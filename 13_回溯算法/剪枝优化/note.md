如何剪枝

1. 优先分配大的任务/优先搜索好的情况

   ```JS
   jobs.sort((a,b)=>b-a) 4088 ms
   ```

   优先分配工作量小的工作会使得工作量大的工作更有可能最后无法被分配

   `2052_将句子分为两行的最小花费.py`

2. 超过 res 立马返回
   ```Python
   # 关键的剪枝1:超出res就返回
   if pathSum > res:
         return
   ```
3. 相同的工作分配只使用第一次(工人都是对称的,要去重)
   ```JS
   if (i >= 1 && workTime[i] === workTime[i - 1]) continue 76 ms
   ```
   类似于`1_全排列允许相同元素` `1_组合总和允许元素相同` `2_子集有重复元素`

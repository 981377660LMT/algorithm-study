用户设置好了自己的网络参数：
　　- 本机的 IP 地址：192.168.1.100
　　- 子网掩码：255.255.255.0
　　- 网关的 IP 地址：192.168.1.1
　　- DNS 的 IP 地址：8.8.8.8
然后他打开浏览器，想要访问 Google，在地址栏输入了网址：www.google.com。
这意味着，浏览器要向 Google 发送一个网页请求的数据包。

1. DNS 协议(域名解析流程)
   **浏览器=>缓存=>hosts=>本地 DNS 服务器=>根域名服务器=>顶级域名服务器=>权威域名服务器=>本地 DNS 服务器=>浏览器**
2. 建立 tcp 连接
3. 发送 HTTP 请求；
4. 服务器处理请求；
5. 返回响应结果；
6. 关闭 TCP 连接；
7. 浏览器解析 HTML；
8. 浏览器布局渲染；

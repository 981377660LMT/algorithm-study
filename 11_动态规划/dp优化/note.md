# 不管是啥 dp，先想暴力

`状态总数x每个状态的决策数x每次状态转移所需时间`

1. 减少状态总数
   - 修改状态表示 (这种题目会有数据量提示，例如 nums[i]<=1e5 `(为什么不是 1e9?)`)
2. 减少状态决策数
   - 单调队列优化
   - 斜率优化 (单调队列维护上/下凸壳)
   - 剪枝
3. 减少状态转移时间
   - 预处理
   - 数据结构优化(线段树查询)

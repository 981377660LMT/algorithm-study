## 1. 两数之和 (Two Sum)

**描述**：给定数组和目标值，返回两个数之和等于目标值的下标。

**思路**：哈希表存"差值→下标"，一次遍历。

```python
def twoSum(nums, target):
    seen = {}
    for i, n in enumerate(nums):
        if target - n in seen:
            return [seen[target - n], i]
        seen[n] = i
```

```typescript
function twoSum(nums: number[], target: number): number[] {
  const visited = new Map<number, number>()
  for (let i = 0; i < nums.length; i++) {
    const need = target - nums[i]
    if (visited.has(need)) return [visited.get(need)!, i]
    visited.set(nums[i], i)
  }
  return []
}
```

## 2. 两数相加 (Add Two Numbers)

**描述**：两个逆序链表表示的整数相加，返回结果链表。

**思路**：模拟加法，维护进位 `carry`，同时遍历两链表。

```python
def addTwoNumbers(l1, l2):
    dummy = cur = ListNode(0)
    carry = 0
    while l1 or l2 or carry:
        val = (l1.val if l1 else 0) + (l2.val if l2 else 0) + carry
        carry, val = divmod(val, 10)
        cur.next = cur = ListNode(val)
        l1 = l1.next if l1 else None
        l2 = l2.next if l2 else None
    return dummy.next
```

```typescript
function addTwoNumbers(l1: ListNode | null, l2: ListNode | null): ListNode | null {
  const dummy = new ListNode(0)
  let cur = dummy,
    carry = 0
  while (l1 || l2 || carry) {
    const val = (l1?.val ?? 0) + (l2?.val ?? 0) + carry
    carry = Math.floor(val / 10)
    cur.next = cur = new ListNode(val % 10)
    l1 = l1?.next ?? null
    l2 = l2?.next ?? null
  }
  return dummy.next
}
```

## 3. 无重复字符的最长子串 (Longest Substring Without Repeating Characters)

**描述**：找字符串中不含重复字符的最长子串长度。

**思路**：滑动窗口，哈希表记录每个字符最新下标，遇到重复则收缩左边界。

```python
def lengthOfLongestSubstring(s):
    last = {}
    res = left = 0
    for right, c in enumerate(s):
        if c in last and last[c] >= left:
            left = last[c] + 1
        last[c] = right
        res = max(res, right - left + 1)
    return res
```

```typescript
function lengthOfLongestSubstring(s: string): number {
  const last = new Map<string, number>()
  let res = 0,
    left = 0
  for (let right = 0; right < s.length; right++) {
    const c = s[right]
    if (last.has(c) && last.get(c)! >= left) left = last.get(c)! + 1
    last.set(c, right)
    res = Math.max(res, right - left + 1)
  }
  return res
}
```

## 4. 寻找两个正序数组的中位数 (Median of Two Sorted Arrays)

**描述**：找两个有序数组合并后的中位数，要求 O(log(m+n))。

**思路**：二分分割较短数组，确保左半部分元素数 = 总数的一半，调整直到左右边界满足条件。

```python
def findMedianSortedArrays(nums1, nums2):
    A, B = sorted([nums1, nums2], key=len)
    m, n = len(A), len(B)
    half = (m + n + 1) // 2
    lo, hi = 0, m
    while lo <= hi:
        i = (lo + hi) // 2
        j = half - i
        if i < m and B[j-1] > A[i]: lo = i + 1
        elif i > 0 and A[i-1] > B[j]: hi = i - 1
        else:
            left_max = max(A[i-1] if i else float('-inf'), B[j-1] if j else float('-inf'))
            if (m + n) % 2: return left_max
            right_min = min(A[i] if i < m else float('inf'), B[j] if j < n else float('inf'))
            return (left_max + right_min) / 2
```

```typescript
function findMedianSortedArrays(nums1: number[], nums2: number[]): number {
  if (nums1.length > nums2.length) [nums1, nums2] = [nums2, nums1]
  const m = nums1.length,
    n = nums2.length,
    half = Math.floor((m + n + 1) / 2)
  let lo = 0,
    hi = m
  while (lo <= hi) {
    const i = (lo + hi) >> 1,
      j = half - i
    if (i < m && nums2[j - 1] > nums1[i]) lo = i + 1
    else if (i > 0 && nums1[i - 1] > nums2[j]) hi = i - 1
    else {
      const lMax = Math.max(i ? nums1[i - 1] : -Infinity, j ? nums2[j - 1] : -Infinity)
      if ((m + n) % 2) return lMax
      const rMin = Math.min(i < m ? nums1[i] : Infinity, j < n ? nums2[j] : Infinity)
      return (lMax + rMin) / 2
    }
  }
  return 0
}
```

## 5. 最长回文子串 (Longest Palindromic Substring)

**描述**：找字符串中最长的回文子串。

**思路**：中心扩展，以每个字符（奇）和相邻字符间（偶）为中心向外扩展。

```python
def longestPalindrome(s):
    res = ""
    for i in range(len(s)):
        for l, r in [(i, i), (i, i+1)]:  # 奇偶中心
            while l >= 0 and r < len(s) and s[l] == s[r]:
                l -= 1; r += 1
            if r - l - 1 > len(res):
                res = s[l+1:r]
    return res
```

```typescript
function longestPalindrome(s: string): string {
  let res = ''
  // [l+1, r) 是回文子串
  const expand = (l: number, r: number) => {
    while (l >= 0 && r < s.length && s[l] === s[r]) {
      l--
      r++
    }
    if (r - l - 1 > res.length) res = s.slice(l + 1, r)
  }
  for (let i = 0; i < s.length; i++) {
    expand(i, i)
    expand(i, i + 1)
  }
  return res
}
```

## 6. Z 字形变换 (Zigzag Conversion)

**描述**：将字符串按Z字形排列后，按行读取。

**思路**：模拟，维护当前行和方向，字符分配到对应行。

```python
def convert(s, numRows):
    if numRows == 1: return s
    rows = [''] * numRows
    row, step = 0, 1
    for c in s:
        rows[row] += c
        if row == 0: step = 1
        elif row == numRows - 1: step = -1
        row += step
    return ''.join(rows)
```

```typescript
function convert(s: string, numRows: number): string {
  if (numRows === 1) return s
  const rows = Array.from({ length: numRows }, () => '')
  let row = 0,
    step = 1
  for (const c of s) {
    rows[row] += c
    if (row === 0) step = 1
    else if (row === numRows - 1) step = -1
    row += step
  }
  return rows.join('')
}
```

## 9. 回文数 (Palindrome Number)

**描述**：判断整数是否为回文数。

**思路**：负数直接 false；翻转后半数字，与前半比较（避免溢出）。

```python
def isPalindrome(x):
    if x < 0 or (x % 10 == 0 and x != 0): return False
    rev = 0
    while x > rev:
        rev = rev * 10 + x % 10
        x //= 10
    return x == rev or x == rev // 10
```

```typescript
function isPalindrome(x: number): boolean {
  if (x < 0 || (x % 10 === 0 && x !== 0)) return false
  let rev = 0
  while (x > rev) {
    rev = rev * 10 + (x % 10)
    x = Math.floor(x / 10)
  }
  return x === rev || x === Math.floor(rev / 10)
}
```

## 10. 正则表达式匹配 (Regular Expression Matching)

**描述**：实现支持 `.` 和 `*` 的正则匹配。

**思路**：DP，`dp[i][j]` 表示 `s[:i]` 与 `p[:j]` 是否匹配。`*` 匹配0次或多次。

```python
def isMatch(s, p):
    m, n = len(s), len(p)
    dp = [[False] * (n+1) for _ in range(m+1)]
    dp[0][0] = True
    for j in range(2, n+1):
        dp[0][j] = dp[0][j-2] and p[j-1] == '*'
    for i in range(1, m+1):
        for j in range(1, n+1):
            if p[j-1] == '*':
                dp[i][j] = dp[i][j-2] or (dp[i-1][j] and (p[j-2] == s[i-1] or p[j-2] == '.'))
            else:
                dp[i][j] = dp[i-1][j-1] and (p[j-1] == s[i-1] or p[j-1] == '.')
    return dp[m][n]
```

```typescript
function isMatch(s: string, p: string): boolean {
  const m = s.length,
    n = p.length
  const dp = Array.from({ length: m + 1 }, () => Array(n + 1).fill(false))
  dp[0][0] = true
  for (let j = 2; j <= n; j++) dp[0][j] = dp[0][j - 2] && p[j - 1] === '*'
  for (let i = 1; i <= m; i++)
    for (let j = 1; j <= n; j++)
      if (p[j - 1] === '*')
        dp[i][j] = dp[i][j - 2] || (dp[i - 1][j] && (p[j - 2] === s[i - 1] || p[j - 2] === '.'))
      else dp[i][j] = dp[i - 1][j - 1] && (p[j - 1] === s[i - 1] || p[j - 1] === '.')
  return dp[m][n]
}
```

## 11. 盛最多水的容器 (Container With Most Water)

**描述**：数组每个元素代表高度，找两条线围成的最大面积。

**思路**：双指针，每次移动**较矮**的那侧（移动较高的不可能增大面积）。

```python
def maxArea(height):
    l, r, res = 0, len(height) - 1, 0
    while l < r:
        res = max(res, min(height[l], height[r]) * (r - l))
        if height[l] < height[r]: l += 1
        else: r -= 1
    return res
```

```typescript
function maxArea(height: number[]): number {
  let l = 0,
    r = height.length - 1,
    res = 0
  while (l < r) {
    res = Math.max(res, Math.min(height[l], height[r]) * (r - l))
    if (height[l] < height[r]) l++
    else r--
  }
  return res
}
```

```python
def intToRoman(num):
```

## 13. 罗马数字转整数 (Roman to Integer)

**描述**：将罗马数字转为整数。

**思路**：从右往左扫描，若当前值 < 右边值则减去，否则加上。

```python
def romanToInt(s):
    d = {'I':1,'V':5,'X':10,'L':50,'C':100,'D':500,'M':1000}
    res = 0
    for i in range(len(s)):
        if i + 1 < len(s) and d[s[i]] < d[s[i+1]]: res -= d[s[i]]
        else: res += d[s[i]]
    return res
```

```typescript
function romanToInt(s: string): number {
  const d: Record<string, number> = { I: 1, V: 5, X: 10, L: 50, C: 100, D: 500, M: 1000 }
  let res = 0
  for (let i = 0; i < s.length; i++) res += d[s[i]] < (d[s[i + 1]] ?? 0) ? -d[s[i]] : d[s[i]]
  return res
}
```

```python
def longestCommonPrefix(strs):
```

## 15. 三数之和 (3Sum)

**描述**：找数组中所有和为0的三元组（不重复）。

**思路**：排序后固定第一个数，剩余两数用双指针，跳过重复元素。

```python
def threeSum(nums):
    nums.sort(); res = []
    for i in range(len(nums) - 2):
        if i > 0 and nums[i] == nums[i-1]: continue
        l, r = i+1, len(nums)-1
        while l < r:
            s = nums[i] + nums[l] + nums[r]
            if s == 0:
                res.append([nums[i], nums[l], nums[r]])
                while l < r and nums[l] == nums[l+1]: l += 1
                while l < r and nums[r] == nums[r-1]: r -= 1
                l += 1; r -= 1
            elif s < 0: l += 1
            else: r -= 1
    return res
```

```typescript
function threeSum(nums: number[]): number[][] {
  nums.sort((a, b) => a - b)
  const res: number[][] = []
  for (let i = 0; i < nums.length - 2; i++) {
    if (i > 0 && nums[i] === nums[i - 1]) continue
    let l = i + 1,
      r = nums.length - 1
    while (l < r) {
      const s = nums[i] + nums[l] + nums[r]
      if (s === 0) {
        res.push([nums[i], nums[l], nums[r]])
        while (l < r && nums[l] === nums[l + 1]) l++
        while (l < r && nums[r] === nums[r - 1]) r--
        l++
        r--
      } else if (s < 0) l++
      else r--
    }
  }
  return res
}
```

```python
def threeSumClosest(nums, target):
```

```python
def fourSum(nums, target):

        for j in range(i+1, n - 2):
            if j > i+1 and nums[j] == nums[j-1]: continue

                    res.append([nums[i], nums[j], nums[l], nums[r]])
                    while l < r and nums[l] == nums[l+1]: l += 1
                    while l < r and nums[r] == nums[r-1]: r -= 1
                    l += 1; r -= 1
                elif s < target: l += 1
                else: r -= 1
    return res
```

```typescript
  }

}
```

## 19. 删除链表的倒数第N个节点 (Remove Nth Node From End of List)

**描述**：删除链表倒数第 n 个节点，一次遍历。

**思路**：快慢指针，快指针先走 n 步，然后同步移动，慢指针到达删除位置的前一个节点。

```python
def removeNthFromEnd(head, n):
    dummy = ListNode(0, head)
    fast = slow = dummy
    for _ in range(n + 1): fast = fast.next
    while fast:
        fast = fast.next; slow = slow.next
    slow.next = slow.next.next
    return dummy.next
```

```typescript
function removeNthFromEnd(head: ListNode | null, n: number): ListNode | null {
  const dummy = new ListNode(0, head)
  let fast: ListNode | null = dummy,
    slow: ListNode | null = dummy
  for (let i = 0; i <= n; i++) fast = fast!.next
  while (fast) {
    fast = fast.next
    slow = slow!.next
  }
  slow!.next = slow!.next!.next
  return dummy.next
}
```

## 20. 有效的括号 (Valid Parentheses)

**描述**：判断括号字符串是否有效（正确闭合）。

**思路**：栈，遇到开括号入栈，遇到闭括号检查栈顶是否匹配。

```python
def isValid(s):
    stack = []
    pairs = {')':'(', ']':'[', '}':'{'}
    for c in s:
        if c in pairs:
            if not stack or stack[-1] != pairs[c]: return False
            stack.pop()
        else:
            stack.append(c)
    return not stack
```

```typescript
function isValid(s: string): boolean {
  const stack: string[] = []
  const pairs: Record<string, string> = { ')': '(', ']': '[', '}': '{' }
  for (const c of s) {
    if (c in pairs) {
      if (stack.pop() !== pairs[c]) return false
    } else stack.push(c)
  }
  return stack.length === 0
}
```

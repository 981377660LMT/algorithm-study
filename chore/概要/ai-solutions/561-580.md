## 561. 数组拆分 (Array Partition)

**描述**：2n 个数配对，最大化 min(pair) 之和。

**思路**：排序后取偶数位。

```python
def arrayPairSum(nums):
    nums.sort()
    return sum(nums[::2])
```

```typescript
function arrayPairSum(nums: number[]): number {
  nums.sort((a, b) => a - b)
  return nums.reduce((s, n, i) => (i % 2 === 0 ? s + n : s), 0)
}
```

## 563. 二叉树的坡度 (Binary Tree Tilt)

**描述**：所有节点坡度之和。坡度 = |左子树和 - 右子树和|。

**思路**：后序遍历返回子树和。

```python
def findTilt(root):
    res = [0]
    def dfs(node):
        if not node: return 0
        l, r = dfs(node.left), dfs(node.right)
        res[0] += abs(l - r)
        return l + r + node.val
    dfs(root); return res[0]
```

```typescript
function findTilt(root: TreeNode | null): number {
  let res = 0
  function dfs(node: TreeNode | null): number {
    if (!node) return 0
    const l = dfs(node.left),
      r = dfs(node.right)
    res += Math.abs(l - r)
    return l + r + node.val
  }
  dfs(root)
  return res
}
```

## 565. 数组嵌套 (Array Nesting)

**描述**：nums 中从 i 开始跳得到的最长环。

**思路**：标记访问过的元素。

```python
def arrayNesting(nums):
    res = 0; visited = [False]*len(nums)
    for i in range(len(nums)):
        if visited[i]: continue
        cnt = 0; j = i
        while not visited[j]:
            visited[j] = True; j = nums[j]; cnt += 1
        res = max(res, cnt)
    return res
```

```typescript
function arrayNesting(nums: number[]): number {
  let res = 0
  const visited = new Array(nums.length).fill(false)
  for (let i = 0; i < nums.length; i++) {
    if (visited[i]) continue
    let cnt = 0,
      j = i
    while (!visited[j]) {
      visited[j] = true
      j = nums[j]
      cnt++
    }
    res = Math.max(res, cnt)
  }
  return res
}
```

## 566. 重塑矩阵 (Reshape the Matrix)

**描述**：将矩阵重塑为 r×c。

**思路**：展平后重填。

```python
def matrixReshape(mat, r, c):
    m, n = len(mat), len(mat[0])
    if m * n != r * c: return mat
    flat = [mat[i][j] for i in range(m) for j in range(n)]
    return [flat[i*c:(i+1)*c] for i in range(r)]
```

```typescript
function matrixReshape(mat: number[][], r: number, c: number): number[][] {
  const m = mat.length,
    n = mat[0].length
  if (m * n !== r * c) return mat
  const flat = mat.flat()
  return Array.from({ length: r }, (_, i) => flat.slice(i * c, (i + 1) * c))
}
```

## 567. 字符串的排列 (Permutation in String)

**描述**：s1 的排列是否是 s2 的子串。

**思路**：滑动窗口 + 字母频率。

```python
from collections import Counter
def checkInclusion(s1, s2):
    n1, n2 = len(s1), len(s2)
    if n1 > n2: return False
    cnt = Counter(s1)
    window = Counter(s2[:n1])
    if window == cnt: return True
    for i in range(n1, n2):
        window[s2[i]] += 1
        old = s2[i-n1]; window[old] -= 1
        if window[old] == 0: del window[old]
        if window == cnt: return True
    return False
```

```typescript
function checkInclusion(s1: string, s2: string): boolean {
  const n1 = s1.length,
    n2 = s2.length
  if (n1 > n2) return false
  const cnt = new Array(26).fill(0),
    win = new Array(26).fill(0)
  const idx = (c: string) => c.charCodeAt(0) - 97
  for (const c of s1) cnt[idx(c)]++
  for (let i = 0; i < n1; i++) win[idx(s2[i])]++
  if (cnt.every((v, i) => v === win[i])) return true
  for (let i = n1; i < n2; i++) {
    win[idx(s2[i])]++
    win[idx(s2[i - n1])]--
    if (cnt.every((v, i) => v === win[i])) return true
  }
  return false
}
```

## 572. 另一棵树的子树 (Subtree of Another Tree)

**描述**：判断 t 是否是 s 的子树。

**思路**：递归判断，每个节点检查是否同结构。

```python
def isSubtree(root, subRoot):
    def same(p, q):
        if not p and not q: return True
        if not p or not q: return False
        return p.val == q.val and same(p.left, q.left) and same(p.right, q.right)
    if not root: return False
    return same(root, subRoot) or isSubtree(root.left, subRoot) or isSubtree(root.right, subRoot)
```

```typescript
function isSubtree(root: TreeNode | null, subRoot: TreeNode | null): boolean {
  function same(p: TreeNode | null, q: TreeNode | null): boolean {
    if (!p && !q) return true
    if (!p || !q) return false
    return p.val === q.val && same(p.left, q.left) && same(p.right, q.right)
  }
  if (!root) return false
  return same(root, subRoot) || isSubtree(root.left, subRoot) || isSubtree(root.right, subRoot)
}
```

## 575. 分糖果 (Distribute Candies)

**描述**：n 种糖果分一半给妹妹，最多几种。

**思路**：种类数和 n/2 取 min。

```python
def distributeCandies(candyType):
    return min(len(set(candyType)), len(candyType) // 2)
```

```typescript
function distributeCandies(candyType: number[]): number {
  return Math.min(new Set(candyType).size, candyType.length >> 1)
}
```

## 576. 出界的路径数 (Out of Boundary Paths)

**描述**：球从 (i,j) 出发最多 N 步，能出界的路径数。

**思路**：DP/记忆化搜索。

```python
def findPaths(m, n, maxMove, startRow, startColumn):
    MOD = 10**9+7
    dp = [[0]*n for _ in range(m)]
    dp[startRow][startColumn] = 1
    res = 0; dirs = [(0,1),(0,-1),(1,0),(-1,0)]
    for _ in range(maxMove):
        ndp = [[0]*n for _ in range(m)]
        for i in range(m):
            for j in range(n):
                if dp[i][j] == 0: continue
                for di, dj in dirs:
                    ni, nj = i+di, j+dj
                    if 0<=ni<m and 0<=nj<n: ndp[ni][nj] = (ndp[ni][nj]+dp[i][j])%MOD
                    else: res = (res+dp[i][j])%MOD
        dp = ndp
    return res
```

```typescript
function findPaths(
  m: number,
  n: number,
  maxMove: number,
  startRow: number,
  startColumn: number
): number {
  const MOD = 1e9 + 7,
    dirs = [
      [0, 1],
      [0, -1],
      [1, 0],
      [-1, 0]
    ]
  let dp = Array.from({ length: m }, () => new Array(n).fill(0))
  dp[startRow][startColumn] = 1
  let res = 0
  for (let mv = 0; mv < maxMove; mv++) {
    const ndp = Array.from({ length: m }, () => new Array(n).fill(0))
    for (let i = 0; i < m; i++)
      for (let j = 0; j < n; j++) {
        if (!dp[i][j]) continue
        for (const [di, dj] of dirs) {
          const ni = i + di,
            nj = j + dj
          if (ni >= 0 && ni < m && nj >= 0 && nj < n) ndp[ni][nj] = (ndp[ni][nj] + dp[i][j]) % MOD
          else res = (res + dp[i][j]) % MOD
        }
      }
    dp = ndp
  }
  return res
}
```

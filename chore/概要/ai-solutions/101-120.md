## 101. 对称二叉树 (Symmetric Tree)

**描述**：判断二叉树是否关于根节点对称。

**思路**：递归比较左子树和右子树的镜像关系。

```python
def isSymmetric(root):
    def check(a, b):
        if not a and not b: return True
        if not a or not b: return False
        return a.val == b.val and check(a.left, b.right) and check(a.right, b.left)
    return check(root.left, root.right)
```

```typescript
function isSymmetric(root: TreeNode | null): boolean {
  const check = (a: TreeNode | null, b: TreeNode | null): boolean => {
    if (!a && !b) return true
    if (!a || !b) return false
    return a.val === b.val && check(a.left, b.right) && check(a.right, b.left)
  }
  return check(root?.left ?? null, root?.right ?? null)
}
```

## 102. 二叉树的层序遍历 (Binary Tree Level Order Traversal)

**描述**：按层返回二叉树节点值。

**思路**：BFS，队列逐层处理。

```python
from collections import deque
def levelOrder(root):
    if not root: return []
    res, queue = [], deque([root])
    while queue:
        level = []
        for _ in range(len(queue)):
            node = queue.popleft()
            level.append(node.val)
            if node.left: queue.append(node.left)
            if node.right: queue.append(node.right)
        res.append(level)
    return res
```

```typescript
function levelOrder(root: TreeNode | null): number[][] {
  if (!root) return []
  const res: number[][] = [],
    queue = [root]
  while (queue.length) {
    const level: number[] = [],
      size = queue.length
    for (let i = 0; i < size; i++) {
      const node = queue.shift()!
      level.push(node.val)
      if (node.left) queue.push(node.left)
      if (node.right) queue.push(node.right)
    }
    res.push(level)
  }
  return res
}
```

## 103. 二叉树的锯齿形层序遍历 (Binary Tree Zigzag Level Order Traversal)

**描述**：层序遍历，但奇数层从右到左。

**思路**：同102，偶数层正常，奇数层反转。

```python
from collections import deque
def zigzagLevelOrder(root):
    if not root: return []
    res, queue = [], deque([root])
    left_to_right = True
    while queue:
        level = []
        for _ in range(len(queue)):
            node = queue.popleft()
            level.append(node.val)
            if node.left: queue.append(node.left)
            if node.right: queue.append(node.right)
        res.append(level if left_to_right else level[::-1])
        left_to_right = not left_to_right
    return res
```

```typescript
function zigzagLevelOrder(root: TreeNode | null): number[][] {
  if (!root) return []
  const res: number[][] = [],
    queue = [root]
  let leftToRight = true
  while (queue.length) {
    const level: number[] = [],
      size = queue.length
    for (let i = 0; i < size; i++) {
      const node = queue.shift()!
      level.push(node.val)
      if (node.left) queue.push(node.left)
      if (node.right) queue.push(node.right)
    }
    res.push(leftToRight ? level : level.reverse())
    leftToRight = !leftToRight
  }
  return res
}
```

## 104. 二叉树的最大深度 (Maximum Depth of Binary Tree)

**描述**：返回二叉树最大深度。

**思路**：递归，`1 + max(左深度, 右深度)`。

```python
def maxDepth(root):
    if not root: return 0
    return 1 + max(maxDepth(root.left), maxDepth(root.right))
```

```typescript
function maxDepth(root: TreeNode | null): number {
  if (!root) return 0
  return 1 + Math.max(maxDepth(root.left), maxDepth(root.right))
}
```

## 105. 从前序与中序遍历序列构造二叉树 (Construct Binary Tree from Preorder and Inorder Traversal)

**描述**：根据前序和中序遍历还原二叉树。

**思路**：前序第一个是根，在中序中找到根的位置，分割左右子树递归构建。

```python
def buildTree(preorder, inorder):
    if not preorder: return None
    root = TreeNode(preorder[0])
    mid = inorder.index(preorder[0])
    root.left = buildTree(preorder[1:mid+1], inorder[:mid])
    root.right = buildTree(preorder[mid+1:], inorder[mid+1:])
    return root
```

```typescript
function buildTree(preorder: number[], inorder: number[]): TreeNode | null {
  if (!preorder.length) return null
  const root = new TreeNode(preorder[0])
  const mid = inorder.indexOf(preorder[0])
  root.left = buildTree(preorder.slice(1, mid + 1), inorder.slice(0, mid))
  root.right = buildTree(preorder.slice(mid + 1), inorder.slice(mid + 1))
  return root
}
```

## 106. 从中序与后序遍历序列构造二叉树 (Construct Binary Tree from Inorder and Postorder Traversal)

**描述**：根据中序和后序遍历还原二叉树。

**思路**：后序最后一个是根，类似105。

```python
def buildTree(inorder, postorder):
    if not postorder: return None
    root = TreeNode(postorder[-1])
    mid = inorder.index(postorder[-1])
    root.left = buildTree(inorder[:mid], postorder[:mid])
    root.right = buildTree(inorder[mid+1:], postorder[mid:-1])
    return root
```

```typescript
function buildTree(inorder: number[], postorder: number[]): TreeNode | null {
  if (!postorder.length) return null
  const root = new TreeNode(postorder[postorder.length - 1])
  const mid = inorder.indexOf(root.val)
  root.left = buildTree(inorder.slice(0, mid), postorder.slice(0, mid))
  root.right = buildTree(inorder.slice(mid + 1), postorder.slice(mid, -1))
  return root
}
```

## 107. 二叉树的层序遍历 II (Binary Tree Level Order Traversal II)

**描述**：自底向上的层序遍历。

**思路**：同102，最后 reverse 结果。

```python
from collections import deque
def levelOrderBottom(root):

    res, queue = [], deque([root])
    while queue:
        level = []

            level.append(node.val)
```

```python
def sortedArrayToBST(nums):
    if not nums: return None
```

```typescript

  return new TreeNode(
    nums[mid],
    sortedArrayToBST(nums.slice(0, mid)),
    sortedArrayToBST(nums.slice(mid + 1))
  )
}
```

## 109. 有序链表转换二叉搜索树 (Convert Sorted List to Binary Search Tree)

**描述**：有序链表转平衡BST。

**思路**：快慢指针找中间节点作为根，断开后递归。

```python
def sortedListToBST(head):
    if not head: return None
    if not head.next: return TreeNode(head.val)
    prev, slow, fast = None, head, head
    while fast and fast.next:
        prev = slow; slow = slow.next; fast = fast.next.next
    prev.next = None
    root = TreeNode(slow.val)
    root.left = sortedListToBST(head)
    root.right = sortedListToBST(slow.next)
    return root
```

```typescript
function sortedListToBST(head: ListNode | null): TreeNode | null {
  if (!head) return null
  if (!head.next) return new TreeNode(head.val)
  let prev: ListNode | null = null,
    slow = head,
    fast: ListNode | null = head
  while (fast && fast.next) {
    prev = slow
    slow = slow.next!
    fast = fast.next.next
  }
  prev!.next = null
  const root = new TreeNode(slow.val)
  root.left = sortedListToBST(head)
  root.right = sortedListToBST(slow.next)
  return root
}
```

## 110. 平衡二叉树 (Balanced Binary Tree)

**描述**：判断二叉树是否高度平衡（任意节点左右子树高度差不超过1）。

**思路**：递归求高度，遇到不平衡返回 -1 提前终止。

```python
def isBalanced(root):
    def height(node):
        if not node: return 0
        l, r = height(node.left), height(node.right)
        if l == -1 or r == -1 or abs(l - r) > 1: return -1
        return 1 + max(l, r)
    return height(root) != -1
```

```typescript
function isBalanced(root: TreeNode | null): boolean {
  const height = (node: TreeNode | null): number => {
    if (!node) return 0
    const l = height(node.left),
      r = height(node.right)
    if (l === -1 || r === -1 || Math.abs(l - r) > 1) return -1
    return 1 + Math.max(l, r)
  }
  return height(root) !== -1
}
```

## 111. 二叉树的最小深度 (Minimum Depth of Binary Tree)

**描述**：根到最近叶子节点的最短路径长度。

**思路**：递归，注意单侧为空时不算叶子。

```python
def minDepth(root):
    if not root: return 0
    if not root.left: return 1 + minDepth(root.right)
    if not root.right: return 1 + minDepth(root.left)
    return 1 + min(minDepth(root.left), minDepth(root.right))
```

```typescript
function minDepth(root: TreeNode | null): number {
  if (!root) return 0
  if (!root.left) return 1 + minDepth(root.right)
  if (!root.right) return 1 + minDepth(root.left)
  return 1 + Math.min(minDepth(root.left), minDepth(root.right))
}
```

## 112. 路径总和 (Path Sum)

**描述**：判断根到叶路径上节点值之和是否等于目标值。

**思路**：递归，到叶子节点时检查剩余值是否为0。

```python
def hasPathSum(root, targetSum):
    if not root: return False
    if not root.left and not root.right: return root.val == targetSum
    return hasPathSum(root.left, targetSum - root.val) or hasPathSum(root.right, targetSum - root.val)
```

```typescript
function hasPathSum(root: TreeNode | null, targetSum: number): boolean {
  if (!root) return false
  if (!root.left && !root.right) return root.val === targetSum
  return hasPathSum(root.left, targetSum - root.val) || hasPathSum(root.right, targetSum - root.val)
}
```

## 113. 路径总和 II (Path Sum II)

**描述**：返回所有根到叶路径总和等于目标值的路径。

**思路**：回溯DFS，记录路径。

```python
def pathSum(root, targetSum):
    res = []
    def dfs(node, remain, path):
        if not node: return
        path.append(node.val)
        if not node.left and not node.right and remain == node.val:
            res.append(path[:])
        dfs(node.left, remain - node.val, path)
        dfs(node.right, remain - node.val, path)
        path.pop()
    dfs(root, targetSum, [])
    return res
```

```typescript
function pathSum(root: TreeNode | null, targetSum: number): number[][] {
  const res: number[][] = []
  const dfs = (node: TreeNode | null, remain: number, path: number[]) => {
    if (!node) return
    path.push(node.val)
    if (!node.left && !node.right && remain === node.val) res.push([...path])
    dfs(node.left, remain - node.val, path)
    dfs(node.right, remain - node.val, path)
    path.pop()
  }
  dfs(root, targetSum, [])
  return res
}
```

## 114. 二叉树展开为链表 (Flatten Binary Tree to Linked List)

**描述**：原地将二叉树展开为单链表（先序遍历顺序，用右指针连接）。

**思路**：前序遍历的反向（右→左→根），用 prev 指针串联。

```python
def flatten(root):
    prev = None
    def dfs(node):
        nonlocal prev
        if not node: return
        dfs(node.right)
        dfs(node.left)
        node.right = prev
        node.left = None
        prev = node
    dfs(root)
```

```typescript
function flatten(root: TreeNode | null): void {
  let prev: TreeNode | null = null
  const dfs = (node: TreeNode | null) => {
    if (!node) return
    dfs(node.right)
    dfs(node.left)
    node.right = prev
    node.left = null
    prev = node
  }
  dfs(root)
}
```

## 115. 不同的子序列 (Distinct Subsequences)

**描述**：s 的子序列中有多少个等于 t。

**思路**：DP，`dp[j]` = s[:i] 中 t[:j] 出现的次数。匹配时 `dp[j] += dp[j-1]`。

```python
def numDistinct(s, t):
    m, n = len(s), len(t)
    dp = [0] * (n + 1)
    dp[0] = 1
    for c in s:
        for j in range(n, 0, -1):
            if c == t[j-1]: dp[j] += dp[j-1]
    return dp[n]
```

```typescript
function numDistinct(s: string, t: string): number {
  const n = t.length,
    dp = new Array(n + 1).fill(0)
  dp[0] = 1
  for (const c of s) for (let j = n; j >= 1; j--) if (c === t[j - 1]) dp[j] += dp[j - 1]
  return dp[n]
}
```

## 116. 填充每个节点的下一个右侧节点指针 (Populating Next Right Pointers in Each Node)

**描述**：完美二叉树，填充每个节点的 next 指针指向右侧节点。

**思路**：层序利用已建立的 next 指针，O(1) 空间。

```python
def connect(root):
    node = root
    while node and node.left:
        cur = node
        while cur:
            cur.left.next = cur.right
            if cur.next: cur.right.next = cur.next.left
            cur = cur.next
        node = node.left
    return root
```

```typescript
function connect(root: Node_ | null): Node_ | null {
  let node = root
  while (node && node.left) {
    let cur: Node_ | null = node
    while (cur) {
      cur.left!.next = cur.right
      if (cur.next) cur.right!.next = cur.next.left
      cur = cur.next
    }
    node = node.left
  }
  return root
}
```

## 118. 杨辉三角 (Pascal's Triangle)

**描述**：生成杨辉三角的前 numRows 行。

**思路**：迭代构建，每行 `row[j] = prev[j-1] + prev[j]`。

```python
def generate(numRows):
    res = [[1]]
    for i in range(1, numRows):
        prev = res[-1]
        row = [1] + [prev[j-1] + prev[j] for j in range(1, i)] + [1]
        res.append(row)
    return res
```

```typescript
function generate(numRows: number): number[][] {
  const res: number[][] = [[1]]
  for (let i = 1; i < numRows; i++) {
    const prev = res[i - 1]
    const row = [1]
    for (let j = 1; j < i; j++) row.push(prev[j - 1] + prev[j])
    row.push(1)
    res.push(row)
  }
  return res
}
```

## 119. 杨辉三角 II (Pascal's Triangle II)

**描述**：返回杨辉三角第 rowIndex 行。

**思路**：一维数组原地更新，从后往前避免覆盖。

```python
def getRow(rowIndex):
    row = [1] * (rowIndex + 1)
    for i in range(2, rowIndex + 1):
        for j in range(i - 1, 0, -1):
            row[j] += row[j-1]
    return row
```

```typescript
function getRow(rowIndex: number): number[] {

  for (let i = 2; i <= rowIndex; i++) for (let j = i - 1; j >= 1; j--) row[j] += row[j - 1]
  return row
}
```

## 120. 三角形最小路径和 (Triangle)

**描述**：三角形从顶到底的最小路径和。

**思路**：自底向上DP，`dp[j] = triangle[i][j] + min(dp[j], dp[j+1])`。

```python
def minimumTotal(triangle):
    dp = triangle[-1][:]
    for i in range(len(triangle)-2, -1, -1):
        for j in range(i+1):
            dp[j] = triangle[i][j] + min(dp[j], dp[j+1])
    return dp[0]
```

```typescript
function minimumTotal(triangle: number[][]): number {
  const dp = [...triangle[triangle.length - 1]]
  for (let i = triangle.length - 2; i >= 0; i--)
    for (let j = 0; j <= i; j++) dp[j] = triangle[i][j] + Math.min(dp[j], dp[j + 1])
  return dp[0]
}
```

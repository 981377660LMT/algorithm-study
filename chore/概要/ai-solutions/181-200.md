## 186-187. 重复的 DNA 序列 (Repeated DNA Sequences)

**描述**：找 DNA 字符串中出现超过一次的长度为10的子串。

**思路**：滑动窗口 + 哈希集合记录看过的子串。

```python
def findRepeatedDnaSequences(s):
    seen, res = set(), set()
    for i in range(len(s) - 9):
        sub = s[i:i+10]
        if sub in seen: res.add(sub)
        else: seen.add(sub)
    return list(res)
```

```typescript
function findRepeatedDnaSequences(s: string): string[] {
  const seen = new Set<string>(),
    res = new Set<string>()
  for (let i = 0; i <= s.length - 10; i++) {
    const sub = s.slice(i, i + 10)
    if (seen.has(sub)) res.add(sub)
    else seen.add(sub)
  }
  return [...res]
}
```

## 188. 买卖股票的最佳时机 IV (Best Time to Buy and Sell Stock IV)

**描述**：最多 k 次买卖的最大利润。

**思路**：DP，`dp[j][0/1]` 表示第 j 次交易持有/不持有的最大利润。k >= n/2 时等价于无限次。

```python
def maxProfit(k, prices):
    n = len(prices)
    if k >= n // 2:
        return sum(max(0, prices[i+1] - prices[i]) for i in range(n-1))
    dp = [[0, float('-inf')] for _ in range(k+1)]
    for p in prices:
        for j in range(k, 0, -1):
            dp[j][0] = max(dp[j][0], dp[j][1] + p)
            dp[j][1] = max(dp[j][1], dp[j-1][0] - p)
    return dp[k][0]
```

```typescript
function maxProfit(k: number, prices: number[]): number {
  const n = prices.length
  if (k >= Math.floor(n / 2)) {
    let res = 0
    for (let i = 1; i < n; i++) res += Math.max(0, prices[i] - prices[i - 1])
    return res
  }
  const dp = Array.from({ length: k + 1 }, () => [0, -Infinity])
  for (const p of prices)
    for (let j = k; j >= 1; j--) {
      dp[j][0] = Math.max(dp[j][0], dp[j][1] + p)
      dp[j][1] = Math.max(dp[j][1], dp[j - 1][0] - p)
    }
  return dp[k][0]
}
```

## 189. 轮转数组 (Rotate Array)

**描述**：将数组右移 k 个位置。

**思路**：三次反转——全部反转，翻转前 k 个，翻转后 n-k 个。

```python
def rotate(nums, k):
    n = len(nums); k %= n
    nums.reverse()
    nums[:k] = nums[:k][::-1]
    nums[k:] = nums[k:][::-1]
```

```typescript
function rotate(nums: number[], k: number): void {
  k %= nums.length
  const reverse = (l: number, r: number) => {
    while (l < r) {
      ;[nums[l], nums[r]] = [nums[r], nums[l]]
      l++
      r--
    }
  }
  reverse(0, nums.length - 1)
  reverse(0, k - 1)
  reverse(k, nums.length - 1)
}
```

## 190. 颠倒二进制位 (Reverse Bits)

**描述**：颠倒32位无符号整数的二进制位。

**思路**：逐位取出，结果左移后更新。

```python
def reverseBits(n):
    res = 0
    for _ in range(32):
        res = (res << 1) | (n & 1)
        n >>= 1
    return res
```

```typescript
function reverseBits(n: number): number {
  let res = 0
  for (let i = 0; i < 32; i++) {
    res = (res << 1) | (n & 1)
    n >>>= 1
  }
  return res >>> 0
}
```

## 191. 位1的个数 (Number of 1 Bits)

**描述**：返回无符号整数二进制中 1 的个数。

**思路**：`n & (n-1)` 消去最低位的 1，数次数。

```python
def hammingWeight(n):
    cnt = 0
    while n:
        n &= n - 1; cnt += 1
    return cnt
```

```typescript
function hammingWeight(n: number): number {
  let cnt = 0
  while (n) {
    n &= n - 1
    cnt++
  }
  return cnt
}
```

## 198. 打家劫舍 (House Robber)

**描述**：不能偷相邻房屋，求最大偷窃金额。

**思路**：DP，`dp[i] = max(dp[i-1], dp[i-2] + nums[i])`，滚动变量优化。

```python
def rob(nums):
    prev2, prev1 = 0, 0
    for n in nums:
        prev2, prev1 = prev1, max(prev1, prev2 + n)
    return prev1
```

```typescript
function rob(nums: number[]): number {
  let prev2 = 0,
    prev1 = 0
  for (const n of nums) [prev2, prev1] = [prev1, Math.max(prev1, prev2 + n)]
  return prev1
}
```

## 199. 二叉树的右视图 (Binary Tree Right Side View)

**描述**：返回从右侧看二叉树能看到的节点值。

**思路**：层序遍历，每层取最后一个节点。

```python
from collections import deque
def rightSideView(root):
    if not root: return []
    res, queue = [], deque([root])
    while queue:
        for i in range(len(queue)):
            node = queue.popleft()
            if i == 0: res.append(node.val)  # 注意：从右往左入队
            if node.right: queue.append(node.right)
            if node.left: queue.append(node.left)
    return res
```

```typescript
function rightSideView(root: TreeNode | null): number[] {
  if (!root) return []
  const res: number[] = [],
    queue = [root]
  while (queue.length) {
    res.push(queue[queue.length - 1].val)
    const size = queue.length
    for (let i = 0; i < size; i++) {
      const node = queue.shift()!
      if (node.left) queue.push(node.left)
      if (node.right) queue.push(node.right)
    }
  }
  return res
}
```

## 200. 岛屿数量 (Number of Islands)

**描述**：二维网格中'1'表示陆地，'0'表示水，求岛屿数量。

**思路**：遍历每个 '1'，DFS/BFS 将连通的 '1' 全标记为 '0'，计数。

```python
def numIslands(grid):
    m, n = len(grid), len(grid[0])
    cnt = 0
    def dfs(i, j):
        if i < 0 or i >= m or j < 0 or j >= n or grid[i][j] != '1': return
        grid[i][j] = '0'
        for di, dj in [(0,1),(0,-1),(1,0),(-1,0)]: dfs(i+di, j+dj)
    for i in range(m):
        for j in range(n):
            if grid[i][j] == '1':
                dfs(i, j); cnt += 1
    return cnt
```

```typescript
function numIslands(grid: string[][]): number {
  const m = grid.length,
    n = grid[0].length
  let cnt = 0
  const dfs = (i: number, j: number) => {
    if (i < 0 || i >= m || j < 0 || j >= n || grid[i][j] !== '1') return
    grid[i][j] = '0'
    dfs(i + 1, j)
    dfs(i - 1, j)
    dfs(i, j + 1)
    dfs(i, j - 1)
  }
  for (let i = 0; i < m; i++)
    for (let j = 0; j < n; j++)
      if (grid[i][j] === '1') {
        dfs(i, j)
        cnt++
      }
  return cnt
}
```

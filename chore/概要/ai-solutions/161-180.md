## 164. 最大间距 (Maximum Gap)

**描述**：无序数组排序后相邻元素最大差值，要求 O(n)。

**思路**：桶排序/基数排序。将数均匀放入桶，最大间距一定出现在不同桶之间。

```python
def maximumGap(nums):
    if len(nums) < 2: return 0
    lo, hi = min(nums), max(nums)
    if lo == hi: return 0
    n = len(nums)
    size = max(1, (hi - lo) // (n - 1))
    bucket_count = (hi - lo) // size + 1
    buckets = [[float('inf'), float('-inf')] for _ in range(bucket_count)]
    for x in nums:
        idx = (x - lo) // size
        buckets[idx][0] = min(buckets[idx][0], x)
        buckets[idx][1] = max(buckets[idx][1], x)
    res, prev_max = 0, lo
    for mn, mx in buckets:
        if mn == float('inf'): continue
        res = max(res, mn - prev_max)
        prev_max = mx
    return res
```

```typescript
function maximumGap(nums: number[]): number {
  if (nums.length < 2) return 0
  const lo = Math.min(...nums),
    hi = Math.max(...nums)
  if (lo === hi) return 0
  const n = nums.length,
    size = Math.max(1, Math.floor((hi - lo) / (n - 1)))
  const cnt = Math.floor((hi - lo) / size) + 1
  const bucketMin = new Array(cnt).fill(Infinity),
    bucketMax = new Array(cnt).fill(-Infinity)
  for (const x of nums) {
    const idx = Math.floor((x - lo) / size)
    bucketMin[idx] = Math.min(bucketMin[idx], x)
    bucketMax[idx] = Math.max(bucketMax[idx], x)
  }
  let res = 0,
    prevMax = lo
  for (let i = 0; i < cnt; i++) {
    if (bucketMin[i] === Infinity) continue
    res = Math.max(res, bucketMin[i] - prevMax)
    prevMax = bucketMax[i]
  }
  return res
}
```

## 165. 比较版本号 (Compare Version Numbers)

**描述**：比较两个版本号字符串。

**思路**：按 `.` 分割后逐段比较整数。

```python
def compareVersion(version1, version2):
    v1 = list(map(int, version1.split('.')))
    v2 = list(map(int, version2.split('.')))
    for i in range(max(len(v1), len(v2))):
        a = v1[i] if i < len(v1) else 0
        b = v2[i] if i < len(v2) else 0
        if a < b: return -1
        if a > b: return 1
    return 0
```

```typescript
function compareVersion(version1: string, version2: string): number {
  const v1 = version1.split('.').map(Number),
    v2 = version2.split('.').map(Number)
  for (let i = 0; i < Math.max(v1.length, v2.length); i++) {
    const a = v1[i] ?? 0,
      b = v2[i] ?? 0
    if (a < b) return -1
    if (a > b) return 1
  }
  return 0
}
```

## 166. 分数到循环小数 (Fraction to Recurring Decimal)

**描述**：将分数转为字符串表示（循环部分用括号标注）。

**思路**：模拟长除法，哈希表记录余数出现的位置检测循环。

```python
def fractionToDecimal(numerator, denominator):
    if numerator == 0: return '0'
    res = ''
    if (numerator < 0) ^ (denominator < 0): res += '-'
    n, d = abs(numerator), abs(denominator)
    res += str(n // d)
    remainder = n % d
    if remainder == 0: return res
    res += '.'
    seen = {}
    while remainder:
        if remainder in seen:
            res = res[:seen[remainder]] + '(' + res[seen[remainder]:] + ')'
            break
        seen[remainder] = len(res)
        remainder *= 10
        res += str(remainder // d)
        remainder %= d
    return res
```

```typescript
function fractionToDecimal(numerator: number, denominator: number): string {
  if (numerator === 0) return '0'
  let res = ''
  if (numerator < 0 !== denominator < 0) res += '-'
  let n = Math.abs(numerator),
    d = Math.abs(denominator)
  res += Math.floor(n / d)
  let remainder = n % d
  if (remainder === 0) return res
  res += '.'
  const seen = new Map<number, number>()
  while (remainder) {
    if (seen.has(remainder)) {
      const idx = seen.get(remainder)!
      return res.slice(0, idx) + '(' + res.slice(idx) + ')'
    }
    seen.set(remainder, res.length)
    remainder *= 10
    res += Math.floor(remainder / d)
    remainder %= d
  }
  return res
}
```

## 167. 两数之和 II - 输入有序数组 (Two Sum II - Input Array Is Sorted)

**描述**：有序数组中找两数之和等于目标值（1-indexed）。

**思路**：双指针从两端收缩。

```python
def twoSum(numbers, target):
    l, r = 0, len(numbers) - 1
    while l < r:
        s = numbers[l] + numbers[r]
        if s == target: return [l+1, r+1]
        elif s < target: l += 1
        else: r -= 1
```

```typescript
function twoSum(numbers: number[], target: number): number[] {
  let l = 0,
    r = numbers.length - 1
  while (l < r) {
    const s = numbers[l] + numbers[r]
    if (s === target) return [l + 1, r + 1]
    else if (s < target) l++
    else r--
  }
  return []
}
```

## 168. Excel 表列名称 (Excel Sheet Column Title)

**描述**：给定列号，返回 Excel 列名（1→A, 28→AB）。

**思路**：26进制转换，注意是1-indexed（每次 n-1 再取模）。

```python
def convertToTitle(columnNumber):
    res = ''
    while columnNumber:
        columnNumber -= 1
        res = chr(columnNumber % 26 + ord('A')) + res
        columnNumber //= 26
    return res
```

```typescript
function convertToTitle(columnNumber: number): string {
  let res = ''
  while (columnNumber) {
    columnNumber--
    res = String.fromCharCode((columnNumber % 26) + 65) + res
    columnNumber = Math.floor(columnNumber / 26)
  }
  return res
}
```

## 169. 多数元素 (Majority Element)

**描述**：找数组中出现次数超过 n/2 的元素。

**思路**：摩尔投票法，候选人和计数器，不同则抵消。

```python
def majorityElement(nums):
    cnt = candidate = 0
    for n in nums:
        if cnt == 0: candidate = n
        cnt += 1 if n == candidate else -1
    return candidate
```

```typescript
function majorityElement(nums: number[]): number {
  let cnt = 0,
    candidate = 0
  for (const n of nums) {
    if (cnt === 0) candidate = n
    cnt += n === candidate ? 1 : -1
  }
  return candidate
}
```

## 171. Excel 表列序号 (Excel Sheet Column Number)

**描述**：Excel 列名转列号（A→1, AB→28）。

**思路**：26进制，逐位累乘。

```python
def titleToNumber(columnTitle):
    res = 0
    for c in columnTitle:
        res = res * 26 + ord(c) - ord('A') + 1
    return res
```

```typescript
function titleToNumber(columnTitle: string): number {
  let res = 0
  for (const c of columnTitle) res = res * 26 + c.charCodeAt(0) - 64
  return res
}
```

## 172. 阶乘后的零 (Factorial Trailing Zeroes)

**描述**：n! 末尾有多少个零。

**思路**：统计因子5的个数（每个5贡献一个0），`n/5 + n/25 + n/125 + ...`。

```python
def trailingZeroes(n):
    res = 0
    while n >= 5:
        n //= 5; res += n
    return res
```

```typescript
function trailingZeroes(n: number): number {
  let res = 0
  while (n >= 5) {
    n = Math.floor(n / 5)
    res += n
  }
  return res
}
```

## 173. 二叉搜索树迭代器 (Binary Search Tree Iterator)

**描述**：实现BST中序遍历迭代器。

**思路**：受控的中序遍历栈——初始压入所有左节点，next 弹栈并处理右子树。

```python
class BSTIterator:
    def __init__(self, root):
        self.stack = []
        self._push_left(root)
    def _push_left(self, node):
        while node:
            self.stack.append(node); node = node.left
    def next(self):
        node = self.stack.pop()
        self._push_left(node.right)
        return node.val
    def hasNext(self):
        return len(self.stack) > 0
```

```typescript
class BSTIterator {
  private stack: TreeNode[] = []
  constructor(root: TreeNode | null) {
    this.pushLeft(root)
  }
  private pushLeft(node: TreeNode | null) {
    while (node) {
      this.stack.push(node)
      node = node.left
    }
  }
  next(): number {
    const node = this.stack.pop()!
    this.pushLeft(node.right)
    return node.val
  }
  hasNext(): boolean {
    return this.stack.length > 0
  }
}
```

## 174. 地下城游戏 (Dungeon Game)

**描述**：公主在右下角，骑士从左上到右下，每格加减血量，求初始最少血量。

**思路**：倒过来DP，从右下往左上，`dp[i][j]` = 进入(i,j)至少需要的血量。

```python
def calculateMinimumHP(dungeon):
    m, n = len(dungeon), len(dungeon[0])
    dp = [[float('inf')] * (n+1) for _ in range(m+1)]
    dp[m][n-1] = dp[m-1][n] = 1
    for i in range(m-1, -1, -1):
        for j in range(n-1, -1, -1):
            dp[i][j] = max(1, min(dp[i+1][j], dp[i][j+1]) - dungeon[i][j])
    return dp[0][0]
```

```typescript
function calculateMinimumHP(dungeon: number[][]): number {
  const m = dungeon.length,
    n = dungeon[0].length
  const dp = Array.from({ length: m + 1 }, () => new Array(n + 1).fill(Infinity))
  dp[m][n - 1] = dp[m - 1][n] = 1
  for (let i = m - 1; i >= 0; i--)
    for (let j = n - 1; j >= 0; j--)
      dp[i][j] = Math.max(1, Math.min(dp[i + 1][j], dp[i][j + 1]) - dungeon[i][j])
  return dp[0][0]
}
```

## 179. 最大数 (Largest Number)

**描述**：将非负整数数组排列成最大的数。

**思路**：自定义排序，比较 `a+b` 和 `b+a` 的字典序。

```python
from functools import cmp_to_key
def largestNumber(nums):
    nums = list(map(str, nums))
    nums.sort(key=cmp_to_key(lambda a, b: 1 if a+b < b+a else -1))
    res = ''.join(nums)
    return '0' if res[0] == '0' else res
```

```typescript
function largestNumber(nums: number[]): string {
  const strs = nums.map(String)
  strs.sort((a, b) => (b + a).localeCompare(a + b))
  const res = strs.join('')
  return res[0] === '0' ? '0' : res
}
```

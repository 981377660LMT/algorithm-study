## 61. 旋转链表 (Rotate List)

**描述**：将链表右旋 k 个位置。

**思路**：连成环，算出总长 n，在 `n - k % n` 处断开。

```python
def rotateRight(head, k):
    if not head or not head.next or k == 0: return head
    cur, n = head, 1
    while cur.next: cur = cur.next; n += 1
    cur.next = head  # 成环
    step = n - k % n
    for _ in range(step): cur = cur.next
    new_head = cur.next
    cur.next = None
    return new_head
```

```typescript
function rotateRight(head: ListNode | null, k: number): ListNode | null {
  if (!head || !head.next || k === 0) return head
  let cur = head,
    n = 1
  while (cur.next) {
    cur = cur.next
    n++
  }
  cur.next = head
  const step = n - (k % n)
  for (let i = 0; i < step; i++) cur = cur.next!
  const newHead = cur.next
  cur.next = null
  return newHead
}
```

## 62. 不同路径 (Unique Paths)

**描述**：m×n 网格从左上到右下的不同路径数（只能向右或向下）。

**思路**：DP（或组合数 C(m+n-2, m-1)）。一维DP：`dp[j] += dp[j-1]`。

```python
def uniquePaths(m, n):
    dp = [1] * n
    for _ in range(1, m):
        for j in range(1, n):
            dp[j] += dp[j-1]
    return dp[-1]
```

```typescript
function uniquePaths(m: number, n: number): number {
  const dp = new Array(n).fill(1)
  for (let i = 1; i < m; i++) for (let j = 1; j < n; j++) dp[j] += dp[j - 1]
  return dp[n - 1]
}
```

## 63. 不同路径 II (Unique Paths II)

**描述**：同62，但网格中有障碍物（值为1）。

**思路**：同样一维DP，遇到障碍物的格子路径数置0。

```python
def uniquePathsWithObstacles(obstacleGrid):
    n = len(obstacleGrid[0])
    dp = [0] * n
    dp[0] = 1
    for row in obstacleGrid:
        for j in range(n):
            if row[j] == 1: dp[j] = 0
            elif j > 0: dp[j] += dp[j-1]
    return dp[-1]
```

```typescript
function uniquePathsWithObstacles(obstacleGrid: number[][]): number {
  const n = obstacleGrid[0].length,
    dp = new Array(n).fill(0)
  dp[0] = 1
  for (const row of obstacleGrid)
    for (let j = 0; j < n; j++) {
      if (row[j] === 1) dp[j] = 0
      else if (j > 0) dp[j] += dp[j - 1]
    }
  return dp[n - 1]
}
```

## 64. 最小路径和 (Minimum Path Sum)

**描述**：m×n 网格，从左上到右下路径上数字总和最小。

**思路**：一维DP，`dp[j] = grid[i][j] + min(dp[j], dp[j-1])`。

```python
def minPathSum(grid):
    m, n = len(grid), len(grid[0])
    dp = [float('inf')] * n
    dp[0] = 0
    for i in range(m):
        dp[0] += grid[i][0]
        for j in range(1, n):
            dp[j] = grid[i][j] + min(dp[j], dp[j-1])
    return dp[-1]
```

```typescript
function minPathSum(grid: number[][]): number {
  const m = grid.length,
    n = grid[0].length
  const dp = new Array(n).fill(Infinity)
  dp[0] = 0
  for (let i = 0; i < m; i++) {
    dp[0] += grid[i][0]
    for (let j = 1; j < n; j++) dp[j] = grid[i][j] + Math.min(dp[j], dp[j - 1])
  }
  return dp[n - 1]
}
```

## 66. 加一 (Plus One)

**描述**：大整数（数组表示）加一。

**思路**：从末位开始加，处理进位；若全部进位则首位插入1。

```python
def plusOne(digits):
    for i in range(len(digits)-1, -1, -1):
        if digits[i] < 9:
            digits[i] += 1; return digits
        digits[i] = 0
    return [1] + digits
```

```typescript
function plusOne(digits: number[]): number[] {
  for (let i = digits.length - 1; i >= 0; i--) {
    if (digits[i] < 9) {
      digits[i]++
      return digits
    }
    digits[i] = 0
  }
  return [1, ...digits]
}
```

## 67. 二进制求和 (Add Binary)

**描述**：两个二进制字符串相加。

**思路**：从末尾逐位相加，维护进位。

```python
def addBinary(a, b):
    res, carry = '', 0
    i, j = len(a)-1, len(b)-1
    while i >= 0 or j >= 0 or carry:
        s = carry
        if i >= 0: s += int(a[i]); i -= 1
        if j >= 0: s += int(b[j]); j -= 1
        res = str(s % 2) + res
        carry = s // 2
    return res
```

```typescript
function addBinary(a: string, b: string): string {
  let i = a.length - 1,
    j = b.length - 1,
    carry = 0,
    res = ''
  while (i >= 0 || j >= 0 || carry) {
    let s = carry
    if (i >= 0) s += +a[i--]
    if (j >= 0) s += +b[j--]
    res = (s % 2) + res
    carry = Math.floor(s / 2)
  }
  return res
}
```

## 68. 文本左右对齐 (Text Justification)

**描述**：将单词数组按指定宽度左右对齐。

**思路**：贪心分行，计算每行空格分配（均匀+余数左优先），最后一行左对齐。

```python
def fullJustify(words, maxWidth):
    res, line, line_len = [], [], 0
    for w in words:
        if line_len + len(w) + len(line) > maxWidth:
            for i in range(maxWidth - line_len):
                line[i % (len(line)-1 or 1)] += ' '
            res.append(''.join(line))
            line, line_len = [], 0
        line.append(w); line_len += len(w)
    res.append(' '.join(line).ljust(maxWidth))
    return res
```

```typescript
function fullJustify(words: string[], maxWidth: number): string[] {
  const res: string[] = []
  let line: string[] = [],
    lineLen = 0
  for (const w of words) {
    if (lineLen + w.length + line.length > maxWidth) {
      for (let i = 0; i < maxWidth - lineLen; i++) line[i % (line.length - 1 || 1)] += ' '
      res.push(line.join(''))
      line = []
      lineLen = 0
    }
    line.push(w)
    lineLen += w.length
  }
  res.push(line.join(' ').padEnd(maxWidth))
  return res
}
```

## 69. x 的平方根 (Sqrt(x))

**描述**：计算 x 的算术平方根（取整）。

**思路**：二分查找，找最大的 `mid` 使得 `mid*mid <= x`。

```python
def mySqrt(x):
    lo, hi = 0, x
    while lo <= hi:
        mid = (lo + hi) // 2
        if mid * mid <= x: lo = mid + 1
        else: hi = mid - 1
    return hi
```

```typescript
function mySqrt(x: number): number {
  let lo = 0,
    hi = x
  while (lo <= hi) {
    const mid = Math.floor((lo + hi) / 2)
    if (mid * mid <= x) lo = mid + 1
    else hi = mid - 1
  }
  return hi
}
```

## 70. 爬楼梯 (Climbing Stairs)

**思路**：斐波那契数列，`f(n) = f(n-1) + f(n-2)`。

```python
def climbStairs(n):
    a, b = 1, 1
    for _ in range(n - 1):
        a, b = b, a + b
    return b
```

```typescript
function climbStairs(n: number): number {

    b = 1
  for (let i = 1; i < n; i++) [a, b] = [b, a + b]

}
```

## 71. 简化路径 (Simplify Path)

**描述**：简化 Unix 风格的绝对路径。

**思路**：用栈，按 `/` 分割，遇 `..` 弹栈，遇 `.` 或空跳过。

```python
def simplifyPath(path):
    stack = []
    for p in path.split('/'):
        if p == '..': stack and stack.pop()
        elif p and p != '.': stack.append(p)
    return '/' + '/'.join(stack)
```

```typescript
function simplifyPath(path: string): string {
  const stack: string[] = []
  for (const p of path.split('/')) {
    if (p === '..') stack.pop()
    else if (p && p !== '.') stack.push(p)
  }
  return '/' + stack.join('/')
}
```

## 72. 编辑距离 (Edit Distance)

**描述**：两个单词之间的最少编辑操作数（插入、删除、替换）。

**思路**：经典DP，`dp[i][j]` = word1[:i] 变为 word2[:j] 的最小操作数。

```python
def minDistance(word1, word2):
    m, n = len(word1), len(word2)
    dp = list(range(n + 1))
    for i in range(1, m + 1):
        prev = dp[0]; dp[0] = i
        for j in range(1, n + 1):
            tmp = dp[j]
            if word1[i-1] == word2[j-1]: dp[j] = prev
            else: dp[j] = 1 + min(prev, dp[j], dp[j-1])
            prev = tmp
    return dp[n]
```

```typescript
function minDistance(word1: string, word2: string): number {
  const m = word1.length,
    n = word2.length
  const dp = Array.from({ length: n + 1 }, (_, i) => i)
  for (let i = 1; i <= m; i++) {
    let prev = dp[0]
    dp[0] = i
    for (let j = 1; j <= n; j++) {
      const tmp = dp[j]
      dp[j] = word1[i - 1] === word2[j - 1] ? prev : 1 + Math.min(prev, dp[j], dp[j - 1])
      prev = tmp
    }
  }
  return dp[n]
}
```

## 73. 矩阵置零 (Set Matrix Zeroes)

**描述**：原地将矩阵中含0的行和列全部置0。

**思路**：用第一行和第一列做标记，两遍扫描。

```python
def setZeroes(matrix):
    m, n = len(matrix), len(matrix[0])
    first_row = any(matrix[0][j] == 0 for j in range(n))
    first_col = any(matrix[i][0] == 0 for i in range(m))
    for i in range(1, m):
        for j in range(1, n):
            if matrix[i][j] == 0: matrix[i][0] = matrix[0][j] = 0
    for i in range(1, m):
        for j in range(1, n):
            if matrix[i][0] == 0 or matrix[0][j] == 0: matrix[i][j] = 0
    if first_row:
        for j in range(n): matrix[0][j] = 0
    if first_col:
        for i in range(m): matrix[i][0] = 0
```

```typescript
function setZeroes(matrix: number[][]): void {
  const m = matrix.length,
    n = matrix[0].length
  let firstRow = false,
    firstCol = false
  for (let j = 0; j < n; j++) if (matrix[0][j] === 0) firstRow = true
  for (let i = 0; i < m; i++) if (matrix[i][0] === 0) firstCol = true
  for (let i = 1; i < m; i++)
    for (let j = 1; j < n; j++)
      if (matrix[i][j] === 0) {
        matrix[i][0] = 0
        matrix[0][j] = 0
      }
  for (let i = 1; i < m; i++)
    for (let j = 1; j < n; j++) if (matrix[i][0] === 0 || matrix[0][j] === 0) matrix[i][j] = 0
  if (firstRow) for (let j = 0; j < n; j++) matrix[0][j] = 0
  if (firstCol) for (let i = 0; i < m; i++) matrix[i][0] = 0
}
```

## 74. 搜索二维矩阵 (Search a 2D Matrix)

**描述**：在行列递增的矩阵中搜索目标值（每行第一个大于前行最后一个）。

**思路**：把矩阵视为一维有序数组，标准二分。

```python
def searchMatrix(matrix, target):
    m, n = len(matrix), len(matrix[0])
    lo, hi = 0, m * n - 1
    while lo <= hi:
        mid = (lo + hi) // 2
        val = matrix[mid // n][mid % n]
        if val == target: return True
        elif val < target: lo = mid + 1
        else: hi = mid - 1
    return False
```

```typescript
function searchMatrix(matrix: number[][], target: number): boolean {
  const m = matrix.length,
    n = matrix[0].length
  let lo = 0,
    hi = m * n - 1
  while (lo <= hi) {
    const mid = (lo + hi) >> 1
    const val = matrix[Math.floor(mid / n)][mid % n]
    if (val === target) return true
    else if (val < target) lo = mid + 1
    else hi = mid - 1
  }
  return false
}
```

## 75. 颜色分类 (Sort Colors)

**描述**：原地排序只有0、1、2的数组（荷兰国旗问题）。

**思路**：三指针，low/mid/high，遇0换到low，遇2换到high，遇1跳过。

```python
def sortColors(nums):
    lo, mid, hi = 0, 0, len(nums) - 1
    while mid <= hi:
        if nums[mid] == 0:
            nums[lo], nums[mid] = nums[mid], nums[lo]
            lo += 1; mid += 1
        elif nums[mid] == 2:
            nums[mid], nums[hi] = nums[hi], nums[mid]
            hi -= 1
        else: mid += 1
```

```typescript
function sortColors(nums: number[]): void {
  let lo = 0,
    mid = 0,
    hi = nums.length - 1
  while (mid <= hi) {
    if (nums[mid] === 0) {
      ;[nums[lo], nums[mid]] = [nums[mid], nums[lo]]
      lo++
      mid++
    } else if (nums[mid] === 2) {
      ;[nums[mid], nums[hi]] = [nums[hi], nums[mid]]
      hi--
    } else mid++
  }
}
```

## 76. 最小覆盖子串 (Minimum Window Substring)

**描述**：在 s 中找包含 t 所有字符的最短子串。

**思路**：滑动窗口，扩右缩左，维护字符频次和满足条件的字符数。

```python
from collections import Counter
def minWindow(s, t):
    need = Counter(t)
    missing = len(t)
    l = start = 0
    res = float('inf')
    for r, c in enumerate(s):
        if need[c] > 0: missing -= 1
        need[c] -= 1
        while missing == 0:
            if r - l + 1 < res:
                res = r - l + 1; start = l
            need[s[l]] += 1
            if need[s[l]] > 0: missing += 1
            l += 1
    return '' if res == float('inf') else s[start:start+res]
```

```typescript
function minWindow(s: string, t: string): string {
  const need = new Map<string, number>()
  for (const c of t) need.set(c, (need.get(c) ?? 0) + 1)
  let missing = t.length,
    l = 0,
    start = 0,
    minLen = Infinity
  for (let r = 0; r < s.length; r++) {
    if ((need.get(s[r]) ?? 0) > 0) missing--
    need.set(s[r], (need.get(s[r]) ?? 0) - 1)
    while (missing === 0) {
      if (r - l + 1 < minLen) {
        minLen = r - l + 1
        start = l
      }
      need.set(s[l], (need.get(s[l]) ?? 0) + 1)
      if ((need.get(s[l]) ?? 0) > 0) missing++
      l++
    }
  }
  return minLen === Infinity ? '' : s.slice(start, start + minLen)
}
```

## 77. 组合 (Combinations)

**描述**：从 1~n 中选 k 个数的所有组合。

**思路**：回溯，从 start 开始选，剪枝：剩余数不够则提前终止。

```python
def combine(n, k):
    res = []
    def bt(start, path):
        if len(path) == k: res.append(path[:]); return
        for i in range(start, n - (k - len(path)) + 2):
            path.append(i); bt(i + 1, path); path.pop()
    bt(1, []); return res
```

```typescript
function combine(n: number, k: number): number[][] {
  const res: number[][] = []
  const bt = (start: number, path: number[]) => {
    if (path.length === k) {
      res.push([...path])
      return
    }
    for (let i = start; i <= n - (k - path.length) + 1; i++) {
      path.push(i)
      bt(i + 1, path)
      path.pop()
    }
  }
  bt(1, [])
  return res
}
```

**描述**：返回数组所有子集。

```python
def subsets(nums):
    res = [[]]
```

## 79. 单词搜索 (Word Search)

**描述**：在二维网格中搜索单词，每个格子只能用一次，上下左右连通。

**思路**：回溯DFS，标记访问过的格子，四个方向递归搜索。

```python
def exist(board, word):
    m, n = len(board), len(board[0])
    def dfs(i, j, k):
        if k == len(word): return True
        if i < 0 or i >= m or j < 0 or j >= n or board[i][j] != word[k]: return False
        tmp, board[i][j] = board[i][j], '#'
        found = any(dfs(i+di, j+dj, k+1) for di, dj in [(0,1),(0,-1),(1,0),(-1,0)])
        board[i][j] = tmp
        return found
    return any(dfs(i, j, 0) for i in range(m) for j in range(n))
```

```typescript
function exist(board: string[][], word: string): boolean {
  const m = board.length,
    n = board[0].length
  const dfs = (i: number, j: number, k: number): boolean => {
    if (k === word.length) return true
    if (i < 0 || i >= m || j < 0 || j >= n || board[i][j] !== word[k]) return false
    const tmp = board[i][j]
    board[i][j] = '#'
    const found = [
      [0, 1],
      [0, -1],
      [1, 0],
      [-1, 0]
    ].some(([di, dj]) => dfs(i + di, j + dj, k + 1))
    board[i][j] = tmp
    return found
  }
  for (let i = 0; i < m; i++) for (let j = 0; j < n; j++) if (dfs(i, j, 0)) return true
  return false
}
```

## 80. 删除有序数组中的重复项 II (Remove Duplicates from Sorted Array II)

**描述**：有序数组原地去重，每个元素最多保留两次。

**思路**：双指针，`nums[k-2] != nums[i]` 时才写入。

```python
def removeDuplicates(nums):
    k = 0
    for n in nums:
        if k < 2 or n != nums[k-2]:
            nums[k] = n; k += 1
    return k
```

```typescript
function removeDuplicates(nums: number[]): number {
  let k = 0
  for (const n of nums) if (k < 2 || n !== nums[k - 2]) nums[k++] = n
  return k
}
```

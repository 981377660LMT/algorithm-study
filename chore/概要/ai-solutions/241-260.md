## 241. 为运算表达式设计优先级 (Different Ways to Add Parentheses)

**描述**：给含 +、-、\* 的表达式，通过加括号改变运算顺序，返回所有可能的结果。

**思路**：分治，以每个运算符为分割点，递归左右两半，组合结果。

```python
def diffWaysToCompute(expression):
    res = []
    for i, c in enumerate(expression):
        if c in '+-*':
            left = diffWaysToCompute(expression[:i])
            right = diffWaysToCompute(expression[i+1:])
            for l in left:
                for r in right:
                    if c == '+': res.append(l + r)
                    elif c == '-': res.append(l - r)
                    else: res.append(l * r)
    if not res: res.append(int(expression))
    return res
```

```typescript
function diffWaysToCompute(expression: string): number[] {
  const res: number[] = []
  for (let i = 0; i < expression.length; i++) {
    const c = expression[i]
    if (c === '+' || c === '-' || c === '*') {
      const left = diffWaysToCompute(expression.slice(0, i))
      const right = diffWaysToCompute(expression.slice(i + 1))
      for (const l of left)
        for (const r of right) res.push(c === '+' ? l + r : c === '-' ? l - r : l * r)
    }
  }
  if (!res.length) res.push(Number(expression))
  return res
}
```

## 242. 有效的字母异位词 (Valid Anagram)

**描述**：判断两个字符串是否是字母异位词。

**思路**：计数比较，或排序比较。

```python
def isAnagram(s, t):
    return sorted(s) == sorted(t)
```

```typescript
function isAnagram(s: string, t: string): boolean {
  if (s.length !== t.length) return false
  const cnt = new Array(26).fill(0)
  for (let i = 0; i < s.length; i++) {
    cnt[s.charCodeAt(i) - 97]++
    cnt[t.charCodeAt(i) - 97]--
  }
  return cnt.every(c => c === 0)
}
```

## 257. 二叉树的所有路径 (Binary Tree Paths)

**描述**：返回所有从根到叶子的路径。

**思路**：DFS，到叶子时记录路径。

```python
def binaryTreePaths(root):
    res = []
    def dfs(node, path):
        if not node: return
        path += str(node.val)
        if not node.left and not node.right:
            res.append(path)
        else:
            dfs(node.left, path + "->")
            dfs(node.right, path + "->")
    dfs(root, "")
    return res
```

```typescript
function binaryTreePaths(root: TreeNode | null): string[] {
  const res: string[] = []
  function dfs(node: TreeNode | null, path: string) {
    if (!node) return
    path += node.val
    if (!node.left && !node.right) res.push(path)
    else {
      dfs(node.left, path + '->')
      dfs(node.right, path + '->')
    }
  }
  dfs(root, '')
  return res
}
```

## 258. 各位相加 (Add Digits)

**描述**：反复将各位数字相加直到剩一位。

**思路**：数学公式：`1 + (n-1) % 9`（数根）。

```python
def addDigits(num):
    return 0 if num == 0 else 1 + (num - 1) % 9
```

```typescript
function addDigits(num: number): number {
  return num === 0 ? 0 : 1 + ((num - 1) % 9)
}
```

## 260. 只出现一次的数字 III (Single Number III)

**描述**：数组中恰好两个元素只出现一次，其余都出现两次，找出这两个元素。

**思路**：全部异或得 a^b，取最低位1分组，组内异或分别得 a 和 b。

```python
def singleNumber(nums):
    xor = 0
    for n in nums: xor ^= n
    bit = xor & (-xor)  # 最低位1
    a = b = 0
    for n in nums:
        if n & bit: a ^= n
        else: b ^= n
    return [a, b]
```

```typescript
function singleNumber(nums: number[]): number[] {
  let xor = 0
  for (const n of nums) xor ^= n
  const bit = xor & -xor
  let a = 0,
    b = 0
  for (const n of nums) {
    if (n & bit) a ^= n
    else b ^= n
  }
  return [a, b]
}
```

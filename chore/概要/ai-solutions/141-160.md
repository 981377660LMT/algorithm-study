## 141. 环形链表 (Linked List Cycle)

**描述**：判断链表是否有环。

**思路**：快慢指针，快指针每次走2步，慢指针走1步，相遇则有环。

```python
def hasCycle(head):
    slow = fast = head
    while fast and fast.next:
        slow = slow.next; fast = fast.next.next
        if slow == fast: return True
    return False
```

```typescript
function hasCycle(head: ListNode | null): boolean {
  let slow = head,
    fast = head
  while (fast && fast.next) {
    slow = slow!.next
    fast = fast.next.next
    if (slow === fast) return true
  }
  return false
}
```

## 142. 环形链表 II (Linked List Cycle II)

**描述**：返回链表环的入口节点。

**思路**：快慢指针相遇后，一个从头走，一个从相遇点走，再次相遇即入口。

```python
def detectCycle(head):
    slow = fast = head
    while fast and fast.next:
        slow = slow.next; fast = fast.next.next
        if slow == fast:
            slow = head
            while slow != fast:
                slow = slow.next; fast = fast.next
            return slow
    return None
```

```typescript
function detectCycle(head: ListNode | null): ListNode | null {
  let slow = head,
    fast = head
  while (fast && fast.next) {
    slow = slow!.next
    fast = fast.next.next
    if (slow === fast) {
      slow = head
      while (slow !== fast) {
        slow = slow!.next
        fast = fast!.next
      }
      return slow
    }
  }
  return null
}
```

## 143. 重排链表 (Reorder List)

**描述**：L0→L1→...→Ln 改为 L0→Ln→L1→Ln-1→...

**思路**：找中点→反转后半段→交替合并两段。

```python
def reorderList(head):
    # 找中点
    slow = fast = head
    while fast.next and fast.next.next:
        slow = slow.next; fast = fast.next.next
    # 反转后半
    prev, cur = None, slow.next
    slow.next = None
    while cur:
        cur.next, prev, cur = prev, cur, cur.next
    # 交替合并
    p1, p2 = head, prev
    while p2:
        n1, n2 = p1.next, p2.next
        p1.next = p2; p2.next = n1
        p1, p2 = n1, n2
```

```typescript
function reorderList(head: ListNode | null): void {
  if (!head || !head.next) return
  let slow = head,
    fast: ListNode | null = head
  while (fast!.next && fast!.next.next) {
    slow = slow.next!
    fast = fast!.next.next
  }
  let prev: ListNode | null = null,
    cur: ListNode | null = slow.next
  slow.next = null
  while (cur) {
    const nxt = cur.next
    cur.next = prev
    prev = cur
    cur = nxt
  }
  let p1: ListNode | null = head,
    p2: ListNode | null = prev
  while (p2) {
    const n1 = p1!.next,
      n2 = p2.next
    p1!.next = p2
    p2.next = n1
    p1 = n1
    p2 = n2
  }
}
```

## 144. 二叉树的前序遍历 (Binary Tree Preorder Traversal)

**描述**：迭代实现前序遍历。

**思路**：栈，先压右后压左。

```python
def preorderTraversal(root):
    if not root: return []
    res, stack = [], [root]
    while stack:
        node = stack.pop()
        res.append(node.val)
        if node.right: stack.append(node.right)
        if node.left: stack.append(node.left)
    return res
```

```typescript
function preorderTraversal(root: TreeNode | null): number[] {
  if (!root) return []
  const res: number[] = [],
    stack = [root]
  while (stack.length) {
    const node = stack.pop()!
    res.push(node.val)
    if (node.right) stack.push(node.right)
    if (node.left) stack.push(node.left)
  }
  return res
}
```

## 145. 二叉树的后序遍历 (Binary Tree Postorder Traversal)

**描述**：迭代实现后序遍历。

**思路**：先做"根→右→左"遍历（前序的镜像），然后反转结果。

```python
def postorderTraversal(root):
    if not root: return []
    res, stack = [], [root]
    while stack:
        node = stack.pop()
        res.append(node.val)
        if node.left: stack.append(node.left)
        if node.right: stack.append(node.right)
    return res[::-1]
```

```typescript
function postorderTraversal(root: TreeNode | null): number[] {
  if (!root) return []
  const res: number[] = [],
    stack = [root]
  while (stack.length) {
    const node = stack.pop()!
    res.push(node.val)
    if (node.left) stack.push(node.left)
    if (node.right) stack.push(node.right)
  }
  return res.reverse()
}
```

## 146. LRU 缓存 (LRU Cache)

**描述**：设计 LRU 缓存，O(1) 的 get 和 put。

**思路**：哈希表 + 双向链表。get/put 时将节点移到链表头部，满时删除尾部。

```python
from collections import OrderedDict
class LRUCache:
    def __init__(self, capacity):
        self.cache = OrderedDict()
        self.cap = capacity
    def get(self, key):
        if key not in self.cache: return -1
        self.cache.move_to_end(key)
        return self.cache[key]
    def put(self, key, value):
        if key in self.cache: self.cache.move_to_end(key)
        self.cache[key] = value
        if len(self.cache) > self.cap:
            self.cache.popitem(last=False)
```

```typescript
class LRUCache {
  private cap: number
  private cache = new Map<number, number>()
  constructor(capacity: number) {
    this.cap = capacity
  }
  get(key: number): number {
    if (!this.cache.has(key)) return -1
    const val = this.cache.get(key)!
    this.cache.delete(key)
    this.cache.set(key, val)
    return val
  }
  put(key: number, value: number): void {
    if (this.cache.has(key)) this.cache.delete(key)
    this.cache.set(key, value)
    if (this.cache.size > this.cap) this.cache.delete(this.cache.keys().next().value!)
  }
}
```

## 147. 对链表进行插入排序 (Insertion Sort List)

**描述**：对链表执行插入排序。

**思路**：维护已排序部分，每次从未排序取一个节点插入正确位置。

```python
def insertionSortList(head):
    dummy = ListNode(0)
    cur = head
    while cur:
        nxt = cur.next
        prev = dummy
        while prev.next and prev.next.val < cur.val:
            prev = prev.next
        cur.next = prev.next; prev.next = cur
        cur = nxt
    return dummy.next
```

```typescript
function insertionSortList(head: ListNode | null): ListNode | null {
  const dummy = new ListNode(0)
  let cur = head
  while (cur) {
    const nxt = cur.next
    let prev = dummy
    while (prev.next && prev.next.val < cur.val) prev = prev.next
    cur.next = prev.next
    prev.next = cur
    cur = nxt
  }
  return dummy.next
}
```

## 148. 排序链表 (Sort List)

**描述**：O(n log n) 排序链表。

**思路**：归并排序，快慢指针找中点，递归切分再合并。

```python
def sortList(head):
    if not head or not head.next: return head
    slow, fast = head, head.next
    while fast and fast.next:
        slow = slow.next; fast = fast.next.next
    mid = slow.next; slow.next = None
    l, r = sortList(head), sortList(mid)
    dummy = cur = ListNode(0)
    while l and r:
        if l.val <= r.val: cur.next = l; l = l.next
        else: cur.next = r; r = r.next
        cur = cur.next
    cur.next = l or r
    return dummy.next
```

```typescript
function sortList(head: ListNode | null): ListNode | null {
  if (!head || !head.next) return head
  let slow = head,
    fast: ListNode | null = head.next
  while (fast && fast.next) {
    slow = slow.next!
    fast = fast.next.next
  }
  const mid = slow.next
  slow.next = null
  let l = sortList(head),
    r = sortList(mid)
  const dummy = new ListNode(0)
  let cur = dummy
  while (l && r) {
    if (l.val <= r.val) {
      cur.next = l
      l = l.next
    } else {
      cur.next = r
      r = r.next
    }
    cur = cur.next!
  }
  cur.next = l ?? r
  return dummy.next
}
```

## 149. 直线上最多的点数 (Max Points on a Line)

**描述**：平面上最多有多少点在同一直线上。

**思路**：枚举每个点，哈希表记录斜率（用最简分数避免浮点），取最大值。

```python
from math import gcd
from collections import defaultdict
def maxPoints(points):
    n = len(points); res = 1
    for i in range(n):
        cnt = defaultdict(int)
        for j in range(i+1, n):
            dx, dy = points[j][0]-points[i][0], points[j][1]-points[i][1]
            g = gcd(dx, dy)
            cnt[(dx//g, dy//g)] += 1
        res = max(res, max(cnt.values(), default=0) + 1)
    return res
```

```typescript
function maxPoints(points: number[][]): number {
  const n = points.length
  let res = 1
  const gcd = (a: number, b: number): number => (b ? gcd(b, a % b) : a)
  for (let i = 0; i < n; i++) {
    const cnt = new Map<string, number>()
    for (let j = i + 1; j < n; j++) {
      let dx = points[j][0] - points[i][0],
        dy = points[j][1] - points[i][1]
      const g = gcd(Math.abs(dx), Math.abs(dy))
      if (g) {
        dx /= g
        dy /= g
      }
      if (dx < 0) {
        dx = -dx
        dy = -dy
      }
      const key = `${dx},${dy}`
      cnt.set(key, (cnt.get(key) ?? 0) + 1)
    }
    for (const v of cnt.values()) res = Math.max(res, v + 1)
  }
  return res
}
```

## 150. 逆波兰表达式求值 (Evaluate Reverse Polish Notation)

**描述**：计算逆波兰表达式的值。

**思路**：栈，遇到数压栈，遇到运算符弹两个数计算后压回。

```python
def evalRPN(tokens):
    stack = []
    for t in tokens:
        if t in '+-*/':
            b, a = stack.pop(), stack.pop()
            if t == '+': stack.append(a + b)
            elif t == '-': stack.append(a - b)
            elif t == '*': stack.append(a * b)
            else: stack.append(int(a / b))  # 向零截断
        else:
            stack.append(int(t))
    return stack[0]
```

```typescript
function evalRPN(tokens: string[]): number {
  const stack: number[] = []
  for (const t of tokens) {
    if ('+-*/'.includes(t)) {
      const b = stack.pop()!,
        a = stack.pop()!
      if (t === '+') stack.push(a + b)
      else if (t === '-') stack.push(a - b)
      else if (t === '*') stack.push(a * b)
      else stack.push(Math.trunc(a / b))
    } else stack.push(parseInt(t))
  }
  return stack[0]
}
```

## 152. 乘积最大子数组 (Maximum Product Subarray)

**描述**：找连续子数组的最大乘积。

**思路**：维护当前最大和最小（负数翻转），每步更新。

```python
def maxProduct(nums):
    res = curMax = curMin = nums[0]
    for n in nums[1:]:
        if n < 0: curMax, curMin = curMin, curMax
        curMax = max(n, curMax * n)
        curMin = min(n, curMin * n)
        res = max(res, curMax)
    return res
```

```typescript
function maxProduct(nums: number[]): number {
  let res = nums[0],
    curMax = nums[0],
    curMin = nums[0]
  for (let i = 1; i < nums.length; i++) {
    if (nums[i] < 0) [curMax, curMin] = [curMin, curMax]
    curMax = Math.max(nums[i], curMax * nums[i])
    curMin = Math.min(nums[i], curMin * nums[i])
    res = Math.max(res, curMax)
  }
  return res
}
```

## 153. 寻找旋转排序数组中的最小值 (Find Minimum in Rotated Sorted Array)

**描述**：旋转数组（无重复）中找最小值。

**思路**：二分，`nums[mid] > nums[hi]` 说明最小值在右边。

```python
def findMin(nums):
    lo, hi = 0, len(nums) - 1
    while lo < hi:
        mid = (lo + hi) // 2
        if nums[mid] > nums[hi]: lo = mid + 1
        else: hi = mid
    return nums[lo]
```

```typescript
function findMin(nums: number[]): number {
  let lo = 0,
    hi = nums.length - 1
  while (lo < hi) {
    const mid = (lo + hi) >> 1
    if (nums[mid] > nums[hi]) lo = mid + 1
    else hi = mid
  }
  return nums[lo]
}
```

## 154. 寻找旋转排序数组中的最小值 II (Find Minimum in Rotated Sorted Array II)

**描述**：同153，但有重复元素。

**思路**：同153，但 `nums[mid] == nums[hi]` 时 `hi--` 跳过。

```python
def findMin(nums):
    lo, hi = 0, len(nums) - 1
    while lo < hi:
        mid = (lo + hi) // 2
        if nums[mid] > nums[hi]: lo = mid + 1
        elif nums[mid] < nums[hi]: hi = mid
        else: hi -= 1
    return nums[lo]
```

```typescript
function findMin(nums: number[]): number {
  let lo = 0,
    hi = nums.length - 1
  while (lo < hi) {
    const mid = (lo + hi) >> 1
    if (nums[mid] > nums[hi]) lo = mid + 1
    else if (nums[mid] < nums[hi]) hi = mid
    else hi--
  }
  return nums[lo]
}
```

## 155. 最小栈 (Min Stack)

**描述**：设计支持 O(1) 获取最小值的栈。

**思路**：辅助栈同步记录当前最小值。

```python
class MinStack:
    def __init__(self):
        self.stack = []
        self.min_stack = [float('inf')]
    def push(self, val):
        self.stack.append(val)
        self.min_stack.append(min(val, self.min_stack[-1]))
    def pop(self):
        self.stack.pop(); self.min_stack.pop()
    def top(self): return self.stack[-1]
    def getMin(self): return self.min_stack[-1]
```

```typescript
class MinStack {
  private stack: number[] = []
  private minStack: number[] = [Infinity]
  push(val: number): void {
    this.stack.push(val)
    this.minStack.push(Math.min(val, this.minStack[this.minStack.length - 1]))
  }
  pop(): void {
    this.stack.pop()
    this.minStack.pop()
  }
  top(): number {
    return this.stack[this.stack.length - 1]
  }
  getMin(): number {
    return this.minStack[this.minStack.length - 1]
  }
}
```

## 160. 相交链表 (Intersection of Two Linked Lists)

**描述**：找两个链表的交点。

**思路**：双指针，A走完走B，B走完走A，路径等长时相遇即交点。

```python
def getIntersectionNode(headA, headB):
    a, b = headA, headB
    while a != b:
        a = a.next if a else headB
        b = b.next if b else headA
    return a
```

```typescript
function getIntersectionNode(headA: ListNode | null, headB: ListNode | null): ListNode | null {
  let a = headA,
    b = headB
  while (a !== b) {
    a = a ? a.next : headB
    b = b ? b.next : headA
  }
  return a
}
```

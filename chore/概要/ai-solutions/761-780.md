## 763. 划分字母区间 (Partition Labels)

**描述**：划分字符串使每个字母只在一个片段中出现，返回每段长度。

**思路**：贪心，记录每个字母最后出现位置。

```python
def partitionLabels(s):
    last = {c: i for i, c in enumerate(s)}
    res = []; start = end = 0
    for i, c in enumerate(s):
        end = max(end, last[c])
        if i == end: res.append(end-start+1); start = i+1
    return res
```

```typescript
function partitionLabels(s: string): number[] {
  const last = new Map<string, number>()
  for (let i = 0; i < s.length; i++) last.set(s[i], i)
  const res: number[] = []
  let start = 0,
    end = 0
  for (let i = 0; i < s.length; i++) {
    end = Math.max(end, last.get(s[i])!)
    if (i === end) {
      res.push(end - start + 1)
      start = i + 1
    }
  }
  return res
}
```

## 766. 托普利茨矩阵 (Toeplitz Matrix)

**描述**：每条对角线上元素相同。

**思路**：检查每个元素和左上是否相等。

```python
def isToeplitzMatrix(matrix):
    for i in range(1, len(matrix)):
        for j in range(1, len(matrix[0])):
            if matrix[i][j] != matrix[i-1][j-1]: return False
    return True
```

```typescript
function isToeplitzMatrix(matrix: number[][]): boolean {
  for (let i = 1; i < matrix.length; i++)
    for (let j = 1; j < matrix[0].length; j++)
      if (matrix[i][j] !== matrix[i - 1][j - 1]) return false
  return true
}
```

## 767. 重构字符串 (Reorganize String)

**描述**：重排字符串使相邻字符不同。

**思路**：贪心，先放频率最高的，间隔放。

```python
from collections import Counter
def reorganizeString(s):
    cnt = Counter(s); n = len(s)
    mx_char, mx_cnt = cnt.most_common(1)[0]
    if mx_cnt > (n+1)//2: return ""
    # 排序后间隔放
    order = sorted(cnt.keys(), key=lambda c: -cnt[c])
    res = [''] * n; idx = 0
    for c in order:
        for _ in range(cnt[c]):
            res[idx] = c; idx += 2
            if idx >= n: idx = 1
    return ''.join(res)
```

```typescript
function reorganizeString(s: string): string {
  const cnt = new Map<string, number>()
  for (const c of s) cnt.set(c, (cnt.get(c) ?? 0) + 1)
  const n = s.length
  let mxCnt = 0
  for (const v of cnt.values()) mxCnt = Math.max(mxCnt, v)
  if (mxCnt > (n + 1) >> 1) return ''
  const order = [...cnt.entries()].sort((a, b) => b[1] - a[1])
  const res = new Array(n)
  let idx = 0
  for (const [c, count] of order)
    for (let i = 0; i < count; i++) {
      res[idx] = c
      idx += 2
      if (idx >= n) idx = 1
    }
  return res.join('')
}
```

## 771. 宝石与石头 (Jewels and Stones)

**描述**：石头中有几个是宝石。

**思路**：集合。

```python
def numJewelsInStones(jewels, stones):
    j = set(jewels)
    return sum(1 for s in stones if s in j)
```

```typescript
function numJewelsInStones(jewels: string, stones: string): number {
  const j = new Set(jewels)
  return [...stones].filter(s => j.has(s)).length
}
```

## 778. 水位上升的泳池中游泳 (Swim in Rising Water)

**描述**：网格中等水位到 t 才能走高度 ≤ t 的格子，求从左上到右下的最小 t。

**思路**：二分答案 + BFS，或 Dijkstra/并查集。

```python
import heapq
def swimInWater(grid):
    n = len(grid)
    heap = [(grid[0][0], 0, 0)]; visited = [[False]*n for _ in range(n)]
    visited[0][0] = True; dirs = [(0,1),(0,-1),(1,0),(-1,0)]
    while heap:
        t, x, y = heapq.heappop(heap)
        if x == n-1 and y == n-1: return t
        for dx, dy in dirs:
            nx, ny = x+dx, y+dy
            if 0<=nx<n and 0<=ny<n and not visited[nx][ny]:
                visited[nx][ny] = True
                heapq.heappush(heap, (max(t, grid[nx][ny]), nx, ny))
    return -1
```

```typescript
function swimInWater(grid: number[][]): number {
  const n = grid.length,
    visited = Array.from({ length: n }, () => new Array(n).fill(false))
  // 简化堆：用排序数组
  const heap: [number, number, number][] = [[grid[0][0], 0, 0]]
  visited[0][0] = true
  const dirs = [
    [0, 1],
    [0, -1],
    [1, 0],
    [-1, 0]
  ]
  while (heap.length) {
    heap.sort((a, b) => a[0] - b[0])
    const [t, x, y] = heap.shift()!
    if (x === n - 1 && y === n - 1) return t
    for (const [dx, dy] of dirs) {
      const nx = x + dx,
        ny = y + dy
      if (nx >= 0 && nx < n && ny >= 0 && ny < n && !visited[nx][ny]) {
        visited[nx][ny] = true
        heap.push([Math.max(t, grid[nx][ny]), nx, ny])
      }
    }
  }
  return -1
}
```

## 779. 第K个语法符号 (K-th Symbol in Grammar)

**描述**：第一行 0，下一行 0→01, 1→10，求第 n 行第 k 个。

**思路**：递归，位数减半。

```python
def kthGrammar(n, k):
    if n == 1: return 0
    parent = kthGrammar(n-1, (k+1)//2)
    return parent if k % 2 else 1 - parent
```

```typescript
function kthGrammar(n: number, k: number): number {
  if (n === 1) return 0
  const parent = kthGrammar(n - 1, Math.ceil(k / 2))
  return k % 2 ? parent : 1 - parent
}
```

## 780. 到达终点 (Reaching Points) — 较难/数学题。

**描述**：从 (sx,sy) 只能变为 (x+y,y) 或 (x,x+y)，能否到 (tx,ty)。

**思路**：从终点反推，大的减小的（取模加速）。

```python
def reachingPoints(sx, sy, tx, ty):
    while tx > sx and ty > sy:
        if tx > ty: tx %= ty
        else: ty %= tx
    if tx == sx and ty == sy: return True
    if tx == sx: return ty > sy and (ty - sy) % sx == 0
    if ty == sy: return tx > sx and (tx - sx) % sy == 0
    return False
```

```typescript
function reachingPoints(sx: number, sy: number, tx: number, ty: number): boolean {
  while (tx > sx && ty > sy) {
    if (tx > ty) tx %= ty
    else ty %= tx
  }
  if (tx === sx && ty === sy) return true
  if (tx === sx) return ty > sy && (ty - sy) % sx === 0
  if (ty === sy) return tx > sx && (tx - sx) % sy === 0
  return false
}
```

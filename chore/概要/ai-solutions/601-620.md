## 605. 种花问题 (Can Place Flowers)

**描述**：花坛 0/1 数组，相邻不能种花，能否再种 n 朵。

**思路**：贪心，遇到连续三个 0 就种。

```python
def canPlaceFlowers(flowerbed, n):
    for i in range(len(flowerbed)):
        if flowerbed[i] == 0 and (i == 0 or flowerbed[i-1] == 0) and (i == len(flowerbed)-1 or flowerbed[i+1] == 0):
            flowerbed[i] = 1; n -= 1
    return n <= 0
```

```typescript
function canPlaceFlowers(flowerbed: number[], n: number): boolean {
  for (let i = 0; i < flowerbed.length; i++)
    if (!flowerbed[i] && !flowerbed[i - 1] && !flowerbed[i + 1]) {
      flowerbed[i] = 1
      n--
    }
  return n <= 0
}
```

## 606. 根据二叉树创建字符串 (Construct String from Binary Tree)

**描述**：二叉树转括号字符串。

**思路**：前序遍历，左空右非空时需要空括号。

```python
def tree2str(root):
    if not root: return ""
    s = str(root.val)
    if root.left or root.right: s += f"({tree2str(root.left)})"
    if root.right: s += f"({tree2str(root.right)})"
    return s
```

```typescript
function tree2str(root: TreeNode | null): string {
  if (!root) return ''
  let s = String(root.val)
  if (root.left || root.right) s += `(${tree2str(root.left)})`
  if (root.right) s += `(${tree2str(root.right)})`
  return s
}
```

## 611. 有效三角形的个数 (Valid Triangle Number)

**描述**：数组中能组成三角形的三元组个数。

**思路**：排序 + 双指针。固定最大边，双指针找满足条件的两条小边。

```python
def triangleNumber(nums):
    nums.sort(); res = 0
    for k in range(2, len(nums)):
        i, j = 0, k-1
        while i < j:
            if nums[i]+nums[j] > nums[k]: res += j-i; j -= 1
            else: i += 1
    return res
```

```typescript
function triangleNumber(nums: number[]): number {
  nums.sort((a, b) => a - b)
  let res = 0
  for (let k = 2; k < nums.length; k++) {
    let i = 0,
      j = k - 1
    while (i < j) {
      if (nums[i] + nums[j] > nums[k]) {
        res += j - i
        j--
      } else i++
    }
  }
  return res
}
```

## 617. 合并二叉树 (Merge Two Binary Trees)

**描述**：两棵树对应节点值相加。

**思路**：递归。

```python
def mergeTrees(root1, root2):
    if not root1: return root2
    if not root2: return root1
    root1.val += root2.val
    root1.left = mergeTrees(root1.left, root2.left)
    root1.right = mergeTrees(root1.right, root2.right)
    return root1
```

```typescript
function mergeTrees(root1: TreeNode | null, root2: TreeNode | null): TreeNode | null {
  if (!root1) return root2
  if (!root2) return root1
  root1.val += root2.val
  root1.left = mergeTrees(root1.left, root2.left)
  root1.right = mergeTrees(root1.right, root2.right)
  return root1
}
```

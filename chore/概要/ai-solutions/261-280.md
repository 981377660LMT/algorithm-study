## 263. 丑数 (Ugly Number)

**描述**：判断 n 是否是丑数（只含因子 2, 3, 5）。

**思路**：不断除以 2、3、5 看是否最终等于 1。

```python
def isUgly(n):
    if n <= 0: return False
    for p in [2, 3, 5]:
        while n % p == 0: n //= p
    return n == 1
```

```typescript
function isUgly(n: number): boolean {
  if (n <= 0) return false
  for (const p of [2, 3, 5]) while (n % p === 0) n /= p
  return n === 1
}
```

## 264. 丑数 II (Ugly Number II)

**描述**：找第 n 个丑数。

**思路**：三指针 DP，分别追踪 ×2、×3、×5 的位置，取最小。

```python
def nthUglyNumber(n):
    dp = [1] * n
    i2 = i3 = i5 = 0
    for i in range(1, n):
        dp[i] = min(dp[i2]*2, dp[i3]*3, dp[i5]*5)
        if dp[i] == dp[i2]*2: i2 += 1
        if dp[i] == dp[i3]*3: i3 += 1
        if dp[i] == dp[i5]*5: i5 += 1
    return dp[-1]
```

```typescript
function nthUglyNumber(n: number): number {
  const dp = [1]
  let i2 = 0,
    i3 = 0,
    i5 = 0
  for (let i = 1; i < n; i++) {
    dp[i] = Math.min(dp[i2] * 2, dp[i3] * 3, dp[i5] * 5)
    if (dp[i] === dp[i2] * 2) i2++
    if (dp[i] === dp[i3] * 3) i3++
    if (dp[i] === dp[i5] * 5) i5++
  }
  return dp[n - 1]
}
```

## 268. 丢失的数字 (Missing Number)

**描述**：0~n 中缺了一个数，找出来。

**思路**：异或，`0^1^...^n ^ 数组所有元素`，剩余即缺失。

```python
def missingNumber(nums):
    n = len(nums)
    return n * (n + 1) // 2 - sum(nums)
```

```typescript
function missingNumber(nums: number[]): number {
  const n = nums.length
  return (n * (n + 1)) / 2 - nums.reduce((a, b) => a + b, 0)
}
```

## 273. 整数转换英文表示 (Integer to English Words)

**描述**：将非负整数转为英文单词。

**思路**：分 billion、million、thousand 三段，每段处理百位+十位+个位。

```python
def numberToWords(num):
    if num == 0: return "Zero"
    below_20 = ["","One","Two","Three","Four","Five","Six","Seven","Eight","Nine",
                 "Ten","Eleven","Twelve","Thirteen","Fourteen","Fifteen","Sixteen",
                 "Seventeen","Eighteen","Nineteen"]
    tens = ["","","Twenty","Thirty","Forty","Fifty","Sixty","Seventy","Eighty","Ninety"]
    thousands = ["", "Thousand", "Million", "Billion"]
    def helper(n):
        if n == 0: return ""
        if n < 20: return below_20[n] + " "
        if n < 100: return tens[n//10] + " " + helper(n%10)
        return below_20[n//100] + " Hundred " + helper(n%100)
    res = ""
    for i, unit in enumerate(thousands):
        if num % 1000:
            res = helper(num % 1000) + unit + " " + res
        num //= 1000
    return res.strip()
```

  const below20 = [

    'Two',
    'Three',
    'Four',
    'Five',
    'Six',
    'Seven',

    'Nine',

    'Thirteen',
    'Fourteen',
    'Fifteen',
    'Sixteen',
    'Seventeen',
    'Eighteen',
    'Nineteen'
  ]
  const tens2 = [
    '',
    '',
    'Twenty',
    'Thirty',
    'Forty',
    'Fifty',
    'Sixty',
    'Seventy',
    'Eighty',
    'Ninety'

    return below20[Math.floor(n / 100)] + ' Hundred ' + helper(n % 100)
  }
  let res = ''
  for (let i = 0; num > 0; i++) {
    if (num % 1000) res = helper(num % 1000) + units[i] + ' ' + res
    num = Math.floor(num / 1000)
  }
  return res.trim()
}

## 274. H 指数 (H-Index)

**描述**：给定引用次数数组，计算 H 指数。

**思路**：排序后从大到小扫描，`citations[i] >= n - i` 即成立。或计数排序 O(n)。

```python
def hIndex(citations):
    citations.sort(reverse=True)
    for i, c in enumerate(citations):
        if c <= i: return i
    return len(citations)
```

```typescript
function hIndex(citations: number[]): number {
  citations.sort((a, b) => b - a)
  for (let i = 0; i < citations.length; i++) {
    if (citations[i] <= i) return i
  }
  return citations.length
}
```

## 275. H 指数 II (H-Index II)

**描述**：排序数组版 H 指数，要求 O(log n)。

**思路**：二分查找，找 `citations[mid] >= n - mid` 的最左位置。

```python
def hIndex(citations):
    n = len(citations)
    lo, hi = 0, n
    while lo < hi:
        mid = (lo + hi) // 2
        if citations[mid] >= n - mid: hi = mid
        else: lo = mid + 1
    return n - lo
```

```typescript
function hIndex(citations: number[]): number {
  const n = citations.length
  let lo = 0,
    hi = n
  while (lo < hi) {
    const mid = (lo + hi) >> 1
    if (citations[mid] >= n - mid) hi = mid
    else lo = mid + 1
  }
  return n - lo
}
```

## 278. 第一个错误的版本 (First Bad Version)

**描述**：版本号 1~n，某个版本之后都是坏的，找第一个坏版本。

**思路**：二分查找。

```python
def firstBadVersion(n):
    lo, hi = 1, n
    while lo < hi:
        mid = (lo + hi) // 2
        if isBadVersion(mid): hi = mid
        else: lo = mid + 1
    return lo
```

```typescript
function firstBadVersion(n: number): number {
  let lo = 1,
    hi = n
  while (lo < hi) {
    const mid = lo + ((hi - lo) >> 1)
    if (isBadVersion(mid)) hi = mid
    else lo = mid + 1
  }
  return lo
}
```

## 279. 完全平方数 (Perfect Squares)

**描述**：返回和为 n 的最少完全平方数的个数。

**思路**：BFS（最短路）或 DP `dp[i] = min(dp[i - j*j]) + 1`。

```python
def numSquares(n):
    dp = [0] + [float('inf')] * n
    for i in range(1, n + 1):
        j = 1
        while j * j <= i:
            dp[i] = min(dp[i], dp[i - j*j] + 1)
            j += 1
    return dp[n]
```

```typescript
function numSquares(n: number): number {
  const dp = [0, ...new Array(n).fill(Infinity)]
  for (let i = 1; i <= n; i++)
    for (let j = 1; j * j <= i; j++) dp[i] = Math.min(dp[i], dp[i - j * j] + 1)
  return dp[n]
}
```

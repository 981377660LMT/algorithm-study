## 81. 搜索旋转排序数组 II (Search in Rotated Sorted Array II)

**描述**：同33，但数组含重复元素。

**思路**：二分，当 `nums[lo] == nums[mid]` 时无法判断有序段，`lo++` 跳过。

```python
def search(nums, target):
    lo, hi = 0, len(nums) - 1
    while lo <= hi:
        mid = (lo + hi) // 2
        if nums[mid] == target: return True
        if nums[lo] == nums[mid]:
            lo += 1; continue
        if nums[lo] <= nums[mid]:
            if nums[lo] <= target < nums[mid]: hi = mid - 1
            else: lo = mid + 1
        else:
            if nums[mid] < target <= nums[hi]: lo = mid + 1
            else: hi = mid - 1
    return False
```

```typescript
function search(nums: number[], target: number): boolean {
  let lo = 0, hi = nums.length - 1
  while (lo <= hi) {
    const mid = (lo + hi) >> 1
    if (nums[mid] === target) return true
    if (nums[lo] === nums[mid]) { lo++; continue }
    if (nums[lo] <= nums[mid]) {
      if (nums[lo] <= target && target < nums[mid]) hi = mid - 1
      else lo = mid + 1
    } else {
      if (nums[mid] < target && target <= nums[hi]) lo = mid + 1
      else hi = mid - 1
    }
  }
  return false
}
```

## 82. 删除排序链表中的重复元素 II (Remove Duplicates from Sorted List II)

**描述**：删除排序链表中所有有重复数字的节点，只留不重复的。

**思路**：哑节点，双指针，遇到重复值跳过所有相同节点。

```python
def deleteDuplicates(head):
    dummy = ListNode(0, head); prev = dummy
    while prev.next:
        cur = prev.next
        while cur.next and cur.next.val == cur.val:
            cur = cur.next
        if prev.next == cur: prev = prev.next
        else: prev.next = cur.next
    return dummy.next
```

```typescript
function deleteDuplicates(head: ListNode | null): ListNode | null {
  const dummy = new ListNode(0, head); let prev = dummy
  while (prev.next) {
    let cur = prev.next
    while (cur.next && cur.next.val === cur.val) cur = cur.next
    if (prev.next === cur) prev = prev.next!
    else prev.next = cur.next
  }
  return dummy.next
}
```

## 83. 删除排序链表中的重复元素 (Remove Duplicates from Sorted List)

**描述**：删除排序链表中的重复元素，使每个值只出现一次。

**思路**：遍历，当前节点值等于下一节点值则跳过下一个。

```python
def deleteDuplicates(head):
    cur = head
    while cur and cur.next:
        if cur.val == cur.next.val: cur.next = cur.next.next
        else: cur = cur.next
    return head
```

```typescript
function deleteDuplicates(head: ListNode | null): ListNode | null {
  let cur = head
  while (cur && cur.next) {
    if (cur.val === cur.next.val) cur.next = cur.next.next
    else cur = cur.next
  }
  return head
}
```

## 84. 柱状图中最大的矩形 (Largest Rectangle in Histogram)

**描述**：柱状图中找能围成的最大矩形面积。

**思路**：单调递增栈，对每个柱子找左右第一个更矮的位置。

```python
def largestRectangleArea(heights):
    stack = [-1]; res = 0
    for i, h in enumerate(heights):
        while stack[-1] != -1 and heights[stack[-1]] >= h:
            height = heights[stack.pop()]
            width = i - stack[-1] - 1
            res = max(res, height * width)
        stack.append(i)
    while stack[-1] != -1:
        height = heights[stack.pop()]
        width = len(heights) - stack[-1] - 1
        res = max(res, height * width)
    return res
```

```typescript
function largestRectangleArea(heights: number[]): number {
  const stack = [-1]; let res = 0
  for (let i = 0; i < heights.length; i++) {
    while (stack[stack.length-1] !== -1 && heights[stack[stack.length-1]] >= heights[i]) {
      const h = heights[stack.pop()!]
      res = Math.max(res, h * (i - stack[stack.length-1] - 1))
    }
    stack.push(i)
  }
  while (stack[stack.length-1] !== -1) {
    const h = heights[stack.pop()!]
    res = Math.max(res, h * (heights.length - stack[stack.length-1] - 1))
  }
  return res
}
```

## 85. 最大矩形 (Maximal Rectangle)

**描述**：在只含 0 和 1 的二维矩阵中找全为 1 的最大矩形面积。

**思路**：逐行构建柱状图高度数组，然后用84的方法求最大矩形。

```python
def maximalRectangle(matrix):
    if not matrix: return 0
    n = len(matrix[0])
    heights = [0] * n
    res = 0
    for row in matrix:
        for j in range(n):
            heights[j] = heights[j] + 1 if row[j] == '1' else 0
        res = max(res, largestRectangleArea(heights))
    return res
```

```typescript
function maximalRectangle(matrix: string[][]): number {
  if (!matrix.length) return 0
  const n = matrix[0].length, heights = new Array(n).fill(0)
  let res = 0
  for (const row of matrix) {
    for (let j = 0; j < n; j++) heights[j] = row[j] === '1' ? heights[j] + 1 : 0
    res = Math.max(res, largestRectangleArea(heights))
  }
  return res
}
```

## 86. 分隔链表 (Partition List)

**描述**：将链表按 x 分隔，所有小于 x 的在前，其余在后，保持原相对顺序。

**思路**：两条链表分别收集 < x 和 >= x 的节点，最后拼接。

```python
def partition(head, x):
    small = s_cur = ListNode(0)
    big = b_cur = ListNode(0)
    while head:
        if head.val < x:
            s_cur.next = head; s_cur = s_cur.next
        else:
            b_cur.next = head; b_cur = b_cur.next
        head = head.next
    b_cur.next = None
    s_cur.next = big.next
    return small.next
```

```typescript
function partition(head: ListNode | null, x: number): ListNode | null {
  const small = new ListNode(0), big = new ListNode(0)
  let s = small, b = big
  while (head) {
    if (head.val < x) { s.next = head; s = s.next }
    else { b.next = head; b = b.next }
    head = head.next
  }
  b.next = null; s.next = big.next
  return small.next
}
```

## 87. 扰乱字符串 (Scramble String)

**描述**：判断 s2 是否是 s1 的扰乱字符串。

**思路**：记忆化递归——枚举分割点，判断两部分交换或不交换是否匹配。

```python
from functools import lru_cache
def isScramble(s1, s2):
    @lru_cache(None)
    def dp(a, b):
        if a == b: return True
        if sorted(a) != sorted(b): return False
        for i in range(1, len(a)):
            if dp(a[:i], b[:i]) and dp(a[i:], b[i:]): return True
            if dp(a[:i], b[-i:]) and dp(a[i:], b[:-i]): return True
        return False
    return dp(s1, s2)
```

```typescript
function isScramble(s1: string, s2: string): boolean {
  const memo = new Map<string, boolean>()
  const dp = (a: string, b: string): boolean => {
    const key = a + '#' + b
    if (memo.has(key)) return memo.get(key)!
    if (a === b) return true
    if ([...a].sort().join('') !== [...b].sort().join('')) { memo.set(key, false); return false }
    for (let i = 1; i < a.length; i++) {
      if (dp(a.slice(0,i), b.slice(0,i)) && dp(a.slice(i), b.slice(i))) { memo.set(key, true); return true }
      if (dp(a.slice(0,i), b.slice(b.length-i)) && dp(a.slice(i), b.slice(0,b.length-i))) { memo.set(key, true); return true }
    }
    memo.set(key, false); return false
  }
  return dp(s1, s2)
}
```

## 88. 合并两个有序数组 (Merge Sorted Array)

**描述**：将 nums2 合并到 nums1 中（nums1 后面预留了空间）。

**思路**：从后往前填充，避免覆盖。

```python
def merge(nums1, m, nums2, n):
    while m > 0 and n > 0:
        if nums1[m-1] >= nums2[n-1]:
            nums1[m+n-1] = nums1[m-1]; m -= 1
        else:
            nums1[m+n-1] = nums2[n-1]; n -= 1
    nums1[:n] = nums2[:n]
```

```typescript
function merge(nums1: number[], m: number, nums2: number[], n: number): void {
  let i = m - 1, j = n - 1, k = m + n - 1
  while (i >= 0 && j >= 0) nums1[k--] = nums1[i] >= nums2[j] ? nums1[i--] : nums2[j--]
  while (j >= 0) nums1[k--] = nums2[j--]
}
```

## 89. 格雷编码 (Gray Code)

**描述**：返回 n 位格雷编码序列。

**思路**：镜像法，每一位在前一位结果的逆序前面加 `1<<i`。

```python
def grayCode(n):
    res = [0]
    for i in range(n):
        res += [x | (1 << i) for x in reversed(res)]
    return res
```

```typescript
function grayCode(n: number): number[] {
  const res = [0]
  for (let i = 0; i < n; i++)
    for (let j = res.length - 1; j >= 0; j--) res.push(res[j] | (1 << i))
  return res
}
```

## 90. 子集 II (Subsets II)

**描述**：给定含重复元素的数组，返回所有不重复子集。

**思路**：排序后回溯，同层跳过重复元素。

```python
def subsetsWithDup(nums):
    nums.sort(); res = []
    def bt(start, path):
        res.append(path[:])
        for i in range(start, len(nums)):
            if i > start and nums[i] == nums[i-1]: continue
            path.append(nums[i]); bt(i+1, path); path.pop()
    bt(0, []); return res
```

```typescript
function subsetsWithDup(nums: number[]): number[][] {
  nums.sort((a,b) => a-b)
  const res: number[][] = []
  const bt = (start: number, path: number[]) => {
    res.push([...path])
    for (let i = start; i < nums.length; i++) {
      if (i > start && nums[i] === nums[i-1]) continue
      path.push(nums[i]); bt(i+1, path); path.pop()
    }
  }
  bt(0, []); return res
}
```

## 91. 解码方法 (Decode Ways)

**描述**：数字字符串解码为字母（A=1...Z=26）的方法数。

**思路**：DP，`dp[i]` = 前 i 个字符的解码数。单独解码 + 两位解码（10~26）。

```python
def numDecodings(s):
    if s[0] == '0': return 0
    prev2, prev1 = 1, 1
    for i in range(1, len(s)):
        cur = 0
        if s[i] != '0': cur += prev1
        two = int(s[i-1:i+1])
        if 10 <= two <= 26: cur += prev2
        prev2, prev1 = prev1, cur
    return prev1
```

```typescript
function numDecodings(s: string): number {
  if (s[0] === '0') return 0
  let prev2 = 1, prev1 = 1
  for (let i = 1; i < s.length; i++) {
    let cur = 0
    if (s[i] !== '0') cur += prev1
    const two = parseInt(s.slice(i-1, i+1))
    if (two >= 10 && two <= 26) cur += prev2
    prev2 = prev1; prev1 = cur
  }
  return prev1
}
```

## 92. 反转链表 II (Reverse Linked List II)

**描述**：反转链表从位置 left 到 right 的部分。

**思路**：找到 left 前一个节点，对 left~right 段进行头插法。

```python
def reverseBetween(head, left, right):
    dummy = ListNode(0, head); prev = dummy
    for _ in range(left - 1): prev = prev.next
    cur = prev.next
    for _ in range(right - left):
        nxt = cur.next
        cur.next = nxt.next
        nxt.next = prev.next
        prev.next = nxt
    return dummy.next
```

```typescript
function reverseBetween(head: ListNode | null, left: number, right: number): ListNode | null {
  const dummy = new ListNode(0, head); let prev = dummy
  for (let i = 0; i < left - 1; i++) prev = prev.next!
  let cur = prev.next!
  for (let i = 0; i < right - left; i++) {
    const nxt = cur.next!
    cur.next = nxt.next; nxt.next = prev.next; prev.next = nxt
  }
  return dummy.next
}
```

## 93. 复原 IP 地址 (Restore IP Addresses)

**描述**：给定只含数字的字符串，生成所有合法 IP 地址。

**思路**：回溯，每段1~3位数字，值0~255，无前导零。

```python
def restoreIpAddresses(s):
    res = []
    def bt(start, parts):
        if len(parts) == 4:
            if start == len(s): res.append('.'.join(parts))
            return
        for length in range(1, 4):
            if start + length > len(s): break
            seg = s[start:start+length]
            if (seg[0] == '0' and length > 1) or int(seg) > 255: continue
            bt(start + length, parts + [seg])
    bt(0, [])
    return res
```

```typescript
function restoreIpAddresses(s: string): string[] {
  const res: string[] = []
  const bt = (start: number, parts: string[]) => {
    if (parts.length === 4) { if (start === s.length) res.push(parts.join('.')); return }
    for (let len = 1; len <= 3; len++) {
      if (start + len > s.length) break
      const seg = s.slice(start, start + len)
      if ((seg[0] === '0' && len > 1) || +seg > 255) continue
      bt(start + len, [...parts, seg])
    }
  }
  bt(0, []); return res
}
```

## 94. 二叉树的中序遍历 (Binary Tree Inorder Traversal)

**描述**：返回二叉树中序遍历结果。

**思路**：迭代，用栈模拟，先一路压左孩子。

```python
def inorderTraversal(root):
    res, stack = [], []
    cur = root
    while cur or stack:
        while cur:
            stack.append(cur); cur = cur.left
        cur = stack.pop()
        res.append(cur.val)
        cur = cur.right
    return res
```

```typescript
function inorderTraversal(root: TreeNode | null): number[] {
  const res: number[] = [], stack: TreeNode[] = []
  let cur = root
  while (cur || stack.length) {
    while (cur) { stack.push(cur); cur = cur.left }
    cur = stack.pop()!
    res.push(cur.val)
    cur = cur.right
  }
  return res
}
```

## 95. 不同的二叉搜索树 II (Unique Binary Search Trees II)

**描述**：返回所有由 1~n 构成的不同BST。

**思路**：递归，枚举根节点 i，左子树用 [lo, i-1]，右子树用 [i+1, hi]。

```python
def generateTrees(n):
    def build(lo, hi):
        if lo > hi: return [None]
        res = []
        for i in range(lo, hi+1):
            for left in build(lo, i-1):
                for right in build(i+1, hi):
                    node = TreeNode(i, left, right)
                    res.append(node)
        return res
    return build(1, n)
```

```typescript
function generateTrees(n: number): Array<TreeNode | null> {
  const build = (lo: number, hi: number): Array<TreeNode | null> => {
    if (lo > hi) return [null]
    const res: Array<TreeNode | null> = []
    for (let i = lo; i <= hi; i++)
      for (const left of build(lo, i-1))
        for (const right of build(i+1, hi))
          res.push(new TreeNode(i, left, right))
    return res
  }
  return build(1, n)
}
```

## 96. 不同的二叉搜索树 (Unique Binary Search Trees)

**描述**：1~n 能构成多少种不同的BST。

**思路**：卡特兰数 DP，`G(n) = Σ G(i-1)*G(n-i)`。

```python
def numTrees(n):
    dp = [0] * (n + 1)
    dp[0] = dp[1] = 1
    for i in range(2, n + 1):
        for j in range(1, i + 1):
            dp[i] += dp[j-1] * dp[i-j]
    return dp[n]
```

```typescript
function numTrees(n: number): number {
  const dp = new Array(n + 1).fill(0)
  dp[0] = dp[1] = 1
  for (let i = 2; i <= n; i++)
    for (let j = 1; j <= i; j++) dp[i] += dp[j-1] * dp[i-j]
  return dp[n]
}
```

## 97. 交错字符串 (Interleaving String)

**描述**：判断 s3 是否由 s1 和 s2 交错组成。

**思路**：DP，`dp[j]` 表示 s1 前 i 个和 s2 前 j 个能否交错成 s3 前 i+j 个。

```python
def isInterleave(s1, s2, s3):
    m, n = len(s1), len(s2)
    if m + n != len(s3): return False
    dp = [False] * (n + 1)
    for i in range(m + 1):
        for j in range(n + 1):
            if i == 0 and j == 0: dp[j] = True
            elif i == 0: dp[j] = dp[j-1] and s2[j-1] == s3[j-1]
            elif j == 0: dp[j] = dp[j] and s1[i-1] == s3[i-1]
            else: dp[j] = (dp[j] and s1[i-1] == s3[i+j-1]) or (dp[j-1] and s2[j-1] == s3[i+j-1])
    return dp[n]
```

```typescript
function isInterleave(s1: string, s2: string, s3: string): boolean {
  const m = s1.length, n = s2.length
  if (m + n !== s3.length) return false
  const dp = new Array(n + 1).fill(false)
  for (let i = 0; i <= m; i++)
    for (let j = 0; j <= n; j++) {
      if (i === 0 && j === 0) dp[j] = true
      else if (i === 0) dp[j] = dp[j-1] && s2[j-1] === s3[j-1]
      else if (j === 0) dp[j] = dp[j] && s1[i-1] === s3[i-1]
      else dp[j] = (dp[j] && s1[i-1] === s3[i+j-1]) || (dp[j-1] && s2[j-1] === s3[i+j-1])
    }
  return dp[n]
}
```

## 98. 验证二叉搜索树 (Validate Binary Search Tree)

**描述**：判断二叉树是否是合法BST。

**思路**：递归传递上下界，每个节点必须在 (lo, hi) 范围内。

```python
def isValidBST(root):
    def check(node, lo, hi):
        if not node: return True
        if node.val <= lo or node.val >= hi: return False
        return check(node.left, lo, node.val) and check(node.right, node.val, hi)
    return check(root, float('-inf'), float('inf'))
```

```typescript
function isValidBST(root: TreeNode | null): boolean {
  const check = (node: TreeNode | null, lo: number, hi: number): boolean => {
    if (!node) return true
    if (node.val <= lo || node.val >= hi) return false
    return check(node.left, lo, node.val) && check(node.right, node.val, hi)
  }
  return check(root, -Infinity, Infinity)
}
```

## 99. 恢复二叉搜索树 (Recover Binary Search Tree)

**描述**：BST 中两个节点被错误交换，恢复它（O(1) 空间）。

**思路**：中序遍历，找两个逆序对的节点，交换它们的值。

```python
def recoverTree(root):
    first = second = prev = None
    def inorder(node):
        nonlocal first, second, prev
        if not node: return
        inorder(node.left)
        if prev and prev.val > node.val:
            if not first: first = prev
            second = node
        prev = node
        inorder(node.right)
    inorder(root)
    first.val, second.val = second.val, first.val
```

```typescript
function recoverTree(root: TreeNode | null): void {
  let first: TreeNode | null = null, second: TreeNode | null = null, prev: TreeNode | null = null
  const inorder = (node: TreeNode | null) => {
    if (!node) return
    inorder(node.left)
    if (prev && prev.val > node.val) {
      if (!first) first = prev
      second = node
    }
    prev = node
    inorder(node.right)
  }
  inorder(root)
  const tmp = first!.val; first!.val = second!.val; second!.val = tmp
}
```

## 100. 相同的树 (Same Tree)

**描述**：判断两棵二叉树是否相同。

**思路**：递归，两节点都为空返回 true，一个为空返回 false，值不等返回 false。

```python
def isSameTree(p, q):
    if not p and not q: return True
    if not p or not q: return False
    return p.val == q.val and isSameTree(p.left, q.left) and isSameTree(p.right, q.right)
```

```typescript
function isSameTree(p: TreeNode | null, q: TreeNode | null): boolean {
  if (!p && !q) return true
  if (!p || !q) return false
  return p.val === q.val && isSameTree(p.left, q.left) && isSameTree(p.right, q.right)
}
```

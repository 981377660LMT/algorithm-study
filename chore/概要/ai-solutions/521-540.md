## 521. 最长特殊序列 I (Longest Uncommon Subsequence I)

**描述**：两个字符串，找最长的「不是两者公共子序列」的子序列。

**思路**：如果两串不同，长的那个就是答案；相同返回 -1。

```python
def findLUSlength(a, b):
    return max(len(a), len(b)) if a != b else -1
```

```typescript
function findLUSlength(a: string, b: string): number {
  return a !== b ? Math.max(a.length, b.length) : -1
}
```

## 522. 最长特殊序列 II (Longest Uncommon Subsequence II)

**描述**：多个字符串中，最长的「不是其他任何字符串子序列」的字符串长度。

**思路**：按长度降序检查，判断是否为其他串的子序列。

```python
def findLUSlength(strs):
    def isSub(a, b):
        it = iter(b)
        return all(c in it for c in a)
    strs.sort(key=len, reverse=True)
    for i, s in enumerate(strs):
        if all(not isSub(s, strs[j]) for j in range(len(strs)) if i != j):
            return len(s)
    return -1
```

```typescript
function findLUSlength(strs: string[]): number {
  const isSub = (a: string, b: string) => {
    let j = 0
    for (const c of b) if (j < a.length && c === a[j]) j++
    return j === a.length
  }
  strs.sort((a, b) => b.length - a.length)
  for (let i = 0; i < strs.length; i++)
    if (strs.every((s, j) => i === j || !isSub(strs[i], s))) return strs[i].length
  return -1
}
```

## 523. 连续的子数组和 (Continuous Subarray Sum)

**描述**：长度 ≥ 2 的子数组和是 k 的倍数。

**思路**：前缀和模 k，哈希表存余数首次出现位置。

```python
def checkSubarraySum(nums, k):
    mp = {0: -1}; prefix = 0
    for i, n in enumerate(nums):
        prefix = (prefix + n) % k
        if prefix in mp:
            if i - mp[prefix] >= 2: return True
        else: mp[prefix] = i
    return False
```

```typescript
function checkSubarraySum(nums: number[], k: number): boolean {
  const mp = new Map<number, number>([[0, -1]])
  let prefix = 0
  for (let i = 0; i < nums.length; i++) {
    prefix = (((prefix + nums[i]) % k) + k) % k
    if (mp.has(prefix)) {
      if (i - mp.get(prefix)! >= 2) return true
    } else mp.set(prefix, i)
  }
  return false
}
```

## 524. 通过删除字母匹配到字典里最长单词 (Longest Word in Dictionary through Deleting)

**描述**：从 s 中删字符能组成 dictionary 中最长的词。

**思路**：双指针判断子序列，排序选最优。

```python
def findLongestWord(s, dictionary):
    dictionary.sort(key=lambda x: (-len(x), x))
    for word in dictionary:
        i = 0
        for c in s:
            if i < len(word) and c == word[i]: i += 1
        if i == len(word): return word
    return ""
```

```typescript
function findLongestWord(s: string, dictionary: string[]): string {
  dictionary.sort((a, b) => b.length - a.length || a.localeCompare(b))
  for (const word of dictionary) {
    let i = 0
    for (const c of s) if (i < word.length && c === word[i]) i++
    if (i === word.length) return word
  }
  return ''
}
```

## 525. 连续数组 (Contiguous Array)

**描述**：0 和 1 个数相等的最长子数组。

**思路**：把 0 当 -1，前缀和 + 哈希。

```python
def findMaxLength(nums):
    mp = {0: -1}; prefix = 0; res = 0
    for i, n in enumerate(nums):
        prefix += 1 if n else -1
        if prefix in mp: res = max(res, i - mp[prefix])
        else: mp[prefix] = i
    return res
```

```typescript
function findMaxLength(nums: number[]): number {
  const mp = new Map<number, number>([[0, -1]])
  let prefix = 0,
    res = 0
  for (let i = 0; i < nums.length; i++) {
    prefix += nums[i] ? 1 : -1
    if (mp.has(prefix)) res = Math.max(res, i - mp.get(prefix)!)
    else mp.set(prefix, i)
  }
  return res
}
```

## 526. 优美的排列 (Beautiful Arrangement)

**描述**：1~n 的排列中，每个位置 i 满足 perm[i] % i == 0 或 i % perm[i] == 0。

**思路**：状压 DP 或回溯。

```python
def countArrangement(n):
    cnt = 0
    def dfs(pos, visited):
        nonlocal cnt
        if pos > n: cnt += 1; return
        for num in range(1, n+1):
            if num not in visited and (num % pos == 0 or pos % num == 0):
                visited.add(num); dfs(pos+1, visited); visited.remove(num)
    dfs(1, set()); return cnt
```

```typescript
function countArrangement(n: number): number {
  let cnt = 0
  function dfs(pos: number, used: number) {
    if (pos > n) {
      cnt++
      return
    }
    for (let num = 1; num <= n; num++)
      if (!(used & (1 << num)) && (num % pos === 0 || pos % num === 0))
        dfs(pos + 1, used | (1 << num))
  }
  dfs(1, 0)
  return cnt
}
```

## 528. 按权重随机选择 (Random Pick with Weight)

**描述**：按权重概率随机选下标。

**思路**：前缀和 + 二分。

```python
import random, bisect
class Solution:
    def __init__(self, w):
        self.prefix = []
        s = 0
        for x in w: s += x; self.prefix.append(s)
        self.total = s

    def pickIndex(self):
        target = random.randint(1, self.total)
        return bisect.bisect_left(self.prefix, target)
```

```typescript
class Solution {
  prefix: number[]
  total: number
  constructor(w: number[]) {
    this.prefix = []
    let s = 0
    for (const x of w) {
      s += x
      this.prefix.push(s)
    }
    this.total = s
  }
  pickIndex(): number {
    const target = Math.floor(Math.random() * this.total) + 1
    let lo = 0,
      hi = this.prefix.length
    while (lo < hi) {
      const mid = (lo + hi) >> 1
      if (this.prefix[mid] < target) lo = mid + 1
      else hi = mid
    }
    return lo
  }
}
```

## 529. 扫雷游戏 (Minesweeper)

**描述**：点击一格后更新扫雷棋盘。

**思路**：DFS/BFS，点到空格递归展开。

```python
def updateBoard(board, click):
    r, c = click
    if board[r][c] == 'M': board[r][c] = 'X'; return board
    m, n = len(board), len(board[0])
    def dfs(i, j):
        if i < 0 or i >= m or j < 0 or j >= n or board[i][j] != 'E': return
        mines = sum(1 for di in range(-1,2) for dj in range(-1,2) if 0<=i+di<m and 0<=j+dj<n and board[i+di][j+dj]=='M')
        if mines: board[i][j] = str(mines)
        else:
            board[i][j] = 'B'
            for di in range(-1,2):
                for dj in range(-1,2): dfs(i+di, j+dj)
    dfs(r, c); return board
```

```typescript
function updateBoard(board: string[][], click: number[]): string[][] {
  const [r, c] = click,
    m = board.length,
    n = board[0].length
  if (board[r][c] === 'M') {
    board[r][c] = 'X'
    return board
  }
  function dfs(i: number, j: number) {
    if (i < 0 || i >= m || j < 0 || j >= n || board[i][j] !== 'E') return
    let mines = 0
    for (let di = -1; di <= 1; di++)
      for (let dj = -1; dj <= 1; dj++)
        if (i + di >= 0 && i + di < m && j + dj >= 0 && j + dj < n && board[i + di][j + dj] === 'M')
          mines++
    if (mines) board[i][j] = String(mines)
    else {
      board[i][j] = 'B'
      for (let di = -1; di <= 1; di++) for (let dj = -1; dj <= 1; dj++) dfs(i + di, j + dj)
    }
  }
  dfs(r, c)
  return board
}
```

## 530. 二叉搜索树的最小绝对差 (Minimum Absolute Difference in BST)

**描述**：BST 中任意两节点差的最小值。

**思路**：中序遍历后相邻差。

```python
def getMinimumDifference(root):
    prev = [None]; res = [float('inf')]
    def inorder(node):
        if not node: return
        inorder(node.left)
        if prev[0] is not None: res[0] = min(res[0], node.val - prev[0])
        prev[0] = node.val
        inorder(node.right)
    inorder(root); return res[0]
```

```typescript
function getMinimumDifference(root: TreeNode | null): number {
  let prev = -1,
    res = Infinity
  function inorder(node: TreeNode | null) {
    if (!node) return
    inorder(node.left)
    if (prev >= 0) res = Math.min(res, node.val - prev)
    prev = node.val
    inorder(node.right)
  }
  inorder(root)
  return res
}
```

## 535. TinyURL 的加密与解密 (Encode and Decode TinyURL)

**描述**：设计 URL 缩短服务。

**思路**：用哈希表 + 自增 id。

```python
class Codec:
    def __init__(self):
        self.map = {}; self.id = 0
    def encode(self, longUrl):
        self.id += 1; self.map[self.id] = longUrl
        return str(self.id)
    def decode(self, shortUrl):
        return self.map[int(shortUrl)]
```

```typescript
class Codec {
  map = new Map<number, string>()
  id = 0
  encode(longUrl: string): string {
    this.id++
    this.map.set(this.id, longUrl)
    return String(this.id)
  }
  decode(shortUrl: string): string {
    return this.map.get(Number(shortUrl))!
  }
}
```

## 537. 复数乘法 (Complex Number Multiplication)

**描述**：两复数相乘，格式 "a+bi"。

**思路**：解析后 `(a+bi)(c+di) = (ac-bd)+(ad+bc)i`。

```python
def complexNumberMultiply(num1, num2):
    a, b = map(int, num1[:-1].split('+'))
    c, d = map(int, num2[:-1].split('+'))
    return f"{a*c-b*d}+{a*d+b*c}i"
```

```typescript
function complexNumberMultiply(num1: string, num2: string): string {
  const [a, b] = num1.slice(0, -1).split('+').map(Number)
  const [c, d] = num2.slice(0, -1).split('+').map(Number)
  return `${a * c - b * d}+${a * d + b * c}i`
}
```

## 538. 把二叉搜索树转换为累加树 (Convert BST to Greater Tree)

**描述**：BST 每个节点值改为 ≥ 该节点值的所有节点值之和。

**思路**：反向中序遍历（右→根→左）累加。

```python
def convertBST(root):
    total = [0]
    def dfs(node):
        if not node: return
        dfs(node.right)
        total[0] += node.val; node.val = total[0]
        dfs(node.left)
    dfs(root); return root
```

```typescript
function convertBST(root: TreeNode | null): TreeNode | null {
  let total = 0
  function dfs(node: TreeNode | null) {
    if (!node) return
    dfs(node.right)
    total += node.val
    node.val = total
    dfs(node.left)
  }
  dfs(root)
  return root
}
```

## 539. 最小时间差 (Minimum Time Difference)

**描述**：时间列表中最小时间差（分钟）。

**思路**：转分钟数排序，相邻差 + 首尾跨天比较。

```python
def findMinDifference(timePoints):
    mins = sorted(int(t[:2])*60+int(t[3:]) for t in timePoints)
    res = 1440 - mins[-1] + mins[0]
    for i in range(1, len(mins)):
        res = min(res, mins[i]-mins[i-1])
    return res
```

```typescript
function findMinDifference(timePoints: string[]): number {
  const mins = timePoints.map(t => +t.slice(0, 2) * 60 + +t.slice(3)).sort((a, b) => a - b)
  let res = 1440 - mins[mins.length - 1] + mins[0]
  for (let i = 1; i < mins.length; i++) res = Math.min(res, mins[i] - mins[i - 1])
  return res
}
```

## 540. 有序数组中的单一元素 (Single Element in a Sorted Array)

**描述**：有序数组中只出现一次的元素，其余出现两次。

**思路**：二分。偶数索引位和右邻居比较判断方向。

```python
def singleNonDuplicate(nums):
    lo, hi = 0, len(nums) - 1
    while lo < hi:
        mid = (lo + hi) // 2
        if mid % 2: mid -= 1  # 保证 mid 为偶数
        if nums[mid] == nums[mid+1]: lo = mid + 2
        else: hi = mid
    return nums[lo]
```

```typescript
function singleNonDuplicate(nums: number[]): number {
  let lo = 0,
    hi = nums.length - 1
  while (lo < hi) {
    let mid = (lo + hi) >> 1
    if (mid % 2) mid--
    if (nums[mid] === nums[mid + 1]) lo = mid + 2
    else hi = mid
  }
  return nums[lo]
}
```

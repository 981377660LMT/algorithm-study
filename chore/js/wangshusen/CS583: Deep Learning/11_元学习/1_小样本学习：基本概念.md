# 小样本学习：基本概念

https://www.bilibili.com/video/BV1V44y1r7cx
Few-Shot Learning

这份 PDF 是 Shusen Wang 教授关于 **元学习 (Meta Learning)** 专题的第一部分，主题为 **小样本学习 (Few-Shot Learning)** 的基础概念。

以下是对该课件的深度分析与结构化解读：

### 1. 核心定义与背景

#### 1.1 什么是 Meta Learning？

- **定义**：Meta Learning 被称为 "Learn to learn"（学习如何学习）。与之相对的传统机器学习是 "Learn mapping from data"。
- **关系**：**Few-Shot Learning (小样本学习)** 是 Meta Learning 的一个重要子领域。它的目标是让模型像人类一样，仅通过极少量的样本就能快速学会识别新物体。

#### 1.2 为什么需要 Few-Shot Learning？

- **传统监督学习 (Supervised Learning)**：
  - 测试样本虽然没见过，但它们所属的**类别 (Classes)** 必须在训练集中出现过（Known Classes）。
  - 比如：训练集有猫狗，测试集也是猫狗。
- **小样本学习 (Few-Shot Learning)**：
  - 测试样本（Query）所属的**类别**从未在训练集中出现过（Unknown Classes）。
  - 比如：训练集是“狗、大象、老虎...”，但测试时给模型看“犰狳”和“穿山甲”，并要求模型识别一张新的图是犰狳还是穿山甲。
  - 模型必须利用推理阶段给出的少量参考图片（**Support Set**）来做判断。

### 2. 关键术语体系

课件清晰地定义了该领域的标准术语，这是阅读 Meta Learning 论文的基础：

- **Training Set (训练集)**：

  - 大规模数据集，包含大量样本。
  - 用于训练模型提取特征或比较相似度的能力。
  - 这里的类别与最终测试时的类别通常**不重叠**。

- **Support Set (支持集)**：

  - 在推断/测试阶段给出的一小组带标签的样本。
  - 相当于模型临时的“参考手册”。
  - 包含 $k$ 个从未见过的类别，每个类别有 $n$ 个样本。

- **Query (查询/测试样本)**：

  - 待分类的样本。
  - 模型需要判断它属于 Support Set 中的哪一类。

- **k-way n-shot**：
  - 描述 Support Set 规模的标准说法。
  - **$k$-way**：Support Set 中包含 $k$ 个不同的类别（分类选项数）。
  - **$n$-shot**：每个类别有 $n$ 个样本（参考样本数）。
  - **难度规律**：
    - **$k$ 越大越难**（选项多了，比如 10-way 比 2-way 难）。
    - **$n$ 越小越难**（参考少了，比如 1-shot 比 5-shot 难）。

### 3. 核心解决思路：学习相似度 (Similarity Learning)

PPT 中提出了一种最直观的解决 Few-Shot Learning 的方法，通常被称为 **基于度量的方法 (Metric-based Approach)**。

1.  **训练阶段**：

    - 不直接学习“这是狗”或“这是猫”。
    - 而是学习一个**相似度函数 (Similarity Function)**：$sim(x, x')$。
    - 目标：如果 $x$ 和 $x'$ 同类，相似度趋近 1；不同类，相似度趋近 0。
    - 利用大规模 Training Set 训练这个函数（本质上是在学习特征提取和距离度量）。

2.  **推理阶段 (Inference)**：
    - 给定一个 Query 图片和 Support Set。
    - 将 Query 图片与 Support Set 中的**每一张**图片计算相似度 $sim(Query, Support\_Sample)$。
    - 找出相似度最高的那个样本，将其标签作为 Query 的预测结果。
    - **举例**：Query 是水獭，它与 Support Set 中的灰猎犬相似度为 0.2，与斗牛犬为 0.1，与水獭为 0.7。因为 0.7 最高，所以判断为水獭。

### 4. 标准数据集 (Benchmarks)

课件介绍了该领域两个最常用的基准数据集：

- **Omniglot** ("机器学习的 MNIST")：

  - 包含 50 种不同的字母系统（如希腊语、梵语等），共 1623 个字符类。
  - 每个字符只有 20 个样本（由 20 个不同人手写）。
  - 非常适合测试 1-shot 或 5-shot 学习能力。
  - 结构：Training Set (30 alphabets) / Test Set (20 alphabets)。

- **Mini-ImageNet**：
  - 因为原始 ImageNet 太大，Vinyals 等人提取的一个子集，专用于 Few-Shot 评估。
  - 包含 100 个类别，每类 600 张图。
  - 通常划分：64 类训练，16 类验证，20 类测试。

### 总结

这份课件主要完成了**概念扫盲**。它指出了 Meta Learning 在 Few-Shot 场景下的本质：不是学习具体的分类知识（因为测试类别是全新的），而是**学习“如何比较”**（Learn to Compare/Measure Similarity）。如果模型学会了如何判断两张图是否属于同一物体，那么给你从未见过的外星生物图片，你也能把它们分好类。

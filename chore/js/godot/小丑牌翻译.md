# Balatro（小丑牌）— 中英文翻译对照及语源讲解

> Balatro 是一款以扑克为核心的 Roguelike 牌组构建游戏。Balatro 一词源自拉丁语 balatro（小丑/弄臣/骗子），指古罗马宴会上逗乐取笑的人。
> 本文档涵盖小丑、塔罗牌、星球牌、幽灵牌、优惠券、盲注、标签、增强、版本、封印、牌组等全部核心内容。

## 综合语源分析

### 命名主题分布

| 类别      | 命名来源                                                                                                                                                                                                                                                                                 |
| --------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 小丑名    | 形容词系列（Greedy/Lusty/Wrathful = 七宗罪）、历史弄臣（Canio/Triboulet/Yorick/Chicot/Perkeo）、戏剧术语（Mime/Sock and Buskin/Troubadour）、食物（Gros Michel/Cavendish/Ramen/Ice Cream/Popcorn）、科学（DNA/Fibonacci/Superposition）、经济（Credit Card/Bull/To the Moon/Bootstraps） |
| 塔罗牌    | 完全对应传统 22 张大阿卡那                                                                                                                                                                                                                                                               |
| 星球牌    | 太阳系天体按大小/距离排列对应牌型强弱                                                                                                                                                                                                                                                    |
| 幽灵牌    | 超自然/灵异/神秘主义（Ouija/Séance/Ectoplasm/Talisman/Sigil）                                                                                                                                                                                                                            |
| 优惠券    | 商业（Clearance Sale/Liquidation）+ 科学（Antimatter）+ 考古（Hieroglyph/Petroglyph）                                                                                                                                                                                                    |
| Boss 盲注 | 身体部位系列（The Hook/The Eye/The Mouth/The Tooth/The Head/The Arm）+ 抽象概念                                                                                                                                                                                                          |
| 牌组      | 颜色编码 + 科学术语（Plasma/Nebula/Anaglyph）                                                                                                                                                                                                                                            |

### 文化亮点

- **七宗罪**：Greedy（贪婪）、Lusty（色欲）、Wrathful（暴怒）、Gluttonous（暴食）——四个花色小丑对应七宗罪中的四宗
- **香蕉灭绝**：Gros Michel（大麦克）和 Cavendish（卡文迪什）——真实的香蕉品种更替历史，大麦克几乎灭绝后被卡文迪什取代，游戏中卡文迪什消失概率极低（1/1000 vs 1/6）
- **历史弄臣**：五位传奇小丑全部有据可查——Canio（歌剧角色）、Triboulet（法国宫廷）、Yorick（莎剧）、Chicot（法国宫廷）、Perkeo（德国宫廷）
- **量子物理**：Superposition（叠加态）——A 既在顺子里又是高牌，如同薛定谔的猫
- **佛罗里达选举**：Hanging Chad——2000 年美国大选的打孔计票争议，被做成了一张牌
- **扑克文化**：The House always wins（赌场必胜），Fish（菜鸟），Card Sharp（老千）

---

> **总结**：Balatro 的翻译体系围绕扑克文化、塔罗占卜、天文学三大核心展开，同时融合了七宗罪、意大利即兴喜剧（Commedia dell'arte）、历史弄臣、量子物理、经济学、食物等多种文化元素。游戏名 "Balatro" 本身就暗示了对整个扑克世界的戏谑颠覆——你不是正经牌手,而是一个 "骗子小丑"。

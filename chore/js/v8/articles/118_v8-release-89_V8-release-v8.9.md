# V8 release v8.9: Top-level await and Stack Optimization

V8 v8.9 见证了多项异步能力和底层性能技术的集中落地。

### 1. 核心特性：顶层 await (Top-level await)

继实验阶段后，顶层 `await` 终于正式上线。它允许开发者在模块顶层直接同步化地等待异步结果，大幅简化了模块依赖的加载逻辑。

### 2. 重要成果：适配帧优化的实装

119 号文章提到的**移除参数适配帧**优化在本版本正式启用。这项底层改动为所有函数调用带来了普适性的性能提升，尤其是那些参数不匹配的场景。

### 3. Wasm 与 调试

提升了 WebAssembly 的调试体验，并且基准编译器 Liftoff 支持了更多的原子操作，为多线程协同提供了更强的指令支持。

### 4. 一针见血的见解

v8.9 是一个承前启后的版本。它在大范围集成底层性能模型重塑（适配帧移除）的同时，通过 Top-level await 完成了 JS 异步语法的最后一块拼图。这使得 Web 开发者的编码体验和应用的运行效率同步得到了质的飞跃。

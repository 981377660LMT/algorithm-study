# V8 release v5.5: Async/await and memory reduction wins

V8 v5.5 是一个极其重要的发布，它不仅带来了异步语法的巅峰，还大幅优化了内存占用。

### 1. 异步之巅：原生 Async/Await

此版本正式支持 ES2017 的 `async/await` 语法。虽然内部仍将其转换为 Promise 链，但通过引擎级别的深度整合，异步代码的执行效率和内存占用都得到了显著改善。

### 2. 内存大扫除：堆占用下降 35%

针对低内存（512MB RAM）设备的专项优化开始结出硕果。相比 v5.3，典型网页的内存占用下降了超过三分之一。这主要归功于更激进的垃圾回收策略和对闲置内存的及时归还。

### 3. 一针见血的见解

v5.5 证明了 V8 能够同时在“高级语法”和“低级内存优化”两个方向上取得成功。它让 JS 应用既能写得优雅（Async/Await），又能在各种简陋的硬件设备上稳定运行。

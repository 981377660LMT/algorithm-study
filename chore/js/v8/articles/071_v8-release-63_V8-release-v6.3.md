# 71. V8 release v6.3 | V8 v6.3 发布：动态导入与 SharedArrayBuffer

V8 v6.3 是一个大版本更新，引入了革命性的动态加载能力。

## 1. 动态 `import()`

正式支持 `import()` 表达式。

- **实现细节**：这要求 V8 的解析器能够处理异步模块解析，并与宿主（如 Chrome）充分协作，以按需拉取和执行脚本，而不是在启动时加载全部。

## 2. SharedArrayBuffer 与 Atomics

为了实现高效的多线程 JS 协作，引入了 `SharedArrayBuffer`。

- **底层支持**：V8 需要处理复杂的内存一致性模型。配合 `Atomics` 原子操作，确保在多核并发读写下不会出现严重数据争用。

## 3. TurboFan 优化：Object.is

`Object.is` 以往比 `===` 慢得多。

- **优化**：V8 v6.3 在 TurboFan 中为 `Object.is` 专门增加了一个内建操作指令。
- **效果**：在处理同类型比较或已知非 NaN 的数字比较时，它的性能表现现在趋近于 `===`。

## 4. Intl.PluralRules

新的国际化 API 支持，用于处理不同语言中的复数形式（如 "1 copy" vs "2 copies"）。

## 5. 性能改进：对象字面量

对大规模对象字面量的解析和初始化进行了优化。之前的实现可能会导致深层嵌套的对象字面量引起栈溢出，v6.3 引入了更加健壮的解析策略。

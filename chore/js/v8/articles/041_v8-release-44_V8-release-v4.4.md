# V8 release v4.4: ES6 core features stable and ready

V8 v4.4 将一系列革命性的 ES6 语法特性从实验标记后移出，标志着 JS 进入了新纪元。

### 1. 计算属性名 (Computed Properties)

支持了 `{[key]: value}` 语法。虽然看似简单，但在引擎内部，这要求属性名在编译阶段必须是动态可解析的。V8 针对这种动态 Key 的隐藏类（Hidden Class）转换路径进行了专门优化，保证了即使是动态生成的属性名也能进入单态 IC（Monomorphic IC）。

### 2. Rest 参数与 Spread 元素

这两大特性在 v4.4 中全面转正。V8 团队针对函数调用时对参数数组的展开逻辑进行了内联优化，解决了以往通过 `arguments` 手动模拟导致的性能不确定性。

### 3. 一针见血的见解

v4.4 是对 ES6 基础语法的“收官之作”。它不仅提供了语法上的完备，更重要的是通过底层优化，让这些抽象程度更高的代码在运行初期的“地板性能”就超出了手写降级代码（Desugared code）的表现。

# V8 release v5.0: RegExp Unicode and ES6 performance

V8 v5.0 标志着对 ES6 规范支持的全面覆盖，并显著提升了现代语法的执行效率。

### 1. 正则补完：Unicode 模式 (`u` 标志)

引入了对 ES6 正则 `u` 标志的支持。这解决了 JavaScript 长期无法正确处理 4 字节 Unicode 字符（如 Emoji `😊`）匹配的问题。同时，通过 Symbol 钩子（`Symbol.match`/`replace`），正则行为变得可编程和可定制。

### 2. 核心特性提速

- **Rest 参数**：通过优化参数数组的分配逻辑，性能提升了 8-10 倍。
- **`Object.keys`**：优化了内部枚举属性的缓存机制，属性访问速度翻倍。

### 3. 一针见血的见解

v5.0 是 V8 从“新特性开发”转向“特性精雕细琢”的转折点。它不仅实现了 ES6，还通过对 Rest 参数等基础特性的重构，确保了现代 JS 语法能够获得与旧语法同级别甚至更高的机器性能。

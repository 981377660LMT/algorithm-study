# 为 Agent 赋予视觉表达：VS Code 中的 MCP 应用支持

链接：https://code.visualstudio.com/blogs/2026/01/26/mcp-apps-support

## 深入分析

### 1. 核心突破：从“对话”到“交互”

MCP Apps 的引入标志着 AI Agent 从纯文本输出进化到了**交互式 UI 输出**。传统 Agent 只能通过 Markdown 表格或代码块展示信息，而 MCP Apps 允许 Agent 直接在聊天窗口渲染拖拽列表、实时火焰图、功能开关面板等动态组件。这极大地降低了开发者在复杂决策（如调整排序、分析性能瓶颈）时的认知负载。

### 2. 人机协作的新范式

- **直观验证**：Agent 不再只是“说”它的分析结果，而是通过可视化手段（如 Storybook 预览）“展示”结果，开发者可以即时预览并修改。
- **闭环操作**：通过集成如 Feature Flag 选择器等 UI，开发者可以在不离开编辑器的环境下，一站式完成查看状态、切换环境、生成代码的闭环。

### 3. 生态意义

MCP Apps 作为 MCP 协议的首个正式扩展，统一了 AI 交互界面的标准。这意味着第三方开发者（如 Storybook）只需开发一次 MCP 适配，即可在支持该标准的任何编辑器（首发为 VS Code）中获得一致的富交互体验。

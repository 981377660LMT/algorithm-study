# Visual Studio Code 在 Build 2020

链接：https://code.visualstudio.com/blogs/2020/05/06/code-at-build

## 深入分析

Build 2020是疫情年的特殊大会，VS Code的演讲反映了远程工作时代的需求。

### 主题

1. **Remote Development的深化** - 不仅SSH，还有DevContainers和GitHub Codespaces的预告
2. **协作开发** - Live Share的正式发布和推广
3. **教育与学习** - VS Code在在线教育中的应用

### 时代背景

- 2020年初，全球进入居家工作模式
- 远程开发工具的需求突然爆发
- VS Code凭借Remote Development功能，成为远程开发的标准选择

### GitHub Codespaces的意义

- 云编辑器不是新概念（Cloud9、Koding等早就存在）
- 但Codespaces的特殊性在于——它基于VS Code，而非独立的web编辑器
- 用户获得了一致的开发体验：无论本地还是云端，都是熟悉的VS Code

### 产品战略

- 微软意识到IDE的未来可能是混合模式——本地+云端
- Codespaces是这一战略的实现，但定价相对较高（需要付费），面向企业用户

# 在 Chromebook 上使用 VS Code 学习

链接：https://code.visualstudio.com/blogs/2020/03/02/chromebook-learning

## 深入分析

这篇文章针对教育市场，特别是资源受限的学校和地区。

### Chromebook的特点
- 便宜（通常$100-300）
- 轻量级（启动快，续航长）
- 安全性好（没有本地文件可被病毒感染）
- 但计算能力有限

### VS Code + Chromebook的组合
1. **VS Code for Web** - 可以在浏览器中运行的轻量版本
2. **编程教学** - Python、JavaScript等入门级编程
3. **跨平台一致性** - 学生回家可以用相同的环境

### 教育意义
- 降低编程教育的硬件成本
- 让贫困地区的学生也能接受专业的编程教育
- VS Code的易用性，特别适合初学者

### 竞争优势
- PyCharm虽然好用，但对Chromebook这样的低端设备太重了
- Jupyter Notebook是个选择，但不如IDE的体验完整

### 社会责任
- 这反映了微软对教育平等性的承诺
- 培养新一代开发者，建立长期竞争力

# 教程的问题

链接：https://code.visualstudio.com/blogs/2022/07/25/tutorial-problem

## 深入分析

这是一篇有趣的思考文章，讨论教程的局限性和VS Code的教育哲学。

### 教程的问题

1. **过时性** - 教程依赖于特定版本的库和工具，易过时
2. **不够深入** - 为了让初学者理解，教程往往跳过重要细节
3. **缺乏完整性** - 教程通常只覆盖"happy path"，不处理错误情况

### VS Code的解决思路

- 不是写更多教程，而是提供**可复现的学习环境**
- 使用DevContainer+VS Code，确保每个学生都有相同的环境
- 减少"跟着教程走，但你的环境不一样"的问题

### 例子

- 传统教程：下载Python 3.9，安装Django 3.0，创建项目...（很容易失败）
- VS Code方式：一键启动DevContainer，环境已预配置，立即开始coding

### 教育哲学的转变

- 从"教程 + 文档"→ "可复现的环境 + 交互式指导"
- 这反映了VS Code对教育的深刻理解

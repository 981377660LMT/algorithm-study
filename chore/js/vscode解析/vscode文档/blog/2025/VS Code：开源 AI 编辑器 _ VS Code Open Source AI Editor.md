# VS Code：开源 AI 编辑器

链接：https://code.visualstudio.com/blogs/2025/05/19/openSourceAIEditor

## 深入分析

### 1. 核心叙事：AI 是编辑器的“原子能力”

VS Code 决定将 GitHub Copilot Chat 开源并将 AI 功能合入核心代码库，宣告了 AI 不再是插件层面的“增量”，而是 IDE 不可或缺的“基座”。这一动作类似于当年内置对 Git 的支持，确立了 AI 原生编辑器的新标准。

### 2. 开源的多重战略价值

- **透明度与信任**：通过公开 Copilot Chat 代码，用户可以清晰地审计数据收集逻辑，回应 AI 背景下的隐私焦虑。
- **防范“秘密配方”依赖**：微软坦承随着大模型能力的提升，传统的 Prompt 工程壁垒已不再是“秘密武器”，通过开源可以换取社区共建底层基座，从而专注于上层极致的 UX 体验。
- **生态赋能**：第三方 AI 扩展作者现在可以参照官方实现进行深度定制，解决了过去黑盒状态下难以测试和集成的问题。

### 3. 未来的社区驱动

通过开源 Prompt 测试基础设施，VS Code 旨在建立一套抗随机性的工程化测试标准，确保社区贡献在不确定的 LLM 响应下依然保持高质量输出。

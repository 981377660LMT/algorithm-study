# 推介 VS Code Insiders 播客

链接：https://code.visualstudio.com/blogs/2025/12/03/introducing-vs-code-insiders-podcast

## 深入分析

### 1. 软实力的构建：超越发布日志

VS Code 团队意识到，单纯的文本发布日志已不足以传达 AI 时代复杂功能的底层逻辑。推介 **VS Code Insiders Podcast** 是为了通过音频载体，让核心开发者直面用户，解释每一个 Feature（如 Planning Agent 或 MCP 实现）背后的权衡与愿景。

### 2. 人本位的开源协作

播客内容涵盖了无障碍设计、社区治理等深度话题。这不仅是为了技术分发，更是为了构建一种“背后有人”的品牌信任。对于开源项目，这种透明的决策沟通常常比代码本身更能吸引高质量的贡献者。

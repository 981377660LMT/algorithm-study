# Sublime Text 扩展综述

链接：https://code.visualstudio.com/blogs/2017/09/14/sublime-roadmap

## 深入分析

这期综述的特殊性在于"为Sublime Text用户推荐VS Code扩展"，反映了VS Code开始蚕食Sublime的市场。

### 背景

- Sublime Text在2016-2017年期间陷入困境（开发团队很小，更新频率低）
- 大量Sublime用户开始考虑迁移
- VS Code团队意识到这是一个市场机会，主动出击

### 迁移战略

1. **提供Sublime快捷键绑定** - 降低Sublime用户的迁移成本
2. **推荐"Sublime精神"的扩展** - 例如，崇尚简洁设计的主题、强大的多选编辑功能
3. **对标Sublime的工作流** - Sublime的Goto Anything在VS Code中通过Command Palette实现

### 心理学考量

- "我们理解你为什么爱Sublime"——这种共鸣很重要
- 不是贬低Sublime，而是帮助Sublime用户找到VS Code中的等价功能
- 这种"友好的交接"，让迁移变成了一个愉快的过程

### 结果

- 这一时期，大量Sublime用户确实迁移到VS Code
- 到2018年，VS Code已成为市场第一编辑器（按使用人数）

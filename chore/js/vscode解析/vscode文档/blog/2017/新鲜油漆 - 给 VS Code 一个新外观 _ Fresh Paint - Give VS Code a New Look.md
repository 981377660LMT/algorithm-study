# 新鲜油漆 - 给 VS Code 一个新外观

链接：https://code.visualstudio.com/blogs/2017/06/20/great-looking-editor-roundup

## 一针见血的分析

1. **外观也是生产力**：VS Code 在 2017 年密集推出各种调色盘、图标主题功能，实际上是补齐了与 Atom 和 Sublime Text 的竞争盲区。开发者对「美观」的追求不亚于代码性能。
2. **社区驱动的配置化**：通过引入 `editor.fontLigatures` 和丰富的文件图标扩展，VS Code 证明了它的 UI 层虽然基于 Electron，但可以做到高度定制化且不损失流畅度。
3. **生态系统的粘性**：主题（Themes）通常是用户接触一个新编辑器的第一步。VS Code 大力推广主题市场，是为了快速扩大其「非深度用户」群，让他们因为「好看」而留下，再被其背后的 LSP/DAP 等硬核技术锁定。

## 摘要

本篇博客回顾了 VS Code 在界面自定义方面的各项改进，旨在帮助开发者打造更具个性化和美观的工作环境。

- **颜色主题 (Color Themes)**：推介了 Sapphire, Nord 和 Ayu 等热门主题。
- **文件图标主题 (File Icon Themes)**：展示了如何通过图标快速识别文件类型，如 `VSCode Great Icons`。
- **字体连字 (Font Ligatures)**：介绍了如何开启连字功能（如 `editor.fontLigatures: true`），并结合 Fira Code 等字体提升代码的可读性与美感。
- **自定义设置**：强调了 VS Code 允许通过简单的 JSON 配置深度定制编辑器外观。

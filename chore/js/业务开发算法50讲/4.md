- Golang 相比于 Java 来说确实是一门更现代的语言，在语言实现上采用了更强的内存分配器，而且通过引入更完备的逃逸分析机制等手段一定程度上`避免了大量小对象的产生`，所以 Golang 并不像 Java 那么需要快速的 GC，也就没必要引入分代 GC 这样的复杂设计了。

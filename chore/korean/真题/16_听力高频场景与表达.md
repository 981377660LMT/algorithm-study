| 한국어                           | 중문          |
| -------------------------------- | ------------- |
| 오늘은 ~에 대해 말씀드리겠습니다 | 今天讲~       |

## 6. 접속 표현 듣기 키 패턴

| 패턴             | 의미       | 듣기 핵심          |
| ---------------- | ---------- | ------------------ |
| A지만 B          | 虽然A但B   | **B가 중심 생각**  |
| 물론 A 그러나 B  | 当然A 但B  | **B가 진짜 의견**  |
| A뿐만 아니라 B도 | 不仅A而且B | A+B 모두 중요      |
| A(으)므로 B      | 因为A所以B | 인과 관계          |
| A는 반면 B       | A的另一面B | 대조               |
| 특히             | 尤其       | **뒤의 내용 강조** |
| 결국/따라서      | 最终/因此  | **결론**           |

## 1~2번: 문법·어휘 빈칸

### 전략

1. **빈칸 앞뒤 문맥**: 접속 관계(이유/대조/조건 등) 파악
2. **문법 패턴**: -(으)니까, -아/어서, -(으)면, -지만 등
3. **시제**: 과거/현재/미래 일치 확인

### 빈출 문법

| 문법         | 기능       | 빈칸 단서      |
| ------------ | ---------- | -------------- |
| -(으)니까    | 이유(주관) | 뒤에 명령/제안 |
| -아/어서     | 이유(객관) | 순서/결과      |
| -(으)면      | 조건       | 가정 상황      |
| -지만        | 대조       | 반대 내용      |
| -(으)ㄴ/는데 | 배경/전환  | 상황 설명+결과 |
| -뿐만 아니라 | 추가       | A+B 모두       |

---

## 3~4번: 유의어 (밑줄 → 비슷한 의미)

### 전략

1. **문맥 대입**: 각 선지를 밑줄 자리에 넣어 자연스러운지 확인
2. **한자어 ↔ 고유어**: 한자어는 다른 한자어로 바꿔 출제
3. **관용 표현**: 관용어의 속뜻 파악

### 빈출 유의어 쌍

| 출제 표현       | 정답               |
| --------------- | ------------------ |
| 급성장하다      | 빠르게 성장하다    |
| 실효성이 없다   | 효과가 없다        |
| 파장을 일으키다 | 영향을 미치다      |
| 역경을 딛다     | 어려움을 극복하다  |
| 통념을 뒤집다   | 기존 믿음을 뒤엎다 |
| 호응을 얻다     | 지지를 받다        |
| 시인하다        | 인정하다           |

---

## 5~8번: 짧은 글 내용 일치

### 전략

1. **선지 먼저 읽기**: 4개 선지를 빠르게 훑고 → 지문에서 확인
2. **키워드 매칭**: 선지의 핵심 단어가 지문에 있는지 확인
3. **왜곡 주의**: 숫자 바꿔치기, 주어 뒤바꿈 등

---

## 9~12번: 화제/주제

### 전략

1. **반복 키워드**: 2~3번 나오는 핵심 단어 = 화제
2. **첫+끝 문장**: 보통 주제문이 존재
3. **범위 적절성**: 너무 넓거나 좁은 선지 제거

---

## 13~15번: 글쓴이의 의도/목적

### 전략

1. **글의 종류 파악**: 안내문? 광고? 호소문? 추천서?
2. **핵심 동사**: "~을 권합니다/요청합니다/안내합니다/소개합니다"
3. **마지막 문장**: 목적이 명확하게 드러남

### 글 종류별 의도

| 글 종류     | 의도                            |
| ----------- | ------------------------------- |
| 안내문      | 정보를 안내하려고               |
| 광고        | 제품/서비스를 홍보하려고        |
| 호소문      | 문제를 알리고 참여를 촉구하려고 |
| 추천서/서평 | 도서/제품을 추천하려고          |
| 민원        | 개선/보수를 요청하려고          |

---

## 16~18번: 글쓴이의 태도/생각

### 전략 ⭐⭐⭐

1. **"~해야 한다"**: 글쓴이의 주장
2. **"그러나/하지만" 이후**: 진짜 견해
3. **균형 vs 한쪽**: 양쪽 장단점 → "균형이 필요하다"

### 태도 키워드

| 표현                  | 태도           |
| --------------------- | -------------- |
| ~(이)라 보기 어렵다   | 부정적/회의적  |
| ~(이)라고 할 수 있다  | 긍정적/동의    |
| ~아/어야 한다         | 당위성/주장    |
| ~문제로 대두되고 있다 | 우려/경각심    |
| ~의 균형이 필요하다   | 중립/균형 추구 |

---

## 19~24번: 중간 길이 내용 일치

### 전략

- 5~8번과 같으나 지문이 더 길고 내용이 구체적
- **고유명사, 연도, 수치** 정확히 확인
- **"모든/항상/절대/유일한"** 등 극단적 표현 = 높은 확률로 오답

---

## 25~27번: 문장 순서 배열 ⭐⭐⭐⭐⭐

### 3단계 풀이법

1. **첫 문장 찾기**: 배경/일반적 사실 = 주어+서술어 완결형
2. **접속어 연결**: 그래서/따라서 → 뒤 / 그러나 → 중간 / 이는 → 앞 참조
3. **지시어 추적**: "이/이러한/그/그러한" → 앞에 근거 필요

### 순서 단서 정리

| 단서             | 위치               |
| ---------------- | ------------------ |
| 일반론/현상 제시 | 첫 번째            |
| 그래서/따라서    | 뒤쪽 (결론)        |
| 특히/예를 들어   | 중간 (구체화)      |
| ~기 때문이다     | 마지막 (이유 후치) |
| 이/이러한/그     | 지시 대상 뒤에     |
| 그러나/하지만    | 전환점             |

---

## 28~31번: 문장 삽입 ⭐⭐⭐⭐⭐

### 3단계 풀이법

1. **<보기> 분석**: 접속어("그러나/이처럼/즉") + 내용
2. **<보기> 앞의 문장**: <보기>가 이어받는 내용이 있어야 함
3. **<보기> 뒤의 문장**: <보기> 다음에 자연스럽게 연결되어야 함

### 접속어별 삽입 위치

| <보기> 시작      | 삽입 위치                  |
| ---------------- | -------------------------- |
| 그러나/하지만    | 긍정 → [삽입] → 부정       |
| 이처럼/이와 같이 | 예시/나열 → [삽입] → 결론  |
| 즉/다시 말해     | 추상 → [삽입(부연)] → 구체 |
| 이는/이것은      | 현상 → [삽입(설명)] → 전개 |

---

## 32~34번: 빈칸 추론 ⭐⭐⭐⭐⭐

### 전략

1. **빈칸 전후 문장**: 결론 표지("따라서/즉/결국") 뒤 빈칸 = 글의 요약
2. **글 전체 요지**: 빈칸은 글의 중심 내용을 함축
3. **비유/격언형**: "~하는 것이 아니라 ~하는 것이다" 구조 빈출

---

## 35~50번: 장문 독해 (2문항 세트) ⭐⭐⭐⭐⭐

### 전략

1. **문제 먼저**: ① 주제/중심 생각 ② 내용 일치 중 유형 확인
2. **문단별 핵심**:
   - 1문단 = 도입/배경
   - 2문단 = 심화/결론/주장
3. **주제 → 첫+끝 문장 / "그러나" 이후**
4. **내용 일치 → 선지 키워드를 지문에서 대조**

### 주제 파악 공식

```
[1단] 일반론/현상 → [2단] "그러나/하지만" → 핵심 주장
= 핵심 주장이 주제/중심 생각
```

## 시간 관리 (총 70분)

| 구간    | 시간    | 전략                    |
| ------- | ------- | ----------------------- |
| 1~12번  | 0~12분  | 빠르게 풀기 (1문제 1분) |
| 13~24번 | 12~25분 | 안정적 (1문제 1분)      |
| 25~34번 | 25~41분 | 집중 (순서/삽입/빈칸)   |
| 35~50번 | 41~70분 | 최고 집중 (2문항 세트)  |

### 핵심 원칙

- **모르는 문제 30초 이상 고민 금지** → 마킹하고 다음으로
- **뒤에서부터 풀기도 전략**: 35~50번이 배점이 높음
- **극단적 표현 = 오답**: "모든/항상/결코/절대/반드시/유일한"

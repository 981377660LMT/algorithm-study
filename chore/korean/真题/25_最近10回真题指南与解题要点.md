## 六、문형별 풀이 요점 (10회 종합)

### 듣기 1~4번 (이어질 말)

- **핵심**: 상대방의 말에 자연스러운 **응답** 선택 (이유 추궁 ✕)
- **함정**: 질문을 되묻는 선지 → 오답

### 듣기 9~12번 (내용 일치)

- **핵심**: 숫자·날짜·장소 **정확히** 확인
- **함정**: 비슷한 숫자로 바꿔치기 (예: 2주 vs 1개월)

### 듣기 17~20번 (중심 생각)

- **핵심**: "그러나/하지만" **이후** = 중심 생각
- **함정**: "그러나" 이전의 내용을 중심 생각으로 착각

### 듣기 37~50번 (세트 문항)

- **핵심**: 문제 2개를 **먼저 읽고** 듣기 시작
- **풀이법**: 1번째 문제 = 전체 요지, 2번째 문제 = 세부 내용 일치

### 읽기 3~4번 (유의어)

- **핵심**: 밑줄 단어를 각 선지로 **대입** → 문맥 자연스러운 것
- **함정**: 한자어의 한 글자만 같은 오답

### 읽기 25~27번 (순서 배열)

- **핵심**: ①첫 문장 = 일반론 ②"따라서" = 뒤 ③"이는/이것은" = 앞 글 참조
- **풀이법**: 접속어·지시어 → 앞뒤 관계 확인

### 읽기 28~31번 (문장 삽입)

- **핵심**: <보기> 첫 단어 분석 (그러나→전환, 이처럼→요약, 즉→부연)
- **풀이법**: <보기> 앞 문장 + <보기> + 뒤 문장 = 자연스러운 흐름

### 읽기 32~34번 (빈칸 추론)

- **핵심**: 빈칸 = 글의 **핵심 요지** 또는 **결론**
- **풀이법**: 글 전체 읽기 → 요지 파악 → 빈칸에 대입

## 1~4번: 알맞은 그림 고르기

### 전략

1. **3대 요소 체크**: 장소, 인물, 행동/물건
2. **위치 표현 집중**: 위에/아래에/옆에/앞에/뒤에
3. **소거법**: 틀린 그림 먼저 제거

### 빈출 장소 단서

- 바다+배 → 해변 장면
- 산+배낭 → 등산 장면
- 책상+컴퓨터 → 사무실/공부 장면

---

## 5~8번: 이어질 수 있는 말

### 전략

1. **질문 유형 파악**: 장소? → 위치 답 / 시간? → 시간 답 / 제안? → 수락/거절
2. **대화 맥락**: 직전 발화의 의도를 정확히 파악
3. **오답 패턴**: 단어는 관련 있지만 맥락이 안 맞는 선지

### 핵심 의도 패턴

| 질문 유형 | 적절한 응답         |
| --------- | ------------------- |
| 장소 질문 | 위치/방향 답변      |
| 시간 질문 | 시간/기간 답변      |
| 이유 질문 | 이유/원인 답변      |
| 제안/초대 | 수락 또는 거절+이유 |
| 부탁/요청 | 수락 또는 거절+대안 |

---

## 9~12번: 대화 내용과 같은 것

### 전략

1. **선지 사전 읽기**: 문제지의 4개 선지를 듣기 전에 빠르게 읽기
2. **키워드 메모**: 숫자, 날짜, 장소, 인물 적음
3. **소거법**: 들리는 내용과 다른 선지 즉시 제거

### 오답 패턴

| 패턴           | 설명                            |
| -------------- | ------------------------------- |
| 주어 바꿔치기  | 남자→여자, 여자→남자            |
| 시간/장소 왜곡 | 화요일→목요일, 강남→종로        |
| 부분만 맞음    | 앞부분은 맞지만 뒷부분이 틀림   |
| 과장/축소      | "모든/절대/항상" 등 극단적 표현 |

---

## 13~16번: 대화의 화제

### 전략

1. **반복 키워드 = 화제**: 대화에서 2~3번 반복되는 핵심 단어
2. **전체 vs 부분**: 화제는 대화 전체를 아우르는 것
3. **너무 넓거나 좁은 선지 제거**

---

## 17~20번: 중심 생각

### 전략 ⭐⭐⭐

1. **마지막 발화 집중**: 상대 의견에 대한 최종 반응이 중심 생각
2. **"그러나/하지만" 이후**: 전환 접속사 뒤가 진짜 의견
3. **"~(이)라고 생각해요"**: 직접적 의견 표현

### 핵심 공식

```
물론 A → 그러나 B → B가 중심 생각
A에 대해 → 저는 B라고 봅니다 → B가 중심 생각
```

---

## 21~24번: 장문 대화 내용 일치

### 전략

- 9~12번과 동일한 전략이지만 대화가 더 길음
- **키워드 메모 필수**: 시간, 장소, 가격, 인원 등 디테일
- 선지 4개 중 3개는 "거의 맞는데 한 부분이 다름"

---

## 25~28번: 뉴스/안내 내용 일치

### 전략

1. **5W1H 메모**: 누가/언제/어디서/무엇을/왜/어떻게
2. **숫자 집중**: %, ○만 원, ○월 ○일 등
3. **뉴스 구조**: 핵심(리드) → 세부 → 배경/전망

### 장르별 키워드

| 장르 | 핵심 포인트                 |
| ---- | --------------------------- |
| 뉴스 | 수치, 정책명, 시행 시기     |
| 안내 | 시간, 장소, 대상, 주의 사항 |
| 광고 | 제품, 특징, 할인, 방법      |
| 다큐 | 주제, 연구 결과, 특이 사실  |

---

## 29~32번: 강연/인터뷰 중심 생각

### 전략 ⭐⭐⭐⭐

1. **마지막 1~2문장 = 결론**: 강연의 핵심은 끝에
2. **"중요한 것은~"**: 핵심 주장 신호
3. **양보→반박**: "물론~그러나~" = 반박이 중심

---

## 33~50번: 2문항 세트

### 문제 구성

- **첫 번째 문제**: 화자 파악 / 주제 / 중심 생각
- **두 번째 문제**: 내용 일치 / 세부 사항

### 전략 ⭐⭐⭐⭐⭐

1. **두 문제 모두 미리 읽기**: 방송 전 선지를 훑어보기
2. **첫 1~2문장으로 화자/주제 파악**: 빠른 판단
3. **전문 용어 = 전문가**: ~교수/~학자/~전문가
4. **찬반 구분**: 토론에서 각 화자 입장 따로 메모

### 고빈도 화자 유형

| 단서                               | 화자                 |
| ---------------------------------- | -------------------- |
| "교수님/선생님, ~에 대해"          | 인터뷰 진행자+전문가 |
| "오늘은 ~에 대해 이야기하겠습니다" | 강연자               |
| "찬반 의견을 들어 보겠습니다"      | 토론 사회자          |
| "연구에 따르면"                    | 연구자/학자          |
| "이 제품은/이 서비스는"            | 광고                 |

## 시간 관리

| 구간    | 시간    | 핵심               |
| ------- | ------- | ------------------ |
| 1~12번  | 0~12분  | 빠르게 → 시간 절약 |
| 13~24번 | 12~25분 | 안정적으로         |
| 25~36번 | 25~40분 | 집중력 유지        |
| 37~50번 | 40~60분 | 최고 집중          |

### 마킹 전략

- 확실한 답은 바로 마킹
- 모르겠으면 직감으로 마킹하고 다음 문제 집중
- **절대 한 문제에 매달리지 말 것** (다음 문제도 놓침)

## 二、과목별 실전 전략 요약

### 듣기 (60분 / 100점)

| 구간    | 전략                                                 |
| ------- | ---------------------------------------------------- |
| 1~12번  | **선지 먼저 읽기** → 핵심 키워드 밑줄 → 소거법       |
| 13~20번 | 첫 문장+마지막 문장 집중 ("그러나" 이후 = 중심 생각) |
| 21~36번 | 문제 2개를 먼저 읽고 → 듣기 시작                     |
| 37~50번 | 메모하며 듣기 (핵심 키워드, 숫자, 입장)              |

**듣기 마킹 전략:**

- 확신 → 바로 마킹
- 2개로 좁힘 → 먼저 마킹, 나중에 재확인
- 모르겠음 → ③ 마킹 후 넘기기 (통계적으로 ③이 정답인 경우가 가장 많음)

---

### 쓰기 (50분 / 100점)

| 문항 | 배점 | 시간 | 핵심                              |
| ---- | ---- | ---- | --------------------------------- |
| Q51  | 10점 | 3분  | 경어체 + 빈칸 맥락에 맞는 한 문장 |
| Q52  | 10점 | 4분  | 경어체 + 빈칸 맥락에 맞는 한 문장 |
| Q53  | 30점 | 13분 | 200~300자 + 수치 정확 + 증감 표현 |
| Q54  | 50점 | 30분 | 600~700자 + 서론/본론/결론        |

**쓰기 필수 표현:**

**Q51-52 경어체 마무리:**

- ~해 주시기 바랍니다
- ~하오니 양해 바랍니다
- ~하시면 감사하겠습니다
- ~(으)시기를 부탁드립니다

**Q53 수치 표현:**

- ~%로 가장 높은 비율을 차지했다
- ~%에서 ~%로 증가/감소하였다
- 가장 큰 폭으로 늘었다/줄었다
- 약 두 배로 늘어났다

**Q54 구조 공식:**

```
[서론 2~3줄]
~(이)라는 주제에 대해 논의가 활발하다.
나는 ~(이)라고 생각한다.

[본론1: 양보 3~4줄]
물론 ~(이)라는 점도 있다.
~라는 측면에서 이해할 수 있다.

[본론2: 주장 6~8줄]
그러나 ~라고 본다.
첫째, ~
둘째, ~
또한 ~

[결론 2~3줄]
결론적으로 ~해야 한다.
~(으)로 나아가야 할 것이다.
```

### 읽기 (70분 / 100점)

| 구간    | 전략                                          |
| ------- | --------------------------------------------- |
| 1~12번  | 빠르게 (1문제 = 1분)                          |
| 13~24번 | 안정적 (1문제 = 1분)                          |
| 25~34번 | 신중 (순서/삽입/빈칸 → 접속어+지시어 확인)    |
| 35~50번 | 최고 집중 (장문 → 문제+선지 먼저 읽고 → 지문) |

**읽기 핵심 규칙:**

1. **극단적 표현 = 오답 가능성 높음** (모든/항상/절대/유일한)
2. **"그러나/하지만" 뒤 = 핵심 주장**
3. **순서배열**: 일반론 → 구체 → 결론 순서
4. **문장삽입**: <보기> 첫 단어(접속어/지시어) 분석 먼저

## 六、고득점 비결 정리

### 듣기 고득점

| 비결            | 설명                                             |
| --------------- | ------------------------------------------------ |
| 선지 선독       | 음성 나오기 전에 선지 키워드 확인                |
| 메모 습관       | 숫자, 날짜, 이름 등 즉시 메모                    |
| 소거법          | 확실한 오답 먼저 제거                            |
| 2번째 기회 활용 | 대화는 2번 들려줌 → 1차에서 키워드, 2차에서 확인 |

### 읽기 고득점

| 비결           | 설명                                        |
| -------------- | ------------------------------------------- |
| 역순 풀이      | 35~50번 먼저 (배점 높음) → 1번으로 돌아가기 |
| 극단 표현 경계 | "모든/항상/절대" → 높은 확률로 오답         |
| 접속어 활용    | 그러나 뒤 = 핵심, 따라서 뒤 = 결론          |
| 시간 분배      | 1~24번: 25분, 25~34번: 16분, 35~50번: 29분  |

### 쓰기 고득점

| 비결      | 설명                                        |
| --------- | ------------------------------------------- |
| 구조 먼저 | 바로 쓰지 말고 구조를 잡고 시작             |
| 고급 표현 | -ㄹ뿐만 아니라, -는 반면, -(으)ㄴ/는 셈이다 |
| 양보+주장 | "물론 ~지만, 그러나 ~라고 생각한다"         |
| 분량 확인 | 자주 글자 수 확인 (원고지 칸 수로)          |

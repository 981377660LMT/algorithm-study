# 易错题集与解析

> 中国学生 TOPIK II 高频错题分析：按题型整理典型错误 + 正确思路

---

## 一、听力易错题

---

### ❌ 错误类型 1：이어질 말 题混淆"原因"和"回应"

**典型错题：**

여자: 주말에 같이 등산 갈래요?
남자: 가고 싶은데 약속이 있어서요.
여자: ******\_\_\_\_******

| 选项 | 内容                   |
| ---- | ---------------------- |
| ①    | 그럼 다음에 같이 가요. |
| ②    | 등산이 좋겠네요.       |
| ③    | 약속이 없으세요?       |
| ④    | 등산을 싫어하시는군요. |

**常见错选**: ③ (追问原因)
**正确答案**: ① (接受拒绝后的自然回应)

**纠正思路**: 이어질 말 题要选"自然延续"的回应，对方已经说了原因（约会），不需要再追问。

---

### ❌ 错误类型 2：내용 일치 题中的"数字陷阱"

**典型错题**：

> 이 행사는 3월 15일부터 4월 15일까지 진행됩니다. 성인 입장료는 10,000원이며, 어린이는 5,000원입니다.

| 选项 | 内容                       |
| ---- | -------------------------- |
| ①    | 행사 기간은 약 2주이다.    |
| ②    | 어린이 입장료는 무료이다.  |
| ③    | 행사 기간은 약 1개월이다.  |
| ④    | 성인 입장료는 5,000원이다. |

**常见错选**: ① (2주로 혼동)
**正确答案**: ③ (3/15~4/15 = 약 1개월)

**纠正思路**: 날짜를 정확히 계산한다. "2주"와 "1개월"을 혼동하지 않도록 주의.

---

### ❌ 错误类型 3：중심 생각 题选太具体或太笼统

**典型错题**：

> 요즘 택배 포장 쓰레기가 큰 문제입니다. 이를 줄이기 위해 기업들이 재활용 가능한 포장재를 사용해야 합니다.

| 选项 | 内容                                  |
| ---- | ------------------------------------- |
| ①    | 택배 포장 방법을 소개하고 있다.       |
| ②    | 재활용 가능한 포장재 사용이 필요하다. |
| ③    | 환경 보호가 중요하다.                 |
| ④    | 택배 서비스를 중단해야 한다.          |

**常见错选**: ③ (너무 넓은 범위)
**正确答案**: ② (구체적 주장과 일치)

**纠正思路**: 중심 생각 = 글쓴이의 구체적 주장. 너무 넓거나 너무 좁은 선지 제거.

---

### ❌ 错误类型 4：화제 题混淆"话题"和"细节"

**典型错题**：

> 한국의 전통 발효 음식은 건강에 좋은 것으로 알려져 있다. 김치에 포함된 유산균은 장 건강에 도움을 주며, 된장은 항암 효과가 있다는 연구 결과도 있다.

| 选项 | 내容                            |
| ---- | ------------------------------- |
| ①    | 김치의 유산균                   |
| ②    | 된장의 항암 효과                |
| ③    | 한국 전통 발효 음식의 건강 효능 |
| ④    | 한국 음식의 역사                |

**常见错选**: ① 或 ② (세부 사항)
**正确答案**: ③ (전체 화제)

**纠正思路**: 화제 = 글 전체를 아우르는 주제. 김치/된장은 예시일 뿐.

---

### ❌ 错误类型 5：대담/토론 题混淆双方立场

**典型错题**：

> 남자: AI가 교육에 활용되면 맞춤형 학습이 가능합니다.
> 여자: 하지만 AI에 과도하게 의존하면 학생의 비판적 사고력이 약해질 수 있습니다.

**问题**: 여자의 입장은?
| 선지 | 내용 |
|------|------|
| ① | AI 교육 도입에 적극 찬성한다. |
| ② | AI 도입에 대해 우려를 표명하고 있다. |

**常见错选**: ① (남자의 입장과 혼동)
**正确答案**: ②

**纠正思路**: "하지만" 다음이 여자의 진짜 입장. 화자별 키워드를 메모.

## 二、阅读易错题

---

### ❌ 错误类型 1：빈칸 题 — 문법 혼동

**핵심 혼동 쌍:**

| 쌍                                     | 혼동 포인트 | 구별법                                          |
| -------------------------------------- | ----------- | ----------------------------------------------- |
| -(으)니까 vs -아/어서                  | 둘 다 이유  | -(으)니까: 뒤에 명령/제안 가능 / -아/어서: 불가 |
| -(으)ㄹ 뿐만 아니라 vs -(으)ㄹ 뿐이다  | "뿐" 공통   | 뿐만 아니라=추가 / 뿐이다=한정                  |
| -는 반면(에) vs -지만                  | 대조        | 반면: 격식체 + 사실 대비 / 지만: 모든 상황      |
| -(으)ㄴ/는 셈이다 vs -(으)ㄴ/는 편이다 | 정도 표현   | 셈이다=결과적으로 보면 / 편이다=비교적          |

**典型错题**：

비가 ( ) 우산을 가져가세요.
① 오니까 ② 와서

**正确答案**: ① (뒤가 명령문이므로 -니까)

---

### ❌ 错误类型 2：유의어 题 — 한자어 뜻 오해

**빈출 오답 패턴:**

| 출제 표현       | 함정 선지 | 정답                | 해설                        |
| --------------- | --------- | ------------------- | --------------------------- |
| 일축하다        | 수용하다  | 거부하다            | 一蹴=단번에 거절            |
| 편중되다        | 고르다    | 치우치다            | 偏重=한쪽으로 쏠림          |
| 심혈을 기울이다 | 포기하다  | 온 힘을 쏟다        | 心血=온 정성                |
| 난관에 봉착하다 | 해결되다  | 어려움에 부딪히다   | 難關逢着=어려운 고비를 만남 |
| 자초하다        | 해결하다  | 스스로 불러일으키다 | 自招=스스로 초래            |

---

### ❌ 错误类型 3：순서 배열 题 — 접속어 판독 오류

**핵심 접속어 + 위치 규칙:**

| 접속어           | 의미      | 위치 규칙                |
| ---------------- | --------- | ------------------------ |
| 그런데/하지만    | 전환/대조 | 앞에 A, 뒤에 반대 내용 B |
| 따라서/그러므로  | 결론      | 마지막 또는 뒤쪽         |
| 예를 들어/특히   | 구체화    | 일반론 뒤                |
| 이는/이것은      | 지시 설명 | 지시 대상 바로 뒤        |
| 이처럼/이와 같이 | 요약      | 나열/예시 뒤             |

**错误习惯**: "따라서"가 있으면 무조건 마지막에 놓음 → 실제로는 "따라서"문장 뒤에 추가 내용이 올 수 있음.

---

### ❌ 错误类型 4：문장 삽입 题 — <보기> 분석 부족

**3단계 풀이 체크리스트:**

- [ ] <보기>의 첫 단어 확인 (접속어? 지시어?)
- [ ] <보기>가 이어받는 내용이 앞 문장에 있는가?
- [ ] <보기> 뒤의 문장이 자연스럽게 연결되는가?

**典型错题**：

**<보기>**: 그러나 이러한 노력만으로는 충분하지 않다.

문단:
정부는 (①) 미세먼지를 줄이기 위해 노력하고 있다. (②) 경유차 운행 제한, 공장 배출 규제 등이 시행되었다. (③) 시민 개개인의 참여와 의식 변화가 함께 이루어져야 한다. (④)

**常见错选**: ① (보기의 "이러한 노력"을 참조할 내용이 아직 없음)
**正确答案**: ② ("노력" = 경유차 제한, 공장 규제 → 보기 삽입 → 시민 참여 필요)

---

### ❌ 错误类型 5：빈칸 추론 题 — 글의 요지 파악 실패

**典型错题**：

> 사람들은 자신에게 이익이 되는 정보만 기억하고, 불리한 정보는 무시하는 경향이 있다. 이러한 '확증 편향'을 극복하려면 의식적으로 ( ).

| 선지 | 내용                         |
| ---- | ---------------------------- |
| ①    | 자신의 의견을 고집해야 한다  |
| ②    | 반대 의견을 찾아봐야 한다    |
| ③    | 정보를 무시해야 한다         |
| ④    | 다른 사람의 말을 따라야 한다 |

**常见错选**: ④ (시키는 대로 하라는 뜻으로 오해)
**正确答案**: ② (확증 편향 극복 = 반대 의견 탐색)

---

### ❌ 错误类型 6：장문 독해 — "극단적 표현" 함정

**자주 나오는 함정 표현 (높은 확률로 오답):**

| 극단적 표현 | 한국어           | 왜 오답인가            |
| ----------- | ---------------- | ---------------------- |
| 모든        | all              | 예외가 있으므로 오답   |
| 항상        | always           | 100%는 거의 없음       |
| 절대(로)    | never/absolutely | 극단적 단정            |
| 반드시      | must             | 예외 불허              |
| 유일한      | only             | 다른 것도 있을 수 있음 |
| 완전히      | completely       | 부분적인 경우 많음     |
| 전혀 ~없다  | not at all       | 일부는 있을 수 있음    |

**실전 규칙**: 선지에 위 표현이 있으면 **먼저 의심**. 지문에서 해당 표현이 정확히 일치하는지 확인 후 판단.

## 三、写作易错题

---

### ❌ 错误类型 1：Q51-52 경어체 불일치

**错误示例:**

> ❌ 신청서를 작성해 주세요. (비격식체)
> ✅ 신청서를 작성하여 제출해 주시기 바랍니다. (격식체)

**规则**: Q51-52는 항상 **격식체 (-ㅂ/습니다, -시기 바랍니다)**

---

### ❌ 错误类型 2：Q53 자료 해석 — 수치 오독

**错误示例:**

> ❌ 2015년 30%에서 2025년 50%로 20%p 증가하였다.
> (실제 자료: 30% → 45%)

**규칙**: 그래프/표의 숫자를 정확히 읽고 **증감률 vs 증감폭(p)**을 구분.

| 표현             | 뜻                 |
| ---------------- | ------------------ |
| ~% 증가하다      | 비율 자체가 변함   |
| ~%p 증가하다     | 백분율 포인트 차이 |
| 두 배로 늘다     | 2배 증가           |
| 약 절반으로 줄다 | 50% 감소           |

---

### ❌ 错误类型 3：Q54 논술 — 구조 부재

**错误 패턴**: 서론-본론-결론 없이 생각나는 대로 씀

**필수 구조:**

```
[서론] 주제 소개 + 자기 입장 밝히기 (~라고 생각한다)
[본론1] 양보: 반대 입장의 타당성 인정 (물론 ~이라는 점도 있다)
[본론2] 주장: 자기 입장의 근거 2~3개 (그러나 / 또한 / 뿐만 아니라)
[결론] 요약 + 제안/전망 (결론적으로 / 따라서)
```

---

### ❌ 错误类型 4：Q54 감점 요인 체크리스트

| 감점 항목          | 빈도  | 예방법                             |
| ------------------ | ----- | ---------------------------------- |
| 반말 사용 (해요체) | ★★★★★ | -ㅂ/습니다체 통일                  |
| 맞춤법 오류        | ★★★★  | 되/돼, 왠지/웬지, -ㄹ까/-ㄹ가 주의 |
| 주어-서술어 불일치 | ★★★   | 긴 문장 피하기                     |
| 분량 부족 (<500자) | ★★★   | 논거를 최소 2개 이상               |
| 접속어 부재        | ★★★   | 그러나/또한/따라서 필수 사용       |
| 서론-결론 누락     | ★★    | 위 구조 암기                       |

## 四、종합 오답 정리 노트 템플릿

실전에서 틀린 문제를 다음 형식으로 정리하면 복습 효과가 극대화됩니다.

```
[날짜] 2026.04.08
[과목/번호] 읽기 38번
[문형] 장문 독해 (내용 일치)
[내 답] ②
[정답] ④
[틀린 이유] 극단적 표현 "모든"을 확인하지 않았음
[핵심 교훈] 선지에 극단적 표현이 있으면 먼저 의심할 것
[관련 표현] 모든/항상/절대/유일한 = 함정 가능성 높음
```

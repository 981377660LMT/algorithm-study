# 听力实战模拟 — 基础篇 (1~20题级别)

> 模拟 TOPIK II 前20题的题型和难度
> 先读题 → 脑中听（想象音频） → 选答案 → 对答案

---

## 第1~2题：选择合适回答

### 第1题

**音频文本：**

```
여: 주말에 같이 등산 가지 않을래요?
```

选项：

- ① 네, 주말에 등산을 갔어요.
- ② 좋아요. 어디 산으로 갈까요?
- ③ 등산을 좋아하지 않아요.
- ④ 주말에 비가 왔어요.

**答案：②** — 回答邀请 + 追问细节

---

### 第2题

**音频文本：**

```
남: 혹시 이 근처에 약국이 어디 있는지 아세요?
```

选项：

- ① 네, 약을 먹었어요.
- ② 저기 편의점 옆에 있어요.
- ③ 약국에서 일하고 있어요.
- ④ 아까 약국에 갔어요.

**答案：②** — 回答地点

## 第3~4题：选择对话场所

### 第3题

**音频文本：**

```
남: 오늘 어떻게 해 드릴까요?
여: 앞머리만 좀 다듬어 주시고요, 뒷머리는 좀 짧게 잘라 주세요.
남: 파마는 안 하세요?
여: 아니요, 커트만 해 주세요.
```

**这段对话发生在哪里？**

选项：

- ① 병원
- ② 미용실
- ③ 옷가게
- ④ 세탁소

**答案：②** — 关键词: 앞머리, 다듬다, 커트, 파마

---

### 第4题

**音频文本：**

```
여: 이 소포 미국으로 보내려고 하는데요.
남: 무게가 삼 킬로그램이네요. 일반이면 만 원, 등기면 만 오천 원입니다.
여: 등기로 부탁드려요. 얼마나 걸리나요?
남: 일주일에서 열흘 정도요.
```

**这段对话发生在哪里？**

选项：

- ① 택배 회사
- ② 우체국
- ③ 공항
- ④ 은행

**答案：②** — 关键词: 소포, 등기, 일반/등기 요금

## 第5~6题：选择行为

### 第5题

**音频文本：**

```
여: 오빠, 커피 가져왔어.
남: 고마워. 근데 여기 회의실 좀 덥지 않아?
여: 그러네. 에어컨을 좀 틀까?
남: 아, 리모컨이 어디 있더라? 내가 찾아볼게.
```

**남자가 이어서 할 행동은?**

选项：

- ① 커피를 마신다
- ② 에어컨 리모컨을 찾는다
- ③ 회의실을 나간다
- ④ 창문을 연다

**答案：②** — 最后一句：내가 찾아볼게

---

### 第6题

**音频文本：**

```
남: 혹시 이번 주 금요일에 회식 참석하실 수 있으세요?
여: 금요일은 출장이라서요. 다른 날은 안 되나요?
남: 그러면 제가 다른 팀원들과 일정 조율해 볼게요.
```

**남자가 이어서 할 행동은?**

选项：

- ① 회식 장소를 예약한다
- ② 출장 일정을 변경한다
- ③ 다른 팀원들에게 일정을 확인한다
- ④ 금요일 회식을 취소한다

**答案：③** — 다른 팀원들과 일정 조율

## 第7~8题：选择合适回答 (中级)

### 第7题

**音频文本：**

```
남: 보고서를 다 작성했는데 한번 확인 좀 해 주시겠어요?
여: 지금은 좀 바쁜데 오후에 봐도 될까요?
남: ________________________
```

选项：

- ① 네, 그럼 오후 세 시쯤 다시 올게요.
- ② 네, 오늘 오전까지 작성하겠습니다.
- ③ 네, 보고서를 내일 제출하겠습니다.
- ④ 네, 지금 바로 확인하겠습니다.

**答案：①** — 대응: 오후에 봐도 될까요 → 좋아요, 오후에 다시

---

### 第8题

**音频文本：**

```
여: 이 제품 지금 할인 중인가요?
남: 네, 이번 주까지 이십 퍼센트 할인입니다.
여: ________________________
```

选项：

- ① 그러면 할인 좀 해 주세요.
- ② 그럼 이거 하나 살게요.
- ③ 할인은 언제부터 시작해요?
- ④ 저번 주에 샀으면 좋았을 텐데.

**答案：②** — 이미 할인 중 → 사겠다

## 第9~12题：目的/主题

### 第9题

**音频文本：**

```
[전화 벨 소리]
남: 여보세요, 시립 도서관입니다.
여: 안녕하세요, 지난주에 빌린 책 반납 기한을 연장하고 싶어서
    전화했는데요.
남: 네, 이름이 어떻게 되세요?
여: 김수연이요. 대출 도서가 두 권인데 일주일만 더 연장 가능할까요?
남: 확인해 보겠습니다. 다른 예약자가 없으면 연장 가능합니다.
```

**여자가 전화한 이유는?**

选项：

- ① 책을 예약하려고
- ② 반납 기한을 연장하려고
- ③ 도서관 운영 시간을 문의하려고
- ④ 분실 도서를 신고하려고

**答案：②**

---

### 第11题

**音频文本：**

```
남: 요즘 커피숍이 정말 많아졌죠?
여: 맞아요. 우리 동네에만 다섯 개가 새로 생겼어요.
남: 경쟁이 심해서 자영업자들이 힘들다고 하더라고요.
여: 그래도 독특한 메뉴나 분위기로 차별화하는 곳은 잘 되는 것 같아요.
```

**무엇에 대해 이야기하고 있습니까?**

选项：

- ① 커피의 종류
- ② 커피숍 창업 방법
- ③ 커피숍의 증가와 경쟁
- ④ 커피 가격 인상

**答案：③**

## 第13~16题：内容一致

### 第13题

**音频文本：**

```
여: 안내 말씀드리겠습니다. 오는 토요일 오후 두 시에
    주민센터 삼 층 강당에서 무료 건강 검진을 실시합니다.
    대상은 육십 세 이상 주민이며,
    사전 예약 없이 신분증만 지참하시면 됩니다.
    검진 항목은 혈압, 혈당, 시력 검사입니다.
```

**들은 내용과 일치하는 것을 고르세요.**

选项：

- ① 건강 검진은 유료이다.
- ② 사전에 예약을 해야 한다.
- ③ 육십 세 이상 주민이 대상이다.
- ④ 검진은 주민센터 이 층에서 한다.

**答案：③**

- ① ✗ 무료
- ② ✗ 사전 예약 없이
- ③ ✓
- ④ ✗ 삼 층 (不是 이 층 — 数字偷换)

### 第15题

**音频文本：**

```
남: 새로 개봉한 영화 '바다의 노래'가 화제입니다.
    개봉 첫 주에 관객 백만 명을 돌파하며 흥행 기록을 세웠습니다.
    이 영화는 부산을 배경으로 한 가족 드라마로,
    김 감독의 세 번째 작품입니다.
    특히 신인 배우 이수진 씨의 연기가 호평을 받고 있습니다.
```

**들은 내용과 일치하는 것을 고르세요.**

选项：

- ① 이 영화는 서울을 배경으로 한다.
- ② 개봉 첫 주에 백만 관객을 넘었다.
- ③ 이 영화는 김 감독의 두 번째 작품이다.
- ④ 베테랑 배우의 연기가 호평을 받았다.

**答案：②**

- ① ✗ 부산 (不是 서울)
- ② ✓ 백만 명 돌파
- ③ ✗ 세 번째 (不是 두 번째)
- ④ ✗ 신인 배우 (不是 베테랑)

## 第17~20题：态度/想法

### 第17题

**音频文本：**

```
여: 이번에 회사에서 유연 근무제를 도입한다고 하던데요.
남: 네, 들었어요. 취지는 좋다고 생각해요.
    그런데 우리 부서는 업무 특성상 다 같이 모여야 할 때가 많잖아요.
    그래서 저는 솔직히 좀 걱정되는 부분이 있어요.
```

**남자의 태도로 알맞은 것은?**

选项：

- ① 유연 근무제에 적극 찬성하고 있다.
- ② 유연 근무제의 도입을 우려하고 있다.
- ③ 유연 근무제에 관심이 없다.
- ④ 유연 근무제를 즉시 폐지해야 한다고 본다.

**答案：②** — "취지는 좋지만...걱정되는 부분" = 우려

---

### 第19题

**音频文本：**

```
남: 온라인 쇼핑몰 리뷰를 얼마나 믿으세요?
여: 솔직히 말하면 예전에는 많이 참고했는데요,
    요즘은 광고성 리뷰가 너무 많아서 믿기 어려워졌어요.
    실제 구매자 리뷰만 따로 볼 수 있으면 좋겠어요.
```

**여자의 태도로 알맞은 것은?**

选项：

- ① 온라인 리뷰를 전적으로 신뢰하고 있다.
- ② 온라인 리뷰 시스템의 개선이 필요하다고 본다.
- ③ 온라인 쇼핑 자체에 반대하고 있다.
- ④ 리뷰를 참고할 필요가 없다고 생각한다.

**答案：②** — 广告性리뷰 문제 지적 + 실제 구매자 리뷰만 → 개선 필요

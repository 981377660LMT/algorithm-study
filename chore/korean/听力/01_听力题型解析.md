## 题型一：选择合适回答 (1~2题, 7~8题)

### 出题模式

播放一句话或问题，从选项中选择最合适的回答。

### 高频问句模式

| 问句               | 含义         | 正确回答方向   |
| ------------------ | ------------ | -------------- |
| ~(으)ㄹ까요?       | 要不要...?   | 同意/拒绝      |
| ~아/어 주시겠어요? | 能帮我...吗? | 好的/不方便    |
| ~(으)ㄹ 수 있어요? | 能...吗?     | 可以/不可以    |
| ~는 게 어때요?     | ...怎么样?   | 好主意/不太... |
| ~(으)ㄴ 적 있어요? | 有过...吗?   | 有/没有+补充   |
| 왜 ~아/어요?       | 为什么?      | 原因句         |
| 언제 ~아/어요?     | 什么时候?    | 时间           |
| 어디에서 ~아/어요? | 在哪里?      | 地点           |
| 어떻게 ~아/어요?   | 怎么...?     | 方式           |

### 解题关键

- **听最后一个词的语调**：升调=疑问，降调=陈述
- **疑问词决定回答类型**：왜→原因，언제→时间，어디→地点
- **注意否定疑问句**：안 갈 거예요? → 回答要对应肯/否

### 常见干扰项

- 词汇重复但意思不对应
- 回答与问句的时态不匹配

## 题型二：选择对话场所 (3~4题)

### 出题模式

播放一段短对话，问："이 대화는 어디에서 하고 있습니까?"

## 题型三：选择合适行为 (5~6题, 29~30题)

### 出题模式

播放对话后问："여자/남자는 이어서 무엇을 할 것 같습니까?" 或 "여자/남자가 할 행동으로 알맞은 것을 고르세요."

### 解题关键

- **只听最后2~3句话**，前面是背景铺垫
- 关注「那我去...」「那先...」等行动指示句：
  - 그러면 제가 ~할게요
  - 먼저 ~해 볼게요
  - 그럼 ~(으)러 가야겠다
  - ~해 주세요 (被请求方会去做)
  - 알겠습니다. ~하겠습니다

### 常见行为动词

| 한국어            | 中文      | 场景 |
| ----------------- | --------- | ---- |
| 전화하다          | 打电话    |      |
| 예약하다          | 预约      |      |
| 알아보다          | 了解/查看 |      |
| 확인하다          | 确认      |      |
| 신청하다          | 申请      |      |
| 접수하다          | 报名/受理 |      |
| 수리하다          | 修理      |      |
| 교환하다          | 换货      |      |
| 환불받다          | 退款      |      |
| 가져오다/가져가다 | 带来/带走 |      |
| 제출하다          | 提交      |      |
| 준비하다          | 准备      |      |

## 题型四：说话目的/主题 (9~12题)

### 出题模式A — 目的

"남자/여자는 왜 전화를 했습니까?" "남자/여자가 이 이야기를 하는 이유는?"

### 出题模式B — 主题

"무엇에 대해 이야기하고 있습니까?" "이 이야기의 주제는?"

### 目的类高频答案

| 답 패턴             | 含义        |
| ------------------- | ----------- |
| ~을/를 부탁하려고   | 为了请求... |
| ~을/를 문의하려고   | 为了询问... |
| ~을/를 확인하려고   | 为了确认... |
| ~을/를 취소하려고   | 为了取消... |
| ~을/를 변경하려고   | 为了变更... |
| ~을/를 알려 주려고  | 为了告知... |
| ~에 대해 사과하려고 | 为了道歉... |
| ~을/를 추천하려고   | 为了推荐... |
| ~을/를 제안하려고   | 为了建议... |

### 解题关键

- 目的题：听第一句话（往往直接说明来意）
- 主题题：听反复出现的关键词

## 题型五：判断内容一致 (13~16题, 23~24题)

### 出题模式

"들은 내용과 일치하는/일치하지 않는 것을 고르세요."

### 解题关键

- **先快速浏览4个选项**（利用题间空隙）
- 听的时候对应每个选项逐一验证
- 注意否定词：안, 못, 없다, 아직, 아니라
- 注意程度词修改：매우→조금, 모든→일부
- 注意时态偷换：했다 vs 할 것이다
- 注意主语偷换：남자 vs 여자

### 常见偷换手法

| 原文                 | 选项偷换               | 方式      |
| -------------------- | ---------------------- | --------- |
| 이번 주에 시작한다   | 지난주에 시작했다      | 时间偷换  |
| 남자가 제안했다      | 여자가 제안했다        | 主语偷换  |
| 대부분 찬성했다      | 모두 찬성했다          | 程度偷换  |
| 아직 결정하지 않았다 | 이미 결정했다          | 否定→肯定 |
| 무료로 제공된다      | 할인 가격으로 제공된다 | 细节偷换  |

## 题型六：说话人态度/想法 (17~20题)

### 出题模式

"남자/여자의 생각/태도로 알맞은 것을 고르세요."

### 态度选项常用词

**肯定态度：**

- 긍정적이다 (肯定的), 찬성하다 (赞成), 동의하다 (同意)
- 만족하다 (满足), 기대하다 (期待), 환영하다 (欢迎)

**否定态度：**

- 부정적이다 (否定的), 반대하다 (反对), 비판하다 (批判)
- 불만이다 (不满), 실망하다 (失望), 우려하다 (担忧)

**中立态度：**

- 객관적이다 (客观的), 중립적이다 (中立的), 신중하다 (慎重的)

### 态度信号词

| 信号                         | 态度倾向             |
| ---------------------------- | -------------------- |
| 좋은 것 같아요 / 괜찮은데요  | 肯定                 |
| 글쎄요... / 좀...            | 否定/犹豫            |
| 저는 그렇게 생각하지 않아요  | 反对                 |
| 맞아요, 저도 그렇게 생각해요 | 同意                 |
| 그것도 일리가 있지만...      | 部分同意后转折       |
| ~면 좋겠는데...              | 期望（可能暗示不满） |
| 진짜요? 그건 좀...           | 怀疑/不满            |

## 题型七：推测前文 (27~28题)

### 出题模式

"남자/여자가 무엇을 이야기하고 있었는지" 或 "앞에 무슨 이야기를 했는지"

### 解题关键

- 听开头的指示代词：그래서(所以), 그런데(但是), 아까 말한(刚才说的)
- 结合对话内容反推：现在在讨论结果→前面讨论的是原因
- 现在在说解决方案→前面说的是问题

## 题型八：长对话/讲座配对题 (31~50题)

### 出题模式

每段音频配2道题：

- 第1题：주제 (主题) 或 중심 생각 (中心思想) 或 태도 (态度)
- 第2题：내용 일치 (内容一致)

### 31~40 题策略 (中等难度)

- 场景：新闻采访、广播节目、讲座片段
- **第1题**听开头定主题，**第2题**听细节
- 可以在听到一半时先选第1题

### 41~50 题策略 (高难度)

- 场景：学术讲座、专家讨论、深度报道
- 语速快、信息密度高、只播放1次
- **笔记关键**：原因、数字、对比、结论
- 如果听不懂全部，抓「总结句」（결국, 따라서, 중요한 것은）

### 高分技巧总结

1. **预读选项**：每段音频前有提示音，利用这几秒读选项
2. **听关键词不听全文**：不需要每个词都懂
3. **前半部分出主题，后半部分出细节**
4. **转折后面是重点**：그러나, 하지만, 그런데 后面才是答案
5. **最后一句话决定态度**：说话人的结论通常在最后

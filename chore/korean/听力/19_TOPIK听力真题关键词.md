## 三、高频出题表达 (真题级别)

### 新闻/报道中的固定表达

| 表达                          | 中文          | 频率  |
| ----------------------------- | ------------- | ----- |
| ~에 따르면                    | 根据...       | ★★★★★ |
| ~(으)ㄴ 것으로 나타났다       | 显示为...     | ★★★★★ |
| ~(으)ㄴ 것으로 밝혀졌다       | 被查明为...   | ★★★★  |
| ~(으)ㄹ 것으로 전망된다       | 预计...       | ★★★★  |
| 전년 대비                     | 与去年相比    | ★★★★  |
| ~(으)ㄹ 계획/예정이다         | 计划/预定...  | ★★★★  |
| 전문가들은 ~(이)라고 분석했다 | 专家分析称... | ★★★★  |
| ~(이)라는 지적이 있다         | 有...的指出   | ★★★   |
| ~(이)라는 우려가 나온다       | 有...的担忧   | ★★★   |

### 讲座中的固定表达

| 表达                                | 中文          | 频率  |
| ----------------------------------- | ------------- | ----- |
| 오늘은 ~에 대해 이야기해 보겠습니다 | 今天来谈谈... | ★★★★★ |
| 일반적으로 ~(이)라고 생각하지만     | 一般认为...但 | ★★★★  |
| 이는 ~기 때문이다                   | 这是因为...   | ★★★★  |
| 예를 들어                           | 例如          | ★★★★  |
| 연구에 따르면                       | 根据研究      | ★★★★  |
| 이처럼 / 결론적으로                 | 如此/总之     | ★★★★  |
| ~할 필요가 있다                     | 需要...       | ★★★   |

### 讨论中的固定表达

| 表达                  | 中文         | 频率  |
| --------------------- | ------------ | ----- |
| 저는 ~(이)라고 봅니다 | 我认为...    | ★★★★★ |
| 물론 ~지만            | 当然...但    | ★★★★★ |
| 그 점은 동의하지만    | 那点同意但   | ★★★★  |
| 현실적으로            | 现实来看     | ★★★   |
| ~(으)ㄹ 수도 있겠지만 | 虽然也有可能 | ★★★   |

## 四、态度题高频选项词

### 正确答案中最常出现的态度描述

| 描述                   | 中文         | 频率  |
| ---------------------- | ------------ | ----- |
| 우려하고 있다          | 表示担忧     | ★★★★★ |
| 필요하다고 보고 있다   | 认为有必要   | ★★★★  |
| 개선이 필요하다고 본다 | 认为需要改善 | ★★★★  |
| 조건부로 찬성한다      | 有条件赞成   | ★★★   |
| 신중한 입장이다        | 慎重立场     | ★★★   |
| 긍정적으로 평가한다    | 积极评价     | ★★★   |
| 비판적 시각을 보인다   | 批判视角     | ★★    |

### 干扰选项中常用的"绝对化"表达（多为错误答案）

| 表达               | 中文       | 为什么常是错的   |
| ------------------ | ---------- | ---------------- |
| 전적으로 찬성      | 完全赞成   | 太绝对           |
| 전면 반대          | 全面反对   | 太绝对           |
| 관심이 없다        | 不关心     | 既然在讨论就不是 |
| 아무 문제 없다     | 没问题     | 太绝对           |
| 즉시 폐지해야 한다 | 应立即废除 | 太激烈           |

## 五、答题节奏时间表

| 题号  | 类型      | 播放 | 建议用时           |
| ----- | --------- | ---- | ------------------ |
| 1~2   | 选回答    | 2次  | 各30秒             |
| 3~4   | 选场所    | 2次  | 各30秒             |
| 5~6   | 选行为    | 2次  | 各30秒             |
| 7~8   | 选回答    | 2次  | 各30秒             |
| 9~12  | 目的/主题 | 2次  | 各40秒             |
| 13~16 | 内容一致  | 2次  | 各40秒             |
| 17~20 | 态度      | 1次  | 各50秒 ★注意       |
| 21~50 | 长文配对  | 1次  | 每对约90秒 ★高难度 |

### 关键提醒

- **17题开始只播放1次** — 从这里开始必须记笔记
- **31题开始都是2题配对** — 先记主题再记细节
- **最后10题是全卷最难** — 如果前面做得好，这里可以放松心态

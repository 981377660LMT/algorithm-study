## 讲座1：心理学 — 决策心理

```
오늘은 의사 결정 과정에서 나타나는 심리적 편향에 대해 이야기해
보겠습니다. 우리는 합리적으로 판단한다고 생각하지만,
실제로는 많은 경우 감정이나 선입견에 의해 결정을 내립니다.

대표적인 예가 '확증 편향'입니다. 확증 편향이란 자신의 기존 믿음을
확인해 주는 정보만 선택적으로 받아들이고, 반대되는 정보는
무시하는 경향을 말합니다.

예를 들어, 어떤 사람이 '유기농 식품이 건강에 좋다'는 믿음을
가지고 있다면, 유기농 식품의 장점에 대한 기사는 열심히 읽지만
과학적으로 차이가 없다는 연구 결과는 무시하게 되는 것입니다.

이러한 확증 편향은 개인의 판단뿐만 아니라
사회적 갈등을 심화시키는 원인이 되기도 합니다.
따라서 중요한 결정을 내릴 때는 의식적으로
반대 의견을 찾아보는 습관을 기르는 것이 필요합니다.
```

**구조 분석:**

1. 도입: 주제 제시 (의사 결정의 심리적 편향)
2. 개념: 확증 편향 정의
3. 예시: 유기농 식품 사례
4. 영향: 사회적 갈등 심화
5. 결론: 반대 의견을 찾는 습관 필요

**출제 예상:**

- 중심 내용: 사람들은 확증 편향에 의해 비합리적 판단을 한다
- 일치: 기존 믿음과 다른 정보를 무시하는 경향이 있다

## 讲座2：经济学 — 行为经济学

```
많은 사람들이 물건을 살 때 가격이 합리적인지
냉철하게 판단한다고 생각합니다.
하지만 행동 경제학에서는 소비자가 심리적 요인에 의해
비합리적인 소비를 한다고 봅니다.

그 대표적인 현상이 '앵커링 효과'입니다.
앵커링 효과란 처음 접한 숫자나 정보가 이후의 판단에
기준점이 되는 현상을 말합니다.

예를 들어, 원래 십만 원짜리 옷에 '정가 이십만 원,
할인가 십만 원'이라고 표시하면 소비자는
이십만 원이라는 앵커에 영향을 받아
십만 원이 매우 싸다고 느끼게 됩니다.
실제로는 같은 가격인데도 말입니다.

이처럼 기업들은 앵커링 효과를 마케팅 전략으로 적극 활용하고 있으며,
소비자는 이를 인식하고 보다 신중한 소비를 할 필요가 있습니다.
```

**구조 분석:**

1. 통념 제시: 합리적 소비라고 생각
2. 반론: 행동 경제학 — 비합리적 소비
3. 개념: 앵커링 효과 정의
4. 예시: 할인 가격 표시
5. 결론: 기업 활용 + 소비자 주의 필요

## 讲座3：社会学 — 社交媒体的影响

```
이 시간에는 소셜 미디어가 현대인의 대인 관계에 미치는
영향에 대해 살펴보겠습니다.

소셜 미디어를 통해 우리는 더 많은 사람과 연결되어 있지만,
역설적으로 외로움을 느끼는 사람이 증가하고 있습니다.

한 연구에 따르면, 소셜 미디어를 하루 세 시간 이상
사용하는 사람은 한 시간 미만 사용하는 사람에 비해
우울감을 느낄 확률이 두 배 이상 높았습니다.

이는 소셜 미디어에서 타인과 자신을 끊임없이 비교하면서
상대적 박탈감을 느끼기 때문이라고 분석됩니다.
특히 이미지 중심의 플랫폼일수록 이러한 현상이
두드러지는 것으로 나타났습니다.

결국 소셜 미디어는 도구에 불과하며,
이를 어떻게 사용하느냐가 중요합니다.
의미 있는 오프라인 관계를 유지하면서
소셜 미디어를 보완적으로 활용하는 자세가 필요합니다.
```

**구조 분석:**

1. 주제: SNS와 대인 관계
2. 역설: 더 연결되지만 더 외로움
3. 연구 결과: 3시간 이상 사용 → 우울감 2배↑
4. 원인: 타인과의 비교, 상대적 박탈감
5. 결론: 오프라인 관계 유지 + SNS 보완적 활용

## 讲座4：环境科学 — 微塑料

```
오늘 강연의 주제는 '미세 플라스틱의 위험성'입니다.

미세 플라스틱은 크기가 오 밀리미터 이하인
작은 플라스틱 조각을 말합니다.
이것은 세탁할 때 합성 섬유에서 떨어져 나오거나,
플라스틱 쓰레기가 자연에서 분해되면서 발생합니다.

문제는 이 미세 플라스틱이 바다로 유입되어
해양 생물의 체내에 축적되고,
이것이 먹이 사슬을 통해 결국 인간에게까지
돌아온다는 점입니다.

세계 보건 기구의 조사에 따르면 한 사람이 일 주일에
약 오 그램, 신용 카드 한 장 분량의 미세 플라스틱을
섭취하고 있는 것으로 추정됩니다.

이러한 미세 플라스틱이 인체에 미치는 장기적인 영향에 대해서는
아직 연구가 진행 중이지만,
호르몬 교란이나 암 발생 위험을 높일 수 있다는
연구 결과가 속속 나오고 있습니다.
```

**출제 예상:**

- 중심 내용: 미세 플라스틱의 위험성과 인체 영향
- 일치: 세탁 과정에서도 미세 플라스틱이 발생한다
- 불일치 함정: "인체 영향이 이미 밝혀졌다" → ✗ (아직 연구 중)

## 讲座5：教育学 — 合作学习

```
지금부터 효과적인 학습 방법에 대해 말씀드리겠습니다.

많은 학생이 혼자 공부하는 것을 선호합니다.
하지만 교육학 연구에서는 협동 학습이
혼자 공부하는 것보다 학습 효과가 높다는 결과가
일관되게 나타나고 있습니다.

그 이유는 크게 세 가지입니다.
첫째, 다른 사람에게 설명하는 과정에서
자신의 이해도가 높아집니다.
둘째, 서로 다른 관점을 접하면서
사고의 폭이 넓어집니다.
셋째, 사회적 상호 작용이 동기 부여를 강화합니다.

물론 협동 학습이 효과적이려면 몇 가지 조건이 필요합니다.
구성원 모두가 적극적으로 참여해야 하며,
역할 분담이 명확해야 합니다.
또한 단순히 모여서 공부하는 것이 아니라
토론과 질문이 활발히 이루어져야 합니다.

결론적으로, 제대로 된 협동 학습은
개인 학습의 한계를 보완하는 강력한 학습 도구입니다.
```

**구조 분석:**

1. 도입: 효과적 학습 방법
2. 주장: 협동 학습 > 개인 학습
3. 이유 3가지: 설명→이해↑, 다양한 관점, 동기 부여
4. 조건: 적극 참여, 역할 분담, 토론·질문
5. 결론: 협동 학습 = 강력한 도구

## 讲座6：艺术/美学

```
오늘은 현대 미술에서 '관객 참여형 작품'이
왜 늘어나고 있는지에 대해 이야기해 보겠습니다.

전통적인 미술에서 관객은 작품을 보는 수동적 존재였습니다.
그러나 현대 미술에서는 관객이 작품의 일부가 되어
직접 참여하는 형태가 증가하고 있습니다.

이러한 변화의 배경에는 '예술의 민주화'라는 개념이 있습니다.
즉, 예술은 특별한 사람만 감상하는 것이 아니라
누구나 참여하고 경험할 수 있어야 한다는 생각입니다.

관객 참여형 작품은 관객의 행동에 따라 작품이 변화하기 때문에
매번 새로운 경험을 제공합니다.
또한 관객이 단순히 보는 것을 넘어 직접 만지고 움직이면서
더 깊은 감정적 연결을 느끼게 됩니다.

이처럼 현대 미술은 더 이상 작가만의 영역이 아니라,
관객과 함께 완성되는 것으로 변화하고 있습니다.
```

## 讲座听力策略总结

### 讲座固定结构

```
① 오늘은 ~에 대해 이야기해 보겠습니다  → 抓住主题
② 일반적으로/많은 사람들이 ~           → 通常认知（可能会被反驳）
③ 그러나/하지만 실제로는 ~             → 核心论点（★重点）
④ 그 이유는/이는 ~ 때문입니다          → 原因
⑤ 예를 들어/한 연구에 따르면           → 例证/数据
⑥ 따라서/결론적으로/이처럼             → 结论（★重点）
```

### 信号词速查

| 听到这个                | 马上准备记 |
| ----------------------- | ---------- |
| ~에 대해 살펴보겠습니다 | 主题来了   |
| 일반적으로 ~지만        | 要反转了   |
| 그 이유는 크게 세 가지  | 数3个要点  |
| 첫째/둘째/셋째          | 分点论述   |
| 연구에 따르면           | 数据来了   |
| 이처럼/결론적으로       | 总结来了   |
| ~할 필요가 있다         | 建议/结论  |

# 语速适应训练素材

> 从慢速到真题速度的阶梯训练
> 同一篇文本，分3个速度层级练习

---

## 训练原理

```
Level 1 (慢速) → 每个词都听清    → 建立文本-声音对应
Level 2 (正常) → 理解句意           → 建立理解习惯
Level 3 (快速) → 抓关键词答题      → 适应真题速度
```

**如何实操：**

- 用 TTS 工具（Papago, NAVER Clova, Google TTS）把下面的文本转成语音
- 或者自己先用0.75x速度朗读 → 再用正常速度 → 再加速到1.25x
- 没有TTS时，可以自己朗读、录音、回放

---

## 素材1：日常对话 (初级速度)

### Level 1 文本（慢速 — 逐句停顿练习）

```
여: | 이번 주말에 | 뭐 할 거예요? |
남: | 아직 | 계획이 없는데요. |
여: | 그러면 | 같이 | 북한산에 | 등산 가지 않을래요? |
남: | 좋아요. | 몇 시에 | 만날까요? |
여: | 아침 | 아홉 시 | 어때요? |
남: | 좀 이른데... | 열 시는 | 안 될까요? |
여: | 그래요. | 열 시에 | 삼호선 | 구파발역에서 | 만나요. |
남: | 알겠어요. | 준비물이 | 뭐 필요해요? |
여: | 편한 옷하고 | 물, | 그리고 | 간식 정도면 | 돼요. |
```

### Level 2 文本（正常语速 — 连贯朗读）

```
여: 이번 주말에 뭐 할 거예요?
남: 아직 계획이 없는데요.
여: 그러면 같이 북한산에 등산 가지 않을래요?
남: 좋아요. 몇 시에 만날까요?
여: 아침 아홉 시 어때요?
남: 좀 이른데... 열 시는 안 될까요?
여: 그래요. 열 시에 삼호선 구파발역에서 만나요.
남: 알겠어요. 준비물이 뭐 필요해요?
여: 편한 옷하고 물, 그리고 간식 정도면 돼요.
```

### Level 3 文本（快速 — 模拟真题语速，部分连读）

> 朗读时刻意加快，连接处不停顿

## 素材2：广播通知 (中级速度)

### Level 1

```
| 안녕하십니까. |
| 승객 여러분께 | 안내 말씀 | 드리겠습니다. |
| 오늘 | 오후 두 시부터 |
| 이호선 | 시청역에서 | 을지로입구역 | 구간의 |
| 열차 운행이 | 일시 중단됩니다. |
| 원인은 | 신호 장비 | 점검 공사 | 때문이며, |
| 대체 교통으로 | 무료 셔틀버스가 | 운행됩니다. |
| 정상 운행은 | 오후 여섯 시부터 | 재개될 예정입니다. |
| 이용에 | 불편을 드려 | 죄송합니다. |
```

### Level 2

```
안녕하십니까. 승객 여러분께 안내 말씀 드리겠습니다.
오늘 오후 두 시부터 이호선 시청역에서 을지로입구역 구간의
열차 운행이 일시 중단됩니다.
원인은 신호 장비 점검 공사 때문이며,
대체 교통으로 무료 셔틀버스가 운행됩니다.
정상 운행은 오후 여섯 시부터 재개될 예정입니다.
이용에 불편을 드려 죄송합니다.
```

**练习：听完后写出**

- 구간: ****\_\_\_\_****
- 중단 시간: ****\_\_\_\_****
- 원인: ****\_\_\_\_****
- 대체: ****\_\_\_\_****
- 재개: ****\_\_\_\_****

## 素材3：新闻报道 (高级速度)

### Level 1

```
| 통계청이 | 발표한 자료에 | 따르면, |
| 올해 | 삼분기 | 가계 소비 지출이 |
| 전년 동기 대비 | 삼 점 이 퍼센트 | 증가한 것으로 |
| 나타났습니다. |
| 특히 | 외식비와 | 여가 활동비의 | 증가가 |
| 두드러졌는데요, |
| 코로나 이후 | 억눌려 있던 | 소비 심리가 |
| 회복되고 있는 | 것으로 | 분석됩니다. |
| 그러나 | 물가 상승률이 | 여전히 |
| 사 퍼센트대를 | 유지하고 있어, |
| 실질 소비 | 여력은 | 크게 나아지지 | 않았다는 |
| 지적도 있습니다. |
```

### Level 2

```
통계청이 발표한 자료에 따르면, 올해 삼분기 가계 소비 지출이
전년 동기 대비 삼 점 이 퍼센트 증가한 것으로 나타났습니다.
특히 외식비와 여가 활동비의 증가가 두드러졌는데요,
코로나 이후 억눌려 있던 소비 심리가 회복되고 있는 것으로
분석됩니다. 그러나 물가 상승률이 여전히 사 퍼센트대를
유지하고 있어, 실질 소비 여력은 크게 나아지지 않았다는
지적도 있습니다.
```

**速记练习：**

```
题：
据：+___% (3분기)
↑：외식, 여가
因：코로나 후 소비심리↑
BUT：물가 __% → 실질소비 ✗
```

## 素材4：学术讲座 (最高速度)

### Level 1

```
| 오늘날 | 도시화가 | 급속히 | 진행되면서, |
| 도시 열섬 현상이 | 심각한 | 환경 문제로 |
| 부상하고 있습니다. |
| 도시 열섬이란 | 도시 중심부의 기온이 |
| 주변 교외 지역보다 | 이 도에서 삼 도 정도 |
| 높게 나타나는 | 현상을 말합니다. |
| 이는 | 아스팔트와 | 콘크리트가 |
| 태양열을 | 흡수하고 | 방열하기 | 때문이며, |
| 건물과 | 자동차에서 나오는 |
| 인공 열도 | 원인 중 | 하나입니다. |
| 이를 | 완화하기 위해 |
| 도시 녹지 확대, | 옥상 녹화, |
| 쿨루프 도료 | 사용 등의 |
| 대책이 | 제시되고 있으며, |
| 실제로 | 서울시에서는 |
| 녹지 면적을 | 이십 퍼센트 | 확대한 결과 |
| 평균 기온이 | 영 점 오 도 | 낮아진 |
| 사례도 있습니다. |
```

### Level 2

```
오늘날 도시화가 급속히 진행되면서, 도시 열섬 현상이 심각한
환경 문제로 부상하고 있습니다. 도시 열섬이란 도시 중심부의
기온이 주변 교외 지역보다 이 도에서 삼 도 정도 높게 나타나는
현상을 말합니다. 이는 아스팔트와 콘크리트가 태양열을 흡수하고
방열하기 때문이며, 건물과 자동차에서 나오는 인공 열도 원인 중
하나입니다. 이를 완화하기 위해 도시 녹지 확대, 옥상 녹화,
쿨루프 도료 사용 등의 대책이 제시되고 있으며, 실제로 서울시에서는
녹지 면적을 이십 퍼센트 확대한 결과 평균 기온이 영 점 오 도
낮아진 사례도 있습니다.
```

## 素材5：讨论 (最高速度)

### Level 2 (直接练快速)

```
여: 저는 대중교통을 무료로 제공해야 한다고 생각합니다.
자가용 이용을 줄여 교통 체증과 대기 오염을
동시에 해결할 수 있으니까요.

남: 취지는 좋지만 재원 문제가 있습니다.
대중교통 무료화에 필요한 예산이 연간 수조 원에 달하는데,
이 비용을 세금으로 충당하면 결국 국민 부담이 커집니다.

여: 하지만 교통 체증으로 인한 사회적 비용도
연간 수십조 원입니다. 장기적으로 보면
오히려 비용이 절약될 수 있습니다.

남: 그런 관점도 있지만, 무료화보다는
요금 인하나 환승 할인 확대 같은
단계적 접근이 현실적이라고 봅니다.
```

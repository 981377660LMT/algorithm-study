# 日常对话核心句型 — 听力场景句

> 按功能分类，每个句型附韩语原句 + 中文 + 常见回答
> 反复朗读到烂熟，听到就秒懂

---

## A. 问候/寒暄

| 韩语                  | 中文         | 常见回答                     |
| --------------------- | ------------ | ---------------------------- |
| 잘 지내셨어요?        | 过得好吗？   | 네, 잘 지냈어요.             |
| 오랜만이에요.         | 好久不见。   | 네, 정말 오랜만이에요.       |
| 요즘 어떻게 지내세요? | 最近怎么样？ | 바쁘게 지내고 있어요.        |
| 식사하셨어요?         | 吃了吗？     | 네, 방금 먹었어요.           |
| 어디 가세요?          | 去哪儿？     | 잠깐 마트에 좀 다녀오려고요. |

## B. 约定/邀请

| 韩语                        | 中文               | 常见回答                              |
| --------------------------- | ------------------ | ------------------------------------- |
| 이번 주말에 시간 있어요?    | 这周末有空吗？     | 토요일은 좀 힘들고 일요일은 괜찮아요. |
| 같이 점심 먹을까요?         | 一起吃午饭吗？     | 좋아요. 몇 시에 만날까요?             |
| 몇 시가 좋으세요?           | 几点合适？         | 두 시쯤 어때요?                       |
| 어디에서 만날까요?          | 在哪见面？         | 강남역 앞에서 만나요.                 |
| 내일 회의 참석할 수 있어요? | 明天能参加会议吗？ | 네, 참석하겠습니다.                   |
| 혹시 토요일에 시간 되세요?  | 周六有时间吗？     | 토요일은 선약이 있어서요...           |

## C. 请求/拜托

| 韩语                          | 中文                 | 常见回答                           |
| ----------------------------- | -------------------- | ---------------------------------- |
| 좀 도와주시겠어요?            | 能帮我一下吗？       | 네, 뭘 도와드릴까요?               |
| 이것 좀 봐 주세요.            | 请看一下这个。       | 네, 어디 봅시다.                   |
| 부탁 하나만 해도 될까요?      | 能拜托你一件事吗？   | 네, 말씀하세요.                    |
| 잠깐만 기다려 주세요.         | 请稍等。             | 네, 알겠습니다.                    |
| 문 좀 닫아 주시겠어요?        | 能帮我关下门吗？     | 네, 닫을게요.                      |
| 이 서류 좀 복사해 주세요.     | 请帮我复印这份文件。 | 몇 부 복사할까요?                  |
| 내일까지 보내 주실 수 있어요? | 明天之前能发给我吗？ | 네, 오늘 중으로 보내 드리겠습니다. |

## D. 建议/提议

| 韩语                            | 中文                | 常见回答                        |
| ------------------------------- | ------------------- | ------------------------------- |
| ~(으)면 어떨까요?               | ...怎么样？         | 좋은 생각이에요. / 글쎄요...    |
| ~는 게 어때요?                  | ...如何？           | 그게 더 나을 것 같아요.         |
| ~(으)ㄹ까요?                    | 要不要...？         | 네, 그래요. / 아니요, 괜찮아요. |
| 차라리 ~는 게 낫지 않을까요?    | 不如...不是更好吗？ | 그것도 일리가 있네요.           |
| ~해 보는 건 어때요?             | 试试...怎么样？     | 한번 해 볼게요.                 |
| 제 생각에는 ~(으)면 좋겠는데요. | 我觉得...就好了。   | 저도 그렇게 생각해요.           |

## E. 同意/附和

| 韩语                  | 中文           |
| --------------------- | -------------- |
| 맞아요.               | 对。           |
| 저도 그렇게 생각해요. | 我也这么想。   |
| 그거 좋은 생각이에요. | 那是个好主意。 |
| 동의합니다.           | 我同意。       |
| 그럼요. 당연하죠.     | 当然了。       |
| 말씀하신 대로예요.    | 如您所说。     |
| 맞는 말씀이에요.      | 说得对。       |

## F. 拒绝/婉拒

| 韩语                             | 中文                  | 语气               |
| -------------------------------- | --------------------- | ------------------ |
| 좀 힘들 것 같아요.               | 恐怕有点困难。        | 委婉               |
| 좀 그런데요...                   | 有点儿那个...         | 委婉               |
| 다음에 하면 안 될까요?           | 下次行不行？          | 委婉               |
| 죄송한데 그날은 선약이 있어서요. | 不好意思那天有约了。  | 标准拒绝           |
| 글쎄요, 좀 생각해 볼게요.        | 嗯，让我想想。        | 犹豫（多半是拒绝） |
| 그건 좀 어려울 것 같은데요.      | 那个恐怕有点困难。    | 委婉               |
| 다른 건 괜찮은데 그건 좀...      | 别的可以但那个有点... | 部分拒绝           |

| 韩语                | 中文           | 回答                      |
| ------------------- | -------------- | ------------------------- |
| 감사합니다.         | 谢谢。         | 별말씀을요. / 아닙니다.   |
| 정말 고마워요.      | 真的谢谢。     | 천만에요.                 |
| 도움이 많이 됐어요. | 帮了大忙了。   | 도움이 됐다니 다행이네요. |

| 신세 많이 졌습니다. | 受了很多照顾。 | 뭘요.                     |

## I. 询问/确认

| 韩语                           | 中文              | 场景          |
| ------------------------------ | ----------------- | ------------- |
| 혹시 ~(으)세요?                | 请问是...吗？     | 确认身份/情况 |
| ~(이)라고 했나요?              | 说的是...吗？     | 听不清时确认  |
| 다시 한번 말씀해 주시겠어요?   | 能再说一遍吗？    | 没听清        |
| 그러니까 ~(이)라는 말씀이시죠? | 也就是说...对吧？ | 确认理解      |
| ~은/는 언제까지인가요?         | ...是到什么时候？ | 询问截止时间  |
| 어떻게 하면 되나요?            | 怎么做就行？      | 询问方法      |
| 혹시 ~(으)ㄴ 거 아세요?        | 您知道...吗？     | 询问信息      |

## J. 表达意见/想法

| 韩语                     | 中文                | 语气     |
| ------------------------ | ------------------- | -------- |
| 제 생각에는요            | 我觉得              | 标准     |
| 저는 ~다고 생각합니다    | 我认为              | 正式     |
| ~(으)ㄴ 것 같아요        | 好像是...           | 委婉     |
| 솔직히 말하면            | 坦白说              | 直接     |
| 개인적으로는             | 就个人来说          | 标准     |
| 사실 저는 ~거든요        | 其实我...           | 说明原因 |
| 글쎄요, 그건 좀...       | 嗯，那个有点...     | 否定     |
| ~(으)ㄹ 수도 있겠지만... | 可能也有道理但是... | 转折前   |

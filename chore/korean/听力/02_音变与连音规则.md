# 韩语音变与连音规则 — 听力致命关卡

> 听力听不懂的第一大原因：**写的和读的不一样**
> 本文列出所有音变规则 + 实际听写对照，反复看直到条件反射

---

## 一、连音化 (연음화) ★★★★★

**规则**：收音后面接元音时，收音移到下一个音节发音

| 书写     | 实际发音   | 中文       |
| -------- | ---------- | ---------- |
| 음악     | [으막]     | 音乐       |
| 한국어   | [한구거]   | 韩国语     |
| 있어요   | [이써요]   | 有/在      |
| 없어요   | [업써요]   | 没有       |
| 읽어요   | [일거요]   | 读         |
| 넓은     | [널분]     | 宽的       |
| 옷을     | [오슬]     | 衣服(宾格) |
| 꽃이     | [꼬치]     | 花(主格)   |
| 부엌에서 | [부어케서] | 在厨房     |
| 삶이     | [살미]     | 生活(主格) |
| 값은     | [갑슨]     | 价格(主格) |
| 닭이     | [달기]     | 鸡(主格)   |
| 젊은     | [절믄]     | 年轻的     |
| 읽은     | [일근]     | 读了的     |
| 없는     | [엄는]     | 没有的     |

### 双收音连音规则

双收音 + 元音 → 前收音留原位，后收音连到下一个音节

| 书写 | 实际发音 | 规则               |
| ---- | -------- | ------------------ |
| 읽어 | [일거]   | ㄺ → ㄹ留 + ㄱ连   |
| 없어 | [업써]   | ㅂㅅ → ㅂ留 + ㅅ连 |
| 삶의 | [살메]   | ㄹㅁ → ㄹ留 + ㅁ连 |
| 넓은 | [널분]   | ㄹㅂ → ㄹ留 + ㅂ连 |
| 닭을 | [달글]   | ㄹㄱ → ㄹ留 + ㄱ连 |

## 二、硬音化/紧音化 (경음화) ★★★★★

### 规则1：收音 ㄱ,ㄷ,ㅂ + ㄱ,ㄷ,ㅂ,ㅅ,ㅈ → 后面变紧音

| 书写   | 实际发音 | 中文 |
| ------ | -------- | ---- |
| 학교   | [학꾜]   | 学校 |
| 식당   | [식땅]   | 餐厅 |
| 입구   | [입꾸]   | 入口 |
| 있다   | [읻따]   | 有   |
| 없다   | [업따]   | 没有 |
| 숙제   | [숙쩨]   | 作业 |
| 국수   | [국쑤]   | 面条 |
| 합격   | [합껵]   | 合格 |
| 접시   | [접씨]   | 盘子 |
| 찹쌀   | [찹쌀]   | 糯米 |
| 학생   | [학쌩]   | 学生 |
| 식사   | [식싸]   | 用餐 |
| 독서   | [독써]   | 读书 |
| 갑자기 | [갑짜기] | 突然 |
| 작성   | [작썽]   | 填写 |

### 规则2：冠形词尾 -(으)ㄹ 后面紧音化

| 书写    | 实际发音  | 中文       |
| ------- | --------- | ---------- |
| 할 것   | [할 껏]   | 要做的     |
| 갈 곳   | [갈 꼳]   | 要去的地方 |
| 볼 수   | [볼 쑤]   | 能看       |
| 할 줄   | [할 쭐]   | 会做       |
| 만들 때 | [만들 때] | 做的时候   |
| 먹을 것 | [머글 껏] | 要吃的     |

### 规则3：汉字词中 ㄹ收音 + ㄷ,ㅅ,ㅈ → 紧音化

| 书写 | 实际发音 | 中文      |
| ---- | -------- | --------- |
| 발전 | [발쩐]   | 发展      |
| 물질 | [물찔]   | 物质      |
| 절대 | [절때]   | 绝对      |
| 갈등 | [갈뜽]   | 葛藤/矛盾 |
| 말살 | [말쌀]   | 抹杀      |

## 三、鼻音化 (비음화) ★★★★★

**规则**：收音 ㄱ,ㄷ,ㅂ + ㄴ,ㅁ → 收音变成对应鼻音

| ㄱ → ㅇ | ㄷ → ㄴ | ㅂ → ㅁ |
| ------- | ------- | ------- |

| 书写   | 实际发音 | 中文         |
| ------ | -------- | ------------ |
| 한국말 | [한궁말] | 韩国话       |
| 작년   | [장년]   | 去年         |
| 국내   | [궁내]   | 国内         |
| 먹는   | [멍는]   | 吃的(现在时) |
| 있는   | [인는]   | 有的         |
| 없는   | [엄는]   | 没有的       |
| 듣는   | [든는]   | 听的         |
| 받는   | [반는]   | 收的         |
| 닫는   | [단는]   | 关的         |
| 입니다 | [임니다] | 是           |
| 합니다 | [함니다] | 做           |
| 십만   | [심만]   | 十万         |
| 앞문   | [암문]   | 前门         |
| 밥맛   | [밤맏]   | 饭味         |
| 숫놀음 | [순노름] | —            |

## 五、送气化 (격음화/유기음화) ★★★★

**规则**：ㅎ + ㄱ,ㄷ,ㅂ,ㅈ (或反过来) → 变成送气音 ㅋ,ㅌ,ㅍ,ㅊ

| 书写   | 实际发音 | 中文   |
| ------ | -------- | ------ |
| 좋다   | [조타]   | 好     |
| 좋고   | [조코]   | 好而且 |
| 놓다   | [노타]   | 放     |
| 많다   | [만타]   | 多     |
| 괜찮다 | [괜찬타] | 没关系 |
| 싫다   | [실타]   | 讨厌   |
| 넣다   | [너타]   | 放入   |
| 못하다 | [모타다] | 不能   |
| 급하다 | [그파다] | 急     |
| 입학   | [이팍]   | 入学   |
| 축하   | [추카]   | 祝贺   |
| 백화점 | [배콰점] | 百货店 |
| 특히   | [트키]   | 尤其   |
| 직후   | [지쿠]   | 之后   |
| 역할   | [여칼]   | 角色   |

## 七、ㅎ脱落 ★★★

**规则**：ㅎ收音 + 元音 → ㅎ不发音

| 书写   | 实际发音 | 中文   |
| ------ | -------- | ------ |
| 좋아   | [조아]   | 好     |
| 좋은   | [조은]   | 好的   |
| 놓아   | [노아]   | 放     |
| 싫어   | [시러]   | 讨厌   |
| 많아   | [마나]   | 多     |
| 괜찮아 | [괜차나] | 没关系 |
| 않아   | [아나]   | 不     |
| 않은   | [아는]   | 不的   |

## 八、收音代表音 ★★★

收音在词尾或辅音前，双收音选一个发音：

| 双收音 | 代表音 | 例词 | 发音          |
| ------ | ------ | ---- | ------------- |
| ㄳ     | ㄱ     | 넋   | [넉]          |
| ㄵ     | ㄴ     | 앉다 | [안따]        |
| ㄶ     | ㄴ     | 않다 | [안타]        |
| ㄺ     | ㄹ     | 읽다 | [익따] ※例外! |
| ㄻ     | ㅁ     | 삶   | [삼]          |
| ㄼ     | ㄹ     | 여덟 | [여덜]        |
| ㄽ     | ㄹ     | 외곬 | [외골]        |
| ㄾ     | ㄹ     | 핥다 | [할따]        |
| ㄿ     | ㄹ     | 읊다 | [을따]        |
| ㅀ     | ㄹ     | 잃다 | [일타]        |
| ㅄ     | ㅂ     | 없다 | [업따]        |

单收音代表音：
| 收音 | 代表音 | 例词 | 发音 |
|------|--------|------|------|
| ㅅ,ㅆ,ㅈ,ㅊ,ㅌ | ㄷ | 옷 [옫], 있[읻], 꽃[꼳], 낮[낟] |
| ㅋ | ㄱ | 부엌 [부억] |
| ㅍ | ㅂ | 앞 [압] |

## 九、ㄴ添加 (ㄴ 첨가) ★★★

**规则**：合成词中，前一个词以收音结尾 + 后一个词以 이,야,여,요,유 开头 → 加 ㄴ

| 书写   | 实际发音 | 中文     |
| ------ | -------- | -------- |
| 서울역 | [서울력] | 首尔站   |
| 한국어 | [한구거] | 韩国语   |
| 색연필 | [생년필] | 彩色铅笔 |
| 꽃잎   | [꼰닙]   | 花瓣     |
| 솜이불 | [솜니불] | 棉被     |
| 부산역 | [부산녁] | 釜山站   |
| 담요   | [담뇨]   | 毯子     |

## 十、听写练习句 (精听用)

> 以下句子包含多种音变，先盲听写出来，再对照

### 练习1：日常对话

```
1. 학교에서 학생들이 같이 읽는 것을 좋아해요.
   [학꾜에서 학쌩드리 가치 잉는 거슬 조아해요]

2. 작년에 한국말을 못했는데 올해는 많이 좋아졌어요.
   [장녀네 한궁마를 모탠는데 올해는 마니 조아져써요]

3. 편리한 연락 방법이 없는 것 같아요.
   [펼리한 열락 방버비 엄는 걷 가타요]

4. 입학 축하해요! 열심히 공부합시다.
   [이팍 추카해요! 열씨미 공부합씨다]

5. 갑자기 비가 와서 옷이 다 젖었어요.
   [갑짜기 비가 와서 오시 다 저저써요]
```

### 练习2：新闻/正式场景

```
6. 국내 경제 상황이 급격히 악화되고 있습니다.
   [궁내 경제 상황이 급껴키 아콰되고 읻씀니다]

7. 관련 법안이 국회에서 통과될 것으로 전망됩니다.
   [괄련 버바니 구쾨에서 통과될 거스로 전망됨니다]

8. 최근 연구 결과에 따르면 환경 문제가 심각합니다.
   [최근 연구 결과에 따르면 환경 문제가 심가캄니다]

9. 많은 전문가들이 이 정책에 대해 우려를 표했습니다.
   [마는 전문가드리 이 정채게 대해 우려를 표햇씀니다]

10. 올해 목표를 달성하기 위해 노력하겠습니다.
    [올해 목표를 달썽하기 위해 노려카겟씀니다]
```

### 练习3：高级/学术

```
11. 독립적인 사고력을 갖추는 것이 중요합니다.
    [동닙쩌긴 사고려글 갇추는 거시 중요함니다]

12. 결론적으로 이 문제는 불가피한 것으로 볼 수밖에 없습니다.
    [결론쩌그로 이 문제는 불가피한 거스로 볼 쑤바게 업씀니다]

13. 특히 합리적인 해결 방안을 모색할 필요가 있습니다.
    [트키 함니쩌긴 해결 방아늘 모새칼 피료가 읻씀니다]
```

# 广告·通知·公告场景 — 完整原文与解析

> TOPIK 듣기 中等难度核心场景（约 9~16题）
> 特点：独白形式，信息密度高，细节多

---

## 类型1：校园通知

### 通知1：图书馆开放时间调整

```
안내 말씀드리겠습니다.
기말시험 기간을 맞이하여 중앙 도서관의 운영 시간이 변경됩니다.
십이월 일일부터 십이월 이십일까지 도서관은
오전 여덟 시부터 밤 열두 시까지 연장 운영합니다.
다만 열람실만 연장 운영되며, 대출 반납 서비스는
기존과 같이 오후 여섯 시까지만 가능합니다.
이용에 참고해 주시기 바랍니다.
```

**핵심 정보:**

- 기간: 12/1~12/21
- 연장 시간: 08:00~24:00
- 주의: 열람실만 연장, 대출반납은 18시까지

---

### 通知2：수강 신청 안내

```
학생 여러분께 안내드립니다.
다음 학기 수강 신청이 다음 주 월요일부터 시작됩니다.
사학년은 월요일, 삼학년은 화요일, 이학년은 수요일, 일학년은 목요일에
각각 오전 아홉 시부터 신청 가능합니다.
정정 기간은 개강 후 일 주일이며, 최대 이십일 학점까지 신청할 수 있습니다.
수강 신청 관련 문의사항은 학사지원과로 연락해 주시기 바랍니다.
```

**핵심 정보:**

- 순서: 4학년(월) → 3학년(화) → 2학년(수) → 1학년(목)
- 시간: 오전 9시
- 학점 제한: 최대 21학점
- 정정 기간: 개강 후 1주

## 类型2：商场/促销广告

### 广告1：百货周年庆

```
고객 여러분, 반갑습니다.
현대백화점 삼십 주년 기념 특별 세일을 안내드립니다.
이번 주 금요일부터 일요일까지 삼 일간 전 품목 이십에서 오십 퍼센트 할인
행사를 진행합니다. 특히 의류와 잡화는 최대 칠십 퍼센트까지 할인됩니다.
또한 오만 원 이상 구매하시는 고객님께는 주차 두 시간 무료 서비스를
제공해 드립니다. 많은 이용 부탁드립니다.
```

**핵심 정보:**

- 기간: 금~일 3일간
- 할인: 전 품목 20~50%, 의류·잡화 최대 70%
- 주차: 5만원 이상 구매 시 2시간 무료

---

### 广告2：超市特价

```
마트를 찾아 주신 고객 여러분, 감사합니다.
오늘의 특가 상품을 안내해 드리겠습니다.
국내산 한우 등심이 백 그램당 삼천구백 원,
유기농 사과 한 박스 만 이천 원에 판매하고 있습니다.
또한 생수 이 리터 여섯 병 묶음이 삼천 원입니다.
특가 상품은 오늘 하루만 진행되며 재고 소진 시 조기 종료될 수 있습니다.
많은 이용 바랍니다.
```

**핵심 정보:**

- 한우 등심: 100g/3,900원
- 사과: 1박스/12,000원
- 생수: 2L×6병/3,000원
- 기한: 오늘만, 재고 소진 시 종료

## 类型3：公共机构通告

### 通告1：地铁运行调整

```
승객 여러분께 안내 말씀드리겠습니다.
시설 점검 공사로 인하여 이번 주 토요일과 일요일 이틀간
삼호선 교대역에서 남부터미널역 구간의 운행이 중단됩니다.
해당 구간은 무료 셔틀버스가 대신 운행되며,
버스 탑승장은 각 역 이번 출구 앞에 있습니다.
열차 운행은 다음 주 월요일 첫차부터 정상 운행됩니다.
불편을 드려 죄송합니다.
```

**핵심 정보:**

- 원인: 시설 점검 공사
- 구간: 3호선 교대역~남부터미널역
- 기간: 토~일 이틀
- 대체: 무료 셔틀버스 (2번 출구 앞)
- 정상 복구: 다음 월요일

---

### 通告2：水费缴纳

```
주민 여러분께 알려 드립니다.
이번 달 수도 요금 고지서가 발송되었습니다.
납부 기한은 이번 달 이십오일까지이며,
은행 창구, 인터넷 뱅킹, 편의점에서 납부 가능합니다.
기한 내에 납부하지 않으실 경우 삼 퍼센트의 가산금이 부과됩니다.
고지서를 분실하신 분은 관리사무소에서 재발급 받으실 수 있습니다.
```

**핵심 정보:**

- 마감: 이달 25일
- 납부 방법: 은행/인터넷뱅킹/편의점
- 연체: 3% 가산금
- 분실: 관리사무소 재발급

## 类型4：活动/展览通知

### 通知1：文化活动

```
시민 여러분, 안녕하십니까.
서울시에서 주최하는 가을 문화제가 다가왔습니다.
행사는 시월 십오일부터 이십일까지 일 주일간 여의도 공원에서 열립니다.
전통 공연, 체험 부스, 먹거리 장터 등 다양한 프로그램이 준비되어 있으며,
입장료는 무료입니다.
다만 일부 체험 프로그램은 사전 예약이 필요합니다.
자세한 내용은 서울시 홈페이지에서 확인해 주세요.
```

**핵심 정보:**

- 기간: 10/15~10/21 (1주)
- 장소: 여의도 공원
- 입장: 무료
- 체험: 사전 예약 필요

---

### 通知2：写作大赛

```
제 이십 회 전국 한국어 글쓰기 대회를 안내드립니다.
참가 대상은 한국어를 배우는 외국인이며,
주제는 '한국에서의 나의 이야기'입니다.
원고 분량은 에이포 용지 세 장에서 다섯 장이며,
접수 기간은 육월 일일부터 칠월 삼십일일까지입니다.
접수는 이메일로만 가능하며,
대상 수상자에게는 상금 백만 원과 장학금이 수여됩니다.
```

**핵심 정보:**

- 대상: 외국인 한국어 학습자
- 주제: 한국에서의 나의 이야기
- 분량: A4 3~5장
- 접수: 6/1~7/31, 이메일만
- 상금: 대상 100만원+장학금

## 类型5：航班/交通广播

### 广播1：机场

```
탑승객 여러분께 안내말씀 드리겠습니다.
인천 발 뉴욕 행 대한항공 칠백이 편은 기상 악화로 인해
출발 시간이 지연되고 있습니다.
원래 오후 두 시 출발 예정이었으나 오후 네 시로 변경되었습니다.
탑승구도 십이번에서 십팔번으로 변경되었으니 확인해 주시기 바랍니다.
불편을 드려 대단히 죄송합니다.
```

**핵심 정보:**

- 편명: 대한항공 702편 (인천→뉴욕)
- 지연 원인: 기상 악화
- 변경: 14:00 → 16:00
- 탑승구: 12번 → 18번

## 广告/通知听力关键词总结

| 카테고리  | 반드시 잡아야 할 정보              |
| --------- | ---------------------------------- |
| 학교 통지 | 날짜, 시간, 학년별 순서, 제한사항  |
| 세일/할인 | 기간, 할인율, 조건, 예외 사항      |
| 공공 안내 | 원인, 구간/범위, 기간, 대체 수단   |
| 행사/대회 | 장소, 기간, 대상, 접수 방법, 비용  |
| 공항/교통 | 편명, 시간 변경, 게이트 변경, 원인 |

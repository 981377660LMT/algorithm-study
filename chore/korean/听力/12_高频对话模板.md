# 高频对话模板 — 按场景分类

> 200+ 对话模板，覆盖 TOPIK 듣기 全部出题场景
> 每段对话精简到考试常见长度

---

## A. 购物 (쇼핑)

### A1. 서점

```
여: 이 책 혹시 재고 있나요?
남: 잠간만요, 확인해 볼게요. 현재 품절인데요,
    주문하시면 삼일 후에 받으실 수 있습니다.
여: 네, 주문할게요. 전화로 연락 주실 수 있나요?
남: 네, 연락처 적어 주시면 입고되면 바로 연락드리겠습니다.
```

### A2. 옷가게

```
여: 이거 다른 색깔도 있어요?
남: 네, 검정, 흰색, 파란색 세 가지 있습니다.
여: 입어 봐도 되나요?
남: 네, 저쪽 탈의실을 이용해 주세요.
여: 이거 좀 큰 것 같은데 한 치수 작은 거 있어요?
남: 확인해 볼게요. 잠시만요.
```

### A3. 전자제품

```
남: 이 노트북 성능이 어때요?
여: 사무용으로 쓰시기에 충분합니다. 가격도 합리적이고요.
남: 배터리는 얼마나 가요?
여: 완충 시 약 열 시간 정도 사용 가능합니다.
남: 보증 기간은요?
여: 일 년 무상 보증이고, 연장 보증은 추가 비용이 들어요.
```

## B. 学校 (학교)

### B1. 교수님과 상담

```
남: 교수님, 리포트 제출 기한을 좀 연장해 주실 수 있을까요?
여: 무슨 일이 있었어요?
남: 지난주에 아파서 준비를 못 했습니다.
여: 그래요? 그러면 이틀만 더 줄게요. 수요일까지 제출하세요.
남: 감사합니다. 꼭 기한 내에 내겠습니다.
```

### B2. 수강 신청

```
여: 이번 학기에 한국어 수업 듣고 싶은데 아직 자리 있어요?
남: 초급은 마감됐고요, 중급에 자리가 세 개 남아 있어요.
여: 중급이면 수준이 좀 높지 않아요?
남: 레벨 테스트를 먼저 보시고 결정하시면 됩니다.
```

### B3. 도서관

```
남: 이 책 대출하려고 하는데요.
여: 학생증 보여 주세요. 현재 대출 중인 책이 세 권이시네요.
    최대 다섯 권까지 가능합니다.
남: 대출 기간은 어떻게 되나요?
여: 이 주일이고요, 한 번 연장 가능합니다.
    온라인이나 전화로 연장하실 수 있어요.
```

## C. 职场 (직장)

### C1. 회의 일정

```
여: 김 대리님, 오늘 오후 회의 참석하실 수 있으세요?
남: 오후에 외부 미팅이 있어서 좀 어려울 것 같은데요.
여: 그러면 내일 오전으로 옮겨도 될까요?
남: 내일 오전은 괜찮습니다. 몇 시에 할까요?
여: 열 시 회의실에서 하죠.
```

### C2. 업무 지시

```
남: 이 보고서 내일까지 수정해서 보내 줄 수 있어요?
여: 네, 어떤 부분을 수정하면 되나요?
남: 삼 페이지에 있는 데이터가 좀 오래된 것 같아요.
    최신 자료로 업데이트해 주세요.
여: 알겠습니다. 내일 오전까지 보내 드리겠습니다.
```

### C3. 점심 약속

```
여: 점심 같이 먹을래요?
남: 좋아요. 뭐 먹을까요?
여: 오늘 비빔밥 어때요? 새로 생긴 한식당이 맛있대요.
남: 좋아요. 열두 시 반에 로비에서 만날까요?
여: 네, 그때 봐요.
```

## D. 约会/朋友 (약속/친구)

### D1. 약속 장소 정하기

```
남: 이번 주말에 만나자. 어디서 볼까?
여: 음... 혹시 홍대 쪽은 어때?
남: 거기 주말에 너무 사람 많지 않아? 강남은?
여: 강남도 복잡하긴 마찬가지야. 그냥 신촌에서 만나자.
남: 그래, 신촌역 삼 번 출구 앞에서 두 시에 보자.
```

### D2. 생일 선물

```
여: 민수 생일에 뭘 선물하면 좋을까?
남: 요즘 커피를 많이 마시니까 텀블러는 어때?
여: 좋은 생각이다. 어디서 사지?
남: 인터넷으로 주문하면 내일 배송 오니까 그러자.
여: 그래. 포장도 해 달라고 하자.
```

### D3. 영화 관람

```
남: 오늘 개봉한 영화 봤어?
여: 아직. 재미있대?
남: 응, 평이 엄청 좋더라. 같이 볼래?
여: 좋아. 몇 시 거 볼까?
남: 여섯 시 꺼는 자리가 없고, 여덟 시 반은 있대.
여: 그럼 여덟 시 반으로 해. 좌석은 네가 예매해 줄래?
남: 응, 알겠어.
```

## E. 旅行 (여행)

### E1. 관광 안내소

```
여: 여기에서 경복궁까지 어떻게 가면 되나요?
남: 지하철 삼호선을 타시고 경복궁역에서 내리시면 됩니다.
여: 입장료는 얼마예요?
남: 한복을 입고 가시면 무료 입장이 가능합니다.
    아니면 삼천 원입니다.
여: 한복 대여하는 데가 근처에 있어요?
남: 네, 안국역 쪽에 여러 곳 있습니다.
```

### E2. 호텔 문의

```
남: 방에 에어컨이 안 되는 것 같은데요.
여: 죄송합니다. 바로 확인해 보겠습니다. 잠시만 기다려 주세요.
남: 그리고 수건 좀 더 받을 수 있을까요?
여: 네, 바로 보내 드리겠습니다. 또 필요하신 것 있으시면
    프런트로 연락해 주세요.
```

## F. 交通 (교통)

### F1. 버스

```
여: 저기요, 이 버스 시청 앞에 서나요?
남: 아니요, 이 버스는 안 서요. 건너편에서 삼백이십 번을 타세요.
여: 한 정거장 지나서요?
남: 아니요, 바로 다음 정거장이에요.
```

### F2. 기차 예매

```
남: 서울에서 부산까지 케이티엑스 표 두 장 주세요.
여: 어떤 시간대로 드릴까요?
남: 오전 열 시 꺼요. 자리 있어요?
여: 일반실은 매진이고요, 특실은 남아 있습니다.
남: 가격이 얼마나 차이 나요?
여: 만 원 차이 납니다.
남: 그럼 특실로 할게요.
```

## G. 运动/兴趣 (운동/취미)

### G1. 운동 약속

```
여: 내일 아침에 같이 조깅할래?
남: 몇 시에?
여: 여섯 시 반에 한강공원에서.
남: 좀 이른데... 일곱 시는 안 돼?
여: 그래, 그럼 일곱 시에 보자.
```

### G2. 요리 교실

```
남: 이번 달 요리 교실 아직 자리 있어요?
여: 한식 클래스는 마감됐고요, 이탈리안 클래스가 남아 있어요.
남: 몇 회 수업이에요?
여: 주 일 회, 총 사 회입니다. 수업료는 십이만 원이에요.
남: 재료비도 포함인가요?
여: 네, 포함입니다.
```

## H. 天气/季节 (날씨)

### H1. 날씨 대화

```
여: 오늘 날씨가 정말 춥다.
남: 내일은 더 춥대. 영하 십 도래.
여: 진짜? 두꺼운 옷 입어야겠다.
남: 눈도 온다고 하니까 우산도 챙겨.
```

## I. 邻居/社区 (이웃)

### I1. 이사 인사

```
남: 안녕하세요, 옆집에 이사 온 김민수입니다.
여: 아, 반갑습니다. 저는 박미영이에요.
    혹시 필요하신 것 있으면 편하게 말씀하세요.
남: 감사합니다. 혹시 근처에 마트가 어디 있는지
    알 수 있을까요?
여: 아파트 뒤쪽으로 나가시면 오 분 거리에 이마트가 있어요.
```

## J. 紧急状况 (긴급)

### J1. 경찰

```
여: 신고 좀 하려고요. 방금 지갑을 잃어버렸어요.
남: 어디에서 잃어버리신 건가요?
여: 지하철 이호선 안에서요. 잠실역에서 내릴 때 없는 걸 알았어요.
남: 지갑 안에 뭐가 들어 있었나요?
여: 신분증, 카드 두 장, 현금 삼만 원 정도요.
남: 알겠습니다. 분실물 센터에 연락해 보시고,
    카드는 바로 정지시키세요.
```

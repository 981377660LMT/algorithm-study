## 二、语气信号词速查

### 肯定/同意信号

| 信号句                 | 含义       |
| ---------------------- | ---------- |
| 맞아요 / 맞습니다      | 对         |
| 저도 그렇게 생각합니다 | 我也这么想 |
| 좋은 생각이에요        | 好主意     |
| 당연하죠               | 当然       |
| 물론이죠               | 那当然     |
| 동의합니다             | 同意       |
| 그거 괜찮은데요        | 那不错啊   |
| 그게 더 나을 것 같아요 | 那样更好   |

### 否定/反对信号

| 信号句                       | 含义                   |
| ---------------------------- | ---------------------- |
| 글쎄요...                    | 嗯...（犹豫=多半否定） |
| 좀 그런데요...               | 有点那个...（否定）    |
| 저는 좀 다르게 생각하는데요  | 我想法不同             |
| 그건 좀 무리가 있지 않을까요 | 那是不是有点勉强       |
| 꼭 그렇지만은 않다고 봅니다  | 未必如此               |
| 현실적으로 어렵다고 봅니다   | 现实上很难             |
| 그건 좀 과장된 것 같아요     | 那有点夸张             |

### 担忧信号

| 信号句                         | 含义              |
| ------------------------------ | ----------------- |
| 좀 걱정이 됩니다               | 有点担心          |
| ~(으)ㄹ 우려가 있습니다        | 有...的担忧       |
| 자칫하면 ~(으)ㄹ 수도 있습니다 | 不小心的话可能... |
| 부작용이 있을 수 있습니다      | 可能有副作用      |
| 慎重해야 합니다                | 必须慎重          |

### 先同意后转折信号 ★★★

| 信号句             | 含义         | 真实态度         |
| ------------------ | ------------ | ---------------- |
| 물론 ~지만         | 当然...但    | **后面才是真意** |
| 그 점은 인정하지만 | 那点承认但   | 反对             |
| 일리가 있기는 한데 | 有道理是有   | 但不同意         |
| 취지는 좋지만      | 初衷虽好但   | 否定可行性       |
| 이해는 하지만      | 理解是理解但 | 实际不同意       |

> ⚠️ **考试陷阱：물론 뒤의 내용이 진짜 입장이다！**
> 如果听到 "물론 좋은 점도 있지만..."，说话人的态度是**否定/担忧**

## 三、态度判断练习

### 练习1

```
여: 이번에 도입된 재택근무 제도에 대해 어떻게 생각하세요?
남: 취지는 좋다고 생각합니다. 하지만 아직 준비가 부족한 상태에서
    시행하면 혼란이 생길 수 있어요. 좀 더 충분한 준비 기간이
    필요했다고 봅니다.
```

→ 남자의 태도? **조건부 우려 (취지 인정 + 시기상조)**

### 练习2

```
남: 이 프로젝트가 잘 될까요?
여: 글쎄요, 시장 상황을 보면 좀 어려울 것 같은데요.
    경쟁사들도 비슷한 제품을 이미 내놓고 있잖아요.
```

→ 여자의 태도? **부정적/회의적**

### 练习3

```
여: 이번 전시회 다녀오셨다면서요? 어떠셨어요?
남: 기대 이상이었어요. 특히 미디어 아트 작품이 정말 인상적이었습니다.
    관람객 참여형 작품도 있어서 재미있었어요.
```

→ 남자의 태도? **긍정적/만족**

### 练习4

```
남: 정부의 새 교육 정책에 대해 어떻게 보십니까?
여: 교육의 질을 높이겠다는 방향은 맞다고 생각합니다.
    다만 현장의 의견을 충분히 반영했는지는
    좀 더 지켜봐야 할 것 같습니다.
```

→ 여자의 태도? **신중/조건부 긍정**

### 练习5

```
남: 온라인 쇼핑이 오프라인 매장을 완전히 대체할 수 있다고 보시나요?
여: 그렇게 단정할 수는 없다고 봅니다. 직접 보고 만져 보고 싶은
    상품도 있고, 오프라인만의 쇼핑 경험도 가치가 있으니까요.
    두 채널이 공존하게 될 거라고 봅니다.
```

→ 여자의 태도? **반대 (완전 대체에 반대, 공존 주장)**

### 练习6

```
여: 이번 봉사 활동 후기 좀 들려주세요.
남: 솔직히 처음에는 의무감에 참여했거든요.
    그런데 막상 해 보니까 보람이 크더라고요.
    다음에도 꼭 참여하고 싶습니다.
```

→ 남자의 태도? **긍정적 (처음 의무감 → 실제로 보람)**

## 四、复杂态度判断 — 高级技巧

### 类型1：表面肯定实际否定

```
네, 좋은 의견이기는 한데요...
→ 否定
```

### 类型2：双重转折后的真正立场

```
물론 문제가 있는 것은 사실입니다. 하지만 그렇다고 해서
이 정책 자체를 폐지해야 한다고는 생각하지 않습니다.
→ 찬성 (虽然承认有问题，但支持继续实施)
```

### 类型3：用反问句表达强烈立场

```
그게 과연 옳은 일이라고 할 수 있을까요?
→ 强烈反对 (반어적 표현)
```

### 类型4：语气递进

```
게다가... 더구나... 심지어...
→ 层层加码 = 强烈支持或强烈反对（看上下文方向）
```

## 五、语调与态度

| 语调特征                  | 可能的态度        |
| ------------------------- | ----------------- |
| 升调结尾 + 长音 "글쎄요~" | 犹豫/否定         |
| 语速快 + 连续递进         | 兴奋/强烈支持     |
| 声音低沉 + 叹气           | 担忧/失望         |
| 笑声 + "아, 정말요?"      | 惊喜/积极         |
| 停顿 + "좀..."            | 难以启齿/委婉否定 |
| 强调重音在"꼭/반드시"     | 强烈主张          |
| "~잖아요" 语气            | 理所当然/略微不满 |

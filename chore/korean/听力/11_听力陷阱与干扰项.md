# 听力陷阱与干扰项分析

> TOPIK 考试中选错的大部分原因不是没听懂，而是被干扰项骗了
> 掌握出题套路 = 多拿10分

---

## 一、六大偷换手法

### 陷阱1：主语偷换 ★★★★★

**原文**：남자가 회의 자료를 준비하기로 했다
**错误选项**：여자가 자료를 준비할 것이다

> 听到"谁做什么"时，立刻在脑中标记主语

### 陷阱2：时态偷换 ★★★★★

| 原文                 | 干扰项             | 偷换方式  |
| -------------------- | ------------------ | --------- |
| 다음 주에 시작한다   | 지난주에 시작했다  | 未来→过去 |
| 아직 결정하지 않았다 | 이미 결정했다      | 未定→已定 |
| 계획 중이다          | 이미 시행하고 있다 | 计划→实施 |
| 고려하고 있다        | 확정했다           | 考虑→确定 |

### 陷阱3：程度偷换 ★★★★

| 原文            | 干扰项        | 偷换方式    |
| --------------- | ------------- | ----------- |
| 대부분 찬성했다 | 모두 찬성했다 | 大部分→全部 |
| 다소 증가했다   | 크게 증가했다 | 稍微→大幅   |
| 일부 지역에서   | 전국적으로    | 部分→全部   |
| 가능성이 있다   | 확실하다      | 可能→确定   |

### 陷阱4：否定词偷换 ★★★★

| 原文                 | 干扰项          |
| -------------------- | --------------- |
| 무료로 제공한다      | 유료로 제공한다 |
| 아직 완성되지 않았다 | 이미 완성되었다 |
| 참가할 수 없다       | 참가할 수 있다  |
| 제한이 있다          | 제한이 없다     |

### 陷阱5：因果倒置 ★★★

**原文**：비 때문에 행사가 취소되었다
**干扰项**：행사가 취소되어서 비가 왔다 ← (荒谬但出现过类似变体)

更常见的：

- 原文：A하므로 B하다 → 干扰：B하므로 A하다
- 原文："因为工作忙取消了旅行" → 干扰："因为取消旅行所以去工作"

### 陷阱6：细节偷换 ★★★

| 原文          | 干扰项       | 偷换内容  |
| ------------- | ------------ | --------- |
| 오후 세 시    | 오전 세 시   | 上下午    |
| 금요일까지    | 금요일부터   | 截止→开始 |
| 삼 층에서     | 이 층에서    | 楼层      |
| 이메일로 접수 | 전화로 접수  | 方式      |
| 선착순 50명   | 선착순 100명 | 数量      |

## 二、题型专属陷阱

### 1~2题（选回答）陷阱

```
问：내일 같이 영화 볼까요?

A. 네, 어제 영화관에 갔어요.     ← 词汇重复(영화)但时态错
B. 내일은 좀 힘들 것 같아요.     ← ✓ 正确回答
C. 영화가 재미있었어요.          ← 词汇重复但答非所问
D. 네, 영화를 좋아해요.          ← 答非所问
```

**规律**：选项中出现原句词汇≠正确答案，要看逻辑对应

### 13~16题（内容一致）陷阱

```
原文：무료 와이파이는 카페 이용 고객에 한해 제공됩니다.

A. 와이파이를 사용하려면 카페에서 음료를 사야 한다. ← ✓
B. 누구나 무료로 와이파이를 사용할 수 있다.        ← ✗ "에 한해" = 限于
C. 와이파이 비밀번호가 따로 필요하다.              ← 원문에 없음
D. 카페에서 음료를 마시면 할인해 준다.             ← 원문에 없음
```

**规律**："에 한해"(限于)、"경우에만"(仅...情况下) 等限定词是重点

### 17~20题（态度）陷阱

```
남: 물론 재택근무도 장점이 있죠. 하지만 팀워크나 소통 면에서는
    출근이 필요하다고 봅니다.

A. 재택근무에 찬성하고 있다.      ← ✗ "물론...하지만" 后面才是真意
B. 재택근무에 반대하고 있다.      ← ✓ 转折后 = 실제 입장
C. 재택·출근 모두 좋다고 본다.    ← ✗ 过度概括
D. 팀워크에 관심이 없다.          ← ✗ 反推
```

### 31~50题（长文+两道题）陷阱

**第1题（主题/态度）**：

- 开头引入的一般观点 ≠ 说话人观点
- "일반적으로 ~라고 하지만" 后面才是主张

**第2题（内容一致）**：

- 讲了5个信息点，4个选项各对应不同信息点
- 选项会微调1个词让它变错
- 未提及的信息作为选项 = 100%错

## 三、防骗策略

### 策略1：预读选项划关键词

利用题间空隙（约10秒）快速看4个选项，划出差异点：

```
A. 남자가 자료를 → 주어: 남자, 행위: 자료
B. 여자가 자료를 → 주어: 여자, 행위: 자료
C. 남자가 회의를 → 주어: 남자, 행위: 회의
D. 여자가 회의를 → 주어: 여자, 행위: 회의
```

→ 只需要听清 **谁** 做 **什么** 即可

### 策略2：听到转折词立刻集中

转折后面 = 出题重点区域

- 그러나 / 하지만 / 그런데
- 반면(에) / 오히려
- 사실은 / 실제로는

### 策略3：标记否定词

听到以下词时在选项旁打×/✓：

- 안, 못, 없다, 아직, 아니라
- 不/未/没有/还没/不是

### 策略4：数字出现时立刻写

不要靠记忆，听到数字就速写：

- 날짜 → D:
- 시간 → T:
- 금액 → ₩:
- 퍼센트 → %:

### 策略5：排除法

如果实在没听清，用排除法：

1. 排除明显矛盾的选项
2. 排除原文未提及的内容
3. 排除过于绝对的表述（모든, 반드시, 절대）
4. 剩下的就是答案

## 四、实战训练 — 找干扰项

### 练习：听以下原文，判断哪个选项是错的

**原文**：

```
이번 주말 봉사 활동은 토요일 오전 아홉 시에 시작합니다.
장소는 강남구 복지관이며, 점심은 제공됩니다.
참가를 원하시는 분은 금요일까지 신청해 주세요.
정원은 삼십 명이며 선착순으로 마감됩니다.
```

| 选项                              | 对/错 | 분석                               |
| --------------------------------- | ----- | ---------------------------------- |
| A. 토요일 오전에 봉사 활동이 있다 | ✓     | 正确                               |
| B. 점심 식사가 무료로 제공된다    | ✓     | "점심은 제공" = 무료 (일반적 해석) |
| C. 토요일까지 신청해야 한다       | ✗     | **금요일**까지! 日期偷换           |
| D. 선착순 삼십 명으로 제한된다    | ✓     | 正确                               |

**答案：C（토요일→금요일 偷换）**

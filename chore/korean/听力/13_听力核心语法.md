# 听力核心语法 — 听懂长句的关键

> 听力中长句听不懂，80%是语法结构没反应过来
> 本文不讲语法理论，只讲"听到什么 → 意思是什么"

---

## 一、终结语尾 — 判断句子类型

### 陈述

| 语尾           | 语体     | 例                                  |
| -------------- | -------- | ----------------------------------- |
| ~ㅂ니다/습니다 | 格式体   | 갑니다, 먹습니다                    |
| ~아/어요       | 非格式体 | 가요, 먹어요                        |
| ~(이)다        | 书面     | ~인 것이다                          |
| ~거든요        | 说明原因 | 바빠서 못 가거든요 (其实我忙去不了) |

### 疑问

| 语尾                  | 例                        |
| --------------------- | ------------------------- |
| ~ㅂ니까/습니까?       | 괜찮습니까?               |
| ~아/어요?             | 괜찮아요?                 |
| ~(으)ㄹ까요?          | 먹을까요? (要吃吗?)       |
| ~나요? / ~(으)ㄴ가요? | 있나요? 괜찮은가요?       |
| ~잖아요?              | 비싸잖아요? (不是很贵嘛?) |

### 命令/请求

| 语尾               | 例                  |
| ------------------ | ------------------- |
| ~(으)세요          | 앉으세요            |
| ~아/어 주세요      | 도와주세요          |
| ~(으)시기 바랍니다 | 확인하시기 바랍니다 |

### 提议

| 语尾         | 例            |
| ------------ | ------------- |
| ~(으)ㅂ시다  | 먹읍시다      |
| ~(으)ㄹ까요? | 갈까요?       |
| ~자          | 가자 (非敬语) |

## 二、连接语尾 — 句与句之间的关系

### 原因/理由 ★★★★★

| 听到              | 意思       | 例                       |
| ----------------- | ---------- | ------------------------ |
| ~아/어서          | 因为       | 비가 와서 못 갔어요      |
| ~(으)니까         | 因为(主观) | 추우니까 코트를 입으세요 |
| ~기 때문에        | 因为(正式) | 교통이 불편하기 때문에   |
| ~(으)므로         | 因为(书面) | 시간이 제한되므로        |
| ~느라고           | 因为忙于   | 공부하느라고 못 갔어요   |
| ~는 바람에        | 突然原因   | 비가 오는 바람에         |
| ~거든요           | 因为(解释) | 내일 시험이거든요        |
| ~(으)ㄴ/는 탓에   | 因为(负面) | 잠을 못 잔 탓에          |
| ~(으)ㄴ/는 덕분에 | 多亏(正面) | 도와준 덕분에            |

### 转折 ★★★★★

| 听到                | 意思       | 例                   |
| ------------------- | ---------- | -------------------- |
| ~지만               | 但是       | 비싸지만 좋아요      |
| ~(으)ㄴ/는데        | 但是/然而  | 갔는데 문을 닫았어요 |
| ~아/어도            | 即使       | 비가 와도 가겠어요   |
| ~더라도             | 即使(假设) | 힘들더라도 해야 해요 |
| ~(으)ㄹ지라도       | 即使(书面) |                      |
| ~(으)ㄴ/는 반면(에) | 另一面     | 편리한 반면 비싸다   |

### 条件 ★★★★

| 听到         | 意思       | 例                   |
| ------------ | ---------- | -------------------- |
| ~(으)면      | 如果       | 시간이 있으면        |
| ~아/어야     | 只有...才  | 열심히 해야 성공해요 |
| ~(으)ㄹ 경우 | ...的情况  | 비가 올 경우         |
| ~다면        | 如果(假设) | 가능하다면           |
| ~(으)ㄹ 때   | ...的时候  | 시간이 될 때         |

### 目的 ★★★★

| 听到         | 意思       | 例                  |
| ------------ | ---------- | ------------------- |
| ~(으)려고    | 为了       | 사려고 왔어요       |
| ~(으)러      | 去做       | 먹으러 가요         |
| ~기 위해(서) | 为了(正式) | 건강을 위해서       |
| ~도록        | 使得/以便  | 알 수 있도록        |
| ~고자        | 想要(书面) | 말씀드리고자 합니다 |

### 时间顺序 ★★★

| 听到           | 意思      | 例                       |
| -------------- | --------- | ------------------------ |
| ~고 나서       | ...之后   | 밥을 먹고 나서           |
| ~(으)ㄴ 후(에) | ...之后   | 졸업한 후에              |
| ~기 전에       | ...之前   | 가기 전에 확인해         |
| ~자마자        | 一...就   | 도착하자마자 전화했다    |
| ~(으)면서      | 同时      | 음악을 들으면서 공부해요 |
| ~다가          | 中途/转而 | 걷다가 넘어졌어요        |

## 三、冠形形语尾 — 修饰名词的关键

| 时态 | 动词    | 形容词   | 名词         |
| ---- | ------- | -------- | ------------ |
| 过去 | ~(으)ㄴ | ~았/었던 | ~이었던/였던 |
| 现在 | ~는     | ~(으)ㄴ  | ~인          |
| 将来 | ~(으)ㄹ | ~(으)ㄹ  | ~일          |

### 听力中的实际运用

```
어제 본 영화        → 昨天看的电影 (过去)
지금 읽는 책        → 现在读的书 (现在)
내일 만날 사람      → 明天要见的人 (将来)
키가 큰 사람        → 个子高的人 (形容词现在)
유명한 배우         → 有名的演员 (形容词现在)
```

> ⚠️ 听到 ~(으)ㄴ 时要根据词性判断时态：
>
> - 동사 + (으)ㄴ = 过去（만난 사람 = 见过的人）
> - 형용사 + (으)ㄴ = 现在（좋은 사람 = 好的人）

## 四、间接引语 — 听力超高频 ★★★★★

### 基本结构

| 句式 | 听到              | 意思    |
| ---- | ----------------- | ------- |
| 陈述 | ~(ㄴ/는)다고 하다 | 说...   |
| 疑问 | ~(으)냐고 하다    | 问...   |
| 命令 | ~(으)라고 하다    | 叫...做 |
| 提议 | ~자고 하다        | 建议... |

### 实际对话中的变体

```
친구가 내일 온다고 했어요.       → 朋友说明天来
선생님이 숙제를 내라고 하셨어요.  → 老师叫交作业
의사가 쉬어야 한다고 했어요.     → 医生说要休息
부장님이 회의에 참석하냐고 물었어요. → 部长问能不能参加会议
```

### 缩略形 （口语中更常听到）

| 完整       | 缩略  | 例                       |
| ---------- | ----- | ------------------------ |
| ~다고 해요 | ~대요 | 간대요 (说要去)          |
| ~냐고 해요 | ~냬요 | 가냬요 (问去不去)        |
| ~라고 해요 | ~래요 | 가래요 (叫去)            |
| ~자고 해요 | ~재요 | 같이 가재요 (建议一起去) |

## 五、被动/使动 — 新闻/讲座高频

### 被动

| 能动         | 被动       | 例                    |
| ------------ | ---------- | --------------------- |
| 만들다 (做)  | 만들어지다 | 새 도로가 만들어졌다  |
| 쓰다 (写/用) | 쓰이다     | 이 표현이 많이 쓰인다 |
| 닫다 (关)    | 닫히다     | 문이 닫혔다           |
| 열다 (开)    | 열리다     | 행사가 열렸다         |
| 바꾸다 (换)  | 바뀌다     | 일정이 바뀌었다       |
| 나누다 (分)  | 나뉘다     | 두 그룹으로 나뉘었다  |

### 使动

| 自动      | 使动   | 例            |
| --------- | ------ | ------------- |
| 줄다 (减) | 줄이다 | 비용을 줄이다 |
| 늘다 (增) | 늘리다 | 시간을 늘리다 |
| 높다 (高) | 높이다 | 질을 높이다   |
| 낮다 (低) | 낮추다 | 가격을 낮추다 |
| 넓다 (宽) | 넓히다 | 범위를 넓히다 |

## 六、推测/猜测表达 — 态度判断关键

| 听到                      | 确信度 | 意思        |
| ------------------------- | ------ | ----------- |
| ~(으)ㄹ 것이다            | 高     | 会...的     |
| ~(으)ㄹ 것 같다           | 中     | 好像会...   |
| ~(으)ㄹ 수 있다           | 中     | 可能...     |
| ~(으)ㄹ지 모르다          | 低     | 说不定...   |
| ~(으)ㄹ 수도 있다         | 低     | 也有可能... |
| ~(으)ㄴ/는 듯하다         | 中     | 似乎...     |
| ~나 보다 / ~(으)ㄴ가 보다 | 中     | 看来是...   |
| ~(으)ㄴ/는 모양이다       | 中     | 好像是...   |

## 七、特殊句式速查

| 句式                | 意思      | 例                     |
| ------------------- | --------- | ---------------------- |
| ~(으)ㄹ 수밖에 없다 | 只能      | 갈 수밖에 없다         |
| ~(으)ㄹ 뿐이다      | 只不过    | 바라볼 뿐이다          |
| ~기 마련이다        | 理所当然  | 실수하기 마련이다      |
| ~는 법이다          | 本就是    | 노력하면 되는 법이다   |
| ~기는커녕           | 别说...了 | 칭찬은커녕 혼났다      |
| ~(으)ㄹ 리가 없다   | 不可能    | 그럴 리가 없다         |
| ~는 셈이다          | 等于是    | 공짜나 다름없는 셈이다 |
| ~에 불과하다        | 不过是    | 소수에 불과하다        |
| ~에 지나지 않다     | 不过是    | 변명에 지나지 않는다   |

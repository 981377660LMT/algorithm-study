## 一、固有数词 vs 汉字数词

### 固有数词 (하나, 둘, 셋...)

用于：个数、时（小时）、岁、杯、瓶、本等

| 数字 | 原形   | 接量词时 | 常见搭配                   |
| ---- | ------ | -------- | -------------------------- |
| 1    | 하나   | 한       | 한 개, 한 시, 한 명, 한 살 |
| 2    | 둘     | 두       | 두 개, 두 시, 두 명, 두 잔 |
| 3    | 셋     | 세       | 세 개, 세 시, 세 번        |
| 4    | 넷     | 네       | 네 개, 네 시, 네 명        |
| 5    | 다섯   | 다섯     | 다섯 개, 다섯 시           |
| 6    | 여섯   | 여섯     | 여섯 시                    |
| 7    | 일곱   | 일곱     | 일곱 시                    |
| 8    | 여덟   | 여덟     | 여덟 시                    |
| 9    | 아홉   | 아홉     | 아홉 시                    |
| 10   | 열     | 열       | 열 개, 열 시               |
| 11   | 열하나 | 열한     | 열한 시                    |
| 12   | 열둘   | 열두     | 열두 시                    |
| 20   | 스물   | 스무     | 스무 살, 스무 개           |
| 30   | 서른   | 서른     | 서른 살                    |
| 40   | 마흔   | 마흔     | 마흔 살                    |
| 50   | 쉰     | 쉰       | 쉰 살                      |
| 60   | 예순   | 예순     | 예순 살                    |
| 70   | 일흔   | 일흔     |                            |
| 80   | 여든   | 여든     |                            |
| 90   | 아흔   | 아흔     |                            |

### 汉字数词 (일, 이, 삼...)

用于：分、秒、日期、月份、电话号码、价格、楼层、号码

| 数字 | 韩语  | 数字 | 韩语 |
| ---- | ----- | ---- | ---- |
| 0    | 영/공 | 6    | 육   |
| 1    | 일    | 7    | 칠   |
| 2    | 이    | 8    | 팔   |
| 3    | 삼    | 9    | 구   |
| 4    | 사    | 10   | 십   |
| 5    | 오    | 100  | 백   |

| 大数 | 韩语 | 例               |
| ---- | ---- | ---------------- |
| 千   | 천   | 삼천 = 3000      |
| 万   | 만   | 오만 = 50000     |
| 十万 | 십만 | 이십만 = 200000  |
| 百万 | 백만 | 삼백만 = 3000000 |
| 千万 | 천만 |                  |
| 亿   | 억   | 일억 = 100000000 |

## 二、时间表达听写训练

### 时刻 (固有数词시 + 汉字数词분)

| 写法     | 读法                     | 中文    |
| -------- | ------------------------ | ------- |
| 1:00     | 한 시                    | 1点     |
| 2:30     | 두 시 삼십 분 / 두 시 반 | 2点30分 |
| 3:15     | 세 시 십오 분            | 3点15分 |
| 4:45     | 네 시 사십오 분          | 4点45分 |
| 7:20     | 일곱 시 이십 분          | 7点20分 |
| 10:05    | 열 시 오 분              | 10点5分 |
| 12:00    | 열두 시                  | 12点    |
| 오전 9시 | 오전 아홉 시             | 上午9点 |
| 오후 6시 | 오후 여섯 시             | 下午6点 |

### 听写练习 — 写出对应时间

```
1. 회의는 오후 세 시 삼십 분에 시작합니다.     → ___:___
2. 비행기는 오전 열한 시 사십오 분에 출발합니다.  → ___:___
3. 수업은 아홉 시 십 분부터 열 시 반까지입니다.  → ___:___ ~ ___:___
4. 마감은 오후 다섯 시까지입니다.               → ___:___
5. 영화는 일곱 시 이십 분에 시작해요.            → ___:___
```

**答案：**

1. 15:30 2. 11:45 3. 9:10~10:30 4. 17:00 5. 19:20

## 三、日期听写训练

### 月份 (汉字数词 + 월)

| 月   | 韩语     | ⚠️注意               |
| ---- | -------- | -------------------- |
| 1月  | 일월     |                      |
| 2月  | 이월     |                      |
| 3月  | 삼월     |                      |
| 4月  | 사월     |                      |
| 5月  | 오월     |                      |
| 6月  | **유**월 | ⚠️ 不是 육월！[유월] |
| 7月  | 칠월     |                      |
| 8月  | 팔월     |                      |
| 9月  | 구월     |                      |
| 10月 | **시**월 | ⚠️ 不是 십월！[시월] |
| 11月 | 십일월   |                      |
| 12月 | 십이월   |                      |

### 日 (汉字数词 + 일)

1日 = 일일, 15日 = 십오일, 31日 = 삼십일일

### 星期

| 韩语   | 中文   |
| ------ | ------ |
| 월요일 | 星期一 |
| 화요일 | 星期二 |
| 수요일 | 星期三 |
| 목요일 | 星期四 |
| 금요일 | 星期五 |
| 토요일 | 星期六 |
| 일요일 | 星期日 |

### 听写练习

```
1. 행사는 삼월 이십오일 토요일에 열립니다.          → ___月___日 星期___
2. 신청 마감은 유월 십오일까지입니다.                → ___月___日
3. 시월 삼일은 개천절입니다.                        → ___月___日
4. 올해 시험은 십일월 십구일 일요일입니다.            → ___月___日 星期___
5. 접수 기간: 팔월 일일부터 구월 삼십일까지           → ___月___日~___月___日
```

**答案：**

1. 3月25日 星期六 2. 6月15日 3. 10月3日 4. 11月19日 星期日 5. 8月1日~9月30日

## 四、价格听写训练

### 韩币读法

| 价格          | 韩语          | 注意      |
| ------------- | ------------- | --------- |
| 500원         | 오백 원       |           |
| 1,000원       | 천 원         | 不说 일천 |
| 3,500원       | 삼천오백 원   |           |
| 10,000원      | 만 원         | 不说 일만 |
| 15,000원      | 만 오천 원    |           |
| 50,000원      | 오만 원       |           |
| 100,000원     | 십만 원       |           |
| 350,000원     | 삼십오만 원   |           |
| 1,000,000원   | 백만 원       |           |
| 25,000,000원  | 이천오백만 원 |           |
| 100,000,000원 | 일억 원       |           |

### 听写练习

```
1. 이 옷은 삼만 구천 원입니다.                       → ____________원
2. 월세가 오십만 원이에요.                            → ____________원
3. 비행기 표가 백이십삼만 원이래요.                     → ____________원
4. 이 아파트가 삼억 오천만 원이에요.                    → ____________원
5. 커피 한 잔에 사천팔백 원이에요.                     → ____________원
6. 수강료가 이십오만 원입니다.                         → ____________원
7. 보증금은 천만 원이고 월세는 오십오만 원입니다.        → 보증금:______ 월세:______
```

**答案：**

1. 39,000원 2. 500,000원 3. 1,230,000원 4. 350,000,000원
2. 4,800원 6. 250,000원 7. 보증금:10,000,000원 월세:550,000원

## 五、电话号码听写

### 读法规则

- 0 读 공 (不读 영)
- 连字符处停顿

### 练习

```
1. 공일공-구삼일이-오육칠팔     → ___-____-____
2. 공이-오삼이-칠팔구공         → __-___-____
3. 공일공-이이삼사-오오육육     → ___-____-____
4. 공일공-삼삼사사-일이삼사     → ___-____-____
```

**答案：**

1. 010-9312-5678 2. 02-532-7890 3. 010-2234-5566 4. 010-3344-1234

## 六、百分比与倍数

| 表达          | 含义   | 用法         |
| ------------- | ------ | ------------ |
| 오십 퍼센트   | 50%    |              |
| 삼십이 퍼센트 | 32%    |              |
| 이 배         | 2倍    | 작년의 이 배 |
| 세 배         | 3倍    |              |
| 절반          | 一半   |              |
| 약 / 대략     | 约     | 약 40%       |
| 이상          | 以上   | 50% 이상     |
| 이하          | 以下   |              |
| 미만          | 未满   | 20세 미만    |
| 초과          | 超过   |              |
| 전체의        | 全体的 | 전체의 30%   |

### 听写练习

```
1. 조사 결과, 응답자의 육십삼 퍼센트가 찬성했습니다.    → ___%
2. 작년에 비해 약 이 배 증가했습니다.                   → ___倍
3. 전체의 삼분의 일이 참가했습니다.                     → ___分之___
4. 이번 달 매출이 삼십 퍼센트 감소했습니다.             → ___%
5. 합격률이 십이 퍼센트에 불과합니다.                   → ___%
```

**答案：**

1. 63% 2. 约2倍 3. 3分之1 4. 30% 5. 12%

## 七、综合听写练习

```
1. 신청 기간은 사월 일일부터 오월 삼십일일까지이며,
   수강료는 이십오만 원입니다.
   → 期间：___月___日~___月___日，学费：_______원

2. 오전 열 시에 출발해서 오후 두 시 삼십 분에 도착합니다.
   소요 시간은 약 네 시간 삼십 분입니다.
   → 出发：_____，到达：_____，耗时：___小时___分

3. 참가자 중 약 칠십 퍼센트가 이십 대에서 삼십 대였으며,
   사십 대 이상은 이십 퍼센트에 그쳤습니다.
   → 20~30多岁：___%，40岁以上：___%

4. 문의 전화는 공이-삼사오-육칠팔구이고,
   접수 마감은 구월 삼십일 금요일 오후 다섯 시입니다.
   → 电话：__-___-____  截止：___月___日 星期___ ___:___
```

**答案：**

1. 4月1日~5月31日，250,000원
2. 10:00出发，14:30到达，约4小时30分
3. 20~30多岁：约70%，40岁以上：20%
4. 02-345-6789 9月30日 星期五 17:00

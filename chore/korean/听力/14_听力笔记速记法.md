# 听力笔记速记法

> TOPIK 듣기 21~50题只播放1次，记笔记是刚需
> 不是写字，是画符号

---

## 一、核心原则

1. **用符号代替文字** — 写字太慢
2. **只记关键信息** — 数字、人物、结论
3. **用母语速记** — 中文比韩文快
4. **提前看选项** — 知道要记什么

---

## 二、通用符号系统

### 基本符号

| 符号 | 含义           | 用法       |
| ---- | -------------- | ---------- |
| ↑    | 增加/上升/提高 | 취업률 ↑   |
| ↓    | 减少/下降/降低 | 가격 ↓     |
| →    | 导致/所以/然后 | 비 → 취소  |
| ←    | 因为/原因      | 취소 ← 비  |
| ⇒    | 结论/因此      | ⇒ 필요     |
| ↔    | 对比/反面      | A ↔ B      |
| =    | 等于/就是      | 결과 = 65% |
| ≠    | 不是/不等于    | ≠ 무료     |
| ✓    | 对/肯定/有     | 주차 ✓     |
| ✗    | 错/否定/没有   | 예약 ✗     |
| ★    | 重要/注意      | ★ 마감 금  |
| ?    | 不确定/疑问    | 시간 ?     |
| !    | 惊喜/强调/注意 | 무료!      |
| ~    | 大约           | ~30%       |
| >    | 大于/多于/优于 | A > B      |
| <    | 小于/少于/劣于 | A < B      |

### 人物符号

| 符号    | 含义              |
| ------- | ----------------- |
| ♂ or 남 | 男                |
| ♀ or 여 | 女                |
| 甲/乙   | 第一个人/第二个人 |
| Prof    | 教授/专家         |

### 时间符号

| 符号    | 含义      |
| ------- | --------- |
| 과      | 过去      |
| 今      | 现在      |
| 来      | 将来      |
| AM / PM | 上午/下午 |
| 周      | 周/星期   |

### 数量符号

| 符号 | 含义        |
| ---- | ----------- |
| ½    | 절반/一半   |
| 2x   | 两倍        |
| W    | 원/韩币     |
| %    | 퍼센트      |
| #    | 数量/....个 |

## 三、场景速记模板

### 模板A：日常商务对话 (1~16题)

```
预备：看选项 → 标出需要确认的信息
记录：
  场：____（场所）
  谁：♂ ♀
  要：____（目的）
  数：____（数字）
  时：____（时间）
  $：____（价格）
```

### 模板B：新闻报道 (21~36题)

```
题：____（话题/主题）
据：____（数据/来源）
因：____（原因）
果：____（结果/影响）
策：____（对策/预定）
BUT：____（但是/反面意见）
```

### 模板C：讲座演讲 (31~50题)

```
题：____（主题）
论：____（核心论点）
  ① ____
  ② ____
  ③ ____
例：____（举例）
⇒：____（结论）
```

### 模板D：讨论辩论 (37~50题)

```
题：____（话题）
♂：赞/反 → ____（理由）
♀：赞/反 → ____（理由）
♂反驳：____
♀反驳：____
合：____（共识/没有共识）
```

## 四、实战演练

### 示例1：新闻报道速记

**听到**：

> 올해 상반기 청년 취업률이 전년 대비 이 퍼센트 포인트 상승한 것으로
> 나타났습니다. 전문가들은 정부의 청년 일자리 정책과 아이티 업종의
> 채용 확대가 주요 원인이라고 분석했습니다. 그러나 비정규직 비율이
> 여전히 높다는 점에서 고용의 질적 개선이 필요하다는 지적도...

**速记**：

```
题：청년취업
据：+2%p ↑ (상반기)
因：정부정책 + IT채용↑
BUT：비정규 ↑ → 질 개선 필요
```

### 示例2：讨论速记

**听到**：

> 남: 재택근무가 더 효율적... 출퇴근 시간 절약... 생산성 13%↑
> 여: 소통 어려움... 번아웃... 신입사원 학습 기회↓
> 남: 화상회의로 보완 가능
> 여: 주3출근+2재택 혼합형이 현실적

**速记**：

```
题：재택 vs 출근
♂：재택찬 → 시간 절약, 생산 +13%
♀：재택반 → 소통✗, 번아웃, 신입↓
♂：화상회의로 OK
♀：3+2 혼합 ⇒ 대안
```

### 示例3：讲座速记

**听到**：

> 오늘은 확증 편향에 대해... 자신의 믿음을 확인하는 정보만
> 받아들이고 반대 정보는 무시... 유기농 식품 예시...
> 사회적 갈등 심화... 반대 의견을 찾는 습관 필요

**速记**：

```
题：확증편향
= 자기믿음만 OK, 반대 ✗
例：유기농
影：사회갈등 ↑
⇒ 반대의견 찾기！
```

## 五、提升速记的日常训练

### 训练1：听新闻速记

1. 打开 KBS 뉴스 (YouTube)
2. 每条新闻用上面的模板速记
3. 结束后对照字幕检查遗漏

### 训练2：听播客速记

1. 听 세바시 或其他韩语播客
2. 用讲座模板速记要点
3. 尝试根据笔记复述内容

### 训练3：做TOPIK真题时严格记笔记

1. 21~50题每题都记
2. 答完后看笔记能否还原关键信息
3. 改进不够高效的地方

---

## 六、注意事项

1. **不要试图记住一切** — 只记选项需要的信息
2. **符号要固定** — 不要每次换，形成肌肉记忆
3. **字要小** — 留空间给下一段
4. **眼睛不离选项** — 手在纸上记，眼睛看选项
5. **练到不思考就能写** — 刻意练习到自动化

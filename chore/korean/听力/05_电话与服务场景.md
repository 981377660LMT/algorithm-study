# 电话与服务场景 — 完整对话原文

> TOPIK 듣기 1~16题常考场景
> 每段对话标注关键词、场所、考点

## 场景1：餐厅预约

**관련 어휘**: 예약, 인원, 자리, 코스, 창가

```
여: 안녕하세요, 맛있는 한식당입니다.
남: 네, 예약 좀 하고 싶은데요. 이번 주 토요일 저녁에 자리 있을까요?
여: 몇 분이세요?
남: 네 명이요.
여: 토요일 저녁은 지금 예약이 많은데요. 일곱 시는 어려울 것 같고
    여덟 시 반은 가능합니다. 괜찮으시겠습니까?
남: 네, 그 시간으로 해 주세요. 혹시 창가 자리로 부탁드려도 될까요?
여: 네, 확인해 보겠습니다. 성함이 어떻게 되세요?
남: 김민수입니다.
여: 확인하겠습니다. 토요일 여덟 시 반, 네 명, 김민수 님으로 예약됐습니다.
```

**考点**: 시간(8:30), 인원(4명), 요청사항(창가 자리)

## 场景2：退换货

**관련 어휘**: 교환, 환불, 영수증, 사이즈, 불량

```
여: 실례합니다. 이 옷을 교환하고 싶은데요.
남: 네, 어떤 문제가 있으세요?
여: 사이즈가 좀 작아서요. 한 치수 큰 걸로 바꿔 주세요.
남: 영수증 가지고 오셨어요?
여: 네, 여기 있습니다.
남: 확인해 보겠습니다. 구매하신 지 이 주 이내니까 교환 가능합니다.
    같은 색상 엘 사이즈가 지금 재고가 없는데요,
    다른 색으로 해 드릴까요, 아니면 환불해 드릴까요?
여: 그럼 환불해 주세요.
남: 네, 카드로 결제하셨으니까 카드 취소 처리해 드리겠습니다.
    영업일 기준 삼일에서 오일 정도 걸립니다.
```

**考点**: 교환 이유(사이즈 작음), 결과(환불), 소요 시간(3~5영업일)

## 场景3：医院/诊所

**관련 어휘**: 증상, 진찰, 처방, 약, 검사

```
남: 어디가 불편하세요?
여: 며칠 전부터 기침이 심하고 목이 아파요. 열도 좀 있는 것 같아요.
남: 언제부터 그러셨어요?
여: 삼일 전부터요. 처음에는 가벼운 감기인 줄 알았는데 점점 심해져서요.
남: 좀 보겠습니다. 목이 많이 부어 있네요. 혹시 기침할 때 가래가 나와요?
여: 네, 노란색 가래가 나와요.
남: 기관지에 염증이 있는 것 같습니다. 약을 처방해 드릴 테니
    하루에 세 번 식후에 드세요. 이삼 일 지나도 안 나으시면 다시 오세요.
여: 네, 감사합니다.
```

**考点**: 증상(기침, 목 아픔, 열), 진단(기관지 염증), 복용법(하루 3번 식후)

## 场景4：银行业务

**관련 어휘**: 계좌, 이체, 입금, 대출, 이자

```
여: 안녕하세요, 무엇을 도와드릴까요?
남: 새 통장을 만들고 싶은데요.
여: 네, 신분증 가지고 오셨나요?
남: 네, 여기 있습니다. 그리고 적금도 하나 들고 싶어요.
여: 현재 일 년 만기 적금 상품이 연 이자율 삼 퍼센트입니다.
    매달 얼마씩 넣으실 예정이세요?
남: 삼십만 원씩 넣으려고요.
여: 네, 그러면 일 년 후에 만기 이자 포함해서 약 삼백육십오만 원 정도
    받으실 수 있습니다. 가입 진행하시겠어요?
남: 네, 부탁드립니다.
```

**考点**: 목적(통장 개설+적금), 금액(월 30万), 이자율(3%)

## 场景5：快递/邮局

**관련 어휘**: 소포, 등기, 배송, 택배, 무게

```
여: 이 소포를 미국으로 보내고 싶은데요.
남: 네, 내용물이 뭔가요?
여: 책하고 과자입니다.
남: 무게를 재 볼게요. 이 킬로그램이네요. 일반 우편이면
    만 오천 원이고 등기로 보내시면 이만 삼천 원입니다.
여: 보통 며칠 정도 걸려요?
남: 일반은 이삼 주 정도 걸리고요, 등기는 일 주일에서 열흘 정도 걸립니다.
여: 등기로 보내 주세요.
남: 네, 받는 분 주소하고 전화번호 여기에 적어 주세요.
```

**考点**: 내용물(책+과자), 방법(등기), 비용(23,000원), 기간(일주일~열흘)

## 场景6：房产中介

**관련 어휘**: 전세, 월세, 보증금, 평수, 관리비

```
남: 안녕하세요, 신촌 쪽에 원룸을 찾고 있는데요.
여: 전세를 원하세요, 아니면 월세를 원하세요?
남: 월세로 알아보고 있어요. 보증금은 천만 원 정도로요.
여: 보증금 천만 원에 월세 오십만 원짜리가 하나 있거든요.
    역에서 걸어서 오 분이고 남향이에요.
남: 관리비는 얼마예요?
여: 관리비는 월 십만 원이고 인터넷하고 수도세는 포함이에요.
    전기세는 따로 나갑니다.
남: 한번 볼 수 있을까요?
여: 네, 내일 오후에 같이 가시죠.
```

**考点**: 조건(보증금 1000万, 월세 50万, 관리비 10万), 위치(역 도보 5분)

## 场景7：旅行社/机票

**관련 어휘**: 왕복, 편도, 직항, 경유, 수하물

```
여: 서울에서 도쿄까지 항공권을 예약하고 싶은데요.
남: 언제 출발하실 건가요?
여: 오월 이십일이요. 왕복으로요.
남: 돌아오시는 날은요?
여: 오월 이십오일이요.
남: 직항은 대한항공이 사십오만 원이고, 경유편은 이십팔만 원입니다.
여: 경유는 어디서 갈아타요?
남: 상하이에서 환승하시는데, 대기 시간이 세 시간 정도 됩니다.
여: 좀 길긴 한데... 직항으로 할게요. 수하물은 몇 킬로까지 되나요?
남: 이십삼 킬로그램까지 무료입니다.
```

**考点**: 일정(5/20~5/25), 가격(직항 45만원), 수하물(23kg)

## 场景8：维修/AS

**관련 어휘**: 고장, 수리, 보증 기간, 무상, 유상

```
남: 안녕하세요, 노트북이 갑자기 안 켜져서요.
여: 언제부터 그러셨어요?
남: 어제부터인데요, 전원 버튼을 눌러도 화면이 안 나와요.
여: 보증 기간이 아직 남아 있으신가요?
남: 네, 산 지 팔 개월밖에 안 됐어요.
여: 그러면 무상 수리가 가능합니다. 다만 점검하는 데 삼일에서
    오일 정도 걸릴 수 있습니다.
남: 데이터는 괜찮을까요?
여: 가능하면 보존해 드리겠지만, 혹시 모르니까 중요한 데이터는
    미리 백업해 두시는 게 좋습니다.
남: 네, 알겠습니다. 수리 맡기겠습니다.
```

**考点**: 증상(전원 안 켜짐), 보증(무상수리), 기간(3~5일)

## 场景9：酒店入住

**관련 어휘**: 체크인, 체크아웃, 조식, 트윈, 더블

```
남: 안녕하세요, 체크인하려고 하는데요.
여: 예약하셨나요? 성함이 어떻게 되세요?
남: 네, 박지훈으로 예약했습니다.
여: 확인됐습니다. 트윈룸으로 이 박 삼 일 예약하셨네요.
    체크아웃은 모레 오전 열한 시까지입니다.
남: 조식은 포함인가요?
여: 네, 조식은 일 층 식당에서 오전 일곱 시부터 아홉 시 반까지
    이용하실 수 있습니다.
남: 주차는 무료인가요?
여: 투숙객은 무료입니다. 지하 일 층에 주차하시면 됩니다.
```

**考点**: 방(트윈룸), 기간(2박3일), 체크아웃(11시), 조식(7:00~9:30)

## 场景10：健身房/会员注册

**관련 어휘**: 회원권, 등록, 트레이너, 이용 시간

```
여: 안녕하세요, 회원 등록하려고 왔는데요.
남: 네, 어떤 프로그램에 관심이 있으세요?
여: 기본 헬스장 이용만 하고 싶어요. 한 달에 얼마예요?
남: 한 달 회원권은 칠만 원이고요, 삼 개월은 십팔만 원,
    육 개월은 삼십만 원입니다.
여: 삼 개월로 할게요. 이용 시간은 어떻게 되나요?
남: 오전 여섯 시부터 밤 열한 시까지입니다.
    개인 트레이닝도 추가하실 수 있는데요, 한 달에 이십만 원입니다.
여: 일단 기본만 할게요. 나중에 추가할 수도 있죠?
남: 네, 물론이죠.
```

**考点**: 가격(3개월 18만원), 이용시간(6:00~23:00), PT(월 20만원)

## 场景11：出租车

**관련 어휘**: 요금, 미터기, 기본요금, 추가 요금

```
남: 어디로 가시겠습니까?
여: 서울역 좀 가 주세요. 얼마나 걸릴까요?
남: 이 시간대면 삼십 분 정도 걸릴 겁니다. 좀 막힐 수도 있어요.
여: 혹시 고속도로로 가면 더 빠를까요?
남: 고속도로는 통행료가 따로 나오는데 괜찮으세요?
여: 네, 빨리 가는 게 좋겠어요. 약속 시간에 늦을 것 같아서요.
남: 알겠습니다. 그쪽으로 가겠습니다.
```

**考点**: 목적지(서울역), 소요시간(30분), 추가비용(고속도로 통행료)

## 场景12：图书馆

**관련 어휘**: 대출, 반납, 연체, 도서 검색

```
여: 이 책들 반납하려고요.
남: 네, 확인해 보겠습니다. 한 권이 반납 기한이 오 일 지났는데요,
    연체료 이천 원이 발생합니다.
여: 아, 죄송해요. 잊고 있었네요. 그리고 새로 책을 빌리고 싶은데
    어떻게 검색하면 되나요?
남: 저쪽에 있는 검색기에서 제목이나 저자를 입력하시면 됩니다.
    한 번에 최대 다섯 권까지 대출 가능하고 대출 기간은 이 주입니다.
여: 네, 고마워요.
```

**考点**: 연체(5일/2000원), 대출 규정(최대 5권/2주)

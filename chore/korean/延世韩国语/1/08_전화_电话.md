## 三、核心语法

### 语法1: V-고 있다 — 进行时（"正在……"）★★★

表示动作正在进行中。

**构成**：动词词干 + **-고 있다**

```
지금 밥을 먹고 있어요.          现在正在吃饭。
친구를 기다리고 있어요.         正在等朋友。
한국어를 공부하고 있어요.       正在学韩语。
전화하고 있어요.               正在打电话。
책을 읽고 있어요.              正在读书。
음악을 듣고 있어요.            正在听音乐。
```

**时态变化**：

```
# 现在进行
먹고 있어요        正在吃

# 过去进行
먹고 있었어요      当时正在吃

# 将来进行（少用）
먹고 있을 거예요    将会在吃
```

> **穿戴状态**：V-고 있다 也用于描述穿着状态
>
> ```
> 빨간 옷을 입고 있어요.       穿着红色衣服。
> 안경을 쓰고 있어요.          戴着眼镜。
> 모자를 쓰고 있어요.          戴着帽子。
> ```
>
> 这种用法表示**状态持续**，不一定是"正在穿"的动作进行中。

---

### 语法2: V-(으)ㄹ 거예요 — 将来时/推测 ★★★

**两个用法**：

1. **计划/打算**（第一人称）："我会/我打算"
2. **推测**（第三人称/一般情况）："大概会"

**规则**：

- 元音/ㄹ结尾 → **-ㄹ 거예요**
- 辅音结尾 → **-을 거예요**

```
# 计划/打算 (我会…)
내일 부산에 갈 거예요.          明天打算去釜山。
주말에 쉴 거예요.              周末打算休息。
오후에 영화를 볼 거예요.        下午打算看电影。
저녁에 친구를 만날 거예요.      晚上打算见朋友。

# 推测 (大概…)
내일 비가 올 거예요.            明天大概会下雨。
수업이 곧 끝날 거예요.          课快要结束了吧。
민수 씨가 잘 할 거예요.         民秀大概能做好。
```

> **-(으)ㄹ 거예요 vs -(으)ㄹ게요 区分**：
>
> - **-ㄹ 거예요**：已有计划/推测 → "我打算…"/"大概会…"
> - **-ㄹ게요**：当场决定的意愿 → "那我就…"
> - 내일 갈 **거예요** (我已经打算好了，明天要去)
> - 그러면 내일 갈**게요** (那好，我明天去吧 → 当场决定)

---

### 语法3: V-아/어 주다 — "为（别人）做……"

表示为他人做某事，带有帮助/服务的语感。

```
# 해요체: V-아/어 줘요
사진을 찍어 줘요?              帮忙拍照好吗？
길을 알려 줘요.                告诉路（帮忙指路）。

# 请求: V-아/어 주세요 (请帮我…)
사진 좀 찍어 주세요.           请帮我拍张照。
천천히 말해 주세요.            请慢慢说。
다시 말해 주세요.              请再说一遍。
전화번호를 알려 주세요.        请告诉我电话号码。
문을 열어 주세요.              请帮我开门。
도와주세요!                   请帮帮我！
```

**自己为别人做的承诺**：V-아/어 줄게요

```
제가 도와줄게요.               我来帮你。
나중에 전화해 줄게요.          稍后我给你打电话。
내일 알려 줄게요.              明天告诉你。
```

---

### 语法4: V-(으)ㄹ까요? — "要不要……？" / "……吗？"

**两个用法**：

1. **提议/征求意见**（我们一起做）
2. **推测性疑问**（会不会/是不是）

```
# 提议
같이 갈까요?                  一起去吗？
점심 먹을까요?                吃午饭吗？
커피 마실까요?                喝杯咖啡吗？
창문을 열까요?                开窗户吗？

# 推测
내일 비가 올까요?             明天会下雨吗？
민수 씨가 올까요?             民秀会来吗？
```

---

### 语法5: -(으)면 — "如果……的话"

表示条件/假设：

**规则**：

- 元音/ㄹ结尾 → **-면**
- 辅音结尾 → **-으면**

```
시간이 있으면 전화해 주세요.     如果有时间请打电话。
날씨가 좋으면 산책할 거예요.     如果天气好就散步。
모르면 물어보세요.              如果不知道就问。
바쁘면 내일 만나요.             如果忙的话明天见。
전화를 안 받으면 문자를 보내세요. 如果不接电话就发短信。
```

## 四、核心会话

### 对话1：打电话找人

```
A: 여보세요.
   喂。

B: 여보세요. 거기 김민수 씨 좀 바꿔 주세요.
   喂。请让金民秀接电话。

A: 잠시만요. ... 지금 전화를 못 받아요. 샤워하고 있어요.
   请稍等。……现在不能接。正在洗澡。

B: 아, 그래요? 그러면 나중에 다시 걸게요.
   啊，是吗？那我稍后再打。

A: 네, 메시지 남기시겠어요?
   好的，要留言吗？

B: 아니요, 괜찮아요. 나중에 전화할게요. 감사합니다.
   不用了。稍后我再打电话。谢谢。
```

### 对话2：电话约见面

```
A: 여보세요. 영희 씨, 저 왕웨이예요.
   喂。英熙，我是王伟。

B: 아, 왕웨이 씨! 잘 지내요?
   啊，王伟！过得好吗？

A: 네, 잘 지내요. 이번 주말에 시간 있어요?
   是的，挺好的。这周末有时间吗？

B: 토요일은 좀 바빠요. 일요일은 괜찮아요.
   周六有点忙。周日没问题。

A: 그러면 일요일에 같이 점심 먹을까요?
   那周日一起吃午饭吗？

B: 좋아요! 몇 시에 만날까요?
   好的！几点见？

A: 열두 시에 강남역에서 만나요.
   12点在江南站见。

B: 네, 그때 봐요!
   好的，到时见！
```

**新词**：

- **잘 지내다** = 过得好
- **이번** = 这次
- **그때** = 那时候
- **봐요** = 见吧（보다 → 봐요）

### 对话3：电话预约

```
A: 여보세요. ○○ 병원이에요?
   喂。是○○医院吗？

B: 네, 맞습니다. 무엇을 도와 드릴까요?
   是的，没错。有什么可以帮您？

A: 예약하려고 하는데요. 이번 주 금요일에 가능해요?
   想预约。这周五可以吗？

B: 금요일 오전에는 자리가 없어요. 오후 세 시는 어떠세요?
   周五上午没有位置了。下午3点怎么样？

A: 네, 괜찮아요. 세 시에 갈게요.
   好的，可以。3点去。

B: 성함이 어떻게 되세요?
   请问贵姓？

A: 왕웨이입니다.
   王伟。

B: 네, 왕웨이 님 금요일 오후 세 시에 예약되었습니다.
   好的，王伟先生周五下午3点已预约。
```

**新词**：

- **병원** = 医院
- **무엇** = 什么 (뭐的正式形)
- **드리다** = 给（谦逊语）
- **예약하다** = 预约
- **자리** = 位子/名额
- **성함** = 姓名（尊敬语）
- **어떻게 되세요** = 是什么呢（礼貌询问）

## 五、电话号码读法

韩语读电话号码用**汉字数词**，数字之间用"의 (에)"或直接连读：

```
010-1234-5678
→ 공일공 에 일이삼사 에 오육칠팔

02-123-4567
→ 공이 에 일이삼 에 사오육칠
```

> **0 的读法**：电话号码中的 0 读 **공** (gong)，不读 영

询问电话号码：

```
전화번호가 뭐예요?             电话号码是多少？
전화번호 좀 알려 주세요.        请告诉我电话号码。
핸드폰 번호가 어떻게 돼요?      手机号是多少？
```

### 练习2：-(으)ㄹ 거예요 选择合适的

3. 주말에 영화를 **\_\_\_\_**. (打算看) → ******\_\_\_******

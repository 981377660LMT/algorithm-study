## 三、核心语法

### 语法1: N(으)로 — 交通方式（"乘坐/用"）

**(으)로** 表示手段/方式/方向：

```
# 交通方式
버스로 가요.              坐公交车去。
지하철로 출근해요.         坐地铁上班。
택시로 갈까요?            坐出租车去吗？
걸어서 가요.              走路去。(注意: 步行用 걸어서，不用 (으)로)

# 工具/手段 (复习)
카드로 계산해요.          用卡结账。
한국어로 말해요.          用韩语说。
```

---

### 语法2: V-(으)ㄹ 수 있다/없다 — "能/不能" ★★★

表示能力或可能性。

**规则**：

- 元音/ㄹ结尾 → **-ㄹ 수 있다/없다**
- 辅音结尾 → **-을 수 있다/없다**

```
# 能力
한국어를 할 수 있어요.           能说韩语。
자전거를 탈 수 있어요.           能骑自行车。
运전할 수 있어요?               能开车吗？

# 可能性
내일 갈 수 있어요.              明天能去。
여기에서 환승할 수 있어요.       在这里能换乘。

# 否定 (不能)
수영할 수 없어요.               不能游泳。
= 수영 못 해요.                 不会游泳。(口语简化)
오늘 갈 수 없어요.              今天不能去。
= 오늘 못 가요.                 今天去不了。
```

> **잘 vs 못 对比**：
>
> - 한국어를 **잘** 해요 = 韩语说得**好**
> - 한국어를 **못** 해요 = 韩语说**不了/不好**
> - 못 放在动词前面，是 -ㄹ 수 없다 的口语简化

---

### 语法3: 얼마나 걸리다 — "花多长时间"

```
얼마나 걸려요?                  花多长时间？

삼십 분쯤 걸려요.               大约花30分钟。
한 시간 걸려요.                 花1小时。
한 시간 반 걸려요.              花1个半小时。
두 시간쯤 걸려요.               大约花2小时。

여기에서 학교까지 얼마나 걸려요?   从这里到学校要多久？
지하철로 삼십 분 걸려요.          坐地铁30分钟。
```

> **시간 (小时) vs 시 (点钟)**：
>
> - 한 **시간** = 1小时（持续时间）
> - 한 **시** = 1点钟（时刻）

---

### 语法4: V-(으)려고 하다 — "打算……"

表示意图、计划：

**规则**：

- 元音/ㄹ结尾 → **-려고 하다**
- 辅音结尾 → **-으려고 하다**

```
내일 부산에 가려고 해요.         打算明天去釜山。
택시를 타려고 해요.              打算坐出租车。
한국어를 배우려고 해요.          打算学韩语。
점심을 먹으려고 해요.            打算吃午饭。
```

---

### 语法5: V-아/어서 — "因为……所以……" / "……然后……"

**-아/어서** 有两个重要用法：

#### 用法1：原因（"因为"）

```
시간이 없어서 택시를 탔어요.       因为没时间所以打了出租车。
길이 복잡해서 늦었어요.           因为路堵所以迟到了。
비가 와서 못 갔어요.              因为下雨所以没去成。
```

#### 用法2：顺序动作（"然后"）→ 限于移动/前后动作

```
버스를 타서 학교에 가요.          坐公交然后去学校。
3호선에서 내려서 환승해요.         在3号线下来然后换乘。
직진해서 좌회전하세요.            直走然后左转。
```

> **-아/어서 vs -고 区别**：
>
> - **-고**：单纯并列两个动作（可交换顺序）
> - **-아/어서**：有因果关系或必须按顺序的动作
> - 밥을 먹**고** 커피를 마셔요 (吃饭然后喝咖啡 → 可换序)
> - 버스를 타**서** 학교에 가요 (坐公交去学校 → 因果/必须先坐车)

> **重要**：-아/어서 表示原因时，后面**不能用命令/共动/过去时语尾**
>
> - 비가 와서 집에 있었어요 ✓ (因为下雨待在了家里)
> - 비가 왔어서 ✗ (아/어서前面不变为过去时)

## 四、核心会话

### 对话1：问路

```
A: 실례합니다. 명동에 어떻게 가요?
   打扰一下。怎么去明洞？

B: 여기에서 2호선을 타세요. 을지로입구역에서 내리세요.
   从这里坐2号线。在乙支路入口站下。

A: 몇 정거장이에요?
   几站？

B: 세 정거장이에요.
   三站。

A: 얼마나 걸려요?
   要多长时间？

B: 십 분쯤 걸려요.
   大约10分钟。

A: 감사합니다.
   谢谢。
```

**新词**：

- **어떻게** = 怎样
- **명동** = 明洞（首尔商业区）
- **정거장** = 站（计数用）
- **쯤** = 大约

### 对话2：坐出租车

```
A: 기사님, 서울역으로 가 주세요.
   司机，请去首尔站。

B: 네, 알겠습니다. 어느 길로 갈까요?
   好的。走哪条路？

A: 빠른 길로 가 주세요.
   请走快的路。

B: 네, 지금 좀 막혀서 삼십 분쯤 걸릴 거예요.
   好的，现在有点堵，大概要30分钟。

A: 괜찮아요.
   没关系。

(到达后)

B: 다 왔습니다. 만 이천 원이에요.
   到了。12,000韩元。

A: 카드로 해도 돼요?
   可以刷卡吗？

B: 네, 됩니다.
   可以。
```

**新词**：

- **기사님** = 司机（尊称）
- **-으로** = 向/往
- **-아/어 주세요** = 请帮我做（请求）
- **막히다** = 堵车
- **-ㄹ 거예요** = 将会（推测/将来时，后续课详讲）
- **다** = 都/全部

### 对话3：比较交通方式

```
A: 강남까지 어떻게 가요?
   怎么去江南？

B: 버스나 지하철로 갈 수 있어요.
   可以坐公交或地铁。

A: 뭐가 더 빨라요?
   哪个更快？

B: 지하철이 더 빨라요. 이십 분쯤 걸려요.
   地铁更快。大约20分钟。

A: 버스는요?
   公交呢？

B: 버스는 삼십 분에서 사십 분쯤 걸려요. 길이 막히면 더 걸려요.
   公交大约30到40分钟。堵车的话更久。

A: 그러면 지하철로 갈게요.
   那我坐地铁去。
```

**新词**：

- **N이나** = N或者
- **빠르다** = 快 → 빨라요 (ㄹ르不规则)
- **-(으)면** = 如果/的话（条件）

## 五、首尔交通实用知识

### 1. 交通卡

- **T-money (티머니)**: 韩国最常用的交通卡
- 便利店可以购买和充值
- 公交、地铁、出租车通用
- 30分钟内换乘免费

### 2. 首尔地铁

- 共9条主要线路 + 多条支线
- 1号线(蓝)、2号线(绿)、3号线(橙)、4号线(天蓝) …
- **주요 역 (主要站)**:
  - 서울역 (首尔站): 1号线、4号线、KTX
  - 강남역 (江南站): 2号线
  - 명동역 (明洞站): 4号线
  - 홍대입구역 (弘大站): 2号线，年轻人聚集地

### 3. 实用路上表达

| 韩文                     | 中文                 |
| ------------------------ | -------------------- |
| 여기에서 내려 주세요     | 请在这里停（出租车） |
| 다음 정류장에서 내릴게요 | 我在下一站下         |
| 이 버스 명동에 가요?     | 这趟公交去明洞吗？   |
| 환승이 필요해요?         | 需要换乘吗？         |
| 길을 잃었어요            | 我迷路了             |
| 지도 좀 보여 주세요      | 请给我看看地图       |

## 三、核心语法

### 语法1: V-(으)ㄹ래요 — "想要做……"

表达说话人的**意愿/想法**（比 -고 싶다 更口语化）：

**规则**：

- 元音/ㄹ结尾 → **-ㄹ래요**
- 辅音结尾 → **-을래요**

```
# 陈述（我想要…）
저는 커피를 마실래요.           我想喝咖啡。
집에서 쉴래요.                 我想在家休息。
여행을 갈래요.                 我想去旅行。

# 疑问（你想要…吗？）
뭐 먹을래요?                  你想吃什么？
같이 갈래요?                  你想一起去吗？
영화 볼래요?                  想看电影吗？
```

> **-(으)ㄹ래요 vs -고 싶다**：
>
> - **-고 싶다**：单纯表达希望/想法 → "我想…"
> - **-(으)ㄹ래요**：倾向于付诸行动的意愿 → "我要…/想做"
> - **-(으)ㄹ래요?**：向对方提议 → "你要…吗？"
> - 한국에 가고 싶어요 (我想去韩国 → 朦胧的愿望)
> - 한국에 갈래요 (我要去韩国 → 更坚定/具体的意愿)

---

### 语法2: V-(으)러 가다/오다 — "去做……/来做……"

表示**移动的目的**：

**规则**：

- 元音/ㄹ结尾 → **-러**
- 辅音结尾 → **-으러**

```
밥 먹으러 가요.               去吃饭。
커피 마시러 카페에 가요.        去咖啡厅喝咖啡。
친구를 만나러 홍대에 가요.      去弘大见朋友。
수영하러 수영장에 가요.        去游泳池游泳。
한국어를 배우러 한국에 왔어요.   来韩国学韩语的。
구경하러 갈까요?               去参观吧？
```

> **注意**：-(으)러 后面只能接 **가다/오다** 等移动动词！
>
> - 친구를 만나러 **가요** ✓
> - 친구를 만나러 **해요** ✗

---

### 语法3: V-아/어야 되다 (하다) — "必须……"

表示义务/必要性：

```
내일 일찍 일어나야 돼요.        明天必须早起。
숙제를 해야 돼요.              必须做作业。
여권을 가져가야 돼요.           必须带护照。
약을 먹어야 돼요.              必须吃药。
비행기 표를 예약해야 돼요.       必须预订机票。
짐을 싸야 돼요.               必须收拾行李。
```

**否定**："不必/不用"

```
안 해도 돼요.                  不用做。
일찍 안 와도 돼요.             不用早来。
걱정 안 해도 돼요.             不用担心。
```

> **-아/어야 되다 = -아/어야 하다**（两者意思相同）
>
> - 가야 돼요 = 가야 해요（必须去）

---

### 语法4: V-고 싶다 — "想要……"（复习+扩展）

```
# 基本用法
여행하고 싶어요.               想旅行。
바다에 가고 싶어요.            想去海边。
맛있는 것을 먹고 싶어요.        想吃好吃的。

# 过去式：想了/曾经想
어제 영화를 보고 싶었어요.      昨天想看电影。

# 第三人称用 -고 싶어 하다
동생이 아이스크림을 먹고 싶어 해요.  弟弟想吃冰淇淋。
```

> **-고 싶다 vs -고 싶어 하다**：
>
> - 第一人称用 **-고 싶다**：저는 가고 싶어요
> - 第三人称用 **-고 싶어 하다**：그 사람은 가고 싶어 해요

---

### 语法5: V-는 것 — 动词名词化（"做……这件事"）

将动词变成名词使用：

```
여행하는 것을 좋아해요.         喜欢旅行。
= 여행하는 것 좋아해요.
= 여행하는 걸 좋아해요. (口语缩略)

노래 부르는 것이 취미예요.      唱歌是爱好。
한국 드라마를 보는 것이 재미있어요.  看韩剧很有趣。
```

> **것의 缩略**：
>
> - 것을 → 걸
> - 것이 → 게
> - 것은 → 건

## 四、核心会话

### 对话1：假期计划

```
A: 이번 방학에 뭐 할 거예요?
   这次放假打算做什么？

B: 제주도에 여행 갈 거예요.
   打算去济州岛旅行。

A: 좋겠다! 누구하고 같이 가요?
   好好啊！和谁一起去？

B: 친구 두 명하고 같이 갈 거예요.
   和两个朋友一起去。

A: 며칠 동안 있을 거예요?
   待几天？

B: 삼 일 동안 있을 거예요. 바다에서 수영도 하고 맛집도 갈 거예요.
   待三天。要在海里游泳，也去美食店。

A: 재미있겠어요. 사진 많이 찍어요!
   一定很有趣。多拍照！
```

**新词**：

- **며칠** = 几天
- **동안** = 期间/之间
- **맛집** = 美食店

### 对话2：邀请朋友

```
A: 이번 주말에 시간 있어요?
   这周末有时间吗？

B: 네, 토요일은 괜찮아요.
   有，周六没问题。

A: 그러면 같이 등산하러 갈래요?
   那一起去爬山怎么样？

B: 좋아요! 어디로 갈 거예요?
   好啊！去哪里？

A: 북한산에 가려고 해요. 단풍이 예쁠 거예요.
   打算去北汉山。红叶应该很漂亮。

B: 좋아요. 몇 시에 만날까요?
   好。几点见？

A: 아침 아홉 시에 만나요. 도시락을 싸 가야 돼요.
   早上9点见。要带便当。

B: 알겠어요. 그러면 토요일에 봐요!
   知道了。那周六见！
```

**新词**：

- **북한산** = 北汉山（首尔著名登山地）
- **도시락** = 便当
- **싸 가다** = 打包带走

### 对话3：回忆假期

```
A: 추석 잘 보냈어요?
   中秋节过得好吗？

B: 네, 고향에 가서 가족을 만났어요.
   是的，回老家见了家人。

A: 뭐 했어요?
   做了什么？

B: 송편을 만들고 성묘도 갔어요.
   그리고 가족들하고 같이 맛있는 음식을 많이 먹었어요.
   做了松糕，也去了扫墓。
   还和家人一起吃了很多好吃的。

A: 좋았겠어요. 저는 친구들하고 여행을 갔어요.
   肯定很好。我和朋友们去旅行了。

B: 어디에 갔어요?
   去了哪里？

A: 부산에 갔어요. 바다가 정말 아름다웠어요.
   去了釜山。海真的很美。
```

**新词**：

- **잘 보내다** = 好好度过
- **고향** = 故乡/老家
- **송편** = 松糕（中秋传统食物）
- **성묘** = 扫墓
- **-들** = ～们（复数标记）

## 五、韩国传统节日文化

### 설날 (春节)

- 农历1月1日，连休3天
- **세배 (拜年)**：向长辈磕头行礼
- **세뱃돈 (压岁钱)**：长辈给晚辈钱
- **떡국 (年糕汤)**：必吃食物，吃了长一岁
- **윷놀이 (投骰游戏)**：传统家庭游戏

### 추석 (中秋节)

- 农历8月15日，连休3天
- **차례 (祭祀)**：祭拜祖先
- **성묘 (扫墓)**：去扫墓
- **송편 (松糕)**：中秋特色食物
- **민족 대이동 (民族大移动)**：全国返乡大潮，类似中国春运

## 八、本课核心句型小结

| 句型               | 用法       | 例句               |
| ------------------ | ---------- | ------------------ |
| V-(으)ㄹ래요       | 想要/提议  | 갈래요?, 먹을래요  |
| V-(으)러 가다/오다 | 去做/来做  | 먹으러 가요        |
| V-아/어야 되다     | 必须       | 해야 돼요          |
| V-고 싶다          | 想要       | 가고 싶어요        |
| V-는 것            | 动词名词化 | 먹는 것을 좋아해요 |

## 三、核心语法

### 语法1: 不规则变化总结（ㅂ / ㄷ / ㅎ 不规则）★★★

这是初学阶段最容易出错的部分，必须牢记。

#### ㅂ 不规则 (ㅂ불규칙)

词干以 ㅂ 结尾的部分形容词/动词，遇到元音语尾时 ㅂ → 우/오

**规则**：ㅂ 脱落 → 加 **우** (极少数加 오)

| 基本形        | 해요体         | -(으)ㄴ 修饰形  | 过去式           |
| ------------- | -------------- | --------------- | ---------------- |
| 덥다 (热)     | 더**워**요     | 더**운** 날     | 더**웠**어요     |
| 춥다 (冷)     | 추**워**요     | 추**운** 날     | 추**웠**어요     |
| 쉽다 (容易)   | 쉬**워**요     | 쉬**운** 일     | 쉬**웠**어요     |
| 어렵다 (难)   | 어려**워**요   | 어려**운** 문제 | 어려**웠**어요   |
| 맵다 (辣)     | 매**워**요     | 매**운** 음식   | 매**웠**어요     |
| 가깝다 (近)   | 가까**워**요   | 가까**운** 곳   | 가까**웠**어요   |
| 아름답다 (美) | 아름다**워**요 | 아름다**운** 꽃 | 아름다**웠**어요 |
| 고맙다 (感谢) | 고마**워**요   | —               | 고마**웠**어요   |
| 돕다 (帮助)   | 도**와**요     | —               | 도**왔**어요     |

> **不是所有 ㅂ 结尾都不规则！**
> **规则变化的 ㅂ 词**：입다(穿), 잡다(抓), 좁다(窄), 씹다(嚼) → 이들은 正常: 입어요, 잡아요 等

#### ㄷ 不规则 (ㄷ불규칙)

词干以 ㄷ 结尾的部分动词，遇到元音语尾时 ㄷ → ㄹ

| 基本形      | 해요体 | 说明    |
| ----------- | ------ | ------- |
| 듣다 (听)   | 들어요 | ㄷ → ㄹ |
| 걷다 (走)   | 걸어요 | ㄷ → ㄹ |
| 묻다 (问)   | 물어요 | ㄷ → ㄹ |
| 싣다 (装载) | 실어요 | ㄷ → ㄹ |

> **规则变化的 ㄷ 词**：닫다(关), 받다(接), 믿다(相信) → 닫아요, 받아요, 믿어요

#### ㅎ 不规则 (ㅎ불규칙)

词干以 ㅎ 结尾的部分形容词，遇到元音语尾时 ㅎ 脱落，并与元音合并。

| 基本形        | 해요体 | -(으)ㄴ 修饰形  |
| ------------- | ------ | --------------- |
| 어떻다 (怎样) | 어때요 | 어떤 (什么样的) |
| 그렇다 (那样) | 그래요 | 그런 (那样的)   |
| 이렇다 (这样) | 이래요 | 이런 (这样的)   |
| 빨갛다 (红)   | 빨개요 | 빨간            |
| 파랗다 (蓝)   | 파래요 | 파란            |
| 노랗다 (黄)   | 노래요 | 노란            |
| 하얗다 (白)   | 하얘요 | 하얀            |
| 까맣다 (黑)   | 까매요 | 까만            |

#### ㄹ 不规则 (ㄹ특수)

词干以 ㄹ 结尾时，遇到 ㄴ/ㅂ/ㅅ 开头的语尾，ㄹ 脱落：

| 基本形      | -(으)ㄴ | -(으)ㅂ니다 | -(으)세요 |
| ----------- | ------- | ----------- | --------- |
| 살다 (住)   | 사는    | 삽니다      | 사세요    |
| 길다 (长)   | 긴      | 깁니다      | —         |
| 알다 (知道) | 아는    | 압니다      | 아세요    |
| 만들다 (做) | 만드는  | 만듭니다    | 만드세요  |

### 语法2: -겠- — 推测/意愿

#### 用法1：推测（"好像会……"/"大概……"）

```
내일 비가 오겠어요.         明天好像会下雨。
날씨가 좋겠어요.           天气应该会好。
많이 춥겠어요.             好像会很冷。
맛있겠다!                 应该很好吃！（感叹）
```

#### 用法2：意愿（第一人称，"我要……"）

```
잘 먹겠습니다.             我开动了。(固定表达)
알겠습니다.               我明白了。
내일 가겠습니다.           明天我会去的。(比较正式)
```

> **-겠- vs -(으)ㄹ 거예요**：
>
> - **-겠-**：根据当下感受/证据的推测，或正式的意愿表达
> - **-(으)ㄹ 거예요**：基于已知信息的推测/已有的计划
> - 하늘이 흐려요. 비가 오**겠**어요. (看天阴了，感觉要下雨)
> - 일기 예보에서 비가 올 **거**래요. (天气预报说会下雨)

---

### 语法3: -(으)니까 — "因为……所以……"

比 -아/어서 更强调主观理由，后面**可以接命令和共动句**。

**规则**：

- 元音/ㄹ结尾 → **-니까**
- 辅音结尾 → **-으니까**

```
날씨가 좋으니까 산책하러 가요.      因为天气好，去散步吧。
비가 오니까 우산을 가져가세요.        因为下雨，带把伞吧。
추우니까 따뜻하게 입으세요.          因为冷，穿暖和点吧。
시간이 없으니까 택시를 타요.          因为没时间，坐出租车吧。
```

> **-아/어서 vs -(으)니까 区别**：
> | | -아/어서 | -(으)니까 |
> |---|---------|---------|
> | 后接命令/共动 | ✗ 不可以 | ✓ 可以 |
> | 过去时附着 | ✗ 不可以 | ✓ 可以 |
> | 语感 | 客观原因 | 主观强调理由 |
>
> - 비가 와서 못 가요 ✓（因为下雨所以去不了 → 陈述）
> - 비가 와서 가지 마세요 ✗（错误！-아/어서后面不能接命令）
> - 비가 오니까 가지 마세요 ✓（因为下雨所以别去 → 命令OK）

---

### 语法4: V-지만 — "虽然……但是……"

表示转折：

```
날씨가 춥지만 산책할 거예요.        虽然天气冷，但会去散步。
비가 오지만 서울역가야 돼요.        虽然下雨，但必须去首尔站。
여름은 덥지만 좋아해요.            夏天虽然热，但我喜欢。
한국어가 어렵지만 재미있어요.       韩语虽然难，但很有趣。
```

## 四、核心会话

### 对话1：谈论天气

```
A: 오늘 날씨가 어때요?
   今天天气怎么样？

B: 오늘은 정말 더워요. 삼십오 도예요.
   今天真的很热。35度。

A: 정말요? 어제는 시원했는데...
   真的吗？昨天还挺凉快的……

B: 네, 여름이니까 갑자기 더워졌어요.
   是啊，因为是夏天嘛，突然变热了。

A: 에어컨을 켜야겠어요.
   得开空调了。
```

**新词**：

- **도** = 度
- **-았/었는데** = 过去的转折（当时是…但…）
- **갑자기** = 突然
- **-아/어지다** = 变得（더워지다 = 变热）
- **켜다** = 打开（电器）
- **-아/어야겠다** = 应该要…（必要性）

### 对话2：最喜欢的季节

```
A: 어느 계절을 제일 좋아해요?
   最喜欢哪个季节？

B: 저는 가을을 제일 좋아해요.
   我最喜欢秋天。

A: 왜요?
   为什么？

B: 날씨가 시원하고 하늘이 맑아요.
   그리고 단풍이 정말 아름다워요.
   天气凉爽，天空晴朗。而且红叶真的很美。

A: 맞아요. 가을은 등산하기 좋은 계절이에요.
   对。秋天是适合登山的季节。

B: 왕웨이 씨는 어떤 계절을 좋아해요?
   王伟你喜欢什么季节？

A: 저는 봄을 좋아해요. 꽃이 예쁘고 따뜻하니까요.
   我喜欢春天。因为花漂亮又暖和。
```

### 对话3：韩国四季

```
A: 한국의 날씨는 어때요?
   韩国的天气怎么样？

B: 한국은 사계절이 뚜렷해요.
   韩国四季分明。

A: 겨울에 많이 추워요?
   冬天很冷吗？

B: 네, 영하 십 도까지 내려가요.
   눈도 많이 와요.
   是的，降到零下10度。也下很多雪。

A: 여름은요?
   夏天呢？

B: 여름은 덥고 습해요.
   유월에서 칠월까지 장마가 있어요. 비가 많이 와요.
   夏天又热又潮。6月到7月有梅雨季。下很多雨。

A: 중국 남쪽이랑 비슷하네요.
   和中国南方很像呢。
```

**新词**：

- **사계절** = 四季
- **뚜렷하다** = 明显/分明
- **영하** = 零下
- **내려가다** = 下降
- **장마** = 梅雨（季）
- **비슷하다** = 相似

## 五、韩国四季文化

### 봄 (春天: 3-5月)

- **벚꽃 축제 (樱花节)**：4月，全国各地赏樱
- **여의도** (汝矣岛)、**경주** (庆州)、**진해** (镇海) 是赏樱名地
- 气温：5-20°C，早晚温差大

### 여름 (夏天: 6-8月)

- **장마 (梅雨)**：6月底-7月中，连续降雨
- **바캉스 (度假)**：8月是海边度假季
- **삼복 (三伏)**：吃 삼계탕(参鸡汤) 补身体
- 气温：25-35°C，高温潮湿

### 가을 (秋天: 9-11月)

- **단풍 (红叶)**：10月中旬-11月，设岳山、雪岳山等赏枫
- **추석 (中秋节)**：韩国最重要的传统节日之一
- 气温：10-20°C，天气凉爽干燥
- 被称为"天高马肥"的好季节

### 겨울 (冬天: 12-2月)

- **크리스마스 (圣诞节)**：韩国是公休日
- **설날 (春节/元旦)**：农历正月初一
- **스키장 (滑雪场)**：全国各地，12月-3月营业
- 气温：-15 ~ 5°C，干冷

### 练习1：ㅂ 不规则变化

1. 덥다 → ******\_\_\_******
2. 춥다 → ******\_\_\_******
3. 가깝다 → ******\_\_\_******
4. 어렵다 → ******\_\_\_******
5. 아름답다 → ******\_\_\_******

<details>
<summary>点击查看答案</summary>

1. 더워요
2. 추워요
3. 가까워요
4. 어려워요
5. 아름다워요

</details>

### 练习2：选择 -아/어서 还是 -(으)니까

1. 비가 **\_\_\_** 우산을 가져가세요. (因为下雨，请带伞) → **\_\_**
2. 피곤해**\_\_\_** 일찍 잤어요. (因为累，早睡了) → **\_\_**
3. 추**\_\_\_** 따뜻하게 입으세요. (因为冷，穿暖和点) → **\_\_**

<details>
<summary>点击查看答案</summary>

1. 비가 **오니까** 우산을 가져가세요. (后接命令 → 用 -(으)니까)
2. 피곤해**서** 일찍 잤어요. (后接陈述 → 都可以，-아/어서 更自然)
3. 추**우니까** 따뜻하게 입으세요. (后接命令 → 用 -(으)니까)

</details>

### 练习3：用 -지만 连接

1. 겨울은 춥다 / 스키를 탈 수 있다 → ******\_\_\_******
2. 한국어가 어렵다 / 재미있다 → ******\_\_\_******
3. 오늘 비가 오다 / 내일은 맑다 → ******\_\_\_******

<details>
<summary>点击查看答案</summary>

1. 겨울은 춥지만 스키를 탈 수 있어요.
2. 한국어가 어렵지만 재미있어요.
3. 오늘 비가 오지만 내일은 맑을 거예요.

</details>

### 练习4：描述你最喜欢的季节

用以下模板写3-4句话：

```
저는 ___을/를 제일 좋아해요.
날씨가 ___고 ___.
___을/를 할 수 있으니까요.
그리고 ___도 ___아/어요.
```

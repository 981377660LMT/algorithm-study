# 준비 (준비) — 韩文字母 한글 入门

## 一、한글 简介

한글（韩文字母）是朝鲜王朝第四代国王世宗大王（세종대왕）于1443年发明的表音文字，原名"训民正音"（훈민정음）。

**核心特点**：

- **表音文字**：每个字母对应一个发音，看到就能读
- **音节块**（syllable block）：辅音和元音组合成方块字
- **共40个字母**：21个元音 + 19个辅音

## 二、基本元音 (모음) — 21个

### 1. 单元音 (단모음) — 10个

| 字母 | 发音 | 中文近似音   | 口型说明                       |
| ---- | ---- | ------------ | ------------------------------ |
| ㅏ   | a    | "啊"         | 嘴巴张大，舌头放平             |
| ㅓ   | eo   | "哦"（偏暗） | 嘴巴半开，比ㅏ小，舌头稍后缩   |
| ㅗ   | o    | "哦"（嘴圆） | 嘴唇圆起，嘴巴小               |
| ㅜ   | u    | "乌"         | 嘴唇圆起前突，开口最小         |
| ㅡ   | eu   | 无对应       | 嘴唇扁平横拉，不圆唇，类似微笑 |
| ㅣ   | i    | "衣"         | 嘴唇扁平，嘴角往两边拉         |
| ㅐ   | ae   | "哎"         | 嘴巴比ㅏ小，舌头稍抬           |
| ㅔ   | e    | "诶"         | 嘴巴比ㅐ更小                   |
| ㅚ   | oe   | 类似"웨"     | 现代发音趋同于ㅙ/ㅞ            |
| ㅟ   | wi   | "위"         | 发ㅜ的口型发ㅣ的音             |

> **学习提示**：ㅐ和ㅔ在现代韩语中发音几乎相同，都接近英语的 /e/。

## 三、辅音 (자음) — 19个

### 1. 基本辅音 (기본 자음) — 9个

| 字母 | 名称 | 发音（初声） | 发音（终声/收音） | 中文近似             |
| ---- | ---- | ------------ | ----------------- | -------------------- |
| ㄱ   | 기역 | g/k          | k                 | 介于"g"和"k"之间     |
| ㄴ   | 니은 | n            | n                 | "n"                  |
| ㄷ   | 디귿 | d/t          | t                 | 介于"d"和"t"之间     |
| ㄹ   | 리을 | r/l          | l                 | 初声弹舌"r"，终声"l" |
| ㅁ   | 미음 | m            | m                 | "m"                  |
| ㅂ   | 비읍 | b/p          | p                 | 介于"b"和"p"之间     |
| ㅅ   | 시옷 | s            | t                 | "s"                  |
| ㅇ   | 이응 | 无音(占位)   | ng                | 初声无音，终声"ng"   |
| ㅈ   | 지읒 | j            | t                 | 介于"j"和"ch"之间    |

> **重点理解**：韩语的 ㄱ/ㄷ/ㅂ/ㅈ 是**松音**（平音），发音时**不送气**，中国人容易听成浊音 g/d/b/j，但实际是不送气的清音。

### 2. 送气音 (격음/激音) — 4个

| 字母 | 名称 | 发音 | 说明                 |
| ---- | ---- | ---- | -------------------- |
| ㅋ   | 키읔 | k    | 强送气，纸片能被吹动 |
| ㅌ   | 티읕 | t    | 强送气               |
| ㅍ   | 피읖 | p    | 强送气               |
| ㅊ   | 치읓 | ch   | 强送气               |
| ㅎ   | 히읗 | h    | 气音，类似"h"        |

### 3. 紧音 / 双辅音 (쌍자음/硬音) — 5个

| 字母 | 名称   | 发音 | 说明                       |
| ---- | ------ | ---- | -------------------------- |
| ㄲ   | 쌍기역 | kk   | 喉头紧绷，不送气，声调略高 |
| ㄸ   | 쌍디귿 | tt   | 喉头紧绷，不送气           |
| ㅃ   | 쌍비읍 | pp   | 喉头紧绷，不送气           |
| ㅆ   | 쌍시옷 | ss   | 喉头紧绷                   |
| ㅉ   | 쌍지읒 | jj   | 喉头紧绷，不送气           |

> **松音 vs 送气音 vs 紧音**（以 ㄱ 系列为例）：
>
> - ㄱ (松音)：不送气，声带放松 → "가" (ga)
> - ㅋ (送气音)：强送气，大量气流 → "카" (ka)
> - ㄲ (紧音)：不送气，喉头紧绷 → "까" (kka)

## 四、音节结构 — 韩文字的拼写方式

韩文每个字都是一个**音节块**，由辅音(C)和元音(V)组成：

```
结构1: C + V          (无收音)    例: 가 = ㄱ + ㅏ
结构2: C + V + C      (有收音)    例: 한 = ㅎ + ㅏ + ㄴ
结构3: C + V + CC     (双收音)    例: 읽 = ㅇ + ㅣ + ㄹㄱ
```

**图解音节组合方式**：

```
┌───┐       ┌───┐       ┌───┐
│ C │       │C│V│       │C│V│
│ V │       ├─┴─┤       ├─┴─┤
└───┘       │ C │       │C C│
 无收音      单收音      双收音
(元音竖写)  (元音竖写)

┌───┐       ┌───┐
│ C │       │ C │
├───┤       ├───┤
│ V │       │ V │
└───┘       ├───┤
 无收音      │ C │
(元音横写)   └───┘
            单收音
           (元音横写)
```

**规则**：

- 竖型元音 (ㅏ, ㅓ, ㅣ等) → 辅音在左，元音在右
- 横型元音 (ㅗ, ㅜ, ㅡ等) → 辅音在上，元音在下
- 复合元音 (ㅘ, ㅝ等) → 辅音在左上角

## 五、收音 (받침) 规则

韩语的收音（终声）只有 **7种实际发音**：

| 代表音  | 发音         | 对应的收音字母             |
| ------- | ------------ | -------------------------- |
| ㄱ [k]  | 嘴闭合不释放 | ㄱ, ㅋ, ㄲ, ㄳ, ㄺ         |
| ㄴ [n]  | 舌尖抵上齿龈 | ㄴ, ㄵ, ㄶ                 |
| ㄷ [t]  | 舌尖抵上齿龈 | ㄷ, ㅌ, ㅅ, ㅆ, ㅈ, ㅊ, ㅎ |
| ㄹ [l]  | 舌尖抵上齿龈 | ㄹ, ㄼ, ㄽ, ㄾ, ㅀ         |
| ㅁ [m]  | 嘴唇闭合     | ㅁ, ㄻ                     |
| ㅂ [p]  | 嘴唇闭合     | ㅂ, ㅍ, ㄿ, ㅄ             |
| ㅇ [ng] | 鼻腔送气     | ㅇ                         |

> **中国人学习提示**：收音类似于中文拼音的韵尾（如 -n, -ng），但韩语还有 -k, -t, -p, -l 四种"阻塞音"收音，发音时气流被阻断但**不释放**（不用爆破）。

## 六、重要音变规则（初学须知）

### 1. 连音 (연음/连读)

当前一个字有收音，后一个字以 ㅇ 开头时，收音移到后面作为初声发音。

```
한국어 → [한·구·거]    (ㄱ移到어前)
음악   → [으·막]       (ㄱ移到악的ㅇ位)
```

### 2. 鼻音化 (비음화)

收音 ㄱ/ㄷ/ㅂ 后接 ㄴ/ㅁ 时，变为对应鼻音。

```
ㄱ → ㅇ :  학년 → [항년]
ㄷ → ㄴ :  걷는 → [건는]
ㅂ → ㅁ :  입니다 → [임니다]
```

### 3. 送气音化 (격음화)

收音 ㄱ/ㄷ/ㅂ/ㅈ + ㅎ（或 ㅎ + ㄱ/ㄷ/ㅂ/ㅈ）合并为送气音。

```
축하 → [추카]     (ㄱ + ㅎ → ㅋ)
좋다 → [조타]     (ㅎ + ㄷ → ㅌ)
```

### 4. 硬音化 (경음화)

收音 ㄱ/ㄷ/ㅂ 后接松音 ㄱ/ㄷ/ㅂ/ㅅ/ㅈ 时，后面变为紧音。

```
학교 → [학꾜]     (ㄱ + ㄱ → ㄲ)
읽다 → [익따]     (ㄱ + ㄷ → ㄸ)
```

## 九、한글 书写练习建议

1. **元音笔顺**：先画竖线/横线，再画短横/短竖
2. **辅音笔顺**：大多从左到右、从上到下
3. **每天练习**：每个字母写一行（约10遍），持续一周即可熟练
4. **推荐App**：
   - Write It! Korean（笔顺练习）
   - Drops（趣味记忆）
   - 한글 공부（基础发音）

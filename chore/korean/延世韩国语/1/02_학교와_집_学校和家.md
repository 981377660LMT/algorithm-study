## 二、核心词汇

### 1. 指示代词

| 韩文 | 中文           | 距离 |
| ---- | -------------- | ---- |
| 이   | 这（近说话人） | 近   |
| 그   | 那（近听话人） | 中   |
| 저   | 那（远离双方） | 远   |
| 이것 | 这个           | 近   |
| 그것 | 那个           | 中   |
| 저것 | 那个（远）     | 远   |
| 여기 | 这里           | 近   |
| 거기 | 那里           | 中   |
| 저기 | 那里（远）     | 远   |
| 어디 | 哪里           | 疑问 |

> **三分体系**：韩语的指示词是**三分**的（이/그/저），不像中文只有"这/那"两分。
>
> - **이**: 靠近说话人的事物
> - **그**: 靠近听话人的事物，或上文提到的事物
> - **저**: 离说话人和听话人都远的事物

## 三、核心语法

### 语法1: N이/가 있다 / 없다 — "有/没有""在/不在"

**있다** 是韩语中极其重要的词，同时表达两个意思：

1. **存在**：某处"有"某物
2. **拥有**：某人"有"某物

**이/가** 是主格助词：

- 辅音结尾 → **이**
- 元音结尾 → **가**

#### 变化形态

| 语体     | 있다     | 없다     |
| -------- | -------- | -------- |
| 格式体   | 있습니다 | 없습니다 |
| 非格式体 | 있어요   | 없어요   |

#### 例句：表示"有/没有"

```
# 某处有某物
교실에 학생이 있어요.              教室里有学生。
책상 위에 책이 있어요.             桌子上有书。
가방 안에 지갑이 없어요.           书包里没有钱包。

# 某人有某物
저는 한국 친구가 있어요.           我有韩国朋友。
시간이 없어요.                    没有时间。
질문이 있어요?                    有问题吗？
```

#### 例句：表示"在/不在"

```
선생님이 교실에 있어요.            老师在教室里。
화장실이 어디에 있어요?            洗手间在哪里？
도서관은 2층에 있어요.             图书馆在2楼。
민수 씨는 지금 집에 없어요.        民秀现在不在家。
```

---

### 语法2: N에 — 场所助词

**에** 是韩语中最常用的助词之一，在本课主要表示：

#### 用法1：表示存在的**地点**（"在……"）

```
학교에 있어요.        在学校。
집에 있어요.          在家。
교실에 학생이 있어요.  教室里有学生。
```

#### 用法2：表示移动的**目的地**（"去……"）

```
학교에 가요.          去学校。
집에 가요.            回家。
도서관에 가요.        去图书馆。
```

> **注意区分**：
>
> - **에**: 地点/方向（无生命主语，或表示目的地）
> - **에서**: 动作发生的场所（"在……做……"）→ 后续课会详细学

---

### 语法3: 이것/그것/저것 + 은/는 — 指示代词做主题

```
이것은 책입니다.       这是书。
그것은 가방이에요.      那是书包。
저것은 뭐예요?         那(远)是什么？
```

**口语缩略形式**（非常常用）：

| 完整形式 | 缩略形式 | 翻译           |
| -------- | -------- | -------------- |
| 이것은   | 이건     | 这个(主题)     |
| 그것은   | 그건     | 那个(主题)     |
| 저것은   | 저건     | 那个(远，主题) |
| 이것이   | 이게     | 这个(主语)     |
| 그것이   | 그게     | 那个(主语)     |
| 저것이   | 저게     | 那个(远，主语) |

```
이건 뭐예요?           这是什么？
그건 제 가방이에요.     那是我的书包。
저건 시계예요.          那(远)是钟。
```

---

### 语法4: N의 — 所有格助词（"的"）

**의** 读作 [에]（实际发音），表示所属关系。

```
저의 책 = 제 책              我的书
선생님의 가방                 老师的书包
한국의 수도                   韩国的首都
학교의 도서관                 学校的图书馆
```

> **口语中**：저의 常缩略为 **제**，나의 常缩略为 **내**。
> **日常对话中**：의 经常被省略。
>
> - 우리 학교 (我们学校) ← 比 우리의 학교 更自然

---

### 语法5: N도 — 添加助词（"也"）

```
저도 학생이에요.              我也是学生。
여기에도 있어요.              这里也有。
책도 있고 연필도 있어요.       有书也有铅笔。
```

> **注意**：도 替换 은/는, 이/가, 을/를 等助词的位置。
>
> - 저**는** 학생이에요 → 저**도** 학생이에요 (不说 저는도)

## 四、核心会话

### 对话1：在教室里

```
A: 이것은 뭐예요?
   这是什么？

B: 그것은 한국어 사전이에요.
   那是韩语词典。

A: 이것도 사전이에요?
   这个也是词典吗？

B: 아니요, 그것은 사전이 아니에요. 교과서예요.
   不是，那不是词典。是教科书。
```

### 对话2：找地方

```
A: 실례합니다. 화장실이 어디에 있어요?
   打扰一下。洗手间在哪里？

B: 화장실은 2층에 있어요.
   洗手间在2楼。

A: 감사합니다. 도서관도 2층에 있어요?
   谢谢。图书馆也在2楼吗？

B: 아니요, 도서관은 3층에 있어요.
   不，图书馆在3楼。
```

### 对话3：介绍房间

```
A: 방에 뭐가 있어요?
   房间里有什么？

B: 책상하고 의자가 있어요. 그리고 침대도 있어요.
   有书桌和椅子。还有床。

A: 텔레비전은요?
   电视呢？

B: 텔레비전은 없어요. 컴퓨터만 있어요.
   没有电视。只有电脑。
```

**新词**：

- **방** = 房间
- **하고** = 和（口语）
- **그리고** = 然后/还有
- **침대** = 床
- **은요?** = "……呢？"（追问）
- **만** = 只/仅仅

## 五、位置表达详解

### 结构：N + 위/아래/앞/뒤/옆/안/밖 + 에

```
책상 위에 책이 있어요.        书桌上面有书。
의자 아래에 가방이 있어요.     椅子下面有书包。
학교 앞에 편의점이 있어요.     学校前面有便利店。
건물 뒤에 공원이 있어요.       建筑后面有公园。
은행 옆에 약국이 있어요.       银行旁边有药店。
가방 안에 뭐가 있어요?        书包里面有什么？
집 밖에 차가 있어요.          家外面有车。
```

**图解位置关系**：

```
         위 (上)
          ↑
왼쪽 ← [물건] → 오른쪽
(左)      ↓        (右)
         아래 (下)

   앞 (前) ← 나 → 뒤 (后)

   안 (里) ↔ 밖 (外)
```

## 六、补充：层数表达

使用汉字数字 + 층（层）：

| 韩文       | 中文    |
| ---------- | ------- |
| 1층 (일층) | 1楼     |
| 2층 (이층) | 2楼     |
| 3층 (삼층) | 3楼     |
| 지하 1층   | 地下1楼 |

> **韩国的楼层**从1开始计数，没有0楼的概念，和中国一样。

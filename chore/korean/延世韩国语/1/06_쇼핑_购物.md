## 三、核心语法

### 语法1: 价格表达 — 汉字数词 + 원

韩国货币单位是 **원 (韩元/Won)**。读价格用**汉字数词**。

```
100원     = 백 원
500원     = 오백 원
1,000원   = 천 원
3,000원   = 삼천 원
5,500원   = 오천오백 원
10,000원  = 만 원
15,000원  = 만 오천 원 (不说 일만)
50,000원  = 오만 원
100,000원 = 십만 원
```

> **大数字读法规则**：
>
> - 万 (만) 前面是"一"时省略일：10,000 = 만 원 (不说 일만)
> - 千 (천) 前面是"一"时省略일：1,000 = 천 원 (不说 일천)
> - 百 (백) 前面是"一"时省略일：100 = 백 원 (不说 일백)

**价格问答**：

```
A: 이거 얼마예요?          这个多少钱？
B: 삼만 원이에요.           3万韩元。

A: 전부 얼마예요?          总共多少钱？
B: 오만 삼천 원이에요.      53,000韩元。

A: 카드로 해도 돼요?       可以刷卡吗？
B: 네, 됩니다.             可以。
```

---

### 语法2: A-(으)ㄴ N — 形容词修饰名词

形容词要变为修饰形才能放在名词前面：

**规则**：

- 元音结尾词干 → **-ㄴ**
- 辅音结尾词干 → **-은**

| 形容词基本形  | 修饰形 | 例句                       |
| ------------- | ------ | -------------------------- |
| 크다 (大)     | 큰     | 큰 가방 (大包)             |
| 작다 (小)     | 작은   | 작은 시계 (小手表)         |
| 비싸다 (贵)   | 비싼   | 비싼 옷 (贵的衣服)         |
| 싸다 (便宜)   | 싼     | 싼 것 (便宜的东西)         |
| 좋다 (好)     | 좋은   | 좋은 선물 (好礼物)         |
| 예쁘다 (漂亮) | 예쁜   | 예쁜 원피스 (漂亮的连衣裙) |
| 새롭다 (新的) | 새로운 | 새로운 신발 (新鞋)         |
| 길다 (长)     | 긴     | 긴 치마 (长裙) → ㄹ脱落    |
| 짧다 (短)     | 짧은   | 짧은 바지 (短裤)           |

> **ㄹ不规则**：词干以 ㄹ 结尾时，ㄹ脱落后加 ㄴ：
>
> - 길다 → 긴 (长的)
> - 멀다 → 먼 (远的)

```
큰 사이즈 있어요?           有大号吗？
더 싼 것 없어요?            没有更便宜的吗？
예쁜 색깔이에요.            是漂亮的颜色。
이 긴 치마가 좋아요.         喜欢这条长裙。
```

---

### 语法3: V-아/어 보다 — "试试看"

**基本含义**：尝试做某事

```
입어 보다        试穿        (입다 → 입어 보다)
신어 보다        试穿(鞋)    (신다 → 신어 보다)
먹어 보다        尝尝        (먹다 → 먹어 보다)
해 보다          试试做       (하다 → 해 보다)
```

**해요体**：V-아/어 봐요

```
이 옷 입어 봐요.             试穿这件衣服吧。
이 신발 신어 봐요.           试穿这双鞋吧。
이거 한번 먹어 봐요.          试吃一下这个。
한번 해 봐요.                试试看。
```

**过去式**：V-아/어 봤어요 → "试过/做过"

```
한국 음식을 먹어 봤어요.       尝过韩国菜。
제주도에 가 봤어요?           去过济州岛吗？
이 영화 봐 봤어요?            看过这部电影吗？
```

---

### 语法4: -(으)ㄹ게요 — "我会……/我来……"

表示说话人的**意愿/承诺**，只用于第一人称：

**规则**：

- 元音/ㄹ结尾 → **-ㄹ게요**
- 辅音结尾 → **-을게요**

```
이거 살게요.               我买这个。
카드로 할게요.             我用卡付。
내일 올게요.               我明天来。
잠깐만 기다릴게요.         我等一下。
다시 올게요.               我再来。
생각해 볼게요.             我考虑看看。
```

---

### 语法5: -아/어도 되다 — "可以……吗？"

表示许可/征得同意：

```
입어 봐도 돼요?            可以试穿吗？
카드로 해도 돼요?          可以刷卡吗？
여기 앉아도 돼요?          可以坐这里吗？
사진 찍어도 돼요?          可以拍照吗？
```

**回答**：

```
네, 되요. / 네, 괜찮아요.    可以。/ 没问题。
아니요, 안 돼요.            不可以。
```

## 四、核心会话

### 对话1：在服装店

```
손님: 저기요, 이 셔츠 얼마예요?
      请问，这件衬衫多少钱？

직원: 삼만 오천 원이에요.
      35,000韩元。

손님: 좀 비싸네요. 할인 안 돼요?
      有点贵啊。不能打折吗？

직원: 지금 세일 중이에요. 삼만 원에 드릴게요.
      现在在打折。给您30,000。

손님: 다른 색깔 있어요?
      有其他颜色吗？

직원: 네, 하얀색하고 파란색이 있어요.
      有，有白色和蓝色。

손님: 파란색으로 입어 봐도 돼요?
      可以试穿蓝色的吗？

직원: 네, 탈의실은 저쪽에 있어요.
      可以，试衣间在那边。

(试穿后)

손님: 잘 맞아요. 이거 살게요. 카드로 해도 돼요?
      很合适。我买这个。可以刷卡吗？

직원: 네, 됩니다. 감사합니다.
      可以。谢谢。
```

**新词**：

- **좀** = 有点/稍微
- **-네요** = ～啊/～呢（感叹）
- **세일 중** = 打折中
- **드리다** = 给（谦逊语，주다的敬语）
- **다른** = 其他的/不同的
- **N(으)로** = 用/选（工具/选择助词）
- **탈의실** = 试衣间
- **맞다** = 合适/对

### 对话2：在超市

```
A: 이 사과 얼마예요?
   这苹果多少钱？

B: 한 개에 천 원이에요.
   一个1000韩元。

A: 다섯 개 주세요. 그리고 우유도 하나 주세요.
   请给我5个。再给我一个牛奶。

B: 네, 전부 만 원이에요.
   好的，总共10,000韩元。

A: 여기요. (给钱)
   给您。

B: 감사합니다. 봉투 필요하세요?
   谢谢。需要袋子吗？

A: 아니요, 괜찮아요.
   不用了，没关系。
```

**新词**：

- **사과** = 苹果
- **한 개에** = 一个（单价用 에）
- **전부** = 总共
- **봉투** = 袋子
- **필요하다** = 需要

### 对话3：网上购物讨论

```
A: 요즘 인터넷 쇼핑 자주 해요?
   最近经常网购吗？

B: 네, 거의 매주 해요. 마트보다 더 싸요.
   是的，几乎每周都买。比超市更便宜。

A: 그래요? 뭘 주로 사요?
   是吗？主要买什么？

B: 옷이나 신발을 많이 사요. 그런데 사이즈가 안 맞을 때도 있어요.
   主要买衣服和鞋。不过有时候尺码不合适。

A: 교환이나 환불 할 수 있어요?
   可以换货或退款吗？

B: 네, 대부분 가능해요.
   是的，大部分都可以。
```

**新词**：

- **요즘** = 最近
- **N보다** = 比N（比较）
- **더** = 更
- **뭘** = 뭐를的缩略 (什么)
- **주로** = 主要
- **-이나** = 或者
- **-을 때** = ～的时候
- **할 수 있다** = 可以/能够
- **대부분** = 大部分
- **가능하다** = 可能的

## 五、比较表达入门

### N보다 (더) A — "比……更……"

```
이것이 그것보다 비싸요.          这个比那个贵。
시장이 백화점보다 더 싸요.       市场比百货商店更便宜。
이 옷이 저 옷보다 예뻐요.       这件衣服比那件漂亮。
```

### 제일 / 가장 A — "最……"

```
이 가방이 제일 예뻐요.          这个包最漂亮。
뭐가 가장 싸요?                什么最便宜？
```

| 句型           | 用法           | 例句             |
| -------------- | -------------- | ---------------- |

## 三、核心语法

### 语法1: N은/는 — 主题助词（"是……的话"/"至于……"）

**功能**：标记句子的主题（topic），类似于"关于……"

**规则**：

- 前面名词以**辅音**结尾 → **은**
- 前面名词以**元音**结尾 → **는**

| 名词结尾      | 助词 | 例句                       | 翻译       |
| ------------- | ---- | -------------------------- | ---------- |
| 辅音 (받침有) | 은   | 저**는** → ✗，이름**은** ✓ | —          |
| 元音 (받침无) | 는   | 저**는** ✓                 | 我（主题） |

**例句**：

```
저는 학생입니다.           我是学生。
이름은 김민수입니다.        名字是金民秀。
이 사람은 선생님입니다.     这个人是老师。
```

> **은/는 vs 이/가 区别**（初学重点）：
>
> - **은/는**（主题）：强调"关于这个人/事"，常用于介绍、对比
> - **이/가**（主格）：强调"谁/什么"做了某事，常用于新信息
> - 例：**저는** 학생입니다（我呢，是学生）→ 介绍自己
> - 例：**제가** 학생입니다（是我，是学生）→ 强调是"我"

---

### 语法2: N이다 — 判断词/系词（"是"）

**功能**：表示"A是B"，相当于英语的 "be" 动词

**基本形态**：

| 语体           | 形式         | 用法                           |
| -------------- | ------------ | ------------------------------ |
| 格式体敬语     | N입니다      | 最正式（演讲、新闻、初次见面） |
| 非格式体敬语   | N이에요/예요 | 礼貌但亲切（日常对话）         |
| 半语（非敬语） | N이야/야     | 朋友之间                       |

**变化规则**：

| 前面名词结尾 | 格式体 | 非格式体 | 半语 |
| ------------ | ------ | -------- | ---- |
| 辅音结尾     | 입니다 | 이에요   | 이야 |
| 元音结尾     | 입니다 | 예요     | 야   |

> 注意：**입니다** 形态不变，不受前面名词影响。

**例句**：

```
# 格式体 (正式)
저는 학생입니다.                  我是学生。
이것은 책입니다.                  这是书。

# 非格式体 (日常礼貌)
저는 학생이에요.                  我是学生。
이것은 책이에요.                  这是书。
여기는 학교예요.                  这里是学校。

# 半语 (朋友间)
나는 학생이야.                    我是学生。
여기는 학교야.                    这里是学校。
```

---

### 语法3: N이/가 아닙니다 — "不是"

**功能**：이다 的否定形

| 语体     | 辅音后      | 元音后      |
| -------- | ----------- | ----------- |
| 格式体   | 이 아닙니다 | 가 아닙니다 |
| 非格式体 | 이 아니에요 | 가 아니에요 |

**例句**：

```
저는 학생이 아닙니다.              我不是学生。
이것은 책이 아닙니다.              这不是书。
저는 일본 사람이 아니에요.          我不是日本人。
여기는 학교가 아니에요.             这里不是学校。
```

> **结构总结**：
>
> - 肯定：A은/는 B입니다 ← A是B
> - 否定：A은/는 B이/가 아닙니다 ← A不是B

---

### 语法4: N입니까? / N이에요? — 疑问形

**格式体疑问**：把 **입니다** 改为 **입니까?**（书面语气，语调上扬）

**非格式体疑问**：이에요? / 예요?（语调上扬即可）

```
학생입니까?                 是学生吗？（正式）
학생이에요?                 是学生吗？（日常）
한국 사람이에요?            是韩国人吗？
중국 사람입니까?            是中国人吗？
```

## 四、核心会话

### 对话1：初次见面自我介绍

```
A: 안녕하세요. 처음 뵙겠습니다.
   你好。初次见面。

B: 안녕하세요. 저는 김민수입니다.
   你好。我是金民秀。

A: 저는 왕웨이입니다. 중국 사람입니다.
   我是王伟。是中国人。

B: 만나서 반갑습니다.
   见到你很高兴。

A: 네, 만나서 반갑습니다.
   是的，见到你很高兴。
```

**分析**：

- **처음 뵙겠습니다** = 初次(처음) + 拜见(뵙다的谦让形) + 格式体
- **만나서 반갑습니다** = 见面(만나다) + 因为(아/어서) + 高兴(반갑다) + 格式体

### 对话2：询问身份

```
A: 실례지만, 이름이 뭐예요?
   冒昧问一下，您叫什么名字？

B: 제 이름은 다나카입니다. 일본 사람이에요.
   我的名字是田中。是日本人。

A: 학생이에요?
   是学生吗？

B: 아니요, 학생이 아니에요. 회사원이에요.
   不是，不是学生。是公司职员。
```

**分析**：

- **실례지만** = 失礼(실례) + 但是(지만) → "冒昧但…"
- **제** = 저의 的缩写 → "我的"
- **뭐예요** = 뭐(什么) + 예요(是) → "是什么？"

### 对话3：确认国籍

```
A: 어느 나라 사람이에요?
   是哪国人？

B: 중국 사람이에요. 베이징에서 왔어요.
   是中国人。从北京来的。

A: 아, 그래요? 반갑습니다.
   啊，是吗？很高兴认识你。
```

## 五、敬语体系入门

韩语的敬语是**必须掌握**的核心概念，初学阶段需要理解两种主要语体：

### 格式体 (합쇼체) — 最正式

| 类型 | 形式           | 例                |
| ---- | -------------- | ----------------- |
| 陈述 | -ㅂ니다/습니다 | 갑니다 (去)       |
| 疑问 | -ㅂ니까/습니까 | 갑니까? (去吗?)   |
| 命令 | -(으)십시오    | 가십시오 (请去)   |
| 共动 | -(으)ㅂ시다    | 갑시다 (一起去吧) |

**使用场景**：工作汇报、新闻播报、军队、初次见面、对长辈

### 非格式体 (해요체) — 日常礼貌

| 类型 | 形式      | 例              |
| ---- | --------- | --------------- |
| 陈述 | -아/어요  | 가요 (去)       |
| 疑问 | -아/어요? | 가요? (去吗?)   |
| 命令 | -아/어요  | 가요 (请去)     |
| 共动 | -아/어요  | 가요 (一起去吧) |

**使用场景**：日常对话、对平辈、稍微熟悉的人

> **初学建议**：先掌握 **해요체（非格式体）**，这是日常生活中最常用的语体。同时了解 **합쇼체（格式体）**，因为课本和正式场合会用到。

## 六、文化补充

### 1. 韩国人的姓名

韩国姓名通常是 **3个音节**：1个字的姓 + 2个字的名

| 常见姓 | 韩文 | 占比 |
| ------ | ---- | ---- |
| 金     | 김   | ~21% |
| 李     | 이   | ~15% |
| 朴     | 박   | ~8%  |
| 崔     | 최   | ~5%  |
| 郑     | 정   | ~4%  |

### 2. 称呼文化

- 韩国人一般**不直接叫名字**，除非关系很亲近
- 用 **姓名 + 씨**（民秀씨）或 **职位**（部长님、课长님）来称呼
- 对老师/前辈统一用 **선생님**
- 对不认识的人可以用 **저기요**（那个/请问）来引起注意

### 3. 鞠躬礼仪

- 初次见面时轻微鞠躬（约15度）
- 对长辈或正式场合鞠躬约30-45度
- 说 안녕하세요 时通常伴随鞠躬

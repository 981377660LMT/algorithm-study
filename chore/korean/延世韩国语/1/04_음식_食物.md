## 三、核心语法 —— 해요体变化（最重要的语法点之一！）

### 语法1: 아/어요 — 非格式体终结语尾

这是韩语日常对话中**最常用**的语体。所有动词/形容词都可以变为해요体。

#### 变化规则

**Step 1**: 去掉基本形的 **다**，得到词干
**Step 2**: 看词干最后一个元音，决定加 **아요** 还是 **어요**

| 词干末元音 | 加的语尾 | 规则                   |
| ---------- | -------- | ---------------------- |
| ㅏ 或 ㅗ   | 아요     | "阳性元音"→ 接 아      |
| 其他元音   | 어요     | "阴性元音"→ 接 어      |
| 하다 结尾  | 해요     | 特殊：하 + 여요 → 해요 |

#### 详细变化示例

**类型1：词干以 ㅏ 结尾 → 아요 (ㅏ + ㅏ 缩写)**

```
가다 (去)  → 가 + 아요 → 가요     (ㅏ+ㅏ 合并)
사다 (买)  → 사 + 아요 → 사요     (ㅏ+ㅏ 合并)
```

**类型2：词干以 ㅗ 结尾 → 아요 (ㅗ + ㅏ → ㅘ)**

```
오다 (来)  → 오 + 아요 → 와요     (ㅗ+ㅏ → ㅘ)
보다 (看)  → 보 + 아요 → 봐요     (ㅗ+ㅏ → ㅘ)
```

**类型3：词干以 ㅓ 结尾 → 어요 (ㅓ + ㅓ 合并)**

```
먹다 (吃) → 먹 + 어요 → 먹어요   (有收音，直接加)
서다 (站) → 서 + 어요 → 서요     (ㅓ+ㅓ 合并)
```

**类型4：词干以 ㅜ 结尾 → 어요 (ㅜ + ㅓ → ㅝ)**

```
주다 (给)  → 주 + 어요 → 줘요     (ㅜ+ㅓ → ㅝ)
배우다 (学) → 배우 + 어요 → 배워요  (ㅜ+ㅓ → ㅝ)
```

**类型5：词干以 ㅣ 结尾 → 어요 (ㅣ + ㅓ → ㅕ)**

```
마시다 (喝) → 마시 + 어요 → 마셔요  (ㅣ+ㅓ → ㅕ)
기다리다 (等) → 기다리 + 어요 → 기다려요
```

**类型6：词干以 ㅡ 结尾 → 특殊 (ㅡ脱落)**

```
쓰다 (写)  → ㅡ脱落，看前一个元音 → 没有前面元音 → 써요
크다 (大)  → ㅡ脱落，看前一个元音 → 没有前面元音 → 커요
바쁘다 (忙) → ㅡ脱落，看前一个바的ㅏ → 바빠요
예쁘다 (漂亮) → ㅡ脱落，看前一个ㅖ → 예뻐요
```

**类型7：하다 结尾 → 해요**

```
공부하다 (学习) → 공부해요
요리하다 (做饭) → 요리해요
좋아하다 (喜欢) → 좋아해요
전화하다 (打电话) → 전화해요
```

### 语法2: 안 V / V-지 않다 — 动词否定

韩语否定动词有两种方式：

#### 方式1：안 + 动词（短否定，口语常用）

```
안 먹어요.          不吃。
안 가요.            不去。
커피를 안 마셔요.    不喝咖啡。
```

#### 方式2：动词词干 + -지 않다（长否定，更正式）

```
먹지 않아요.         不吃。
가지 않아요.         不去。
커피를 마시지 않아요.  不喝咖啡。
```

> **注意**：하다类动词的短否定，안 放在 하다 前面：
>
> - 공부 **안** 해요 ✓ (不学习)
> - 안 공부해요 ✗ (错误)

### 语法3: V-(으)세요 — "请……" / 尊敬语

表示礼貌地请求、建议，或描述尊敬对象的动作。

**规则**：

- 词干以元音或ㄹ结尾 → **세요**（ㄹ脱落）
- 词干以辅音结尾 → **으세요**

```
# 请求/建议
여기 앉으세요.        请坐这里。
이것 드세요.          请吃这个。(드시다 → 드세요)
메뉴를 보세요.        请看菜单。
많이 드세요.          请多吃。

# 描述尊敬对象的动作
어머니가 요리하세요.   妈妈在做饭。
선생님이 가르치세요.   老师在教。
할아버지가 주무세요.   爷爷在睡觉。(자다的尊敬 → 주무시다)
```

---

### 语法4: N 주세요 — "请给我……"

点餐/购物时最实用的句型：

```
물 주세요.              请给我水。
비빔밥 하나 주세요.      请给我一份拌饭。
커피 두 잔 주세요.       请给我两杯咖啡。
이것 주세요.            请给我这个。
메뉴 주세요.            请给我菜单。
계산서 주세요.          请给我账单。
```

## 四、核心会话

### 对话1：在餐厅点餐

```
직원: 어서 오세요. 몇 명이세요?
      欢迎光临。几位？

손님: 두 명이에요.
      两位。

직원: 이쪽으로 앉으세요. 메뉴 여기 있어요.
      请坐这边。菜单在这里。

손님: 감사합니다. 비빔밥 하나하고 된장찌개 하나 주세요.
      谢谢。请给我一份拌饭和一份大酱汤。

직원: 음료는요?
      饮料呢？

손님: 콜라 두 개 주세요.
      请给我两瓶可乐。

직원: 네, 알겠습니다. 잠시만 기다리세요.
      好的，明白了。请稍等。
```

**词汇**：

- **직원** = 工作人员/服务员
- **손님** = 客人
- **어서 오세요** = 欢迎光临
- **이쪽** = 这边
- **-(으)로** = 向/往（方向助词）
- **음료** = 饮料
- **알겠습니다** = 我知道了/明白了
- **잠시만** = 稍微/一会儿

### 对话2：谈论食物喜好

```
A: 어떤 음식을 좋아해요?
   喜欢什么食物？

B: 저는 삼겹살을 제일 좋아해요. 왕웨이 씨는요?
   我最喜欢五花肉。王伟呢？

A: 저는 비빔밥을 좋아해요. 김치도 좋아해요.
   我喜欢拌饭。也喜欢泡菜。

B: 매운 음식도 잘 먹어요?
   辣的食物也能吃吗？

A: 네, 매운 음식을 아주 좋아해요. 중국 음식이 매우니까요.
   是的，非常喜欢辣的食物。因为中国菜很辣嘛。
```

**词汇**：

- **어떤** = 什么样的/哪种
- **제일** = 最
- **아주** = 非常
- **매우니까요** = 因为辣嘛（매우다 → ㅂ不规则 → 매우니까, 后续课讲解）

### 对话3：做饭

```
A: 요리를 자주 해요?
   经常做饭吗？

B: 네, 거의 매일 해요. 오늘은 김치찌개를 만들었어요.
   是的，几乎每天做。今天做了泡菜汤。

A: 와, 맛있겠다! 저는 요리를 잘 못 해요.
   哇，一定很好吃！我不太会做饭。

B: 김치찌개는 쉬워요. 제가 가르쳐 줄게요.
   泡菜汤很简单。我来教你。
```

**词汇**：

- **자주** = 经常
- **거의** = 几乎
- **매일** = 每天
- **맛있겠다** = 一定好吃（推测）
- **잘 못 하다** = 不太会做
- **쉽다** = 容易的
- **가르치다** = 教
- **-아/어 줄게요** = 我来帮你做（后续课详讲）

## 五、文化补充：韩国饮食文化

### 1. 用餐礼仪

- **어른 먼저 (长辈先)**：长辈先动筷子后，晚辈才能开始吃
- **숟가락과 젓가락 (勺子和筷子)**：韩国用金属扁筷 + 长柄勺
  - 汤/饭用**勺子（숟가락）**
  - 菜/肉用**筷子（젓가락）**
  - 不要同时用筷子和勺子
- **碗不端起来**：韩国文化中碗放在桌上吃，不端起来（和中日不同）
- **밥상 (饭桌)**：韩式用餐有很多小菜（반찬），通常免费续加

### 2. 韩国特色饮酒文化

- **소주 (烧酒)**：韩国国民酒，度数约16-20度
- **건배 (干杯)**：举杯说 "건배!"
- **서로 따르다 (互相倒酒)**：自己不倒酒，别人帮你倒，你帮别人倒
- **고개를 돌리다 (转头喝酒)**：在长辈面前喝酒时要侧过身子

### 3. 常见点餐用语

| 韩文              | 中文               |
| ----------------- | ------------------ |
| 여기요!           | 这里！（叫服务员） |
| 메뉴 주세요       | 请给我菜单         |
| 추천해 주세요     | 请推荐             |
| 이거 뭐예요?      | 这是什么？         |
| 이거 맵어요?      | 这个辣吗？         |
| 덜 맵게 해 주세요 | 请做不那么辣       |
| 계산해 주세요     | 请结账             |
| 잘 먹겠습니다     | 我开动了（饭前）   |
| 잘 먹었습니다     | 我吃好了（饭后）   |

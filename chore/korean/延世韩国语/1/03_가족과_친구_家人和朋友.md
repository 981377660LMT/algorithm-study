> - 男性说哥哥 = **형**，男性说姐姐 = **누나**
> - 女性说哥哥 = **오빠**，女性说姐姐 = **언니**
> - 这四个词也用于非血缘关系的年长者（比如年长的同学、前辈、甚至明星）

## 三、核心语法

### 语法1: 固有数词 + 量词

韩语数数量时，使用**固有数词**（不是汉字数词）。

#### 固有数词完整表

| 数字 | 独立形 | 接量词形 | 注意     |
| ---- | ------ | -------- | -------- |
| 1    | 하나   | 한       | 形态变化 |
| 2    | 둘     | 두       | 形态变化 |
| 3    | 셋     | 세       | 形态变化 |
| 4    | 넷     | 네       | 形态变化 |
| 5    | 다섯   | 다섯     | 不变     |
| 6    | 여섯   | 여섯     | 不变     |
| 7    | 일곱   | 일곱     | 不变     |
| 8    | 여덟   | 여덟     | 不变     |
| 9    | 아홉   | 아홉     | 不变     |
| 10   | 열     | 열       | 不变     |
| 11   | 열하나 | 열한     | 10+1     |
| 12   | 열둘   | 열두     | 10+2     |
| 20   | 스물   | 스무     | 形态变化 |
| 30   | 서른   | 서른     | 不变     |
| 40   | 마흔   | 마흔     | 不变     |
| 50   | 쉰     | 쉰       | 不变     |
| 60   | 예순   | 예순     | 不变     |
| 70   | 일흔   | 일흔     | 不变     |
| 80   | 여든   | 여든     | 不变     |
| 90   | 아흔   | 아흔     | 不변     |

> **注意**：1-4和20在接量词时形态会缩短（하나→한, 둘→두, 셋→세, 넷→네, 스물→스무）

#### 使用结构

```
名词 + 固有数词(接量词形) + 量词

학생 세 명          三名学生
책 두 권            两本书
커피 한 잔          一杯咖啡
고양이 네 마리      四只猫
사과 다섯 개        五个苹果
```

> **语序灵活**：量词短语也可以放在名词前面
>
> - 세 명의 학생 = 학생 세 명 (三名学生)

---

### 语法2: 몇 + 量词 — "几……"

```
가족이 몇 명이에요?           家里有几口人？
→ 네 명이에요. (다섯 명이에요.)  四口人。(五口人。)

책이 몇 권 있어요?            有几本书？
→ 세 권 있어요.               有三本。

몇 살이에요?                  几岁了？（用固有数词回答）
→ 스물두 살이에요.             22岁。
```

> **나이 (年龄)** 的两种说法：
>
> - 固有数词 + 살：스물두 살 (22岁) ← 日常口语
> - 汉字数词 + 세：이십이 세 (22岁) ← 正式/书面

---

### 语法3: N와/과, N하고, N(이)랑 — "和"

韩语有多种表达"和"的方式：

| 助词   | 规则                 | 语体        |
| ------ | -------------------- | ----------- |
| 와     | 元音后               | 书面/正式   |
| 과     | 辅音后               | 书面/正式   |
| 하고   | 不变                 | 口语/日常   |
| (이)랑 | 辅音后이랑，元音后랑 | 口语/非正式 |

```
# 와/과 (正式)
아버지와 어머니                 爸爸和妈妈
형과 동생                      哥哥和弟弟

# 하고 (日常口语)
아빠하고 엄마                   爸爸和妈妈
친구하고 같이 가요              和朋友一起去

# (이)랑 (口语亲切)
오빠랑 언니                     哥哥和姐姐
친구랑 놀아요                   和朋友玩
```

---

### 语法4: 그리고 / 그런데 / 그래서 — 连接句子

| 连接词 | 中文      | 用法         |
| ------ | --------- | ------------ |
| 그리고 | 然后/而且 | 并列添加信息 |
| 그런데 | 但是/不过 | 转折         |
| 그래서 | 所以      | 因果         |

```
가족이 네 명이에요. 그리고 고양이도 한 마리 있어요.
家里有四口人。而且还有一只猫。

형이 있어요. 그런데 누나는 없어요.
有哥哥。但是没有姐姐。

한국어를 공부해요. 그래서 한국 친구가 많아요.
学韩语。所以韩国朋友很多。
```

---

### 语法5: N을/를 — 宾格助词（"把……"）

标记动作的对象（宾语）：

- 辅音结尾 → **을**
- 元音结尾 → **를**

```
한국어를 공부해요.              学韩语。
음악을 들어요.                  听音乐。
커피를 마셔요.                  喝咖啡。
친구를 만나요.                  见朋友。
```

> 这个语法点会在后续课程中大量使用，这里先初步了解。

## 四、核心会话

### 对话1：介绍家人

```
A: 가족이 몇 명이에요?
   家里有几口人？

B: 네 명이에요. 부모님하고 언니가 있어요.
   四口人。有父母和姐姐。

A: 언니는 뭐 해요?
   姐姐做什么的？

B: 언니는 대학생이에요. 서울대학교에 다녀요.
   姐姐是大学生。在首尔大学上学。
```

**新词**：

- **대학생** = 大学生
- **다니다** = 上（学）/去（公司），表示反复前往

### 对话2：介绍朋友

```
A: 이 사람은 누구예요?
   这个人是谁？

B: 제 친구예요. 이름은 마이클이에요.
   是我朋友。名字叫Michael。

A: 어느 나라 사람이에요?
   哪国人？

B: 미국 사람이에요. 그런데 한국어를 잘해요.
   美国人。但是韩语说得很好。

A: 아, 그래요? 몇 살이에요?
   啊，是吗？几岁了？

B: 스물세 살이에요. 저하고 같은 나이예요.
   23岁。和我同龄。
```

**新词**：

- **누구** = 谁
- **잘하다** = 做得好
- **같은** = 相同的
- **나이** = 年龄

### 对话3：宠物

```
A: 집에 동물이 있어요?
   家里有动物吗？

B: 네, 고양이 두 마리하고 개 한 마리가 있어요.
   是的，有两只猫和一只狗。

A: 와, 많네요! 이름이 뭐예요?
   哇，好多啊！叫什么名字？

B: 고양이는 나비하고 별이에요. 개는 초코예요.
   猫叫"蝴蝶"和"星星"。狗叫"巧克力"。
```

**新词**：

- **동물** = 动物
- **고양이** = 猫
- **개** = 狗
- **나비** = 蝴蝶
- **별** = 星星
- **초코** = 巧克力 (chocolate)

## 五、文化补充

### 1. 한국의 가족 문화 (韩国的家族文化)

- **호칭 (称呼) 极其重要**：韩国人非常注重辈分和年龄，即使只差一岁也要用敬语
- **우리 (我们)**：韩国人说"我的家人"会说 **우리 가족**（我们家族），"我妈妈"说 **우리 엄마**（我们妈妈），体现集体文化
- **존댓말 (敬语)**：对父母、老师、前辈必须用敬语
- **나이 (年龄)**：韩国人见面经常先问年龄，以确定使用何种语体

### 2. 韩国年龄系统

韩国曾有独特的"韩国年龄"（한국 나이）算法：出生即1岁，每年1月1日加1岁。但2023年6月起，韩国已统一使用**国际年龄**（만 나이）。

```
만 나이 = 国际年龄 = 当前年 - 出生年（过生日后）
```

## 六、助词总结（1-3课）

到目前为止学过的助词：

| 助词   | 功能      | 接续规则        | 例           |
| ------ | --------- | --------------- | ------------ |
| 은/는  | 主题      | 辅音은/元音는   | 저는, 이름은 |
| 이/가  | 主语      | 辅音이/元音가   | 책이, 제가   |
| 을/를  | 宾语      | 辅音을/元音를   | 한국어를     |
| 에     | 地点/方向 | 不变            | 학교에       |
| 의     | 所有      | 不变(读[에])    | 저의 책      |
| 도     | 也        | 不变            | 저도         |
| 와/과  | 和(正式)  | 元音와/辅音과   | 아버지와     |
| 하고   | 和(口语)  | 不变            | 아빠하고     |
| (이)랑 | 和(亲切)  | 辅音이랑/元音랑 | 친구랑       |

> **记忆规律**：大部分有两种形态的助词，规则是"辅音后形态更长"。

### 练习2：选择正确的"和"

用 와/과, 하고, (이)랑 中任意一个填空：

1. 아버지**\_\_\_** 어머니 (正式场合)
2. 친구**\_\_\_** 같이 (日常对话)
3. 빵**\_\_\_** 우유 (和朋友聊天)

<details>
<summary>点击查看答案</summary>

1. 아버지**와** 어머니 (아버지以元音结尾 → 와)
2. 친구**하고** 같이
3. 빵**이랑** 우유 (빵以辅音结尾 → 이랑)

</details>

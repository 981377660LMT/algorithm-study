# 제5과: 하루 생활 (日常生活)

> 延世韩国语 第一册 · 第5课

---

## 一、学习目标

- 掌握韩语时间表达（小时、分钟）
- 学会描述每天的日常活动
- **重点**：掌握过去时 **-았/었/했어요**
- 学会 에서 (动作发生场所)、부터～까지 (从～到～)

## 三、核心语法

### 语法1: 时间表达

#### 小时：固有数词 + 시

| 韩文    | 中文 |
| ------- | ---- |
| 한 시   | 1点  |
| 두 시   | 2点  |
| 세 시   | 3点  |
| 네 시   | 4点  |
| 다섯 시 | 5点  |
| 여섯 시 | 6点  |
| 일곱 시 | 7点  |
| 여덟 시 | 8点  |
| 아홉 시 | 9点  |
| 열 시   | 10点 |
| 열한 시 | 11点 |
| 열두 시 | 12点 |

#### 分钟：汉字数词 + 분

| 韩文         | 中文    |
| ------------ | ------- |
| 오 분        | 5分     |
| 십 분        | 10分    |
| 십오 분      | 15分    |
| 이십 분      | 20分    |
| 삼십 분 / 반 | 30分/半 |
| 사십오 분    | 45分    |

#### 完整时间表达

```
지금 몇 시예요?                  现在几点？

아홉 시예요.                     9点。
아홉 시 삼십 분이에요.            9点30分。
아홉 시 반이에요.                 9点半。
오후 세 시 이십 분이에요.          下午3点20分。
```

> **关键规则**：
>
> - **小时** → 用固有数词 (한, 두, 세…)
> - **分钟** → 用汉字数词 (일, 이, 삼…)
> - 这是初学者最容易搞混的地方！

---

### 语法2: N에 (时间) — 时间助词

**에** 接在时间词后面，表示"在……（时候）"：

```
아침에 일어나요.          早上起床。
일곱 시에 일어나요.       7点起床。
월요일에 수업이 있어요.    星期一有课。
주말에 뭐 해요?           周末做什么？
```

> **例外**：以下时间词**不加 에**：
>
> - 지금 (现在)、오늘 (今天)、어제 (昨天)、내일 (明天)、매일 (每天)
> - 这些词本身含有时间意义，不需要助词

---

### 语法3: N에서 — 动作场所助词（"在……做……"）

**에서** 表示动作发生的场所，区别于 에（存在位置/目的地）：

```
학교에서 공부해요.           在学校学习。
집에서 텔레비전을 봐요.      在家看电视。
식당에서 밥을 먹어요.        在餐厅吃饭。
도서관에서 책을 읽어요.      在图书馆读书。
회사에서 일해요.            在公司工作。
```

> **에 vs 에서 对比**：
> | 助词 | 用法 | 例句 |
> |------|------|------|
> | 에 | 存在/目的地 | 학교**에** 있어요 (在学校) |
> | 에 | 去向 | 학교**에** 가요 (去学校) |
> | 에서 | 动作场所 | 학교**에서** 공부해요 (在学校学习) |
>
> **简单判断法**：
>
> - 있다/없다 → 用 에
> - 가다/오다 → 用 에
> - 其他动作动词 → 用 에서

---

### 语法4: N부터 N까지 — "从……到……"

**부터** = 从（起点）
**까지** = 到（终点）

可以用于时间和地点：

```
# 时间
아홉 시부터 여섯 시까지 일해요.        从9点到6点工作。
월요일부터 금요일까지 수업이 있어요.    从周一到周五有课。

# 也可以单独使用
몇 시부터예요?                  从几点开始？
세 시까지 와 주세요.             请3点之前来。
```

---

### 语法5: V-았/었/했어요 — 过去时 ★★★

**过去时变化规则和해요体完全一样**，只是把 -아요/어요 换成 -았어요/었어요：

| 规则       | 현재 (现在) | 과거 (过去) |
| ---------- | ----------- | ----------- |
| ㅏ/ㅗ → 았 | 가요        | 갔어요      |
| 其他 → 었  | 먹어요      | 먹었어요    |
| 하다 → 했  | 해요        | 했어요      |

#### 详细变化

```
# ㅏ/ㅗ 元音 → 았어요
가다   → 갔어요        去了
오다   → 왔어요        来了
만나다  → 만났어요      见了
보다   → 봤어요        看了
사다   → 샀어요        买了

# 其他元音 → 었어요
먹다   → 먹었어요      吃了
읽다   → 읽었어요      读了
마시다  → 마셨어요      喝了
쉬다   → 쉬었어요      休息了
배우다  → 배웠어요      学了

# 하다 → 했어요
공부하다 → 공부했어요    学习了
운동하다 → 운동했어요    运动了
요리하다 → 요리했어요    做饭了
청소하다 → 청소했어요    打扫了

# 이다 → 이었어요/였어요
학생이다 → 학생이었어요   是学生（过去）
```

#### 过去否定

```
안 갔어요.             没去。
안 먹었어요.           没吃。
공부 안 했어요.        没学习。
가지 않았어요.         没去。（长否定过去式）
```

## 四、核心会话

### 对话1：描述一天

```
A: 보통 몇 시에 일어나요?
   通常几点起床？

B: 일곱 시에 일어나요.
   7点起床。

A: 아침에 뭐 해요?
   早上做什么？

B: 씻고 아침을 먹어요. 그리고 여덟 시에 회사에 가요.
   洗漱吃早饭。然后8点去公司。

A: 몇 시까지 일해요?
   工作到几点？

B: 아홉 시부터 여섯 시까지 일해요. 퇴근 후에 운동해요.
   从9点工作到6点。下班后运动。
```

**新词**：

- **보통** = 通常
- **-고** = "然后/并且"（连接词尾，并列两个动作）
- **후에** = ～之后

### 对话2：问昨天做了什么

```
A: 어제 뭐 했어요?
   昨天做了什么？

B: 어제 친구를 만났어요.
   昨天见了朋友。

A: 어디에서 만났어요?
   在哪里见的？

B: 강남에서 만났어요. 같이 점심을 먹고 영화를 봤어요.
   在江南见的。一起吃了午饭看了电影。

A: 재미있었어요?
   有趣吗？

B: 네, 아주 재미있었어요!
   是的，非常有趣！
```

**新词**：

- **강남** = 江南（首尔地名）
- **같이** = 一起
- **영화** = 电影
- **재미있다** = 有趣的 → 재미있었어요 (过去式)

### 对话3：周末计划

```
A: 주말에 보통 뭐 해요?
   周末通常做什么？

B: 토요일에는 청소하고 빨래해요.
   周六打扫和洗衣服。

A: 일요일에는요?
   周日呢？

B: 일요일에는 쉬어요. 집에서 음악을 듣거나 책을 읽어요.
   周日休息。在家听音乐或者读书。

A: 저는 주말에 한국어를 공부해요.
   我周末学韩语。

B: 대단해요! 한국어 공부 화이팅!
   好厉害！韩语学习加油！
```

**新词**：

- **-거나** = 或者（连接动词）
- **대단하다** = 了不起/厉害
- **화이팅** = 加油 (Fighting!)

## 五、V-고 — 并列连接词尾

**-고** 连接两个动作，表示"然后"或"并且"：

```
씻고 밥을 먹어요.              洗漱然后吃饭。
아침을 먹고 학교에 가요.        吃完早饭去学校。
운동하고 샤워해요.              运动然后洗澡。
커피를 마시고 공부해요.          喝完咖啡学习。
```

> **-고 vs 그리고**：
>
> - **-고** 连接词尾，用在句中连接两个动词
> - **그리고** 连接副词，用在句首连接两个句子
> - 밥을 먹**고** 커피를 마셔요 = 밥을 먹어요. **그리고** 커피를 마셔요.

## 六、一天的日程（综合运用）

```
저는 매일 일곱 시에 일어나요.
아침에 샤워하고 이를 닦고 옷을 입어요.
일곱 시 반에 아침을 먹어요.
여덟 시에 집에서 나가요.
아홉 시부터 여섯 시까지 회사에서 일해요.
점심은 열두 시에 회사 식당에서 먹어요.
여섯 시에 퇴근해요.
퇴근 후에 헬스장에서 운동해요.
여덟 시에 집에 와요.
저녁을 먹고 텔레비전을 봐요.
열한 시에 자요.
```

**翻译**：
我每天7点起床。早上洗澡、刷牙、穿衣服。7点半吃早饭。8点出门。从9点到6点在公司工作。午饭12点在公司食堂吃。6点下班。下班后在健身房运动。8点回家。吃完晚饭看电视。11点睡觉。

## 八、本课核心句型小结

| 句型           | 用法      | 例句                     |
| -------------- | --------- | ------------------------ |
| N시 N분        | 时间表达  | 세 시 이십 분            |
| N에 (时间)     | 在某时    | 아홉 시에                |
| N에서 V        | 在某处做… | 학교에서 공부해요        |
| N부터 N까지    | 从…到…    | 9시부터 6시까지          |
| V-았/었/했어요 | 过去时    | 먹었어요, 갔어요, 했어요 |
| V-고 V         | 并列/然后 | 먹고 마셔요              |

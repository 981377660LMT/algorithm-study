> 延世韩国语 第二册 · 第6课

---

## 一、学习目标

- 能够在邮局、银行、公共机关办理事务
- **重点语法**：-(으)면 되다 (只要…就行)、-아/어 놓다 (做好放着)、-아/어 두다 (留着备用)、-(으)ㄹ 수밖에 없다 (只能/不得不)
- 掌握办事流程常用表达
- 了解韩国公共服务体系

## 三、核心语法

### 语法1: -(으)면 되다 — "只要…就行了" ★★★

告诉对方"做到…就够了/就可以了"，给出**简单的解决方案**。

**构成**：动词/形容词词干 + -(으)면 되다

```
# 基本用法
여기에 이름을 쓰면 돼요.
只要在这里写名字就行了。

여권을 가져오면 돼요.
带护照来就行了。

3번 창구에 가면 돼요.
去3号窗口就行了。

이 서류를 작성하면 됩니다.
填写这个文件就行了。
```

**否定：-지 않으면 되다 / 안 -면 되다**

```
걱정하지 않으면 돼요.           不用担心就行了。
늦지 않으면 돼요.              不迟到就行了。
```

**疑问：어떻게 하면 돼요?**（非常实用）

```
환전하려면 어떻게 하면 돼요?     想换汇的话怎么做就行？
비자를 연장하려면 어떻게 하면 돼요? 想延长签证怎么办？
```

---

### 语法2: -아/어 놓다 — "做好放着" ★★★

表示**做完某个动作后，维持着那个结果状态**。强调"已完成"状态的持续。

**构成**：动词词干 + -아/어 놓다

```
문을 열어 놓았어요.              把门打开了（保持开着的状态）。
돈을 모아 놓았어요.              把钱存好了（存着呢）。
짐을 싸 놓았어요.               把行李收拾好了（放着呢）。
미리 예약해 놓았어요.            提前预约好了。
음식을 만들어 놓았어요.           把菜做好了（做好放着）。
서류를 준비해 놓았어요.           把材料准备好了。
```

---

### 语法3: -아/어 두다 — "做好留着/备用" ★★★

与 -아/어 놓다 相似，但更强调**为将来备用**的含义。

**构成**：动词词干 + -아/어 두다

```
여행 전에 호텔을 예약해 뒀어요.   旅行前把酒店预订好了（留着备用）。
나중에 먹으려고 음식을 냉장고에 넣어 뒀어요.
为了以后吃把食物放冰箱里了。

필요한 서류를 복사해 뒀어요.      把需要的材料复印好了。
중요한 전화번호를 적어 뒀어요.    把重要的电话号码记下来了。
```

> **-아/어 놓다 vs -아/어 두다**：
>
> - **놓다**：强调"做完了"的结果状态 → 문을 열어 놓았어요 (把门开着)
> - **두다**：强调"为将来准备/保管" → 돈을 모아 두었어요 (攒钱留着用)
> - 很多情况下两者可以互换，但语义侧重不同。
> - 口语中 두다 常缩略：두었어요 → 뒀어요

---

### 语法4: -(으)ㄹ 수밖에 없다 — "只能/不得不/没办法只好" ★★

表示**没有其他选择**，只能这样做。

**构成**：动词词干 + -(으)ㄹ 수밖에 없다

```
비가 와서 택시를 탈 수밖에 없었어요.
下雨了只能坐出租车。

마감이 내일이라서 야근할 수밖에 없어요.
截止日期是明天，只能加班。

현금이 없어서 카드로 낼 수밖에 없어요.
没有现金，只能刷卡。

한국에서 살려면 한국어를 배울 수밖에 없어요.
想在韩国生活就不得不学韩语。
```

> **-(으)ㄹ 수밖에 없다 vs -아/어야 하다**：
>
> - **-아/어야 하다**：应该/必须（义务、规范）
> - **-(으)ㄹ 수밖에 없다**：没有其他选择，被迫只能（无奈）

## 四、核心会话

### 对话1：在邮局寄包裹

```
A: 안녕하세요. 중국에 소포를 보내고 싶은데요.
   你好。我想往中国寄包裹。

B: 네, 여기에 올려놓으세요. 무게를 재 볼게요.
   好的，请放上来。我称一下重量。
   3.5킬로그램이네요.

A: 얼마나 걸려요?
   要多久？

B: 일반 우편은 2주 정도 걸리고, 빠른 우편은 5일 정도요.
   普通邮寄大约2周，快递大约5天。

A: 빠른 우편으로 보내면 얼마예요?
   快递寄的话多少钱？

B: 35,000원이에요. 이 서류에 보내는 사람과 받는 사람의
   이름, 주소를 쓰면 돼요.
   35000韩元。在这个单子上写寄件人和收件人的姓名地址就行。

A: 네, 알겠습니다.
   好的，知道了。
```

### 对话2：在银行

```
A: 안녕하세요. 계좌를 만들고 싶은데요.
   你好。我想开账户。

B: 신분증 가져오셨어요?
   带身份证了吗？

A: 네, 외국인등록증이요.
   是的，外国人登录证。

B: 네, 이 신청서를 작성하시면 돼요.
   비밀번호는 네 자리 숫자로 만드시면 됩니다.
   好的，填这个申请表就行。
   密码设4位数字就行。

A: 체크카드도 같이 만들 수 있어요?
   借记卡也可以一起办吗？

B: 네, 카드는 일주일 후에 받을 수 있어요.
   可以，卡一周后可以拿到。
   그동안 통장으로 사용하시면 돼요.
   这期间用存折就行。
```

### 对话3：签证延长

```
A: 비자를 연장하려면 어떻게 하면 돼요?
   想延长签证要怎么办？

B: 출입국관리사무소에 가야 해요.
   要去出入境管理局。
   미리 온라인으로 예약해 놓으면 빨라요.
   提前在网上预约好的话会快。

A: 필요한 서류가 뭐예요?
   需要什么材料？

B: 여권, 외국인등록증, 재학증명서를 준비해 두면 돼요.
   准备好护照、外国人登录证、在学证明就行。
   사진도 한 장 가져가야 해요.
   也要带一张照片。

A: 온라인으로 예약 안 하면 안 돼요?
   不在网上预约不行吗？

B: 안 해도 되는데 오래 기다릴 수밖에 없어요.
   不预约也行，但只能等很久。

A: 그러면 미리 예약해 놓을게요.
   那我提前预约好。
```

**新词**：

- **연장하다** = 延长
- **재학증명서** = 在学证明
- **오래** = 很久

## 六、语法练习

### 练习1：-(으)면 되다

用 -(으)면 되다 回答"怎么办"：

1. A: 도서관에서 책을 빌리려면 어떻게 해요?
   B: 도서관 카드를 (만들다) → ******\_\_\_******

2. A: 한국에서 핸드폰을 사려면?
   B: 외국인등록증을 (가져가다) → ******\_\_\_******

3. A: 이 서류는 어디에 내요?
   B: 저기 접수처에 (내다) → ******\_\_\_******

<details>
<summary>点击查看答案</summary>

1. 도서관 카드를 만들면 돼요.
2. 외국인등록증을 가져가면 돼요.
3. 저기 접수처에 내면 돼요.

</details>

### 练习2：-아/어 놓다 / -아/어 두다

用合适的表达完成句子：

1. 이따 회의가 있으니까 자료를 복사해 \_\_\_.
2. 이미 비행기 표를 예약해 \_\_\_.
3. 여행 가기 전에 여권을 찾아 \_\_\_ 세요.

<details>
<summary>点击查看答案</summary>

1. 복사해 놓으세요 / 복사해 두세요 (为会议准备好)
2. 예약해 놓았어요 / 예약해 뒀어요 (已经做好)
3. 찾아 두세요 / 찾아 놓으세요 (为旅行准备好)

</details>

### 练习3：-(으)ㄹ 수밖에 없다

用"只能…"表达无奈：

1. 지하철이 끊겨서 (택시를 타다) → ******\_\_\_******
2. 돈이 부족해서 (싼 것을 사다) → ******\_\_\_******
3. 한국어로 서류를 써야 해서 (한국 친구에게 부탁하다) → ******\_\_\_******

<details>
<summary>点击查看答案</summary>

1. 택시를 탈 수밖에 없었어요.
2. 싼 것을 살 수밖에 없어요.
3. 한국 친구에게 부탁할 수밖에 없어요.

</details>

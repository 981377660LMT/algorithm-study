> 延世韩国语 第二册 · 第8课

---

## 一、学习目标

- 能够描述身体不适和症状
- **重点语法**：-지 말다 (不要做)、-아/어도 (即使…也)、-(으)ㄴ 지 (自从…以来)、-기 때문에 (因为)
- 学会在医院/药店进行对话
- 掌握健康相关的常用表达

## 三、核心语法

### 语法1: -지 말다 — "不要做……" ★★★

用于**禁止/劝阻/请求不要做某事**。

**构成**：动词词干 + -지 말다

| 形式     | 变化              | 例句               |
| -------- | ----------------- | ------------------ |
| 命令     | -지 마세요 (敬语) | 걱정하지 마세요    |
| 命令     | -지 마 (半语)     | 걱정하지 마        |
| 共动     | -지 맙시다        | 포기하지 맙시다    |
| 共动     | -지 말자          | 싸우지 말자        |
| 间接引语 | -지 말라고        | 먹지 말라고 했어요 |

**例句**：

```
무리하지 마세요.                  不要勉强。
찬 음식을 먹지 마세요.            不要吃凉的食物。
술을 마시지 마세요.               不要喝酒。
약을 빈속에 먹지 마세요.          不要空腹吃药。
늦지 마세요.                     不要迟到。
여기에서 담배를 피우지 마세요.     请不要在这里抽烟。
```

> **-지 않다 (不做) vs -지 말다 (不要做)**：
>
> - **-지 않다**：描述"不做某事"（陈述事实）→ 저는 술을 마시지 않아요 (我不喝酒)
> - **-지 말다**：命令/请求"不要做" → 술을 마시지 마세요 (不要喝酒)

---

### 语法2: -아/어도 — "即使…也…" / "就算…也…" ★★★

表示**让步**，前面条件不影响后面结果。

**构成**：动词/形容词词干 + -아/어도

```
# 基本用法
비가 와도 갈 거예요.              即使下雨也要去。
약을 먹어도 안 나아요.             即使吃药也不好。
아무리 바빠도 밥은 먹어야 해요.     再忙也要吃饭。
좀 비싸도 건강에 좋은 음식을 먹어요.  即使贵点也吃对健康好的食物。
피곤해도 운동해야 해요.            即使累也要运动。
```

**아무리 + -아/어도**：强调"无论多么…也…"

```
아무리 아파도 병원에 안 가요.       无论多疼也不去医院。
아무리 추워도 매일 조깅해요.        无论多冷也每天慢跑。
아무리 좋아도 너무 많이 먹으면 안 돼요.
无论多好吃也不能吃太多。
```

---

### 语法3: -(으)ㄴ 지 — "自从…以来已经" ★★★

表示从某个时间点到现在过了多久。

**构成**：动词词干 + -(으)ㄴ 지 + 时间 + 되다/넘다/지나다

```
한국에 온 지 1년이 됐어요.
来韩国已经1年了。

이 약을 먹은 지 3일이 됐어요.
吃这个药已经3天了。

감기에 걸린 지 일주일이 됐는데 아직 안 나았어요.
感冒已经一周了但还没好。

운동을 시작한 지 한 달이 넘었어요.
开始运动已经超过一个月了。

이 회사에서 일한 지 5년이 됐어요.
在这家公司工作已经5年了。
```

> **질문**:
>
> - "한국에 온 지 얼마나 됐어요?" = 来韩国多久了？
> - "이 약을 먹은 지 얼마나 됐어요?" = 吃这个药多久了？

---

### 语法4: -기 때문에 — "因为……所以" ★★★

表示**明确的原因/理由**，比 -아/어서 更正式。

**构成**：

- 动词/形容词词干 + **-기 때문에**
- 名词 + **때문에** / **이기 때문에**

```
# V/A + -기 때문에
감기에 걸렸기 때문에 학교에 못 갔어요.
因为感冒了所以没能去学校。

너무 아프기 때문에 병원에 가야 해요.
因为太疼了所以要去医院。

약을 안 먹었기 때문에 안 나았어요.
因为没吃药所以没好。

# N + 때문에
감기 때문에 학교에 못 갔어요.
因为感冒没能去学校。

날씨 때문에 운동을 못 했어요.
因为天气没能运动。

시험 때문에 바빠요.
因为考试很忙。
```

> **원인 표현 비교 (原因表达对比)**：
> | 表达 | 特点 | 例句 |
> |------|------|------|
> | -아/어서 | 客观原因，不接命令/共动 | 아파서 못 갔어요 |
> | -(으)니까 | 主观原因，可接命令/共动 | 아프니까 쉬세요 |
> | -기 때문에 | 正式/书面，强调因果 | 아프기 때문에 쉬어야 합니다 |
> | -거든요 | 解释新信息给对方 | 아프거든요 |

## 四、核心会话

### 对话1：看医生

```
의사: 어디가 아파서 오셨어요?
      哪里不舒服来的？

환자: 어제부터 열이 나고 기침이 심해요.
      从昨天开始发烧咳嗽厉害。

의사: 열이 몇 도예요?
      烧多少度？

환자: 아까 재 봤는데 38.5도였어요.
      刚才量了是38.5度。

의사: 목도 좀 부었네요. 감기에 걸린 것 같아요.
      嗓子也有点肿。好像得了感冒。
      감기약을 처방해 드릴게요.
      我给您开感冒药。

환자: 네, 감사합니다.
      好的，谢谢。

의사: 하루에 세 번, 식후 30분에 드세요.
      一天三次，饭后30分钟吃。
      찬 음식을 먹지 마시고 푹 쉬세요.
      不要吃凉的，好好休息。

환자: 며칠 정도 쉬면 돼요?
      休息几天就行了？

의사: 약을 먹으면 2~3일이면 나을 거예요.
      吃药的话2-3天就会好。
      그래도 안 나으면 다시 오세요.
      如果还没好的话再来。
```

### 对话2：在药店

```
환자: 안녕하세요. 처방전이요.
      你好。这是处方。

약사: 네, 잠시만요.
      好的，请稍等。
      알레르기가 있는 약이 있어요?
      有过敏的药物吗？

환자: 아니요, 없어요.
      没有。

약사: 이 약은 식후에 드시면 돼요. 하루 세 번이에요.
      这个药饭后吃就行。一天三次。
      이 약은 좀 졸릴 수 있어요.
      这个药可能会犯困。

환자: 그러면 운전하면 안 돼요?
      那就不能开车了？

약사: 네, 약을 먹은 후에는 운전하지 마세요.
      是的，吃完药后不要开车。
      그리고 술은 절대 마시지 마세요.
      而且绝对不要喝酒。

환자: 네, 알겠습니다. 감사합니다.
      好的，知道了。谢谢。
```

### 对话3：关心朋友

```
A: 얼굴이 안 좋아 보여요. 어디 아파요?
   脸色看起来不好。哪里不舒服？

B: 요즘 머리가 자주 아파요.
   最近经常头疼。

A: 병원에 가 봤어요?
   去医院看了吗？

B: 아뇨, 약을 먹어도 안 나아서요.
   没有，吃药也不好。

A: 두통이 시작된 지 얼마나 됐어요?
   头痛开始多久了？

B: 한 2주 된 것 같아요.
   大概2周了。

A: 2주나요? 그러면 꼭 병원에 가 보세요.
   2周啊？那一定要去医院看看。
   아무리 바빠도 건강이 제일 중요하잖아요.
   再忙健康也是最重要的嘛。

B: 맞아요. 이번 주에 병원에 가기로 했어요.
   对。这周决定去医院了。

A: 무리하지 말고 좀 쉬세요.
   不要勉强，休息一下。
```

## 五、文化补充：韩国医疗体系

```
🏥 한국 의료 시스템
  - 의원 (诊所): 먼저 가까운 의원(클리닉)에 가요.
    先去附近的诊所。
  - 병원 (医院): 큰 병원은 의원의 소견서가 필요해요.
    大医院需要诊所的转诊单。
  - 응급실 (急诊): 응급 상황에는 119에 전화해요.
    急诊情况拨打119。

💊 약국 시스템
  - 한국은 의약분업이에요.
    韩国实行医药分离。
  - 의사의 처방전이 있어야 약국에서 약을 받을 수 있어요.
    需要有医生的处方才能在药店取药。
  - 간단한 약 (두통약, 소화제)은 처방전 없이 살 수 있어요.
    简单的药（头痛药、消化药）可以不需处方购买。

🏥 건강보험
  - 한국에 살면 건강보험에 가입해야 해요.
    在韩国生活要加入健康保险。
  - 외국인도 6개월 이상 체류하면 건강보험 가입 대상이에요.
    外国人居留6个月以上也是健康保险参保对象。
  - 진료비의 약 30%만 내면 돼요.
    只需支付诊疗费的约30%。
```

## 六、语法练习

### 练习1：-지 마세요 禁止表达

用 -지 마세요 给出建议：

1. 感冒了的人 → (찬 물을 마시다) ******\_\_\_******
2. 腿受伤的人 → (무리하게 운동하다) ******\_\_\_******
3. 吃药后 → (술을 마시다) ******\_\_\_******
4. 图书馆里 → (큰 소리로 말하다) ******\_\_\_******

<details>
<summary>点击查看答案</summary>

1. 찬 물을 마시지 마세요.
2. 무리하게 운동하지 마세요.
3. 술을 마시지 마세요.
4. 큰 소리로 말하지 마세요.

</details>

### 练习2：-아/어도 让步

完成句子：

1. 비가 (오다) \_\_\_ 매일 운동해요.
2. 아무리 (힘들다) \_\_\_ 포기하지 않을 거예요.
3. 약을 (먹다) \_\_\_ 아직 안 나았어요.
4. (피곤하다) \_\_\_ 병원에 가야 해요.

<details>
<summary>点击查看答案</summary>

1. 와도
2. 힘들어도
3. 먹어도
4. 피곤해도

</details>

### 练习3：-(으)ㄴ 지 + 时间

用"自从…已经…了"造句：

1. 韩国생活 / 2年 → ******\_\_\_******
2. 吃药 / 3天 → ******\_\_\_******
3. 学韩语 / 6个月 → ******\_\_\_******
4. 戒烟 / 1年 → ******\_\_\_******

<details>
<summary>点击查看答案</summary>

1. 한국에서 산 지 2년이 됐어요.
2. 약을 먹은 지 3일이 됐어요.
3. 한국어를 배운 지 6개월이 됐어요.
4. 담배를 끊은 지 1년이 됐어요.

</details>

## 七、本课核心句型小结

| 句型              | 用法       | 例句                      |
| ----------------- | ---------- | ------------------------- |
| -지 마세요        | 不要做…    | 무리하지 마세요           |
| -아/어도          | 即使…也    | 약을 먹어도 안 나아요     |
| -(으)ㄴ 지 + 时间 | 自从…已经  | 온 지 1년이 됐어요        |
| -기 때문에        | 因为(正式) | 아프기 때문에 쉬어야 해요 |

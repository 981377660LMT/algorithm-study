# 제7과: 전화 (电话) — 간접화법 완전정복

> 延世韩国语 第二册 · 第7课

---

## 一、学习目标

- 完全掌握**间接引语（간접화법）**四大形式
- **重点语法**：-다고 하다 (说)、-(으)냐고 하다 (问)、-자고 하다 (提议)、-(으)라고 하다 (命令/要求)
- 能够转述他人的话、传达消息
- 掌握更复杂的电话通话表达

## 三、核心语法 — 间接引语四大天王 ★★★★★

间接引语是从第二册到第三册**最重要的语法**，必须彻底掌握。

> 💡 核心概念：把别人说的话"转述"出来。
> 直接引语：민수가 "내일 바빠요" 라고 말했어요. (民秀说"明天忙")
> 间接引语：민수가 내일 바쁘**다고** 말했어요. (民秀说明天忙)

---

### 语法1: -다고 하다 — 转述陈述句 "说……" ★★★★★

#### 构成规则

| 词性       | 现在            | 过去              | 将来                |
| ---------- | --------------- | ----------------- | ------------------- |
| **动词**   | V-ㄴ/는**다고** | V-았/었**다고**   | V-(으)ㄹ 거**라고** |
| **形容词** | A-**다고**      | A-았/었**다고**   | A-(으)ㄹ 거**라고** |
| **名词**   | N-(이)**라고**  | N-이었/였**다고** | N-일 거**라고**     |

#### 详细变化

```
# ★ 动词现在时
먹다 → 먹는다고    가다 → 간다고    하다 → 한다고
오다 → 온다고     마시다 → 마신다고  살다 → 산다고 (ㄹ脱落)

# ★ 形容词现在时 (直接 + 다고)
크다 → 크다고     예쁘다 → 예쁘다고   좋다 → 좋다고
바쁘다 → 바쁘다고  맛있다 → 맛있다고

# ★ 过去时 (不分动词/形容词)
먹었다고, 갔다고, 예뻤다고, 좋았다고

# ★ 名词 (이라고)
학생이라고, 한국 사람이라고, 의사라고 (元音后省이)
```

#### 例句

```
민수 씨가 내일 회사에 안 간다고 했어요.
民秀说明天不去公司。

선생님이 이 책이 좋다고 했어요.
老师说这本书好。

친구가 어제 영화를 봤다고 했어요.
朋友说昨天看了电影。

그 사람이 자기가 의사라고 했어요.
那个人说自己是医生。

뉴스에서 내일 비가 올 거라고 했어요.
新闻说明天要下雨。
```

**口语缩略 -다고 → -대(요)**：

```
민수가 내일 바쁘대요. = 민수가 내일 바쁘다고 해요.
엄마가 밥 먹으래요. (命令的缩略，见下文)
```

---

### 语法2: -(으)냐고 하다 — 转述疑问句 "问……" ★★★★★

#### 构成

| 词性            | 形式              |
| --------------- | ----------------- |
| **动词/形容词** | V/A-**(으)냐고**  |
| **名词**        | N-**(이)냐고**    |
| **过去**        | V/A-**았/었냐고** |

```
# 基本变化
가다 → 가냐고         먹다 → 먹냐고
좋다 → 좋으냐고       바쁘다 → 바쁘냐고
학생 → 학생이냐고      한국 사람 → 한국 사람이냐고
```

#### 例句

```
친구가 어디 가냐고 물어봤어요.
朋友问我去哪里。

엄마가 밥 먹었냐고 전화했어요.
妈妈打电话问吃饭了没有。

선생님이 숙제 했냐고 물었어요.
老师问做作业了没有。

동료가 내일 회의에 참석하냐고 했어요.
同事问明天参不参加会议。

지수 씨가 저한테 한국 사람이냐고 했어요.
智秀问我是不是韩国人。
```

---

### 语法3: -자고 하다 — 转述共动句 "说一起……" ★★★★

转述"一起…吧"这类提议。

#### 构成

动词词干 + **-자고 하다**

```
같이 가다 → 같이 가자고 했어요
같이 먹다 → 같이 먹자고 했어요
```

#### 例句

```
민수 씨가 같이 영화 보자고 했어요.
民秀说一起看电影。

친구가 주말에 등산 가자고 했어요.
朋友说周末一起去爬山。

동료가 점심에 같이 먹자고 했어요.
同事说中午一起吃。

지수 씨가 다음에 만나자고 했어요.
智秀说下次见面。

팀장님이 회의를 시작하자고 했어요.
组长说开始开会。
```

---

### 语法4: -(으)라고 하다 — 转述命令/请求 "让……" ★★★★

转述命令、请求、指示。

#### 构成

动词词干 + **-(으)라고 하다**

- 元音/ㄹ → -라고
- 辅音 → -으라고

```
# 基本变化
가다 → 가라고       오다 → 오라고
먹다 → 먹으라고     읽다 → 읽으라고
하다 → 하라고       공부하다 → 공부하라고
```

#### 例句

```
선생님이 열심히 공부하라고 했어요.
老师让努力学习。

엄마가 일찍 오라고 했어요.
妈妈让早点来。

의사가 약을 먹으라고 했어요.
医生让吃药。

부장님이 보고서를 내일까지 내라고 했어요.
部长让明天之前交报告。

경찰이 여기에 차를 세우지 말라고 했어요.
警察让不要在这里停车。
```

> ⚠️ **否定命令**：-지 **말라고** 하다 （让不要做…）
>
> 의사가 술을 마시지 말라고 했어요. 医生说别喝酒。

---

### 间接引语总结表 ★★★★★

| 类型       | 原句          | 间接引语         | 缩略形      |
| ---------- | ------------- | ---------------- | ----------- |
| **陈述**   | "바쁩니다"    | 바쁘다고 해요    | 바쁘대요    |
| **疑问**   | "바쁩니까?"   | 바쁘냐고 해요    | 바쁘냬요    |
| **共动**   | "같이 가요"   | 같이 가자고 해요 | 같이 가재요 |
| **命令**   | "빨리 오세요" | 빨리 오라고 해요 | 빨리 오래요 |
| **否定命** | "가지 마세요" | 가지 말라고 해요 | 가지 말래요 |

## 四、核心会话

### 对话1：转接电话

```
A: 여보세요, 김민수 씨 좀 바꿔 주세요.
   喂，请帮我转接金民秀。

B: 잠시만요, 민수 씨 지금 통화 중인데요.
   请稍等，民秀正在通话中呢。

A: 그러면 전해 주실 수 있어요?
   那能帮我转达吗？

B: 네, 말씀하세요.
   好的，请说。

A: 내일 회의가 오후 세 시로 바뀌었다고 전해 주세요.
   请转达明天的会议改到下午3点了。

B: 네, 내일 회의 오후 세 시. 알겠습니다. 전화 왔다고 전할게요.
   好的，明天会议下午3点。知道了。我会转达你来过电话。
```

### 对话2：传达他人的话

```
A: 민수 씨가 뭐래요?
   民秀说什么？

B: 이번 주말에 같이 부산에 가자고 해요.
   说这周末一起去釜山。

A: 부산? 갑자기?
   釜山？突然？

B: 네, 부산에 맛있는 식당을 찾았다고 했어요.
   是，说找到了好吃的餐厅。
   그리고 꼭 회를 먹어 보라고 했어요.
   还让一定要尝尝生鱼片。

A: 재미있겠다! 몇 시에 출발하냐고 물어봐 줘.
   应该很好玩！帮我问问几点出发。

B: 민수 씨한테 전화해서 물어볼게요.
   我给民秀打电话问问。
```

### 对话3：复杂转述

```
A: 아까 부장님이 뭐라고 하셨어요?
   刚才部长说什么了？

B: 이번 프로젝트가 중요하다고 하셨어요.
   说这次项目很重要。
   그리고 다음 주까지 보고서를 준비하라고 하셨어요.
   然后让下周之前准备好报告。

A: 팀원들한테도 알려야겠네요.
   得告诉组员们呢。

B: 네, 그리고 금요일에 다 같이 회의하자고 하셨어요.
   是的，还说周五大家一起开会。
   늦지 말라고도 하셨어요.
   也说了不要迟到。

A: 알겠어요. 모두한테 금요일 회의가 있다고 전할게요.
   知道了。我跟大家说周五有会议。
```

**新词**：

- **바꾸다** = 换/转接
- **바뀌다** = 被换/改变 (被动)
- **갑자기** = 突然
- **회(刺身)** = 生鱼片
- **알려야겠네요** = 得告诉…呢

## 五、间接引语活用练习模板

### 将直接引语转换为间接引语

| 直接引语              | 类型     | 间接引语                 |
| --------------------- | -------- | ------------------------ |
| "한국어가 어렵습니다" | 陈述     | 한국어가 어렵다고 했어요 |
| "어디에 살아요?"      | 疑问     | 어디에 사냐고 했어요     |
| "같이 밥 먹읍시다"    | 共动     | 같이 밥 먹자고 했어요    |
| "문을 닫으세요"       | 命令     | 문을 닫으라고 했어요     |
| "늦지 마세요"         | 否定命令 | 늦지 말라고 했어요       |

## 七、本课核心句型小结

| 句型                              | 类型         | 例句               |
| --------------------------------- | ------------ | ------------------ |
| V-ㄴ/는다고 / A-다고 / N-(이)라고 | 陈述转述     | 바쁘다고 했어요    |
| V/A-(으)냐고                      | 疑问转述     | 어디 가냐고 했어요 |
| V-자고                            | 共动转述     | 같이 가자고 했어요 |
| V-(으)라고                        | 命令转述     | 공부하라고 했어요  |

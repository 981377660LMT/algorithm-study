# 제9과: 여행 (旅行)

> 延世韩国语 第二册 · 第9课

---

## 一、学习目标

- 能够描述旅行计划、经历和感受
- **重点语法**：-(으)려면 (想要…的话)、-는 게 좋다 (最好…)、-(으)ㄹ 텐데 (应该会…但)、-다가 (做着做着/中途)
- 学会讨论旅行计划和分享旅行经历
- 掌握交通、住宿、观光相关表达

## 三、核心语法

### 语法1: -(으)려면 — "想要做…的话" ★★★

表示"如果想做某事，就需要/应该…"。是 -(으)려고 하면 的缩略。

**构成**：动词词干 + -(으)려면

```
비행기 표를 싸게 사려면 미리 예약해야 해요.
想买便宜机票的话要提前预订。

한국어를 잘하려면 매일 연습해야 해요.
想韩语好的话要每天练习。

제주도에 가려면 비행기를 타야 해요.
想去济州岛的话要坐飞机。

숙소를 좋은 곳으로 잡으려면 일찍 예약하는 게 좋아요.
想订好的住处的话最好早点预约。

건강하려면 규칙적으로 운동해야 해요.
想健康的话要有规律地运动。
```

---

### 语法2: -는 게 좋다 — "最好…/…比较好" ★★★

给出建议，表示"做…比较好"。

**构成**：动词词干 + **-는 게 좋다** (= -는 것이 좋다)

```
여행 전에 날씨를 확인하는 게 좋아요.
旅行前最好确认天气。

비행기 표는 미리 예매하는 게 좋아요.
机票最好提前预订。

제주도는 렌터카를 빌리는 게 좋아요.
济州岛最好租车。

여행 중에 현금을 좀 가지고 다니는 게 좋아요.
旅行中最好带些现金。

일찍 출발하는 게 좋겠어요.
最好早点出发。
```

**否定：안 -는 게 좋다 / -지 않는 게 좋다**

```
여행 짐은 너무 많이 싸지 않는 게 좋아요.
旅行行李最好不要装太多。

주말에는 차로 가지 않는 게 좋아요. 막히거든요.
周末最好不要开车去。因为会堵。
```

---

### 语法3: -(으)ㄹ 텐데 — "应该会…（但是/所以）" ★★

表示**对未来情况的推测**，后面接担心或建议。

**构成**：动词/形容词词干 + -(으)ㄹ 텐데(요)

```
주말에 사람이 많을 텐데 미리 예약하세요.
周末人应该会很多，提前预约吧。

길이 막힐 텐데 지하철로 가는 게 좋겠어요.
路应该会堵，坐地铁去比较好。

날씨가 추울 텐데 따뜻하게 입으세요.
天气应该会冷，穿暖和点。

음식이 맵을 텐데 괜찮으세요?
食物应该会辣，没事吗？

힘들었을 텐데 고생했어요.
应该很辛苦吧，辛苦了。
```

---

### 语法4: -다가 — "做着做着/中途转到" ★★★

表示**一个动作进行中，中途转换到另一个动作**。

**构成**：动词词干 + -다가

```
# 动作中途转换
걷다가 넘어졌어요.               走着走着摔了。
여행하다가 친구를 만났어요.        旅行途中遇到了朋友。
사진을 찍다가 핸드폰을 떨어뜨렸어요.
拍照时手机掉了。

# 反复/来回
서울에서 살다가 부산으로 이사했어요.
在首尔住着后搬到了釜山。

회사에 다니다가 그만두고 여행을 갔어요.
上着班辞职去旅行了。

# 中途中断
자다가 깼어요.                   睡着睡着醒了。
밥을 먹다가 전화가 왔어요.         正吃饭电话来了。
```

> **-다가 vs -고 vs -아/어서**：
>
> - **-다가**：前一动作进行中途 → 걷다가 넘어졌어요 (走着走着摔了)
> - **-고**：顺序连接 → 밥을 먹고 영화를 봤어요 (吃了饭看了电影)
> - **-아/어서**：因果顺序 → 배가 아파서 병원에 갔어요 (肚子疼去了医院)

## 四、核心会话

### 对话1：讨论旅行计划

```
A: 이번 여름 휴가에 어디 갈 거예요?
   今年暑假要去哪里？

B: 제주도에 가려고 해요.
   打算去济州岛。

A: 좋겠다! 패키지여행이에요, 자유여행이에요?
   真好！跟团还是自由行？

B: 자유여행이요. 자유여행이 더 자유롭잖아요.
   自由行。自由行更自由嘛。

A: 제주도에 가려면 비행기 표를 미리 사는 게 좋아요.
   想去济州岛的话最好提前买机票。
   여름에는 비싸질 텐데요.
   夏天应该会涨价呢。

B: 맞아요. 이번 주에 예매하기로 했어요.
   对。这周决定预订了。

A: 숙소는? 게스트하우스가 어때요?
   住处呢？Guest House怎么样？

B: 에어비앤비를 알아볼까 해요.
   在考虑找Airbnb。
```

### 对话2：旅行经历

```
A: 부산 여행 어땠어요?
   釜山旅行怎么样？

B: 정말 좋았어요! 해운대 바다가 너무 아름다웠어요.
   真的很好！海云台的海太美了。

A: 뭐가 제일 기억에 남아요?
   什么最记忆深刻？

B: 감천문화마을이요. 걸어다니다가 예쁜 벽화를 많이 봤어요.
   甘川文化村。走着走着看到了很多漂亮的壁画。

A: 음식은 어땠어요?
   食物怎么样？

B: 자갈치 시장에서 회를 먹었는데 정말 신선하고 맛있었어요.
   在札嘎其市场吃了生鱼片，真的新鲜又好吃。
   그런데 부산 사투리를 잘 못 알아들었어요.
   但是釜山方言不太听得懂。

A: 하하, 부산 사투리가 좀 다르긴 하죠.
   哈哈，釜山方言确实有点不同。
```

### 对话3：旅行中的问题

```
A: 여행 중에 무슨 일이 있었어요?
   旅行中有什么事？

B: 경주에서 자전거를 타다가 넘어졌어요.
   在庆州骑自行车时摔了。

A: 어머, 괜찮았어요?
   天哪，没事吧？

B: 무릎을 좀 다쳤는데 크게 다치지는 않았어요.
   膝盖受了点伤但没大碍。

A: 여행할 때는 조심하는 게 좋아요.
   旅行时最好小心。

B: 맞아요. 그래도 경주가 정말 좋았어요.
   对。不过庆州真的很好。
   역사적인 건물이 많아서 볼거리가 많았어요.
   因为有很多历史建筑，看点很多。

A: 다음에 저도 가 봐야겠어요.
   下次我也要去看看。
```

**新词**：

- **렌터카** = 租车
- **벽화** = 壁画
- **자갈치 시장** = 札嘎其市场（釜山著名海鲜市场）
- **사투리** = 方言
- **다치다** = 受伤
- **역사적** = 历史性的

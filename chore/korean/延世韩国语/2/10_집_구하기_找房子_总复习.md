## 三、核心语法

### 语法1: -(으)ㄴ/는/(으)ㄹ 것 같다 — "好像……" ★★★★★

第二册**最重要的推测语法**，使用频率极高。

#### 构成

| 词性       | 过去                 | 现在             | 将来/推测        |
| ---------- | -------------------- | ---------------- | ---------------- |
| **动词**   | V-(으)ㄴ 것 같다     | V-는 것 같다     | V-(으)ㄹ 것 같다 |
| **形容词** | A-았/었던 것 같다    | A-(으)ㄴ 것 같다 | A-(으)ㄹ 것 같다 |
| **名词**   | N이었던/였던 것 같다 | N인 것 같다      | N일 것 같다      |

#### 例句

```
# 过去推测
어제 비가 온 것 같아요.           好像昨天下雨了。
그 사람이 벌써 간 것 같아요.       那个人好像已经走了。

# 现在推测
이 집이 좀 작은 것 같아요.        这个房子好像有点小。
지금 비가 오는 것 같아요.          好像现在在下雨。
그 사람은 학생인 것 같아요.        那个人好像是学生。

# 将来推测
내일 비가 올 것 같아요.           好像明天要下雨。
이 집은 비쌀 것 같아요.           这个房子好像会贵。
```

**婉转表达（非常常用）**：

```
이 원룸이 좋을 것 같아요.          这个单间好像不错。
여기가 편할 것 같아요.             这里好像会方便。
이 집은 좀 좁을 것 같은데요.       这个房子好像有点窄呢。
월세가 좀 높은 것 같아요.          月租好像有点高。
```

### 语法2: -아/어 버리다 — "彻底做完/不幸做了" ★★★

表示动作**彻底完成**，带有"完了/遗憾/痛快"等感情色彩。

**构成**：动词词干 + -아/어 버리다

#### 用法1：遗憾/不幸（负面结果）

```
열쇠를 잃어버렸어요.              把钥匙弄丢了。
약속을 잊어버렸어요.              把约定忘了。
좋은 집을 놓쳐 버렸어요.          错过了好房子。
계약금을 이미 내 버렸어요.         已经交了定金了（没办法了）。
핸드폰이 고장 나 버렸어요.         手机坏了。
```

#### 用法2：痛快/干脆（积极）

```
다 먹어 버렸어요.                 全吃光了。
숙제를 다 해 버렸어요.            作业全做完了。
짐을 다 싸 버렸어요.              行李全收拾好了。
결정해 버렸어요.                  索性决定了。
```

> **感情色彩**：-아/어 버리다 给动作加上感情 →
> 먹었어요 (吃了) vs 먹어 버렸어요 (竟然吃掉了/索性吃光了)

---

### 语法3: -(으)ㄹ지 모르다 — "不知道是否/说不定" ★★

#### 用法1：不知道是否

```
이 집이 마음에 들지 모르겠어요.
不知道会不会喜欢这个房子。

언제 이사할지 아직 모르겠어요.
还不知道什么时候搬家。

보증금을 돌려받을 수 있을지 모르겠어요.
不知道能不能拿回押金。
```

#### 用法2：-(으)ㄹ지도 모르다 → "说不定/可能"

```
내일 비가 올지도 몰라요.           说不定明天下雨。
이 집이 벌써 나갔을지도 몰라요.     这个房子说不定已经租出去了。
다음 달에 이사할지도 몰라요.       说不定下个月搬家。
```

---

### 语法4: -도록 — "以便/为了/到…程度" ★★

#### 用法1：目的 "以便/为了"

```
편하게 살 수 있도록 좋은 집을 찾고 있어요.
为了能住得舒服在找好房子。

늦지 않도록 일찍 출발하세요.
为了不迟到早点出发。

공부에 집중할 수 있도록 조용한 곳에 살고 싶어요.
为了能集中学习想住在安静的地方。
```

#### 用法2：程度 "到…程度"

```
목이 아프도록 노래를 불렀어요.
唱歌唱到嗓子疼。

배가 터지도록 먹었어요.
吃到肚子要撑破。

눈물이 나도록 감동적이었어요.
感动到流泪。
```

## 四、核心会话

### 对话1：找房子

```
A: 안녕하세요. 원룸을 찾고 있는데요.
   你好。我在找单间。

부동산: 네, 어떤 조건을 원하세요?
        好的，您想要什么条件？

A: 학교 근처에 월세 50만 원 이하면 좋겠어요.
   学校附近月租50万韩元以下就好。

부동산: 보증금은 얼마 정도 생각하세요?
        押金考虑多少？

A: 500만 원 정도요.
   500万韩元左右。

부동산: 그러면 이 집 어떠세요? 학교에서 걸어서 10분이에요.
        那这个房子怎么样？走路到学校10分钟。
        월세 45만 원에 관리비 5만 원이에요.
        月租45万，管理费5万。

A: 풀옵션인가요?
   是全家具家电吗？

부동산: 네, 세탁기, 냉장고, 에어컨 다 있어요.
        是的，洗衣机、冰箱、空调都有。

A: 채광은 어때요?
   采光怎么样？

부동산: 남향이라서 채광이 좋을 것 같아요. 한번 보시겠어요?
        朝南所以采光应该好。要看一下吗？

A: 네, 보러 갈게요.
   好的，去看看。
```

### 对话2：看房

```
A: 이 집은 깨끗한 것 같은데 좀 좁은 것 같아요.
   这个房子看起来干净但好像有点窄。

부동산: 원룸이라 그럴 수 있는데, 혼자 살기에는 충분해요.
        因为是单间可能会那样，但一个人住足够了。

A: 그런데 화장실에 창문이 없어서 좀 불편할 것 같아요.
   但卫生间没窗户好像会有点不方便。

부동산: 환풍기가 있어서 괜찮을 거예요.
        有换气扇所以应该可以。
      다른 집도 하나 더 보시겠어요?
       要再看另一个房子吗？

A: 네, 좀 넓은 곳을 보고 싶어요.
   好，想看稍大一点的。
   서둘러야 해요. 좋은 집은 금방 나가 버리잖아요.
   得抓紧。好房子马上就被租走了嘛。
```

### 对话3：签约与搬家

```
A: 어제 본 그 집으로 결정했어요.
   决定选昨天看的那个房子了。

B: 잘됐네요! 언제 계약해요?
   太好了！什么时候签约？

A: 이번 주 금요일에 계약하기로 했어요.
   约好了这周五签约。

B: 이사는 언제 해요?
   什么时候搬家？

A: 다음 달 1일에 입주할 것 같아요.
   好像下个月1号入住。
   이삿짐 센터를 예약해 놓아야 하는데 어디가 좋을지
   모르겠어요.
   要预约搬家公司但不知道哪家好。

B: 제가 지난번에 사용한 곳이 좋았는데 소개해 줄까요?
   我上次用的那家不错，给你介绍？

A: 네, 부탁해요! 이사 준비가 잘 되도록 도와주세요.
   好，拜托！帮忙让搬家准备顺利。

B: 그럼요. 이삿짐 싸는 것도 도와줄게요.
   当然。收拾搬家行李也帮你。
```

**新词**：

- **풀옵션** = 全家具家电齐全
- **남향** = 朝南
- **환풍기** = 换气扇
- **서두르다** = 赶快/着急
- **금방** = 马上/很快
- **이삿짐** = 搬家行李

## 五、文化补充：韩国的住房制度

```
🏠 전세 (全租) — 韩国特有的制度
  - 一次性交大笔押金（几千万到几亿韩元）
  - 不用付月租！
  - 退租时全额退还押金
  - 对房东：用押金投资获利
  - 对租客：不用每月付钱

💰 월세 (月租)
  - 보증금 (押金) + 月租
  - 보증금越高，월세越低
  - 例：보증금 500, 월세 45 = "오백에 사십오"

📝 계약 注意事项
  - 계약서 (合同) 认真看！
  - 확정일자 (확정日期) 一定要去住民中心盖章
  - 등기부등본 (房产登记簿) 要确认
  - 계약 기간: 通常 2年
  - 중개 수수료 (中介费): 按比例收取

🚛 이사 (搬家)
  - 이삿짐 센터: 韩国有专业搬家公司
  - "손 없는 날": 传统上搬家要选"没有鬼神的日子"
  - 새집에 이사하면 떡을 돌려요: 搬新家要给邻居送年糕
```

## 六、语法练习

### 练习1：-(으)ㄴ/는 것 같다

用"好像…"表达推测：

1. 이 집 / 좀 작다 → ******\_\_\_******
2. 내일 / 비가 오다 → ******\_\_\_******
3. 그 사람 / 벌써 이사하다 (过去) → ******\_\_\_******

<details>
<summary>点击查看答案</summary>

1. 이 집이 좀 작은 것 같아요. (现在-形容词)
2. 내일 비가 올 것 같아요. (将来)
3. 그 사람이 벌써 이사한 것 같아요. (过去-动词)

</details>

### 练习2：-아/어 버리다

加上感情色彩：

1. 열쇠를 잃다 (遗憾) → ******\_\_\_******
2. 좋은 집을 놓치다 (遗憾) → ******\_\_\_******
3. 짐을 다 싸다 (痛快) → ******\_\_\_******

<details>
<summary>点击查看答案</summary>

1. 열쇠를 잃어버렸어요.
2. 좋은 집을 놓쳐 버렸어요.
3. 짐을 다 싸 버렸어요.

</details>

### 练习3：综合应用

翻译成韩语：

1. 为了不迟到最好早点出发 →
2. 不知道什么时候搬家 →
3. 说不定好房子已经被租走了 →

<details>
<summary>点击查看答案</summary>

1. 늦지 않도록 일찍 출발하는 게 좋아요.
2. 언제 이사할지 모르겠어요.
3. 좋은 집이 벌써 나갔을지도 몰라요.

</details>

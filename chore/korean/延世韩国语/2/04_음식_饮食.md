# 제4과: 음식 (饮食)

> 延世韩国语 第二册 · 第4课

---

## 一、学习目标

- 能够详细描述食物的味道、外观和烹饪方法
- **重点语法**：-아/어 보이다 (看起来)、-(으)ㄴ데 (背景转折)、-기는 하지만 (虽然…但是)、-게 (副词化)
- 学会在餐厅进行复杂对话
- 掌握韩国饮食文化和礼仪

## 三、核心语法

### 语法1: -아/어 보이다 — "看起来……" ★★★

表示凭外观/第一印象做出的判断。

**构成**：形容词词干 + -아/어 보이다

```
# 基本变化
맛있다 → 맛있어 보이다        看起来好吃
예쁘다 → 예뻐 보이다          看起来漂亮
피곤하다 → 피곤해 보이다       看起来累
젊다 → 젊어 보이다            看起来年轻
건강하다 → 건강해 보이다       看起来健康
비싸다 → 비싸 보이다          看起来贵
```

**例句**：

```
이 케이크가 맛있어 보여요!
这个蛋糕看起来好吃！

오늘 기분이 좋아 보여요.
你今天看起来心情好。

이 음식은 맵지 않아 보여요.
这个菜看起来不辣。

그 식당이 비싸 보이는데 실제로는 저렴해요.
那家餐厅看起来贵，实际上很便宜。

어제 야근했어요? 피곤해 보여요.
昨天加班了？看起来很累。
```

> **-아/어 보이다 vs -(으)ㄴ 것 같다**：
>
> - **-아/어 보이다**：基于视觉/外表判断 → 맛있어 보여요 (看着好吃)
> - **-(으)ㄴ 것 같다**：基于各种推测 → 맛있는 것 같아요 (好像好吃)

---

### 语法2: -(으)ㄴ데 / -는데 — 背景·转折·铺垫 ★★★★★

这是韩语中**使用频率最高的连接语尾之一**，贯穿整个韩语学习。

#### 基本含义

1. **背景铺垫**："…，而…" / "…，那么…"
2. **转折**："虽然…但是…"
3. **引出请求/问题的前奏**

#### 构成

| 词性       | 现在           | 过去                  |
| ---------- | -------------- | --------------------- |
| **动词**   | V-**는데**     | V-**았/었는데**       |
| **形容词** | A-**(으)ㄴ데** | A-**았/었는데**       |
| **名词**   | N-**인데**     | N-**이었는데/였는데** |

#### 用法1：背景铺垫（引出话题）

```
저는 일본에서 왔는데 한국어를 배우고 있어요.
我从日本来的，（背景）正在学韩语。

이 식당이 유명한데 같이 가 볼래요?
这家餐厅很有名，（铺垫）要一起去看看吗？

내일 시간이 있는데 뭐 할까요?
明天有时间，（铺垫）做什么好呢？
```

#### 用法2：转折

```
비빔밥을 시켰는데 맛이 없었어요.
点了拌饭但是不好吃。

열심히 공부했는데 시험을 못 봤어요.
努力学了但考试没考好。

좀 비싼데 맛있어요.
有点贵但好吃。
```

#### 用法3：婉转请求/提建议

```
저기요, 물 좀 주시겠는데요.
不好意思，（想说的是）能给点水吗？

다른 메뉴는 없는데요?
（想确认）没有别的菜单吗？
```

> ⚠️ **-(으)ㄴ데 是第二册最重要的语法之一**，后面的课还会不断用到。要培养"先铺垫，再说重点"的韩语思维。

---

### 语法3: -기는 하지만 / -기는 -지만 — "虽然确实…但是…" ★★

承认前半句事实的同时，提出不同的看法或补充。

```
맛있기는 하지만 좀 비싸요.
确实好吃是好吃，但有点贵。

맵기는 하지만 먹을 만해요.
确实辣是辣，但还能吃。

좋기는 한데 너무 멀어요.
确实好是好，但是太远了。

알기는 아는데 설명하기 어려워요.
确实知道是知道，但很难解释。
```

**口语缩略形**：-기는 → -긴

```
맛있긴 하지만 비싸요.        好吃是好吃但贵。
맵긴 한데 괜찮아요.          辣是辣但还行。
크긴 큰데 무거워요.          大是大但重。
```

---

### 语法4: -게 — 副词化语尾 ★★

把形容词变成副词（修饰动词），"…地"。

**构成**：形容词词干 + -게

```
# 变化
맛있다 → 맛있게             好吃地 → 맛있게 드세요 (请好好吃)
예쁘다 → 예쁘게             漂亮地 → 예쁘게 만들었어요 (做得很漂亮)
크다 → 크게                大地 → 크게 말해 주세요 (请大声说)
작다 → 작게                小地 → 작게 잘라 주세요 (请切小)
깨끗하다 → 깨끗하게          干净地 → 깨끗하게 씻어요 (洗干净)
맵다 → 맵게                辣地 → 맵게 해 주세요 (请做辣)
```

**实用例句**：

```
맛있게 드세요! (= 잘 먹겠습니다 的回应)
请好好享用！（韩国人做饭后常说的话）

맵지 않게 해 주세요.
请做得不辣。

싱겁지 않게 간을 맞춰 주세요.
请调味不要太淡。

따뜻하게 입으세요.
请穿暖和。
```

## 四、核心会话

### 对话1：在餐厅

```
A: 여기요! 주문할게요.
   这里！要点单。

B: 네, 뭘 드시겠어요?
   好的，您要吃什么？

A: 이 식당에서 뭐가 제일 맛있어요?
   这家餐厅什么最好吃？

B: 저희 식당은 김치찌개가 유명한데, 한번 드셔 보세요.
   我们餐厅泡菜汤有名，尝尝看。

A: 많이 매워요? 맵지 않게 해 주실 수 있어요?
   很辣吗？能做得不辣吗？

B: 네, 안 맵게 해 드릴게요. 몇 인분 드릴까요?
   好，会做不辣的。给您几人份？

A: 2인분이요. 그리고 공기밥 하나 추가해 주세요.
   两人份。再加一碗米饭。

B: 네, 잠시만 기다려 주세요.
   好的，请稍等。
```

### 对话2：聊美食

```
A: 한국 음식 중에서 뭘 제일 좋아해요?
   韩国料理里最喜欢什么？

B: 저는 삼겹살을 좋아해요. 직접 구워 먹는 게 재미있거든요.
   我喜欢五花肉。因为自己烤着吃很有趣。

A: 저도 삼겹살 좋아하는데, 좀 느끼하지 않아요?
   我也喜欢五花肉，但不是有点油腻吗？

B: 느끼하긴 한데 상추에 싸서 먹으면 괜찮아요.
   确实油腻是油腻，但用生菜包着吃就没事。

A: 아, 맞아요. 한국에서는 고기를 상추에 싸서 먹는 게
   특별해 보여요.
   啊对。在韩国用生菜包肉吃看起来很特别。

B: 네, 처음에는 이상하게 보이는데 먹어 보면 정말 맛있어요.
   是，一开始看着奇怪但吃了就真的好吃。
```

**新词**：

- **삼겹살** = 五花肉（三层肉）
- **직접** = 亲自/直接
- **상추** = 生菜
- **싸다** = 包裹 → 싸서 먹다 = 包着吃
- **이상하다** = 奇怪

### 对话3：做饭

```
A: 오늘 저녁에 뭐 해 먹을까요?
   今天晚上做什么吃？

B: 불고기 어때요? 제가 만들 수 있는데.
   烤肉怎么样？我能做。

A: 좋아요! 재료가 있어요?
   好！有材料吗？

B: 고기는 있는데 양파하고 당근을 사야 해요.
   肉有但要买洋葱和胡萝卜。

A: 그러면 제가 마트에 가서 사 올게요.
   那我去超市买回来。

B: 고마워요. 야채는 작게 잘라 주세요.
   그래야 맛있게 볶을 수 있어요.
   谢谢。蔬菜请切小块。
   那样才能炒得好吃。
```

## 五、韩国饮食文化补充

### 用餐礼仪

```
1. 어른보다 먼저 먹지 않아요.
   不比长辈先吃。

2. 어른 앞에서 술을 마실 때 고개를 돌려요.
   在长辈面前喝酒时要侧头。

3. 밥그릇을 들고 먹지 않아요. (중국과 다른 점!)
   不端着饭碗吃。（和中国不同！）

4. 숟가락으로 밥을, 젓가락으로 반찬을 먹어요.
   用汤匙吃饭，用筷子吃小菜。

5. "잘 먹겠습니다" (吃之前) / "잘 먹었습니다" (吃完后)
   两句用餐敬语必须记住。
```

### 韩国代表饮食

```
🍲 찌개류 (锅类): 김치찌개, 된장찌개, 순두부찌개
🥘 탕류 (汤类): 갈비탕, 삼계탕, 설렁탕
🥩 구이류 (烤类): 삼겹살, 불고기, 갈비
🍜 면류 (面类): 냉면, 비빔국수, 잔치국수
🥗 반찬류 (小菜): 김치, 나물, 젓갈
```

## 六、语法练习

### 练习1：-아/어 보이다

看图片（想象）描述：

1. 一碗热气腾腾的汤 → ******\_\_\_******
2. 一个很累的朋友 → ******\_\_\_******
3. 一件很贵的衣服 → ******\_\_\_******
4. 一道辣的菜 → ******\_\_\_******

<details>
<summary>点击查看答案</summary>

1. 따뜻해 보여요. / 맛있어 보여요.
2. 피곤해 보여요. / 힘들어 보여요.
3. 비싸 보여요.
4. 매워 보여요.

</details>

### 练习2：-(으)ㄴ데/는데 连接

用 -(으)ㄴ데/는데 连接两个句子：

1. 한국 음식을 좋아하다 + 잘 못 만들다 → ******\_\_\_******
2. 이 식당은 유명하다 + 가 본 적이 없다 → ******\_\_\_******
3. 배가 고프다 + 뭐 먹을까요? → ******\_\_\_******

<details>
<summary>点击查看答案</summary>

1. 한국 음식을 좋아하는데 잘 못 만들어요.
2. 이 식당은 유명한데 가 본 적이 없어요.
3. 배가 고픈데 뭐 먹을까요?

</details>

### 练习3：-기는 하지만 承认+转折

1. 김치 / 맵다 / 맛있다 → ******\_\_\_******
2. 이 식당 / 비싸다 / 분위기가 좋다 → ******\_\_\_******
3. 한국어 / 재미있다 / 어렵다 → ******\_\_\_******

<details>
<summary>点击查看答案</summary>

1. 김치가 맵기는 하지만 맛있어요. / 맵긴 한데 맛있어요.
2. 이 식당이 비싸기는 하지만 분위기가 좋아요.
3. 한국어가 재미있기는 하지만 어려워요.

</details>

# 제1과: 취미 (兴趣爱好)

> 延世韩国语 第二册 · 第1课

---

## 一、学习目标

- 能够谈论自己的兴趣爱好和特长
- **重点语法**：-(으)ㄴ 적이 있다/없다 (经验)、-기 (名词化)、-(으)ㄹ 줄 알다/모르다 (会/不会)、-(으)면서 (同时)
- 掌握多种休闲活动的表达方式
- 学会用更丰富的句式描述爱好

## 三、核心语法

### 语法1: -(으)ㄴ 적이 있다/없다 — "做过/没做过" ★★★

表示是否有过某种**经历/经验**。

**构成**：动词词干 + -(으)ㄴ 적이 있다/없다

- 元音/ㄹ结尾 → **-ㄴ 적이 있다**
- 辅音结尾 → **-은 적이 있다**

```
# 有过经验
한국에 간 적이 있어요.                去过韩国。
비빔밥을 먹은 적이 있어요.            吃过拌饭。
스키를 탄 적이 있어요.               滑过雪。
한국 드라마를 본 적이 있어요.          看过韩剧。

# 没有过经验
일본에 간 적이 없어요.               没去过日本。
번지점프를 한 적이 없어요.            没蹦过极。
한국어로 편지를 쓴 적이 없어요.        没用韩语写过信。

# 疑问形
제주도에 가 본 적이 있어요?            去过济州岛吗？
김치를 만들어 본 적이 있어요?          做过泡菜吗？
```

> **-(으)ㄴ 적이 있다 vs -아/어 본 적이 있다**：
>
> - 둘 다 표현 가능하지만, "-아/어 본 적이 있다" 更强调"尝试过"的语感
> - 한국에 **간** 적이 있어요 = 한국에 **가 본** 적이 있어요（都OK）
> - 但日常口语中 **-아/어 봤어요** 更常用：한국에 가 봤어요?

---

### 语法2: V-기 — 动词名词化 ★★★

将动词变为名词，相当于英语的 "-ing"。用途非常广泛。

**构成**：动词词干 + **-기**

```
# 基本名词化
읽기    读（名词）: 읽기 시험 (阅读考试)
쓰기    写（名词）: 쓰기 연습 (写作练习)
듣기    听（名词）: 듣기 평가 (听力测试)
말하기  说（名词）: 말하기 대회 (演讲比赛)
```

#### 用法1: V-기가 A — "做某事是……的"

```
한국어를 배우기가 어려워요.          学韩语很难。
이 노래를 부르기가 쉬워요.          唱这首歌很容易。
아침에 일어나기가 힘들어요.          早上起床很困难。
혼자 여행하기가 무서워요.           一个人旅行很害怕。
```

#### 用法2: V-기를 좋아하다/싫어하다 — "喜欢/讨厌做"

```
저는 음악 듣기를 좋아해요.          我喜欢听音乐。
요리하기를 싫어해요.               讨厌做饭。
```

> **-기 vs -는 것** 都是动词名词化，但用法稍有不同：
>
> - **-기**：更常用于固定搭配（如 -기 전에, -기 때문에, -기로 하다）
> - **-는 것**：更通用，可用于几乎所有场合
> - 요리하**기**를 좋아해요 ≈ 요리하**는 것**을 좋아해요（都可以）

#### 用法3: 固定搭配

| 搭配            | 意思      | 例句                                  |
| --------------- | --------- | ------------------------------------- |
| -기 전에        | 做…之前   | 자기 전에 책을 읽어요                 |
| -기 후에        | 做…之后   | 운동하기 후에 → 但更常用 -(으)ㄴ 후에 |
| -기 시작하다    | 开始做    | 한국어를 배우기 시작했어요            |
| -기로 하다      | 决定做    | 여행을 가기로 했어요                  |
| -기 때문에      | 因为做    | 바쁘기 때문에 못 갔어요               |
| -기 쉽다/어렵다 | 容易/难做 | 이해하기 쉬워요                       |
| -기 위해(서)    | 为了做    | 건강을 위해 운동해요                  |

---

### 语法3: -(으)ㄹ 줄 알다/모르다 — "会/不会做" ★★★

表示是否具备某种**技能/能力**。

**构成**：

- 元音/ㄹ结尾 → **-ㄹ 줄 알다/모르다**
- 辅音结尾 → **-을 줄 알다/모르다**

```
# 会做
피아노를 칠 줄 알아요.             会弹钢琴。
한국어를 할 줄 알아요.             会说韩语。
자전거를 탈 줄 알아요.             会骑自行车。
수영할 줄 알아요.                 会游泳。
요리할 줄 알아요.                 会做饭。

# 不会做
운전할 줄 몰라요.                 不会开车。
기타를 칠 줄 몰라요.              不会弹吉他。
스키를 탈 줄 몰라요.              不会滑雪。
```

> **-(으)ㄹ 줄 알다 vs -(으)ㄹ 수 있다 区别**：
> | | -(으)ㄹ 줄 알다 | -(으)ㄹ 수 있다 |
> |--|-----------------|----------------|
> | 含义 | 具备技能/知道方法 | 有能力/有可能性 |
> | 重点 | "我学过/我懂" | "我能够/情况允许" |
> | 例 | 수영할 줄 알아요 (学过游泳) | 수영할 수 있어요 (能游泳/身体允许) |
>
> - 피아노를 칠 **줄** 알아요 → 我学过/会弹钢琴
> - 피아노를 칠 **수** 있어요 → 我能弹钢琴（有空/有琴/身体OK）

---

### 语法4: V-(으)면서 — "一边……一边……" ★★

表示两个动作**同时进行**（同一主语）。

**构成**：

- 元音/ㄹ结尾 → **-면서**
- 辅音结尾 → **-으면서**

```
음악을 들으면서 공부해요.           一边听音乐一边学习。
커피를 마시면서 신문을 읽어요.       一边喝咖啡一边看报纸。
운동하면서 노래를 들어요.           一边运动一边听歌。
텔레비전을 보면서 밥을 먹어요.       一边看电视一边吃饭。
걸으면서 이야기해요.               一边走一边聊天。
```

> **重要制约**：-(으)면서 的前后动作**主语必须相同**。
>
> - 저는 음악을 들으면서 (저는) 공부해요 ✓（同一人）
> - 저는 음악을 들으면서 친구는 잠을 자요 ✗（不同人→要用 -는 동안）

---

### 语法5: -게 되다 — "变得……/开始……"

表示某种变化的结果或非意愿的结果。

```
한국 친구를 사귀게 되었어요.         结交了韩国朋友。（自然而然地）
한국어를 좋아하게 되었어요.          开始喜欢韩语了。
이 회사에서 일하게 되었어요.         开始在这家公司工作了。
한국 문화에 관심을 갖게 되었어요.     开始对韩国文化产生兴趣了。
서울에 살게 되었어요.               开始住在首尔了。
```

> **-게 되다** 强调"非刻意的变化"或"自然而然的结果"，不是自己主动选择的。

## 四、核心会话

### 对话1：谈论爱好

```
A: 왕웨이 씨, 취미가 뭐예요?
   王伟，你的兴趣爱好是什么？

B: 저는 사진 찍는 것을 좋아해요. 주말마다 사진을 찍으러 나가요.
   我喜欢拍照。每个周末都出去拍照。

A: 그래요? 어떤 사진을 찍어요?
   是吗？拍什么样的照片？

B: 주로 풍경 사진을 찍어요. 산이나 바다 풍경을 좋아해요.
   主要拍风景照。喜欢山或海的风景。

A: 멋있네요! 저는 그림 그리기가 취미예요.
   好帅！我的爱好是画画。

B: 아, 정말요? 어릴 때부터 그렸어요?
   啊，真的吗？从小就画吗？

A: 네, 어릴 때부터 그림을 그리기 시작했어요.
   벌써 십 년이 넘었어요.
   是的，从小就开始画画了。已经超过10年了。
```

**新词**：

- **주말마다** = 每个周末（-마다 = 每）
- **나가다** = 出去
- **풍경** = 风景
- **주로** = 主要
- **멋있다** = 帅/酷
- **어릴 때** = 小时候
- **벌써** = 已经
- **넘다** = 超过

### 对话2：谈论特长

```
A: 혹시 악기를 연주할 줄 알아요?
   你会演奏乐器吗？

B: 네, 기타를 칠 줄 알아요. 고등학교 때 배웠어요.
   会，我会弹吉他。高中时学的。

A: 저도 기타를 배우고 싶어요. 기타 치기가 어려워요?
   我也想学吉他。弹吉他难吗？

B: 처음에는 좀 어렵지만, 연습하면 괜찮아요.
   一开始有点难，但练习的话就还好。

A: 얼마나 연습하면 잘 칠 수 있어요?
   练多久能弹好？

B: 매일 삼십 분씩 연습하면 한 달 후에 간단한 노래를 칠 수 있어요.
   每天练30分钟的话，一个月后能弹简单的歌。

A: 그래요? 그러면 저도 내일부터 시작할래요!
   是吗？那我也从明天开始！
```

**新词**：

- **혹시** = 万一/或许/碰巧
- **악기** = 乐器
- **고등학교** = 高中
- **처음에는** = 一开始
- **연습하다** = 练习
- **매일** = 每天
- **-씩** = 每（量词后缀）
- **후에** = …之后
- **간단하다** = 简单的

### 对话3：经历分享

```
A: 스카이다이빙을 해 본 적이 있어요?
   跳过伞吗？

B: 아니요, 해 본 적이 없어요. 높은 곳이 무서워요.
   没有，没试过。怕高。

A: 저는 작년에 해 봤는데 정말 재미있었어요.
   我去年试了一次，真的很有趣。

B: 무섭지 않았어요?
   不害怕吗？

A: 처음에는 무서웠지만 뛰고 나서는 정말 짜릿했어요.
   一开始很害怕，但跳完之后真的很刺激。

B: 그래요? 저도 한번 도전해 보고 싶어지네요.
   是吗？我也开始想挑战一下了。
```

**新词**：

- **스카이다이빙** = 跳伞
- **높다** = 高的
- **무섭다** = 可怕的/害怕
- **작년** = 去年
- **-는데** = …但是/…呢（转折/背景）
- **뛰다** = 跳
- **-고 나서** = 做完之后
- **짜릿하다** = 刺激/惊心动魄
- **도전하다** = 挑战

## 五、관형사형 (冠形词形/修饰形) 入门

### 动词的修饰形 — 过去/现在/将来

这是第二册的核心语法之一，用于动词修饰名词：

| 时态     | 规则         | 例                       |
| -------- | ------------ | ------------------------ |
| **过去** | V-(으)ㄴ + N | 먹은 음식 (吃过的食物)   |
| **现在** | V-는 + N     | 먹는 음식 (正在吃的食物) |
| **将来** | V-(으)ㄹ + N | 먹을 음식 (将要吃的食物) |

```
# 过去修饰 -(으)ㄴ
어제 본 영화가 재미있었어요.          昨天看的电影很有趣。
지난주에 만난 사람이 누구예요?        上周见的人是谁？
제가 읽은 책을 추천해 줄게요.         推荐我看过的书。

# 现在修饰 -는
지금 읽는 책이 뭐예요?              你现在读的书是什么？
매일 마시는 커피가 있어요.           有每天喝的咖啡。
한국어를 배우는 학생이 많아요.        学韩语的学生很多。

# 将来修饰 -(으)ㄹ
내일 볼 영화를 골랐어요.            选好了明天要看的电影。
마실 물이 없어요.                  没有要喝的水。
할 일이 많아요.                    要做的事情很多。
```

> **形容词的修饰形**（复习+对比）：
>
> - 形容词现在时用 **-(으)ㄴ**：큰 집, 좋은 날씨, 예쁜 꽃
> - 形容词过去时用 **-았/었던**：예뻤던 꽃 (曾经美丽的花)
> - **动词 vs 形容词 修饰形对比**：
>   - 动词现在：-는 (먹는)
>   - 形容词现在：-(으)ㄴ (큰)
>   - ⚠️ 千万不要搞混！

## 八、本课核心句型小结

| 句型                   | 用法          | 例句                          |
| ---------------------- | ------------- | ----------------------------- |
| -(으)ㄴ 적이 있다/없다 | 有/没有经历   | 가 본 적이 있어요             |
| V-기                   | 动词名词化    | 읽기, 쓰기, 배우기가 어려워요 |
| -(으)ㄹ 줄 알다/모르다 | 会/不会       | 칠 줄 알아요                  |

| V-(으)ㄴ/는/(으)ㄹ + N | 动词修饰名词  | 본 영화, 보는 영화, 볼 영화   |

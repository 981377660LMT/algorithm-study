# 제2과: 일상생활 (日常生活)

> 延世韩国语 第二册 · 第2课

---

## 一、学习目标

- 能够描述复杂的日常生活和习惯
- **重点语法**：-는 동안 (期间)、-기 전에 (之前)、-(으)ㄴ 후에 (之后)、-(으)ㄹ 때 (的时候)
- 掌握 -아/어야 하다 的各种应用
- 学会描述习惯和规律性活动

## 三、核心语法

### 语法1: V-는 동안 — "在做……的期间" ★★★

表示某个动作进行**期间**，另一个动作同时发生。

**构成**：动词词干 + **-는 동안(에)**

```
# 基本用法
제가 요리하는 동안 남편이 청소했어요.
我做饭的期间丈夫打扫了卫生。

수업을 듣는 동안 졸았어요.
听课的时候打瞌睡了。

비가 오는 동안 카페에서 기다렸어요.
下雨期间在咖啡厅等着。

엄마가 전화하는 동안 아이가 울었어요.
妈妈打电话期间孩子哭了。
```

**名词 + 동안**：

```
방학 동안 여행을 갔어요.           放假期间去旅行了。
두 시간 동안 공부했어요.           学了两个小时。
삼 년 동안 한국에서 살았어요.       在韩国住了三年。
```

> **-는 동안 vs -(으)면서 区别**：
>
> - **-(으)면서**：同一主语同时做两件事 → 음악을 들으면서 공부해요
> - **-는 동안**：两个不同主语各做各的，或不同的时间段 → 내가 공부하는 동안 동생은 잤어요

---

### 语法2: V-기 전에 — "做……之前" ★★★

**构成**：动词词干 + **-기 전에**

```
자기 전에 이를 닦아요.              睡觉之前刷牙。
수업 시작하기 전에 복습해요.         上课开始之前复习。
밥 먹기 전에 손을 씻어요.           吃饭之前洗手。
출근하기 전에 운동해요.             上班之前运动。
한국에 오기 전에 한국어를 공부했어요.  来韩国之前学了韩语。
```

**名词 + 전에**：

```
수업 전에 = 수업을 시작하기 전에    上课之前
식사 전에                         吃饭之前
출발 전에                         出发之前
삼 년 전에                        三年之前
```

---

### 语法3: V-(으)ㄴ 후에 / V-고 나서 — "做……之后" ★★★

#### -(으)ㄴ 후에

**构成**：动词词干 + -(으)ㄴ 후에

- 元音/ㄹ结尾 → **-ㄴ 후에**
- 辅音结尾 → **-은 후에**

```
밥을 먹은 후에 커피를 마셔요.        吃完饭后喝咖啡。
수업이 끝난 후에 도서관에 가요.      下课后去图书馆。
졸업한 후에 회사에 다녔어요.         毕业后上班了。
운동한 후에 샤워해요.               运动后洗澡。
```

#### -고 나서 (做完后)

**构成**：动词词干 + **-고 나서** （强调先完成前一个动作）

```
손을 씻고 나서 밥을 먹어요.          洗完手后吃饭。
숙제를 하고 나서 게임을 해요.        做完作业后玩游戏。
이 책을 읽고 나서 빌려줄게요.        读完这本书后借给你。
```

**名词 + 후에**：

```
식사 후에                          吃饭之后
졸업 후에                          毕业之后
한 시간 후에                        一个小时之后
```

> **-(으)ㄴ 후에 vs -고 나서 vs -고**：
>
> - **-(으)ㄴ 후에**：正式，明确的"之后" → 졸업한 후에 취직했어요
> - **-고 나서**：强调"完成后才" → 다 먹고 나서 나가자
> - **-고**：简单顺序连接 → 밥을 먹고 커피를 마셨어요

---

### 语法4: -(으)ㄹ 때 — "做……的时候" ★★★

**构成**：

- 动词/形容词词干 + -(으)ㄹ 때 → 将来/习惯性的"时候"
- 动词词干 + -(으)ㄴ 때 → 过去的"时候"（回忆）
- 名词 + 때 → "…的时候"

#### 现在/习惯性：V-(으)ㄹ 때

```
시간이 있을 때 운동해요.             有空的时候运动。
심심할 때 영화를 봐요.              无聊的时候看电影。
피곤할 때 커피를 마셔요.             累的时候喝咖啡。
모를 때 선생님께 물어보세요.          不懂的时候问老师。
```

#### 过去的某个时间点：V-았/었을 때

```
한국에 처음 왔을 때 한국어를 못 했어요.
第一次来韩国的时候不会韩语。

고등학생이었을 때 매일 운동했어요.
高中生的时候每天运动。
```

#### 名词 + 때

```
어릴 때 꿈이 뭐였어요?              小时候梦想是什么？
대학 때 친구를 만났어요.             大学时遇到了朋友。
점심시간 때 뭐 해요?               午饭时间干什么？
방학 때 여행 갈 거예요.             放假时要去旅行。
```

---

### 语法5: -거든요 — "因为（你不知道但其实是）……" ★★

表示向对方**解释理由/说明原因**，带有"你可能不知道，其实是这样的"语气。

```
A: 왜 한국어를 배워요?
   为什么学韩语？
B: 한국 드라마를 좋아하거든요.
   因为我喜欢韩剧嘛（其实是这样的）。

A: 내일 같이 갈래요?
   明天一起去吗？
B: 미안해요. 내일 시험이 있거든요.
   对不起。因为明天有考试。

A: 왜 이렇게 피곤해요?
   怎么这么累？
B: 어제 야근했거든요.
   因为昨天加班了。
```

> **-거든요 vs -아/어서 vs -(으)니까**：
>
> - **-아/어서**：客观原因陈述
> - **-(으)니까**：主观强调理由（可接命令/共动）
> - **-거든요**：向对方解释新信息，"你不知道吧？其实是因为…"

## 四、核心会话

### 对话1：描述日常

```
A: 보통 하루 일과가 어떻게 돼요?
   通常一天的日程是怎样的？

B: 아침 여섯 시에 일어나요.
   일어난 후에 샤워하고 아침을 먹어요.
   早上6点起床。起床后洗澡吃早饭。

A: 아침을 꼭 먹어요?
   一定吃早饭吗？

B: 네, 아침을 안 먹으면 수업 시간에 집중을 못 하거든요.
   是的，因为不吃早饭的话上课不能集中精力。

A: 수업이 끝난 후에는 뭐 해요?
   下课后干什么？

B: 도서관에서 숙제를 하고 나서 알바하러 가요.
   在图书馆做完作业后去打工。

A: 바쁘네요! 쉬는 시간은 있어요?
   好忙啊！有休息时间吗？

B: 주말에는 좀 쉬어요. 친구를 만나거나 영화를 봐요.
   周末稍微休息。见朋友或看电影。
```

**新词**：

- **일과** = 日程/一天的活动
- **꼭** = 一定
- **집중하다** = 集中
- **쉬는 시간** = 休息时间

### 对话2：时间安排

```
A: 이번 주말에 같이 등산 갈래요?
   这周末一起去爬山吗？

B: 좋아요! 토요일 아침 어때요?
   好啊！周六早上怎么样？

A: 출발하기 전에 도시락을 준비해야 돼요.
   出发之前要准备便当。

B: 그러면 아침 일곱 시에 만나서 준비하고 여덟 시에 출발하면 어때요?
   那早上7点见面准备，8点出发怎么样？

A: 좋아요. 등산하는 동안 사진도 많이 찍어요.
   好。爬山的时候多拍照。

B: 네! 산에 도착한 후에 같이 점심 먹어요.
   好！到山上后一起吃午饭。
```

### 对话3：回忆过去

```
A: 어릴 때 뭐 되고 싶었어요?
   小时候想当什么？

B: 저는 어릴 때 가수가 되고 싶었어요.
   我小时候想当歌手。

A: 진짜요? 노래를 잘해요?
   真的吗？唱歌唱得好吗？

B: 네, 초등학교 때 합창단에 있었거든요.
   是的，因为小学时在合唱团。

A: 그런데 왜 가수가 안 됐어요?
   那为什么没当歌手呢？

B: 중학교 때 프로그래밍을 배우기 시작했는데,
   그때부터 프로그래머가 되고 싶어졌어요.
   初中时开始学编程，
   从那时起就想当程序员了。

A: 그래서 지금 프로그래머예요?
   所以现在是程序员吗？

B: 네, 대학교를 졸업한 후에 IT 회사에 들어갔어요.
   是的，大学毕业后进了IT公司。
```

**新词**：

- **되다** = 成为
- **진짜** = 真的
- **초등학교** = 小学
- **합창단** = 合唱团
- **중학교** = 初中/中学
- **프로그래밍** = 编程
- **프로그래머** = 程序员
- **졸업하다** = 毕业
- **들어가다** = 进入

## 五、时间顺序表达总结

用一段完整的叙述来展示所有时间表达：

```
아침에 일어나자마자 핸드폰을 확인해요.
一早起来就看手机。

세수하기 전에 물 한 잔을 마셔요.
洗脸之前喝一杯水。

샤워한 후에 옷을 입어요.
洗完澡后穿衣服。

출근 준비를 하는 동안 뉴스를 들어요.
准备上班期间听新闻。

출근할 때 지하철에서 책을 읽어요.
上班的时候在地铁里读书。

일하고 나서 헬스장에 가요.
工作完后去健身房。

운동하면서 음악을 들어요.
一边运动一边听音乐。

자기 전에 일기를 써요.
睡觉之前写日记。
```

**新词**：

- **-자마자** = 一…就… （下节课详讲）
- **확인하다** = 确认/查看
- **준비하다** = 准备

## 六、语法练习

### 练习1：V-기 전에 / V-(으)ㄴ 후에

将两个动作按顺序连接（用括号中的语法）：

1. 밥을 먹다 → 이를 닦다 (-기 전에) → ******\_\_\_******
2. 운동하다 → 샤워하다 (-(으)ㄴ 후에) → ******\_\_\_******
3. 한국에 오다 → 한국어를 배우다 (-기 전에) → ******\_\_\_******
4. 졸업하다 → 취직하다 (-고 나서) → ******\_\_\_******

<details>
<summary>点击查看答案</summary>

1. 밥을 먹기 전에 이를 닦아요. (吃饭之前刷牙)
   或 이를 닦은 후에 밥을 먹어요. (刷完牙后吃饭)
2. 운동한 후에 샤워해요.
3. 한국에 오기 전에 한국어를 배웠어요.
4. 졸업하고 나서 취직했어요.

</details>

## 七、本课核心句型小结

| 句型          | 用法             | 例句                        |
| ------------- | ---------------- | --------------------------- |
| V-는 동안     | 做…期间          | 기다리는 동안 책을 읽었어요 |
| V-기 전에     | 做…之前          | 자기 전에 이를 닦아요       |
| V-(으)ㄴ 후에 | 做…之后          | 먹은 후에 커피를 마셔요     |
| V-고 나서     | 做完…后          | 숙제를 하고 나서 놀아요     |
| V-(으)ㄹ 때   | 做…的时候        | 심심할 때 영화를 봐요       |
| -거든요       | 因为(解释新信息) | 내일 시험이 있거든요        |

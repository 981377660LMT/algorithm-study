## 三、核心语法

### 语法1: 관형사형 심화 — 动词/形容词修饰名词 ★★★★

这是韩语中**极重要的语法**：用动词/形容词修饰名词，相当于中文的"…的（名词）"。

#### 形容词修饰名词：A-(으)ㄴ + N

| 形容词            | -(으)ㄴ + N | 例句                        |
| ----------------- | ----------- | --------------------------- |
| 덥다 → 더운       | 더운 날     | 더운 날에는 수영하고 싶어요 |
| 춥다 → 추운       | 추운 겨울   | 추운 겨울이 싫어요          |
| 맑다 → 맑은       | 맑은 하늘   | 맑은 하늘이 예뻐요          |
| 따뜻하다 → 따뜻한 | 따뜻한 봄   | 따뜻한 봄을 좋아해요        |
| 시원하다 → 시원한 | 시원한 바람 | 시원한 바람이 불어요        |

#### 动词修饰名词：时态三形式

| 时态     | 形式         | 例句                                                     |
| -------- | ------------ | -------------------------------------------------------- |
| **过去** | V-(으)ㄴ + N | 어제 **본** 영화가 재미있었어요 (昨天**看的**电影很有趣) |
| **现在** | V-는 + N     | 지금 **읽는** 책이 뭐예요? (现在**读的**书是什么?)       |
| **将来** | V-(으)ㄹ + N | 내일 **만날** 사람이 누구예요? (明天**要见的**人是谁?)   |

**详细变化表**：

```
# ★ 过去 -(으)ㄴ
먹다 → 먹은 (吃了的)     먹은 음식 = 吃过的食物
보다 → 본 (看了的)       본 영화 = 看过的电影
만들다 → 만든 (做了的)    만든 요리 = 做的菜

# ★ 现在 -는
먹다 → 먹는 (正在吃的)   먹는 음식 = 正在吃的食物
보다 → 보는 (正在看的)   지금 보는 드라마 = 正在看的电视剧
만들다 → 만드는 (正做的)  지금 만드는 요리 = 正在做的菜

# ★ 将来 -(으)ㄹ
먹다 → 먹을 (要吃的)     먹을 음식 = 要吃的食物
보다 → 볼 (要看的)       볼 영화 = 要看的电影
만들다 → 만들 (要做的)    만들 요리 = 要做的菜
```

**实际运用**：

```
제가 좋아하는 계절은 봄이에요.      我喜欢的季节是春天。
어제 내린 비 때문에 길이 미끄러워요.  因为昨天下的雨路很滑。
내일 입을 옷을 준비했어요.          准备了明天要穿的衣服。
서울에서 살고 있는 친구가 전화했어요. 住在首尔的朋友打电话了。
```

> ⚠️ **형용사 (形容词) vs 동사 (动词) 的区别**：
>
> - 形容词修饰名词只用 **A-(으)ㄴ**（因为形容词没有时态变化作定语）
> - 动词修饰名词有时态区分：**V-(으)ㄴ** (过去), **V-는** (现在), **V-(으)ㄹ** (将来)

### 语法2: -아/어지다 — "变得……" ★★★

表示状态的**自然变化**，"变得…起来了"。

**构成**：形容词词干 + -아/어지다

```
# 기본 변화
따뜻하다 → 따뜻해지다          变暖
추워지다                       变冷
더워지다                       变热
좋아지다                       变好
나빠지다                       变差
길어지다                       变长
짧아지다                       变短
예뻐지다                       变漂亮
```

**例句**：

```
봄이 되면 날씨가 따뜻해져요.
到春天天气变暖。

요즘 낮이 짧아졌어요.
最近白天变短了。

한국어가 점점 좋아지고 있어요.
韩语在一点点变好。

비가 온 후에 공기가 깨끗해졌어요.
下雨之后空气变干净了。

키가 많이 커졌네요!
长高了很多啊！

운동을 시작한 후에 건강해졌어요.
开始运动后变健康了。
```

> **注意**：-아/어지다 只能接 **形容词**，不能接动词。
>
> - ✅ 따뜻해지다 (变暖)
> - ❌ 먹어지다 (❌ 不能说"变吃")

---

### 语法3: -(으)ㄴ가요? / -나요? — 柔和疑问 ★★

比 -아/어요? 更柔和、礼貌的疑问形式，常用于询问信息。

**构成**：

- 形容词词干 + -(으)ㄴ가요? → 추운가요? / 따뜻한가요?
- 动词词干 + -나요? → 오나요? / 먹나요?
- 名词 + 인가요? → 학생인가요?

```
내일 날씨가 좋은가요?            明天天气好吗？
몇 시에 출발하나요?              几点出发？
이 근처에 은행이 있나요?          这附近有银行吗？
혹시 한국 사람인가요?            请问你是韩国人吗？
이 약을 언제 먹으면 되나요?       这个药什么时候吃就行？
```

---

### 语法4: -대요 / -래요 (传闻) — 间接引语入门 ★★

转述听来的消息，"听说……"。

**构成**：

- 陈述句 → -대요 (= -다고 해요 的缩写)
- 命令/请求 → -래요 (= -(으)라고 해요 的缩写)

```
# -대요 (转述陈述)
내일 비가 온대요.                听说明天下雨。
서울이 춥대요.                   听说首尔很冷。
그 식당이 맛있대요.              听说那家餐厅好吃。
민수 씨가 다음 주에 결혼한대요.   听说民秀下周结婚。

# -래요 (转述命令/请求)
선생님이 숙제를 내래요.          老师说让交作业。
엄마가 빨리 오래요.              妈妈说让快点来。
의사가 약을 먹으래요.            医生说让吃药。
```

> 📝 这是间接引语 (간접화법) 的入门，第7课会全面展开。

## 四、核心会话

### 对话1：谈论天气

```
A: 요즘 날씨가 어때요?
   最近天气怎么样？

B: 점점 추워지고 있어요. 이제 겨울이 오는 것 같아요.
   越来越冷了。好像冬天要来了。

A: 한국 겨울은 많이 춥나요?
   韩国冬天很冷吗？

B: 네, 영하 10도까지 내려가요.
   是的，降到零下10度。

A: 와, 정말 추운 겨울이네요!
   哇，真是寒冷的冬天啊！

B: 그래서 두꺼운 옷을 준비해야 해요.
   所以要准备厚衣服。
```

### 对话2：季节偏好

```
A: 가장 좋아하는 계절이 뭐예요?
   最喜欢的季节是什么？

B: 저는 가을을 좋아해요. 시원한 바람이 불고
   단풍이 정말 예뻐지거든요.
   我喜欢秋天。因为凉风吹来，红叶变得很漂亮。

A: 가을에 보통 뭐 해요?
   秋天通常做什么？

B: 산에 가서 단풍 구경을 해요.
   去山上赏红叶。
   작년에 설악산에서 본 단풍이 정말 아름다웠어요.
   去年在雪岳山看的红叶真的很美。

A: 뉴스에서 이번 주말에 단풍이 절정이래요.
   新闻说这周末红叶到高峰。

B: 그러면 이번 주말에 단풍 구경 갈래요?
   那这周末去赏红叶吗？
```

**新词**：

- **절정** = 高峰/鼎盛
- **두꺼운** = 厚的 (두껍다 → 두꺼운: ㅂ不规则)
- **아름답다** = 美丽 (아름다운: ㅂ不规则)

### 对话3：天气预报

```
A: 일기예보 봤어요?
   看天气预报了吗？

B: 네, 내일부터 장마가 시작된대요.
   是，听说从明天开始进入梅雨季。

A: 얼마 동안 계속되나요?
   持续多久？

B: 보통 2주에서 3주 동안 계속돼요.
   通常持续2到3周。

A: 장마 때 뭘 주의해야 돼요?
   梅雨季要注意什么？

B: 우산을 꼭 가지고 다녀야 하고,
   습해지니까 빨래가 잘 안 말라요.
   一定要随身带伞，
   因为变潮湿所以衣服不容易干。
```

## 五、韩国四季文化补充

```
🌸 봄 (春天 3~5月)
- 벚꽃 축제: 여의도, 진해 벚꽃 축제가 유명해요.
  樱花节：汝矣岛、镇海樱花节很有名。
- 황사/미세먼지: 봄에는 중국에서 황사가 와요.
  沙尘暴/雾霾：春天从中国刮来沙尘暴。

☀️ 여름 (夏天 6~8月)
- 장마: 6월 말~7월에 약 한 달 동안 비가 많이 와요.
  梅雨：6月底到7月大约一个月下很多雨。
- 복날: 초복, 중복, 말복에 삼계탕을 먹어요.
  伏天：初伏、中伏、末伏吃参鸡汤。

🍂 가을 (秋天 9~11月)
- 추석: 한국의 가장 큰 명절이에요.
  中秋节：韩国最大的传统节日。
- 단풍: 설악산, 내장산의 단풍이 유명해요.
  红叶：雪岳山、内藏山的红叶很有名。

❄️ 겨울 (冬天 12~2月)
- 김장: 겨울에 먹을 김치를 많이 담가요.
  腌泡菜：腌制冬天吃的大量泡菜。
- 스키장: 강원도에 스키장이 많아요.
  滑雪场：江原道有很多滑雪场。
```

## 六、语法练习

### 练习1：관형사형（定语形式）

用定语形式完成句子：

1. 제가 (좋아하다 - 现在) \_\_\_ 음식은 비빔밥이에요.
2. 어제 (사다 - 过去) \_\_\_ 옷이 마음에 들어요.
3. 내일 (보다 - 将来) \_\_\_ 영화 제목이 뭐예요?
4. (맛있다) \_\_\_ 식당을 알아요?

<details>
<summary>点击查看答案</summary>

1. 좋아하는 (现在进行/习惯 → -는)
2. 산 (사다 → 산, 过去 → -(으)ㄴ)
3. 볼 (将来 → -(으)ㄹ)
4. 맛있는 (맛있다 虽然是形容词,但以 있다 结尾的用 -는)

</details>

### 练习2：-아/어지다 变化句

把括号里的形容词变成"变得…"：

1. 봄이 되면 날씨가 (따뜻하다) → ******\_\_\_******
2. 한국어를 공부한 후에 한국어가 (쉽다) → ******\_\_\_******
3. 운동을 시작하니까 몸이 (건강하다) → ******\_\_\_******
4. 가을이 되면 밤이 (길다) → ******\_\_\_******

<details>
<summary>点击查看答案</summary>

1. 따뜻해져요 / 따뜻해졌어요
2. 쉬워졌어요 (쉽다 → 쉬워지다: ㅂ不规则)
3. 건강해졌어요
4. 길어져요 / 길어졌어요

</details>

### 练习3：-대요 传闻练习

用 -대요 转述以下内容：

1. (일기예보) 내일은 맑겠습니다 → ******\_\_\_******
2. (친구) 부산이 좋아요 → ******\_\_\_******
3. (뉴스) 다음 주에 태풍이 와요 → ******\_\_\_******

<details>
<summary>点击查看答案</summary>

1. 내일은 맑대요. (听说明天晴)
2. 부산이 좋대요. (听说釜山好)
3. 다음 주에 태풍이 온대요. (听说下周来台风)

</details>

## 七、本课核心句型小结

| 句型                  | 用法             | 例句               |
| --------------------- | ---------------- | ------------------ |
| A-(으)ㄴ N            | 形容词修饰名词   | 따뜻한 봄이 좋아요 |
| V-(으)ㄴ N            | 过去动词修饰名词 | 어제 본 영화       |
| V-는 N                | 现在动词修饰名词 | 지금 읽는 책       |
| V-(으)ㄹ N            | 将来动词修饰名词 | 내일 만날 사람     |
| -아/어지다            | 变得…            | 날씨가 추워졌어요  |
| -(으)ㄴ가요? / -나요? | 柔和疑问         | 비가 오나요?       |
| -대요                 | 传闻/听说        | 내일 춥대요        |

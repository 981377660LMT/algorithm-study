# 제5과: 하루 일과 (一天的日程)

> 延世韩国语 第二册 · 第5课

---

## 一、学习目标

- 能够详细描述一天中的约定、计划和变更
- **重点语法**：-는데요 (句尾用法)、-기로 하다 (决定做)、-(으)ㄹ까 하다 (打算做)、-잖아요 (你也知道嘛)
- 学会谈论约定和计划变动
- 掌握表达意图和决心

---

## 三、核心语法

### 语法1: -는데요 — 句尾"…呢" ★★★

作为 **句尾** 使用时，表示：

1. 委婉地陈述状况，暗示"怎么办？"
2. 表达惊讶或感叹
3. 引出对方的反应

**构成**（与连接语尾 -는데 相同，只是加了요做终结）：

- 动词 → V-**는데요**
- 形容词 → A-**(으)ㄴ데요**
- 名词 → N-**인데요**
- 过去 → -**았/었는데요**

#### 用法1：说明情况，暗示"怎么办/你看呢"

```
A: 내일 같이 영화 볼래요?
   明天一起看电影吗？
B: 미안해요. 내일은 약속이 있는데요...
   对不起。明天有约了呢……（暗示不行）

A: 김 선생님 계세요?
   金老师在吗？
B: 지금 회의 중인데요.
   现在正在开会呢。（暗示不方便）
```

#### 用法2：表达惊讶/感叹

```
이 식당 음식이 정말 맛있는데요!
这家餐厅的菜真好吃呢！

한국어를 잘하시는데요!
韩语说得很好呢！

경치가 아름다운데요!
风景很美呢！
```

#### 用法3：电话/拜访开场

```
여보세요, 김민수인데요.
喂，我是金民秀。

실례합니다, 학생인데요...
打扰了，我是学生……（铺垫后面的请求）
```

---

### 语法2: -기로 하다 — "决定做……" ★★★

表示自己**做出决定**或**和别人约定好**要做某事。

**构成**：动词词干 + -기로 하다

```
# 基本用法
주말에 등산하기로 했어요.         决定周末去爬山。
다이어트하기로 했어요.            决定减肥。
내년에 한국에 가기로 했어요.      决定明年去韩国。

# 和他人的约定
내일 세 시에 만나기로 했어요.     约好明天3点见面。
같이 여행 가기로 했어요.          约好一起去旅行。
다음 달에 결혼하기로 했어요.      决定下个月结婚。
```

**否定/取消决定**：

```
안 가기로 했어요.                决定不去了。
그 계획은 취소하기로 했어요.      决定取消那个计划。
```

> **-기로 하다 vs -(으)려고 하다**：
>
> - **-기로 하다**：已经做出的**确定决定** → 留학하기로 했어요 (决定了要留学)
> - **-(으)려고 하다**：打算/想要（还不确定） → 유학하려고 해요 (打算留学)

---

### 语法3: -(으)ㄹ까 하다 — "想要/打算做……" ★★

表示模糊的**打算或考虑**，语气比 -(으)려고 하다 更不确定。

**构成**：动词词干 + -(으)ㄹ까 하다

```
이번 주말에 영화를 볼까 해요.
打算这周末看电影。（只是想想）

올해는 일본어도 배울까 해요.
今年打算也学日语。（在考虑中）

점심에 냉면을 먹을까 해요.
午饭想吃冷面。（还没确定）

하반기에 이직할까 해요.
下半年在考虑换工作。
```

> **确定程度对比**：
> -(으)ㄹ까 하다 < -(으)려고 하다 < -기로 하다
> (在考虑) < (打算) < (决定了)

---

### 语法4: -잖아요 — "你也知道嘛/不是…吗" ★★

提醒对方**已经知道或应该知道**的事实，表示"你不是也知道吗"。

**构成**：

- 动词/形容词 + -잖아요
- 名词 + (이)잖아요

```
A: 왜 약속을 취소해요?
   为什么取消约定？
B: 내일 시험이잖아요. 공부해야 해요.
   明天不是有考试嘛。要学习。

A: 이거 왜 이렇게 비싸요?
   这个怎么这么贵？
B: 여기가 명동이잖아요.
   这里不是明洞嘛。（明洞本来就贵你也知道）

A: 오늘 회식 안 가요?
   今天不去聚餐吗？
B: 저 술 못 마시잖아요.
   我不是不能喝酒嘛。（你知道的呀）

내일은 일요일이잖아요. 쉬어요.
明天不是周日嘛。休息吧。
```

## 四、核心会话

### 对话1：约定

```
A: 이번 토요일에 뭐 해요?
   这周六做什么？

B: 아직 계획이 없는데요. 왜요?
   还没有计划呢。怎么了？

A: 같이 노래방 갈까 해요.
   想一起去KTV。

B: 좋아요! 몇 시에 만날까요?
   好！几点见？

A: 저녁 여섯 시에 만나기로 해요.
   定晚上6点见面吧。

B: 네, 좋아요. 그 전에 저녁도 같이 먹어요.
   好，行。之前一起吃晚饭吧。

A: 그러면 다섯 시에 만나서 밥 먹고 노래방 가기로 해요.
   那5点见面吃饭再去KTV吧。

B: 네, 그렇게 하기로 해요!
   好，就这么定了！
```

### 对话2：计划变更

```
A: 민수 씨, 내일 회의 준비 다 했어요?
   民秀，明天会议准备好了吗？

B: 거의 다 했는데요. 발표 자료를 좀 더 수정해야 해요.
   差不多了呢。发表资料要再修改一下。

A: 그런데 내일 회의가 취소됐어요.
   但是明天的会议取消了。

B: 네? 왜요?
   什么？为什么？

A: 부장님이 출장 가시잖아요. 다음 주로 미루기로 했어요.
   部长不是出差嘛。决定推到下周了。

B: 아, 맞다. 부장님이 이번 주에 부산 가신다고 했었죠.
   啊对。部长说这周去釜山来着。
   그러면 시간이 좀 더 있으니까 자료를 더 잘 만들어야겠어요.
   那有更多时间了，得把资料做更好。
```

**新词**：

- **발표 자료** = 发表资料/PPT
- **수정하다** = 修改
- **부장님** = 部长
- **출장** = 出差

### 对话3：规律生活

```
A: 매일 아침에 운동해요?
   每天早上运动吗？

B: 네, 올해부터 매일 운동하기로 했어요.
   是的，从今年开始决定每天运动。

A: 대단하네요! 저도 운동을 시작할까 해요.
   好厉害！我也在考虑开始运动。

B: 처음에는 힘든데 하다 보면 습관이 돼요.
   一开始很累但做着做着就成习惯了。

A: 어떤 운동을 하기로 했어요?
   决定做什么运动了？

B: 월, 수, 금은 달리기를 하고 화, 목은 수영하기로 했어요.
   周一三五跑步，周二四游泳。

A: 와, 계획이 구체적이네요!
   哇，计划很具体啊！
```

## 五、文化补充：韩国人的约定文化

```
📱 카카오톡 (KakaoTalk)
  - 한국에서 약속은 주로 카카오톡으로 잡아요.
    在韩国约定主要通过KakaoTalk定。
  - 단톡방 (群聊) 에서 시간과 장소를 정해요.
    在群聊里定时间和地点。

⏰ 시간 약속
  - "한국 시간" 이란 말이 있어요.
    한국 사람들은 약속 시간에 좀 늦는 경향이 있어요.
    有"韩国时间"的说法。韩国人有约会迟到的倾向。
  - 하지만 회사에서는 시간을 꼭 지켜야 해요!
    但在公司必须守时！

🍻 회식 문화
  - 회사에서는 회식이 자주 있어요.
    公司经常有聚餐。
  - "1차, 2차, 3차" = 第一场、第二场、第三场
  - 요즘은 개인 시간을 중요시해서 바뀌고 있어요.
    最近因为重视个人时间正在改变。
```

## 六、语法练习

### 练习1：-기로 하다 vs -(으)ㄹ까 하다

选择合适的语法填空：

1. 다음 달에 일본에 여행 (가다) \_\_\_. (已经买了机票)
2. 올해는 중국어도 (배우다) \_\_\_. (只是在想)
3. 친구하고 주말에 (만나다) \_\_\_. (已经约好了)
4. 점심에 뭘 (먹다) \_\_\_. (还没决定)

<details>
<summary>点击查看答案</summary>

1. 가기로 했어요 (已经决定)
2. 배울까 해요 (在考虑中)
3. 만나기로 했어요 (约好了)
4. 먹을까 해요 (还在想)

</details>

### 练习2：-잖아요 练习

用 -잖아요 回答：

1. A: 왜 집에서 쉬어요? → B: 오늘 일요일(이다) \_\_\_
2. A: 왜 우산을 가져가요? → B: 오늘 비가 (오다) \_\_\_
3. A: 왜 한국어를 배워요? → B: 저 한국 회사에서 (일하다) \_\_\_

<details>
<summary>点击查看答案</summary>

1. 오늘 일요일이잖아요.
2. 오늘 비가 오잖아요.
3. 저 한국 회사에서 일하잖아요.

</details>

### 练习3：-는데요 应答

用 -는데요 回答以下情况：

1. 有人约你明天见面，但你明天有考试 → ******\_\_\_******
2. 有人找你的同事，但同事已经下班了 → ******\_\_\_******
3. 你第一次吃到好吃的韩国料理 → ******\_\_\_******

<details>
<summary>点击查看答案</summary>

1. 내일은 시험이 있는데요... / 내일은 좀 바쁜데요...
2. 벌써 퇴근했는데요. / 지금 안 계시는데요.
3. 와, 정말 맛있는데요!

</details>

## 三、核心语法

### 语法1: -(으)ㄹ진저 — "应该…/当…才好" ★★★

极度文言体，表达**应该如此/最好如此**。诗歌、格言中常见。

```
법 앞에 만인이 평등할진저.
在法律面前人人应当平等。

정의를 실현하기 위해 노력할진저.
应当为实现正义而努力。

자유를 소중히 여길진저.
当珍视自由才好。
```

> 현대 한국어에서는 극히 드물게 사용. 격언/연설/문학 전용.

---

### 语法2: -는 바이다 — "所要说/要求的是…" ★★★★★

表示**正式声明/表态**。公文、声明、学术结论常用。

```
본 법안이 추구하는 바는
국민의 기본권 보장이다.
本法案所追求的是
国民基本权利的保障。

강조하고자 하는 바는
절차적 정의의 중요성이다.
所要强调的是
程序正义的重要性。

이에 다음과 같이 건의하는 바이다.
特此建议如下。

심각하게 우려하는 바이다.
（对此）深表忧虑。
```

> **-는 바이다**：公文/声明的固定结尾。极正式。

---

### 语法3: -기에 이르기까지 — "从…一直发展到…" ★★★★

表示**事物发展的漫长过程→到达现在的状态**。

```
고대 법전에서부터 현대 헌법에
이르기까지 법은 계속 발전해 왔다.
从古代法典一直到现代宪法
法律持续发展着。

인권 개념이 자연법 사상에서 시작하여
세계인권선언에 이르기까지
오랜 역사가 있다.
人权概念从自然法思想开始
到世界人权宣言
有漫长的历史。

불의에 대한 저항에서부터
법 개정에 이르기까지
시민의 노력이 있었다.
从对不义的抵抗
到法律修订
有市民的努力。
```

---

### 语法4: -거늘 — "然而/可是" (古语连接词尾) ★★★

古典文学体的转折，与 -건만 类似但更古雅。

```
법은 만인의 평등을 보장하거늘
현실은 그렇지 못하다.
法律保障人人平等
然而现实却不是如此。

제도는 완비되었거늘
집행이 뒤따르지 않는다.
制度虽已完备
然而执行跟不上。

자유를 선언하거늘
차별은 여전하다.
虽然宣布了自由
然而歧视依然存在。
```

> **전환 표현 최종 계보 (转折表达最终谱系)**：-지만 (口语) → -건만 (文学) → -거늘 (古语) → -(으)나 (书面)

## 四、核心会话

### 对话：法治与正义

```
A: 법치주의란 무엇일까요?
   法治主义是什么呢？

B: 법 앞에 만인이 평등하고
   누구도 법 위에 있을 수 없다는 원칙이에요.
   法律面前人人平等
   谁也不能凌驾于法律之上的原则。

A: 제도는 잘 갖추어져 있거늘
   현실에서는 불공정한 경우가 있잖아요.
   制度虽然完备
   但现实中有不公正的情况。

B: 그래서 강조하고자 하는 바는
   절차적 정의예요.
   所以想要强调的是
   程序正义。

A: 보나마나 힘 있는 사람에게 유리하다고
   생각하는 사람도 있어요.
   也有人认为不用说也知道
   对有权势的人有利。

B: 그런 인식을 바꾸기로서니
   하루아침에 되겠습니까.
   시민의 감시가 중요합니다.
   就算要改变那种认知
   怎么能一朝一夕实现呢。
   市民的监督很重要。
```

## 四、让步/转折表达 终极总结

韩语让步系统是 TOPIK 6급最重要的考点之一：

```
口语 ─────────────────────────────── 文语
  -지만  →  -더라도  →  -ㄹ지언정  →  -거늘
  虽然     即使       即使…绝不     然而(古)

  -건만  →  -ㄹ망정  →  -로되  →  -옵거니와
  可是     就算…也    虽然(古)   不但…(古敬)
```

## 七、TOPIK II 6급 备考要点

### 阅读理解策略

1. **古语识别**：遇到 -거늘, -로되, -리로다 等 → 立刻判断为文学/古文体
2. **公文识别**：看到 -는 바이다 → 判断为声明/公文
3. **让步辨析**：区分 -ㄹ지언정(坚决), -ㄹ망정(承认), -거늘(古语转折)
4. **因果推理**：-는지라(文), -느니만큼(正式) 后必跟结论

### 写作策略

1. **论述文**：善用 -는 바이다, -느니만큼, -(으)ㄹ 따름이다
2. **说明文**：善用 -기에 이르기까지, -는 이만큼
3. **议论文**：善用 -(으)ㄹ지언정, -는 한이 있더라도

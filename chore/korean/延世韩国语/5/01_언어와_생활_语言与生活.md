## 三、核心语法

### 语法1: -(으)ㄹ진대 — "既然如此/要是…的话" ★★★

表示**既然承认前面的事实，后面理当如此**。非常正式/文学体。

**构成**：动词/形容词词干 + -(으)ㄹ진대

```
외국어를 배울진대 제대로 배워야 한다.
既然要学外语就要好好学。

약속을 할진대 반드시 지켜야 한다.
既然做出承诺就必须遵守。

살아갈진대 후회 없이 살아야 한다.
既然要活下去就要活得无悔。

이 일을 맡을진대 끝까지 책임져야 한다.
既然要承担这件事就要负责到底。

가르칠진대 정확하게 가르쳐야 한다.
既然要教就要准确地教。
```

> **语感**：极度正式/文言化，现代口语几乎不用。
> 口语替换：-(으)ㄹ 바에야 / -(으)려면 / -는 이상
>
> - 배울진대 ≈ 배우는 이상 ≈ 배우려면 (既然学 / 要学的话)

---

### 语法2: -는 터에 — "在…的情况/处境下" ★★★★

表示**已经处于某种状况**，在此基础上发生后续事情。常含"已经够麻烦了"的语感。

**构成**：

| 词类   | 构成          |
| ------ | ------------- |
| 动词   | -는 터에      |
| 形容词 | -(으)ㄴ 터에  |
| 过去   | -았/었던 터에 |

```
시간도 없는 터에 갑자기 일이 더 생겼다.
在时间也不够的情况下突然又增加了工作。

돈도 부족한 터에 또 고장이 났다.
在钱也不够的情况下又出了故障。

바쁜 터에 도움까지 청하니 미안하다.
在忙碌的情况下还来求帮忙很不好意思。

공부할 시간도 없는 터에 아르바이트까지 해야 한다.
在连学习时间都没有的情况下还得打工。

이미 지친 터에 밤새 일하라니 무리다.
在已经疲惫的情况下让通宵工作太过分了。
```

> **-는 터에 vs -는 데다가**：
>
> - -는 터에：强调"已有的不利处境"+雪上加霜
> - -는 데다가：单纯"加上/而且还"（正负皆可）

---

### 语法3: -(으)ㄹ래야 -(으)ㄹ 수 없다 — "想…也不能" ★★★★

表示**即使想做也不可能做到**。强调"完全不可能"。

**构成**：动词词干 + -(으)ㄹ래야 + 同一动词 + -(으)ㄹ 수 없다

```
잊을래야 잊을 수 없는 추억이다.
想忘也忘不了的回忆。

참을래야 참을 수 없을 만큼 화가 났다.
想忍也忍不了地生气了。

부정할래야 부정할 수 없는 사실이다.
想否认也否认不了的事实。

감출래야 감출 수 없는 감정이었다.
想藏也藏不住的感情。

피할래야 피할 수 없는 현실이다.
想逃也逃不掉的现实。

외면할래야 외면할 수 없는 문제다.
想回避也回避不了的问题。
```

> **常用定语形**：-(으)ㄹ래야 -(으)ㄹ 수 없는 + 名词
> 这是一个非常高级且地道的表达，TOPIK 高级阅读中常见。

---

### 语法4: -건만 (-거니와) — "然而/可是/…但是" ★★★★

表示**承认前面的事实，但后面结果却出乎意料或令人遗憾**。书面/文学体。

**构成**：

| 词类   | 构成            |
| ------ | --------------- |
| 动词   | -는건만 / -건만 |
| 形容词 | -(으)ㄴ건만     |
| 过去   | -았/었건만      |

```
노력은 했건만 결과가 좋지 않았다.
虽然努力了但结果不好。

알고는 있건만 실천하기 어렵다.
虽然知道但难以实践。

세월은 흘렀건만 그 기억은 생생하다.
岁月虽已流逝但那记忆依然鲜明。

말은 쉽건만 행동으로 옮기기는 어렵다.
话说起来容易但付诸行动很难。

수없이 읽었건만 여전히 감동적이다.
虽然读了无数遍但依然令人感动。
```

> **-건만 vs -지만**：
>
> - -지만：通用转折 → 바쁘지만 (虽然忙但…)
> - -건만：文学/感性转折，含遗憾或感叹 → 노력했건만 (努力了然而…)
>
> **-거니와**：与 -건만 同源，但更偏"不仅…而且"（递进感）
>
> - 실력도 뛰어나거니와 성격도 좋다 (实力出色且性格也好)

## 四、核心会话

### 对话1：语言学习的困难

```
A: 한국어를 오래 공부했는데도
   아직 원어민처럼 말하기 어려워요.
   虽然学了很久韩语
   但还是很难说得像母语者一样。

B: 외국어란 잊을래야 잊을 수 없는
   모국어의 영향에서 벗어나기 어렵죠.
   学外语的话想摆脱也摆脱不了
   母语的影响确实困难。

A: 문법은 아는 터에
   실제 대화에서는 잘 안 나와요.
   语法是知道的
   但在实际对话中说不出来。

B: 저도 마찬가지예요. 공부는 꾸준히 했건만
   유창해지기까지는 시간이 걸리더라고요.
   我也一样。虽然一直学着
   但到变流利为止需要时间。

A: 언어를 배울진대
   그 나라 문화까지 알아야 한다던데요.
   听说既然要学语言
   就得了解那个国家的文化。

B: 맞아요. 맥락을 모르면
   말은 할 수 있어도 소통이 안 되거든요.
   对。不了解语境的话
   虽然能说话但无法真正沟通。
```

### 对话2：敬语文化

```
A: 한국어 경어법이 왜 이렇게 복잡해요?
   韩语敬语法为什么这么复杂？

B: 한국은 상하 관계를 중시하는 터에
   말에도 그게 반영되는 거예요.
   韩国重视上下关系
   这自然也反映在语言上。

A: 높임말, 반말, 격식체, 비격식체...
   외국인이 구별하기 어렵죠.
   敬语、非敬语、格式体、非格式体……
   外国人确实难区分。

B: 한국인도 가끔 어색할 때가 있어요.
   처음 만난 동갑내기한테 뭐라고 해야 할지...
   韩国人有时也觉得尴尬。
   第一次见同龄人不知道该怎么说……

A: 그래도 부정할래야 부정할 수 없는 게
   경어법이 한국 문화의 핵심이라는 거죠.
   但想否认也否认不了的是
   敬语法是韩国文化的核心。

B: 네, 처음엔 어렵건만
   익숙해지면 미묘한 뉘앙스까지 느낄 수 있어요.
   嗯，虽然一开始难
   但习惯之后能感受到微妙的语感差异。
```

### 对话3：语言与思维

```
A: '언어가 사고를 결정한다'는
   사피어-워프 가설에 대해 어떻게 생각하세요?
   你怎么看'语言决定思维'的
   萨丕尔-沃尔夫假说？

B: 언어가 사고를 완전히 결정한다고 할 수는 없건만
   영향을 미치는 것은 분명해요.
   虽然不能说语言完全决定思维
   但影响思维是确定的。

A: 예를 들면요?
   比如呢？

B: 한국어에는 '정'이라는 단어가 있잖아요.
   영어로 번역할래야 할 수 없는
   독특한 감정 표현이에요.
   韩语有"情"这个词不是吗？
   想翻译成英语也翻译不了
   是独特的情感表达。

A: 그러고 보니 '눈치'도 그렇네요.
   这么一看"看人眼色"也是那样。

B: 언어를 공부할진대
   그 안에 담긴 세계관도 함께 배우는 셈이에요.
   既然要学语言
   也算是一起学习其中蕴含的世界观。
```

**新词**：

- **사고 (思考)** = 思维
- **가설** = 假说
- **정 (情)** = 韩国特有的"情"的概念
- **눈치** = 察言观色/看眼色
- **세계관** = 世界观

## 五、文化补充：韩语的语言特征

| 特征                | 说明                                                   |
| ------------------- | ------------------------------------------------------ |
| **경어법 (敬语法)** | 7种语体层级，世界罕见的复杂敬语系统                    |
| **어순 (语序)**     | SOV语序 (主宾谓)，与中/英不同                          |
| **조사 (助词)**     | 은/는、이/가、을/를 等丰富的助词系统                   |
| **의성어·의태어**   | 拟声词·拟态词极其丰富 (반짝반짝、두근두근…)            |
| **한글 (韩文字)**   | 世宗大王1443年创制，科学性极高的文字                   |
| **높임법 종류**     | 주체높임(-시-), 객체높임(-드리다), 상대높임(합쇼체 등) |

## 六、语法练习

### 练习1：-(으)ㄹ래야 -(으)ㄹ 수 없다

用"想…也不能"完成句子：

1. 想忘也忘不了的经历 → ******\_\_\_******
2. 想否认也否认不了的事实 → ******\_\_\_******

<details>
<summary>点击查看答案</summary>

1. 잊을래야 잊을 수 없는 경험이다.
2. 부정할래야 부정할 수 없는 사실이다.

</details>

### 练习2：-건만

用"虽然…但/然而"连接：

1. 虽然知道但做不到 → ******\_\_\_******
2. 时间过去了然而记忆鲜明 → ******\_\_\_******

<details>
<summary>点击查看答案</summary>

1. 알고 있건만 실천하지 못한다.
2. 시간이 흘렀건만 기억은 생생하다.

</details>

### 练习3：-는 터에

用"在…的情况下"：

1. 在时间也不够的情况下又来了新工作 → ******\_\_\_******
2. 在已经很累的情况下还要加班 → ******\_\_\_******

<details>
<summary>点击查看答案</summary>

1. 시간도 없는 터에 새 일이 또 들어왔다.
2. 이미 피곤한 터에 야근까지 해야 한다.

</details>

## 三、核心语法

### 语法1: -아/어야지(요) — "得做/下决心做" ★★★

表示自己**下决心**要做某事，或**应该做**某事（带有自我提醒的语气）。

**构成**：动词词干 + -아/어야지(요)

```
# 下决心/自我激励
이제부터 열심히 공부해야지!
从现在开始得努力学习了！

내일부터 다이어트해야지.
明天开始得减肥了。

올해는 한국어 시험에 합격해야지요.
今年得通过韩语考试。

건강을 위해서 매일 운동해야지.
为了健康每天得运动。
```

```
# 提醒对方（轻度责备/叮嘱）
약속을 했으면 지켜야지요.
既然约了就得遵守啊。

학생이면 공부를 해야지.
是学生就得学习啊。

아침을 먹어야지, 왜 안 먹어요?
得吃早饭啊，为什么不吃？
```

> **-아/어야지 vs -아/어야 하다**：
>
> - **-아/어야 하다**：客观义务 "应该/必须" → 숙제를 해야 해요 (应该做作业)
> - **-아/어야지**：主观决心/自我激励 "得做!" → 숙제를 해야지! (得做作业了!)

---

### 语法2: -(으)ㄴ/는 대로 — "按照/一…就…" ★★★

有两个重要含义：

#### 含义1：按照…的样子 (방식)

```
선생님이 말씀하신 대로 했어요.
按照老师说的做了。

제가 하는 대로 따라 하세요.
按照我做的跟着做。

레시피에 쓰인 대로 요리했어요.
按照食谱写的做了饭。

들은 대로 말씀드릴게요.
按照听到的告诉您。
```

#### 含义2：一…就… (시간 즉시)

动词词干 + **-는 대로** → "一做完…就…"

```
한국에 도착하는 대로 연락할게요.
一到韩国就联系你。

일이 끝나는 대로 갈게요.
一下班就去。

준비되는 대로 출발합시다.
一准备好就出发。

소식을 듣는 대로 알려 드릴게요.
一听到消息就通知您。
```

#### 名词 + 대로

```
계획대로 진행하겠습니다.          按计划推进。
예상대로 비가 왔어요.            如预想的那样下雨了。
마음대로 하세요.                 随你意/随便。
```

---

### 语法3: -게 되다 深化 — "变得/来到了…的状态" ★★★

第一册已学过基础用法，这里深入活用。

#### 各种场景

```
# 自然而然的变化
한국에 살다 보니까 매운 음식을 잘 먹게 됐어요.
在韩国住着住着就变得能吃辣了。

시간이 지나면서 한국 생활에 적응하게 됐어요.
随着时间过去适应了韩国生活。

한국 친구를 사귀게 되면서 한국어가 많이 늘었어요.
交了韩国朋友后韩语进步了很多。

# 非自愿/命运安排
회사 때문에 갑자기 한국에 오게 됐어요.
因为公司突然来了韩国。

처음에는 관심이 없었는데 점점 좋아하게 됐어요.
一开始没有兴趣但渐渐喜欢上了。

# 结果/结论
결국 그 회사에서 일하게 됐어요.
最终在那家公司工作了。

여러 가지를 비교한 끝에 이 집에 살게 됐어요.
比较了各种之后住了这个房子。
```

> **-게 되다 的核心含义**：说话者并非完全出于自己意志，而是由于某种原因/经过，自然而然达到了某个结果。搭配 "보니(까)"、"면서"、"다 보니" 等非常自然。

---

### 语法4: -(으)ㄹ 만하다 — "值得做/能做到" ★★★

#### 含义1：值得做

```
이 영화는 볼 만해요.              这部电影值得看。
이 식당은 갈 만해요.              这家餐厅值得去。
이 책은 읽을 만해요.              这本书值得读。
한국 생활은 해 볼 만해요.          韩国生活值得体验。
```

#### 含义2：还能忍受/还行

```
매운데 먹을 만해요.               虽辣但还能吃。
힘들지만 참을 만해요.             虽累但还能忍。
멀지만 걸어갈 만해요.             虽远但还能走。
추운데 견딜 만해요.               虽冷但还能受。
```

## 四、核心会话

### 对话1：聊韩国生活

```
A: 한국에 온 지 얼마나 됐어요?
   来韩国多久了？

B: 1년 정도 됐어요. 처음에는 정말 힘들었어요.
   大概1年了。一开始真的很辛苦。

A: 뭐가 제일 힘들었어요?
   什么最辛苦？

B: 쓰레기 분리수거요! 중국에 비해서 정말 복잡하거든요.
   垃圾分类！和中国相比真的很复杂。
   그런데 한국 친구가 하는 대로 따라 했더니
   금방 익숙해졌어요.
   但按照韩国朋友做的跟着做了之后很快就熟悉了。

A: 지금은 어때요? 적응이 됐어요?
   现在怎么样？适应了吗？

B: 네, 많이 적응하게 됐어요.
   嗯，适应了很多了。
   한국 음식도 잘 먹게 됐고, 대중교통도 편하게 이용해요.
   韩国料理也能吃得好了，公共交通也方便使用。

A: 한국 생활이 해 볼 만해요?
   韩国生活值得体验吗？

B: 네, 정말 해 볼 만해요!
   是，真的值得体验！
   특히 배달 문화가 정말 편해요.
   特别是外卖文化真的很方便。
```

### 对话2：文化差异

```
A: 한국에서 가장 놀란 문화 차이가 뭐예요?
   在韩国最吃惊的文化差异是什么？

B: 한국은 나이를 정말 중요하게 생각하는 것이요.
   韩国真的很重视年龄。

A: 맞아요. 한국에서는 나이에 따라 말이 달라지잖아요.
   对。在韩国根据年龄用语不同嘛。

B: 처음에는 존댓말이 정말 어려웠어요.
   一开始敬语真的很难。
   누구한테 반말을 쓰고 누구한테 존댓말을 써야 하는지
   몰라서 실수를 많이 했어요.
   不知道对谁用半语对谁用敬语犯了很多错。

A: 지금은 자연스러워졌어요?
   现在自然了吗？

B: 네, 이제 자연스럽게 되게 됐어요.
   嗯，现在自然了。
   한국 사람들이 친절하게 알려 줘서요.
   因为韩国人亲切地告诉我。

A: 그래도 힘든 점은 있어요?
   不过有辛苦的地方吗？

B: 가끔 향수병이 와요. 그럴 때는 고향 음식을 먹어야지,
   하면서 중국 식당에 가요.
   偶尔会想家。那时候就想"得吃家乡菜了"，
   然后去中国餐厅。
```

### 对话3：新年决心

```
A: 새해 계획 세웠어요?
   新年计划定了吗？

B: 네. 올해는 토픽 4급에 합격해야지요!
   定了。今年得通过TOPIK 4级！

A: 좋은 목표네요! 어떻게 준비할 거예요?
   好目标呢！打算怎么准备？

B: 선생님이 추천해 주신 대로 매일 한 시간씩 공부하려고요.
   准备按照老师推荐的每天学一小时。

A: 한 시간이면 충분해요?
   一小时够吗？

B: 부족하면 더 해야지요.
   不够就得多做。
   주말에는 한국어 스터디에도 참가하게 됐어요.
   周末也开始参加韩语学习会了。

A: 열심히 하면 합격할 만해요. 화이팅!
   努力的话值得通过。加油！
```

## 七、本课核心句型小结

| 句型            | 用法         | 例句                       |
| --------------- | ------------ | -------------------------- |

# 제7과: 실수와 사과 (失误与道歉)

> 延世韩国语 第三册 · 第7课

---

## 一、学习目标

- 能够用韩语表达后悔、道歉和遗憾
- **重点语法**：-(으)ㄹ걸 그랬다 (早该…的)、-았/었어야 했는데 (本应该…)、-기는요 (哪里/怎么会)、-다니 (竟然…)
- 掌握韩国社交中道歉和表达遗憾的方式

## 三、核心语法

### 语法1: -(으)ㄹ걸 그랬다 — "早该…的" ★★★★

表示**后悔没做某事**，意为"早知道（当时）就…了"。

**构成**：动词词干 + -(으)ㄹ걸 (그랬다)

```
일찍 일어날걸 그랬어요.
早该早起的（后悔没早起）。

공부를 더 할걸 그랬어요.
早该多学习的。

우산을 가져갈걸...
该带伞的…

그때 사과할걸 그랬어요.
早该那时候道歉的。

택시를 탈걸 그랬어요.
该打车的（后悔没打车）。

좀 더 생각해 볼걸 그랬어요.
该多想想的。
```

> **口语中常省略 그랬다**：
>
> - 일찍 갈걸... = 일찍 갈걸 그랬다 (该早走的…)
> - 안 먹을걸... = 안 먹을걸 그랬다 (该不吃的…)

---

### 语法2: -았/었어야 했는데 — "本应该…（但没有）" ★★★★

表示**过去应该做但没做**，包含后悔或遗憾。

**构成**：动词词干 + -았/었어야 했는데

```
미리 예약했어야 했는데 깜빡했어요.
本该提前预约的但忘了。

더 열심히 공부했어야 했는데 놀기만 했어요.
本该更努力学习的但只知道玩。

일찍 출발했어야 했는데 늦게 나왔어요.
本该早出发的但出来晚了。

사과했어야 했는데 기회를 놓쳤어요.
本该道歉的但错过了机会。

확인했어야 했는데 그냥 보냈어요.
本该确认的但就那样发了。
```

> **-(으)ㄹ걸 그랬다 vs -았/었어야 했는데**：
>
> - -(으)ㄹ걸 그랬다：更口语，感叹后悔 → 일찍 갈걸…（该早走的）
> - -았/었어야 했는데：更理性/叙述，后接补充说明 → 일찍 갔어야 했는데 늦게 나왔다（本该早走但出来晚了）

---

### 语法3: -기는요 — "哪里/怎么会" ★★★

对别人的话表示**谦虚否认**或**客气反驳**。

**构成**：

- 动词/形容词 + -기는요
- 也可以直接重复对方的用词 + -기는요
- 口语中常缩略为 **-긴요**

```
A: 한국어를 정말 잘하시네요.    韩语说得真好啊。
B: 잘하기는요. 아직 멀었어요.   哪里啊。还差得远呢。

A: 어제 많이 바빴죠?           昨天很忙吧？
B: 바쁘기는요. 할 일이 별로 없었어요.
   忙什么啊。没什么事儿。

A: 시험 잘 봤어요?             考试考好了吗？
B: 잘 보기는요. 완전 망했어요.   好什么啊。完全考砸了。

A: 요리를 잘하시네요.           做饭很好啊。
B: 잘하긴요. 그냥 대충 했어요.   哪里。就随便做的。
```

> **语气**：不是真正的否定，而是**谦虚/客气**的表达。韩国文化中对别人的夸奖要谦虚回应。

---

### 语法4: -다니 — "竟然…" ★★★

表示对某件事感到**惊讶、意外、难以置信**。

**构成**：

| 词类   | 构成                 | 例                      |
| ------ | -------------------- | ----------------------- |
| 动词   | -다니 / -(ㄴ/는)다니 | 간다니 (竟然去)         |
| 形容词 | -다니                | 예쁘다니 (竟然漂亮)     |
| 名词   | -(이)라니            | 학생이라니 (竟然是学生) |
| 过去   | -았/었다니           | 갔다니 (竟然去了)       |

```
그 사람이 거짓말을 하다니 믿을 수 없어요.
那个人竟然说谎真不敢相信。

시험에 떨어지다니 정말 속상해요.
考试竟然没过真的很难过。

10년 만에 만나다니 정말 반갑다.
竟然10年后才见面真的很高兴。

100점을 받다니 대단해요!
竟然拿了100分太厉害了！

벌써 졸업이라니 시간이 정말 빨라요.
竟然已经毕业了时间真快。

혼자 그걸 다 했다니 놀랍다.
竟然一个人全做了真令人惊讶。
```

> **-다니 的两种用法**：
>
> 1. **惊讶**：그게 사실이라니! (那竟然是真的！)
> 2. **引用+反问**（间接引用的变体）：뭐라고요? 간다니요? (什么？说要去？)

## 四、核心会话

### 对话1：后悔与道歉

```
A: 어제 약속에 왜 안 왔어요?
   昨天约会为什么没来？

B: 아, 정말 죄송해요. 날짜를 착각했어요.
   啊，真的对不起。我搞错了日期。

A: 전화라도 해 줬어야 했는데...
   至少也该打个电话的…

B: 네, 연락했어야 했는데 깜빡했어요.
   嗯，本该联系的但忘了。
   미리 확인할걸 그랬어요.
   早该提前确认的。

A: 다음부터는 날짜를 꼭 확인하세요.
   以后一定要确认日期。

B: 네, 다시는 안 그럴게요.
   嗯，不会再那样了。
   정말 미안해요. 제가 밥 살게요.
   真的对不起。我请吃饭。
```

### 对话2：谦虚回应

```
A: 와, 한국어 실력이 정말 많이 늘었네요!
   哇，韩语实力真的进步很大啊！

B: 늘기는요. 아직 모르는 표현이 너무 많아요.
   进步什么啊。不知道的表达还太多。

A: 아니에요. 3개월밖에 안 됐는데
   이 정도면 대단한 거예요.
   不是的。才3个月
   能到这种程度很厉害了。

B: 감사합니다. 그런데 어제 실수해서
   한국 친구한테 사과했어야 했는데 못 했어요.
   谢谢。但是昨天犯了错误
   本该向韩国朋友道歉但没做到。

A: 무슨 실수요?
   什么错误？

B: "배 불러요"를 "배 빨라요"라고 했어요.
   把"肚子饱了"说成了"肚子快"。

A: 하하, 그 정도는 괜찮아요.
   哈哈，那种程度没关系啦。
   오히려 귀엽다고 생각했을 거예요.
   反而会觉得可爱吧。
```

### 对话3：惊讶感叹

```
A: 민수 씨가 회사를 그만뒀대요.
   听说民秀辞职了。

B: 뭐라고요? 그만두다니!
   일을 그렇게 좋아했는데...
   什么？竟然辞职了！
   那么喜欢工作的说…

A: 네, 저도 깜짝 놀랐어요.
   嗯，我也吓了一跳。

B: 왜 그만뒀대요?
   为什么辞的？

A: 유학을 간대요. 미국으로.
   说要去留学。去美国。

B: 그렇구나. 미리 얘기해 줄걸 그랬는데.
   这样啊。早该提前告诉我们的。

A: 마지막 날에 송별회를 하기로 했어요.
   决定最后一天办欢送会。
```

**新词**：

- **깜빡하다** = 忘了（忘记去做某事）
- **대충** = 大概/随便
- **오히려** = 反而
- **그만두다** = 辞职/停止

## 五、文化补充：韩国的道歉文化

| 特点            | 说明                                           |
| --------------- | ---------------------------------------------- |
| **정중한 사과** | 韩国非常重视正式道歉，越严重的事越需要正式场合 |
| **몸으로 사과** | 严重场合可能鞠躬90度（큰절）来道歉             |
| **선물 사과**   | 送礼物/请吃饭也是道歉的方式之一                |
| **체면**        | 面子文化，公开道歉比私下道歉更有分量           |
| **겸양**        | 谦虚是美德，-기는요 是典型的谦虚回应           |

> 韩语中"对不起"的程度：
> **죄송합니다** (最正式) > **미안합니다** > **미안해요** > **미안해** (最非正式)

1. 没复习，考试考砸了 → ******\_\_\_******
2. 没带伞，淋了雨 → ******\_\_\_******
3. 没道歉，朋友生气了 → ******\_\_\_******

<summary>点击查看答案</summary>

2. 우산을 가져갈걸 그랬어요.
3. 사과할걸 그랬어요.

</details>

### 练习2：-기는요

用谦虚回应改写：

1. A: 요리를 잘하시네요. B: ******\_\_\_******
2. A: 한국어가 유창하시네요. B: ******\_\_\_******
3. A: 바쁘시죠? B: ******\_\_\_******

<details>
<summary>点击查看答案</summary>

1. 잘하기는요. / 잘하긴요. 그냥 좀 할 줄 아는 정도예요.
2. 유창하기는요. 아직 부족한 게 많아요.
3. 바쁘기는요. / 바쁘긴요. 오늘은 좀 한가해요.

</details>

### 练习3：-다니

用"竟然…"表达惊讶：

1. 亲自来了 → ******\_\_\_******
2. 10万韩元 → ******\_\_\_******
3. 一个人去旅行 → ******\_\_\_******

<details>
<summary>点击查看答案</summary>

1. 직접 오시다니 감사합니다!
2. 10만 원이라니 너무 비싸요!
3. 혼자 여행을 가다니 대단해요!

</details>

# 제9과: 부탁과 거절 (请求与拒绝)

> 延世韩国语 第三册 · 第9课

---

## 一、学习目标

- 能够礼貌地提出请求和委婉拒绝
- **重点语法**：-(으)ㄹ래요? (你要…吗？)、-아/어 줄래요? (能帮我…吗？)、-(으)ㄹ까 봐(서) (担心…)、-거든요 (因为…你知道)
- 理解韩国文化中请求与拒绝的礼仪

## 三、核心语法

### 语法1: -(으)ㄹ래요? — "你要…吗？/想…吗？" ★★★

表示**征求对方意愿**或**自己的意愿**，比 -(으)ㄹ까요 更随意/亲近。

**构成**：动词词干 + -(으)ㄹ래요

#### 疑问句：征求意愿

```
커피 마실래요?                想喝咖啡吗？
같이 갈래요?                  一起去吗？
뭐 먹을래요?                  想吃什么？
오늘 영화 볼래요?              今天要看电影吗？
이 자리에 앉을래요?            要坐这个位子吗？
```

#### 陈述句：表达自己的意愿

```
저는 이거 먹을래요.            我要吃这个。
나는 집에 있을래요.            我要待在家。
안 갈래요. 피곤해요.           不想去。累了。
```

> **-(으)ㄹ래요 vs -(으)ㄹ까요 vs -(으)ㄹ 거예요**：
> | 表达 | 语感 | 场景 |
> |------|------|------|
> | -(으)ㄹ래요? | 亲切/随意 | 朋友间提议 |
> | -(으)ㄹ까요? | 礼貌/提议 | 较正式提议 |
> | -(으)ㄹ 거예요? | 确认计划 | 问对方的计划 |

> **注意**：-(으)ㄹ래요 **不能用于第三人称**。
>
> - ✅ 같이 갈래요? (你要一起去吗？)
> - ❌ ~~철수 씨가 갈래요?~~ (不能用于别人)

---

### 语法2: -아/어 줄래요? — "能帮我…吗？" ★★★

表示**请求对方为自己做某事**，语气亲切随意。

**构成**：动词词干 + -아/어 줄래(요)?

```
이것 좀 도와줄래요?            能帮一下忙吗？
사진 좀 찍어 줄래요?            能帮拍个照吗？
여기 좀 잡아 줄래요?            能帮扶一下这里吗？
내일 좀 일찍 와 줄래?           明天能早点来吗？
이 짐 좀 들어 줄래?             能帮拎一下这个行李吗？
```

> **请求敬语等级从随意到正式**：
>
> ```
> -아/어 줘.           帮—吧。          (半语/朋友)
> -아/어 줄래(요)?     能帮—吗？         (亲切请求)
> -아/어 주세요.       请帮—。           (礼貌请求)
> -아/어 주시겠어요?    能否帮—呢？       (更礼貌)
> -아/어 주실 수 있으세요?  您能帮—吗？   (最礼貌)
> ```

---

### 语法3: -(으)ㄹ까 봐(서) — "担心…/怕…" ★★★★

表示因为**担心某个不好的事情可能发生**，所以做了/要做某件事。

**构成**：动词/形容词词干 + -(으)ㄹ까 봐(서)

```
비가 올까 봐 우산을 가져왔어요.
怕下雨带了伞。

늦을까 봐 택시를 탔어요.
怕迟到坐了出租车。

시험에 떨어질까 봐 밤새 공부했어요.
怕考试不过通宵学习了。

다칠까 봐 조심했어요.
怕受伤小心了。

잊어버릴까 봐 메모해 놨어요.
怕忘记做了笔记。

감기에 걸릴까 봐 옷을 따뜻하게 입었어요.
怕感冒穿暖了。

부모님이 걱정하실까 봐 말 안 했어요.
怕父母担心没说。
```

> **核心结构**：**担心A + 所以做了B**
>
> - A = -(으)ㄹ까 봐 (担心的内容)
> - B = 采取的行动

---

### 语法4: -거든요 — "因为…你知道/其实是因为" ★★★★

用在句末，为对方**提供补充说明/理由**，让对方理解自己的行为或情况。

**构成**：

- 动词/形容词词干 + -거든요
- 名词 + (이)거든요
- 过去: -았/었거든요

```
A: 왜 밥 안 먹어요?
B: 아까 먹었거든요.           因为刚才吃过了。

A: 추워요?
B: 오늘 코트를 안 입었거든요.   因为今天没穿外套。

A: 한국어를 잘하시네요.
B: 한국에서 3년 살았거든요.    因为在韩国住了3年。

A: 왜 일찍 가요?
B: 내일 시험이거든요.          因为明天有考试。

A: 왜 운동을 시작했어요?
B: 요즘 살이 많이 쪘거든요.    因为最近胖了很多。
```

> **-거든요 vs -잖아요**：
>
> - **-거든요**：对方**不知道**的理由 → 新信息 → "其实是因为…"
> - **-잖아요**：对方**知道**的理由 → 共有信息 → "你也知道嘛…"
>
> ```
> 한국에서 3년 살았거든요. (对方不知道 → 新信息)
> 한국에서 3년 살았잖아요. (对方知道 → 提醒)
> ```

## 四、核心会话

### 对话1：请求帮忙

```
A: 혹시 부탁 하나 해도 돼요?
   可以拜托一件事吗？

B: 네, 뭔데요?
   好，什么事？

A: 내일 이사하는데 와서 좀 도와줄래요?
   明天搬家能来帮忙吗？

B: 내일이요? 몇 시쯤이에요?
   明天？大概几点？

A: 아침 10시부터요. 짐이 많지는 않은데
   혼자 하기 힘들거든요.
   从早上10点。行李不多但
   一个人做的话很吃力。

B: 알겠어요. 갈게요.
   好的。我去。

A: 고마워요! 끝나고 밥 살게요.
   谢谢！结束后我请吃饭。
```

### 对话2：委婉拒绝

```
A: 이번 주말에 바다에 같이 갈래요?
   这个周末一起去海边吗？

B: 아... 그게 좀...
   啊…那个有点…

A: 혹시 다른 일 있어요?
   该不会有别的事吧？

B: 네, 사실은 시험이 얼마 안 남았거든요.
   嗯，其实是考试没剩多少天了。
   점수가 나쁠까 봐 공부를 많이 해야 돼요.
   怕分数不好得多学习。

A: 그렇구나. 그럼 시험 끝나고 가요.
   这样啊。那考试结束后去吧。

B: 네, 그때 꼭 같이 가요! 미안해요.
   好，那时一定一起去！抱歉。

A: 괜찮아요. 시험 잘 봐요!
   没关系。考试加油！
```

### 对话3：帮忙的连锁

```
A: 수미 씨, 오늘 좀 바빠요?
   秀美，今天忙吗？

B: 아니요, 별로요. 왜요?
   不，不太忙。怎么了？

A: 저 대신 교수님께 과제를 좀 내 줄래요?
   能代我把作业交给教授吗？

B: 오늘 못 가요?
   今天不能去？

A: 네, 갑자기 알바를 하게 됐거든요.
   嗯，突然有了兼职。
   교수님이 늦게 내면 감점할까 봐
   부탁드리는 거예요.
   怕教授扣分才拜托你的。

B: 알겠어요. 몇 시까지 내면 돼요?
   好的。几点之前交就行？

A: 5시까지요. 사무실에 가는 김에
   저 대신 출석 체크도 해 줄래요?
   5点之前。趁着去办公室
   也帮我签到好吗？

B: 하하, 그건 좀... 출석은 직접 해야죠.
   哈哈，那个有点…出勤得自己来吧。
```

**新词**：

- **감점** = 扣分
- **출석 체크** = 签到/出勤确认
- **대신** = 代替

## 五、文化补充：韩国的请求与拒绝文化

| 特点                | 说明                                                                           |
| ------------------- | ------------------------------------------------------------------------------ |
| **간접적 거절**     | 韩国人很少直接说"안 돼요"(不行)，而是委婉地说"좀 어려울 것 같아요"(恐怕有点难) |
| **눈치**            | 读懂对方暗示很重要；如果对方犹豫、叹气，可能就是在拒绝                         |
| **"생각해 볼게요"** | "我想想看"经常是委婉拒绝的意思                                                 |
| **"다음에요"**      | "下次吧"也常常是拒绝                                                           |
| **반복 부탁**       | 拒绝一次后对方可能再问；但韩国人一般不会勉强太多次                             |
| **감사와 보답**     | 接受帮助后通常请对方吃饭或送小礼物表示感谢                                     |

> 委婉拒绝常用模式：
>
> ```
> [犹豫] + [理由(-거든요)] + [替代方案] + [道歉]
>
> 예) "그게 좀... 사실 그날 약속이 있거든요.
>      다음 주는 어때요? 미안해요."
> (那个有点…其实那天有约。下周怎么样？不好意思。)
> ```

3. 怕感冒 → 穿了很多 → ******\_\_\_******

2. A: 왜 안 먹어요? B: (肚子疼) → ******\_\_\_******

### 练习3：请求场景

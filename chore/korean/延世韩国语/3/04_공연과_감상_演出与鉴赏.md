# 제4과: 공연과 감상 (演出与鉴赏)

> 延世韩国语 第三册 · 第4课

---

## 一、学习目标

- 能够表达对演出、电影、展览等的感想和评价
- **重点语法**：-(으)ㄴ/는 줄 알았다/몰랐다 (原以为/没想到)、-던 (回忆定语)、-았/었더니 (做了…结果)、-(으)ㄹ 뿐만 아니라 (不仅…而且)
- 学会发表个人感想和艺术评论
- 掌握叙述出人意料的经历

## 三、核心语法

### 语法1: -(으)ㄴ/는 줄 알았다/몰랐다 — "原以为/没想到" ★★★★

表示与事实不同的**预想**或对事实感到意外。

#### -(으)ㄴ/는 줄 알았다 — "还以为…（其实不是）"

```
그 영화가 재미없는 줄 알았는데 정말 재미있었어요.
还以为那电影没意思，结果真的很有趣。

한국 사람인 줄 알았어요.
还以为是韩国人呢。（原来不是）

쉬운 줄 알았는데 어렵더라고요.
还以为很简单，结果很难。

이미 끝난 줄 알았어요.
还以为已经结束了。
```

#### -(으)ㄴ/는 줄 몰랐다 — "没想到…（竟然是这样）"

```
이렇게 감동적인 줄 몰랐어요.
没想到这么感动。

한국어를 그렇게 잘하는 줄 몰랐어요.
没想到韩语说得那么好。

그 가수가 이렇게 유명한 줄 몰랐어요.
没想到那个歌手这么有名。

시간이 이렇게 빨리 지나간 줄 몰랐어요.
没想到时间过得这么快。
```

> **구분**:
>
> - 알았다: 我**以为**是那样 → 其实不是
> - 몰랐다: 我**不知道**竟然是这样 → 表示惊讶

---

### 语法2: -던 — 回忆性定语 "曾经…的/以前…的" ★★★★

表示回忆过去**未完成、反复发生、或中断的动作/状态**。

#### -던 + 名词

```
# 过去经常/反复做的
어릴 때 자주 가던 식당이 없어졌어요.
小时候经常去的餐厅没了。

대학교 때 듣던 노래를 다시 들었어요.
大学时常听的歌又听了。

# 过去进行中的（未完成）
읽던 책을 다시 찾았어요.
正在读的书又找到了。

먹던 음식이 테이블 위에 있어요.
正在吃的食物在桌子上。

# 回忆中的状态
예전에 살던 동네에 다시 가 봤어요.
以前住过的小区又去看了。
```

#### -던 vs -(으)ㄴ 的区别

|      | -던                               | -(으)ㄴ                  |
| ---- | --------------------------------- | ------------------------ |
| 含义 | 回忆·未完成·反复                  | 完成·定论                |
| 例1  | 읽**던** 책 (正读/常读的书)       | 읽**은** 책 (读完了的书) |
| 例2  | 살**던** 집 (以前住过的/曾住的家) | 산 집 (买了的房子)       |
| 例3  | 만나**던** 사람 (以前常见的人)    | 만**난** 사람 (见了的人) |

```
# -았/었던: 过去完成后回忆
내가 어렸을 때 갔었던 바다가 기억나요.
想起了我小时候去过的海。

작년에 봤었던 영화를 또 보고 싶어요.
想再看去年看过的电影。
```

---

### 语法3: -았/었더니 — "做了…结果/却" ★★★

表示**前面做了某个动作，后面是意外的结果或发现**。

#### 主语相同：做了…之后（结果）

```
열심히 연습했더니 실력이 많이 늘었어요.
努力练习了之后实力进步了很多。

약을 먹었더니 금방 나았어요.
吃了药之后很快好了。

밤새 공부했더니 너무 피곤해요.
通宵学习了之后太累了。

많이 걸었더니 다리가 아파요.
走了很多路之后腿疼。
```

#### 主语不同：我做了…对方/情况却…

```
전화했더니 안 받더라고요.
我打电话了但对方没接。

식당에 갔더니 문이 닫혀 있었어요.
去了餐厅结果门关着。

공연장에 갔더니 이미 매진이었어요.
去了演出场结果已经售罄。
```

> **-았/었더니 vs -았/었는데**：
>
> - **-았/었더니**：强调做了A"**所以**"出现了B（因果性更强）
> - **-았/었는데**：做了A"结果"B（单纯背景/转折）

---

### 语法4: -(으)ㄹ 뿐만 아니라 — "不仅…而且…" ★★★

**构成**：V/A-(으)ㄹ 뿐만 아니라 / N뿐만 아니라

```
이 뮤지컬은 노래가 좋을 뿐만 아니라 스토리도
감동적이에요.
这个音乐剧不仅歌好而且故事也很感动。

그 배우는 연기를 잘할 뿐만 아니라 노래도 잘해요.
那个演员不仅演技好而且唱歌也唱得好。

한국어뿐만 아니라 영어도 잘해요.
不仅韩语而且英语也说得好。

이 식당은 맛있을 뿐만 아니라 가격도 저렴해요.
这家餐厅不仅好吃而且价格也便宜。

비가 올 뿐만 아니라 바람도 많이 불어요.
不仅下雨而且风也很大。
```

> **类似表达**：
>
> - -(으)ㄹ 뿐만 아니라 = 不仅…而且… (最常用)
> - -(으)ㄹ 뿐 아니라 = 同上（省略만）
> - -을 뿐더러 = 不仅…而且… (更书面)

## 四、核心会话

### 对话1：看完音乐剧

```
A: 어제 뮤지컬 어땠어요?
   昨天音乐剧怎么样？

B: 정말 감동적이었어요!
   真的很感动！
   이렇게 좋은 줄 몰랐어요.
   没想到这么好。

A: 처음에는 별로인 줄 알았는데?
   一开始还以为一般呢？

B: 맞아요. 솔직히 기대를 안 했는데
   보고 났더니 완전히 생각이 바뀌었어요.
   对。说实话没有期待但
   看了之后想法完全变了。

   배우들이 노래를 잘할 뿐만 아니라 연기도 훌륭했어요.
   演员们不仅歌唱得好而且演技也出色。

A: 그래요? 저도 보러 가야겠다.
   是吗？我也要去看看。

B: 꼭 가 보세요. 대학 때 자주 듣던 노래도 나오더라고요.
   一定去看。大学时经常听的歌也出来了。
```

### 对话2：电影评价

```
A: 요즘 재미있는 영화 있어요?
   最近有好看的电影吗？

B: 지난주에 본 한국 영화가 정말 좋았어요.
   上周看的韩国电影真的很好。
   처음에는 공포 영화인 줄 알았는데
   사실은 감동적인 가족 영화였어요.
   一开始以为是恐怖片结果其实是感人的家庭电影。

A: 어떤 점이 인상적이었어요?
   什么地方印象深刻？

B: 스토리가 독특할 뿐만 아니라
   배우들의 연기가 정말 생동감이 있었어요.
   故事独特不说，演员们的演技也真的很生动。

A: 지루하지 않았어요?
   不无聊吗？

B: 전혀요. 2시간 동안 완전히 몰입했어요.
   完全不。2小时完全沉浸了。
   영화가 끝났더니 관객들이 다 박수를 치더라고요.
   电影结束后观众都鼓了掌。
```

### 对话3：展览观后感

```
A: 이번에 미술관에서 하는 전시회 봤어요?
   这次美术馆的展览看了吗？

B: 네, 어제 다녀왔어요. 대학 때 좋아하던 화가의
   전시회라서 꼭 가고 싶었거든요.
   看了，昨天去了。是大学时喜欢的画家的展览所以一定要去。

A: 어땠어요?
   怎么样？

B: 예전에 책에서만 보던 그림을 직접 봤더니
   느낌이 완전히 달랐어요.
   以前只在书上看过的画亲眼看了之后感觉完全不同。

A: 비쌌어요?
   贵吗？

B: 학생증을 보여 줬더니 할인해 줬어요.
   出示了学生证就给打折了。
   학생이면 꼭 학생증을 가지고 가는 게 좋아요.
   如果是学生一定要带学生证。
```

**新词**：

- **기대하다** = 期待
- **매진** = 售罄
- **박수를 치다** = 鼓掌
- **화가** = 画家
- **느낌** = 感觉
- **학생증** = 学生证

完成句子：

1. 容易 / 其实很难 → 쉬운 줄 ******\_\_\_******
2. 没想到 / 这么感人 → 이렇게 감동적인 줄 ******\_\_\_******
3. 以为是韩国人 → 한국 사람인 줄 ******\_\_\_******

<details>
<summary>点击查看答案</summary>

### 练习3：-았/었더니

用"做了…结果"造句：

1. 운동을 열심히 하다 → ******\_\_\_******

## 七、本课核心句型小结

| 句型                 | 用法              | 例句                          |
| -------------------- | ----------------- | ----------------------------- |
| -(으)ㄴ/는 줄 알았다 | 还以为…(其实不是) | 쉬운 줄 알았어요              |

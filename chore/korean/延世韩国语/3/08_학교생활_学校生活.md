## 三、核心语法

### 语法1: -느라(고) — "因为忙着做…（所以）" ★★★★

表示因为**忙于做某件事**，导致另一件事受到影响（通常是负面的）。

**构成**：动词词干 + -느라(고)

**注意事项**：

1. **只接动词**（不接形容词）
2. **主语必须相同**（前后句同一个人）
3. **后句通常是负面结果**
4. 前句是**持续性的动作**

```
과제 하느라 밤을 새웠어요.
因为忙着做作业通宵了。

아르바이트 하느라 수업에 못 갔어요.
因为忙着打工没能去上课。

전화 받느라 수업에 집중 못 했어요.
因为忙着接电话上课没集中。

이사 준비하느라 바빴어요.
因为忙着准备搬家很忙。

시험 공부하느라 친구를 못 만났어요.
因为忙着备考没能见朋友。

보고서 쓰느라 점심도 못 먹었어요.
因为忙着写报告连午饭都没吃。
```

> **-느라고 vs -는 바람에**（都表达导致负面结果的原因）：
> | | -느라(고) | -는 바람에 |
> |---|----------|----------|
> | 主语 | **必须相同** | 可以不同 |
> | 行为 | **主动/持续性动作** | **突发/意外** |
> | 例 | 공부하느라 못 갔어요 | 비가 오는 바람에 못 갔어요 |

---

### 语法2: -(으)ㄹ 수밖에 없다 — "只能…/不得不…" ★★★★

表示**没有其他选择**，只能那样做。

**构成**：动词/形容词词干 + -(으)ㄹ 수밖에 없다

```
버스가 끊겨서 택시를 탈 수밖에 없었어요.
公交停运了只能坐出租车。

모르는 문제가 많아서 찍을 수밖에 없었어요.
不会的题太多只能瞎猜了。

돈이 없으니까 집에서 밥을 먹을 수밖에 없어요.
没有钱只能在家吃饭。

그 상황에서는 포기할 수밖에 없었어요.
在那种情况下只能放弃了。

시간이 부족해서 벼락치기를 할 수밖에 없었어요.
时间不够只能临时抱佛脚了。

공사 중이라 돌아갈 수밖에 없었어요.
正在施工只能绕路。
```

> **-(으)ㄹ 수밖에 없다 vs -아/어야 하다**：
>
> - -(으)ㄹ 수밖에 없다：强调**无奈/没选择** → 그렇게 할 수밖에 없었어요 (只能那么做)
> - -아/어야 하다：强调**义务/必须** → 그렇게 해야 해요 (得那么做)

---

### 语法3: -는 김에 — "趁着…（顺便）" ★★★

表示**趁着做某件事的机会**，顺便做另一件事。

**构成**：

- 动词词干 + **-는 김에** (现在)
- 动词词干 + **-(으)ㄴ 김에** (过去)

```
도서관에 가는 김에 책도 빌려 올게요.
趁着去图书馆顺便借书回来。

마트에 가는 김에 과일도 좀 사 올게요.
趁着去超市顺便买点水果回来。

나온 김에 커피 한잔 하고 가자.
既然出来了顺便喝杯咖啡走吧。

한국에 온 김에 제주도도 가고 싶어요.
既然来了韩国（趁这机会）也想去济州岛。

청소하는 김에 빨래도 돌렸어요.
趁着打扫卫生顺便也洗了衣服。

이야기 나온 김에 하나만 더 물어볼게요.
既然说到了顺便再问一个。
```

---

### 语法4: -(으)ㄹ 겸 — "顺便/兼顾" ★★★

表示做某件事**兼顾/也有另一个目的**。常两个目的并列。

**构成**：动词词干 + -(으)ㄹ 겸 (+ 동사 -(으)ㄹ 겸)

```
운동할 겸 산책할 겸 공원에 갔어요.
兼顾运动和散步去了公园。

구경할 겸 쇼핑할 겸 명동에 갔어요.
顺便逛逛顺便购物去了明洞。

공부할 겸 커피 마실 겸 카페에 왔어요.
兼顾学习和喝咖啡来了咖啡厅。

기분 전환할 겸 여행을 갔어요.
顺便换个心情去旅行了。

건강도 챙길 겸 헬스장에 다니기 시작했어요.
也为了注意健康开始去健身房了。
```

> **-는 김에 vs -(으)ㄹ 겸**：
>
> - **-는 김에**：已经在做A→**顺便**做B → 가는 김에 (去的话顺便)
> - **-(으)ㄹ 겸**：带着A和B两个目的出发 → 운동할 겸 (为了运动也为了…)

## 四、核心会话

### 对话1：忙碌的大学生活

```
A: 요즘 얼굴 보기가 힘드네요. 뭐 하고 있어요?
   最近很难见到你啊。在干嘛？

B: 기말시험 준비하느라 정신이 없어요.
   因为忙着准备期末考试焦头烂额。

A: 시험이 언제예요?
   考试什么时候？

B: 다음 주부터예요. 과제도 밀려 있어서
   밤을 새울 수밖에 없어요.
   从下周开始。作业也积压着
   只能通宵了。

A: 이번 학기 과목이 많이 힘든가 봐요.
   这学期课程看来很难啊。

B: 네, 전공 네 과목에 교양 두 과목...
   더 열심히 했어야 했는데 많이 놀았어요.
   嗯，专业四门通选两门…
   本该更努力的但玩了很多。

A: 도서관 갈 거면 같이 가요.
   如果去图书馆一起去吧。
   가는 김에 공부도 같이 해요.
   趁着去顺便一起学习。

B: 좋아요!
   好！
```

### 对话2：顺便做点事

```
A: 주말에 뭐 할 거예요?
   周末打算做什么？

B: 운동할 겸 기분 전환할 겸 한강에 가려고요.
   兼顾运动和换个心情打算去汉江。

A: 좋겠다. 나도 같이 가도 돼요?
   好好啊。我也能一起去吗？

B: 그럼요! 가는 김에 치킨도 사 가자.
   当然了！趁着去也带炸鸡去吧。

A: 좋아요. 그런데 과제는 다 했어요?
   好。但作业都做了吗？

B: 아직이요... 과제 때문에 주말에도 놀 수밖에
   없다고 하면 안 되겠죠?
   还没…如果说因为作业周末也只能玩
   不太好吧？

A: 하하, 그건 변명이에요.
   哈哈，那是借口。

B: 알아요. 일요일에 밤새서라도 할게요.
   我知道。周日就算通宵也做完。
```

### 对话3：选课烦恼

```
A: 수강 신청 했어요?
   选课了吗？

B: 네, 그런데 듣고 싶은 과목이
   시간이 겹치는 바람에 하나를 포기할 수밖에 없었어요.
   嗯，但想听的课时间重合了
   只能放弃一门。

A: 어떤 과목을 포기했어요?
   放弃了哪门？

B: 한국 문학을 포기했어요.
   放弃了韩国文学。
   다음 학기에 들을걸 그랬는데
   이미 신청해 버렸어요.
   该留到下学期听的但已经申请了。

A: 정정 기간에 바꿀 수 있지 않아요?
   修改期间不能换吗？

B: 맞다! 정정 기간이 있었지.
   对！有修改期间呢。
   수강 신청 사이트에 들어가는 김에
   시간표도 다시 확인해 볼게요.
   趁着上选课网站也重新确认一下课程表。
```

**新词**：

- **밀리다** = 积压/堆积
- **기분 전환** = 换个心情
- **겹치다** = 重叠
- **정정 기간** = 修改/更正期间

## 五、文化补充：韩国大学文化 (한국 대학 문화)

| 特征               | 说明                                                                |
| ------------------ | ------------------------------------------------------------------- |
| **선후배 문화**    | 前辈/晚辈关系非常重要，첫 만남时要问"몇 학번이에요?" (哪一年入学？) |
| **MT**             | Membership Training，社团/系新生欢迎活动，通常去郊外住一两天        |
| **축제**           | 校园庆典期间有各种公演、摊位和活动，很热闹                          |
| **수강 신청 전쟁** | 选课像打仗一样竞争激烈，热门课几秒就满了                            |
| **카공족**         | 카페+공부+족，在咖啡厅学习的族群                                    |
| **대학 동아리**    | 大学社团种类丰富：학술(学术)、봉사(志愿)、운동(运动)、음악(音乐) 等 |
| **학점 관리**      | 学分管理非常重要，影响就业、奖学金、研究生申请                      |

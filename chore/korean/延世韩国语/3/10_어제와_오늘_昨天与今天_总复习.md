## 三、核心语法

### 语法1: 간접 화법 종합 — 间接引用（转述） ★★★★★

间接引用是中级韩语最重要的语法之一。将别人说的话或自己的想法转述给第三方。

#### 基本构成表

|            | 现在            | 过去              | 未来/推测           |
| ---------- | --------------- | ----------------- | ------------------- |
| **动词**   | -ㄴ/는다고 하다 | -았/었다고 하다   | -(으)ㄹ 거라고 하다 |
| **形容词** | -다고 하다      | -았/었다고 하다   | -(으)ㄹ 거라고 하다 |
| **名词**   | -(이)라고 하다  | -이었/였다고 하다 | -(이)ㄹ 거라고 하다 |

#### 各句型转述

| 句型     | 间接引用                    | 缩略型      |
| -------- | --------------------------- | ----------- |
| **陈述** | -다고 하다                  | **-대(요)** |
| **疑问** | -냐고 하다 / -(으)냐고 하다 | **-냬(요)** |
| **命令** | -(으)라고 하다              | —           |
| **请求** | -아/어 달라고 하다          | —           |
| **建议** | -자고 하다                  | **-재(요)** |

#### 陈述句转述

```
직접: "한국어를 공부해요." (学韩语)
간접: 한국어를 공부한다고 했어요. (说学韩语)
축약: 한국어를 공부한대요. (说学韩语呢)

직접: "어제 비가 왔어요." (昨天下雨了)
간접: 어제 비가 왔다고 했어요. (说昨天下雨了)

직접: "날씨가 좋아요." (天气好)
간접: 날씨가 좋다고 했어요. (说天气好)

직접: "저는 학생이에요." (我是学生)
간접: 자기가 학생이라고 했어요. (说自己是学生)
```

#### 疑问句转述

```
직접: "몇 시에 만날까요?" (几点见？)
간접: 몇 시에 만나냐고 했어요. (问几点见)

직접: "배고파요?" (饿吗？)
간접: 배고프냐고 했어요. (问饿不饿)
```

#### 命令句转述

```
직접: "일찍 오세요." (请早点来)
간접: 일찍 오라고 했어요. (让早点来)

직접: "조용히 하세요." (请安静)
간접: 조용히 하라고 했어요. (让安静)
```

#### 请求转述

```
직접: "도와주세요." (请帮忙)
간접: 도와 달라고 했어요. (拜托帮忙)
```

#### 提议转述

```
직접: "같이 가요." (一起去吧)
간접: 같이 가자고 했어요. (说一起去)
```

#### 缩略型（日常口语高频）

```
비가 온대요.         (据说下雨呢 = 비가 온다고 해요)
내일 간대요.         (说明天去呢 = 내일 간다고 해요)
맛있대요.           (说好吃呢 = 맛있다고 해요)
학생이래요.          (说是学生呢 = 학생이라고 해요)
뭐 먹냬요.          (问吃什么呢 = 뭐 먹느냐고 해요)
같이 가재요.         (说一起去呢 = 같이 가자고 해요)
```

> **引用时人称/时间变化规则**：
>
> - 저/나 → 자기 (他自己)
> - 이/그/저 → 根据实际情况调整
> - 여기 → 거기/그곳
> - 오늘 → 그날

### 语法2: -다면서(요)? — "听说…是吗？/…是真的吗？" ★★★★

用来**确认从别处听到的信息**，带有"我听说了，是真的吗？"的语气。

**构成**：

| 词类        | 构成                  |
| ----------- | --------------------- |
| 动词 (现在) | -ㄴ/는다면서(요)?     |
| 形容词      | -다면서(요)?          |
| 名词        | -(이)라면서(요)?      |
| 过去        | -았/었다면서(요)?     |
| 未来        | -(으)ㄹ 거라면서(요)? |

```
한국에 간다면서요? 정말이에요?
听说要去韩国？真的？

시험에 합격했다면서요? 축하해요!
听说考试通过了？恭喜！

결혼한다면서요? 언제 해요?
听说要结婚了？什么时候？

새로 생긴 식당이 맛있다면서요?
听说新开的餐厅好吃？

의사라면서요? 좀 봐 주세요.
听说你是医生？帮看看吧。

내일 비가 온다면서요?
听说明天下雨？
```

> **-다면서(요) vs -다고 하던데**：
>
> - -다면서요：**向当事人确认** → "听说你要去？(是真的吗？)"
> - -다고 하던데：**转述给第三者** → "听说他要去呢(我记得)"

## 四、核心会话

### 对话1：回顾变化

```
A: 한국에 온 지 벌써 1년이 됐네요.
   来韩国已经1年了呢。

B: 네, 시간이 정말 빠르네요.
   처음 왔을 때 생각나요?
   嗯，时间过得真快。
   还记得刚来的时候吗？

A: 그럼요. 한국어를 하나도 못 해서
   편의점에서 물건 사는 것도 힘들었어요.
   当然。韩语一点都不会
   连在便利店买东西都很难。

B: 지금은 한국어도 잘하고 친구도 많고
   정말 많이 달라졌네요.
   现在韩语也好朋友也多
   真的变了很多。

A: 맞아요. 한국 생활에 적응하느라 힘들었는데
   지금은 편해졌어요.
   对。忙着适应韩国生活很累
   但现在轻松了。

B: 한국에 계속 있을 거라면서요?
   听说要继续留在韩国？

A: 네, 대학원에 진학하려고요.
   嗯，打算考研究生。
```

### 对话2：转述消息

```
A: 민수 씨 소식 들었어요?
   听说民秀的消息了吗？

B: 아니요, 무슨 일이에요?
   没有，什么事？

A: 외국에서 취직했대요. 일본이래요.
   听说在国外找到工作了。说是日本。

B: 정말요? 언제 가냬요?
   真的？问什么时候去？

A: 다음 달에 간다고 했어요.
   说下个月去。
   그래서 이번 주에 모이자고 했어요.
   所以说这周聚一聚。

B: 가기 전에 꼭 만나야겠네요.
   去之前一定得见一面。
   가는 김에 선물도 하나 준비하자.
   趁这机会也准备个礼物吧。
```

### 对话3：今昔对比

```
A: 요즘 한국이 옛날과 많이 달라졌다면서요?
   听说最近韩国跟以前很不同了？

B: 네, 정말 많이 변했어요.
   예전에는 집에 전화기 하나뿐이었는데
   지금은 모두 스마트폰을 쓰잖아요.
   嗯，真的变了很多。
   以前家里就一部电话，
   现在大家都用智能手机嘛。

A: 생활 방식도 많이 바뀌었겠네요.
   生活方式也变了很多吧。

B: 물론이죠. 옛날에는 시장에서 물건을 샀는데
   지금은 인터넷으로 다 시킬 수 있어요.
   当然。以前在市场买东西
   现在在网上都能点。

A: 편리해진 건 좋은데
   사라진 것들이 아쉬울 때도 있겠죠?
   变方便是好的
   但也会有觉得消失的东西可惜的时候吧？

B: 맞아요. 이웃 간의 정이 예전만 못한 것 같아요.
   对。邻里之间的情谊好像不如以前了。
```

**新词**：

- **적응하다** = 适应
- **대학원** = 研究生院
- **진학하다** = 升学
- **이웃** = 邻居
- **정** = 情谊/感情

## 七、语法练习

### 练习1：间接引用

将直接引用改为间接引用：

1. "내일 시험이 있어요." → ******\_\_\_******
2. "몇 시에 만날까요?" → ******\_\_\_******
3. "조용히 하세요." → ******\_\_\_******
4. "같이 밥 먹자." → ******\_\_\_******

<details>
<summary>点击查看答案</summary>

1. 내일 시험이 있다고 했어요.
2. 몇 시에 만나냐고 했어요.
3. 조용히 하라고 했어요.
4. 같이 밥 먹자고 했어요.

</details>

### 练习2：-다면서(요)?

确认听到的消息：

1. 听说要去旅行？ → ******\_\_\_******
2. 听说考试通过了？ → ******\_\_\_******
3. 听说新开的店好吃？ → ******\_\_\_******

<details>
<summary>点击查看答案</summary>

1. 여행 간다면서요?
2. 시험에 합격했다면서요?
3. 새로 생긴 가게가 맛있다면서요?

</details>

### 练习3：综合运用

用第三册学过的语法完成对话：

```
A: 한국어 시험 봤다면서요? 어땠어요?
B: 시험이 생각보다 어려워서 _______ 수밖에 없었어요. (猜)
   더 열심히 _______ 그랬어요. (早该学习)
A: 그래도 시험 _______ 잘하는 편이잖아요. (治)
B: _______(요). 아직 모르는 게 많아요. (哪里)
```

<details>
<summary>点击查看答案</summary>

B: 시험이 생각보다 어려워서 **찍을** 수밖에 없었어요.
더 열심히 **공부할걸** 그랬어요.
A: 그래도 시험 **치르는 게** 잘하는 편이잖아요. (或: 시험은 잘 보는 편이잖아요.)
B: **잘하기는요**. 아직 모르는 게 많아요.

</details>

## 八、本课核心句型小结

| 句型                           | 用法        | 例句            |
| ------------------------------ | ----------- | --------------- |

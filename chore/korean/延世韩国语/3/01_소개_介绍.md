## 三、核心语法

### 语法1: -게 하다 — 使动·"让/使…做" ★★★★★

**第三册最重要的新语法之一**。表示主语让/使某人做某事。

#### 构成

动词/形容词词干 + **-게 하다**

#### 基本含义

```
# "让某人做某事"
선생님이 학생들에게 책을 읽게 했어요.
老师让学生们读书。

부모님이 저를 한국에 유학 오게 하셨어요.
父母让我来韩国留学。

엄마가 동생에게 방을 청소하게 했어요.
妈妈让弟弟打扫房间。
```

#### 三种含义

| 含义            | 例句                                               |
| --------------- | -------------------------------------------------- |
| **허락 (允许)** | 아이에게 게임을 하게 해 주세요. (请允许孩子玩游戏) |
| **지시 (指示)** | 학생들에게 숙제를 하게 했어요. (让学生做作业)      |
| **원인 (导致)** | 그 소식이 저를 기쁘게 했어요. (那个消息让我高兴)   |

#### 更多例句

```
# 허락 (允许)
아이들을 밖에서 놀게 해 주세요.
请让孩子们在外面玩。

# 지시 (指示/命令)
사장님이 직원들에게 야근하게 했어요.
老板让员工加班。

부장님이 저한테 보고서를 다시 쓰게 했어요.
部长让我重新写报告。

# 원인 (使得/导致)
이 영화가 저를 울게 했어요.
这部电影让我哭了。

그 사람의 말이 저를 화나게 했어요.
那个人的话让我生气了。

좋은 음악이 마음을 편하게 해요.
好的音乐让心灵平静。
```

> **-게 하다 vs 사동사 (使动词)**：
>
> - **-게 하다**：通用使动形式，所有动词都可以用
> - **사동사**：特定词根的固定变化（下文详述）
> - 먹게 하다 = 먹이다 (喂)，但 -게 하다 更通用

---

### 语法2: 피동 (被动) 入门 — "-이/히/리/기- 被动词" ★★★★★

韩语的被动通过在动词词根上加 **-이/히/리/기** 后缀构成。

#### 被动词变化表

| 后缀     | 主动         | 被动                     | 中文 |
| -------- | ------------ | ------------------------ | ---- |
| **-이-** | 보다 (看)    | 보이다 (被看到/看得见)   |      |
|          | 쓰다 (用)    | 쓰이다 (被使用)          |      |
|          | 놓다 (放)    | 놓이다 (被放)            |      |
|          | 바꾸다 (换)  | 바뀌다 (被换/改变)       |      |
| **-히-** | 읽다 (读)    | 읽히다 (被读)            |      |
|          | 잡다 (抓)    | 잡히다 (被抓)            |      |
|          | 먹다 (吃)    | 먹히다 (被吃)            |      |
|          | 막다 (挡)    | 막히다 (被堵)            |      |
|          | 닫다 (关)    | 닫히다 (被关)            |      |
| **-리-** | 듣다 (听)    | 들리다 (被听到/听得见)   |      |
|          | 팔다 (卖)    | 팔리다 (被卖)            |      |
|          | 열다 (开)    | 열리다 (被开)            |      |
|          | 풀다 (解)    | 풀리다 (被解开)          |      |
| **-기-** | 안다 (抱)    | 안기다 (被抱)            |      |
|          | 쫓다 (追)    | 쫓기다 (被追)            |      |
|          | 빼다 (拔/减) | 빼기다 → 빼앗기다 (被夺) |      |

#### 被动句的结构变化

```
# 主动: A가 B를 V → 被动: B가 A에게/한테 V-이/히/리/기

주동 (主动): 경찰이 도둑을 잡았어요.
             警察抓了小偷。
피동 (被动): 도둑이 경찰에게 잡혔어요.
             小偷被警察抓了。

주동: 엄마가 문을 열었어요.      妈妈开了门。
피동: 문이 열렸어요.             门被打开了/门开了。

주동: 많은 사람들이 이 책을 읽어요.  很多人读这本书。
피동: 이 책이 많은 사람들에게 읽혀요. 这本书被很多人读。
```

#### 实际高频被动词

```
길이 막혀요.                    路堵了。（막다→막히다）
여기서 산이 보여요.              从这里能看到山。（보다→보이다）
음악 소리가 들려요.              听到音乐声。（듣다→들리다）
이 가게에서 빵이 잘 팔려요.      这店的面包卖得好。（팔다→팔리다）
문이 안 열려요.                  门打不开。（열다→열리다）
```

> ⚠️ 被动是韩语中级**最大的难点之一**，需要大量练习。后续课程会持续深入。

### 语法3: -(으)ㄴ/는 편이다 — "偏向于…/算是…" ★★★

表示虽然不是绝对的，但大致**倾向于**某种状态。

#### 构成

- 动词 → V-**는 편이다**
- 形容词 → A-**(으)ㄴ 편이다**
- 过去 → V-**(으)ㄴ 편이다**

```
# 形容词
저는 내향적인 편이에요.           我算是偏内向的。
한국 겨울은 추운 편이에요.        韩国冬天算冷的。
이 식당은 비싼 편이에요.          这家餐厅算贵的。

# 动词（习惯）
저는 운동을 자주 하는 편이에요.    我算是经常运动的。
책을 많이 읽는 편이에요.          我算是读很多书的。
일찍 일어나는 편이에요.           我算是早起的。

# 过去
어제는 일찍 잠든 편이에요.        昨天算是睡得早的。
```

**否定：안 V/A-는/(으)ㄴ 편이다**

```
매운 음식을 잘 안 먹는 편이에요.
我算是不太吃辣的。

저는 키가 크지 않은 편이에요.
我算是不太高的。
```

---

### 语法4: -에 대해(서) / -에 관해(서) — "关于……" ★★★

表示"关于某个话题/对象"。

```
# -에 대해(서) 更口语化
한국 문화에 대해 관심이 많아요.
对韩国文化很感兴趣。

이 문제에 대해서 이야기해 봅시다.
关于这个问题讨论一下吧。

자기소개에 대해 준비해 오세요.
请准备关于自我介绍的内容。

# -에 관해(서) 更书面化
이 연구에 관한 논문을 읽었어요.
读了关于这项研究的论文。

환경 문제에 관해서 발표하겠습니다.
关于环境问题进行发表。
```

**定语形式**: -에 대한 / -에 관한 + 名词

```
한국 역사에 대한 책을 읽고 있어요.
正在读关于韩国历史的书。

건강에 관한 정보를 찾고 있어요.
在找关于健康的信息。
```

## 四、核心会话

### 对话1：深度自我介绍

```
A: 자기소개 좀 해 주시겠어요?
   能做一下自我介绍吗？

B: 네, 저는 중국에서 온 왕웨이라고 합니다.
   好的，我叫王伟，从中国来的。
   베이징 출신인데 지금은 서울에서 살고 있어요.
   北京出身，现在住在首尔。

A: 한국에 오시게 된 계기가 뭐예요?
   来韩国的契机是什么？

B: 대학교 때 한국 드라마를 보고 한국어에 관심을 갖게 됐어요.
   大学的时候看了韩剧开始对韩语感兴趣了。
   그래서 부모님이 한국에 유학 오게 해 주셨어요.
   所以父母让我来了韩国留学。

A: 성격이 어떤 편이에요?
   性格怎么样？

B: 활발한 편이에요. 새로운 사람을 만나는 걸 좋아하거든요.
   算是活泼的。因为喜欢见新的人。
   그런데 가끔 덜렁대는 편이라 실수도 많아요.
   但有时候比较粗心所以失误也多。

A: 관심사나 취미는 뭐예요?
   兴趣爱好是什么？

B: 한국 문화에 대해 관심이 많아요.
   对韩国文化很感兴趣。
   특히 한국 음식에 대해 배우는 걸 좋아해요.
   特别是喜欢学习韩国料理。
```

### 对话2：介绍朋友

```
A: 저 사람이 누구예요?
   那个人是谁？

B: 제 동아리 선배인 민수 씨예요.
   是我社团的前辈民秀。
   정말 성실한 편이고 책임감이 있어요.
   算是很踏实的，有责任感。

A: 첫인상이 좀 무뚝뚝해 보이는데요.
   第一印象看起来有点冷漠呢。

B: 처음에는 그렇게 보이는데 알고 보면 정말 다정한 사람이에요.
   一开始看起来是那样但了解了后是真的温柔的人。
   저를 많이 도와주셔서 한국 생활에 빨리 적응하게 해 주신 분이에요.
   帮了我很多，是让我快速适应韩国生活的人。

A: 멋지네요. 소개해 주세요!
   好帅啊。给我介绍一下！

B: 그럼요! 저쪽에서 이야기하고 있으니까 가서 인사해요.
   当然！在那边聊着呢，过去打个招呼吧。
```

### 对话3：面试场景

```
면접관: 자기소개와 지원 동기에 대해 말씀해 주세요.
        请说一下自我介绍和求职动机。

지원자: 안녕하십니까. 저는 이번에 졸업한 김지수입니다.
        您好。我是今年毕业的金智秀。
        대학에서 컴퓨터공학을 전공했습니다.
        在大学主修了计算机工程。

면접관: 자신의 장점과 단점에 대해 이야기해 주세요.
        请说说自己的优缺点。

지원자: 장점은 꼼꼼한 편이라 실수가 적은 것입니다.
        优点是比较细心所以失误少。
        단점은 가끔 너무 꼼꼼해서 일이 느려지는 것인데,
        缺点是有时候太仔细导致工作变慢，
        이 단점을 고치려고 시간 관리를 하고 있습니다.
        为了改掉这个缺点正在进行时间管理。

면접관: 이 회사에 지원하게 된 이유가 뭐예요?
        来这家公司应聘的原因是什么？

지원자: 이 회사의 기술력에 대해 많이 들었습니다.
        关于这家公司的技术实力听说了很多。
        제 전공을 살릴 수 있는 회사라고 생각합니다.
        我认为是能发挥我专业的公司。
```

**新词**：

- **계기** = 契机/起因
- **적응하다** = 适应
- **면접관** = 面试官
- **지원자** = 应聘者
- **지원 동기** = 求职动机
- **전공하다** = 主修
- **고치다** = 改正/修理
- **살리다** = 发挥/活用

## 五、使动与被动 基础对照表

| 主动      | 使动(-게 하다)   | 使动词(사동사)  | 被动词(피동사)  |
| --------- | ---------------- | --------------- | --------------- |
| 먹다 (吃) | 먹게 하다 (让吃) | 먹이다 (喂)     | 먹히다 (被吃)   |
| 보다 (看) | 보게 하다 (让看) | 보이다 (让看见) | 보이다 (被看到) |
| 읽다 (读) | 읽게 하다 (让读) | 읽히다 (让读)   | 읽히다 (被读)   |
| 웃다 (笑) | 웃게 하다 (使笑) | 웃기다 (逗笑)   | —               |
| 울다 (哭) | 울게 하다 (使哭) | 울리다 (弄哭)   | —               |
| 입다 (穿) | 입게 하다 (让穿) | 입히다 (给穿)   | —               |
| 눕다 (躺) | 눕게 하다 (让躺) | 눕히다 (放躺)   | —               |
| 앉다 (坐) | 앉게 하다 (让坐) | 앉히다 (让坐下) | —               |

> 💡 **记忆技巧**：使动词和被动词的后缀相同（-이/히/리/기），要通过**语境**区分。

## 六、文化补充：韩国的自我介绍文化

```
🤝 한국의 자기소개 문화

1. 나이를 물어봐요!
   韩国人认识后很快就会问年龄。
   → 这是为了确定 반말/존댓말 的使用。
   → "몇 년생이세요?" (几几年的？) 是常用问法。

2. 선후배 관계가 중요해요.
   前后辈关系很重要。
   → 大学里 동기 (同期)、선배 (学长)、후배 (学弟妹)
   → 工作中 선임 (前任)、후임 (后任)

3. 명함 문화
   还有名片文化。
   → 两手递名片和接名片
   → 先仔细看名片内容再收好

4. "저는 ○○○이라고 합니다"
   正式自我介绍用 "-라고 합니다" 而非 "-예요/이에요"
```

## 八、本课核心句型小结

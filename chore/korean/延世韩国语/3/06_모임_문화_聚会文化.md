## 三、核心语法

### 语法1: -(으)ㄹ 테니(까) — "因为我打算…/因为会…" ★★★★

表示说话人的**意志/推测**作为理由，然后给对方建议或请求。

**构成**：

- 有收音：-을 테니(까) → 먹을 테니까
- 无收音：-ㄹ 테니(까) → 갈 테니까
- ㄹ收音：脱ㄹ + -ㄹ 테니(까) → 만들 테니까

#### 表意志（第一人称）

```
제가 예약할 테니까 장소 걱정 마세요.
我来预约，(你）不用担心场地。

내가 준비할 테니까 그냥 와.
我来准备，你来就行。

내가 차를 가져갈 테니 같이 타고 가자.
我开车去，一起坐车走吧。
```

#### 表推测（第三人称/一般情况）

```
내일 비가 올 테니까 우산을 가져가세요.
明天会下雨，带伞吧。

주말에는 사람이 많을 테니 미리 예약합시다.
周末人会多，提前预约吧。

길이 막힐 테니까 일찍 출발하세요.
路会堵，早点出发吧。
```

> **-(으)ㄹ 테니까 vs -(으)ㄹ 거니까**：
>
> - -(으)ㄹ 테니까：更强的意志/确信，常带**命令/建议** → 제가 할 테니까 쉬세요
> - -(으)ㄹ 거니까：温和的计划/推测 → 제가 할 거니까 걱정 마세요

---

### 语法2: -는 척하다 — "假装…" ★★★★

表示装作/假装做某种行为或处于某种状态。

**构成**：

| 词类   | 现在           | 过去           |
| ------ | -------------- | -------------- |
| 动词   | -는 척하다     | -(으)ㄴ 척하다 |
| 形容词 | -(으)ㄴ 척하다 | —              |
| 名词   | 인 척하다      | —              |

```
모르는 척하다                假装不知道
잠이 든 척하다               假装睡着了
아픈 척하다                  假装生病
안 들은 척하다               假装没听见
괜찮은 척하다                假装没事
학생인 척하다                假装是学生
```

#### 实际例句

```
수업이 지루해서 듣는 척만 했어요.
课很无聊只是假装在听。

상사가 와서 바쁜 척했어요.
上司来了假装很忙。

처음 만나는 척하지 마. 우리 전에 만났잖아.
别假装第一次见面。我们之前见过嘛。

슬펐지만 괜찮은 척했어요.
虽然伤心但假装没事。

놀란 척하지 마세요. 다 알고 있었잖아요.
别装惊讶了，你都知道嘛。
```

> **注意**：**-는 체하다** 和 -는 척하다 意思一样，**체하다** 更书面。

---

### 语法3: -(으)ㄹ 정도로 — "到…的程度" ★★★

表示达到某种程度，强调事物的程度之深。

**构成**：动词/形容词词干 + -(으)ㄹ 정도(로)

```
눈물이 날 정도로 감동적이었어요.
感动到流泪的程度。

못 알아볼 정도로 변했어요.
变得认不出来了（到认不出来的程度）。

걸을 수 없을 정도로 아팠어요.
疼到走不了路的程度。

한국 사람인 줄 알았을 정도로 한국어를 잘해요.
韩语好到以为是韩国人的程度。

배가 터질 정도로 많이 먹었어요.
吃了很多到肚子要撑破的程度。

정신을 못 차릴 정도로 바빴어요.
忙到缓不过神的程度。
```

#### 搭配变体

```
-(으)ㄹ 정도이다/-(으)ㄹ 정도다    → 达到某种程度
-(으)ㄹ 정도로                      → 到某种程度地（修饰后面动词/形容词）
-(으)ㄹ 정도의 + 名词               → 达到某种程度的（修饰名词）
```

### 语法4: -고 보니(까) — "做了…之后才发现/才知道" ★★★

表示做了某个动作**之后才发现/才知道**了某个事实。

**构成**：动词词干 + -고 보니(까)

```
한국에 오고 보니 생각보다 물가가 비쌌어요.
来了韩国才发现物价比想象中贵。

직접 해 보고 보니 생각보다 어려웠어요.
亲自做了才发现比想象中难。

결혼하고 보니 혼자 살 때가 편했어요.
结婚后才发现一个人住的时候更舒服。

알고 보니 우리 같은 학교 출신이었어요.
了解之后才知道我们是同一个学校出来的。

사고 보니 이미 가지고 있던 거였어요.
买了之后才发现已经有了。
```

> **-고 보니 vs -다 보니**：
>
> - **-고 보니**：一次性动作后发现 → 먹고 보니 맛있었어요 (吃了才知道好吃)
> - **-다 보니**：反复/持续动作中发现 → 먹다 보니 맛있었어요 (一直吃着吃着发现好吃)

## 四、核心会话

### 对话1：计划聚会

```
A: 이번에 동호회 모임은 언제 해요?
   这次兴趣社团聚会什么时候？

B: 이번 주 토요일이요. 참석할 수 있어요?
   这周六。能来吗？

A: 토요일이요? 제가 장소를 잡을 테니까
   시간만 알려 주세요.
   周六？我来定场地，
   你告诉我时间就行。

B: 고마워요. 6시에 시작할까요?
   谢谢。6点开始怎么样？

A: 좋아요. 참석 인원이 어떻게 돼요?
   好的。参加人数多少？

B: 한 열 명 정도요. 작은 식당 하나 잡아 주세요.
   大概十个人。定个小餐厅就行。

A: 알겠어요. 회비는 각자 내는 거죠?
   知道了。会费各自付吧？

B: 네, 각자 내요.
   嗯，各自付。
```

### 对话2：假装没事

```
A: 어제 회식 재미있었어요?
   昨天聚餐好玩吗？

B: 하나도 안 재미있었는데 재미있는 척했어요.
   一点也不好玩但假装很开心。

A: 왜요? 분위기가 안 좋았어요?
   为什么？气氛不好吗？

B: 부장님이 계속 노래를 시키시는 바람에
   노래를 못 하는 저는 죽을 정도로 힘들었어요.
   部长一直让唱歌，
   不会唱歌的我累到要死的程度。

A: 하하, 그래도 잘한 척은 했어요?
   哈哈，但假装唱得好了吗？

B: 아니요, 잘 부르는 척도 못 했어요.
   没有，连假装唱得好都没做到。
   그냥 가서 노래를 부르고 보니
   의외로 다 박수를 쳐 줘서 기분이 좋아졌어요.
   但上去一唱发现大家意外地都给鼓掌心情就好了。
```

**新词**：

- **부장님** = 部长
- **노래를 시키다** = 让唱歌 (使动)
- **의외로** = 意外地
- **박수를 치다** = 鼓掌

## 五、文化补充：韩国的聚会文化 (한국의 모임 문화)

| 特征                | 说明                                                                |
| ------------------- | ------------------------------------------------------------------- |
| **1차 → 2차 → 3차** | 韩国聚会常常不止一场，1次 = 正餐，2次 = 酒吧/노래방，3次 = 深夜小吃 |
| **회식**            | 公司聚餐，过去几乎强制参加，现在越来越尊重个人选择                  |
| **건배사**          | 干杯词，常用: "위하여!", 也有创意干杯词                             |
| **선배·후배**       | 聚会中前辈/晚辈文化明显，晚辈倒酒要转头喝                           |
| **각자 vs 한턱**    | 年轻人倾向AA制，但前辈请客仍常见                                    |
| **번개**            | 在SNS上突然约的聚会，"번개 = 闪电"→ 闪聚                            |

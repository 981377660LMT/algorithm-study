# 제3과: 날씨와 생활 (天气与生活)

> 延世韩国语 第三册 · 第3课

---

## 一、学习目标

- 能够用更丰富的表达讨论天气对生活的影响
- **重点语法**：-다 보면/보니(까) (做着做着就…/做了发现…)、-(으)ㄴ/는 탓에 (因为…的缘故·归咎)、-아/어 가다·오다 (逐渐…)、-더라(고요) (回忆追述)
- 掌握中级阶段的叙事能力：回忆、归因、渐变过程

## 三、核心语法

### 语法1: -다 보면 / -다 보니(까) — 做着做着就… ★★★★

这是中级核心语法，表示**持续做某事的过程中，产生了某种结果或发现**。

#### -다 보면：做着做着就会… (假设性/一般性)

```
한국에 살다 보면 한국어가 자연스럽게 늘어요.
在韩国住着住着韩语自然就会进步。

운동을 하다 보면 체력이 좋아질 거예요.
运动着运动着体力会变好的。

열심히 하다 보면 좋은 결과가 있을 거예요.
努力着努力着会有好结果的。

한국 드라마를 보다 보면 한국 문화를 이해하게 돼요.
看着韩剧看着看着就会理解韩国文化。
```

#### -다 보니(까)：做着做着发现/结果是… (实际经验)

```
한국에 살다 보니까 매운 음식도 잘 먹게 됐어요.
在韩国住着住着发现辣的也能吃了。

공부하다 보니까 벌써 밤 12시였어요.
学着学着发现已经半夜12点了。

일하다 보니까 시간이 금방 지나갔어요.
工作着工作着时间很快就过去了。

혼자 살다 보니까 요리를 잘하게 됐어요.
独居着独居着做菜变好了。
```

> **-다 보면 vs -다 보니까**：
>
> - **-다 보면**：一般性推测 "做着做着就**会**…" → 还没发生/普遍规律
> - **-다 보니까**：亲身经验 "做着做着**发现**…" → 已经发生了的事实

---

### 语法2: -(으)ㄴ/는 탓에 — "因为…的缘故（归咎·负面）" ★★★

表示**负面结果的原因**，带有"怪…/都是因为…"的语气。

**构成**：

- 动词过去 → V-(으)ㄴ 탓에
- 动词现在 → V-는 탓에
- 形容词 → A-(으)ㄴ 탓에
- 名词 → N 탓에

```
비가 많이 온 탓에 길이 막혔어요.
因为下了很多雨路堵了。

미세먼지가 심한 탓에 밖에 나가지 못했어요.
因为雾霾严重没能出门。

날씨가 너무 더운 탓에 잠을 잘 수 못 잤어요.
因为天气太热没睡好觉。

폭설이 온 탓에 비행기가 취소됐어요.
因为暴雪飞机取消了。

내 탓이에요. / 제 탓이에요.
是我的错。

날씨 탓에 기분이 안 좋아요.
因为天气心情不好。
```

> **탓 vs 덕분 vs 때문**：
> | 表达 | 语气 | 例句 |
> |------|------|------|
> | -탓에 | **负面**归咎 | 비 탓에 못 갔어요 (怪下雨没去成) |
> | -덕분에 | **正面**归功 | 선생님 덕분에 합격했어요 (多亏老师通过了) |
> | -때문에 | **中性**原因 | 비 때문에 못 갔어요 (因为下雨没去) |

---

### 语法3: -아/어 가다 · -아/어 오다 — "逐渐…" ★★★

表示某种变化的**渐进过程**。

#### -아/어 가다：从现在向未来渐变 (离开说话者)

```
날씨가 점점 추워져 가요.
天气渐渐变冷了（还会继续冷）。

해가 지면서 어두워져 가요.
太阳落下来渐渐暗了。

시간이 지나가면서 잊혀져 가요.
随着时间过去渐渐被忘记了。

겨울이 다가오면서 낮이 짧아져 가요.
冬天临近白天渐渐变短了。
```

#### -아/어 오다：从过去到现在渐变 (朝说话者方向)

```
한국어 실력이 좋아져 왔어요.
韩语水平渐渐提高了（到现在为止）。

지금까지 열심히 살아왔어요.
到现在一直努力地生活了。

봄이 되면서 날씨가 따뜻해져 왔어요.
到了春天天气渐渐变暖了。

이 문제에 대해 오랫동안 생각해 왔어요.
关于这个问题长时间地思考过来了。
```

> **가다 vs 오다 的时间方向**：
>
> - **-아/어 가다**：现在→未来 (远离说话时间) → 추워져 **가요** (还在变冷)
> - **-아/어 오다**：过去→现在 (靠近说话时间) → 따뜻해져 **왔어요** (已变暖到现在)

---

### 语法4: -더라(고요) — "（回忆）我发现…/我记得…" ★★★★

表示说话者**回忆过去亲身经历或目睹的事情**，并将这个信息传达给对方。

**构成**：动词/形容词词干 + -더라고요 (敬语) / -더라 (半语)

```
어제 밖에 나갔는데 날씨가 정말 춥더라고요.
昨天出去了，天气真的很冷（我发现）。

그 식당에 가 봤는데 음식이 맛있더라고요.
去了那家餐厅，食物很好吃（我发现）。

한국 사람들은 매운 음식을 정말 잘 먹더라고요.
韩国人真的很能吃辣的（我观察到的）。

직접 해 보니까 생각보다 어렵더라고요.
亲自做了发现比想象的难。

처음에는 힘들었는데 하다 보니까 재미있더라고요.
一开始很难但做着做着发现挺有趣的。
```

> ⚠️ **-더라고요 的限制**：
>
> 1. 只能用于**第一人称的经历/观察** → "我"亲眼见、亲身感受
> 2. 主语是"我"时，只能用于**非意志性**状态 → ✅ 배가 고프더라고요 ❌ 밥을 먹더라고요
> 3. 不能用于**正在发生的事** → 是对过去的回忆

## 四、核心会话

### 对话1：天气影响

```
A: 요즘 날씨가 미세먼지 때문에 안 좋죠?
   最近因为雾霾天气不好吧？

B: 네, 미세먼지가 심한 탓에 밖에 못 나가겠어요.
   是啊，因为雾霾严重出不了门。

A: 어제 밖에 나갔는데 하늘이 완전 뿌옇더라고요.
   昨天出去了发现天空完全灰蒙蒙的。

B: 맞아요. 저도 마스크를 쓰고 나갔는데 그래도 목이 아프더라고요.
   对。我也戴着口罩出去了但嗓子还是疼。

A: 이렇게 계속 나쁘면 건강이 안 좋아져 갈 텐데요.
   这样继续差的话健康也会变差的。

B: 맞아요. 공기청정기를 사야지, 하고 있었는데
   아직 못 샀어요.
   对。一直想着得买空气净化器但还没买。
```

### 对话2：季节变化

```
A: 한국의 사계절 중에서 뭐가 제일 좋았어요?
   韩国四季中最喜欢什么？

B: 가을이요. 한국에 처음 왔을 때 가을이었는데,
   단풍이 정말 아름답더라고요.
   秋天。第一次来韩国时是秋天，
   红叶真的很美。

A: 한국 가을은 정말 예쁘죠.
   韩国秋天确实很美。

B: 네, 그런데 한국에 살다 보니까 가을이
   점점 짧아져 가는 것 같아요.
   是，但在韩国住着发现秋天好像渐渐变短了。

A: 기후 변화 탓에 그런 것 같아요.
   好像是气候变化的缘故。
   여름은 점점 길어져 오고 겨울은 갑자기 와요.
   夏天渐渐变长冬天突然就来了。

B: 맞아요. 지구 온난화 문제에 대해 생각하게 돼요.
   对。让人开始思考全球变暖问题。
```

### 对话3：生活经验分享

```
A: 한국에서 겨울을 처음 보내는 건 어때요?
   在韩国第一次过冬怎么样？

B: 정말 춥더라고요! 상상한 것보다 훨씬 추워요.
   真的很冷！比想象的冷多了。

A: 한국 겨울은 바람이 많이 불어서 더 춥게 느껴져요.
   韩国冬天风大所以更冷。

B: 맞아요. 바람이 강한 탓에 체감 온도가 영하 20도까지
   내려가더라고요.
   对。因为风大体感温度降到零下20度。

A: 살다 보면 익숙해질 거예요.
   住着住着就会习惯的。
   한국 사람들은 겨울에도 아이스 아메리카노를 마시거든요.
   韩国人冬天也喝冰美式呢。

B: 하하, 그건 아직 이해가 안 돼요.
   哈哈，那个我还理解不了。
```

**新词**：

- **뿌옇다** = 灰蒙蒙/模糊
- **공기청정기** = 空气净化器
- **체감 온도** = 体感温度
- **상상하다** = 想象

3. 运动 / 做着做着 / 体力会变好（推测） → ******\_\_\_******

1. 선생님 ( ) 시험에 합격했어요.
2. 태풍 ( ) 행사가 취소됐어요.
3. 열심히 공부한 ( ) 좋은 성적을 받았어요.

<summary>点击查看答案</summary>

</details>

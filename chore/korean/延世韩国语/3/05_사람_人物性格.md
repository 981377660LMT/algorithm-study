# 제5과: 사람 (人物/性格)

> 延世韩国语 第三册 · 第5课

---

## 一、学习目标

- 能够深入描述人的外貌、性格和行为习惯
- **重点语法**：사동사 심화 (使动词深化)、-아/어 있다 vs -고 있다 (状态持续 vs 动作进行)、-(으)ㄹ 뻔하다 (差点…)、-는 바람에 (因为…结果)
- 掌握更细腻的人物描写能力

## 三、核心语法

### 语法1: 사동사 심화 — 使动词详解 ★★★★★

第1课学了 -게 하다（通用使动），本课深入学习**使动词后缀**。

#### 使动后缀：-이/히/리/기/우/추/구-

| 后缀     | 原动词      | 使动词                 | 含义    |
| -------- | ----------- | ---------------------- | ------- |
| **-이-** | 먹다 (吃)   | **먹이다** (喂)        | 让吃→喂 |
|          | 보다 (看)   | **보이다** (让看/显示) |         |
|          | 죽다 (死)   | **죽이다** (杀)        | 让死→杀 |
|          | 높다 (高)   | **높이다** (提高)      |         |
|          | 줄다 (减)   | **줄이다** (减少)      |         |
| **-히-** | 읽다 (读)   | **읽히다** (让读/被读) |         |
|          | 입다 (穿)   | **입히다** (给穿)      |         |
|          | 앉다 (坐)   | **앉히다** (让坐)      |         |
|          | 눕다 (躺)   | **눕히다** (放躺)      |         |
|          | 익다 (熟)   | **익히다** (使熟/掌握) |         |
| **-리-** | 울다 (哭)   | **울리다** (弄哭)      |         |
|          | 알다 (知道) | **알리다** (通知/让知) |         |
|          | 살다 (活)   | **살리다** (救活)      |         |
|          | 돌다 (转)   | **돌리다** (转动/旋转) |         |
| **-기-** | 웃다 (笑)   | **웃기다** (逗笑)      |         |
|          | 벗다 (脱)   | **벗기다** (给脱)      |         |
|          | 맡다 (闻)   | **맡기다** (托付/委托) |         |
| **-우-** | 깨다 (醒)   | **깨우다** (叫醒)      |         |
|          | 자다 (睡)   | **재우다** (哄睡)      |         |
|          | 타다 (烧)   | **태우다** (烧/让烧)   |         |
| **-추-** | 낮다 (低)   | **낮추다** (降低)      |         |
|          | 맞다 (对)   | **맞추다** (对准/调整) |         |

#### 实际例句

```
엄마가 아이에게 밥을 먹였어요.       妈妈喂了孩子饭。
선생님이 학생을 앉혔어요.            老师让学生坐下。
그 이야기가 모두를 울렸어요.          那个故事让所有人哭了。
그 코미디언이 관객을 웃겼어요.        那个喜剧演员逗笑了观众。
아침에 동생을 깨웠어요.             早上叫醒了弟弟。
음악 소리를 좀 낮춰 주세요.          请把音乐声音降低一些。
```

> **사동사 vs -게 하다**：
>
> - **사동사**：更自然/直接 → 아이를 재웠어요 (哄睡了孩子)
> - **-게 하다**：更间接/通用 → 아이를 자게 했어요 (让孩子睡了)
> - 能用사동사的地方优先用사동사，更像母语者

### 语法2: -아/어 있다 vs -고 있다 — 状态持续 vs 动作进行 ★★★★

#### -아/어 있다：动作完成后状态持续

```
문이 열려 있어요.               门是开着的（被打开后保持着）。
의자에 앉아 있어요.             坐着（坐下后保持着）。
벽에 그림이 걸려 있어요.         墙上挂着画。
눈이 쌓여 있어요.              雪堆着。
책이 테이블 위에 놓여 있어요.     书放在桌子上。
```

#### -고 있다：动作正在进行

```
지금 문을 열고 있어요.           现在正在开门。
의자에 앉고 있어요.             正在坐下。
그림을 걸고 있어요.             正在挂画。
지금 공부하고 있어요.            正在学习。
밥을 먹고 있어요.               正在吃饭。
```

#### 核心区别

|      | -아/어 있다              | -고 있다               |
| ---- | ------------------------ | ---------------------- |
| 释义 | 动作完成后的**状态**持续 | 动作正**进行中**       |
| 例   | 앉아 있다 (坐着·已坐好)  | 앉고 있다 (正在坐下)   |
| 例   | 서 있다 (站着·已站好)    | 서고 있다 (正在站起来) |
| 搭配 | 多用自动词/被动          | 可用所有动词           |

**特殊情况：穿着**

```
모자를 쓰고 있어요.              = 戴着帽子 (状态 或 正在戴)
안경을 쓰고 있어요.              = 戴着眼镜
빨간 옷을 입고 있어요.            = 穿着红衣服
# 穿戴类动词 -고 있다 既表进行也表状态
```

### 语法3: -(으)ㄹ 뻔하다 — "差点…" ★★★

表示某件事**差一点就发生了但最终没发生**。

**构成**：动词词干 + -(으)ㄹ 뻔하다 (→ 过去: -(으)ㄹ 뻔했다)

```
길에서 넘어질 뻔했어요.           差点在路上摔倒。
버스를 놓칠 뻔했어요.             差点错过公交车。
약속을 잊어버릴 뻔했어요.          差点忘了约定。
늦잠을 자서 지각할 뻔했어요.       因为睡过了差点迟到。
비밀을 말할 뻔했어요.             差点说出了秘密。
그 사람과 부딪힐 뻔했어요.         差点和那个人撞上。
울 뻔했어요.                     差点哭了。
```

---

### 语法4: -는 바람에 — "因为（突发/意外）…结果" ★★★

表示因为**突发、意外的事情**发生，导致了（通常不好的）结果。

**构成**：动词词干 + **-는 바람에** (只接动词)

```
갑자기 비가 오는 바람에 옷이 다 젖었어요.
因为突然下雨衣服全湿了。

알람이 안 울리는 바람에 늦잠을 잤어요.
因为闹钟没响睡过了头。

친구가 갑자기 오는 바람에 약속을 취소했어요.
因为朋友突然来了取消了约定。

길이 막히는 바람에 약속에 늦었어요.
因为堵车约会迟到了。

전화가 오는 바람에 집중을 못 했어요.
因为电话来了没能集中。
```

> **原因表达大对比**：
> | 表达 | 原因特点 | 结果倾向 |
> |------|---------|---------|
> | -아/어서 | 一般原因 | 中性 |
> | -(으)니까 | 主观/发现 | 中性 |
> | -기 때문에 | 正式/书面 | 中性 |
> | -는 바람에 | **突发/意外** | **负面** |
> | -(으)ㄴ/는 탓에 | **归咎** | **负面** |
> | -덕분에 | **归功** | **正面** |

## 四、核心会话

### 对话1：描述人物

```
A: 새로 온 팀원 봤어요? 어떤 사람이에요?
   看到新来的组员了吗？什么样的人？

B: 키가 크고 안경을 쓰고 있는 사람이요?
   个子高戴着眼镜的人？

A: 네, 그 분이요.
   对，那位。

B: 첫인상은 좀 조용한 편이었는데,
   이야기를 나눠 보니까 유머가 있고 적극적이더라고요.
   第一印象算是比较安静的，
   但聊了之后发现幽默又积极。

A: 일은 잘해요?
   工作做得好吗？

B: 네, 꼼꼼한 편이라 실수가 거의 없어요.
   嗯，算是很细心的所以几乎没有失误。
   그리고 눈치가 빨라서 분위기 파악을 잘하더라고요.
   而且察言观色很快能看懂气氛。
```

### 对话2：差点出事

```
A: 어제 무슨 일 있었어요? 얼굴이 안 좋아 보여요.
   昨天有什么事？脸色看起来不好。

B: 어제 차 사고가 날 뻔했어요.
   昨天差点出车祸。

A: 어머, 정말요? 무슨 일이 있었어요?
   天哪，真的？怎么了？

B: 횡단보도를 건너고 있었는데
   갑자기 차가 달려오는 바람에 깜짝 놀랐어요.
   正过人行横道突然车冲过来吓了一跳。

A: 다치지는 않았어요?
   没受伤吧？

B: 네, 옆에 있던 사람이 저를 잡아 줘서 괜찮았어요.
   嗯，旁边的人拉住了我所以没事。
   정말 크게 다칠 뻔했어요.
   真的差点受大伤。

A: 정말 다행이다. 앞으로 조심하세요.
   真万幸。以后小心。
```

### 对话3：使动词活用

```
A: 어제 조카가 놀러 왔는데 정말 힘들었어요.
   昨天侄子来玩真的很累。

B: 왜요? 뭐가 힘들었어요?
   怎么了？什么累？

A: 먹이고, 입히고, 재우고...
   하루 종일 돌봤어요.
   喂饭、给穿衣服、哄睡觉…
   照顾了一整天。

B: 아이 돌보는 게 정말 힘들죠.
   照顾孩子真的很累对吧。

A: 네, 밤에 재우는 게 제일 힘들었어요.
   嗯，晚上哄睡觉最累。
   계속 울리는 바람에 한 시간이나 걸렸어요.
   因为一直哭了一个小时才哄睡。
   저도 같이 울 뻔했어요.
   我也差点一起哭了。

B: 하하, 고생했네요.
   哈哈，辛苦了。
```

**新词**：

- **횡단보도** = 人行横道
- **깜짝 놀라다** = 吓一跳
- **조카** = 侄子/外甥
- **돌보다** = 照顾

## 五、语法练习

### 练习1：사동사

用使动词改写：

1. 아이가 밥을 먹다 → 엄마가 아이에게 밥을 **\_\_\_**
2. 동생이 울다 → 형이 동생을 **\_\_\_**
3. 아이가 자다 → 엄마가 아이를 **\_\_\_**
4. 소리가 낮다 → 소리를 **\_\_\_**

<details>
<summary>点击查看答案</summary>

1. 먹였어요 (먹이다)
2. 울렸어요 (울리다)
3. 재웠어요 (재우다)
4. 낮췄어요 (낮추다)

</details>

### 练习2：-아/어 있다 vs -고 있다

选择合适的形式：

1. 문이 닫\_\_\_ 있어요. (关着的状态)
2. 지금 밥을 먹\_\_\_ 있어요. (正在吃)
3. 벽에 사진이 걸\_\_\_ 있어요. (挂着的状态)
4. 빨간 옷을 입\_\_\_ 있어요. (穿着)

<details>
<summary>点击查看答案</summary>

1. 닫혀 있어요 (-아/어 있다, 状态)
2. 먹고 있어요 (-고 있다, 进行)
3. 걸려 있어요 (-아/어 있다, 状态)
4. 입고 있어요 (-고 있다, 穿着状态)

</details>

### 练习3：-(으)ㄹ 뻔하다

用"差点…"造句：

1. 迟到 → ******\_\_\_******
2. 摔倒 → ******\_\_\_******
3. 哭 → ******\_\_\_******

<details>
<summary>点击查看答案</summary>

1. 지각할 뻔했어요. / 늦을 뻔했어요.
2. 넘어질 뻔했어요.
3. 울 뻔했어요.

</details>

## 六、本课核心句型小结

| 句型                        | 用法            | 例句                      |
| --------------------------- | --------------- | ------------------------- |
| 사동사 (-이/히/리/기/우/추) | 让/使…做        | 아이를 재웠어요           |
| -아/어 있다                 | 状态持续        | 문이 열려 있어요          |
| -고 있다                    | 动作进行        | 지금 먹고 있어요          |
| -(으)ㄹ 뻔하다              | 差点…           | 넘어질 뻔했어요           |
| -는 바람에                  | 因为(突发)…结果 | 비가 오는 바람에 젖었어요 |

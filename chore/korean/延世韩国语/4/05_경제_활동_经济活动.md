> 延世韩国语 第四册 · 第5课

---

## 一、学习目标

- 能够用韩语讨论经济、消费、投资等话题
- **重点语法**：-는 데다(가) (加上/而且还)、-(으)ㄹ 텐데 (估计…/应该会…不过)、-는 대신(에) (代替…/作为…的交换)、-만 못하다 (不如…)
- 掌握经济金融基础词汇

## 三、核心语法

### 语法1: -는 데다(가) — "加上/不仅…而且还" ★★★★

在已有情况的基础上**追加**另一个情况，两者叠加。

**构成**：

| 词类   | 构成             |
| ------ | ---------------- |
| 动词   | -는 데다(가)     |
| 形容词 | -(으)ㄴ 데다(가) |
| 过去   | -(으)ㄴ 데다(가) |

```
물가가 오르는 데다가 월급은 그대로예요.
物价在涨加上月薪没变。

비가 오는 데다가 바람까지 불어요.
下雨加上还刮风。

맛있는 데다가 가격도 싸요.
好吃加上价格也便宜。

교통이 편리한 데다가 주변에 편의시설도 많아요.
交通方便加上周边便利设施也多。

감기에 걸린 데다가 일도 많아서 힘들어요.
感冒了加上工作也多所以很累。

경험이 많은 데다가 성격도 좋아서 인기가 많아요.
经验丰富加上性格好所以很受欢迎。
```

> **"加上"系列对比**：
> | 表达 | 特点 |
> |------|------|
> | -는 데다(가) | **累积叠加** (既A又B) |
> | -(으)ㄹ 뿐(만) 아니라 | 不仅A而且B |
> | -(으)ㄹ 뿐더러 | 不仅A而且B (书面) |
> | -(으)ㄴ/는 데다(가) | 在A基础上还B |

---

### 语法2: -(으)ㄹ 텐데 — "估计…不过/应该会…但是" ★★★★

表示**推测某种情况**，然后接转折、担心、建议等。

**构成**：动词/形容词词干 + -(으)ㄹ 텐데

```
지금쯤 도착했을 텐데 왜 연락이 없지?
现在应该到了但为什么没联系？

주말이라 사람이 많을 텐데 미리 예약하세요.
周末人会多的不过提前预约吧。

혼자서 하기 어려울 텐데 도와줄까요?
一个人做应该挺难的要帮忙吗？

피곤할 텐데 좀 쉬세요.
应该累了休息一会儿吧。

돈이 많이 들 텐데 괜찮아요?
应该要花很多钱·没关系吗？

주식 투자가 쉽지 않을 텐데 괜찮겠어요?
股票投资应该不容易·会没事吗？
```

> **-(으)ㄹ 텐데 vs -(으)ㄹ 테니까**：
>
> - -(으)ㄹ 텐데：推测+**转折/担心** → 힘들 텐데 괜찮아요? (会累的不过没事吗?)
> - -(으)ㄹ 테니까：推测/意志→**建议/命令的理由** → 제가 할 테니까 쉬세요 (我来做所以你休息)

---

### 语法3: -는 대신(에) — "代替…/作为…的交换" ★★★★

表示两个情况之间的**替代/交换**关系。

**构成**：

| 词类   | 构成             |
| ------ | ---------------- |
| 动词   | -는 대신(에)     |
| 形容词 | -(으)ㄴ 대신(에) |
| 名词   | 대신(에)         |

#### "作为…的交换/补偿"

```
야근한 대신에 내일 일찍 퇴근해도 돼요.
加了班作为交换明天可以早下班。

집이 작은 대신에 월세가 싸요.
房子小作为交换月租便宜。

맛은 없는 대신에 건강에 좋아요.
味道不好不过对健康好。

가격이 비싼 대신에 품질이 좋아요.
价格贵不过质量好。
```

#### "代替"

```
택시 대신에 버스를 탔어요.
代替出租车坐了公交。

밥 대신에 빵을 먹었어요.
代替米饭吃了面包。

제가 가는 대신에 동생이 갔어요.
代替我去弟弟去了。

현금 대신에 카드로 계산했어요.
代替现金用卡结算了。
```

---

### 语法4: -만 못하다 — "不如…" ★★★

表示A**不如**B，B比A好/强。

**构成**：名词 + 만 못하다 / 动词-(으)ㄴ 것만 못하다

```
올해 경기가 작년만 못해요.
今年经济形势不如去年。

레스토랑 음식이 집밥만 못해요.
餐厅的饭不如家里做的。

아무리 좋은 약도 예방만 못해요.
再好的药也不如预防。

말로 하는 것이 직접 보여 주는 것만 못해요.
用语言说不如直接示范。

지금 상황이 예전만 못해요.
现在的情况不如以前。

여행사 패키지가 자유 여행만 못해요.
旅行社的套餐不如自由行。
```

> **"不如"系列**：
> | 表达 | 例 |
> |------|-----|
> | A만 못하다 | 집밥만 못해요 (不如家饭) |
> | A보다 못하다 | 예전보다 못해요 (比以前差) |
> | A(으)ㄴ 것만 못하다 | 직접 하는 것만 못하다 (不如亲自做) |

## 四、核心会话

### 对话1：物价讨论

```
A: 요즘 장 보러 가면 깜짝 놀라요.
   最近去买菜就吓一跳。

B: 맞아요. 물가가 오르는 데다가
   제품 크기는 줄어들었어요.
   对。物价上涨加上
   产品大小缩小了。

A: 올해 물가가 작년만 못하다고 느끼는 건
   저만 그런 건가요?
   觉得今年物价不如去年的
   是只有我这样吗？

B: 아니요, 다들 그래요.
   월급은 안 오르는 데다가 세금은 올라서
   정말 힘든 지경이에요.
   不是，大家都这样。
   月薪不涨加上税涨了
   真的到了困难的地步。

A: 절약하는 대신에 삶의 질이 떨어지니까
   그것도 답이 아닌 것 같아요.
   节约作为代价生活质量下降
   那也不是答案啊。

B: 투자를 해 볼까 하는데
   주식은 위험할 텐데 어떻게 생각해요?
   想试试投资
   但股票应该有风险你怎么看？
```

### 对话2：储蓄与投资

```
A: 요즘 적금 금리가 너무 낮지 않아요?
   最近定期储蓄利率太低了不是吗？

B: 네, 은행에 넣어 봤자 이자가 별로라
   투자하는 사람이 늘었어요.
   嗯，存银行利息没多少
   投资的人增多了。

A: 그래도 원금이 보장되는 반면에 안전하잖아요.
   但另一方面本金有保障安全嘛。

B: 맞아요. 하지만 인플레이션을 생각하면
   가만히 있는 것이 투자하는 것만 못할 수도 있어요.
   对。但考虑到通胀的话
   什么都不做可能不如投资。

A: 투자 경험이 없으면 어려울 텐데요.
   没有投资经验的话应该挺难的。

B: 처음에는 소액으로 시작하는 대신에
   공부를 많이 하는 게 좋아요.
   一开始小额投资作为代价
   多学习比较好。
```

### 对话3：省钱技巧

```
A: 가계부를 쓰기 시작했는데
   생각보다 지출이 많아서 놀랐어요.
   开始写记账本了
   没想到支出多吓了一跳。

B: 어디에 가장 많이 써요?
   在哪儿花最多？

A: 외식비가 제일 많아요.
   외식하는 게 편한 데다가
   배달 앱이 너무 편리하거든요.
   外食费最多。
   外食方便加上
   外卖APP太方便了。

B: 직접 요리하면 확실히 절약되는 반면에
   시간이 많이 들잖아요.
   自己做饭确实能省不过
   很花时间嘛。

A: 그래서 밀키트를 사 볼 셈이에요.
   所以打算买料理包试试。

B: 좋은 방법이네요. 밀키트가 외식만 못할 줄 알았는데
   요즘 것들은 꽤 맛있더라고요.
   好方法。以为料理包不如外食
   但最近的挺好吃的。
```

**新词**：

- **장 보다** = 买菜/购物
- **원금** = 本金
- **소액** = 小额
- **가계부** = 记账本
- **밀키트** = Meal Kit/料理包

## 五、文化补充：韩国的经济生活

| 项目                | 说明                                       |
| ------------------- | ------------------------------------------ |
| **카드 사회**       | 韩国信用卡使用率极高，很多地方甚至不收现金 |
| **전세/월세**       | 独特的全租(전세)制度：付大额保证金免月租   |

| **영끌**            | 영혼까지 끌어모아 (倾尽一切) 来买房/投资   |
| **가성비 / 가심비** | 性价比/心价比(满足心理的消费)              |
| **주식 열풍**       | 近年股票投资热潮，尤其年轻人               |

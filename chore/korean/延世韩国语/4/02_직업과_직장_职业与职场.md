## 三、核心语法

### 语法1: -(으)ㅁ/음 — 名词化 ★★★★★

将动词/形容词**转化为名词**，比 -기 更书面、更正式。

**构成**：

- 无收音/ㄹ收音：-ㅁ → 살다 → 삶 (生活), 알다 → 앎 (知道)
- 有收音：-음 → 있다 → 있음, 죽다 → 죽음 (死亡)

#### 作为正式书面名词

```
삶 (살다 → 삶)           生活/人生
죽음 (죽다 → 죽음)        死亡
웃음 (웃다 → 웃음)        笑/笑声
울음 (울다 → 울음)        哭/哭声
슬픔 (슬프다 → 슬픔)      悲伤
기쁨 (기쁘다 → 기쁨)      喜悦
아픔 (아프다 → 아픔)      痛苦
느낌 (느끼다 → 느낌)      感觉
믿음 (믿다 → 믿음)        信任/信仰
꿈 (꾸다 → 꿈)            梦
```

#### 用于正式文体/书面语

```
노력함으로써 성공할 수 있다.
通过努力能够成功。

그가 떠남을 알았다.
知道了他离开这件事。

준비가 부족함을 느꼈다.
感到了准备不足。

시간이 없음에도 불구하고 참석했다.
尽管没有时间还是出席了。

그의 성실함이 인정받았다.
他的诚实得到了认可。
```

> **-(으)ㅁ vs -기**：
> | | -(으)ㅁ/음 | -기 |
> |---|-----------|-----|
> | 正式度 | **高 (书面/正式)** | 中 (口语也用) |
> | 搭配 | -에도 불구하고, -을 알다 | -가 쉽다/어렵다, -를 바라다 |
> | 例 | 있음을 알다 (知道存在) | 있기가 어렵다 (难以存在) |

---

### 语法2: -는 한 — "只要…/在…的范围内" ★★★★

表示在**某个条件/范围内**就一定如何。

**构成**：

| 词类   | 构成       |
| ------ | ---------- |
| 动词   | -는 한     |
| 形容词 | -(으)ㄴ 한 |
| 名词   | -인 한     |

```
노력하는 한 성공할 수 있어요.
只要努力就能成功。

건강한 한 계속 일할 거예요.
只要健康就会继续工作。

내가 살아 있는 한 너를 도와줄게.
只要我活着就会帮你。

시간이 허락하는 한 최대한 많이 연습하세요.
在时间允许的范围内尽可能多练习。

가능한 한 빨리 연락해 주세요.
尽可能快地联系我。

제 힘이 닿는 한 도와드리겠습니다.
在我能力所及的范围内帮您。
```

> **常用搭配**：
>
> - **가능한 한** = 尽可能 (= 될 수 있는 한)
> - **할 수 있는 한** = 尽可能

---

### 语法3: -(으)ㄹ 따름이다 — "只是…/只不过…" ★★★

表示**仅仅如此**，强调谦虚或局限。类似"只是…而已"。

**构成**：动词词干 + -(으)ㄹ 따름이다

```
최선을 다할 따름이에요.
只是尽最大努力而已。

저는 시키는 대로 할 따름이에요.
我只是按照吩咐做而已。

감사할 따름이에요.
只有感谢。（只是感激不尽）

지켜볼 따름이에요.
只能看着（束手无策）。

주어진 일을 열심히 할 따름이에요.
只是认真做分内的事而已。
```

> **语感**：相当正式/书面，口语中不太用。多用于正式发言、采访、书信中。

---

### 语法4: -는 법이다 — "应该…/理当…/一般都会…" ★★★★

表示按照**道理/常识/规律**应该如何或一般都会如何。

**构成**：动词词干 + -는 법이다

```
어른 앞에서는 예의를 지키는 법이에요.
在长辈面前应该守礼仪。

좋은 결과를 위해서는 꾸준히 노력하는 법이에요.
为了好结果理当持续努力。

남의 물건을 함부로 만지지 않는 법이에요.
不应该随意碰别人的东西。

회의 중에는 핸드폰을 꺼 놓는 법이에요.
开会时理应关掉手机。

사람은 서로 도우며 사는 법이에요.
人理当互相帮助地生活。
```

> **-는 법이다 vs -기 마련이다**：
>
> - **-는 법이다**：带有**规范/道义**色彩 → "应该这样做"
> - **-기 마련이다**：带有**自然规律**色彩 → "自然会这样"

## 四、核心会话

### 对话1：求职面试准备

```
A: 요즘 취업 준비하고 있다면서요?
   听说最近在准备就业？

B: 네, 이력서를 쓰는 데 생각보다 시간이 많이 걸려요.
   嗯，写简历方面比想象中花时间。

A: 어떤 분야에 지원할 셈이에요?
   打算应聘什么领域？

B: IT 분야요. 개발자로 취직할 셈이에요.
   IT领域。打算作为开发者就业。

A: 요즘 그 분야 경쟁이 심하던데...
   最近那个领域竞争很激烈…

B: 네, 그래도 포기하지 않는 한 기회는 올 거라고
   믿고 있어요.
   嗯，但只要不放弃机会就会来
   我这么相信。
   최선을 다할 따름이에요.
   只是尽最大努力而已。
```

### 对话2：职场生活

```
A: 이직을 생각하고 있어요?
   在考虑跳槽吗？

B: 네, 솔직히 요즘 야근이 많을 뿐더러
   연봉도 마음에 안 들거든요.
   嗯，说实话最近加班多不说
   年薪也不满意。

A: 그래도 지금 회사가 복리후생은 좋잖아요.
   但现在公司福利待遇不是挺好的么。

B: 맞아요. 그래서 고민이에요.
   对。所以很纠结。

A: 직장을 옮기려면 신중해야 하는 법이에요.
   要换工作的话理应慎重。
   급하게 결정하면 후회하기 마련이니까요.
   匆忙决定的话免不了后悔的。

B: 맞는 말이에요. 좀 더 생각해 볼게요.
   说得对。我再想想。
```

### 对话3：面试经验分享

```
A: 면접 잘 봤어요?
   面试顺利吗？

B: 글쎄요... 면접관 앞에서 너무 긴장했어요.
   不好说…面试官面前太紧张了。

A: 처음에는 긴장하기 마련이에요. 어떤 질문을 받았어요?
   一开始免不了紧张。被问了什么问题？

B: "왜 이 회사에 지원했느냐"고 물었는데
   준비가 부족함을 느꼈어요.
   问了"为什么应聘这家公司"
   感到了准备不足。

A: 면접은 많이 볼수록 늘는 법이에요.
   面试越多参加越会提高的。

B: 네, 경험이 쌓이는 데 시간이 걸리겠지만
   포기하지 않을 셈이에요.
   嗯，积累经验方面虽然需要时间
   但打算不放弃。
```

**新词**：

- **신중하다** = 慎重
- **급하다** = 急/匆忙
- **면접관** = 面试官
- **지원하다** = 应聘/申请
- **쌓이다** = 积累

## 五、文化补充：韩国职场文化

| 特征          | 说明                                                                   |
| ------------- | ---------------------------------------------------------------------- |
| **직급 문화** | 韩国公司有严格的职级体系：사원→대리→과장→차장→부장→이사→상무→전무→사장 |
| **호칭**      | 上级一般称"○○님" 或 직급+"님" (과장님, 부장님)                         |
| **야근 문화** | 虽然"워라밸(Work-Life Balance)"意识在增强，加班仍较普遍                |
| **회식**      | 公司聚餐，上司常请客，年轻世代逐渐更注重个人时间                       |
| **명함 교환** | 交换名片时需双手递交，仔细看后放好                                     |
| **존댓말**    | 即使年纪相近，只要职级不同就用敬语                                     |
| **취업 준비** | 스펙(Spec): 学历+资格证+语言成绩+实习经验等的积累                      |

## 六、语法练习

### 练习1：-(으)ㅁ/음

将以下动词/形容词名词化：

1. 슬프다 → **\_\_\_** (悲伤)
2. 기쁘다 → **\_\_\_** (喜悦)
3. 느끼다 → **\_\_\_** (感觉)
4. 살다 → **\_\_\_** (人生/生活)

<details>
<summary>点击查看答案</summary>

1. 슬픔
2. 기쁨
3. 느낌
4. 삶

</details>

### 练习2：-는 한

用"只要…"完成：

1. 只要健康就会继续工作 → ******\_\_\_******
2. 只要不下雨就去 → ******\_\_\_******
3. 尽可能快地来 → ******\_\_\_******

<details>
<summary>点击查看答案</summary>

1. 건강한 한 계속 일할 거예요.
2. 비가 오지 않는 한 갈 거예요.
3. 가능한 한 빨리 올게요.

</details>

### 练习3：-는 법이다 vs -기 마련이다

选择合适的表达：

1. 어른에게는 인사를 하\_\_\_ (理应)
2. 노력하면 성공하\_\_\_ (自然会)
3. 남의 것을 탐내지 않\_\_\_ (理应)

<details>
<summary>点击查看答案</summary>

1. 인사를 하는 법이에요. (-는 법: 规范)
2. 성공하기 마련이에요. (-기 마련: 规律)
3. 탐내지 않는 법이에요. (-는 법: 规范)

</details>

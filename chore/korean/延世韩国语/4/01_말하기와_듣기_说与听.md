## 三、核心语法

### 语法1: -는 데(에) — "在…方面/对于…来说" ★★★★★

表示在某个**方面、过程、事情**上。是第四册最高频语法之一。

**构成**：

| 词类            | 构成           |
| --------------- | -------------- |
| 动词            | -는 데(에)     |
| 형용사 (形容词) | -(으)ㄴ 데(에) |
| 过去            | -(으)ㄴ 데(에) |
| 名词            | -인 데(에)     |

```
한국어를 배우는 데 시간이 많이 걸려요.
学韩语方面需要很多时间。

이 책은 문법을 이해하는 데 도움이 돼요.
这本书对理解语法有帮助。

건강을 유지하는 데 운동이 중요해요.
在保持健康方面运动很重要。

발음이 좋아지는 데 6개월이 걸렸어요.
发音变好花了6个月。

이 일을 하는 데 어려움이 많았어요.
做这件事方面困难很多。

자취하는 데 한 달에 얼마나 들어요?
自己住一个月大概花多少？
```

> ⚠️ **-는 데 vs -는데**：注意空格！
>
> - **-는 데** (名词性)：在…方面 → 공부하는 데 도움이 돼요
> - **-는데** (连接语尾)：但是/…的时候 → 공부하는데 전화가 왔어요

---

### 语法2: -(으)ㄹ 뿐더러 — "不仅…而且" ★★★★

比第三册的 -(으)ㄹ 뿐만 아니라 更书面/正式。

**构成**：动词/形容词词干 + -(으)ㄹ 뿐더러

```
그 사람은 한국어를 잘할 뿐더러 일본어도 잘해요.
那个人不仅韩语好而且日语也好。

이 식당은 맛있을 뿐더러 가격도 싸요.
这家餐厅不仅好吃而且价格也便宜。

비가 올 뿐더러 바람까지 불었어요.
不仅下雨而且还刮风。

그분은 실력이 좋을 뿐더러 성격도 좋아요.
那位不仅实力好而且性格也好。

경험이 부족할 뿐더러 자신감도 없었어요.
不仅经验不足而且也没有自信。
```

> **"不仅…而且"系列对比**：
> | 表达 | 正式度 | 例 |
> |------|--------|-----|
> | -고 | 低 (口语) | 맛있고 싸요 |
> | -(으)ㄹ 뿐만 아니라 | 中 | 맛있을 뿐만 아니라 싸요 |
> | -(으)ㄹ 뿐더러 | **高 (书面)** | 맛있을 뿐더러 싸요 |
> | -(으)ㄹ 뿐 아니라 | 高 | 맛있을 뿐 아니라 싸요 |

---

### 语法3: -는 셈이다 — "算是…/等于是…" ★★★★

表示虽然不完全是，但**大致可以算作/等于**某种情况。

**构成**：

| 词类        | 构成                   |
| ----------- | ---------------------- |
| 动词 (现在) | -는 셈이다             |
| 동사 (过去) | -(으)ㄴ 셈이다         |
| 형용사      | -(으)ㄴ 셈이다         |
| 미래        | -(으)ㄹ 셈이다 → 打算… |

#### "算是/等于"用法

```
매일 2시간씩 공부하니까 꽤 열심히 하는 셈이에요.
每天学2小时·算是很努力了。

한국에서 5년 살았으니까 거의 반 한국 사람인 셈이에요.
在韩国住了5年·算是半个韩国人了。

한 시간에 만 원이면 싼 셈이에요.
一小时一万韩元的话·算是便宜了。

한국어 능력 시험 4급을 따면 중급은 끝낸 셈이에요.
考过TOPIK 4级的话中级算是结束了。
```

#### "打算"用法（-(으)ㄹ 셈이다）

```
올해 안에 TOPIK 시험을 볼 셈이에요.
打算今年内考TOPIK。

졸업 후에 한국에서 취직할 셈이에요.
毕业后打算在韩国就职。

주말에 집에서 쉴 셈이에요.
打算周末在家休息。
```

---

### 语法4: -기 마련이다 — "必然…/免不了…/自然会…" ★★★★

表示按照**常理/规律**，某个结果是必然的、不可避免的。

**构成**：动词/形容词词干 + -기 마련이다

```
노력하면 결과가 좋기 마련이에요.
努力的话结果自然会好。

외국어를 배우면 처음에는 어렵기 마련이에요.
学外语的话一开始必然会难。

세월이 지나면 사람은 변하기 마련이에요.
随着岁月流逝人必然会变。

실수를 하다 보면 실력이 늘기 마련이에요.
犯错犯着犯着实力自然会提高。

처음에는 서투르기 마련이니까 너무 걱정하지 마세요.
一开始免不了生疏所以不用太担心。

사람들이 많으면 의견이 다르기 마련이에요.
人多的话意见必然会不同。
```

> **语感**：像是在说"这是自然规律/道理"，带有安慰或总结的语气。

## 四、核心会话

### 对话1：语言学习方法

```
A: 한국어 공부한 지 얼마나 됐어요?
   学韩语多久了？

B: 한 2년 됐어요. 매일 2시간씩 공부하니까
   꽤 열심히 하는 셈이에요.
   大概2年了。每天学2小时
   算是挺努力的了。

A: 발음도 좋고 말도 유창하시네요.
   发音好说话也流利啊。

B: 처음에는 발음이 어려워서 고생했어요.
   한국어 발음을 익히는 데 한 6개월 걸린 것 같아요.
   一开始发音难吃了不少苦。
   习惯韩语发音方面好像花了6个月左右。

A: 어떻게 연습하셨어요?
   怎么练习的？

B: 한국 드라마를 많이 봤어요.
   듣기 연습이 될 뿐더러 자연스러운 표현도
   배울 수 있거든요.
   看了很多韩剧。
   不仅能练听力而且还能学到自然的表达。

A: 맞아요. 드라마가 도움이 많이 되죠.
   对。电视剧帮助很大。

B: 네, 그리고 처음에 실수를 많이 해도
   꾸준히 하면 실력이 늘기 마련이에요.
   嗯，而且一开始即使犯很多错
   坚持做的话实力必然会提高。
```

### 对话2：有效的沟通

```
A: 발표 준비 잘돼 가고 있어요?
   发表准备得怎么样了？

B: 자료를 준비하는 데 시간이 너무 많이 걸려서
   아직 다 못 했어요.
   准备资料方面花了太多时间
   还没准备完。

A: 발표할 때 뭐가 제일 중요해요?
   发表的时候什么最重要？

B: 듣는 사람의 입장에서 생각하는 게 중요한 것 같아요.
   我觉得站在听众的立场思考比较重要。
   내용이 좋을 뿐더러 전달력도 있어야 해요.
   内容好不说传达力也得有。

A: 그런데 사람들 앞에서 말하면 떨리지 않아요?
   但是在人们面前说话不紧张吗？

B: 당연히 떨리죠. 처음에는 긴장하기 마련이에요.
   当然紧张。一开始免不了紧张。
   그래도 많이 연습하면 자신감이 생길 셈이에요.
   但多练习的话自信心算是会有的。

A: 좋은 조언이네요. 감사합니다.
   好建议。谢谢。
```

### 对话3：方言与标准语

```
A: 한국에 방언이 많다면서요?
   听说韩国方言很多？

B: 네, 지역마다 사투리가 달라요.
   부산 사투리, 전라도 사투리, 제주 사투리 등이 있어요.
   嗯，每个地区方言不同。
   有釜山方言、全罗道方言、济州方言等。

A: 서울 말이 표준어예요?
   首尔话是标准语吗？

B: 표준어의 기반이 된 셈이에요.
   算是成为标准语的基础。
   그런데 서울 사람도 사투리를 쓸 때가 있어요.
   不过首尔人也有用方言的时候。

A: 외국인이 사투리를 배우면 어때요?
   外国人学方言怎么样？

B: 일단 표준어를 배우는 데 집중하는 게 좋아요.
   首先集中在学标准语方面比较好。
   표준어를 잘하면 사투리도 이해하기 마련이에요.
   标准语好的话方言也自然会理解。
```

**新词**：

- **서투르다** = 生疏/不熟练
- **꾸준히** = 持续地/坚持地
- **전달력** = 传达力
- **긴장하다** = 紧张
- **기반** = 基础/基盘

## 五、文化补充：韩语的敬语体系

| 等级         | 名称        | 终结语尾        | 场景                  |
| ------------ | ----------- | --------------- | --------------------- |
| 格式体最尊敬 | 하십시오체  | -습니다/-ㅂ니다 | 新闻、演讲、军队      |
| 非格式体尊敬 | 해요체      | -아/어요        | 日常对话最常用        |
| 格式体一般   | 하오체      | -오/-소         | 古语/文学/偶尔长辈间  |
| 格式体下称   | 하게체      | -네/-게         | 教授对学生/长辈对晚辈 |
| 非格式体一般 | 해체 (반말) | -아/어          | 朋友/晚辈             |
| 格式体最不恭 | 해라체      | -다/-라         | 书面/新闻/小说叙述    |

> **实际使用**：现代韩国社会日常主要用 **해요체** 和 **합니다체**，半语(**반말**)用于亲密关系。

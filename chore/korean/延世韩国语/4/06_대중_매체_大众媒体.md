## 一、学习目标

- 能够用韩语讨论媒体、网络文化和信息素养
- **重点语法**：-(으)ㄹ 뿐이다 (只是…而已)、-는 양/듯/체 (假装/好像…的样子)、-도록 (以便/直到)、-(으)ㄹ 만큼 (达到…的程度/足够)

## 三、核心语法

### 语法1: -(으)ㄹ 뿐이다 — "只是…而已" ★★★★

表示**仅此而已**，限定范围。比 -(으)ㄹ 따름이다 略口语化。

**构成**：动词/形容词词干 + -(으)ㄹ 뿐이다

```
저는 들은 대로 말할 뿐이에요.
我只是照听到的说而已。

아는 것을 설명할 뿐이에요.
只是解释知道的而已。

제 의견을 말할 뿐이에요.
只是说我的意见而已。

사실을 전할 뿐이에요.
只是传达事实而已。

최선을 다할 뿐이에요.
只是尽最大努力而已。
```

#### 与名词/形容词搭配

```
이것은 소문에 불과할 뿐이에요.
这只不过是传闻而已。

그 사람은 유명할 뿐 실력이 없어요.
那个人只是有名而已没有实力。

돈만 많을 뿐 행복하지 않아요.
只是钱多而已并不幸福。
```

> **"只是/仅仅"系列**：
> | 表达 | 正式度 | 例 |
> |------|--------|-----|
> | -(으)ㄹ 뿐이다 | 中-高 | 말할 뿐이에요 |
> | -(으)ㄹ 따름이다 | **高 (最正式)** | 감사할 따름입니다 |
> | -기만 하다 | 低-中 | 보기만 했어요 |

---

### 语法2: -는 양/듯/체 — "假装/好像…的样子" ★★★

表示做出某种**样子/姿态**，常搭配 하다。

#### -는 양 = -는 듯 = -는 체：表面看起来如何

```
아무것도 모르는 양 행동했어요.
装作什么都不知道的样子。

잘 아는 체하지 마세요.
别装作很懂的样子。

관심 없는 듯 행동했지만 사실 궁금했어요.
装作不感兴趣的样子但其实很好奇。

안 들은 체했어요.
假装没听见。

바쁜 양 서류를 뒤적거렸어요.
装出忙的样子翻文件。
```

> **-는 체하다 = -는 척하다**（第三册学过）：
>
> - **-는 체하다**：书面/正式
> - **-는 척하다**：更口语
> - **-는 양하다**：文学/书面
> - **-는 듯하다**：常用于"好像"的意思（不一定假装）

#### -는 듯하다 单独用=好像

```
비가 올 듯해요.              好像要下雨。
이해한 듯한 표정이었어요.      是好像理解了的表情。
알 듯 모를 듯한 내용이었어요.   似懂非懂的内容。
```

---

### 语法3: -도록 — "以便…/直到…/为了让…" ★★★★★

多功能语法，有三种主要用法。

#### 用法1：以便/为了让…（目的）

```
잘 들을 수 있도록 앞에 앉으세요.
为了能听清请坐前面。

다 볼 수 있도록 큰 화면을 사용했어요.
为了能都看到使用了大屏。

오해가 없도록 분명하게 말할게요.
为了不产生误会会说清楚。

늦지 않도록 서두르세요.
为了不迟到请赶紧。
```

#### 用法2：直到…为止

```
밤 12시가 되도록 잠을 못 잤어요.
到晚上12点都没睡着。

입이 닳도록 말했어요.
说到嘴都磨破了（反复说了无数次）。

눈물이 나도록 웃었어요.
笑到流眼泪了。

쓰러지도록 일했어요.
工作到要倒下的程度。
```

#### 用法3：使/让（使动性）

```
선생님이 학생들이 이해하도록 설명했어요.
老师为了让学生们理解进行了说明。

부모님이 걱정하지 않도록 연락했어요.
为了让父母不担心联系了。
```

---

### 语法4: -(으)ㄹ 만큼 — "达到…的程度/足够…" ★★★★

表示达到某种**足够的程度**。

**构成**：动词/形容词词干 + -(으)ㄹ 만큼

```
한국어로 대화할 수 있을 만큼 실력이 늘었어요.
韩语实力提高到了能对话的程度。

혼자 할 수 있을 만큼 경험이 쌓였어요.
经验积累到了能独自做的程度。

먹을 만큼만 주문하세요.
点足够吃的量就行。

입이 찢어질 만큼 웃었어요.
笑到嘴要裂开的程度。

감동받을 만큼 좋은 영화였어요.
是好到令人感动的电影。

참을 만큼 참았어요.
忍到了能忍的极限。
```

> **-(으)ㄹ 만큼 vs -(으)ㄹ 정도로 vs -(으)ㄹ 지경이다**：
> | 表达 | 感情色彩 | 例 |
> |------|---------|-----|
> | -(으)ㄹ 만큼 | **中性（足够）** | 먹을 만큼 먹었어요 |
> | -(으)ㄹ 정도로 | **中性（程度）** | 울 정도로 감동적 |
> | -(으)ㄹ 지경이다 | **负面（极端）** | 미칠 지경이에요 |

## 四、核心会话

### 对话1：假新闻讨论

```
A: 이 기사 봤어요? 충격적인 내용이에요.
   看了这篇报道吗？很震惊的内容。

B: 그건 가짜 뉴스일 뿐이에요.
   那只不过是假新闻而已。
   온라인에서 검증 없이 퍼지는 거예요.
   是在网上没经过验证就扩散的。

A: 어떻게 알아요?
   怎么知道？

B: 출처가 불분명한 데다가
   다른 주요 언론에서 보도한 적이 없거든요.
   出处不明确加上
   其他主要媒体都没有报道过。

A: 요즘 가짜 뉴스가 진짜 뉴스처럼 보이도록
   정교하게 만들어져서 구별하기 어려워요.
   最近假新闻做得精巧到看起来像真新闻
   很难区分。

B: 맞아요. 그래서 비판적으로 정보를
   판단할 수 있을 만큼의 능력이 중요해요.
   对。所以能批判性判断信息
   达到那种程度的能力很重要。
```

### 对话2：网红文化

```
A: 요즘 유튜브에서 인플루언서가 된 사람이 많죠?
   最近在YouTube成为网红的人很多吧？

B: 네, 유명한 양 행동하는 사람도 있고
   진짜 실력 있는 사람도 있어요.
   嗯，装作有名的样子表示的人也有
   真正有实力的人也有。

A: 조회수만 많을 뿐 내용이 없는 영상도 많지 않아요?
   不是也有很多只是播放量多而已没有内容的视频吗？

B: 맞아요. 자극적인 콘텐츠가 인기를 끄는 반면에
   유익한 콘텐츠는 주목받기 어려워요.
   对。刺激性内容受欢迎另一方面
   有益的内容很难受到关注。

A: 그래도 좋은 콘텐츠를 만드는 사람들은
   꾸준히 구독자가 늘기 마련이에요.
   但做好内容的人
   订阅者必然会稳步增加的。

B: 네, 결국 콘텐츠의 질이 중요한 법이에요.
   嗯，最终内容的质量才是应当重视的。
```

### 对话3：手机中毒

```
A: 요즘 스마트폰 사용 시간이
   하루에 8시간이 넘을 정도예요.
   最近智能手机使用时间
   一天超过8小时的程度。

B: 그 정도면 중독이라고 해도 될 만큼이에요.
   到那个程度说是上瘾也足够了。

A: 줄이려고 하는데 쉽지 않아요.
   想减少但不容易。

B: 자기 전에 핸드폰을 안 보도록
   거실에 놓고 자 보세요.
   为了不在睡前看手机
   试试放在客厅睡。

A: 그래 봤는데 결국 다시 가져왔어요.
   试过了但最终又拿回来了。

B: 아무리 의지가 강하더라도
   환경을 안 바꾸면 어려운 법이에요.
   就算意志再强
   不改变环境的话是难的。
   앱 사용 제한을 설정해 놓도록 하세요.
   设置APP使用限制吧。
```

**新词**：

- **출처** = 出处/来源
- **불분명하다** = 不明确
- **정교하다** = 精巧/精密
- **자극적** = 刺激性的
- **유익하다** = 有益的

## 五、文化补充：韩国的媒体文化

| 项目                | 说明                                            |
| ------------------- | ----------------------------------------------- |
| **포털 사이트**     | 네이버(Naver)、다음(Daum) 是主要新闻门户        |
| **실시간 검색어**   | 曾经的"实时搜索词"影响力巨大，现已取消          |
| **댓글 문화**       | 韩国评论文化非常活跃，但악플(恶评)问题严重      |
| **OTT**             | 넷플릭스, 왓챠, 웨이브, 쿠팡플레이 等流媒体平台 |
| **1인 미디어**      | 个人媒体蓬勃发展，많은 사람이 유튜버를 꿈꿈     |
| **미디어 리터러시** | 学校开始教媒体素养教育                          |

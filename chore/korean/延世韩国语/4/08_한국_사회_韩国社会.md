# 제8과: 한국 사회 (韩国社会)

> 延世韩国语 第四册 · 第8课

---

## 一、学习目标

- 能够用韩语讨论韩国社会现象和社会问题
- **重点语法**：-는 이상 (既然…)、-(으)ㄹ 바에(야) (与其…不如)、-(으)ㄴ/는 셈 치다 (就当做…)、-아/어 봤자 (就算…也没用)
- 掌握表达社会观点和立场的高级表达

## 三、核心语法

### 语法1: -는 이상 — "既然…/只要…(就)" ★★★★

表示**既然**某个事实/条件已成立，后面就应该如何。

**构成**：

| 词类   | 构成         |
| ------ | ------------ |
| 动词   | -는 이상     |
| 形容词 | -(으)ㄴ 이상 |
| 过去   | -(으)ㄴ 이상 |
| 名词   | -인 이상     |

```
약속한 이상 지켜야 해요.
既然约好了就应该遵守。

시작한 이상 끝까지 해야 해요.
既然开始了就应该做到底。

한국에 사는 이상 한국어를 배워야 해요.
既然住在韩国就应该学韩语。

학생인 이상 공부가 최우선이에요.
既然是学生学习是第一优先。

결정한 이상 후회하지 마세요.
既然决定了就不要后悔。

이 사회에서 살아가는 이상
사회 문제에 관심을 가져야 해요.
既然在这个社会生活
就应该关注社会问题。
```

> **-는 이상 vs -는 한**：
>
> - **-는 이상**：强调**既定事实/责任** → 약속한 이상 지켜야 해요 (既然约了就得守)
> - **-는 한**：强调**条件范围** → 건강한 한 일할 거예요 (只要健康就工作)

---

### 语法2: -(으)ㄹ 바에(야) — "与其…不如" ★★★★

表示**与其做A（不好的/不值得的），不如做B**。

**构成**：动词词干 + -(으)ㄹ 바에(야)

```
이렇게 고생할 바에 그냥 포기할래요.
与其这么受苦不如放弃算了。

시간 낭비할 바에 차라리 집에서 쉬는 게 나아요.
与其浪费时间还不如在家休息。

후회할 바에야 지금이라도 행동하세요.
与其后悔不如现在就行动。

대충 할 바에 안 하는 게 나아요.
与其马虎做不如不做。

혼자 고민할 바에 다른 사람과 상의하세요.
与其独自烦恼不如和别人商量。

불평만 할 바에 직접 나서서 바꿔 보세요.
与其光抱怨不如亲自出来改变试试。
```

> **常搭配**：-(으)ㄹ 바에(야) + 차라리/그냥 + 后句

---

### 语法3: -(으)ㄴ/는 셈 치다 — "就当做…" ★★★

表示把实际情况**当作/视为**另一种情况来看待。

**构成**：

- 动词 (过去)：-(으)ㄴ 셈 치다 → 잃어버린 셈 치다 (当做丢了)
- 动词 (现在)：-는 셈 치다
- 形容词：-(으)ㄴ 셈 치다

```
돈은 없어진 셈 치세요.
钱就当丢了吧。

그냥 한번 속은 셈 치고 해 보세요.
就当被骗一次试试看吧。

여행 간 셈 치고 맛있는 거 먹자.
就当去了旅行吃好吃的吧。

공부한 셈 치고 책을 한 권 사세요.
就当是学习了买一本书吧。

안 본 셈 치고 넘어갈게요.
就当没看到算了。

운동한 셈 치고 걸어갈까요?
就当运动了走过去怎么样？
```

> **-는 셈이다 vs -는 셈 치다**：
>
> - -는 셈이다：**算是…**（客观判断） → 잘하는 셈이에요 (算是做得好)
> - -는 셈 치다：**就当做是…**（主观假设） → 잘한 셈 치세요 (就当做得好吧)

---

### 语法4: -아/어 봤자 — "就算…也(没用)" ★★★★

表示**即使做了也没有什么好结果/没用**。带有消极、否定的语气。

**构成**：动词词干 + -아/어 봤자

```
불평해 봤자 소용없어요.
就算抱怨也没用。

지금 후회해 봤자 아무 소용없어요.
现在后悔也没有任何用。

가 봤자 사람이 많아서 못 들어갈 거예요.
就算去了人多也进不去。

말해 봤자 안 들을 거예요.
就算说了也不会听的。

아무리 서둘러 봤자 이미 늦었어요.
就算再怎么赶也已经晚了。

혼자 고민해 봤자 답이 안 나와요.
就算一个人烦恼也想不出答案。
```

> **-아/어 봤자 vs -더라도**：
>
> - -아/어 봤자：**没用/消极** → 해 봤자 소용없어요 (做了也没用)
> - -더라도：**让步/意志** → 어려워하더라도 해 볼 거예요 (即使难也要试)

## 四、核心会话

### 对话1：低出生率讨论

```
A: 한국의 출산율이 세계 최저 수준이라면서요?
   听说韩国出生率是世界最低水平？

B: 네, 주거비가 비싼 데다가 교육비도 많이 들어서
   아이를 낳기 어려운 거예요.
   嗯，住房费贵加上教育费也高
   所以很难生孩子。

A: 정부에서 정책을 만들어도
   효과가 없는 것 같아요.
   政府制定政策也
   好像没有效果。

B: 출산 장려금만 줘 봤자
   근본적인 문제가 해결되지 않거든요.
   就算只给生育奖励金也
   根本问题不能解决。

A: 그럼 어떻게 해야 할까요?
   那应该怎么办？

B: 사회 구조를 바꿔야 해요.
   주거 문제는 물론이고 육아와 직장 양립 문제도
   해결해야 해요.
   得改变社会结构。
   住房问题不用说育儿和工作兼顾问题也
   得解决。

A: 일하는 이상 아이를 키우기 어렵다는 건
   정말 심각한 문제예요.
   既然要工作养孩子很难这一点
   真的是很严重的问题。
```

### 对话2：代际差异

```
A: 요즘 세대 갈등이 심해졌다고 하는데
   어떻게 생각해요?
   听说最近代际冲突加深了
   你怎么看？

B: MZ 세대와 기성세대의 가치관이
   많이 다른 것 같아요.
   MZ世代和既成世代的价值观
   好像很不同。

A: 서로 이해하려고 노력해 봤자
   쉽지 않은 것 같아요.
   就算努力互相理解
   好像也不容易。

B: 그래도 같은 사회에서 사는 이상
   소통하려고 노력해야 하지 않겠어요?
   但既然在同一个社会生活
   不是应该努力沟通吗？

A: 맞아요. 서로를 비난할 바에
   차라리 대화를 통해 이해하는 게 나을 것 같아요.
   对。与其互相批评
   不如通过对话理解更好。

B: 좋은 경험이 있는가 하면 나쁜 경험도 있듯이
   어느 세대든 완벽할 수는 없잖아요.
   就像有好经验也有坏经验一样
   哪个世代都不可能完美嘛。
```

### 对话3：多文化社会

```
A: 한국도 다문화 사회가 됐는데
   아직 편견이 많지 않아요?
   韩国也成了多文化社会
   但偏见不是还很多吗？

B: 네, 예전보다 나아졌지만
   차별 문제가 완전히 해결된 건 아니에요.
   嗯，比以前好了但
   歧视问题没有完全解决。

A: 법만 바꿔 봤자 인식이 안 바뀌면 소용없죠.
   就算只改法律意识不变也没用吧。

B: 맞아요. 어릴 때부터 다문화 교육을
   받은 셈 치고 시작해야 해요.
   对。从小得把多文化教育
   当做必须的来开始。

A: 결국 편견을 불평할 바에
   직접 다른 문화를 경험해 보는 게 나을 것 같아요.
   最终与其抱怨偏见
   不如亲自体验不同文化更好。

B: 네, 알면 알수록 편견이 줄어들기 마련이에요.
   嗯，越了解偏见必然会减少。
```

**新词**：

- **출산율** = 出生率
- **장려금** = 奖励金
- **양립** = 兼顾/并立
- **가치관** = 价值观
- **기성세대** = 既成世代/老一辈
- **비난하다** = 批评/指责

## 五、文化补充：韩国当代社会关键词

| 关键词       | 说明                                               |
| ------------ | -------------------------------------------------- |
| **MZ 세대**  | Millennial + Z世代，指1980-2000年代出生的年轻世代  |
| **N포 세대** | 放弃N样东西的一代（恋爱、结婚、生子、就业、住房…） |
| **헬조선**   | "地狱朝鲜"，讽刺韩国年轻人面临的社会压力           |
| **수저론**   | "汤匙论"，金汤匙/银汤匙/泥汤匙论阶层固化           |
| **갑질**     | 以甲方身份对乙方施压/欺凌                          |
| **꼰대**     | 老古板/爱说教的长辈                                |
| **워라밸**   | Work-Life Balance，追求工作与生活平衡              |

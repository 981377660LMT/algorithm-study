## 三、核心语法

### 语法1: -(으)ㄹ 테지만 — "虽然会…但是" ★★★

表示**承认推测的情况**（会如此），但同时提出不同看法或转折。

**构成**：动词/形容词词干 + -(으)ㄹ 테지만

```
AI가 편리할 테지만 윤리적 문제도 생길 거예요.
虽然AI会方便但也会产生伦理问题。

기술이 발전할 테지만 모든 문제를 해결하지는 못할 거예요.
虽然技术会发展但不会解决所有问题。

자동화로 편해질 테지만 일자리가 줄어들 수 있어요.
虽然因自动化会变方便但就业岗位可能减少。

반대하는 사람도 있을 테지만 변화는 불가피해요.
虽然反对的人也会有但变化不可避免。

많은 시간이 걸릴 테지만 결국 해낼 수 있을 거예요.
虽然会花很多时间但最终能做到。
```

> **-(으)ㄹ 테지만 vs -겠지만**：
>
> - -(으)ㄹ 테지만：更确信的推测+转折 → AI가 편리할 테지만…
> - -겠지만：较温和的推测+转折 → 편리하겠지만…
>   两者可互换，但 -(으)ㄹ 테지만 语气更强。

---

### 语法2: -는 가운데 — "在…之中/当中" ★★★★

表示在某种**状况/背景**下。用于书面/正式场合。

**构成**：

| 词类   | 构成           |
| ------ | -------------- |
| 动词   | -는 가운데     |
| 形容词 | -(으)ㄴ 가운데 |
| 名사   | -인 가운데     |

```
기술이 빠르게 발전하는 가운데
사회 변화도 가속화되고 있어요.
在技术快速发展之中
社会变化也在加速。

경기가 어려운 가운데 청년 실업 문제가 심각해요.
在经济困难之中青年失业问题严重。

환경 오염이 심각해지는 가운데
친환경 운동이 확산되고 있어요.
在环境污染严重化之中
环保运动在扩散。

많은 관심 속에서/많은 관심인 가운데
행사가 진행됐어요.
在很多关注之中活动进行了。

논란이 이어지는 가운데 정부가 입장을 발표했어요.
在争论持续之中政府发表了立场。
```

> **语感**：新闻/报道体常用。日常口语中更多用 **-는 중에** 或 **-는 상황에서**。

---

### 语法3: -(으)ㄹ망정 — "宁可…也(不)" ★★★

表示**宁可接受A（虽然不好），也不做B（更不能接受的）**。

**构成**：动词/形容词词干 + -(으)ㄹ망정

```
굶을망정 남의 것을 훔치지 않겠어요.
宁可饿着也不偷别人的东西。

실패할망정 거짓말은 하지 않겠어요.
宁可失败也不说谎。

힘들망정 포기하지 않을 거예요.
宁可辛苦也不放弃。

돈을 못 벌망정 양심에 어긋나는 일은 안 해요.
宁可赚不到钱也不做违背良心的事。

느릴망정 정확하게 해야 해요.
宁可慢也要做准确。

작을망정 내 집이 있는 게 좋아요.
虽然小但有自己的房子好。(宁可小也要有)
```

> **语感**：非常正式/文学化，口语中通常用 **-더라도**：
>
> - 굶을망정 = 굶더라도 (宁可饿 = 即使饿)
> - -(으)ㄹ망정 更强调**底线/原则**

---

### 语法4: -(으)ㄹ 나위 없다 — "再好不过/无可挑剔" ★★★

表示已经达到**最好的程度**，没有比这更好的了。

**构成**：动词/形容词词干 + -(으)ㄹ 나위 없다 (常用 **더할 나위 없다**)

```
날씨가 더할 나위 없이 좋아요.
天气好得无可挑剔。

이 이상 더 좋을 나위가 없어요.
不可能比这更好了。

오늘 발표가 더할 나위 없이 훌륭했어요.
今天的发表好得无以复加。

건강하고 가족이 함께 있으니
더할 나위 없이 행복해요.
健康又和家人在一起
幸福得无可挑剔。

이번 성과는 더할 나위 없는 결과예요.
这次成果是无可挑剔的结果。
```

> **最常用搭配**：**더할 나위 없다** / **더할 나위 없이 + 형용사**
> 已经固定化为一个表达，意思就是"无可挑剔地/再…不过了"。

## 四、核心会话

### 对话1：AI与未来

```
A: AI가 앞으로 우리 생활을 어떻게 바꿀까요?
   AI今后会怎样改变我们的生活？

B: 기술이 발전하는 가운데
   AI가 많은 직업을 대체할 거라는 전망이 있어요.
   在技术发展之中
   有预测说AI会替代很多职业。

A: 편리해질 테지만 일자리 걱정이 되네요.
   虽然会变方便但担心就业。

B: 맞아요. 단순 반복 업무는 AI가
   더할 나위 없이 효율적으로 처리할 수 있어요.
   对。简单重复的业务AI
   能无可挑剔地高效处理。

A: 그래도 창의적인 일은 사람만이 할 수 있지 않을까요?
   但创意性的工作不是只有人能做吗？

B: 그렇게 생각하고 싶지만
   AI가 예술 작품까지 만드는 가운데
   장담하기 어려워요.
   虽然想那样想
   但在AI连艺术作品都做的情况下
   很难保证。

A: 기술을 두려워할 바에
   적극적으로 활용하는 게 나을 것 같아요.
   与其害怕技术
   不如积极利用更好。
```

### 对话2：环境问题

```
A: 기후 변화 문제가 갈수록 심각해지고 있어요.
   气候变化问题越来越严重了。

B: 네, 탄소 배출이 줄지 않는 가운데
   지구 온도가 계속 오르고 있어요.
   嗯，在碳排放不减的情况下
   地球温度继续上升。

A: 개인이 노력해 봤자 소용없는 거 아니에요?
   个人努力也没用不是吗？

B: 그렇게 생각할 수도 있을 테지만
   작은 실천이 모이면 큰 변화가 되기 마련이에요.
   虽然可能那样想
   但小小的实践积累起来必然会成为大变化。

A: 맞아요. 불편할망정
   일회용품 사용을 줄여야 해요.
   对。宁可不方便
   也得减少一次性用品使用。

B: 지구에 사는 이상
   환경 보호는 선택이 아니라 의무예요.
   既然住在地球
   环保不是选择而是义务。
```

### 对话3：未来畅想

```
A: 20년 후의 세상은 어떨까요?
   20年后的世界会怎样？

B: 자율주행차가 보편화되는 것은 물론이고
   우주 여행도 가능해질 거예요.
   自动驾驶汽车普及化不用说
   太空旅行也会成为可能。

A: 상상만 해도 기대돼요.
   光是想象就期待。

B: 기술이 편리해지는 반면에
   개인 정보 보호 문제는 더 심각해질 텐데요.
   技术变方便的另一方面
   个人信息保护问题会更严重吧。

A: 결국 기술 발전과 윤리 사이에서
   균형을 찾는 게 중요하겠네요.
   最终在技术发展和伦理之间
   找到平衡很重要。

B: 네, 기술이 발달하면 할수록
   인간다움의 가치가 더 중요해질 거예요.
   嗯，技术越发达
   人性的价值会越重要。
```

**新词**：

- **불가피하다** = 不可避免
- **가속화** = 加速化
- **장담하다** = 保证/打包票
- **보편화** = 普遍化/普及
- **인간다움** = 人性/人的本质

## 六、语法练习

### 练习1：-(으)ㄹ 테지만

用"虽然会…但"：

1. 虽然会很难但不会放弃 → ******\_\_\_******
2. 虽然技术会发展但问题也会有 → ******\_\_\_******

<details>
<summary>点击查看答案</summary>

1. 어려울 테지만 포기하지 않을 거예요.
2. 기술이 발전할 테지만 문제도 생길 거예요.

</details>

### 练习2：-(으)ㄹ망정

用"宁可…也不"：

1. 宁可失败也不说谎 → ******\_\_\_******
2. 宁可少赚也要做正确的事 → ******\_\_\_******

<details>
<summary>점击查看답案</summary>

1. 실패할망정 거짓말은 하지 않겠어요.
2. 적게 벌망정 올바른 일을 할 거예요.

</details>

### 练习3：更할 나위 없다

用"再好不过/无可挑剔"：

1. 天气好得无可挑剔 → ******\_\_\_******
2. 今天的表演好得无以复加 → ******\_\_\_******

<details>
<summary>点击查看答案</summary>

1. 날씨가 더할 나위 없이 좋아요.
2. 오늘 공연이 더할 나위 없이 훌륭했어요.

</details>

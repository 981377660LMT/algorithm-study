## 一、学习目标

- 能够用韩语讨论传统文化的保存与变化

## 三、核心语法

### 语法1: -(으)ㄴ가 하면 — "一方面…另一方面" ★★★★

表示两种**不同或对立的情况并存**。"有的这样，有的那样"。

**构成**：动词/形容词 + -(으)ㄴ가 하면

```
전통을 중요하게 생각하는 사람이 있는가 하면
현대화를 더 중시하는 사람도 있어요.
一方面有重视传统的人
另一方面也有更重视现代化的人。

한복을 좋아하는 사람이 있는가 하면
불편하다고 싫어하는 사람도 있어요.
有喜欢韩服的人也有说不方便不喜欢的人。

경제가 좋아지는가 하면 다시 나빠지기도 해요.
经济好一阵又变差。

주중에는 바쁜가 하면 주말에는 한가해요.
平日忙另一方面周末闲。

비가 오는가 하면 갑자기 해가 나기도 해요.
下着雨一会儿突然又出太阳。
```

> **语感**：A和B对比并存，不做价值判断，较客观。

---

### 语法2: -는 것은 물론(이고) — "…不用说/当然…而且还" ★★★★

表示**理所当然**包含A，并且还有B。

**构成**：

- 动词：-는 것은 물론(이고)
- 形容词：-(으)ㄴ 것은 물론(이고)
- 名词：은/는 물론(이고)

```
한국어는 물론이고 일본어도 잘해요.
韩语不用说日语也好。

문법은 물론이고 발음도 중요해요.
语法不用说发音也重要。

전통 음식은 물론이고 현대 퓨전 요리도 인기가 있어요.
传统食物不用说现代融合料理也受欢迎。

국내 관광객은 물론이고 외국인 관광객도 많이 방문해요.
国内游客不用说外国游客也大量访问。

맛있는 것은 물론이고 건강에도 좋아요.
好吃不用说对健康也好。

어른은 물론이고 아이들도 좋아해요.
大人不用说孩子们也喜欢。
```

---

### 语法3: -았/었더니 심화 — "做了…结果/发现" ★★★★

第三册学过基础，这里深入到更多用法。

#### 用法1：自己做了…→ 发现结果

```
전통 시장에 가 봤더니 생각보다 물건이 다양했어요.
去了传统市场发现东西比想象中多样。

한옥에서 하룻밤 묵었더니 정말 편안했어요.
在韩屋住了一晚发现真的很舒适。

열심히 연습했더니 실력이 많이 늘었어요.
认真练习了结果实力进步很多。
```

#### 用法2：因为我做了…→ 对方/其他事物的反应

```
한복을 입었더니 사람들이 사진을 찍어 달라고 했어요.
穿了韩服结果人们说帮拍照。

전통 음식을 만들었더니 외국인 친구들이 좋아했어요.
做了传统食物结果外国朋友们很喜欢。

한국 역사를 공부했더니 드라마가 더 재미있어요.
学了韩国历史结果电视剧更有趣了。
```

#### 用法3：对方做了…→ 我观察到的结果

```
아이가 한옥에 가 봤더니 신기해했어요.
孩子去了韩屋觉得新奇。

친구가 제사를 경험했더니 한국 문화를 더 이해하게 됐어요.
朋友体验了祭祀后更理解韩国文化了。
```

---

### 语法4: -(으)면 -(으)ㄹ수록 — "越…越…" ★★★★★

表示程度**随前面条件的增加而增加**。

**构成**：动词/形容词词干 + -(으)면 -(으)ㄹ수록

```
배우면 배울수록 재미있어요.
越学越有趣。

알면 알수록 한국 문화가 깊어요.
越了解韩国文化越深。

하면 할수록 어려워요.
越做越难。

사람은 나이가 들면 들수록 지혜로워져요.
人越上年纪越有智慧。

생각하면 할수록 복잡해져요.
越想越复杂。

전통은 지키면 지킬수록 가치가 높아져요.
传统越守护价值越高。
```

> **简略形**：可省略 -(으)면，直接用 **-(으)ㄹ수록**：
>
> ```
> 갈수록 더워요.     越来越热。
> 갈수록 어려워요.    越来越难。
> 볼수록 예뻐요.     越看越漂亮。
> ```

## 四、核心会话

### 对话1：传统文化保存

```
A: 최근에 한옥 마을에 다녀왔어요.
   最近去了韩屋村。

B: 어땠어요? 볼 만했어요?
   怎么样？值得看吗？

A: 네, 전통 한옥은 물론이고 체험 프로그램도
   다양해서 좋았어요.
   嗯，传统韩屋不用说体验项目也
   多样很好。

B: 한복 입어 봤어요?
   试穿了韩服吗？

A: 네, 한복을 입었더니 기분이 달라졌어요.
   嗯，穿了韩服心情就不同了。
   직접 입어 보면 볼수록 한복의 아름다움을
   느끼게 돼요.
   亲自穿着越看越能
   感受到韩服的美丽。

B: 전통을 잘 보존하는 곳이 있는가 하면
   관광지화돼서 본래 모습을 잃은 곳도 있잖아요.
   一方面有好好保存传统的地方
   另一方面也有因为旅游化失去原来面貌的地方。

A: 맞아요. 개발과 보존의 균형이 중요한 것 같아요.
   对。开发和保存的平衡好像很重要。
```

### 对话2：传统与现代融合

```
A: 요즘 한복을 현대적으로 바꾼 디자인이
   인기라면서요?
   听说最近把韩服变现代设计的很受欢迎？

B: 네, 전통적인 한복을 좋아하는 사람이 있는가 하면
   현대 한복을 선호하는 사람도 많아요.
   嗯，喜欢传统韩服的人有
   另一方面偏好现代韩服的人也多。

A: 전통과 현대를 융합하는 건 어떻게 생각해요?
   你觉得融合传统和现代怎么样？

B: 전통의 가치를 지키는 것은 물론이고
   현대에 맞게 변화하는 것도 필요하다고 봐요.
   守护传统的价值不用说
   适应现代需要变化我也这么认为。

A: 일본도 기모노를 현대화했더니
   젊은 층에서 인기가 많아졌대요.
   日本也把和服现代化了结果
   在年轻群体中受欢迎了。

B: 결국 전통은 알면 알수록
   현대에도 충분히 매력적인 것 같아요.
   最终传统越了解
   在现代也是足够有魅力的。
```

### 对话3：传统节日

```
A: 한국 명절에 대해 알려 주세요.
   请告诉我韩国传统节日。

B: 대표적으로 설날과 추석이 있어요.
   代表性的有春节和中秋。
   설날에는 떡국을 먹는 것은 물론이고
   세배도 해요.
   春节吃年糕汤不用说
   还要磕头拜年。

A: 세배는 어른에게 하는 거죠?
   拜年是向长辈做的吧？

B: 네, 어른에게 세배하면 할수록
   새해 복이 많이 온다고 해요.
   嗯，据说越向长辈拜年
   新年福气会越多。

A: 요즘 젊은 사람들도 제사를 지내요?
   最近年轻人也做祭祀吗？

B: 간소화하는 사람이 있는가 하면
   전통 방식 그대로 지키는 집도 있어요.
   简化的人有
   另一方面也有按传统方式原样守护的家庭。
   하면 할수록 의미를 알게 되지만
   부담스러워하는 사람도 많아요.
   越做越了解意义
   但觉得有负担的人也很多。
```

**新词**：

- **관광지화** = 旅游化
- **균형** = 平衡
- **융합** = 融合
- **매력적** = 有魅力的
- **간소화하다** = 简化
- **부담스럽다** = 有负担/感到压力

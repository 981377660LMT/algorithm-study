# 表达意图·愿望·计划 (의도·소망·계획 표현)

---

## 一、-(으)ㄹ게요 (我会…/我来…) ★★★★★

说话者的承诺/意志（承诺给对方，需有听者）

仅用于**第一人称**

| 例句                  | 翻译               |
| --------------------- | ------------------ |
| 제가 할**게요**.      | 我来做。           |
| 내일 전화할**게요**.  | 明天我给你打电话。 |
| 다시는 안 할**게요**. | 我不会再做了。     |
| 먼저 갈**게요**.      | 我先走了。         |
| 잘 생각해 볼**게요**. | 我好好想想。       |

---

## 二、-겠- (意志/推测) ★★★★★

### (1) 意志（第一人称）

| 例句                        | 翻译           |
| --------------------------- | -------------- |
| 제가 하**겠**습니다.        | 我来做(正式)。 |
| 열심히 공부하**겠**습니다.  | 我会努力学习。 |
| 다시는 늦지 않**겠**습니다. | 不会再迟到了。 |

### (2) 推测（第二三人称）→ 见23课

| 例句            | 翻译           |
| --------------- | -------------- |
| 맛있**겠**다!   | 看起来好吃！   |
| 힘들**겠**어요. | 应该很辛苦吧。 |

### -(으)ㄹ게요 vs -겠-

| -(으)ㄹ게요          | -겠-                          |
| -------------------- | ----------------------------- |
| 对听者承诺           | 自己决心/正式宣言             |
| 口语                 | 正式                          |
| 제가 할게요 (我来做) | 제가 하겠습니다 (我来做·正式) |
| 只用于第一人称       | 一人称=意志，二三人称=推测    |

---

## 三、-(으)ㄹ래요 (我要/你要吗) ★★★★★

口语化的意愿/询问对方意愿

| 例句                    | 翻译         |
| ----------------------- | ------------ |
| 저는 이거 먹을**래요**. | 我要吃这个。 |
| 같이 갈**래요**?        | 一起去吗？   |
| 뭐 마실**래요**?        | 想喝什么？   |
| 집에 갈**래요**.        | 我要回家。   |

| -(으)ㄹ래요     | -(으)ㄹ게요        |
| --------------- | ------------------ |
| 自己意愿(我想)  | 对对方承诺(我会)   |
| 可问对方(你要?) | 不问对方           |
| 뭐 먹을래요? ✓  | ~~뭐 먹을게요?~~ ✗ |

---

## 四、-(으)려고 하다 (打算) ★★★★★

表示计划/打算/将要做

| 例句                            | 翻译                     |
| ------------------------------- | ------------------------ |
| 한국에 가**려고 해요**.         | 打算去韩国。             |
| 직장을 옮기**려고 한다**.       | 打算换工作。             |
| 일찍 자**려고 했는데** 못 잤다. | 本来打算早睡，但没睡成。 |
| 뭐 하**려고 해요**?             | 打算做什么？             |

### -(으)려고 했다 (本来打算，但没实现)

| 예문                               | 翻译                 |
| ---------------------------------- | -------------------- |
| 일찍 출발하**려고 했는데** 늦었다. | 本打算早出发但晚了。 |
| 전화하**려고 했는데** 잊었다.      | 本打算打电话但忘了。 |

## 五、-고 싶다 (想做) ★★★★★

| 例句                        | 翻译           |
| --------------------------- | -------------- |
| 여행 가**고 싶다**.         | 想去旅行。     |
| 한국어를 잘하**고 싶어요**. | 想说好韩语。   |
| 만나**고 싶었어요**.        | 一直想见(你)。 |

### 否定

| 形式          | 例句                    |
| ------------- | ----------------------- |
| -고 싶지 않다 | 가고 싶지 않다 (不想去) |
| 안 -고 싶다   | ✗ 不能用                |

### 第三人称: -고 싶어하다

| 例句                         | 翻译             |
| ---------------------------- | ---------------- |
| 그 아이가 놀**고 싶어한다**. | 那个孩子想玩。   |
| 친구가 보**고 싶어했다**.    | 朋友想见(我)了。 |

## 六、-(으)ㄹ 것이다 (将要) ★★★★★

### (1) 计划/意志（第一人称）

| 例句                      | 翻译         |
| ------------------------- | ------------ |
| 내년에 졸업할 **거예요**. | 明年要毕业。 |
| 열심히 할 **겁니다**.     | 会努力的。   |

### (2) 推测（第三人称）→ 见23课

| 例句                | 翻译       |
| ------------------- | ---------- |
| 비가 올 **거예요**. | 会下雨吧。 |

---

## 七、-아/어야겠다 (得…/应该…) ★★★★

表示经过判断后的意志决心

| 例句                      | 翻译           |
| ------------------------- | -------------- |
| 이제 가**야겠다**.        | 得走了。       |
| 공부를 더 해**야겠어요**. | 得再多学习。   |
| 병원에 가 봐**야겠다**.   | 得去看看医院。 |
| 살을 빼**야겠다**.        | 得减肥了。     |

---

## 八、-(으)ㄹ까 하다 (想…/在考虑…) ★★★★

犹豫中的打算（没有完全确定）

| 例句                  | 翻译           |
| --------------------- | -------------- |
| 여행을 갈**까 해요**. | 在想着去旅行。 |
| 이직할**까 합니다**.  | 在考虑换工作。 |
| 뭘 공부할**까 해요**? | 在想学什么？   |

---

## 九、-(으)면 좋겠다 (希望…就好了) ★★★★

| 例句                           | 翻译             |
| ------------------------------ | ---------------- |
| 내일 날씨가 좋**으면 좋겠다**. | 希望明天天气好。 |
| 빨리 끝나**면 좋겠다**.        | 希望快点结束。   |
| 합격하**면 좋겠어요**.         | 希望能合格。     |
| 눈이 오**면 좋겠다**.          | 希望下雪。       |

### -(으)면 하다 (比较温和的愿望)

| 예문                        | 翻译           |
| --------------------------- | -------------- |
| 같이 갈 수 있**으면 해요**. | 希望能一起去。 |

---

## 十、-았/었으면 좋겠다 (真希望…了就好了) ★★★★

对现在/未来的愿望（用过去时表示更强烈的渴望）

| 例句                             | 翻译                 |
| -------------------------------- | -------------------- |
| 시험이 빨리 끝**났으면 좋겠다**. | 真希望考试快点结束。 |
| 키가 좀 더 컸**으면 좋겠다**.    | 真希望再高一点。     |
| 비가 안 왔**으면 좋겠다**.       | 真希望不要下雨。     |

---

## 十一、-기(를) 바라다 (希望/祝愿) ★★★

正式/书面的愿望表达

| 例句                               | 翻译               |
| ---------------------------------- | ------------------ |
| 행복하**기를 바랍니다**.           | 祝您幸福。         |
| 건강하시**기 바랍니다**.           | 祝您健康。         |
| 많은 참여 부탁드리**기 바랍니다**. | 期望大家积极参与。 |

---

## 十二、总览对比

| 表达           | 核心意思   | 强度  | 场景       |
| -------------- | ---------- | ----- | ---------- |
| -고 싶다       | 想做       | 中    | 通用       |
| -(으)ㄹ래요    | 我要/你要? | 轻→中 | 口语       |
| -(으)ㄹ게요    | 我会(承诺) | 强    | 口语       |
| -겠-           | 决心/意志  | 最强  | 正式       |
| -(으)려고 하다 | 打算       | 中    | 通用       |
| -(으)ㄹ 것이다 | 将要/计划  | 中→强 | 通用       |
| -아/어야겠다   | 得.../应该 | 强    | 判断后决心 |
| -(으)ㄹ까 하다 | 在考虑     | 弱    | 犹豫       |
| -(으)면 좋겠다 | 希望       | 中    | 愿望       |
| -기 바라다     | 祝愿/希望  | 正式  | 书面       |

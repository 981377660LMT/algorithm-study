# 名词化与转成

> 让动词/形容词变成名词的方法

---

## 一、-기 ★★★★★

### 接续

词干 + -기

### 意思

将动作/状态名词化（更偏口语，常用于固定搭配）

### 基本用法

| 例句                        | 翻译           |
| --------------------------- | -------------- |
| 한국어 배우**기**가 어렵다. | 学韩语很难。   |
| 여행 가**기**로 했다.       | 决定去旅行了。 |
| 요리하**기**를 좋아한다.    | 喜欢做菜。     |

### 高频固定搭配 ★★★★★

| 搭配           | 意思          | 例句                             |
| -------------- | ------------- | -------------------------------- |
| -기로 하다     | 决定做        | 유학 가**기로 했다**.            |
| -기 전에       | 之前          | 자**기 전에** 양치해요.          |
| -기 시작하다   | 开始做        | 비가 오**기 시작했다**.          |
| -기 때문에     | 因为          | 바쁘**기 때문에** 못 갔다.       |
| -기 위해(서)   | 为了          | 합격하**기 위해** 열심히 했다.   |
| -기 바라다     | 希望          | 건강하시**기 바랍니다**.         |
| -기(가) 쉽다   | 容易做        | 사용하**기 쉽다**.               |
| -기(가) 어렵다 | 难做          | 이해하**기 어렵다**.             |
| -기(가) 좋다   | 适合做        | 살**기 좋은** 도시               |
| -기(가) 나쁘다 | 不适合做      | 걷**기 나쁜** 길                 |
| -기(가) 힘들다 | 做起来辛苦    | 혼자 하**기 힘들다**.            |
| -기는 하다     | 是做了(但...) | 먹**기는** 먹었다. (吃是吃了)    |
| -기만 하다     | 只做          | 듣**기만** 한다.                 |
| -기도 하다     | 也做          | 울**기도** 하고 웃**기도** 한다. |
| -기에          | 因为(书面)    | 시간이 없**기에** 미루었다.      |
| -기는커녕      | 别说做        | 먹**기는커녕** 볼 시간도 없다.   |

## 二、-(으)ㅁ ★★★

### 接续

- 有收音 + -음 / 无收音 + -ㅁ

### 意思

将动作/状态名词化（更偏书面）

| 例句                                        | 翻译                             |
| ------------------------------------------- | -------------------------------- |
| 그가 범인**임**이 밝혀졌다.                 | 他是犯人的事实被查明了。         |
| 노력해도 안 된**다**는 사실에 실망했**음**. | 对努力了也不行这一事实感到失望。 |
| 밝**음**과 어두**움**                       | 光明和黑暗                       |
| 기쁨, 슬픔, 아픔                            | 喜悦、悲伤、疼痛                 |
| 죽**음**, 삶                                | 死亡、生活                       |

### -ㅁ 已固化为名词的词

| 原形   | 名词化 | 翻译     |
| ------ | ------ | -------- |
| 기쁘다 | 기쁨   | 喜悦     |
| 슬프다 | 슬픔   | 悲伤     |
| 아프다 | 아픔   | 疼痛     |
| 높다   | 높음   | 高(程度) |
| 젊다   | 젊음   | 年轻     |
| 늙다   | 늙음   | 衰老     |
| 죽다   | 죽음   | 死亡     |
| 울다   | 울음   | 哭声     |
| 웃다   | 웃음   | 笑声     |
| 꿈꾸다 | 꿈     | 梦       |

---

## 三、-기 vs -(으)ㅁ 区别 ★★★

|           | -기           | -(으)ㅁ         |
| --------- | ------------- | --------------- |
| 语境      | 口语 + 书面   | 纯书面          |
| 抽象度    | 过程/动作本身 | 完成的事实/结果 |
| 固定搭配  | 极多          | 少              |
| TOPIK写作 | ✓             | ✓ (更高级)      |

### 对比例句

| -기                                 | -(으)ㅁ                                      |
| ----------------------------------- | -------------------------------------------- |
| 운동하**기**가 좋다. (运动这件事好) | 운동의 중요**함**을 알다. (知道运动的重要性) |
| 배우**기** 어렵다. (学起来难)       | 배**움**에는 끝이 없다. (学习没有尽头)       |

---

## 四、-는 것 ★★★★★

### 接续

动词 -는 것 / 形容词 -(으)ㄴ 것 / 过去 -(으)ㄴ 것

### 意思

...的事/东西（最常用的名词化方式）

| 例句                           | 翻译               |
| ------------------------------ | ------------------ |
| 먹**는 것**을 좋아한다.        | 喜欢吃(的事)。     |
| 중요한 **것**은 노력이다.      | 重要的是努力。     |
| 내가 한 **것**이 맞다.         | 我做的是对的。     |
| 공부하**는 것**이 재미있다.    | 学习是有趣的。     |
| 운동하**는 것**보다 쉬고 싶다. | 比起运动更想休息。 |

### 缩写 것 → 거 (口语) ★

| 书面      | 口语                |
| --------- | ------------------- |
| 먹는 것   | 먹는 거             |
| 좋은 것   | 좋은 거             |
| 할 것이다 | 할 거다 / 할 거예요 |
| 먹을 것   | 먹을 거             |

---

## 五、-기 vs -는 것 区别 ★★★

|          | -기                         | -는 것                   |
| -------- | --------------------------- | ------------------------ |
| 功能     | 固定搭配中                  | 一般名词化               |
| 例       | 배우**기** 어렵다           | 배우**는 것**이 어렵다   |
| 例       | 하**기**로 했다 (✓)         | 하**는 것**으로 했다 (✗) |
| 固定搭配 | -기 시작하다, -기 바라다 등 | 없음                     |

> **固定搭配用 -기，一般描述用 -는 것**

## 六、其他名词化/转成方式

### -(으)ㄹ 것 — 将要做的事

| 例句                   | 翻译               |
| ---------------------- | ------------------ |
| 내일 **할 것**이 많다. | 明天要做的事很多。 |
| **먹을 것** 좀 사 와.  | 买点吃的来。       |

### -이 / -히 / -게 — 副词化

| 形容词        | 副词                    | 例子            |
| ------------- | ----------------------- | --------------- |
| 깨끗하다      | 깨끗**이**/깨끗**하게** | 깨끗이 청소하다 |
| 조용하다      | 조용**히**/조용**하게** | 조용히 하세요   |
| 빠르다        | 빠르**게**              | 빠르게 달리다   |
| 크다          | 크**게**                | 크게 말하다     |
| 높다          | 높**이**                | 높이 날다       |
| 같다          | 같**이**                | 같이 가다       |
| 일찍 → 일찍이 | —                       | 부사            |

### -이/히/리/기 — 被动/使动接尾词 (详见文件16, 17)

## 七、名词化在TOPIK写作中的应用

### 话题句常用名词化 ★★★★

| 模式                 | 例句                                   |
| -------------------- | -------------------------------------- |
| -는 것이 중요하다    | 규칙적으로 운동하**는 것이 중요하다**. |
| -는 것이 필요하다    | 외국어를 배우**는 것이 필요하다**.     |
| -(으)ㅁ을 알 수 있다 | 경제가 성장하고 있**음을 알 수 있다**. |
| -기 위해서는         | 건강해지**기 위해서는** 노력해야 한다. |
| -기 때문이다         | 인구가 감소하**기 때문이다**.          |
| -기가 쉽지 않다      | 혼자서 하**기가 쉽지 않다**.           |

# 易混淆语法辨析 (헷갈리기 쉬운 문법 비교)

---

## 一、-아/어서 vs -(으)니까 ★★★★★

|          | -아/어서                        | -(으)니까                     |
| -------- | ------------------------------- | ----------------------------- |
| 语感     | 客观自然                        | 主观强调                      |
| 过去时   | ✗ 前句不加过去                  | ✓ 前句可加过去                |
| 命令共动 | ✗ 后句不可                      | ✓ 后句可以                    |
| 例       | 비가 와서 못 갔다               | 비가 오니까 우산 가져가       |
| 例       | ~~비가 왔어서~~ ✗               | 비가 왔으니까 ✓               |
| 例       | ~~배가 아파서 병원에 가세요~~ ✗ | 배가 아프니까 병원에 가세요 ✓ |

---

## 二、-(으)면 vs -다면 ★★★★

|        | -(으)면                                   | -다면                                            |
| ------ | ----------------------------------------- | ------------------------------------------------ |
| 可能性 | 一般/现实                                 | 假设/虚拟                                        |
| 例     | 시간이 있**으면** 가겠다 (如果有时间就去) | 내가 새라**면** 날아갈 텐데 (如果我是鸟就飞走了) |
| 适用   | 现实可能的条件                            | 不太可能的假设                                   |

---

## 三、-는 것 같다 vs -나 보다 ★★★★★

|          | -것 같다                     | -나 보다                    |
| -------- | ---------------------------- | --------------------------- |
| 依据     | 自己判断(直接/间接)          | 间接证据                    |
| 语感     | 一般推测                     | "看来是…"                   |
| 自己感觉 | ✓ 피곤한 것 같다             | ✗ (不用于自己)              |
| 例       | 비가 올 것 같다 (好像会下雨) | 비가 오나 봐요 (看来在下雨) |

---

## 四、-겠- vs -(으)ㄹ 것이다 ★★★★★

|        | -겠-                          | -(으)ㄹ 것이다           |
| ------ | ----------------------------- | ------------------------ |
| 一人称 | 意志(正式)                    | 计划                     |
| 三人称 | 即时推测(感叹)                | 有根据的推测             |
| 例     | 맛있겠다! (看起来好吃!)       | 맛있을 거예요 (应该好吃) |
| 例     | 제가 하겠습니다 (我来做·正式) | 제가 할 거예요 (我会做)  |

---

## 五、-(으)ㄹ게요 vs -(으)ㄹ래요 vs -겠- ★★★★★

|        | -(으)ㄹ게요 | -(으)ㄹ래요  | -겠습니다   |
| ------ | ----------- | ------------ | ----------- |
| 意思   | 对对方承诺  | 自己意愿     | 正式意志    |
| 人称   | 1人称       | 1·2人称      | 1人称(意志) |
| 问对方 | ✗           | ✓ 뭐 먹을래? | ✗           |
| 语感   | 考虑对方    | 自我导向     | 庄重        |

---

## 六、-고 vs -아/어서 (顺序) ★★★★

|      | -고                                  | -아/어서                                 |
| ---- | ------------------------------------ | ---------------------------------------- |
| 关联 | 单纯并列/顺序                        | 因果/紧密关联                            |
| 主语 | 可不同                               | 一般相同                                 |
| 例   | 손을 씻고 밥을 먹었다 (洗手然后吃饭) | 시장에 가서 과일을 샀다 (去市场买了水果) |
| 要点 | 两件事独立                           | 第一件是第二件的前提                     |

---

## 七、-(으)로 vs -에 (方向) ★★★★

|      | -(으)로                        | -에                      |
| ---- | ------------------------------ | ------------------------ |
| 意思 | 方向(向着)                     | 目的地(到达)             |
| 例   | 서울**로** 가다 (向首尔方向去) | 서울**에** 가다 (去首尔) |
| 侧重 | 路径/方向                      | 到达点                   |

---

## 八、-은/는 vs -이/가 ★★★★★

|        | 은/는                                         | 이/가                                             |
| ------ | --------------------------------------------- | ------------------------------------------------- |
| 功能   | 主题(对比/已知)                               | 主格(新信息/焦点)                                 |
| 例     | 저**는** 학생이에요 (我呢是学生)              | 누가 왔어요? 민수**가** 왔어요 (谁来了？民秀来了) |
| 对比   | 나**는** 좋아하는데 너**는**? (我喜欢但你呢?) | (无对比功能)                                      |
| 存在句 | ✗ ~~책은 있다~~ (不自然)                      | ✓ 책**이** 있다                                   |

---

## 九、-에서 vs -에 ★★★★★

|      | -에서                              | -에                       |
| ---- | ---------------------------------- | ------------------------- |
| 动态 | ✓ 做事的场所                       | ✗ 存在/方向               |
| 例   | 학교**에서** 공부하다 (在学校学习) | 학교**에** 있다 (在学校)  |
| 例   | 집**에서** 요리하다 (在家做饭)     | 집**에** 가다 (去家/回家) |

---

## 十、-아/어 있다 vs -고 있다 ★★★★

|      | -아/어 있다             | -고 있다                  |
| ---- | ----------------------- | ------------------------- |
| 意思 | 结果状态                | 进行中                    |
| 用于 | 自动词                  | 他动词                    |
| 例   | 앉아 있다 (坐着)        | 밥을 먹고 있다 (正在吃)   |
| 例   | 문이 열려 있다 (门开着) | 문을 열고 있다 (正在开门) |

⚠️ 穿衣类特殊：입고 있다 = 穿着(状态)/正在穿

---

## 十一、-기 때문에 vs -(으)므로 ★★★★

|      | -기 때문에            | -(으)므로                  |
| ---- | --------------------- | -------------------------- |
| 场景 | 口语+书面             | 纯书面/正式                |
| 后句 | 可用命令/共动         | 可用命令/共动              |
| 例   | 바쁘기 때문에 못 가요 | 바쁘므로 참석하지 못합니다 |

---

## 十二、-는 동안 vs -(으)면서 ★★★

|      | -는 동안                                        | -(으)면서                                     |
| ---- | ----------------------------------------------- | --------------------------------------------- |
| 主语 | 可不同                                          | 一般相同                                      |
| 意思 | 在…期间                                         | 边…边…                                        |
| 例   | 내가 자**는 동안** 비가 왔다 (我睡觉期间下了雨) | 음악을 들**으면서** 공부하다 (边听音乐边学习) |

---

## 十三、-도록 vs -게 ★★★

|      | -도록                    | -게                                |
| ---- | ------------------------ | ---------------------------------- |
| 意思 | 以便/到…程度             | 使得/方式                          |
| 使役 | ✓ 공부하도록 하다        | ✓ 공부하게 하다                    |
| 程度 | 죽도록 일하다 (拼命工作) | 알아볼 수 없게 바뀌다 (变得认不出) |

---

## 十四、-지만 vs -(으)ㄴ/는데 ★★★★★

|      | -지만                      | -(으)ㄴ/는데                                  |
| ---- | -------------------------- | --------------------------------------------- |
| 意思 | 但是                       | 背景/转折/引出话题                            |
| 范围 | 纯转折                     | 转折+背景+惊讶+铺垫                           |
| 例   | 비싸**지만** 좋다 (贵但好) | 비싼**데** 그래도 살 거야? (贵呢，还是要买？) |
| 语气 | 客观对比                   | 语气柔和                                      |

---

## 十五、-(으)ㄹ 것 같다 vs -(으)ㄹ 듯하다 ★★★

|      | -(으)ㄹ 것 같다     | -(으)ㄹ 듯하다 |
| ---- | ------------------- | -------------- |
| 场景 | 口语+书面           | 书面为主       |
| 意思 | 完全相同(好像/似乎) | 完全相同       |
| 例   | 올 것 같다          | 올 듯하다      |

---

## 十六、-기에 vs -길래 ★★★

|      | -기에                     | -길래                     |
| ---- | ------------------------- | ------------------------- |
| 场景 | 书面                      | 口语                      |
| 意思 | 因为…所以                 | 因为(观察到)…所以         |
| 人称 | 无限制                    | 后句一般第一人称          |
| 例   | 날씨가 좋기에 산책을 갔다 | 날씨가 좋길래 산책을 갔어 |

---

## 十七、-아/어야 하다 vs -(으)ㄹ 필요가 있다 ★★★

|      | -아/어야 하다          | -(으)ㄹ 필요가 있다             |
| ---- | ---------------------- | ------------------------------- |
| 意思 | 应该/必须              | 有必要                          |
| 强度 | 义务                   | 建议/必要性                     |
| 例   | 공부해야 한다 (得学习) | 공부할 필요가 있다 (有必要学习) |

---

## 十八、速查辨析表

| 对比组                  | 关键区别                        |
| ----------------------- | ------------------------------- |
| -아서 vs -니까          | 前者客观/后者主观；后者可接命令 |
| -면 vs -다면            | 现实 vs 虚拟假设                |
| 것 같다 vs 나 보다      | 直接/间接判断 vs 间接证据       |
| -겠 vs -(으)ㄹ 것이다   | 即时判断 vs 有据推测            |
| -(으)ㄹ게 vs -(으)ㄹ래  | 承诺 vs 自愿                    |
| -고 vs -아서(顺序)      | 独立并列 vs 紧密关联            |
| 은/는 vs 이/가          | 主题旧信息 vs 主格新信息        |
| -에서 vs -에            | 动态场所 vs 静态位置/方向       |
| -아/어 있다 vs -고 있다 | 结果状态 vs 进行中              |
| -거든요 vs -(으)니까요  | 补充说明 vs 理由强调            |
| -지만 vs -는데          | 纯转折 vs 背景铺垫              |

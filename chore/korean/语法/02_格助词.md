# 格助词 (격조사)

> 格助词附着在名词后，标记该名词在句子中的语法角色
> ★ = TOPIK 常考

---

## 一、主格助词 이/가

### 基本用法

- 有收音 + **이** / 无收音 + **가**
- 标记主语

| 例句                 | 翻译       |
| -------------------- | ---------- |
| **비가** 온다.       | 下雨了。   |
| **학생이** 공부한다. | 学生学习。 |
| **날씨가** 좋다.     | 天气好。   |
| **시간이** 없다.     | 没有时间。 |

### 特殊用法

#### 1) 新信息标记 ★

第一次提到/回答"谁"的问题时用 이/가

> Q: 누가 왔어요? (谁来了?)
> A: **철수가** 왔어요. (哲秀来了。)

#### 2) 存在句主语

- **교실에 학생이** 있다. (教室里有学生。)
- **집에 아무도** 없다. (家里没人。)

#### 3) 되다/아니다 前的补语

- 물이 **얼음이** 되었다. (水变成了冰。)
- 그것은 **사실이** 아니다. (那不是事实。)

#### 4) 能力/可能的对象 ★

- 한국어**가** 어렵다. (韩语难。)
- 김치**가** 맛있다. (泡菜好吃。)
- 영어**가** 되다. (会英语。)

## 二、宾格助词 을/를

### 基本用法

- 有收音 + **을** / 无收音 + **를**
- 标记宾语（动作的对象）

| 例句             | 翻译     |
| ---------------- | -------- |
| 밥**을** 먹다.   | 吃饭。   |
| 영화**를** 보다. | 看电影。 |
| 편지**를** 쓰다. | 写信。   |

### 特殊用法

#### 1) 移动动词的出发点/经过点 ★

- 집**을** 나가다. (离开家。)
- 길**을** 건너다. (过马路。)
- 다리**를** 지나다. (经过桥。)
- 학교**를** 졸업하다. (从学校毕业。)

#### 2) 时间持续

- 하루**를** 기다렸다. (等了一天。)
- 일 년**을** 공부했다. (学了一年。)

#### 3) 하다 动词的对象（常省略）

- 공부(**를**) 하다
- 운동(**을**) 하다
- 요리(**를**) 하다

---

## 三、属格助词 의

### 基本用法

- 表示所属关系：N1의 N2 = N1的N2

| 例句            | 翻译       |
| --------------- | ---------- |
| **나의** 책     | 我的书     |
| **한국의** 문화 | 韩国的文化 |
| **회사의** 규칙 | 公司的规则 |

### 注意事项

#### 1) 口语中常省略

- 나의 → 내, 너의 → 네(니)
- 우리의 집 → 우리 집
- 한국의 음식 → 한국 음식

#### 2) 书面语中必须用

- 민주주의**의** 발전 (民主主义的发展)
- 교육**의** 중요성 (教育的重要性)

## 四、与格助词 에게 / 한테 / 께

### 表示动作的对象（给谁/对谁）

| 助词     | 用法      | 例子                        |
| -------- | --------- | --------------------------- |
| **에게** | 书面/标准 | 친구**에게** 편지를 보냈다. |
| **한테** | 口语      | 친구**한테** 전화했다.      |
| **께**   | 敬语 ★    | 선생님**께** 말씀드렸다.    |

### 相反方向：에게서 / 한테서 / 께서（从谁那里）

| 助词       | 例子                                               |
| ---------- | -------------------------------------------------- |
| **에게서** | 친구**에게서** 선물을 받았다. (从朋友那收到礼物。) |
| **한테서** | 동생**한테서** 연락이 왔다. (弟弟那来了联系。)     |

## 五、处格助词 에 / 에서

### 에 的用法 ★★★

#### 1) 场所（存在的位置）

- 학교**에** 있다. (在学校。)
- 서울**에** 산다. (住在首尔。)

#### 2) 时间

- 3시**에** 만나자. (3点见。)
- 월요일**에** 시작한다. (周一开始。)

#### 3) 方向/目的地

- 학교**에** 가다. (去学校。)
- 부산**에** 도착했다. (到达釜山。)

#### 4) 原因

- 비**에** 젖었다. (被雨淋湿了。)
- 감기**에** 걸렸다. (得了感冒。)

#### 5) 单位/基准

- 한 개**에** 천 원이다. (一个一千元。)

### 에서 的用法 ★★★

#### 1) 场所（动作发生的地点）

- 도서관**에서** 공부하다. (在图书馆学习。)
- 식당**에서** 밥을 먹다. (在餐厅吃饭。)

#### 2) 出发点

- 서울**에서** 출발하다. (从首尔出发。)
- 여기**에서** 저기까지. (从这里到那里。)

#### 3) 组织/团体主语 ★

- 정부**에서** 발표했다. (政府发表了。)
- 학교**에서** 결정했다. (学校决定了。)

### 에 vs 에서 辨析 ★★★

|          | 에                        | 에서                          |
| -------- | ------------------------- | ----------------------------- |
| 核心     | 静态/存在                 | 动态/动作                     |
| 있다     | 학교**에** 있다 ✓         | ✗                             |
| 공부하다 | ✗                         | 학교**에서** 공부하다 ✓       |
| 가다     | 학교**에** 가다 (去→学校) | 학교**에서** 가다 (从学校→去) |
| 살다     | 서울**에** 살다 ✓         | 서울**에서** 살다 ✓ (都可)    |

## 六、工具格助词 (으)로

### 基本形式

- 收音(ㄹ除外) + **으로** / 无收音或ㄹ收音 + **로**

### 用法 ★★★

#### 1) 方向

- 오른쪽**으로** 가세요. (请往右走。)
- 서울**로** 이사했다. (搬到首尔了。)

#### 2) 手段/工具

- 버스**로** 가다. (坐公交去。)
- 젓가락**으로** 먹다. (用筷子吃。)
- 한국어**로** 말하다. (用韩语说。)

#### 3) 材料

- 나무**로** 만들다. (用木头做。)
- 쌀**로** 떡을 만들다. (用米做年糕。)

#### 4) 原因/理由

- 감기**로** 결석했다. (因感冒缺席。)
- 교통사고**로** 다쳤다. (因交通事故受伤。)

#### 5) 身份/资格

- 대표**로** 참석했다. (作为代表出席。)
- 선물**로** 주다. (作为礼物送。)

#### 6) 变化的结果

- 얼음이 물**로** 변했다. (冰变成了水。)
- 한국어 선생님**으로** 일하다. (做韩语老师。)

## 七、呼格助词 아/야

- 有收音 + **아** / 无收音 + **야**
- 用于称呼（非敬语）

| 例句                 | 翻译                      |
| -------------------- | ------------------------- |
| 철수**야**, 이리 와! | 哲秀，过来！              |
| 민수**야**, 뭐 해?   | 民秀，你干嘛？            |
| 언니**야**           | (很少见，通常直接说 언니) |

---

## 八、比较格助词 보다

- 表示比较的对象

| 例句                            | 翻译             |
| ------------------------------- | ---------------- |
| 오늘이 어제**보다** 더 춥다.    | 今天比昨天更冷。 |
| 한국어가 일본어**보다** 어렵다. | 韩语比日语难。   |
| 기대**보다** 맛있었다.          | 比期待的更好吃。 |
| 생각**보다** 비쌌다.            | 比想的更贵。     |

---

## 九、引用格助词 (이)라고 / 고

- 标记引用内容（详见文件18\_间接引语）

| 例句                                 | 翻译             |
| ------------------------------------ | ---------------- |
| 이것을 한국어**로** 뭐**라고** 해요? | 这用韩语怎么说？ |
| 내일 간다**고** 했다.                | 说明天去。       |

## 十、格助词总览表

| 格       | 助词          | 功能           | 例子                  |
| -------- | ------------- | -------------- | --------------------- |
| 主格     | 이/가         | 主语           | 비**가** 온다         |
| 主格(敬) | 께서          | 敬语主语       | 선생님**께서** 오셨다 |
| 宾格     | 을/를         | 宾语           | 책**을** 읽다         |
| 属格     | 의            | 所属           | 나**의** 책           |
| 与格(人) | 에게/한테/께  | 对象(人)       | 친구**에게** 주다     |
| 处格(静) | 에            | 位置/时间/方向 | 학교**에** 가다       |
| 处格(动) | 에서          | 动作场所/出发  | 학교**에서** 공부하다 |
| 工具格   | (으)로        | 手段/方向/原因 | 버스**로** 가다       |
| 比较格   | 보다          | 比较           | 어제**보다** 춥다     |
| 呼格     | 아/야         | 称呼           | 철수**야**            |
| 引用格   | (이)라고      | 引用           | 뭐**라고** 했어?      |
| 共同格   | 와/과, (이)랑 | 和             | 나**와** 너           |

## 十一、格助词叠加规则

格助词一般不能叠用，但格助词 + 补助词可以叠：

| 组合        | 例子                               | 翻译                   |
| ----------- | ---------------------------------- | ---------------------- |
| 에 + 는     | 학교**에는** 학생이 많다.          | 在学校学生多。         |
| 에서 + 는   | 한국**에서는** 김치를 많이 먹는다. | 在韩国吃很多泡菜。     |
| 에게 + 는   | 나**에게는** 꿈이 있다.            | 我有梦想。             |
| (으)로 + 는 | 버스**로는** 갈 수 없다.           | 坐公交去不了。         |
| 에서 + 도   | 일본**에서도** 한국어를 배운다.    | 在日本也学韩语。       |
| 보다 + 는   | 어제**보다는** 낫다.               | 比昨天好一些（强调）。 |
| 보다 + 도   | 생각**보다도** 어렵다.             | 比想的还要难。         |

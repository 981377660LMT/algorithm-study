# 表达程度·强调 (정도·강조 표현)

---

## 一、얼마나 -(으)ㄴ/는지 모르다 (不知道有多…) ★★★★★

感叹程度之深，相当于"多么…啊"

| 例句                                 | 翻译             |
| ------------------------------------ | ---------------- |
| **얼마나** 기쁜**지 몰라요**.        | 不知道有多高兴。 |
| **얼마나** 맛있는**지 몰라요**.      | 不知道有多好吃。 |
| **얼마나** 힘들었는**지 모른다**.    | 不知道有多辛苦。 |
| **얼마나** 보고 싶었는**지 몰라요**. | 不知道有多想你。 |

---

## 二、-(으)ㄹ 정도로 / -(으)ㄹ 정도이다 (到…的程度) ★★★★★

| 例句                               | 翻译                 |
| ---------------------------------- | -------------------- |
| 눈물이 날 **정도로** 감동적이었다. | 感动到要流泪的程度。 |
| 걸을 수 없을 **정도로** 다쳤다.    | 伤到走不了路的程度。 |
| 밤을 잘 수 없을 **정도였다**.      | 到了睡不着觉的程度。 |
| 놀랄 **정도로** 빠르다.            | 快到让人吃惊的程度。 |

---

## 三、-(으)ㄹ 만큼 (到…的程度/足够…) ★★★★

| 例句                                | 翻译                         |
| ----------------------------------- | ---------------------------- |
| 먹을 **만큼** 먹었다.               | 吃够了。                     |
| 죽을 **만큼** 힘들다.               | 累得要死。                   |
| 다 알아들을 **만큼** 실력이 늘었다. | 实力提高到都能听懂的程度了。 |
| 깜짝 놀랄 **만큼** 예쁘다.          | 美到让人大吃一惊。           |

---

## 四、-기 짝이 없다 (…极了) ★★★

非常…（书面/强调到极致）

| 例句                    | 翻译       |
| ----------------------- | ---------- |
| 무례하**기 짝이 없다**. | 无礼至极。 |
| 불편하**기 짝이 없다**. | 不便至极。 |
| 위험하**기 짝이 없다**. | 危险至极。 |
| 어리석**기 짝이 없다**. | 愚蠢至极。 |

---

## 五、-기 그지없다 (无比…) ★★★

书面语，意思与 -기 짝이 없다 相近

| 例句                   | 翻译       |
| ---------------------- | ---------- |
| 기쁘**기 그지없다**.   | 无比高兴。 |
| 감사하**기 그지없다**. | 无比感谢。 |
| 안타깝**기 그지없다**. | 无比遗憾。 |

---

## 六、-(으)ㄹ 수 없을 만큼 / 정도로 ★★★

到…不了的程度

| 例句                            | 翻译             |
| ------------------------------- | ---------------- |
| 말할 **수 없을 만큼** 슬펐다.   | 悲伤到无法言说。 |
| 헤아릴 **수 없을 정도로** 많다. | 多到数不清。     |

---

## 七、-다 못해 (…不了了/实在忍不住) ★★★

做到极限之后转向另一行动

| 例句                         | 翻译                 |
| ---------------------------- | -------------------- |
| 참**다 못해** 화를 냈다.     | 忍无可忍发了火。     |
| 기다리**다 못해** 먼저 갔다. | 等不了了先走了。     |
| 울**다 못해** 웃었다.        | 哭到不能再哭就笑了。 |

---

## 八、-(으)ㄹ 뿐이다 (只是/仅仅) ★★★★

限定程度（强调只有这点）

| 例句                    | 翻译           |
| ----------------------- | -------------- |
| 바라볼 **뿐이었다**.    | 只是看着而已。 |
| 최선을 다할 **뿐이다**. | 只能尽全力了。 |
| 기다릴 **뿐이다**.      | 只能等了。     |

---

## 九、-기는 하다 (倒是…) ★★★★

承认事实，但暗含"不过/虽然"的语气

| 例句                               | 翻译                   |
| ---------------------------------- | ---------------------- |
| 맛있**기는 해요**. (근데 비싸요.)  | 倒是好吃。(不过很贵。) |
| 알**기는 하는데** 설명하기 어렵다. | 倒是知道，但难以解释。 |
| 했**기는 했어요**.                 | 倒是做了。             |

### 缩略: -긴 하다

| 例句               | 翻译          |
| ------------------ | ------------- |
| 맛있**긴 해요**.   | 倒是好吃。    |
| 알**긴 아는데**... | 倒是知道但... |
| 좋**긴 좋다**.     | 倒是好。      |

---

## 十、-다(가) 보면 (一直做就…) ★★★★

持续做某事就会到达某程度/状态

| 例句                           | 翻译                 |
| ------------------------------ | -------------------- |
| 연습하**다 보면** 잘하게 된다. | 一直练就会做好。     |
| 살**다 보면** 좋은 일도 있다.  | 活着活着也会有好事。 |
| 듣**다 보면** 알게 돼요.       | 一直听就会明白。     |

---

## 十一、-(으)ㄹ수록 (越…越…) ★★★★★

| 例句                 | 翻译         |
| -------------------- | ------------ |
| 갈**수록** 어렵다.   | 越来越难。   |
| 볼**수록** 예쁘다.   | 越看越漂亮。 |
| 먹을**수록** 맛있다. | 越吃越好吃。 |
| 할**수록** 재미있다. | 越做越有趣。 |

### 常用搭配: -면 -(으)ㄹ수록

| 例句                                | 翻译                   |
| ----------------------------------- | ---------------------- |
| 공부하**면 할수록** 모르는 게 많다. | 越学越觉得不知道的多。 |
| 생각하**면 할수록** 답이 안 나온다. | 越想越想不出答案。     |

## 十二、부사 강조 (副词强调) ★★★★★

| 副词   | 意思       | 例句                             |
| ------ | ---------- | -------------------------------- |
| 아주   | 非常       | **아주** 좋다 (非常好)           |
| 매우   | 非常(正式) | **매우** 중요하다 (非常重要)     |
| 너무   | 太/非常    | **너무** 예쁘다 (太漂亮了)       |
| 정말   | 真的       | **정말** 맛있다 (真的好吃)       |
| 진짜   | 真的(口语) | **진짜** 좋다 (真的好)           |
| 엄청   | 超级(口语) | **엄청** 크다 (超级大)           |
| 굉장히 | 非常       | **굉장히** 빠르다 (非常快)       |
| 상당히 | 相当       | **상당히** 어렵다 (相当难)       |
| 대단히 | 非常(正式) | **대단히** 감사합니다 (非常感谢) |
| 무척   | 非常       | **무척** 기쁘다 (非常高兴)       |
| 몹시   | 非常       | **몹시** 춥다 (非常冷)           |

### 程度由弱到强

약간 < 조금 < 꽤 < 상당히 < 아주/매우 < 무척/굉장히 < 너무 < 엄청

## 十三、其他强调表达

### -만 해도 (光…就)

| 例句                             | 翻译                 |
| -------------------------------- | -------------------- |
| 서울**만 해도** 인구가 천만이다. | 光首尔人口就一千万。 |
| 이번 달**만 해도** 세 번 갔다.   | 光这个月就去了三次。 |

### 무려 (竟然)

| 例句                          | 翻译             |
| ----------------------------- | ---------------- |
| **무려** 10시간이나 기다렸다. | 竟然等了10小时。 |

### (이)나 (强调多)

| 例句                  | 翻译              |
| --------------------- | ----------------- |
| 10개**나** 먹었다.    | 吃了(竟然)10个。  |
| 3시간**이나** 걸렸다. | (竟然)花了3小时。 |

### 밖에 (只有，含少的意外)

| 例句                    | 翻译           |
| ----------------------- | -------------- |
| 3명**밖에** 안 왔다.    | 只来了3个人。  |
| 10분**밖에** 안 남았다. | 只剩10分钟了。 |

## 十四、总览

| 表达              | 核心        | 例句                 |
| ----------------- | ----------- | -------------------- |
| 얼마나 ~지 모르다 | 多么…啊     | 얼마나 좋은지 몰라요 |
| -(으)ㄹ 정도로    | 到…程度     | 울 정도로 슬펐다     |
| -(으)ㄹ 만큼      | 足够…的程度 | 죽을 만큼 힘들다     |
| -기 짝이 없다     | …极了       | 무례하기 짝이 없다   |
| -다 못해          | 忍不住      | 참다 못해 화냈다     |
| -(으)ㄹ수록       | 越…越…      | 볼수록 예쁘다        |
| -기는 하다        | 倒是…       | 좋긴 좋다            |
| -다 보면          | 一直做就    | 하다 보면 잘된다     |

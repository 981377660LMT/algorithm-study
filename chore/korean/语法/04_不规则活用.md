# 不规则活用 (불규칙 활용)

> 韩语有8种不规则活用，词干在接特定语尾时发生形态变化
> ★ = 出现频率高，必须掌握

---

## 一、ㄷ 不规则 ★★★

### 规则

词干末收音 **ㄷ** + 以**元音**开头的语尾 → ㄷ 变 **ㄹ**

### 常见词汇

| 原形   | + 아/어 | + (으)면 | + (으)니 | 中文 |
| ------ | ------- | -------- | -------- | ---- |
| 듣다   | 들어    | 들으면   | 들으니   | 听   |
| 걷다   | 걸어    | 걸으면   | 걸으니   | 走   |
| 묻다   | 물어    | 물으면   | 물으니   | 问   |
| 싣다   | 실어    | 실으면   | 실으니   | 装载 |
| 깨닫다 | 깨달아  | 깨달으면 | 깨달으니 | 领悟 |

### 规则活用的 ㄷ（不变化）

| 原形 | + 아/어       | 中文 |
| ---- | ------------- | ---- |
| 받다 | 받아 (✗ 발아) | 收到 |
| 닫다 | 닫아 (✗ 달아) | 关   |
| 믿다 | 믿어 (✗ 밀어) | 相信 |
| 얻다 | 얻어 (✗ 얼어) | 得到 |

### 辨析：묻다

| 词   | 意思 | 活用 | 类型   |
| ---- | ---- | ---- | ------ |
| 묻다 | 问   | 물어 | 不规则 |
| 묻다 | 埋   | 묻어 | 规则   |

## 二、ㅂ 不规则 ★★★★★

### 规则

词干末收音 **ㅂ** + 以**元音**开头的语尾 → ㅂ 变 **우** (or 오)

### 常见词汇

| 原形     | + 아/어  | + (으)면   | + (으)ㄴ | 中文       |
| -------- | -------- | ---------- | -------- | ---------- |
| 춥다     | 추워     | 추우면     | 추운     | 冷         |
| 덥다     | 더워     | 더우면     | 더운     | 热         |
| 어렵다   | 어려워   | 어려우면   | 어려운   | 难         |
| 쉽다     | 쉬워     | 쉬우면     | 쉬운     | 容易       |
| 무섭다   | 무서워   | 무서우면   | 무서운   | 可怕       |
| 아름답다 | 아름다워 | 아름다우면 | 아름다운 | 美丽       |
| 가볍다   | 가벼워   | 가벼우면   | 가벼운   | 轻         |
| 무겁다   | 무거워   | 무거우면   | 무거운   | 重         |
| 귀엽다   | 귀여워   | 귀여우면   | 귀여운   | 可爱       |
| 맵다     | 매워     | 매우면     | 매운     | 辣         |
| 고맙다   | 고마워   | 고마우면   | 고마운   | 感谢       |
| 반갑다   | 반가워   | 반가우면   | 반가운   | 高兴(见面) |
| 즐겁다   | 즐거워   | 즐거우면   | 즐거운   | 快乐       |
| 슬프다   | 슬퍼     | 슬프면     | 슬픈     | 悲伤       |
| 기쁘다   | 기뻐     | 기쁘면     | 기쁜     | 高兴       |
| 나쁘다   | 나빠     | 나쁘면     | 나쁜     | 坏         |
| 바쁘다   | 바빠     | 바쁘면     | 바쁜     | 忙         |
| 예쁘다   | 예뻐     | 예쁘면     | 예쁜     | 漂亮       |
| 돕다     | 도와 ★   | 도우면     | 도운     | 帮助       |
| 곱다     | 고와 ★   | 고우면     | 고운     | 美/细      |

### 规则活用的 ㅂ（不变化）

| 原形 | + 아/어 | 中文   |
| ---- | ------- | ------ |
| 잡다 | 잡아    | 抓     |
| 좁다 | 좁아    | 窄     |
| 뽑다 | 뽑아    | 拔/选  |
| 씹다 | 씹어    | 嚼     |
| 업다 | 업어    | 背(人) |
| 입다 | 입어    | 穿     |

## 三、ㅅ 不规则 ★★★

### 规则

词干末收音 **ㅅ** + 以**元音**开头的语尾 → ㅅ **脱落**

### 常见词汇

| 原形 | + 아/어 | + (으)면 | + (으)ㄴ | 中文      |
| ---- | ------- | -------- | -------- | --------- |
| 짓다 | 지어    | 지으면   | 지은     | 建/造     |
| 낫다 | 나아    | 나으면   | 나은     | 好(痊愈)  |
| 잇다 | 이어    | 이으면   | 이은     | 连接      |
| 붓다 | 부어    | 부으면   | 부은     | 肿/倒(水) |
| 젓다 | 저어    | 저으면   | 저은     | 搅        |
| 긋다 | 그어    | 그으면   | 그은     | 画(线)    |

### 规则活用的 ㅅ（不变化）

| 原形 | + 아/어 | 中文 |
| ---- | ------- | ---- |
| 웃다 | 웃어    | 笑   |
| 씻다 | 씻어    | 洗   |
| 벗다 | 벗어    | 脱   |
| 빗다 | 빗어    | 梳   |

## 四、ㄹ 不规则 ★★★★★

### 规则

词干末收音 **ㄹ** + **ㄴ, ㅂ, ㅅ** 开头的语尾 → ㄹ **脱落**

### 变化示例

| 原形   | + ㄴ (冠形) | + ㅂ니다 | + ㅅ니까 | + 세요   | 中文  |
| ------ | ----------- | -------- | -------- | -------- | ----- |
| 살다   | 사는        | 삽니다   | —        | 사세요   | 住/活 |
| 알다   | 아는        | 압니다   | —        | 아세요   | 知道  |
| 만들다 | 만드는      | 만듭니다 | —        | 만드세요 | 做    |
| 팔다   | 파는        | 팝니다   | —        | 파세요   | 卖    |
| 열다   | 여는        | 엽니다   | —        | 여세요   | 开    |
| 놀다   | 노는        | 놉니다   | —        | 노세요   | 玩    |
| 멀다   | 먼          | 멉니다   | —        | 머세요   | 远    |
| 길다   | 긴          | 깁니다   | —        | 기세요   | 长    |
| 달다   | 단          | 답니다   | —        | 다세요   | 甜    |
| 풀다   | 푸는        | 풉니다   | —        | 푸세요   | 解开  |
| 울다   | 우는        | 웁니다   | —        | 우세요   | 哭    |
| 들다   | 드는        | 듭니다   | —        | 드세요   | 拿/进 |

### 注意

- 接 **-아/어, -고, -(으)면** 等时，ㄹ **不脱落**
  - 살다 → 살아, 살고, 살면
- 只在接 **ㄴ, ㅂ, ㅅ** 时脱落

## 五、르 不规则 ★★★★

### 规则

词干以 **르** 结尾 + **-아/어** → 르 变为 **ㄹ라/ㄹ러**

### 常见词汇

| 原形     | + 아/어 | 中文   |
| -------- | ------- | ------ |
| 모르다   | 몰라    | 不知道 |
| 부르다   | 불러    | 叫/唱  |
| 다르다   | 달라    | 不同   |
| 빠르다   | 빨라    | 快     |
| 고르다   | 골라    | 选     |
| 누르다   | 눌러    | 按     |
| 자르다   | 잘라    | 切     |
| 기르다   | 길러    | 养     |
| 마르다   | 말라    | 干/瘦  |
| 오르다   | 올라    | 上     |
| 흐르다   | 흘러    | 流     |
| 서두르다 | 서둘러  | 赶紧   |
| 게으르다 | 게을러  | 懒     |

### 规则

```
[前面的字] + ㄹ + [라 or 러]
모르 → 모 + ㄹ + 라 = 몰라 (ㅗ → 라)
기르 → 기 + ㄹ + 러 = 길러 (ㅣ → 러)
```

## 六、ㅎ 不规则 ★★★★

### 规则

词干末收音 **ㅎ** + 以**元音**开头的语尾 → ㅎ **脱落**
特别：+ **-아/어** → 变为 **-애/에**

### 常见词汇

| 原形                                              | + 아/어 | + (으)면      | + (으)ㄴ | 中文     |
| ------------------------------------------------- | ------- | ------------- | -------- | -------- |
| 그렇다                                            | 그래    | 그러면        | 그런     | 那样     |
| 이렇다                                            | 이래    | 이러면        | 이런     | 这样     |
| 저렇다                                            | 저래    | 저러면        | 저런     | 那样(远) |
| 어떻다                                            | 어때    | 어떠면→어떻게 | 어떤     | 怎样     |
| 빨갛다                                            | 빨개    | 빨가면        | 빨간     | 红       |
| 파랗다                                            | 파래    | 파라면        | 파란     | 蓝       |
| 노랗다                                            | 노래    | 노라면        | 노란     | 黄       |
| 하얗다                                            | 하얘    | 하야면        | 하얀     | 白       |
| 까맣다                                            | 까매    | 까마면        | 까만     | 黑       |
| 그렇다 系列 → 그래서, 그러니까, 그러면, 그런데... |

### 그렇다 系列全家福 ★★★★★

| 词     | +아/어서 | +(으)ㄴ데 | +(으)면 | +(으)ㄴ | +(으)니까 |
| ------ | -------- | --------- | ------- | ------- | --------- |
| 그렇다 | 그래서   | 그런데    | 그러면  | 그런    | 그러니까  |
| 이렇다 | 이래서   | 이런데    | 이러면  | 이런    | 이러니까  |
| 저렇다 | 저래서   | 저런데    | 저러면  | 저런    | 저러니까  |
| 어떻다 | 어때서   | 어떤데    | 어떻게  | 어떤    | —         |

## 七、으 不规则 ★★★★

### 规则

词干以 **으** 结尾 + **-아/어** → 으 **脱落**，根据 으 前的元音决定 아/어

### 常见词汇

| 原形   | 으前元音 | + 아/어 | 中文     |
| ------ | -------- | ------- | -------- |
| 쓰다   | (无)     | 써      | 写/用/戴 |
| 끄다   | (无)     | 꺼      | 关(灯)   |
| 크다   | (无)     | 커      | 大       |
| 뜨다   | (无)     | 떠      | 浮/编织  |
| 바쁘다 | ㅏ       | 바빠    | 忙       |
| 나쁘다 | ㅏ       | 나빠    | 坏       |
| 아프다 | ㅏ       | 아파    | 疼       |
| 예쁘다 | ㅖ(→어)  | 예뻐    | 漂亮     |
| 슬프다 | ㅡ(→어)  | 슬퍼    | 悲伤     |
| 기쁘다 | ㅣ(→어)  | 기뻐    | 高兴     |
| 고프다 | ㅗ       | 고파    | 饿       |
| 모으다 | ㅗ       | 모아    | 收集     |
| 잠그다 | ㅏ       | 잠가    | 锁       |

### 规则

```
으 前无字 → 어 (쓰다 → 써)
으 前有 ㅏ/ㅗ → 아 (바쁘다 → 바빠)
으 前有其他 → 어 (슬프다 → 슬퍼)
```

## 八、여 不规则（하다 类） ★★★★★

### 规则

**하다** + **-아/어** → **해** (不是 하어 或 하아)

### 这是韩语中最常见的活用

| 原形     | + 아/어 | + 았/었 | 中文 |
| -------- | ------- | ------- | ---- |
| 하다     | 해      | 했      | 做   |
| 공부하다 | 공부해  | 공부했  | 学习 |
| 좋아하다 | 좋아해  | 좋아했  | 喜欢 |
| 시작하다 | 시작해  | 시작했  | 开始 |
| 필요하다 | 필요해  | 필요했  | 需要 |
| 중요하다 | 중요해  | 중요했  | 重要 |
| 조용하다 | 조용해  | 조용했  | 安静 |
| 깨끗하다 | 깨끗해  | 깨끗했  | 干净 |

## 九、不规则活用总览表

| 类型      | 条件          | 变化            | 代表词        |
| --------- | ------------- | --------------- | ------------- |
| ㄷ 不规则 | ㄷ + 元音语尾 | ㄷ → ㄹ         | 듣다 → 들어   |
| ㅂ 不规则 | ㅂ + 元音语尾 | ㅂ → 우/오      | 춥다 → 추워   |
| ㅅ 不规则 | ㅅ + 元音语尾 | ㅅ 脱落         | 짓다 → 지어   |
| ㄹ 不规则 | ㄹ + ㄴ/ㅂ/ㅅ | ㄹ 脱落         | 살다 → 삽니다 |
| 르 不规则 | 르 + -아/어   | 르 → ㄹ라/ㄹ러  | 모르다 → 몰라 |
| ㅎ 不规则 | ㅎ + 元音语尾 | ㅎ 脱落(+애/에) | 그렇다 → 그래 |
| 으 不规则 | 으 + -아/어   | 으 脱落         | 쓰다 → 써     |
| 여 不规则 | 하 + -아/어   | 하여 → 해       | 하다 → 해     |

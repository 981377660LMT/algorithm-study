# 否定表达 (부정 표현)

---

## 一、안 否定 ★★★★★

### 用法

**安** + 动词/形容词（能力/意愿上不做）

| 例句           | 翻译     |
| -------------- | -------- |
| **안** 간다.   | 不去。   |
| **안** 먹었다. | 没吃。   |
| **안** 예쁘다. | 不漂亮。 |
| **안** 좋다.   | 不好。   |

### 하다 动词的 안 否定

名词 + **안** + 하다 (不是放在名词前)

| 错误           | 正确             |
| -------------- | ---------------- |
| ~~안공부하다~~ | 공부 **안** 하다 |
| ~~안청소하다~~ | 청소 **안** 하다 |
| ~~안운동하다~~ | 운동 **안** 하다 |

---

## 二、-지 않다 ★★★★★

### 用法

词干 + **-지 않다**（和 안 同义，更书面/正式）

| 例句                 | 翻译     |
| -------------------- | -------- |
| 가**지 않는다**.     | 不去。   |
| 먹**지 않았다**.     | 没吃。   |
| 예쁘**지 않다**.     | 不漂亮。 |
| 공부하**지 않는다**. | 不学习。 |
| 비싸**지 않다**.     | 不贵。   |

### 안 vs -지 않다

|        | 안               | -지 않다          |
| ------ | ---------------- | ----------------- |
| 位置   | 谓语前           | 谓语后            |
| 语感   | 口语             | 正式/书面         |
| 意思   | 完全相同         | 完全相同          |
| 하다类 | 공부 **안** 하다 | 공부하**지 않다** |

---

## 三、못 否定 ★★★★★

### 用法

**못** + 动词（能力上做不了 / 情况不允许）

| 例句                       | 翻译               |
| -------------------------- | ------------------ |
| **못** 간다.               | 去不了。           |
| **못** 먹었다.             | 没能吃。           |
| 시간이 없어서 **못** 갔다. | 因为没时间去不了。 |
| 너무 매워서 **못** 먹겠다. | 太辣了吃不了。     |

### 하다 类动词

| 错误           | 正确             |
| -------------- | ---------------- |
| ~~못공부하다~~ | 공부 **못** 하다 |
| ~~못운동하다~~ | 운동 **못** 하다 |

### 못 + 形容词 ✗

못 一般**不用于形容词**

| 错误          | 正确                    |
| ------------- | ----------------------- |
| ~~못 예쁘다~~ | 예쁘지 않다 / 안 예쁘다 |
| ~~못 크다~~   | 크지 않다 / 안 크다     |

例外：못되다(坏), 못나다(丑), 못생기다(长得丑) — 这些是固定词

---

## 四、-지 못하다 ★★★★★

### 用法

词干 + **-지 못하다**（和 못 同义，更正式）

| 例句                     | 翻译       |
| ------------------------ | ---------- |
| 가**지 못한다**.         | 去不了。   |
| 먹**지 못했다**.         | 没能吃。   |
| 참석하**지 못했습니다**. | 未能出席。 |
| 이해하**지 못했다**.     | 没能理解。 |

---

## 五、안 vs 못 对比 ★★★★★

|        | 안 / -지 않다                   | 못 / -지 못하다                         |
| ------ | ------------------------------- | --------------------------------------- |
| 意思   | 不(意愿)                        | 不了/不能(能力)                         |
| 语感   | 不想做 / 选择不做               | 想做但做不了                            |
| 例     | 밥을 **안** 먹었다 (不吃[不想]) | 밥을 **못** 먹었다 (没能吃[想吃但不能]) |
| 形容词 | ✓                               | ✗ (一般不用)                            |

## 六、말다 ★★★★★

### 用法

**-지 말다** = 不要做...（禁止/劝阻）

只用于**命令句和共动句**

| 例句                           | 翻译             |
| ------------------------------ | ---------------- |
| 가**지 마**.                   | 不要去。         |
| 걱정하**지 마세요**.           | 不要担心。       |
| 여기서 사진 찍**지 마십시오**. | 请不要在这拍照。 |
| 포기하**지 맙시다**.           | 别放弃吧。       |
| 늦**지 마**!                   | 别迟到！         |
| 말하**지 마**.                 | 别说。           |

### 말다 的活用

| 形式            | 例子                        |
| --------------- | --------------------------- |
| -지 마          | 가지 마 (别去)              |
| -지 마세요      | 가지 마세요 (请别去)        |
| -지 마십시오    | 가지 마십시오 (请勿去)      |
| -지 맙시다      | 가지 맙시다 (别去了)        |
| -지 말라고 하다 | 가지 말라고 했다 (说叫别去) |
| -지 말 것       | 가지 말 것 (禁止去)         |

## 七、부정 의문 (否定疑问) ★★★

否定疑问句往往含有"期待肯定回答"的语气:

| 例句              | 翻译       | 期待回答     |
| ----------------- | ---------- | ------------ |
| **안** 갈 거예요? | 不要去吗？ | 期待"去"     |
| 맛**없**어요?     | 不好吃吗？ | 期待"好吃"   |
| 모르**세요**?     | 不知道吗？ | 觉得应该知道 |

---

## 八、双重否定 ★★★

-지 않을 수 없다 = 不得不...

| 例句                       | 翻译         |
| -------------------------- | ------------ |
| 인정하**지 않을 수 없다**. | 不得不承认。 |
| 가**지 않을 수 없었다**.   | 不得不去了。 |
| 놀라**지 않을 수 없었다**. | 不得不惊讶。 |

## 九、其他否定/否认表达

### 아니다 (不是)

| 例句                 | 翻译       |
| -------------------- | ---------- |
| 학생이 **아니다**.   | 不是学生。 |
| 사실이 **아닙니다**. | 不是事实。 |

### 없다 (没有)

| 例句             | 翻译       |
| ---------------- | ---------- |
| 시간이 **없다**. | 没有时间。 |
| 방법이 **없다**. | 没有办法。 |

### -(으)ㄹ 리가 없다 (不可能)

| 例句                | 翻译           |
| ------------------- | -------------- |
| 그럴 **리가 없다**. | 不可能那样。   |
| 모를 **리가 없다**. | 不可能不知道。 |

### -(으)ㄴ/는 적이 없다 (从没)

| 例句                      | 翻译           |
| ------------------------- | -------------- |
| 한 번도 간 **적이 없다**. | 一次都没去过。 |

### -ㄴ/는 것은 아니다 (并不是说)

| 例句                    | 翻译           |
| ----------------------- | -------------- |
| 싫다**는 것은 아니다**. | 并不是说讨厌。 |
| 모르**는 것은 아니다**. | 并不是不知道。 |

---

## 十、否定表达总览

| 表达              | 意思       | 用于        |
| ----------------- | ---------- | ----------- |
| 안 / -지 않다     | 不(意愿)   | 动词+形容词 |
| 못 / -지 못하다   | 不了(能力) | 动词        |
| -지 말다          | 不要(禁止) | 命令/共动   |
| 아니다            | 不是       | 名词        |
| 없다              | 没有       | 存在        |
| -(으)ㄹ 수 없다   | 不能       | 可能性      |
| -(으)ㄹ 리가 없다 | 不可能     | 推测否定    |
| -지 않을 수 없다  | 不得不     | 双重否定    |

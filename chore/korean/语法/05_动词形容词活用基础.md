# 动词形容词活用基础

> 韩语谓语的活用 = 词干 + 语尾
> 本文整理语尾接续的完整规则

---

## 一、词干的提取

**词干 = 原形去掉 다**

| 原形   | 词干 | 末元音 | 有无收音 |
| ------ | ---- | ------ | -------- |
| 가다   | 가   | ㅏ     | 无       |
| 먹다   | 먹   | ㅓ     | 有(ㄱ)   |
| 오다   | 오   | ㅗ     | 无       |
| 읽다   | 읽   | ㅣ     | 有(ㄺ)   |
| 배우다 | 배우 | ㅜ     | 无       |
| 크다   | 크   | ㅡ     | 无       |
| 하다   | 하   | ㅏ     | 无       |

## 二、语尾接续的三大类型

### 类型1：直接接续（以辅音开头的语尾）

不用考虑词干末尾的元音或收音，直接接。

| 语尾      | 가다      | 먹다      | 좋다      |
| --------- | --------- | --------- | --------- |
| -고       | 가고      | 먹고      | 좋고      |
| -지만     | 가지만    | 먹지만    | 좋지만    |
| -지 않다  | 가지 않다 | 먹지 않다 | 좋지 않다 |
| -는데(动) | 가는데    | 먹는데    | —         |
| -겠-      | 가겠다    | 먹겠다    | 좋겠다    |

### 类型2：아/어 接续

根据词干末元音的阴阳性选择 아 或 어。

| 规则 | 条件          | 例子                             |
| ---- | ------------- | -------------------------------- |
| + 아 | 末元音 ㅏ, ㅗ | 가 + 아 → 가, 오 + 아 → 와       |
| + 어 | 其他元音      | 먹 + 어 → 먹어, 배우 + 어 → 배워 |
| + 해 | 하다 类       | 하 + 여 → 해                     |

#### 아/어 缩写规则 ★

| 词干末 | + 아/어 | 缩写        | 例子             |
| ------ | ------- | ----------- | ---------------- |
| ㅏ     | + 아    | 省略一个 아 | 가 + 아 → 가     |
| ㅗ     | + 아    | → 와        | 오 + 아 → 와     |
| ㅓ     | + 어    | 省略一个 어 | 서 + 어 → 서     |
| ㅜ     | + 어    | → 워        | 배우 + 어 → 배워 |
| ㅣ     | + 어    | → 여        | 마시 + 어 → 마셔 |
| ㅡ     | + 아/어 | 으脱落      | 쓰 + 어 → 써     |
| ㅐ, ㅔ | + 어    | 省略        | 보내 + 어 → 보내 |

### 类型3：(으) 接续

根据词干有无收音，选择是否加 으。

| 条件   | 规则                      | 例子               |
| ------ | ------------------------- | ------------------ |
| 有收音 | + 으 + 语尾               | 먹 + 으면 → 먹으면 |
| 无收音 | 直接 + 语尾               | 가 + 면 → 가면     |
| ㄹ收音 | 直接 + 语尾(去ㄹ or 保留) | 살 + 면 → 살면     |

#### (으) 类语尾一览

| 语尾        | 有收音   | 无收音 | ㄹ收音 |
| ----------- | -------- | ------ | ------ |
| -(으)면     | 먹으면   | 가면   | 살면   |
| -(으)ㄴ     | 먹은     | 간     | 산     |
| -(으)ㄹ     | 먹을     | 갈     | 살     |
| -(으)니     | 먹으니   | 가니   | 사니   |
| -(으)니까   | 먹으니까 | 가니까 | 사니까 |
| -(으)려고   | 먹으려고 | 가려고 | 살려고 |
| -(으)러     | 먹으러   | 가러   | 살러   |
| -(으)세요   | 읽으세요 | 가세요 | 사세요 |
| -(으)ㅂ시다 | 먹읍시다 | 갑시다 | 삽시다 |
| -(으)ㅂ니다 | 먹습니다 | 갑니다 | 삽니다 |

## 三、完整活用表（以5个代表动词为例）

### 가다 (去) — 无收音, ㅏ

| 语尾        | 活用   | 语尾        | 活用      |
| ----------- | ------ | ----------- | --------- |
| -아/어요    | 가요   | -고         | 가고      |
| -았/었어요  | 갔어요 | -지만       | 가지만    |
| -(으)면     | 가면   | -지 않다    | 가지 않다 |
| -(으)ㄴ     | 간     | -는(动현재) | 가는      |
| -(으)ㄹ     | 갈     | -(으)려고   | 가려고    |
| -(으)세요   | 가세요 | -겠-        | 가겠다    |
| -(스)ㅂ니다 | 갑니다 | -자         | 가자      |

### 먹다 (吃) — 有收音 ㄱ, ㅓ

| 语尾        | 活用     | 语尾        | 活用      |
| ----------- | -------- | ----------- | --------- |
| -아/어요    | 먹어요   | -고         | 먹고      |
| -았/었어요  | 먹었어요 | -지만       | 먹지만    |
| -(으)면     | 먹으면   | -지 않다    | 먹지 않다 |
| -(으)ㄴ     | 먹은     | -는(动현재) | 먹는      |
| -(으)ㄹ     | 먹을     | -(으)려고   | 먹으려고  |
| -(으)세요   | 드세요 ★ | -겠-        | 먹겠다    |
| -(스)ㅂ니다 | 먹습니다 | -자         | 먹자      |

### 만들다 (做) — ㄹ收音

| 语尾        | 活用       | 语尾        | 活用        |
| ----------- | ---------- | ----------- | ----------- |
| -아/어요    | 만들어요   | -고         | 만들고      |
| -았/었어요  | 만들었어요 | -지만       | 만들지만    |
| -(으)면     | 만들면     | -지 않다    | 만들지 않다 |
| -(으)ㄴ     | 만든 ★     | -는(动현재) | 만드는 ★    |
| -(으)ㄹ     | 만들       | -(으)려고   | 만들려고    |
| -(으)세요   | 만드세요 ★ | -겠-        | 만들겠다    |
| -(스)ㅂ니다 | 만듭니다 ★ | -자         | 만들자      |

### 좋다 (好) — 形容词, 有收音 ㅎ(规则)

| 语尾       | 活用     | 语尾     | 活用      |
| ---------- | -------- | -------- | --------- |
| -아/어요   | 좋아요   | -고      | 좋고      |
| -았/었어요 | 좋았어요 | -지만    | 좋지만    |
| -(으)면    | 좋으면   | -지 않다 | 좋지 않다 |
| -(으)ㄴ    | 좋은     | -(으)ㄹ  | 좋을      |
| -(으)세요  | —        | -겠-     | 좋겠다    |

### 하다 (做) — 여불규칙

| 语尾        | 活用   | 语尾        | 活用      |
| ----------- | ------ | ----------- | --------- |
| -아/어요    | 해요   | -고         | 하고      |
| -았/었어요  | 했어요 | -지만       | 하지만    |
| -(으)면     | 하면   | -지 않다    | 하지 않다 |
| -(으)ㄴ     | 한     | -는(动현재) | 하는      |
| -(으)ㄹ     | 할     | -(으)려고   | 하려고    |
| -(으)세요   | 하세요 | -겠-        | 하겠다    |
| -(스)ㅂ니다 | 합니다 | -자         | 하자      |

## 四、动词 vs 形容词的活用差异 ★★★

| 语尾                          | 动词                   | 形容词                   |
| ----------------------------- | ---------------------- | ------------------------ |
| -는 (现在冠形)                | 먹**는** 사람 ✓        | ~~큰는~~ ✗ → **큰** 사람 |
| -(으)ㄴ (过去冠形/形容词现在) | 먹**은** 사람 (吃了的) | **큰** 사람 (大的)       |
| -고 있다 (进行)               | 먹**고 있다** ✓        | ~~크고 있다~~ ✗          |
| -(으)세요 (命令)              | 먹**으세요** ✓         | ~~크세요~~ ✗             |
| -(으)ㅂ시다 (共动)            | 먹**읍시다** ✓         | ~~큽시다~~ ✗             |

### 冠形词形语尾对比表 ★★★★★

|            | 过去             | 现在           | 将来           |
| ---------- | ---------------- | -------------- | -------------- |
| **动词**   | -(으)ㄴ (먹은)   | -는 (먹는)     | -(으)ㄹ (먹을) |
| **形容词** | -던 (좋던)       | -(으)ㄴ (좋은) | -(으)ㄹ (좋을) |
| **이다**   | -이던 (학생이던) | -인 (학생인)   | -일 (학생일)   |

## 五、이다 / 아니다 的活用

### 이다 (是)

| 语尾         | 有收音          | 无收音                           |
| ------------ | --------------- | -------------------------------- |
| -예요/이에요 | 학생**이에요**  | 의사**예요**                     |
| -입니다      | 학생**입니다**  | 의사**입니다** (口语读 의삽니다) |
| -(이)라서    | 학생**이라서**  | 의사**라서**                     |
| -(이)라고    | 학생**이라고**  | 의사**라고**                     |
| -(이)라면    | 학생**이라면**  | 의사**라면**                     |
| -인 (冠形)   | 학생**인** 사람 | 의사**인** 사람                  |

### 아니다 (不是)

| 语尾     | 活用                                  |
| -------- | ------------------------------------- |
| -아/어요 | 아니**에요**                          |
| -ㅂ니다  | 아닙니다                              |
| -(으)면  | 아니면                                |
| -(으)ㄴ  | 아닌                                  |
| -라서    | 아니라서 (不是因为) → 但常说 아니어서 |

## 六、있다 / 없다 的特殊性

**있다/없다** 具有动词和形容词的双重性。

| 用作             | 冠形词形         | 例子                          |
| ---------------- | ---------------- | ----------------------------- |
| 形容词 (有/没有) | -(으)ㄴ → 있는 ★ | 돈이 **있는** 사람 (有钱的人) |
| 动词 (在/存在)   | -는 → 있는       | 집에 **있는** 사람 (在家的人) |

> 特殊：있다/없다 的冠形词形**固定用 -는**，不用 -(으)ㄴ
> 맛있는 음식 ✓ / ~~맛있은 음식~~ ✗
> 재미없는 영화 ✓ / ~~재미없은 영화~~ ✗

# 连接语尾：条件假设

> 表示"如果、要是、只要"等条件关系

---

## 一、-(으)면 ★★★★★

### 接续

- 有收音 + -으면 / 无收音 + -면 / ㄹ收音 + -면

### 意思

如果...的话

| 例句                                  | 翻译                 |
| ------------------------------------- | -------------------- |
| 시간이 있**으면** 같이 가요.          | 如果有时间一起去。   |
| 비가 오**면** 집에 있을래요.          | 如果下雨想在家待着。 |
| 열심히 공부하**면** 합격할 수 있어요. | 如果努力学习能合格。 |
| 모르**면** 물어보세요.                | 不知道的话请问。     |
| 살**면** 살수록 어렵다.               | 越活越难。           |

### 固定搭配

| 表达                | 翻译                  |
| ------------------- | --------------------- |
| -(으)면 되다        | ...就行了             |
| -(으)면 좋겠다      | 要是...就好了         |
| -(으)면 안 되다     | 不能...               |
| -았/었으면 좋겠다   | 要是...了就好了(愿望) |
| -(으)면 -(으)ㄹ수록 | 越...越...            |

| 例句                            | 翻译                   |
| ------------------------------- | ---------------------- |
| 여기에 사인하**면 돼요**.       | 在这签名就行了。       |
| 내일 날씨가 좋**으면 좋겠다**.  | 要是明天天气好就好了。 |
| 여기서 사진 찍**으면 안 돼요**. | 这里不能拍照。         |
| 비가 안 **왔으면 좋겠다**.      | 要是不下雨就好了。     |
| 먹**으면 먹을수록** 맛있다.     | 越吃越好吃。           |

## 二、-거든 ★★★★

### 接续

词干 + -거든 (直接接)

### 意思

如果...的话（比 -(으)면 更强调"如果真的发生了"）

| 例句                                 | 翻译                   |
| ------------------------------------ | ---------------------- |
| 서울에 오**거든** 연락해.            | 如果来首尔就联系我。   |
| 시간이 있**거든** 도와줘.            | 如果有时间就帮帮我。   |
| 필요하**거든** 말해.                 | 如果需要就说。         |
| 혹시 그 사람을 만나**거든** 전해 줘. | 万一见到那个人就转告。 |

### -(으)면 vs -거든 ★★

|        | -(으)면      | -거든             |
| ------ | ------------ | ----------------- |
| 可能性 | 一般假设     | 可能性较低/不确定 |
| 语感   | 如果（一般） | 如果真的/万一     |
| 后句   | 都可         | 多搭配命令/共动   |

---

## 三、-다면 ★★★★

### 接续

词干 + -다면 (해라体 + 면)

### 意思

如果说...（假设性更强，可能是虚拟的）

| 例句                                | 翻译                      |
| ----------------------------------- | ------------------------- |
| 내가 새**라면** 하늘을 날고 싶다.   | 如果我是鸟想在天上飞。    |
| 시간을 되돌릴 수 있**다면** 좋겠다. | 如果能倒流时间就好了。    |
| 100만 원이 있**다면** 뭘 하겠어요?  | 如果有100万元你会做什么？ |
| 그것이 사실이**라면** 큰 문제다.    | 如果那是事实就是大问题。  |

### -(으)면 vs -다면 ★

|        | -(으)면                   | -다면                    |
| ------ | ------------------------- | ------------------------ |
| 现实性 | 可能发生                  | 虚拟/不太可能            |
| 例     | 내일 비가 오면 (可能下雨) | 내가 새라면 (不可能是鸟) |

---

## 四、-았/었더라면 ★★★

### 接续

词干 + -았/었더라면

### 意思

如果当时...的话（对过去的虚拟假设 = 后悔）

| 例句                                     | 翻译                       |
| ---------------------------------------- | -------------------------- |
| 열심히 공부했**더라면** 합격했을 텐데.   | 要是当时努力学了就合格了。 |
| 그때 말했**더라면** 이렇게 안 됐을 거야. | 要是那时说了就不会这样。   |
| 일찍 출발했**더라면** 안 늦었을 거예요.  | 要是早出发就不会迟到。     |
| 그 사람을 만나지 않았**더라면**...       | 要是没遇到那个人的话...    |

---

## 五、-(으)려면 ★★★★

### 接续

词干 + -(으)려면

### 意思

如果想要...的话（= -(으)려고 하면 的缩写）

| 例句                                             | 翻译                       |
| ------------------------------------------------ | -------------------------- |
| 한국어를 잘하**려면** 많이 연습해야 해요.        | 想学好韩语的话要多练习。   |
| 건강해지**려면** 운동을 해야 합니다.             | 想变健康的话要运动。       |
| 서울에 가**려면** KTX를 타세요.                  | 想去首尔的话坐KTX。        |
| TOPIK 6급을 받**으려면** 230점 이상 받아야 한다. | 想拿TOPIK 6级要230分以上。 |

---

## 六、-아/어야 ★★★★

### 接续

词干 + -아/어야

### 意思

只有...才... / 必须...才...

| 例句                        | 翻译                 |
| --------------------------- | -------------------- |
| 돈이 있**어야** 살 수 있다. | 只有有钱才能买。     |
| 열심히 해**야** 성공한다.   | 只有努力才能成功。   |
| 직접 먹어 봐**야** 안다.    | 只有亲自尝了才知道。 |
| 봐**야** 알지.              | 看了才知道。         |

### -아/어야 되다/하다 ★★★★★

| 例句                    | 翻译         |
| ----------------------- | ------------ |
| 숙제를 해**야 돼요**.   | 必须做作业。 |
| 일찍 일어나**야 해요**. | 必须早起。   |
| 약을 먹어**야 합니다**. | 必须吃药。   |

### -아/어야만 (强调"只有")

| 例句                           | 翻译               |
| ------------------------------ | ------------------ |
| 노력해**야만** 성공할 수 있다. | 只有努力才能成功。 |

## 七、-(으)ㄴ/는다면 (间接引用 + 条件) ★★★

| 例句                                            | 翻译                           |
| ----------------------------------------------- | ------------------------------ |
| 환경을 생각한**다면** 대중교통을 이용해야 한다. | 如果考虑环境就应该用公共交通。 |
| 건강을 원한**다면** 규칙적으로 운동해야 한다.   | 如果要健康就应该规律运动。     |

---

## 八、-기만 하면 ★★★

### 意思

只要一...就...

| 例句                            | 翻译                   |
| ------------------------------- | ---------------------- |
| 먹**기만 하면** 배가 아파요.    | 只要一吃肚子就疼。     |
| 시간**만 있으면** 갈 수 있어요. | 只要有时间就能去。     |
| 비**만 오면** 기분이 안 좋다.   | 只要一下雨心情就不好。 |

---

## 九、-(으)ㄹ 경우(에) ★★★

### 意思

在...的情况下（正式/书面）

| 例句                                  | 翻译                     |
| ------------------------------------- | ------------------------ |
| 문제가 생기**ㄹ 경우** 연락해 주세요. | 发生问题的情况下请联系。 |
| 위반할 **경우** 벌금을 물어야 한다.   | 违反的情况下要交罚款。   |

---

## 十、条件假设语尾总览

| 语尾         | 意思        | 使用场景         |
| ------------ | ----------- | ---------------- |
| -(으)면      | 如果        | 一般条件(最常用) |
| -거든        | 如果真的    | 不确定/万一      |
| -다면        | 假如        | 虚拟假设         |
| -았/었더라면 | 要是当时    | 过去虚拟(后悔)   |
| -(으)려면    | 想要...的话 | 目的条件         |
| -아/어야     | 只有才/必须 | 必要条件         |
| -기만 하면   | 只要一就    | 充分条件         |
| -(으)ㄹ 경우 | 在...情况下 | 正式/书面        |

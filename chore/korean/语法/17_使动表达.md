# 使动表达 (사동 표현)

> 使动 = 让/使别人做某事
> 韩语有两种使动方式：接尾词使动、-게 하다

---

## 一、接尾词使动 ★★★★

### 规则

在动词/形容词词干后加 **-이-, -히-, -리-, -기-, -우-, -추-, -키-** 变成使动动词

### -이- 使动

| 原动词     | 使动词     | 例句                              |
| ---------- | ---------- | --------------------------------- |
| 먹다(吃)   | 먹**이**다 | 아이에게 밥을 먹인다 (给孩子喂饭) |
| 보다(看)   | 보**이**다 | 사진을 보인다 (给看照片)          |
| 죽다(死)   | 죽**이**다 | 벌레를 죽인다 (杀虫子)            |
| 속다(被骗) | 속**이**다 | 사람을 속인다 (欺骗人)            |
| 녹다(融)   | 녹**이**다 | 얼음을 녹인다 (融化冰)            |

### -히- 使动

| 原动词   | 使动词     | 例句                                |
| -------- | ---------- | ----------------------------------- |
| 읽다(读) | 읽**히**다 | 아이에게 책을 읽힌다 (让孩子读书)   |
| 앉다(坐) | 앉**히**다 | 아이를 의자에 앉힌다 (让孩子坐椅子) |
| 입다(穿) | 입**히**다 | 아이에게 옷을 입힌다 (给孩子穿衣)   |
| 넓다(宽) | 넓**히**다 | 시야를 넓힌다 (开阔视野)            |
| 좁다(窄) | 좁**히**다 | 범위를 좁힌다 (缩小范围)            |
| 맞다(对) | 맞**히**다 | 답을 맞힌다 (答对答案)              |

### -리- 使动

| 原动词   | 使动词       | 例句                     |
| -------- | ------------ | ------------------------ |
| 살다(活) | 살**리**다   | 환자를 살린다 (救活病人) |
| 알다(知) | 알**리**다   | 소식을 알린다 (通知消息) |
| 울다(哭) | 울**리**다   | 아이를 울린다 (让孩子哭) |
| 돌다(转) | 돌**리**다   | 머리를 돌린다 (转头)     |
| 날다(飞) | 날**리**다   | 연을 날린다 (放风筝)     |
| 걸다(挂) | 걸**리**다 ★ | (注意这个更常做被动)     |

### -기- 使动

| 原动词     | 使动词     | 例句                        |
| ---------- | ---------- | --------------------------- |
| 웃다(笑)   | 웃**기**다 | 사람을 웃긴다 (让人笑→搞笑) |
| 벗다(脱)   | 벗**기**다 | 옷을 벗긴다 (给脱衣服)      |
| 맡다(承担) | 맡**기**다 | 일을 맡긴다 (委托工作)      |
| 숨다(藏)   | 숨**기**다 | 비밀을 숨긴다 (隐藏秘密)    |
| 남다(留)   | 남**기**다 | 메시지를 남긴다 (留言)      |
| 감다(闭眼) | 감**기**다 | 눈을 감긴다                 |

### -우- 使动

| 原动词      | 使动词     | 例句                     |
| ----------- | ---------- | ------------------------ |
| 자다(睡)    | 재**우**다 | 아이를 재운다 (让孩子睡) |
| 타다(烧/坐) | 태**우**다 | 종이를 태운다 (烧纸)     |
| 깨다(醒)    | 깨**우**다 | 아이를 깨운다 (叫醒孩子) |
| 서다(站)    | 세**우**다 | 차를 세운다 (停车)       |

### -추- 使动

| 原动词   | 使动词     | 例句                       |
| -------- | ---------- | -------------------------- |
| 낮다(低) | 낮**추**다 | 목소리를 낮춘다 (压低声音) |
| 늦다(晚) | 늦**추**다 | 속도를 늦춘다 (减速)       |
| 맞다(合) | 맞**추**다 | 시계를 맞춘다 (对表)       |

### -키- 使动

| 原动词   | 使动词     | 例句                   |
| -------- | ---------- | ---------------------- |
| 크다(大) | 키**우**다 | 아이를 키운다 (养孩子) |
| 끓다(沸) | 끓**이**다 | 물을 끓인다 (烧水)     |

## 二、-게 하다 ★★★★★

### 接续

动词/形容词 + -게 하다

### 意思

让/使（某人）做...

| 例句                           | 翻译             |
| ------------------------------ | ---------------- |
| 학생들을 공부하**게 했다**.    | 让学生们学习了。 |
| 아이를 일찍 자**게 해요**.     | 让孩子早睡。     |
| 친구를 웃**게 했다**.          | 让朋友笑了。     |
| 사람들이 놀라**게 했다**.      | 使人们吃惊了。   |
| 걱정하지 않**게 해 주세요**.   | 请让我不担心。   |
| 환경을 깨끗하**게 해야 한다**. | 要使环境干净。   |

### 特征

- **所有动词/形容词**都可以用 -게 하다 构成使动
- 比接尾词使动**更万能**

---

## 三、接尾词使动 vs -게 하다 对比 ★★★

|          | 接尾词使动                    | -게 하다                         |
| -------- | ----------------------------- | -------------------------------- |
| 适用范围 | 固定的动词                    | 任何动词/形容词                  |
| 直接程度 | 更直接（亲手做）              | 间接（指示/允许）                |
| 例       | 아이를 **웃기다** (逗笑)      | 아이를 **웃게 하다** (让孩子笑)  |
| 例       | 아이에게 밥을 **먹이다** (喂) | 아이를 밥을 **먹게 하다** (让吃) |

### 语感差异

| 接尾词使动    | 含义           | -게 하다         | 含义           |
| ------------- | -------------- | ---------------- | -------------- |
| 아이를 입히다 | 亲手给穿       | 아이를 입게 하다 | 让(孩子自己)穿 |
| 아이를 먹이다 | 亲手喂         | 아이를 먹게 하다 | 让孩子吃       |
| 차를 세우다   | 停车(自己操作) | 차를 서게 하다   | 让车停下来     |

---

## 四、使动句的结构

### 基本结构

| 主语(使动者) | 对象              | 使动动词    |
| ------------ | ----------------- | ----------- |
| 엄마가       | 아이를 / 아이에게 | 옷을 입혔다 |
| (妈妈)       | (给孩子)          | (穿了衣服)  |

### 对象标记

| 标记  | 用于                 | 例子                                    |
| ----- | -------------------- | --------------------------------------- |
| 을/를 | 直接使动             | 아이**를** 재웠다 (让孩子睡了)          |
| 에게  | 间接使动（让某人做） | 아이**에게** 밥을 먹였다 (给孩子喂了饭) |

---

## 五、被动+使动 同形词 ★★★

有些词既是被动又是使动，要靠语境区分：

| 词     | 被动意思            | 使动意思            |
| ------ | ------------------- | ------------------- |
| 보이다 | 被看到(산이 보인다) | 给看(사진을 보이다) |
| 읽히다 | 被读(책이 읽힌다)   | 让读(책을 읽히다)   |
| 안기다 | 被抱(아이가 안기다) | 给抱(아이를 안기다) |

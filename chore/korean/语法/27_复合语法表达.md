# 复合语法表达 (복합 문법 표현)

> TOPIK II 中高级常见的由多个语法单位组合而成的固定表达。

---

## 一、-(으)ㄹ 수밖에 없다 (只能/不得不) ★★★★★

| 例句                               | 翻译               |
| ---------------------------------- | ------------------ |
| 인정할 **수밖에 없다**.            | 只能承认。         |
| 기다릴 **수밖에 없었다**.          | 只能等着。         |
| 그렇게 할 **수밖에 없었어요**.     | 只能那样做了。     |
| 포기할 **수밖에 없는** 상황이었다. | 是只能放弃的情况。 |

---

## 二、-(으)ㄹ 뿐만 아니라 (不仅…而且) ★★★★★

| 例句                                       | 翻译                       |
| ------------------------------------------ | -------------------------- |
| 한국어**뿐만 아니라** 일본어도 할 수 있다. | 不仅韩语，日语也会。       |
| 맛있을 **뿐만 아니라** 건강에도 좋다.      | 不仅好吃，对健康也好。     |
| 학생**뿐만 아니라** 직장인도 많이 온다.    | 不仅学生，上班族也来很多。 |

---

## 三、-는 데(에) (在…方面/为了…) ★★★★★

| 例句                                    | 翻译                       |
| --------------------------------------- | -------------------------- |
| 한국어를 배우**는 데** 얼마나 걸려요?   | 学韩语要多久？             |
| 이 일을 하**는 데** 도움이 필요해요.    | 做这件事需要帮助。         |
| 건강을 유지하**는 데** 운동이 중요하다. | 对维持健康来说运动很重要。 |
| 돈을 버**는 데** 관심이 없다.           | 对赚钱没有兴趣。           |

---

## 四、-는 셈이다 (算是…) ★★★★

| 例句                        | 翻译               |
| --------------------------- | ------------------ |
| 거의 다 한 **셈이다**.      | 算是几乎都做完了。 |
| 싸게 산 **셈이다**.         | 算是便宜买到的。   |
| 매일 운동하는 **셈이에요**. | 算是每天运动了。   |

### -(으)ㄴ/는 셈 치다 (就当作)

| 例句                     | 翻译           |
| ------------------------ | -------------- |
| 그냥 없는 **셈 치세요**. | 就当作没有吧。 |
| 모르는 **셈 칠게요**.    | 就当作不知道。 |

---

## 五、-기(가) 마련이다 / -는 법이다 (必然) ★★★

| 例句                                   | 翻译                   |
| -------------------------------------- | ---------------------- |
| 노력하면 성공하**기 마련이다**.        | 努力就必然成功。       |
| 늙으면 기억력이 떨어지**기 마련이다**. | 变老就必然记忆力下降。 |
| 열심히 하면 결과가 나오**는 법이다**.  | 努力了结果必然会出来。 |

---

## 六、-(으)ㄹ 수 있다/없다 (能/不能) ★★★★★

| 例句                         | 翻译           |
| ---------------------------- | -------------- |
| 한국어를 말할 **수 있어요**. | 会说韩语。     |
| 올 **수 없습니다**.          | 来不了。       |
| 그럴 **수도 있다**.          | 也有可能那样。 |

---

## 七、-(으)ㄹ 줄 알다/모르다 (会/不会 | 以为) ★★★★★

### (1) 技能：会/不会

| 例句                   | 翻译         |
| ---------------------- | ------------ |
| 수영할 **줄 알아요**.  | 会游泳。     |
| 운전할 **줄 몰라요**.  | 不会开车。   |
| 기타 칠 **줄 아세요**? | 会弹吉他吗？ |

### (2) 以为…（与事实相反）

| 例句                       | 翻译             |
| -------------------------- | ---------------- |
| 쉬울 **줄 알았다**.        | 以为会简单。     |
| 이렇게 추울 **줄 몰랐다**. | 没想到会这么冷。 |

---

## 八、-는 바람에 / -는 통에 (因为…导致) ★★★★

### -는 바람에 (因为突发→消极结果)

| 例句                                 | 翻译                 |
| ------------------------------------ | -------------------- |
| 비가 오**는 바람에** 소풍을 못 갔다. | 因为下雨没能去野餐。 |
| 늦게 일어나**는 바람에** 지각했다.   | 因为起晚了迟到了。   |

### -는 통에 (因为混乱→消极结果)

| 例句                                 | 翻译                 |
| ------------------------------------ | -------------------- |
| 시끄러운 **통에** 잠을 못 잤다.      | 因为吵闹没睡着。     |
| 갑자기 비가 오**는 통에** 다 젖었다. | 因为突然下雨全湿了。 |

---

## 九、-기는커녕 (别说…连…) ★★★★

| 例句                                 | 翻译                       |
| ------------------------------------ | -------------------------- |
| 감사**하기는커녕** 오히려 화를 냈다. | 别说感谢，反而生气了。     |
| 휴가**는커녕** 주말도 못 쉬었다.     | 别说休假，连周末都没休息。 |
| 도움**은커녕** 방해만 됐다.          | 别说帮忙，只是碍事。       |

---

## 十、-(으)ㄹ 뿐(이다) (只是…) ★★★★

| 例句                        | 翻译                   |
| --------------------------- | ---------------------- |
| 보고 있을 **뿐이다**.       | 只是看着而已。         |
| 최선을 다할 **뿐이다**.     | 只能尽全力了。         |
| 울 **뿐** 아무 말도 못했다. | 只是哭，什么话也没说。 |

---

## 十一、-(으)ㄴ/는 탓에 / 덕분에 (因为…/多亏…) ★★★★

### -탓에 (消极原因：因为…)

| 例句                                | 翻译                   |
| ----------------------------------- | ---------------------- |
| 비가 온 **탓에** 경기가 취소됐다.   | 因为下雨比赛取消了。   |
| 연습 부족**인 탓에** 실수가 많았다. | 因为练习不足失误很多。 |

### -덕분에 (积极原因：多亏…)

| 例句                           | 翻译               |
| ------------------------------ | ------------------ |
| 선생님 **덕분에** 합격했다.    | 多亏老师合格了。   |
| 좋은 날씨 **덕분에** 즐거웠다. | 多亏好天气很开心。 |

---

## 十二、-(으)ㄹ 만하다 (值得/足够) ★★★★

| 例句                      | 翻译             |
| ------------------------- | ---------------- |
| 이 영화는 볼 **만해요**.  | 这电影值得看。   |
| 참을 **만하다**.          | 忍得了/还能忍。  |
| 먹을 **만한** 식당이에요. | 是值得吃的餐厅。 |
| 읽을 **만한** 책이에요.   | 是值得读的书。   |

---

## 十三、-다가는 (再这样下去就) ★★★★

| 例句                                  | 翻译                   |
| ------------------------------------- | ---------------------- |
| 이렇게 하**다가는** 큰일 난다.        | 再这样的话要出大事。   |
| 공부 안 하**다가는** 시험에 떨어진다. | 再不学习就要考不过了。 |
| 무리하**다가는** 쓰러진다.            | 再勉强的话会倒下。     |

---

## 十四、-(으)ㄴ/는 대신(에) (代替/取而代之) ★★★★

| 例句                             | 翻译               |
| -------------------------------- | ------------------ |
| 고기 **대신** 채소를 먹자.       | 代替肉吃蔬菜吧。   |
| 비싼 **대신** 품질이 좋다.       | 虽然贵但质量好。   |
| 도움을 준 **대신** 밥을 사 줬다. | 代替帮忙请了吃饭。 |

---

## 十五、-ㄹ/을 겸(해서) (兼/顺便) ★★★

| 例句                         | 翻译               |
| ---------------------------- | ------------------ |
| 운동할 **겸** 산책을 나갔다. | 顺便运动去散步了。 |
| 구경할 **겸** 쇼핑도 했다.   | 顺便观光也购物了。 |

---

## 十六、-(으)ㄹ 리(가) 없다/있다 (不可能/有可能) ★★★★

| 例句                        | 翻译             |
| --------------------------- | ---------------- |
| 그럴 **리가 없다**.         | 不可能那样。     |
| 거짓말일 **리가 없어요**.   | 不可能是谎话。   |
| 그런 일이 있을 **리 없다**. | 不可能有那种事。 |

---

## 十七、-아/어서 그런지 (可能因为…所以) ★★★★

| 例句                                  | 翻译                         |
| ------------------------------------- | ---------------------------- |
| 날씨가 추워**서 그런지** 사람이 없다. | 可能因为天冷所以没人。       |
| 피곤해**서 그런지** 집중이 안 된다.   | 可能因为累所以无法集中。     |
| 연습을 많이 해**서 그런지** 잘한다.   | 可能因为练了很多所以做得好。 |

---

## 十八、-느니 차라리 (与其…不如) ★★★

| 例句                                 | 翻译                 |
| ------------------------------------ | -------------------- |
| 밥을 먹**느니 차라리** 굶겠다.       | 与其吃那个不如饿着。 |
| 후회하**느니 차라리** 시도해 보겠다. | 与其后悔不如试试。   |
| 이렇게 기다리**느니 차라리** 가겠다. | 与其这样等不如走了。 |

## 十九、其他高频复合表达

| 表达                  | 意思            | 例句                      |
| --------------------- | --------------- | ------------------------- |
| -(으)ㄹ 수 있도록     | 以便能…         | 참가할 수 있도록 준비하다 |
| -는 것은 물론(이고)   | 不用说…当然     | 학생은 물론 선생님도      |
| -(으)ㄹ지 모르다      | 不知道会不会…   | 비가 올지 모르겠다        |
| -에 불과하다          | 不过是…         | 시작에 불과하다           |
| -에 지나지 않다       | 只不过是…       | 변명에 지나지 않다        |
| -아/어 봤자           | 就算做了也…     | 해 봤자 소용없다          |
| -(으)ㄹ 따름이다      | 只是…而已       | 감사할 따름이다           |
| -고 말겠다            | 一定要…         | 성공하고 말겠다           |
| -(으)ㄹ 지경이다      | 到了…地步       | 쓰러질 지경이다           |
| -기 나름이다          | 取决于…         | 생각하기 나름이다         |
| -기 일쑤이다          | 动不动就        | 늦기 일쑤이다             |
| -는 만큼              | 因为…（相应地） | 노력하는 만큼 결과가 좋다 |
| -(으)ㄹ 겸 -(으)ㄹ 겸 | 一方面…另一方面 | 운동할 겸 산책할 겸       |
| -다 보니(까)          | 做着做着        | 하다 보니 잘하게 됐다     |
| -(으)ㄹ 법하다        | 有可能…/合理    | 그럴 법하다               |
| -는 한이 있더라도     | 即使…也         | 죽는 한이 있더라도        |

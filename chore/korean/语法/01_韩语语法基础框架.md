# 韩语语法基础框架

> 在学习具体语法之前，先理解韩语的整体语法结构

---

## 一、韩语的基本语序

### SOV 语序

韩语是 **主语(S) + 宾语(O) + 谓语(V)** 语序的语言。

| 中文       | 韩语                 | 结构                              |
| ---------- | -------------------- | --------------------------------- |
| 我吃饭     | 나는 밥을 먹는다     | S(나는) + O(밥을) + V(먹는다)     |
| 他读书     | 그는 책을 읽는다     | S(그는) + O(책을) + V(읽는다)     |
| 姐姐喝咖啡 | 언니는 커피를 마신다 | S(언니는) + O(커피를) + V(마신다) |

### 核心规则

1. **谓语永远在句末** — 这是韩语最铁的规则
2. **修饰语在被修饰语前面** — 예쁜 꽃 (漂亮的花)
3. **助词标记成分** — 语序可以调换，靠助词判断谁是主语/宾语

### 语序灵活性

| 标准语序               | 强调语序               | 说明                   |
| ---------------------- | ---------------------- | ---------------------- |
| 나는 밥을 먹는다       | 밥을 나는 먹는다       | 强调"饭"(不是别的)     |
| 철수가 영희를 좋아한다 | 영희를 철수가 좋아한다 | 强调"英希"(被喜欢的人) |

## 二、韩语的九大词类

| 词类   | 韩语名 | 能否活用 | 例子               |
| ------ | ------ | -------- | ------------------ |
| 名词   | 명사   | ✗        | 사람, 학교, 행복   |
| 代词   | 대명사 | ✗        | 나, 너, 그, 이것   |
| 数词   | 수사   | ✗        | 하나, 둘, 일, 이   |
| 动词   | 동사   | ✓        | 가다, 먹다, 읽다   |
| 形容词 | 형용사 | ✓        | 크다, 예쁘다, 좋다 |
| 冠形词 | 관형사 | ✗        | 새, 헌, 온, 모든   |
| 副词   | 부사   | ✗        | 매우, 빨리, 아직   |
| 感叹词 | 감탄사 | ✗        | 아, 어머, 글쎄     |
| 助词   | 조사   | ✗        | 이/가, 을/를, 에서 |

### 关键区分

#### 动词 vs 形容词

- **动词**：表示动作/行为 → 가다, 먹다, 공부하다
- **形容词**：表示状态/性质 → 크다, 작다, 아름답다
- **语法差异**：

| 区别点         | 动词            | 形容词            |
| -------------- | --------------- | ----------------- |
| 现在时冠形词形 | -는 (먹는 사람) | -(으)ㄴ (큰 사람) |
| -(으)시-       | 가시다 ✓        | 크시다 ✓          |
| -고 있다       | 먹고 있다 ✓     | ~~크고 있다~~ ✗   |
| 命令形         | 가세요 ✓        | ~~크세요~~ ✗      |
| 共动形         | 갑시다 ✓        | ~~큽시다~~ ✗      |

#### 이다 的特殊性

- **이다**（是）既不是纯动词也不是纯形容词
- 接在名词后面：학생이다 (是学生)
- 有自己独特的活用方式

## 三、句子的基本成分

### 必须成分

| 成分 | 标记助词          | 例子                   |
| ---- | ----------------- | ---------------------- |
| 主语 | 이/가, 은/는      | **학생이** 공부한다    |
| 谓语 | (动词/形容词活用) | 학생이 **공부한다**    |
| 宾语 | 을/를             | 학생이 **책을** 읽는다 |
| 补语 | 이/가             | 물이 **얼음이** 되었다 |

### 修饰成分

| 成分         | 位置   | 例子          |
| ------------ | ------ | ------------- |
| 冠形语(定语) | 名词前 | **예쁜** 꽃   |
| 副词语(状语) | 谓语前 | **빨리** 가다 |

### 独立成分

| 成分   | 例子                 |
| ------ | -------------------- |
| 感叹语 | **아**, 좋은 날씨다! |
| 呼语   | **철수야**, 이리 와! |

## 四、谓语的构造

韩语谓语 = **词干** + **语尾**

```
먹 + 는다 = 먹는다 (吃)
먹 + 었다 = 먹었다 (吃了)
먹 + 겠다 = 먹겠다 (要吃)
먹 + 고 싶다 = 먹고 싶다 (想吃)
```

### 语尾的层次结构

```
词干 + [先语末语尾] + [语末语尾]

例: 가 + 시 + 었 + 습니다
    词干  敬语  过去  终结语尾
    = 가셨습니다 (去了 - 敬语)
```

### 语尾的三大分类

| 分类         | 功能         | 例子                       |
| ------------ | ------------ | -------------------------- |
| **终结语尾** | 结束句子     | -ㅂ니다, -아/어요, -다     |
| **连接语尾** | 连接两个句子 | -고, -(으)면, -지만        |
| **转成语尾** | 改变词类     | -(으)ㄴ, -는, -기, -(으)ㅁ |

## 五、韩语句子的类型

### 按语气分

| 类型   | 终结语尾                | 例子             |
| ------ | ----------------------- | ---------------- |
| 陈述句 | -ㅂ니다/-아요/-다       | 날씨가 좋습니다. |
| 疑问句 | -ㅂ니까/-아요?/-니?     | 날씨가 좋습니까? |
| 命令句 | -(으)십시오/-아요/-아라 | 빨리 오십시오.   |
| 共动句 | -(으)ㅂ시다/-아요/-자   | 같이 갑시다.     |
| 感叹句 | -는구나/-네             | 날씨가 좋구나!   |

### 按结构分

| 类型   | 说明         | 例子                                |
| ------ | ------------ | ----------------------------------- |
| 单句   | 一个谓语     | 꽃이 예쁘다.                        |
| 复句   | 两个以上分句 | 비가 오**고** 바람이 분다.          |
| 包孕句 | 句中有句     | 나는 [그가 온다**는** 것을] 알았다. |

## 六、助词系统概览

助词附着在名词/代词后，标记语法关系。

| 类别     | 功能         | 主要助词                           |
| -------- | ------------ | ---------------------------------- |
| 格助词   | 标记句子成分 | 이/가, 을/를, 의, 에, 에서, (으)로 |
| 补助词   | 添加额外意义 | 은/는, 도, 만, 까지, 부터, 밖에    |
| 接续助词 | 连接名词     | 와/과, (이)랑, 하고                |

### 有收音 vs 无收音

很多助词有两种形式：

| 条件   | 主格 | 宾格 | 主题 | 连接 |
| ------ | ---- | ---- | ---- | ---- |
| 有收音 | 이   | 을   | 은   | 과   |
| 无收音 | 가   | 를   | 는   | 와   |

例：

- 학생**이** (有收音 ㅇ → 이)
- 나**가** (无收音 → 가)

## 七、活用的基本概念

**活用** = 动词/形容词的词干后接不同语尾，形态发生变化

### 基本活用（规则活用）

以 **먹다** (吃) 为例：

| 语尾类型      | 接续      | 结果   |
| ------------- | --------- | ------ |
| 以辅音开头    | 먹 + 고   | 먹고   |
| 以 아/어 开头 | 먹 + 어   | 먹어   |
| 以 (으) 开头  | 먹 + 으면 | 먹으면 |

以 **가다** (去) 为例：

| 语尾类型      | 接续    | 结果              |
| ------------- | ------- | ----------------- |
| 以辅音开头    | 가 + 고 | 가고              |
| 以 아/어 开头 | 가 + 아 | 가(아→ 가)        |
| 以 (으) 开头  | 가 + 면 | 가면 (无收音省으) |

### 母音和谐（元音调和）

接 -아/어 时：

- 词干最后元音是 **ㅏ, ㅗ** → 接 **-아**
- 其他元音 → 接 **-어**
- **하다** → **해** (하 + 여 → 해)

| 词干末元音 | 动词   | 活用             |
| ---------- | ------ | ---------------- |
| ㅏ         | 가다   | 가 + 아 → 가     |
| ㅗ         | 오다   | 오 + 아 → 와     |
| ㅓ         | 먹다   | 먹 + 어 → 먹어   |
| ㅜ         | 배우다 | 배우 + 어 → 배워 |
| ㅡ         | 쓰다   | 쓰 + 어 → 써     |
| ㅣ         | 마시다 | 마시 + 어 → 마셔 |
| ㅐ         | 보내다 | 보내 + 어 → 보내 |
| 하         | 하다   | 하 + 여 → 해     |

## 八、句子分析练习

### 例句1

**그 학생은 도서관에서 열심히 한국어를 공부합니다.**

| 成分       | 词       | 助词/语尾 | 功能           |
| ---------- | -------- | --------- | -------------- |
| 冠形语     | 그       | -         | 修饰 학생      |
| 主语       | 학생     | 은        | 主题           |
| 状语(场所) | 도서관   | 에서      | 在...（场所）  |
| 状语(方式) | 열심히   | -         | 副词（努力地） |
| 宾语       | 한국어   | 를        | 宾格           |
| 谓语       | 공부하다 | ㅂ니다    | 格式体终结     |

→ 那个学生在图书馆努力地学习韩语。

### 例句2

**어제 친구에게 선물을 줬는데 친구가 매우 기뻐했다.**

| 成分       | 词       | 助词/语尾          | 功能         |
| ---------- | -------- | ------------------ | ------------ |
| 状语(时间) | 어제     | -                  | 昨天         |
| 间接宾语   | 친구     | 에게               | 给...        |
| 直接宾语   | 선물     | 을                 | 宾格         |
| 谓语1      | 주다     | 줬는데 (过去+转折) | 给了...但    |
| 主语       | 친구     | 가                 | 主格         |
| 状语(程度) | 매우     | -                  | 副词（非常） |
| 谓语2      | 기뻐하다 | 기뻐했다 (过去)    | 高兴了       |

→ 昨天给朋友送了礼物，朋友非常高兴。

# 补助动词 (보조 동사)

> 韩语中补助动词接在本动词后面，补充或微调语义。结构为 **V-아/어 + 보조동사**。

---

## 一、-아/어 주다 (给…做) ★★★★★

表示为他人做某事（施恩/服务）

| 例句                    | 翻译           |
| ----------------------- | -------------- |
| 도와**줄게요**.         | 我来帮你。     |
| 설명해 **주세요**.      | 请给我解释。   |
| 길을 가르쳐 **주었다**. | 告诉了我路。   |
| 문을 열어 **줄까요**?   | 要帮你开门吗？ |

### -아/어 드리다 (谦虚：给尊长做)

| 例句                     | 翻译           |
| ------------------------ | -------------- |
| 제가 들어 **드릴게요**.  | 我来帮您拿。   |
| 안내해 **드리겠습니다**. | 我来给您引路。 |

---

## 二、-아/어 보다 (试试/曾经) ★★★★★

### (1) 尝试

| 例句              | 翻译         |
| ----------------- | ------------ |
| 한번 먹어 **봐**. | 试试吃吃看。 |
| 입어 **보세요**.  | 请试穿。     |
| 해 **봅시다**.    | 试试吧。     |

### (2) 经历（过去时）

| 例句                       | 翻译             |
| -------------------------- | ---------------- |
| 한국에 가 **봤어요**.      | 去过韩国。       |
| 이 영화 봐 **봤어요**?     | 看过这部电影吗？ |
| 살아 **보니까** 알게 됐다. | 活过了才知道。   |

---

## 三、-아/어 있다 (结果状态持续) ★★★★★

表示动作完成后的状态持续（多用于自动词）

| 例句                       | 翻译         |
| -------------------------- | ------------ |
| 문이 열려 **있다**.        | 门开着。     |
| 앉아 **있어요**.           | 坐着。       |
| 불이 켜져 **있다**.        | 灯开着。     |
| 1층에 서 **있었다**.       | 站在1楼。    |
| 벽에 그림이 걸려 **있다**. | 墙上挂着画。 |

### -고 있다 vs -아/어 있다

| -고 있다                         | -아/어 있다               |
| -------------------------------- | ------------------------- |
| 进行中                           | 结果状态                  |
| 밥을 먹**고 있다** (正在吃)      | 앉아 **있다** (坐着)      |
| 옷을 입**고 있다** (正在穿/穿着) | 문이 열려 **있다** (开着) |

⚠️ 입고 있다 既可表示"正在穿"也可表示"穿着"(状态)

---

## 四、-아/어 놓다 / -아/어 두다 (做好放着) ★★★★

### -아/어 놓다 (做好放着，完成→放置)

| 例句                     | 翻译                 |
| ------------------------ | -------------------- |
| 미리 예약해 **놓았다**.  | 提前预约好了。       |
| 창문을 열어 **놨다**.    | 把窗户打开了(放着)。 |
| 숙제를 다 해 **놨어요**. | 作业都做好了。       |

### -아/어 두다 (做好留着，为将来准备)

| 例句                      | 翻译                 |
| ------------------------- | -------------------- |
| 돈을 모아 **두었다**.     | 把钱存着了(为将来)。 |
| 자료를 준비해 **두세요**. | 请准备好资料(备用)。 |
| 메모해 **둘게요**.        | 我记下来(备用)。     |

### 놓다 vs 두다

| -아/어 놓다               | -아/어 두다                   |
| ------------------------- | ----------------------------- |
| 做好→状态                 | 做好→为将来                   |
| 强调"完成放在那里"        | 强调"目的性保留"              |
| 문을 열어 놓다 (开着不管) | 문을 열어 두다 (开着为了通风) |

---

## 五、-아/어 가다 / -아/어 오다 ★★★★

### -아/어 가다 (逐渐…下去，从现在到将来)

| 例句                       | 翻译                         |
| -------------------------- | ---------------------------- |
| 날씨가 추워져 **간다**.    | 天气越来越冷了(继续冷下去)。 |
| 점점 좋아져 **갈 거예요**. | 会越来越好的。               |
| 살아 **가다**.             | 活下去。                     |

### -아/어 오다 (一直…过来，从过去到现在)

| 例句                          | 翻译                     |
| ----------------------------- | ------------------------ |
| 지금까지 열심히 해 **왔다**.  | 到现在一直努力着。       |
| 날씨가 따뜻해져 **왔다**.     | 天气(到现在)一直变暖着。 |
| 한국어를 3년간 배워 **왔다**. | 学韩语学了3年了。        |

| -아/어 가다  | -아/어 오다    |
| ------------ | -------------- |
| 现在→未来    | 过去→现在      |
| 变化持续进行 | 变化持续到现在 |

---

## 六、-아/어 버리다 (做完了/遗憾/痛快) ★★★★

表示动作彻底完成，带有感情色彩（遗憾/轻松/痛快）

| 例句                     | 翻译             | 色彩      |
| ------------------------ | ---------------- | --------- |
| 다 먹어 **버렸다**.      | 全吃光了。       | 遗憾/痛快 |
| 잊어 **버렸다**.         | 忘掉了。         | 遗憾      |
| 다 팔아 **버릴 거예요**. | 要全卖掉。       | 痛快      |
| 떠나 **버렸다**.         | 走掉了。         | 遗憾      |
| 숙제를 다 해 **버렸다**. | 作业全做完了。   | 轻松      |
| 고백해 **버렸다**.       | (豁出去)表白了。 | 痛快      |

---

## 七、-아/어 내다 (做出来/完成) ★★★

强调努力后终于做到/完成

| 例句                              | 翻译         |
| --------------------------------- | ------------ |
| 끝까지 참아 **냈다**.             | 忍到了最后。 |
| 결국 해 **냈다**!                 | 最终做到了！ |
| 문제를 풀어 **냈다**.             | 解出了题。   |
| 살아 **남다** (다른어: 살아남다). | 活了下来。   |

---

## 八、-아/어 대다 (反复/不停地做) ★★★

带有"过度/反感"的语感

| 例句                | 翻译       |
| ------------------- | ---------- |
| 자꾸 먹어 **대다**. | 不停地吃。 |
| 떠들어 **대다**.    | 吵个不停。 |
| 울어 **대다**.      | 一直哭。   |
| 물어 **대다**.      | 不停地问。 |

---

## 九、-고 싶다 (想…) ★★★★★

| 例句             | 翻译            |
| ---------------- | --------------- |
| 먹**고 싶다**.   | 想吃。          |
| 가**고 싶어요**. | 想去。          |
| 보**고 싶었다**. | 想看了/想你了。 |

否定: -고 싶지 않다 = 不想...

| 例句                | 翻译     |
| ------------------- | -------- |
| 가고 **싶지 않다**. | 不想去。 |

---

## 十、-고 말다 (终于/到底还是) ★★★

### (1) 遗憾：最终还是…了

| 例句                        | 翻译                   |
| --------------------------- | ---------------------- |
| 비밀을 얘기하**고 말았다**. | 还是(不小心)说了秘密。 |
| 넘어지**고 말았다**.        | 还是摔倒了。           |

### (2) 决心：一定要…(-(으)ㄹ 것이다)

| 예문                       | 翻译         |
| -------------------------- | ------------ |
| 꼭 성공하**고 말겠다**.    | 一定要成功。 |
| 끝까지 해내**고 말 거야**. | 一定做到底。 |

---

## 十一、-아/어 지다 (变得) ★★★★★

### 形容词 + 아/어 지다 = 变得...

| 例句                        | 翻译             |
| --------------------------- | ---------------- |
| 날씨가 따뜻해**졌다**.      | 天气变暖了。     |
| 예뻐**지고 싶다**.          | 想变漂亮。       |
| 한국어 실력이 좋아**졌다**. | 韩语水平提高了。 |
| 점점 비싸**지고 있다**.     | 越来越贵了。     |

---

## 十二、-게 되다 (变得/到了…的地步) ★★★★★

表示自然发展、非意愿的变化

| 例句                            | 翻译                       |
| ------------------------------- | -------------------------- |
| 한국에 오**게 됐어요**.         | (自然发展到)来了韩国。     |
| 알**게 됐다**.                  | 知道了(自然得知)。         |
| 그 사람을 좋아하**게 됐다**.    | (不知不觉)喜欢上了那个人。 |
| 회사를 그만두**게 되었습니다**. | 辞了职(不得已)。           |

---

## 十三、기타 보조 동사 (其他补助动词)

### -아/어 먹다 (贬义：做了可恶的事)

| 例句                 | 翻译             |
| -------------------- | ---------------- |
| 다 때려 **먹었다**.  | 全打了(恶行)。   |
| 거짓말해 **먹었다**. | (可恶地)说谎了。 |

### -아/어 치우다 (迅速处理掉)

| 例句                  | 翻译               |
| --------------------- | ------------------ |
| 숙제를 해 **치웠다**. | 把作业赶紧做完了。 |
| 밥을 먹어 **치우자**. | 赶紧把饭吃掉吧。   |

### -아/어 가지고 (口语：做了之后/因为)

| 例句                        | 翻译             |
| --------------------------- | ---------------- |
| 사 **가지고** 왔다.         | 买了带来了。     |
| 비가 와 **가지고** 못 갔다. | 因为下雨没去成。 |

## 十四、补助动词总览

| 补助动词      | 意思         | 星级  |
| ------------- | ------------ | ----- |
| -아/어 주다   | 为他人做     | ★★★★★ |
| -아/어 보다   | 试试/经验    | ★★★★★ |
| -아/어 있다   | 状态持续     | ★★★★★ |
| -고 있다      | 进行         | ★★★★★ |
| -아/어 놓다   | 做好放着     | ★★★★  |
| -아/어 두다   | 做好备用     | ★★★★  |
| -아/어 가다   | 逐渐(→未来)  | ★★★★  |
| -아/어 오다   | 一直(→现在)  | ★★★★  |
| -아/어 버리다 | 彻底完成     | ★★★★  |
| -아/어 내다   | 做到/完成    | ★★★   |
| -아/어 대다   | 反复/不停    | ★★★   |
| -고 싶다      | 想做         | ★★★★★ |
| -고 말다      | 终于还是     | ★★★   |
| -아/어 지다   | 变得         | ★★★★★ |
| -게 되다      | 变得(非意愿) | ★★★★★ |
| -아/어 치우다 | 赶紧处理     | ★★    |
| -아/어 먹다   | 贬义做       | ★★    |

# 书面体与高级表达 (문어체·고급 표현)

> TOPIK II 읽기·쓰기에 자주 나오는 격식체/문어체 표현

---

## 一、书面体终结语尾 ★★★★★

### 1. 陈述: -ㄴ다/는다, -다

| 口语       | 书面         | 翻译   |
| ---------- | ------------ | ------ |
| 가요       | 간**다**     | 去     |
| 먹어요     | 먹는**다**   | 吃     |
| 좋아요     | 좋**다**     | 好     |
| 학생이에요 | 학생이**다** | 是学生 |

### 2. 疑问: -(으)ㄴ가, -는가

| 口语             | 书面               | 翻译           |
| ---------------- | ------------------ | -------------- |
| 좋아요?          | 좋은**가**?        | 好吗？         |
| 아는 사람이에요? | 아는 사람인**가**? | 是认识的人吗？ |
| 가능해요?        | 가능한**가**?      | 可能吗？       |

### 3. -(으)ㄹ 것이다 (将要)

| 口语      | 书面          |
| --------- | ------------- |
| 할 거예요 | 할 **것이다** |
| 될 거예요 | 될 **것이다** |

## 二、书面体连接语尾 ★★★★★

### -(으)며 (而且/并且) = -고 的书面体

| 例句                                      | 翻译                               |
| ----------------------------------------- | ---------------------------------- |
| 서울은 한국의 수도**이며** 최대 도시이다. | 首尔是韩国的首都，也是最大的城市。 |
| 아름다우**며** 웅장하다.                  | 美丽且壮观。                       |
| 읽**으며** 생각하다.                      | 边读边想。                         |

### -(으)므로 (因为) = -기 때문에 的书面体

| 例句                                 | 翻译               |
| ------------------------------------ | ------------------ |
| 비가 오**므로** 우산을 가져가십시오. | 因为下雨请带伞。   |
| 위험하**므로** 접근하지 마십시오.    | 因为危险请勿接近。 |

### -(으)나 (但是) = -지만 的书面体

| 例句                         | 翻译                 |
| ---------------------------- | -------------------- |
| 노력하였**으나** 실패하였다. | 虽然努力了但失败了。 |
| 작**으나** 강하다.           | 虽小但强。           |

### -되 (但是，带条件限制)

| 例句                                 | 翻译                   |
| ------------------------------------ | ---------------------- |
| 참가는 가능하**되** 신청이 필요하다. | 可以参加，但需要申请。 |
| 사용할 수 있**되** 주의해야 한다.    | 可以使用，但要注意。   |

### -(으)ㄹ뿐더러 (不仅…而且) 书面

| 例句                          | 翻译               |
| ----------------------------- | ------------------ |
| 똑똒할**뿐더러** 성격도 좋다. | 不仅聪明性格也好。 |

## 三、高级连接表达

### 1. -(으)ㄴ/는 바 (根据…/所以…) ★★★★

| 例句                                       | 翻译                       |
| ------------------------------------------ | -------------------------- |
| 조사한 **바에 의하면** 문제가 없다.        | 根据调查没有问题。         |
| 알려진 **바와 같이** 이번 사건은 심각하다. | 正如已知的，此事件很严重。 |
| 경험한 **바로는** 그 방법이 효과적이다.    | 根据经验那个方法有效。     |

### 2. -는 데(에) 있어(서) (在…方面) ★★★★

| 例句                                             | 翻译                         |
| ------------------------------------------------ | ---------------------------- |
| 건강을 유지하**는 데 있어** 운동이 중요하다.     | 在保持健康方面运动很重要。   |
| 문제를 해결하**는 데 있어서** 소통이 필수적이다. | 在解决问题方面沟通是必须的。 |

### 3. -(으)ㄴ/는 한 (只要…/在…范围内) ★★★

| 例句                               | 翻译                   |
| ---------------------------------- | ---------------------- |
| 가능한 **한** 빨리 오세요.         | 尽可能快来。           |
| 내가 아**는 한** 그런 일은 없었다. | 据我所知没有那样的事。 |
| 살아 있**는 한** 희망이 있다.      | 只要活着就有希望。     |

### 4. -노라면 (做着做着…) ★★★

持续做就会…（文学色彩）

| 例句                                  | 翻译                   |
| ------------------------------------- | ---------------------- |
| 기다리**노라면** 좋은 날이 올 것이다. | 等着等着好日子会来的。 |
| 살아가**노라면** 힘든 일도 있다.      | 活着活着也有辛苦的事。 |

### 5. -(으)ㄹ진대 (既然…) ★★

| 例句                                       | 翻译                       |
| ------------------------------------------ | -------------------------- |
| 이미 시작한 이상 끝까지 해야 할 **것인즉** | 既然已经开始了就该做到最后 |

### 6. -건대 (个人看法：依我看) ★★

| 例句                                    | 翻译               |
| --------------------------------------- | ------------------ |
| 제가 생각**건대** 이 방법이 좋겠습니다. | 依我看这个方法好。 |

## 四、正式/公文常用句式

### 1. 명사 + (으)로 인해(서) (由于…) ★★★★

| 例句                                  | 翻译                     |
| ------------------------------------- | ------------------------ |
| 태풍**으로 인해** 수업이 취소되었다.  | 由于台风课程取消了。     |
| 코로나**로 인해** 많은 변화가 생겼다. | 由于新冠产生了很多变化。 |

### 2. 명사 + 에 따라(서) (根据…/随着…) ★★★★

| 例句                             | 翻译               |
| -------------------------------- | ------------------ |
| 상황**에 따라** 다르다.          | 根据情况不同。     |
| 결과**에 따라** 결정하겠다.      | 将根据结果决定。   |
| 시간**에 따라** 가격이 변합니다. | 随着时间价格变化。 |

### 3. 명사 + 에 대해(서) (关于…) ★★★★★

| 例句                                 | 翻译                   |
| ------------------------------------ | ---------------------- |
| 이 문제**에 대해** 논의합시다.       | 关于这个问题讨论吧。   |
| 한국 문화**에 대해** 발표하겠습니다. | 关于韩国文化进行发表。 |

### 4. 명사 + 에 비해(서) (相比…) ★★★★

| 例句                                | 翻译                       |
| ----------------------------------- | -------------------------- |
| 작년**에 비해** 올해 물가가 올랐다. | 与去年相比今年物价上涨了。 |
| 나**에 비해** 그 친구는 키가 크다.  | 跟我比那个朋友个子高。     |

### 5. 명사 + (으)로 말미암아 (由于) 高级书面 ★★

| 例句                                      | 翻译                 |
| ----------------------------------------- | -------------------- |
| 전쟁**으로 말미암아** 많은 사람이 죽었다. | 由于战争很多人死了。 |

### 6. -기에 (因为…所以) ★★★

| 例句                           | 翻译                 |
| ------------------------------ | -------------------- |
| 날씨가 좋**기에** 산책을 갔다. | 因为天气好去散步了。 |
| 너무 비싸**기에** 안 샀다.     | 因为太贵了没买。     |

## 五、引用与转述的书面表达

### -다고 한다 → -다고 하다 → -다 한다 ★★★★

| 完整            | 缩略          | 翻译   |
| --------------- | ------------- | ------ |
| 좋**다고 한다** | 좋**다 한다** | 据说好 |
| 간**다고 한다** | 간**다 한다** | 据说去 |

### -(이)라고 하다 → -(이)라 하다

| 例句                        | 翻译             |
| --------------------------- | ---------------- |
| 이것을 '한글'**이라 한다**. | 这个叫做"韩文"。 |

## 六、-는 것으로 나타나다/드러나다/밝혀지다 ★★★

新闻/论文常用：经调查/研究显示

| 例句                                      | 翻译                     |
| ----------------------------------------- | ------------------------ |
| 흡연이 건강에 해로운 **것으로 나타났다**. | (研究)显示吸烟有害健康。 |
| 원인이 시스템 오류인 **것으로 밝혀졌다**. | 查明原因是系统错误。     |
| 범인이 내부자인 **것으로 드러났다**.      | 发现犯人是内部人员。     |

---

## 七、-도록 하다 (使…/请…) 正式指示 ★★★★

| 例句                            | 翻译               |
| ------------------------------- | ------------------ |
| 내일까지 제출하**도록 하세요**. | 请于明天之前提交。 |
| 조용히 하**도록 하겠습니다**.   | 我会保持安静的。   |
| 다시 확인하**도록 하세요**.     | 请再确认一下。     |

---

## 八、-아/어야 할 것이다 (应该…) ★★★

书面中的义务/建议

| 例句                              | 翻译               |
| --------------------------------- | ------------------ |
| 더 노력해**야 할 것이다**.        | 应该更努力。       |
| 이 문제를 해결해**야 할 것이다**. | 应该解决这个问题。 |
| 원인을 분석해**야 할 것이다**.    | 应该分析原因。     |

| 副词          | 意思   | 例句                              |
| ------------- | ------ | --------------------------------- |
| 차라리        | 宁可   | 차라리 안 하겠다                  |
| 도리어/오히려 | 反而   | 도움이 도리어 방해가 됐다         |
| 하물며        | 何况   | 어른도 힘든데 하물며 아이는       |
| 더욱이        | 更何况 | 더욱이 날씨까지 나빴다            |
| 비록          | 虽然   | 비록 어렵지만 포기하지 않겠다     |
| 설령          | 即便   | 설령 실패하더라도 후회하지 않겠다 |
| 결코          | 绝不   | 결코 잊지 않겠다                  |
| 과연          | 果然   | 과연 소문대로였다                 |
| 진정          | 真正地 | 진정 사랑한다면                   |
| 실로          | 实在   | 실로 놀라운 결과다                |

## 十、书面体 vs 口语体 对照表

| 功能 | 口语                | 书面               |
| ---- | ------------------- | ------------------ |
| 并且 | -고                 | -(으)며            |
| 因为 | -아/어서 / -니까    | -(으)므로          |
| 但是 | -지만 / -는데       | -(으)나 / -되      |
| 如果 | -(으)면             | -(으)ㄹ 경우       |
| 不仅 | -(으)ㄹ 뿐만 아니라 | -(으)ㄹ뿐더러      |
| 关于 | -에 대해서          | -에 관하여         |
| 由于 | -때문에             | -(으)로 인하여     |
| 显示 | 보여 주다           | 나타나다/드러나다  |
| 将要 | -(으)ㄹ 거예요      | -(으)ㄹ 것이다     |
| 应该 | -아/어야 해요       | -아/어야 할 것이다 |
| 做吧 | -세요               | -도록 하십시오     |

# 时制与体貌

> 韩语的时制（过去/现在/将来）和体貌（进行/完成/经历）

---

## 一、现在时

### 1) 非格式体 -아/어요

| 动词/形容词 | 现在时   | 翻译 |
| ----------- | -------- | ---- |
| 가다        | 가요     | 去   |
| 먹다        | 먹어요   | 吃   |
| 공부하다    | 공부해요 | 学习 |
| 크다        | 커요     | 大   |
| 좋다        | 좋아요   | 好   |

### 2) 格式体 -(스)ㅂ니다

| 条件   | 形式             | 例子                       |
| ------ | ---------------- | -------------------------- |
| 有收音 | -습니다          | 먹습니다, 있습니다         |
| 无收音 | -ㅂ니다          | 갑니다, 합니다             |
| ㄹ收音 | (ㄹ脱落) -ㅂ니다 | 삽니다(살다), 압니다(알다) |

### 3) 半语 -아/어, -ㄴ다/는다

口语半语：가, 먹어, 해

书面/独白半语（动词）：

- 无收音 + **-ㄴ다**: 가**ㄴ다** → 간다
- 有收音 + **-는다**: 먹**는다** → 먹는다

书面半语（形容词）：直接 + **-다**

- 크**다**, 좋**다**, 예쁘**다**

## 二、过去时 ★★★★★

### 规则

词干 + **-았/었/했** + 语尾

| 条件            | 过去先语末 | 例子               |
| --------------- | ---------- | ------------------ |
| 阳性元音(ㅏ,ㅗ) | -았-       | 가 + 았다 → 갔다   |
| 阴性元音(其他)  | -었-       | 먹 + 었다 → 먹었다 |
| 하다            | -했-       | 하 + 였다 → 했다   |

### 各种语尾的过去时

| 语尾        | 现在       | 过去               |
| ----------- | ---------- | ------------------ |
| -아/어요    | 가요       | 갔어요             |
| -(스)ㅂ니다 | 갑니다     | 갔습니다           |
| -아/어서    | 가서       | 갔어서 (但有限制★) |
| -(으)ㄴ데   | 가는데(动) | 갔는데             |
| -다 (半语)  | 간다       | 갔다               |

### 过去时 + 연결어미 的限制 ★

| 语尾      | 能否加过去？          | 说明                                   |
| --------- | --------------------- | -------------------------------------- |
| -아/어서  | ✗ (大部分不加)        | ~~갔어서~~ → 가서 (前句时制由后句决定) |
| -(으)니까 | ✓                     | 갔으니까                               |
| -고       | ✓                     | 갔고                                   |
| -지만     | ✓                     | 갔지만                                 |
| -(으)면   | ✗ (需要用 -았/었으면) | 갔으면 (只用于虚拟/后悔)               |

## 三、将来时/推测 ★★★★

### 1) -(으)ㄹ 것이다 — 推测/计划

| 例句                       | 翻译         |
| -------------------------- | ------------ |
| 내일 비가 **올 거예요**.   | 明天会下雨。 |
| 저는 한국에 **갈 거예요**. | 我要去韩国。 |
| 그 사람은 **올 겁니다**.   | 那人会来的。 |

### 2) -겠- — 意志/推测

| 用法              | 例句                 | 翻译              |
| ----------------- | -------------------- | ----------------- |
| 说话者意志(1人称) | 제가 하**겠**습니다. | 我来做。          |
| 推测(2/3人称)     | 내일 춥**겠**다.     | 明天大概会冷。    |
| 客套话            | 알**겠**습니다.      | 我知道了/明白了。 |
|                   | 맛있**겠**다!        | 看起来好吃！      |

### -(으)ㄹ 것이다 vs -겠- 区别 ★★

|      | -(으)ㄹ 것이다              | -겠-                 |
| ---- | --------------------------- | -------------------- |
| 根据 | 外部信息/计划               | 直觉/当场判断        |
| 主语 | 1/2/3人称都可               | 1人称意志, 2/3推测   |
| 例   | (看天气预报) 비가 올 거예요 | (看天空) 비가 오겠다 |

## 四、过去完成 / 大过去 -았/었었- ★★★

表示过去的状态/动作，现在已经改变了（"曾经..."）

| 例句                          | 翻译             | 言外之意     |
| ----------------------------- | ---------------- | ------------ |
| 작년에 서울에 살**았었**어요. | 去年曾住在首尔。 | 现在不住了   |
| 옛날에는 예뻤**었**어요.      | 以前曾经漂亮。   | 现在不漂亮了 |
| 아까 비가 왔**었**어요.       | 刚才下过雨。     | 现在雨停了   |

### -았/었- vs -았/었었- 的区别

|          | -았/었-   | -았/었었-          |
| -------- | --------- | ------------------ |
| 살았다   | 住了/住过 | —                  |
| 살았었다 | —         | 曾经住过(现在不住) |

## 五、进行体 -고 있다 ★★★★★

### 基本用法

动词 + **-고 있다** = 正在...

| 例句                    | 翻译       |
| ----------------------- | ---------- |
| 밥을 먹**고 있다**.     | 正在吃饭。 |
| 책을 읽**고 있어요**.   | 正在看书。 |
| 비가 오**고 있습니다**. | 正在下雨。 |

### 持续状态用法

穿戴类动词 + -고 있다 = 穿着/戴着（状态）

| 例句                  | 翻译       |
| --------------------- | ---------- |
| 모자를 쓰**고 있다**. | 戴着帽子。 |
| 안경을 끼**고 있다**. | 戴着眼镜。 |
| 치마를 입**고 있다**. | 穿着裙子。 |

### 过去进行

-고 있**었**다 = 当时正在...

| 例句                    | 翻译           |
| ----------------------- | -------------- |
| 그때 자**고 있었어요**. | 那时正在睡觉。 |

## 六、完成/结果持续 -아/어 있다 ★★★★

### 基本用法

自动词(不及物动词) + **-아/어 있다** = ...着（结果状态持续）

| 例句                  | 翻译     |
| --------------------- | -------- |
| 문이 열려 **있다**.   | 门开着。 |
| 불이 꺼져 **있다**.   | 灯关着。 |
| 그는 앉아 **있다**.   | 他坐着。 |
| 꽃이 피어 **있다**.   | 花开着。 |
| 편지가 놓여 **있다**. | 信放着。 |

### -고 있다 vs -아/어 있다 ★★★

|      | -고 있다             | -아/어 있다           |
| ---- | -------------------- | --------------------- |
| 含义 | 动作进行中           | 动作完成后的状态持续  |
| 用法 | 他动词/自动词        | 主要自动词            |
| 앉다 | 앉고 있다 (正在坐下) | 앉아 있다 (坐着)      |
| 죽다 | 죽고 있다 (正在死去) | 죽어 있다 (死了/死着) |

---

## 七、经验 -(으)ㄴ 적이 있다/없다 ★★★★

| 例句                            | 翻译           |
| ------------------------------- | -------------- |
| 한국에 가**ㄴ 적이 있어요**.    | 去过韩国。     |
| 김치를 먹어 **본 적이 있어요**. | 吃过泡菜。     |
| 스키를 타 **본 적이 없어요**.   | 没滑过雪。     |
| 그런 말을 들은 **적이 없다**.   | 没听过那种话。 |

---

## 八、回想 -더- ★★★★

### -더라 / -더라고(요) — 回忆自己亲身经历/看到的

| 例句                          | 翻译                         |
| ----------------------------- | ---------------------------- |
| 한국 음식이 맛있**더라고요**. | (我发现)韩国菜好吃。         |
| 어제 날씨가 춥**더라**.       | 昨天天气冷(我亲身感受)。     |
| 그 영화가 재미있**더라고요**. | 那部电影有趣(我看了后觉得)。 |

### -았/었더라고(요) — 回忆过去看到的已完成状态

| 例句                                    | 翻译                             |
| --------------------------------------- | -------------------------------- |
| 도착했을 때 이미 끝나 있**었더라고요**. | 到的时候已经结束了(我当时看到)。 |

### -던 — 回想中的冠形词形（过去未完/反复）

| 例句               | 翻译                       |
| ------------------ | -------------------------- |
| 자주 가**던** 식당 | 常去的餐厅(过去经常)       |
| 읽**던** 책        | 在读的书(读了一半还没读完) |
| 좋아하**던** 노래  | 以前喜欢的歌               |

### -(으)ㄴ vs -던 区别 ★★★

|      | -(으)ㄴ                         | -던                                     |
| ---- | ------------------------------- | --------------------------------------- |
| 含义 | 完成的事实                      | 回忆/未完成                             |
| 예   | 어제 **읽은** 책 (昨天读完的书) | 어제 **읽던** 책 (昨天在读的书[没读完]) |
| 예   | 내가 **만난** 사람 (见过的人)   | 내가 **만나던** 사람 (以前常见的人)     |

## 九、时制总览表

| 时制            | 标记                 | 例子 (가다)  |
| --------------- | -------------------- | ------------ |
| 现在            | -(ㄴ/는)다, -아/어요 | 간다, 가요   |
| 过去            | -았/었-              | 갔다, 갔어요 |
| 大过去          | -았/었었-            | 갔었다       |
| 将来(计划)      | -(으)ㄹ 것이다       | 갈 거예요    |
| 将来(意志/推测) | -겠-                 | 가겠다       |
| 进行            | -고 있다             | 가고 있다    |
| 完成持续        | -아/어 있다          | 가 있다      |
| 经验            | -(으)ㄴ 적이 있다    | 간 적이 있다 |
| 回想            | -더-                 | 가더라       |

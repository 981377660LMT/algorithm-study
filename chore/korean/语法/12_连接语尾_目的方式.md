# 连接语尾：目的方式

> 表示"为了、以便、通过、按照"等目的和方式

---

## 一、-(으)러 ★★★★★

### 接续
动词 + -(으)러 + 移动动词(가다/오다/나가다 等)

### 意思
去/来做...（移动的目的）

| 例句 | 翻译 |
|------|------|
| 밥 먹**으러** 갈까요? | 去吃饭好吗？ |
| 영화 보**러** 가요. | 去看电影。 |
| 한국어 배우**러** 한국에 왔어요. | 来韩国学韩语。 |
| 친구 만나**러** 나갔다. | 出去见朋友了。 |
| 책 빌리**러** 도서관에 갔다. | 去图书馆借书了。 |

### 限制
- 后面**必须是移动动词**（가다, 오다, 다니다, 나가다, 들어오다 等）

---

## 二、-(으)려고 ★★★★★

### 接续
- 有收音 + -으려고 / 无收音 + -려고

### 意思
为了...（打算做...）

| 例句 | 翻译 |
|------|------|
| 한국어를 배우**려고** 학원에 다녀요. | 为了学韩语上补习班。 |
| 건강해지**려고** 매일 운동해요. | 为了变健康每天运动。 |
| 돈을 모으**려고** 아르바이트를 해요. | 为了攒钱做兼职。 |
| 일찍 자**려고** 했는데 못 잤어요. | 打算早睡但没睡成。 |

### 限制
- 前后主语必须相同
- 后句不能用命令/共动

---

## 三、-기 위해(서) ★★★★

### 接续
动词 + -기 위해(서)
名词 + 을/를 위해(서)

### 意思
为了...（比 -려고 更正式）

| 例句 | 翻译 |
|------|------|
| 건강**을 위해서** 운동한다. | 为了健康运动。 |
| 합격하**기 위해서** 열심히 공부했다. | 为了合格努力学习了。 |
| 환경을 보호하**기 위해** 노력해야 한다. | 为了保护环境要努力。 |
| 미래**를 위해** 지금 투자해야 한다. | 为了未来现在要投资。 |

---

## 四、-도록 ★★★★

### 接续
动词/形容词 + -도록

### 用法1：目的（以便...）

| 例句 | 翻译 |
|------|------|
| 잊지 않**도록** 메모하세요. | 以免忘记请记笔记。 |
| 늦지 않**도록** 일찍 출발하세요. | 为了不迟到请早出发。 |
| 모두 볼 수 있**도록** 크게 쓰세요. | 写大一点以便大家都能看到。 |

### 用法2：程度（到...的程度）

| 例句 | 翻译 |
|------|------|
| 배가 터지**도록** 먹었다. | 吃到肚子要撑破的程度。 |
| 귀가 아프**도록** 들었다. | 听到耳朵疼的程度。 |
| 밤이 새**도록** 이야기했다. | 聊到天亮。 |

### 用法3：使役（让...）

| 例句 | 翻译 |
|------|------|
| 학생들이 이해하**도록** 설명했다. | 解释了让学生们理解。 |
| 아이가 먹**도록** 했다. | 让孩子吃了。 |

---

## 五、-게 ★★★★

### 接续
形容词/动词 + -게

### 用法1：方式（...地）

| 例句 | 翻译 |
|------|------|
| 크**게** 말하세요. | 请大声说。 |
| 예쁘**게** 쓰세요. | 请写漂亮。 |
| 맛있**게** 드세요. | 请好好地吃。(客套语) |
| 빠르**게** 가다. | 快快地走。 |

### 用法2：使役（让...做...）

-게 하다 = 让/使...

| 例句 | 翻译 |
|------|------|
| 웃**게** 하다. | 让(人)笑。 |
| 울**게** 하다. | 让(人)哭。 |
| 걱정하지 않**게** 해 주세요. | 请让我不担心。 |

## 六、-(으)려고 vs -기 위해서 vs -도록 vs -(으)러 对比 ★★★

| | -(으)러 | -(으)려고 | -기 위해서 | -도록 |
|---|---|---|---|---|
| 意思 | 去做 | 为了 | 为了 | 以便/到...程度 |
| 正式度 | 口语 | 口语 | 书面 | 口语+书面 |
| 后句限制 | 移动动词 | 不能命令/共动 | 无特别限制 | 无 |
| 前后主语 | 相同 | 相同 | 可不同 | 可不同 |
| 例 | 먹으러 가다 | 먹으려고 준비하다 | 먹기 위해 요리하다 | 먹도록 만들다 |

---

## 七、-아/어다(가) ★★★

### 接续
动词 + -아/어다(가)

### 意思
做了A(转移)然后B（在一个地方做了然后带到另一个地方）

| 例句 | 翻译 |
|------|------|
| 책을 빌려**다가** 읽었다. | 借了书然后读了。 |
| 음식을 사**다가** 먹었다. | 买了食物然后吃了。 |
| 꽃을 사 **다** 선물했다. | 买了花然后送了。 |

---

## 八、대로 / -는 대로 ★★★

### 名词 + 대로 — 按照...

| 例句 | 翻译 |
|------|------|
| 계획**대로** 하자. | 按计划来吧。 |
| 약속**대로** 했다. | 按约定做了。 |

### 动词 + -는 대로 — 一...就.../按照...做

| 例句 | 翻译 |
|------|------|
| 도착하**는 대로** 연락하세요. | 一到达就联系我。 |
| 끝나**는 대로** 가겠습니다. | 一结束就过去。 |
| 선생님이 하시**는 대로** 따라 하세요. | 请照老师做的做。 |

---

## 九、-(으)ㄹ 겸 ★★★

### 接续
动词 + -(으)ㄹ 겸

### 意思
顺便...（一举两得的目的）

| 例句 | 翻译 |
|------|------|
| 운동할 **겸** 산책을 나갔다. | 顺便运动出去散步了。 |
| 구경할 **겸** 시장에 갔다. | 顺便逛逛去了市场。 |
| 인사할 **겸** 집들이 선물을 가져갔다. | 顺便问好带了乔迁礼物。 |

### A할 겸 B할 겸

| 例句 | 翻译 |
|------|------|
| 운동**할 겸** 산책**할 겸** 나갔다. | 既为了运动也为了散步出去了。 |

---

## 十、目的方式语尾总览

| 语尾 | 意思 | 关键特征 |
|------|------|---------|
| -(으)러 | 去做 | 后必须移动动词 |
| -(으)려고 | 为了 | 前后同主语 |
| -기 위해(서) | 为了(正式) | 可不同主语 |
| -도록 | 以便/程度/使 | 多功能 |
| -게 | ...地/让 | 副词化+使役 |
| -아/어다(가) | 做了带过来 | 空间转移 |
| -는 대로 | 一就/照做 | 时间+方式 |
| -(으)ㄹ 겸 | 顺便 | 兼顾目的 |

# 间接引语 (간접 화법)

> 间接引语 = 转述别人说的话
> "他说明天来" / "她问我去不去"

---

## 一、基本结构 ★★★★★

### 间接引语 = 引用内容 + 고 + 传达动词

传达动词：말하다(说), 하다(说), 묻다(问), 대답하다(回答), 생각하다(想) 等

---

## 二、陈述句的间接引语

### 动词

| 时制 | 结构           | 例子                        |
| ---- | -------------- | --------------------------- |
| 现在 | -ㄴ/는다고     | 간**다고** 했다 (说要去)    |
| 过去 | -았/었다고     | 갔**다고** 했다 (说去了)    |
| 将来 | -(으)ㄹ 거라고 | 갈 **거라고** 했다 (说要去) |

| 原话             | 间接引语                    |
| ---------------- | --------------------------- |
| "나는 학생이다." | (그는) 학생이**라고** 했다. |
| "매일 운동한다." | 매일 운동한**다고** 했다.   |
| "어제 갔다."     | 어제 갔**다고** 했다.       |
| "내일 올 거다."  | 내일 올 **거라고** 했다.    |

### 形容词

| 时制 | 结构       | 例子                       |
| ---- | ---------- | -------------------------- |
| 现在 | -다고      | 좋**다고** 했다 (说好)     |
| 过去 | -았/었다고 | 좋았**다고** 했다 (说好过) |

| 原话               | 间接引语                    |
| ------------------ | --------------------------- |
| "한국어가 어렵다." | 한국어가 어렵**다고** 했다. |
| "날씨가 좋았다."   | 날씨가 좋았**다고** 했다.   |

### 이다/아니다

| 原话             | 间接引语                  |
| ---------------- | ------------------------- |
| "학생이다."      | 학생이**라고** 했다.      |
| "학생이 아니다." | 학생이 아니**라고** 했다. |

---

## 三、疑问句的间接引语

### 结构

动词 -느냐고 / -(으)냐고
形容词 -(으)냐고
이다 -(이)냐고

| 原话             | 间接引语                    |
| ---------------- | --------------------------- |
| "어디 가냐?"     | 어디 가**냐고** 물었다.     |
| "밥 먹었니?"     | 밥 먹었**냐고** 물었다.     |
| "한국어 어렵니?" | 한국어 어렵**냐고** 물었다. |
| "학생이니?"      | 학생이**냐고** 물었다.      |
| "뭐 할 거니?"    | 뭐 할 **거냐고** 물었다.    |

### 口语缩写

-냐고 → -냐고 / -느냐고 → -느냐고 (书面更常见)

---

## 四、命令句的间接引语

### 结构

-(으)라고

| 原话         | 间接引语                |
| ------------ | ----------------------- |
| "빨리 와!"   | 빨리 오**라고** 했다.   |
| "공부해라."  | 공부하**라고** 했다.    |
| "조용히 해." | 조용히 하**라고** 했다. |
| "일찍 자."   | 일찍 자**라고** 했다.   |
| "먹어라."    | 먹**으라고** 했다.      |

### 敬语命令 → -달라고 (请求)

| 原话             | 间接引语                        |
| ---------------- | ------------------------------- |
| "도와주세요."    | 도와**달라고** 했다. (请求帮忙) |
| "기다려 주세요." | 기다려 **달라고** 했다.         |

## 五、共动句的间接引语

### 结构

-자고

| 原话           | 间接引语                |
| -------------- | ----------------------- |
| "같이 가자."   | 같이 가**자고** 했다.   |
| "밥 먹자."     | 밥 먹**자고** 했다.     |
| "내일 만나자." | 내일 만나**자고** 했다. |
| "게임하자."    | 게임하**자고** 했다.    |

## 六、间接引语总表 ★★★★★

| 句型       | 动词         | 形容词     | 이다      | 아니다      |
| ---------- | ------------ | ---------- | --------- | ----------- |
| 陈述(现在) | -ㄴ/는다고   | -다고      | -(이)라고 | -아니라고   |
| 陈述(过去) | -았/었다고   | -았/었다고 | -이었다고 | -아니었다고 |
| 疑问       | -느냐고/냐고 | -(으)냐고  | -(이)냐고 | -아니냐고   |
| 命令       | -(으)라고    | —          | —         | —           |
| 共动       | -자고        | —          | —         | —           |

## 七、缩写形式 ★★★★★

口语中间接引语常缩写：

### -다고 하다 → -다고 → -대(요)

| 完整        | 缩写       | 翻译     |
| ----------- | ---------- | -------- |
| 간다고 해요 | 간**대요** | (说)要去 |
| 좋다고 해요 | 좋**대요** | (说)好   |
| 갔다고 해요 | 갔**대요** | (说)去了 |

### -(이)라고 하다 → -(이)래(요)

| 完整            | 缩写           | 翻译       |
| --------------- | -------------- | ---------- |
| 학생이라고 해요 | 학생이**래요** | (说)是学生 |

### -냐고 하다 → -냬(요)

| 完整           | 缩写          | 翻译         |
| -------------- | ------------- | ------------ |
| 뭐 하냐고 해요 | 뭐 하**냬요** | (问)在做什么 |

### -(으)라고 하다 → -(으)래(요) (命令)

| 完整        | 缩写       | 翻译   |
| ----------- | ---------- | ------ |
| 오라고 해요 | 오**래요** | (让)来 |

### -자고 하다 → -재(요)

| 完整             | 缩写            | 翻译         |
| ---------------- | --------------- | ------------ |
| 같이 가자고 해요 | 같이 가**재요** | (提议)一起去 |

## 八、间接引语的其他传达动词 ★★★

| 动词           | 意思   | 例句                       |
| -------------- | ------ | -------------------------- |
| 말하다         | 说     | 내일 간다**고 말했다**     |
| 하다           | 说     | 좋다**고 했다**            |
| 묻다(물어보다) | 问     | 어디 가냐**고 물었다**     |
| 대답하다       | 回答   | 괜찮다**고 대답했다**      |
| 생각하다       | 想     | 좋다**고 생각한다**        |
| 듣다           | 听说   | 한국에 간다**고 들었다**   |
| 쓰다           | 写(着) | 합격이라**고 쓰여 있다**   |
| 부탁하다       | 拜托   | 도와 달라**고 부탁했다**   |
| 약속하다       | 约定   | 내일 만나자**고 약속했다** |
| 경고하다       | 警告   | 조심하라**고 경고했다**    |

## 九、间接引语常见句型 ★★★★

### -다고/라고 하는 N — 叫做N的...

| 例句                      | 翻译            |
| ------------------------- | --------------- |
| 김치**라고 하는** 음식    | 叫做泡菜的食物  |
| TOPIK이**라고 하는** 시험 | 叫做TOPIK的考试 |

### -다는 N — ...的N (冠形词形)

| 例句                  | 翻译         |
| --------------------- | ------------ |
| 간**다는** 소식       | 要去的消息   |
| 합격했**다는** 이야기 | 合格了的消息 |
| 좋**다는** 식당       | 据说好的餐厅 |

### -다니 — 居然说...（惊讶）

| 例句                   | 翻译               |
| ---------------------- | ------------------ |
| 거짓말이라**니**!      | 居然说是谎话！     |
| 그만둔**다니** 놀랐다. | 听说辞职了很惊讶。 |

## 十、人称代词的变化

间接引语中，人称需要调整：

| 直接引语             | 间接引语                         |
| -------------------- | -------------------------------- |
| "**나는** 학생이다." | **자기는** 학생이**라고** 했다.  |
| "**너** 어디 가?"    | **나에게** 어디 가**냐고** 했다. |
| "**내일** 올게."     | **다음 날** 오겠**다고** 했다.   |
| "**여기**가 좋다."   | **거기**가 좋**다고** 했다.      |
| "**오늘** 바쁘다."   | **그날** 바쁘**다고** 했다.      |

# 连接语尾：转折让步

> 表示"虽然、但是、即使、尽管"等转折/让步关系

---

## 一、-지만 ★★★★★

### 接续

词干 + -지만（直接接，不分收音）

### 意思

虽然...但是...（最基本的转折）

| 例句                                | 翻译                 |
| ----------------------------------- | -------------------- |
| 비싸**지만** 맛있어요.              | 虽然贵但好吃。       |
| 한국어는 어렵**지만** 재미있습니다. | 韩语虽然难但有趣。   |
| 바쁘**지만** 갈게요.                | 虽然忙但我会去。     |
| 노력했**지만** 실패했다.            | 虽然努力了但失败了。 |
| 알**지만** 말하지 않겠다.           | 虽然知道但不说。     |

---

## 二、-(으)ㄴ데 / -는데 ★★★★★

### 接续

| 词类   | 现在                          | 过去                      |
| ------ | ----------------------------- | ------------------------- |
| 动词   | -는데 (먹**는데**)            | -았/었는데 (먹었**는데**) |
| 形容词 | -(으)ㄴ데 (크**ㄴ데** → 큰데) | -았/었는데                |
| 이다   | -인데 (학생**인데**)          | -이었는데                 |

### 用法1：转折（但是）

| 例句                         | 翻译             |
| ---------------------------- | ---------------- |
| 열심히 했**는데** 안 됐어요. | 努力做了但没成。 |
| 비싼**데** 별로예요.         | 虽然贵但一般。   |

### 用法2：背景信息（引出话题）

| 例句                               | 翻译                     |
| ---------------------------------- | ------------------------ |
| 저는 학생인**데** 질문이 있어요.   | 我是学生，有个问题。     |
| 내일 시험인**데** 같이 공부할래요? | 明天考试，要一起学习吗？ |
| 지금 가**는데** 같이 갈래?         | 我现在要去，要一起吗？   |

### 用法3：意外/惊讶

| 例句                     | 翻译           |
| ------------------------ | -------------- |
| 한국어 잘하시**는데요**! | 韩语说得好呢！ |
| 생각보다 맛있**는데요**! | 比想的好吃呢！ |

---

## 三、-지만 vs -(으)ㄴ데 对比 ★★★

|          | -지만    | -(으)ㄴ/는데       |
| -------- | -------- | ------------------ |
| 核心     | 纯转折   | 转折 + 背景 + 铺垫 |
| 语感     | 直接对立 | 委婉/引出话题      |
| 多功能   | ✗        | ✓ (转折+背景+惊讶) |
| 使用频率 | 常用     | 超常用(口语之王)   |

---

## 四、-아/어도 ★★★★★

### 接续

词干 + -아/어도 (按阴阳元音)

### 意思

即使...也...（让步）

| 例句                              | 翻译               |
| --------------------------------- | ------------------ |
| 비가 와**도** 갈 거예요.          | 即使下雨也会去。   |
| 아무리 바빠**도** 밥은 먹어야 해. | 再怎么忙也要吃饭。 |
| 돈이 많아**도** 행복하지 않다.    | 即使钱多也不幸福。 |
| 열 번을 해**도** 안 돼요.         | 做十次也不行。     |
| 아무리 생각해**도** 모르겠다.     | 怎么想也不明白。   |

### 아무리 + -아/어도 ★★★

| 例句                                  | 翻译                 |
| ------------------------------------- | -------------------- |
| **아무리** 노력해**도** 안 되다.      | 再怎么努力也不行。   |
| **아무리** 먹어**도** 배가 안 부르다. | 再怎么吃也不饱。     |
| **아무리** 좋아**도** 참아야 한다.    | 再怎么喜欢也要忍住。 |

---

## 五、-더라도 ★★★★

### 接续

词干 + -더라도

### 意思

即使...也...（比 -아/어도 **更强烈/假设性更强**）

| 例句                                | 翻译                   |
| ----------------------------------- | ---------------------- |
| 실패하**더라도** 포기하지 마세요.   | 即使失败也不要放弃。   |
| 힘들**더라도** 끝까지 해야 합니다.  | 即使辛苦也要做到最后。 |
| 비가 오**더라도** 출발합시다.       | 即使下雨也出发吧。     |
| 시간이 걸리**더라도** 잘 해 보겠다. | 即使花时间也要好好做。 |

### -아/어도 vs -더라도 ★

|        | -아/어도  | -더라도     |
| ------ | --------- | ----------- |
| 语感   | 一般让步  | 更强烈/假设 |
| 常用于 | 一般情况  | 决心/激励   |
| 书面   | 口语+书面 | 更偏书面    |

---

## 六、-(으)ㄹ지라도 ★★

### 接续

词干 + -(으)ㄹ지라도

### 意思

即使（可能）...也...（书面语，假设+让步）

| 例句                              | 翻译                   |
| --------------------------------- | ---------------------- |
| 어렵**ㄹ지라도** 도전해 보겠다.   | 即使可能难也要挑战。   |
| 실패할**지라도** 후회하지 않겠다. | 即使可能失败也不后悔。 |

---

## 七、-(으)ㄹ지언정 ★★

### 接续

词干 + -(으)ㄹ지언정

### 意思

宁可...也不...（强烈的让步+对比）

| 例句                                    | 翻译                 |
| --------------------------------------- | -------------------- |
| 굶**을지언정** 남의 것을 빼앗지 않겠다. | 宁可饿也不抢别人的。 |
| 죽**을지언정** 비밀을 지키겠다.         | 宁可死也要守秘密。   |

---

## 八、-건만 / -거늘 / -되 ★★

### -건만 (虽然...但是... 书面+遗憾)

| 例句                                        | 翻译                       |
| ------------------------------------------- | -------------------------- |
| 열심히 공부했**건만** 성적이 오르지 않았다. | 虽然努力学了但成绩没提高。 |

### -거늘 (虽然...但是... 古语/文学)

| 例句                                         | 翻译                   |
| -------------------------------------------- | ---------------------- |
| 때가 이르렀**거늘** 아직 준비가 되지 않았다. | 时机已到但还没准备好。 |

### -되 (虽然...但有条件)

| 例句                           | 翻译                 |
| ------------------------------ | -------------------- |
| 가**되** 빨리 돌아와라.        | 去可以，但要快回来。 |
| 말하**되** 예의 바르게 하세요. | 说可以，但要有礼貌。 |

---

## 九、-(으)나 ★★★

### 接续

词干 + -(으)나

### 意思

虽然...但是...（书面体，和 -지만 同义但更正式）

| 例句                                    | 翻译                       |
| --------------------------------------- | -------------------------- |
| 노력하였**으나** 실패하였다.            | 虽然努力了但失败了。       |
| 여러 번 시도하였**으나** 결과는 같았다. | 虽然尝试了多次但结果一样。 |
| 알고는 있**으나** 실천이 어렵다.        | 虽然知道但实践很难。       |

---

## 十、-ㄴ/는 반면(에) ★★★

### 接续

动词 -는 반면 / 形容词 -(으)ㄴ 반면 / 名词 인 반면

### 意思

... 另一方面 ...（对比两个相反的方面）

| 例句                                   | 翻译                             |
| -------------------------------------- | -------------------------------- |
| 이 식당은 맛있**는 반면에** 비싸요.    | 这餐厅好吃但另一方面贵。         |
| 도시는 편리한 **반면** 공기가 안 좋다. | 城市方便但另一方面空气不好。     |
| 수입이 증가한 **반면** 지출도 늘었다.  | 收入增加但另一方面支出也增加了。 |

---

## 十一、-ㄴ/는 데 반해(서) ★★

### 意思

与...相反（更强调对比）

| 例句                                                   | 翻译                           |
| ------------------------------------------------------ | ------------------------------ |
| 작년에 비해 수출은 증가한 **데 반해** 수입은 감소했다. | 出口增加，与此相反进口减少了。 |

---

## 十二、转折让步语尾总览

| 语尾          | 意思           | 口语/书面 | 强度  |
| ------------- | -------------- | --------- | ----- |
| -지만         | 虽然但是       | 口语+书面 | ★★    |
| -(으)ㄴ/는데  | 但是/铺垫      | 口语      | ★★    |
| -아/어도      | 即使也         | 口语+书面 | ★★★   |
| -더라도       | 即使也(假设)   | 书面偏多  | ★★★★  |
| -(으)ㄹ지라도 | 即使也(强)     | 书面      | ★★★★  |
| -(으)ㄹ지언정 | 宁可也不       | 书面      | ★★★★★ |
| -(으)나       | 虽然但是(正式) | 纯书面    | ★★    |
| -건만         | 虽然但遗憾     | 纯书面    | ★★★   |
| -되           | 虽然但有条件   | 书面      | ★★    |
| -는 반면(에)  | 另一方面       | 书面      | ★★    |

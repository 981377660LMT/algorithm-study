# 被动表达 (피동 표현)

> 韩语的被动有三种方式：接尾词被动、-아/어지다、-게 되다

---

## 一、接尾词被动 ★★★★★

### 规则

在他动词词干后加 **-이-, -히-, -리-, -기-** 变成被动动词

### -이- 被动

| 主动       | 被动       | 主动例句    | 被动例句                   |
| ---------- | ---------- | ----------- | -------------------------- |
| 보다(看)   | 보**이**다 | 산을 본다   | 산이 보인다 (山被看到)     |
| 쓰다(用)   | 쓰**이**다 | 돈을 쓴다   | 돈이 쓰인다 (钱被使用)     |
| 놓다(放)   | 놓**이**다 | 책을 놓는다 | 책이 놓인다 (书被放着)     |
| 쌓다(堆)   | 쌓**이**다 | 눈을 쌓는다 | 눈이 쌓인다 (雪堆积)       |
| 바꾸다(换) | 바뀌다 ★   | —           | 계획이 바뀌었다 (计划变了) |
| 섞다(混)   | 섞**이**다 | —           | 물과 기름이 안 섞인다      |

### -히- 被动

| 主动     | 被动       | 例句                                    |
| -------- | ---------- | --------------------------------------- |
| 먹다(吃) | 먹**히**다 | 토끼가 여우에게 먹힌다 (兔子被狐狸吃)   |
| 잡다(抓) | 잡**히**다 | 도둑이 경찰에게 잡혔다 (小偷被警察抓了) |
| 읽다(读) | 읽**히**다 | 이 책은 많이 읽힌다 (这书被很多人读)    |
| 닫다(关) | 닫**히**다 | 문이 닫혔다 (门关上了)                  |
| 막다(堵) | 막**히**다 | 길이 막혔다 (路堵了)                    |
| 밟다(踩) | 밟**히**다 | 발이 밟혔다 (脚被踩了)                  |
| 뽑다(拔) | 뽑**히**다 | 이가 뽑혔다 (牙被拔了)                  |

### -리- 被动

| 主动     | 被动         | 例句                                 |
| -------- | ------------ | ------------------------------------ |
| 듣다(听) | 들**리**다   | 음악이 들린다 (听到音乐)             |
| 열다(开) | 열**리**다   | 문이 열렸다 (门开了)                 |
| 걸다(挂) | 걸**리**다   | 그림이 걸려 있다 (画挂着)            |
| 팔다(卖) | 팔**리**다   | 이 제품은 잘 팔린다 (这产品卖得好)   |
| 풀다(解) | 풀**리**다   | 문제가 풀렸다 (问题解决了)           |
| 밀다(推) | 밀**리**다   | 차가 밀렸다 (被车挤了→堵车了)        |
| 알다(知) | 알**리**다 ★ | 이것은 많이 알려져 있다 (这广为人知) |

> 알리다 更常作"告知"(使动)，被动常用 알려지다

### -기- 被动

| 主动       | 被动         | 例句                                    |
| ---------- | ------------ | --------------------------------------- |
| 안다(抱)   | 안**기**다   | 아이가 엄마에게 안겼다 (孩子被妈妈抱着) |
| 빼앗다(夺) | 빼앗**기**다 | 공을 빼앗겼다 (球被抢了)                |
| 쫓다(追)   | 쫓**기**다   | 시간에 쫓긴다 (被时间追赶)              |
| 끊다(断)   | 끊**기**다   | 전화가 끊겼다 (电话断了)                |

## 二、-아/어지다 ★★★★★

### 接续

形容词 + -아/어지다 = 变得...（状态变化）
动词 + -아/어지다 = 被...（被动）

### 形容词 + -아/어지다（变化）

| 例句                   | 翻译         |
| ---------------------- | ------------ |
| 날씨가 추워**졌다**.   | 天气变冷了。 |
| 한국어가 좋아**졌다**. | 韩语变好了。 |
| 예뻐**졌다**.          | 变漂亮了。   |
| 건강해**졌다**.        | 变健康了。   |
| 조용해**졌다**.        | 变安静了。   |
| 물가가 비싸**졌다**.   | 物价变贵了。 |

### 动词 + -아/어지다（被动）

| 例句                            | 翻译                     |
| ------------------------------- | ------------------------ |
| 문제가 해결되다 / 풀어**지다**. | 问题被解决了。           |
| 유리가 깨**졌다**.              | 玻璃被打碎了。           |
| 약속이 지켜**졌다**.            | 约定被遵守了。           |
| 가방이 찢어**졌다**.            | 包被撕破了。             |
| 이름이 알려**졌다**.            | 名字被知道了。(变得有名) |

## 三、-게 되다 ★★★★★

### 接续

动词/形容词 + -게 되다

### 意思

变得.../（自然而然地）开始.../被安排...

| 例句                        | 翻译                     |
| --------------------------- | ------------------------ |
| 한국에 가**게 되었다**.     | (机缘巧合)去了韩国。     |
| 한국어를 배우**게 되었다**. | (自然而然)开始学韩语了。 |
| 그 소식을 알**게 되었다**.  | 知道了那个消息。         |
| 이해하**게 되었다**.        | 变得理解了。             |
| 좋아하**게 되었다**.        | 变得喜欢了。             |
| 결혼하**게 되었습니다**.    | 要结婚了(宣布)。         |

### 特征

- 强调**不是主动选择的结果**，而是自然而然的发展
- 比直接用主动态更委婉

## 四、被动句的结构 ★★★

### 主动 → 被动 转换

| 主动句                        | 被动句                  |
| ----------------------------- | ----------------------- |
| 경찰이 도둑을 잡았다.         | 도둑이 경찰에게 잡혔다. |
| (警察抓了小偷)                | (小偷被警察抓了)        |
| 사람들이 이 책을 많이 읽는다. | 이 책은 많이 읽힌다.    |
| (人们常读这本书)              | (这本书被常读)          |
| 바람이 문을 열었다.           | 문이 바람에 열렸다.     |
| (风开了门)                    | (门被风打开了)          |

### 被动句中的施动者标记

| 标记   | 用于          | 例子                        |
| ------ | ------------- | --------------------------- |
| 에게   | 人/动物       | 도둑이 경찰**에게** 잡혔다  |
| 에     | 自然力/无生物 | 문이 바람**에** 열렸다      |
| (으)로 | 工具/手段     | 이 건물은 돌**로** 지어졌다 |

## 五、三种被动方式对比

|           | 接尾词被动      | -아/어지다              | -게 되다      |
| --------- | --------------- | ----------------------- | ------------- |
| 接续      | 特定动词        | 形容词(变化)/动词(被动) | 动词+形容词   |
| 特点      | 固定搭配需记忆  | 最灵活                  | 强调非主动    |
| 例        | 문이 닫**히**다 | 날씨가 추워**지다**     | 알**게 되다** |
| 口语/书面 | 都可            | 都可                    | 都可          |

### 注意：不要双重被动 ★★

- ~~잡혀지다~~ ✗ (已经是被动不要再加 -지다)
- ~~읽혀지다~~ ✗
- 잡히다 ✓
- 읽히다 ✓

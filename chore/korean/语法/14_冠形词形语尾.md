# 冠形词形语尾 (관형사형 어미)

> 冠形词形语尾 = 让动词/形容词变成定语修饰名词
> 中文的"吃的饭、漂亮的花" → 韩语用冠形词形语尾实现

---

## 一、核心对照表 ★★★★★

|               | 过去            | 现在       | 将来/推测  |
| ------------- | --------------- | ---------- | ---------- |
| **动词**      | -(으)ㄴ         | -는        | -(으)ㄹ    |
| **形容词**    | -던             | -(으)ㄴ    | -(으)ㄹ    |
| **이다**      | -이던 / -이었던 | -인        | -일        |
| **있다/없다** | -있던/없던      | -있는/없는 | -있을/없을 |

---

## 二、动词的冠形词形

### 过去 -(으)ㄴ — 已完成的动作

| 例句               | 翻译         |
| ------------------ | ------------ |
| 어제 **먹은** 음식 | 昨天吃的食物 |
| 내가 **읽은** 책   | 我读过的书   |
| 작년에 **간** 곳   | 去年去的地方 |
| 아까 **만난** 사람 | 刚才见的人   |
| 이미 **끝난** 일   | 已经结束的事 |

### 现在 -는 — 正在进行/习惯性

| 例句                   | 翻译         |
| ---------------------- | ------------ |
| 지금 **먹는** 음식     | 正在吃的食物 |
| 매일 **읽는** 신문     | 每天读的报纸 |
| 여기서 **사는** 사람   | 住在这里的人 |
| 내가 **좋아하는** 노래 | 我喜欢的歌   |
| 학교에 **다니는** 학생 | 上学的学生   |

### 将来 -(으)ㄹ — 将要做的/推测

| 例句               | 翻译           |
| ------------------ | -------------- |
| 내일 **먹을** 음식 | 明天要吃的食物 |
| 앞으로 **읽을** 책 | 接下来要读的书 |
| **갈** 곳이 없다   | 没有要去的地方 |
| **할** 일이 많다   | 要做的事很多   |
| **올** 수 있다     | 能来           |

## 三、形容词的冠形词形

### 现在 -(으)ㄴ — 当前状态

| 例句                | 翻译                  |
| ------------------- | --------------------- |
| **큰** 집           | 大房子                |
| **예쁜** 꽃         | 漂亮的花              |
| **좋은** 날씨       | 好天气                |
| **비싼** 옷         | 贵衣服                |
| **맛있는** 음식 ★   | 好吃的食物 (있다特殊) |
| **재미없는** 영화 ★ | 无聊的电影 (없다特殊) |

### 将来 -(으)ㄹ — 推测/可能

| 例句               | 翻译     |
| ------------------ | -------- |
| **좋을** 것 같다   | 好像会好 |
| **비쌀** 것 같다   | 好像会贵 |
| **어려울** 수 있다 | 可能会难 |

---

## 四、-던 — 回想/未完成 ★★★★

### 过去的回忆/反复/未完成

| 例句                        | 翻译             | 含义       |
| --------------------------- | ---------------- | ---------- |
| 자주 가**던** 식당          | 常去的餐厅       | 过去反复   |
| 읽**던** 책                 | 在读的书         | 过去未完成 |
| 좋아하**던** 사람           | 以前喜欢的人     | 过去回忆   |
| 살**던** 집                 | 以前住的家       | 过去回忆   |
| 어렸을 때 좋아하**던** 음식 | 小时候喜欢的食物 | 过去回忆   |

### -았/었던 — 过去完成后回忆

| 例句                   | 翻译           | 含义        |
| ---------------------- | -------------- | ----------- |
| 어제 **갔던** 식당     | 昨天去过的餐厅 | 已完成+回忆 |
| 작년에 **읽었던** 책   | 去年读过的书   | 已完成+回忆 |
| 내가 **좋아했던** 사람 | 我曾喜欢过的人 | 已结束+回忆 |

### -(으)ㄴ vs -던 vs -았/었던 辨析 ★★★★★

|               | -(으)ㄴ    | -던             | -았/었던     |
| ------------- | ---------- | --------------- | ------------ |
| 含义          | 完成的事实 | 未完成/反复回忆 | 完成后的回忆 |
| **읽은** 책   | 读完的书   | —               | —            |
| **읽던** 책   | —          | 读了一半的书    | —            |
| **읽었던** 책 | —          | —               | 以前读完的书 |

## 五、常见搭配 ★★★★★

### -(으)ㄹ 수 있다/없다 — 能/不能

| 例句                     | 翻译       |
| ------------------------ | ---------- |
| 한국어를 할 **수 있다**. | 能说韩语。 |
| 갈 **수 없다**.          | 不能去。   |
| 이해할 **수 있어요**?    | 能理解吗？ |

### -(으)ㄹ 것이다 — 将来/推测

| 例句                     | 翻译         |
| ------------------------ | ------------ |
| 내일 비가 올 **것이다**. | 明天会下雨。 |
| 곧 끝날 **거예요**.      | 马上会结束。 |

### -(으)ㄹ 줄 알다/모르다 — 会/不会（技能）

| 例句                     | 翻译               |
| ------------------------ | ------------------ |
| 운전할 **줄 알아요**?    | 会开车吗？         |
| 요리할 **줄 몰라요**.    | 不会做菜。         |
| 이렇게 될 **줄 몰랐다**. | 没想到会变成这样。 |

### -(으)ㄴ/는 것 — 名词化（...的事/东西）

| 例句                      | 翻译                 |
| ------------------------- | -------------------- |
| 먹는 **것**을 좋아한다.   | 喜欢吃(的事)。       |
| 중요한 **것**은 노력이다. | 重要的(东西)是努力。 |
| 내가 한 **것**이 맞다.    | 我做的是对的。       |

### -(으)ㄹ 때 — ...的时候

| 例句               | 翻译         |
| ------------------ | ------------ |
| 학생이었을 **때**  | 当学生的时候 |
| 시간이 있을 **때** | 有时间的时候 |
| 어릴 **때**        | 小时候       |

### -(으)ㄴ/는 편이다 — 算是…的一方

| 例句                        | 翻译               |
| --------------------------- | ------------------ |
| 한국어를 잘하는 **편이다**. | 算是韩语说得好的。 |
| 키가 큰 **편이에요**.       | 算是高的。         |

## 六、不规则活用在冠形词形中的表现

| 原形         | -(으)ㄴ | -는    | -(으)ㄹ |
| ------------ | ------- | ------ | ------- |
| 듣다(ㄷ不)   | 들은    | 듣는   | 들을    |
| 춥다(ㅂ不)   | 추운    | —      | 추울    |
| 짓다(ㅅ不)   | 지은    | 짓는   | 지을    |
| 살다(ㄹ不)   | 산      | 사는   | 살      |
| 모르다(르不) | 모른    | 모르는 | 모를    |
| 그렇다(ㅎ不) | 그런    | —      | 그럴    |
| 쓰다(으不)   | 쓴      | 쓰는   | 쓸      |
| 하다(여不)   | 한      | 하는   | 할      |
| 있다(特殊)   | 있던    | 있는 ★ | 있을    |
| 없다(特殊)   | 없던    | 없는 ★ | 없을    |

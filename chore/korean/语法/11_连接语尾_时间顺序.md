# 连接语尾：时间顺序

> 表示"然后、之后、一边、之前、刚一就"等时间关系

---

## 一、-고 ★★★★★

### 接续
词干 + -고

### 意思
1. **并列** — ...和...
2. **顺序** — ...然后...

### 并列用法

| 例句 | 翻译 |
|------|------|
| 이 옷은 예쁘**고** 싸요. | 这衣服又漂亮又便宜。 |
| 나는 커피를 마시**고** 그는 차를 마신다. | 我喝咖啡他喝茶。 |

### 顺序用法

| 例句 | 翻译 |
|------|------|
| 밥을 먹**고** 학교에 갔다. | 吃了饭然后去了学校。 |
| 손을 씻**고** 밥을 먹어요. | 洗手然后吃饭。 |
| 숙제를 하**고** 놀았다. | 做了作业然后玩了。 |

---

## 二、-고 나서 ★★★★

### 接续
动词 + -고 나서

### 意思
做完...之后（强调前一动作结束后）

| 例句 | 翻译 |
|------|------|
| 밥을 먹**고 나서** 커피를 마셨다. | 吃完饭之后喝了咖啡。 |
| 졸업하**고 나서** 취직했다. | 毕业之后找了工作。 |
| 영화를 보**고 나서** 이야기합시다. | 看完电影之后聊吧。 |
| 생각해 보**고 나서** 결정하겠습니다. | 想好之后再决定。 |

---

## 三、-아/어서 (顺序) ★★★★★

### 用法（先后顺序）

| 例句 | 翻译 |
|------|------|
| 도서관에 가**서** 공부했다. | 去图书馆学习了。 |
| 앉아**서** 이야기합시다. | 坐下来聊吧。 |
| 만나**서** 같이 밥 먹었어요. | 见了面一起吃了饭。 |

### -고 vs -아/어서 顺序用法区别 ★

| | -고 | -아/어서 |
|---|---|---|
| 前后关联性 | 弱（可分开的两件事） | 强（第二件是第一件的延续） |
| 可分主语 | ✓ | ✗ |
| 例 | 밥을 먹**고** TV를 봤다 | 시장에 가**서** 과일을 샀다 |

---

## 四、-(으)면서 ★★★★★

### 接续
- 有收音 + -으면서 / 无收音 + -면서

### 意思
一边...一边...（同时进行）

| 例句 | 翻译 |
|------|------|
| 음악을 들**으면서** 공부해요. | 一边听音乐一边学习。 |
| 밥을 먹**으면서** 이야기했다. | 边吃饭边聊天了。 |
| 운전하**면서** 전화하면 안 돼요. | 不能边开车边打电话。 |
| 걸**으면서** 생각했다. | 边走边想。 |
| 울**면서** 말했다. | 边哭边说。 |

### 限制
- 前后主语必须相同
- 两个动作可以同时做

---

## 五、-(으)ㄴ 후에 / -(으)ㄴ 다음에 / -(으)ㄴ 뒤에 ★★★★

### 接续
动词 + -(으)ㄴ 后 + 에

### 意思
...之后

| 表达 | 例句 | 翻译 |
|------|------|------|
| -(으)ㄴ 후에 | 졸업한 **후에** 취직했다. | 毕业后找了工作。 |
| -(으)ㄴ 다음에 | 밥을 먹은 **다음에** 출발합시다. | 吃完饭之后出发吧。 |
| -(으)ㄴ 뒤에 | 수업이 끝난 **뒤에** 만납시다. | 上课结束后见面吧。 |

---

## 六、-기 전에 ★★★★

### 接续
动词 + -기 전에

### 意思
...之前

| 例句 | 翻译 |
|------|------|
| 자**기 전에** 양치해요. | 睡前刷牙。 |
| 밥을 먹**기 전에** 손을 씻으세요. | 吃饭前请洗手。 |
| 출발하**기 전에** 확인하세요. | 出发前请确认。 |
| 결정하**기 전에** 잘 생각해 보세요. | 决定前好好想想。 |

---

## 七、-자마자 ★★★★

### 接续
动词 + -자마자

### 意思
一...就...（紧接着）

| 例句 | 翻译 |
|------|------|
| 집에 도착하**자마자** 잤어요. | 一到家就睡了。 |
| 수업이 끝나**자마자** 나갔다. | 一下课就出去了。 |
| 소식을 듣**자마자** 달려왔다. | 一听到消息就跑来了。 |
| 눈을 뜨**자마자** 휴대폰을 확인했다. | 一睁眼就看手机了。 |

---

## 八、-다가 ★★★★

### 接续
动词 + -다가

### 意思
1. 正在做A中途转做B（中断）
2. 做A的过程中发生了B

| 例句 | 翻译 |
|------|------|
| 공부하**다가** 잠들었어요. | 学着学着睡着了。 |
| 걷**다가** 넘어졌다. | 走着走着摔倒了。 |
| 밥을 먹**다가** 전화를 받았다. | 吃饭过程中接了电话。 |
| 서울에 살**다가** 부산으로 이사했다. | 在首尔住了一段后搬到了釜山。 |
| 울**다가** 웃었다. | 哭着哭着又笑了。 |

---

## 九、-자 ★★★

### 接续
动词 + -자

### 意思
一...就...（书面/叙述体）

| 例句 | 翻译 |
|------|------|
| 문을 열**자** 고양이가 뛰어나왔다. | 一开门猫就跑出来了。 |
| 선생님이 들어오시**자** 학생들이 조용해졌다. | 老师一进来学生们就安静了。 |
| 해가 뜨**자** 안개가 걷혔다. | 太阳出来雾就散了。 |

---

## 十、-는 동안(에) ★★★★

### 接续
动词 + -는 동안(에)

### 意思
在...期间

| 例句 | 翻译 |
|------|------|
| 기다리**는 동안** 커피를 마셨다. | 等的期间喝了咖啡。 |
| 자**는 동안** 비가 왔다. | 睡觉期间下了雨。 |
| 방학 **동안** 여행을 갔다. | 假期期间去旅行了。 |

---

## 十一、-는 사이(에) ★★★

### 意思
在...之间/当...的时候（不知不觉间）

| 例句 | 翻译 |
|------|------|
| 잠깐 한눈파**는 사이에** 아이가 사라졌다. | 一走神的功夫孩子就不见了。 |
| 점심을 먹**는 사이에** 배달이 왔다. | 吃午饭的时候快递来了。 |

---

## 十二、-아/어 가지고 ★★★

### 接续
动词 + -아/어 가지고

### 意思
口语中的"...然后..."（= -아/어서 的口语替代）

| 例句 | 翻译 |
|------|------|
| 돈을 모아 **가지고** 여행 갈 거야. | 攒钱然后去旅行。 |
| 일찍 일어나 **가지고** 운동했어. | 早起然后运动了。 |

---

## 十三、时间语尾总览

| 语尾 | 意思 | 时间关系 |
|------|------|---------|
| -고 | 然后 | A → B (顺序) |
| -고 나서 | 做完后 | A完成 → B |
| -아/어서 | 然后(关联) | A → B(紧密) |
| -(으)면서 | 一边一边 | A ‖ B (同时) |
| -(으)ㄴ 후에 | 之后 | A完成 → B |
| -기 전에 | 之前 | B → A |
| -자마자 | 一就 | A → B(立即) |
| -다가 | 做着做着 | A中途 → B |
| -자 | 一就(书面) | A → B(立即) |
| -는 동안 | 在期间 | A 持续中 → B |
| -는 사이에 | 之间 | A的间隙 → B |

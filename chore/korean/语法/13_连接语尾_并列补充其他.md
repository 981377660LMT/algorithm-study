# 连接语尾：并列补充其他

> 表示选择、递进、比例、列举等关系的连接语尾

---

## 一、-거나 ★★★★★

### 接续

词干 + -거나

### 意思

或者（选择）

| 例句                                       | 翻译                     |
| ------------------------------------------ | ------------------------ |
| 주말에 영화를 보**거나** 책을 읽어요.      | 周末看电影或看书。       |
| 커피를 마시**거나** 차를 마신다.           | 喝咖啡或喝茶。           |
| 좋**거나** 싫**거나** 해야 한다.           | 不管喜不喜欢都要做。     |
| 비가 오**거나** 눈이 오**거나** 갈 거예요. | 不管下雨还是下雪都会去。 |

---

## 二、-든지 / -든(가) ★★★★

### 接续

词干 + -든지

### 意思

不管...都（= 无论...都）

| 例句                                     | 翻译               |
| ---------------------------------------- | ------------------ |
| 뭐**든지** 잘 먹어요.                    | 不管什么都吃得好。 |
| 어디**든지** 갈 수 있어요.               | 不管哪里都能去。   |
| 가**든지** 말**든지** 네 맘대로 해.      | 去不去随你。       |
| 비가 오**든지** 안 오**든지** 나갈 거야. | 下不下雨都要出去。 |
| 누구**든지** 참여할 수 있다.             | 不管谁都能参加。   |

---

## 三、-(으)ㄹ수록 ★★★★

### 接续

词干 + -(으)ㄹ수록

### 意思

越...越...

| 例句                                           | 翻译               |
| ---------------------------------------------- | ------------------ |
| 갈**수록** 어려워진다.                         | 越走越难。         |
| 먹**을수록** 맛있다.                           | 越吃越好吃。       |
| 알**면** 알**수록** 어렵다.                    | 越了解越难。       |
| 생각하**면** 생각할**수록** 모르겠다.          | 越想越不明白。     |
| 바쁘**면** 바쁠**수록** 건강에 신경 써야 한다. | 越忙越要注意健康。 |

### -(으)면 -(으)ㄹ수록 ★ 固定搭配

同一动词重复加强语气。

---

## 四、-(으)ㄹ 뿐만 아니라 ★★★★

### 接续

动词 -(으)ㄹ 뿐만 아니라 / 形容词 -(으)ㄹ 뿐만 아니라
名词 + 뿐만 아니라

### 意思

不仅...而且...

| 例句                                       | 翻译                   |
| ------------------------------------------ | ---------------------- |
| 한국어**뿐만 아니라** 일본어도 할 수 있다. | 不仅韩语日语也会。     |
| 맛있**을 뿐만 아니라** 건강에도 좋다.      | 不仅好吃而且对健康好。 |
| 공부를 잘할 **뿐만 아니라** 운동도 잘한다. | 不仅学习好运动也好。   |

---

## 五、-(으)ㄹ 뿐(이다) ★★★★

### 接续

词干 + -(으)ㄹ 뿐이다

### 意思

只是...而已 / 只不过...

| 例句                              | 翻译               |
| --------------------------------- | ------------------ |
| 듣기만 했**을 뿐** 말하지 않았다. | 只是听了而已没说。 |
| 노력했**을 뿐**이다.              | 只是努力了而已。   |
| 보기만 했**을 뿐** 사지 않았다.   | 只是看了而已没买。 |

---

## 六、-는데다(가) ★★★

### 接续

动词 -는데다(가) / 形容词 -(으)ㄴ데다(가)

### 意思

不仅...而且还...（递进/叠加）

| 例句                                 | 翻译                 |
| ------------------------------------ | -------------------- |
| 비가 오**는데다가** 바람까지 불었다. | 不仅下雨还刮风了。   |
| 맛이 없**는데다가** 비싸기까지 하다. | 不仅不好吃还贵。     |
| 열심히 한**데다가** 운까지 좋았다.   | 不仅努力了运气还好。 |

---

## 七、-고도 ★★★

### 접속

词干 + -고도

### 意思

明明做了...却...（做了A但结果出乎意料）

| 例句                            | 翻译                   |
| ------------------------------- | ---------------------- |
| 많이 먹**고도** 배가 안 부르다. | 明明吃了很多但没饱。   |
| 알**고도** 모르는 척했다.       | 明明知道却装不知道。   |
| 듣**고도** 못 들은 척했다.      | 明明听见了却装没听见。 |

---

## 八、-(으)ㄴ/는 셈이다 ★★★

### 意思

算是.../等于是...

| 例句                                             | 翻译                       |
| ------------------------------------------------ | -------------------------- |
| 매일 한 시간씩 하니까 꽤 열심히 하**는 셈이다**. | 每天一小时，算是相当努力。 |
| 반도 안 왔으니 실패**한 셈이다**.                | 一半都没来算是失败了。     |

---

## 九、-(으)ㄴ/는 김에 ★★★

### 접속

动词 -는 김에 / -(으)ㄴ 김에

### 意思

既然...顺便...

| 例句                                 | 翻译                         |
| ------------------------------------ | ---------------------------- |
| 나가**는 김에** 우유 좀 사다 줘.     | 既然出去顺便买点牛奶。       |
| 청소하**는 김에** 빨래도 했다.       | 既然打扫了顺便洗了衣服。     |
| 서울에 **온 김에** 경복궁도 가 봤다. | 既然来了首尔顺便去了景福宫。 |
| 일어나**ㄴ 김에** 운동했다.          | 已经起来了顺便运动了。       |

---

## 十、-기는커녕 ★★★

### 접속

名词 + 은/는커녕
动词 + -기는커녕

### 意思

别说...连...都...（反预期递进）

| 例句                            | 翻译                   |
| ------------------------------- | ---------------------- |
| 돈**은커녕** 시간도 없다.       | 别说钱了连时间都没有。 |
| 도와주**기는커녕** 방해만 했다. | 别说帮忙了光捣乱。     |
| 사과하**기는커녕** 화를 냈다.   | 别说道歉了还发火。     |
| 칭찬**은커녕** 욕만 먹었다.     | 别说表扬了光挨骂。     |

---

## 十一、-(으)ㄹ 만큼 ★★★

### 意思

达到...的程度 / 足以...

| 例句                             | 翻译                   |
| -------------------------------- | ---------------------- |
| 먹**을 만큼** 먹었다.            | 吃了足够的量。         |
| 울**만큼** 울었으니 이제 그만해. | 哭够了所以现在停吧。   |
| 이해할 **만큼** 설명했다.        | 解释到能理解的程度了。 |

---

## 十二、-ㄴ/는 만큼 ★★★

### 意思

和...一样多/因为...所以...

| 例句                                  | 翻译                         |
| ------------------------------------- | ---------------------------- |
| 노력하**는 만큼** 결과가 좋을 것이다. | 付出多少努力就有多好的结果。 |
| 가격이 비싼 **만큼** 질이 좋다.       | 价格贵，相应地质量也好。     |

---

## 十三、-다시피 ★★★

### 접속

动词 + -다시피

### 意思

正如...（引用已知事实）

| 例句                                 | 翻译                   |
| ------------------------------------ | ---------------------- |
| 아시**다시피** 한국어는 어렵습니다.  | 如你所知韩语很难。     |
| 보시**다시피** 결과는 좋지 않습니다. | 如您所见结果不好。     |
| 알**다시피** 오늘은 시험이에요.      | 如大家知道的今天考试。 |

---

## 十四、-치고 ★★

### 意思

作为...来说（后面通常是反预期）

| 例句                            | 翻译                       |
| ------------------------------- | -------------------------- |
| 학생**치고** 돈이 많다.         | 作为学生来说钱很多。       |
| 외국인**치고** 한국어를 잘한다. | 作为外国人来说韩语说得好。 |
| 싼 것**치고** 품질이 좋다.      | 作为便宜的来说质量好。     |

---

## 十五、并列补充语尾总览

| 语尾            | 意思     | 类型       |
| --------------- | -------- | ---------- |
| -거나           | 或者     | 选择       |
| -든지           | 不管都   | 无条件选择 |
| -(으)ㄹ수록     | 越越     | 比例递进   |
| -ㄹ 뿐만 아니라 | 不仅而且 | 递进       |
| -(으)ㄹ 뿐이다  | 只是而已 | 限定       |
| -는데다(가)     | 还加上   | 叠加       |
| -고도           | 明明却   | 反预期     |
| -는 김에        | 顺便     | 附带目的   |
| -기는커녕       | 别说连   | 反预期递进 |
| -(으)ㄹ 만큼    | 到程度   | 程度       |
| -다시피         | 正如     | 引用       |
| -치고           | 作为来说 | 反预期     |

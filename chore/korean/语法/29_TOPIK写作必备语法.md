# TOPIK 写作必备语法 (쓰기 필수 문법)

> TOPIK II 쓰기 제51~54번 답안 작성에 필요한 핵심 문법과 문형

---

## 一、쓰기 51·52번 (填空题) 常用文型

### 1. 原因结果类

| 文型               | 例句                               |
| ------------------ | ---------------------------------- |
| 명사 + (으)로 인해 | 태풍**으로 인해** 피해가 발생했다  |
| -기 때문이다       | 경제가 어렵**기 때문이다**         |
| -(으)ㄴ/는 탓에    | 준비 부족**인 탓에** 실패했다      |
| -(으)ㄴ/는 덕분에  | 기술 발전 **덕분에** 편리해졌다    |
| -아/어서 그런지    | 피곤해**서 그런지** 집중이 안 된다 |

### 2. 对比类

| 文型               | 例句                                 |
| ------------------ | ------------------------------------ |
| -에 비해           | 작년**에 비해** 수출이 증가했다      |
| -는 반면(에)       | 남자는 줄어든 **반면** 여자는 늘었다 |
| -(으)ㄴ/는 데 반해 | 예상과 달리 증가한 **데 반해**       |

### 3. 条件假设类

| 文型         | 例句                                      |
| ------------ | ----------------------------------------- |
| -(으)ㄹ 경우 | 문제가 발생할 **경우** 즉시 보고해야 한다 |
| -다면        | 노력한**다면** 성공할 수 있을 것이다      |

### 4. 关于/对于类

| 文型               | 例句                               |
| ------------------ | ---------------------------------- |
| 명사 + 에 대해(서) | 환경 문제**에 대해** 논의했다      |
| 명사 + 에 관해(서) | 이 주제**에 관해** 발표하겠다      |
| -는 것에 대해      | 변화하**는 것에 대해** 생각해 봤다 |

## 二、쓰기 53번 (图表描述) 必备表达

### 1. 图表引入

| 表达                               | 翻译                   |
| ---------------------------------- | ---------------------- |
| 위의 그래프는 ~을/를 나타낸 것이다 | 上面的图表表示了…      |
| 위의 표에 따르면                   | 根据上面的表格         |
| ~에 대한 조사/설문 결과이다        | 是关于…的调查/问卷结果 |

### 2. 增加

| 表达                       | 翻译          |
| -------------------------- | ------------- |
| 증가하다 / 늘어나다 / 늘다 | 增加          |
| 급증하다 / 급격히 증가하다 | 急剧增加      |
| 꾸준히 증가하다            | 持续增加      |
| 소폭/대폭 증가하다         | 小幅/大幅增加 |

### 3. 减少

| 表达                       | 翻译     |
| -------------------------- | -------- |
| 감소하다 / 줄어들다 / 줄다 | 减少     |
| 급감하다 / 급격히 감소하다 | 急剧减少 |
| 꾸준히 감소하다            | 持续减少 |

### 4. 变化

| 表达                          | 翻译                 |
| ----------------------------- | -------------------- |
| ~에서 ~(으)로 변했다/바뀌었다 | 从…变为了…           |
| ~%에서 ~%로 증가/감소했다     | 从…%增加/减少到…%    |
| ~배(로) 증가했다              | 增加了…倍            |
| ~%p 상승/하락했다             | 上升/下降了…个百分点 |

### 5. 比较

| 表达                               | 翻译                |
| ---------------------------------- | ------------------- |
| ~보다 높다/낮다                    | 比…高/低            |
| ~보다 ~배 더 많다/적다             | 比…多/少…倍         |
| 가장 높은/낮은 비율을 차지하다     | 占最高/最低比例     |
| ~에 비해 ~이/가 높게/낮게 나타났다 | 与…相比…显示较高/低 |
| ~은/는 ~와/과 반대의 경향을 보였다 | …显示与…相反的趋势  |

### 6. 总结

| 表达                        | 翻译            |
| --------------------------- | --------------- |
| 전반적으로 ~하는 추세이다   | 总体呈…趋势     |
| 이를 통해 ~을/를 알 수 있다 | 通过这可以知道… |
| 이러한 결과를 볼 때         | 从这些结果来看  |

## 三、쓰기 54번 (论述文) 核心文型

### 1. 开头段：话题引入

| 文型                                 | 例句                    |
| ------------------------------------ | ----------------------- |
| 현대 사회에서 ~은/는 중요한 문제이다 | 在现代社会…是重要的问题 |
| 최근 ~에 대한 관심이 높아지고 있다   | 最近对…的关注正在增加   |
| ~에 관해 다양한 의견이 있다          | 关于…有各种观点         |
| ~은/는 찬반양론이 있다               | …有赞反两论             |
| 이에 대해 살펴보고자 한다            | 就此想来探讨            |
| ~의 원인과 해결 방안을 살펴보겠다    | 来看看…的原因和解决方案 |

### 2. 正文段：展开论述

#### 提出论点

| 文型                       | 翻译           |
| -------------------------- | -------------- |
| 우선/먼저 ~을/를 살펴보면  | 首先来看…      |
| 그 이유는 ~기 때문이다     | 其原因是因为…  |
| ~(으)ㄹ 뿐만 아니라 ~도    | 不仅…而且也…   |
| 또한/게다가/더불어         | 另外/而且/加之 |
| 다음으로 ~을/를 들 수 있다 | 其次可以举出…  |

#### 举例

| 文型                     | 翻译                  |
| ------------------------ | --------------------- |
| 예를 들어/예를 들면      | 例如                  |
| 이를테면                 | 比如说                |
| ~는 좋은/대표적인 예이다 | 是很好的/代表性的例子 |
| ~에서 볼 수 있듯이       | 正如在…中所见         |

#### 对比·转折

| 文型                   | 翻译         |
| ---------------------- | ------------ |
| 그러나/하지만/그렇지만 | 但是         |
| 반면(에)               | 另一方面     |
| 한편                   | 另一方面     |
| 그에 반해/그와 반대로  | 与此相反     |
| ~는 반면에 ~는         | …另一方面…则 |

#### 因果

| 文型                   | 翻译      |
| ---------------------- | --------- |
| 따라서/그러므로/그래서 | 因此      |
| 그 결과                | 结果      |
| ~(으)므로              | 因为…所以 |
| 이로 인해              | 由此      |
| 그런 까닭에            | 因此      |

### 3. 結尾段：总结归纳

| 文型                                          | 翻译              |
| --------------------------------------------- | ----------------- |
| 결론적으로/요약하면/이상에서 살펴본 바와 같이 | 总的来说/综上所述 |
| ~(으)ㄹ 필요가 있다                           | 有必要…           |
| ~아/어야 할 것이다                            | 应该…             |
| ~(으)ㄹ 때 비로소 ~(으)ㄹ 수 있을 것이다      | 只有…才能…        |
| ~도록 노력해야 한다                           | 应该努力以便…     |
| ~하는 자세가 필요하다                         | 需要…的态度       |
| ~는 것이 무엇보다 중요하다                    | …比什么都重要     |
| ~해 나가야 할 것이다                          | 应该持续…下去     |

## 四、54번 常用句式模板

### 模板1: 原因分析 + 解决方案

```
현대 사회에서 [주제]은/는 중요한 문제이다.
이에 대해 원인과 해결 방안을 살펴보고자 한다.

먼저, [주제]의 원인을 살펴보면 [원인1]을/를 들 수 있다.
[구체적 설명]. 또한 [원인2]도 그 원인 중 하나이다.
[구체적 설명].

이러한 문제를 해결하기 위해서는 [방안1]아/어야 할 것이다.
[구체적 설명]. 뿐만 아니라 [방안2]는 것도 필요하다.
[구체적 설명].

결론적으로, [주제] 문제를 해결하기 위해서는
[방안]도록 노력해야 할 것이다.
```

### 模板2: 赞反两论

```
최근 [주제]에 대한 관심이 높아지고 있다.
이에 대해 찬성과 반대의 의견이 있다.

찬성하는 입장에서는 [이유1]기 때문이라고 주장한다.
[구체적 설명]. 또한 [이유2]ㄹ 뿐만 아니라
[부가 설명]는 점도 강조한다.

그러나 반대하는 입장에서는 [이유]를 들며
[부정적 측면]을/를 지적한다.
[구체적 설명].

이상에서 살펴본 바와 같이, [주제]에 대해서는
다양한 관점이 존재한다. [자신의 의견]는 것이
무엇보다 중요하다고 생각한다.
```

## 五、쓰기 高分核心技巧

### 1. 격식체 必须（해요体→书面体）

| ✗ 不可用  | ✓ 应该用                      |
| --------- | ----------------------------- |
| 좋아요    | 좋다                          |
| 있어요    | 있다                          |
| 할 거예요 | 할 것이다                     |
| 해야 돼요 | 해야 한다 / 해야 할 것이다    |
| 같아요    | 것으로 보인다 / (으)ㄹ 것이다 |

### 2. 口语→书面 替换

| 口语      | 书面                          |
| --------- | ----------------------------- |
| 그래서    | 따라서 / 그러므로             |
| 그래도    | 그럼에도 불구하고             |
| 그런데    | 그러나 / 하지만               |
| 되게/엄청 | 매우 / 상당히                 |
| 안 좋다   | 부정적이다                    |
| 좋은 점   | 장점 / 긍정적인 측면          |
| 나쁜 점   | 단점 / 부정적인 측면          |
| 요즘      | 최근 / 현대 사회에서          |
| 많은 사람 | 많은 사람들 / 대다수의 사람들 |

### 3. 段落结构

| 段落  | 内容              | 行数建议 |
| ----- | ----------------- | -------- |
| 开头  | 引入主题+提出论点 | 2~3行    |
| 正文1 | 第一论据+举例     | 3~4行    |
| 正文2 | 第二论据+举例     | 3~4行    |
| 结尾  | 总结+展望/建议    | 2~3行    |

### 4. 接续词使用

| 功能 | 接续词                            |
| ---- | --------------------------------- |
| 并列 | 또한, 게다가, 뿐만 아니라, 더불어 |
| 转折 | 그러나, 하지만, 반면에, 한편      |
| 因果 | 따라서, 그러므로, 이로 인해       |
| 递进 | 특히, 무엇보다, 더 나아가         |
| 举例 | 예를 들어, 이를테면, 가령         |
| 总结 | 결론적으로, 요약하면, 정리하면    |

## 六、쓰기 常犯错误

| 错误               | 正确                | 说明             |
| ------------------ | ------------------- | ---------------- |
| ~~좋은 것 같아요~~ | 좋다고 생각한다     | 书面不用 것 같다 |
| ~~나는~~           | 필자는 / (省略主语) | 第一人称用法     |
| ~~진짜~~           | 매우 / 상당히       | 口语→书面        |
| ~~-아/어요~~       | -ㄴ다/는다/-다      | 终结语尾         |
| ~~근데~~           | 그러나 / 하지만     | 接续词           |
| ~~-(으)ㄹ 거예요~~ | -(으)ㄹ 것이다      | 将来时           |
| ~~-잖아요~~        | -(으)ㄴ/는 것이다   | 强调方式         |

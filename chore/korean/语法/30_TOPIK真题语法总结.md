# TOPIK 真题语法总结 (기출 문법 정리)

> TOPIK II 읽기·듣기에서 반복 출제되는 문법 항목을 빈도별로 정리

## 一、最高频出题语法 (每届必考) ★★★★★

### 1. 连接语尾

| 语法         | 意思       | 出题重点          |
| ------------ | ---------- | ----------------- |
| -아/어서     | 因为/然后  | vs -(으)니까 辨析 |
| -(으)니까    | 因为(主观) | 后接命令/共动     |
| -지만        | 但是       | 基本转折          |
| -(으)ㄴ/는데 | 背景/转折  | 多义辨析★         |
| -(으)면      | 如果       | 条件假设          |
| -(으)면서    | 边…边…     | 同时进行/矛盾     |
| -고          | 然后/并且  | 顺序 vs 并列      |
| -(으)려고    | 为了       | 目的(同主语)      |
| -기 위해(서) | 为了       | 目的(正式)        |
| -아/어도     | 即使       | 让步              |
| -거나        | 或者       | 选择              |
| -(으)ㄹ수록  | 越…越…     | 读解选择题        |

### 2. 终结语尾

| 语法        | 意思       | 出题重点       |
| ----------- | ---------- | -------------- |
| -(으)ㄹ게요 | 我会(承诺) | vs -(으)ㄹ래요 |
| -(으)ㄹ래요 | 我想/你要? | 口语意愿       |
| -겠-        | 决心/推测  | 一人称=意志    |
| -네요       | 发现感叹   | 亲历感受       |
| -군요/-구나 | 感叹       | 新发现         |
| -거든요     | 补充理由   | 听力对话       |
| -잖아요     | 你不是知道 | 提醒           |

### 3. 冠形词形+补助表达

| 语法                 | 意思    | 出题重点  |
| -------------------- | ------- | --------- |
| -(으)ㄹ 수 있다/없다 | 能/不能 | 基本可能  |
| -(으)ㄹ 것 같다      | 好像    | 推测★★★   |
| -(으)ㄴ 적이 있다    | 做过    | 经验      |
| -는 것               | 名词化  | …的事情   |
| -(으)ㄹ 때           | …的时候 | 时间      |
| -(으)ㄹ 것이다       | 将要/会 | 推测+意志 |

## 四、各题型语法攻略

### 읽기 1~2번 (广告/通知 选主题)

常见语法：

- -(으)ㄹ 수 있습니다 (能…)
- -아/어 주시기 바랍니다 (请…)
- -에 관한/대한 (关于…)
- -(으)ㄹ 예정이다 (预定…)
- -고자 하다 (想要…)

### 읽기 3~4번 (看图选句)

考点：终结语尾辨析

- -(으)ㄹ까요? vs -(으)ㄹ래요?
- -(으)세요 vs -(으)ㄹ게요
- -아/어 줄까요? vs -아/어 주세요

### 읽기 5~8번 (相近意思选择)

**最常考对比组**：

| 题目语法   | 选项中可能的近义           |
| ---------- | -------------------------- |
| -아/어서   | -기 때문에, -(으)니까      |
| -는 데     | -기 위해, -는 것에         |
| -(으)면서  | -는 동시에, -ㄴ 채로       |
| -다가      | -는 도중에                 |
| -도록      | -기 위해, -게              |
| -는 셈이다 | -는 것과 같다/마찬가지이다 |
| -곤 하다   | -는 경우가 많다/자주 -는다 |

### 읽기 9~12번 (문장 넣기/순서 배열)

关键：识别连接线索
| 标记 | 功能 |
|------|------|
| 그래서/따라서 | 因果→结果 |
| 그러나/하지만 | 转折 |
| 또한/게다가 | 递进 |
| 예를 들어 | 举例 |
| 즉/다시 말해 | 换言之 |
| 이처럼/이와 같이 | 如此/总结 |
| 그런데 | 话题转换 |

### 듣기 주화자 의견 (说话者观点)

高频判断表达：
| 表达 | 态度 |
|------|------|
| -(으)ㄹ 필요가 있다 | 建议/支持 |
| -아/어야 한다 | 义务/支持 |
| -(으)면 안 되다 | 反对/禁止 |
| -(으)ㄹ 수밖에 없다 | 无奈/必然 |
| -는 것이 바람직하다 | 建议(正式) |
| -는 것이 중요하다 | 强调重要性 |
| -기를 바라다 | 期望/愿望 |
| -(으)ㄹ 것으로 보인다 | 推测(客观) |

## 五、语法辨析真题回顾

### 常见辨析对（每届1~3道）

| 对比组                     | 区别要点            |
| -------------------------- | ------------------- |
| -아서 vs -니까             | 客观/主观; 后句限制 |
| -겠 vs -(으)ㄹ 것이다      | 即时/有据推测       |
| -는데 vs -지만             | 背景铺垫/纯转折     |
| -도록 vs -게               | 目的程度/方式使役   |
| -(으)면서 vs -는 동안      | 同主语同时/可不同   |
| -기 때문에 vs -(으)므로    | 通用/纯书面         |
| -(으)ㄹ게요 vs -(으)ㄹ래요 | 承诺/意愿           |
| -나 보다 vs 것 같다        | 间接推断/一般推测   |

## 六、分级语法检查清单

### 3级必会 (TOPIK II 基础)

- [ ] 격조사: 이/가, 을/를, 에, 에서, (으)로, 의, 보다, 와/과
- [ ] 보조사: 은/는, 도, 만, 까지, 부터
- [ ] 연결: -고, -아/어서, -(으)니까, -지만, -(으)면, -거나
- [ ] 종결: 해요체, -겠-, -(으)ㄹ게요, -(으)ㄹ래요
- [ ] 관형: -(으)ㄴ/는/(으)ㄹ + 명사
- [ ] 보조: -(으)ㄹ 수 있다, -고 싶다, -아/어 보다, -아/어 주다

### 4级必会

- [ ] 연결: -느라고, -는 바람에, -더라도, -다가, -자마자, -(으)ㄹ수록
- [ ] 종결: -나 보다, -(으)ㄹ 텐데, -거든요, -잖아요
- [ ] 간접 인용: -다고/냐고/라고/자고

### 5级必会

- [ ] 연결: -는 한, -느니, -(으)ㄹ 뿐만 아니라, -기는커녕
- [ ] 문어체: -(으)며, -(으)므로, -(으)나
- [ ] 복합: -(으)ㄹ 수밖에 없다, -기 마련이다, -는 셈이다
- [ ] 고급: -(으)ㄹ 따름이다, -기에 달려 있다

### 6级必会

- [ ] 고급 문어: -(으)ㄹ진대, -건대, -노라면, -(으)ㄹ뿐더러
- [ ] 관용: -는 법이 없다, -(으)ㄹ 법하다, -기 일쑤이다
- [ ] 학술: -것으로 나타나다, -것으로 밝혀지다, -(으)ㄴ/는 바

# 表达推测·判断 (추측·판단 표현)

---

## 一、-(으)ㄹ 것 같다 (好像会/大概) ★★★★★

**最常用的推测表达**，可用于过去/现在/未来

### 1. 现在推测

| 例句                  | 翻译         |
| --------------------- | ------------ |
| 비가 올 **것 같다**.  | 好像会下雨。 |
| 맛있을 **것 같아요**. | 好像很好吃。 |
| 좀 비쌀 **것 같다**.  | 好像有点贵。 |

### 2. 过去推测

| 例句                          | 翻译             |
| ----------------------------- | ---------------- |
| 벌써 갔을 **것 같다**.        | 好像已经走了。   |
| 어제 비가 왔을 **것 같아요**. | 昨天好像下雨了。 |

### 3. 否定推测

| 例句                | 翻译         |
| ------------------- | ------------ |
| 안 올 **것 같다**.  | 好像不会来。 |
| 모를 **것 같아요**. | 好像不知道。 |

### 冠形词形时制搭配

| 时态 | 动词             | 形容词           |
| ---- | ---------------- | ---------------- |
| 过去 | -았/었을 것 같다 | -았/었을 것 같다 |
| 现在 | -는 것 같다      | -(으)ㄴ 것 같다  |
| 未来 | -(으)ㄹ 것 같다  | -(으)ㄹ 것 같다  |

| 例句               | 时态    | 翻译         |
| ------------------ | ------- | ------------ |
| 먹**는** 것 같다   | V现在   | 好像在吃     |
| 먹**은** 것 같다   | V过去   | 好像吃了     |
| 먹**을** 것 같다   | V未来   | 好像会吃     |
| 예**쁜** 것 같다   | Adj现在 | 好像漂亮     |
| 예**뻤을** 것 같다 | Adj过去 | 好像以前漂亮 |

---

## 二、-겠- (推测/感叹) ★★★★★

### (1) 根据眼前情况推测

| 例句              | 翻译             |
| ----------------- | ---------------- |
| 맛있**겠**다!     | (看起来)好吃！   |
| 힘들**겠**어요.   | 应该很辛苦吧。   |
| 춥**겠**다.       | (看起来)很冷吧。 |
| 재미있**겠**어요! | 肯定很有趣！     |

### (2) 较有把握的推测

| 例句                    | 翻译             |
| ----------------------- | ---------------- |
| 내일도 비가 오**겠**다. | 明天也会下雨吧。 |
| 12시쯤 되**겠**다.      | 大概12点了吧。   |

---

## 三、-(으)ㄹ 것이다 (会/将) ★★★★★

较有把握的推测/预测

| 例句                       | 翻译             |
| -------------------------- | ---------------- |
| 내일 비가 올 **거예요**.   | 明天会下雨吧。   |
| 곧 도착할 **겁니다**.      | 应该快到了。     |
| 시간이 좀 걸릴 **거예요**. | 可能要花点时间。 |

### -겠- vs -(으)ㄹ 것이다

| -겠-                | -(으)ㄹ 것이다                      |
| ------------------- | ----------------------------------- |
| 即时判断            | 有根据的推测                        |
| 眼前感受            | 一般推测                            |
| （看到菜）맛있겠다! | 이 식당 음식이 맛있을 거예요(听说). |

---

## 四、-나 보다 / -(으)ㄴ가 보다 (看来/似乎) ★★★★★

根据**间接证据**做出推测

### 动词

| 形态           | 例句                | 翻译           |
| -------------- | ------------------- | -------------- |
| V-나 보다      | 비가 오**나 봐요**. | 看来在下雨。   |
| V-았/었나 보다 | 벌써 갔**나 봐요**. | 看来已经走了。 |

### 形容词

| 形态              | 例句             | 翻译             |
| ----------------- | ---------------- | ---------------- |
| Adj-(으)ㄴ가 보다 | 바**쁜가 봐요**. | 看来很忙。       |
| Adj-았/었나 보다  | 아팠**나 봐요**. | 看来(之前)病了。 |

### 名词: 인가 보다

| 例句               | 翻译         |
| ------------------ | ------------ |
| 학생**인가 봐요**. | 看来是学生。 |

---

## 五、-(으)ㄴ/는 모양이다 (看样子) ★★★★

根据观察到的情况推测

| 时态  | 形态             | 例句                    | 翻译             |
| ----- | ---------------- | ----------------------- | ---------------- |
| V现在 | -는 모양이다     | 비가 오**는 모양이다**. | 看样子在下雨。   |
| V过去 | -(으)ㄴ 모양이다 | 벌써 간 **모양이다**.   | 看样子已经走了。 |
| V将来 | -(으)ㄹ 모양이다 | 곧 올 **모양이다**.     | 看样子快来了。   |
| Adj   | -(으)ㄴ 모양이다 | 바**쁜 모양이다**.      | 看样子很忙。     |

### -나 보다 vs -(으)ㄴ/는 모양이다

| -나 보다 | 모양이다 |
| -------- | -------- |
| 口语多   | 稍正式   |
| 间接证据 | 观察推断 |
| 语感相近 | 语感相近 |

---

## 六、-(으)ㄹ 텐데 (应该…吧/估计) ★★★★

推测 + 担心/遗憾

| 例句                             | 翻译                       |
| -------------------------------- | -------------------------- |
| 시험이 어려울 **텐데** 걱정이다. | 考试应该很难吧，真担心。   |
| 피곤할 **텐데** 좀 쉬세요.       | 应该很累吧，休息一下。     |
| 배고플 **텐데** 뭐 먹을래요?     | 应该饿了吧，要吃什么？     |
| 혼자서 힘들 **텐데** 도와줄까요? | 自己应该很难吧，要帮忙吗？ |

---

## 七、-(으)ㄹ 테니까 (估计…所以) ★★★★

推测 + 给出理由/建议

| 例句                               | 翻译                     |
| ---------------------------------- | ------------------------ |
| 비가 올 **테니까** 우산 가져가.    | 估计会下雨，带伞去。     |
| 바쁠 **테니까** 나중에 전화하세요. | 估计很忙，之后再打电话。 |
| 제가 할 **테니까** 걱정 마세요.    | 我来做，(你)别担心。     |

---

## 八、-(으)ㄹ 리가 없다 (不可能) ★★★★

强烈否定推测

| 例句                            | 翻译                 |
| ------------------------------- | -------------------- |
| 그럴 **리가 없다**!             | 不可能！             |
| 모를 **리가 없다**.             | 不可能不知道。       |
| 거짓말할 **리가 없어요**.       | 不可能说谎。         |
| 그 사람이 범인일 **리가 없다**. | 那个人不可能是犯人。 |

---

## 九、-(으)ㄹ지도 모르다 (说不定/也许) ★★★★

不确定的推测（也许）

| 例句                        | 翻译               |
| --------------------------- | ------------------ |
| 비가 올**지도 몰라요**.     | 说不定会下雨。     |
| 늦을**지도 모르겠다**.      | 也许会迟到。       |
| 아직 모를**지도 몰라요**.   | 说不定还不知道。   |
| 벌써 끝났을**지도 모른다**. | 说不定已经结束了。 |

---

## 十、-ㄴ/는 듯하다 (似乎/好像) ★★★

书面化的推测

| 时态  | 形态           | 例句                              | 翻译     |
| ----- | -------------- | --------------------------------- | -------- |
| V现在 | -는 듯하다     | 알**는 듯하다** → 아**는 듯하다** | 似乎知道 |
| V过去 | -(으)ㄴ 듯하다 | 간 **듯하다**                     | 似乎走了 |
| V将来 | -(으)ㄹ 듯하다 | 올 **듯하다**                     | 似乎会来 |
| Adj   | -(으)ㄴ 듯하다 | 추**운 듯하다**                   | 似乎冷   |

---

## 十一、-(으)ㄹ 수(도) 있다 (可能/也许) ★★★★★

表示可能性

| 例句                        | 翻译             |
| --------------------------- | ---------------- |
| 내일 비가 올 **수도 있다**. | 明天可能会下雨。 |
| 그럴 **수도 있어요**.       | 也许是那样。     |
| 틀릴 **수도 있다**.         | (也)可能是错的。 |

---

## 十二、-아/어 보이다 (看起来) ★★★★

| 例句                 | 翻译           |
| -------------------- | -------------- |
| 피곤해 **보여요**.   | 看起来很累。   |
| 맛있어 **보인다**.   | 看起来好吃。   |
| 행복해 **보이네요**. | 看起来很幸福。 |
| 젊어 **보여요**.     | 看起来年轻。   |

---

## 十三、-(으)ㄴ/는/(으)ㄹ 줄 알았다 ★★★

以为…（但其实不是）

| 例句                         | 翻译                   |
| ---------------------------- | ---------------------- |
| 한국 사람인 **줄 알았어요**. | 以为是韩国人。         |
| 벌써 간 **줄 알았다**.       | 以为已经走了。         |
| 쉬울 **줄 알았는데** 어렵다. | 以为会简单，结果很难。 |

---

## 十四、推测表达强度排列

| 表达                | 确信度      | 场景      |
| ------------------- | ----------- | --------- |
| -(으)ㄹ 리가 없다   | ★★★★★(否定) | 不可能    |
| -(으)ㄹ 것이다      | ★★★★        | 较有把握  |
| -겠-                | ★★★★        | 即时判断  |
| -(으)ㄹ 것 같다     | ★★★         | 一般推测  |
| -나 보다 / 모양이다 | ★★★         | 间接推断  |
| -(으)ㄹ 텐데        | ★★★         | 推测+担心 |
| -(으)ㄹ 수도 있다   | ★★          | 可能性    |
| -(으)ㄹ지도 모르다  | ★           | 说不定    |

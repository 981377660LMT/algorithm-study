# 终结语尾 (종결어미)

> 终结语尾 = 结束句子的语尾，决定语气（陈述/疑问/命令/共动/感叹）和敬语等级

---

## 一、韩语的六种语体

| 语体   | 名称         | 敬语等级 | 使用场景             |
| ------ | ------------ | -------- | -------------------- |
| 합쇼체 | 格式体敬语   | ★★★★★    | 新闻、军队、正式演讲 |
| 해요체 | 非格式体敬语 | ★★★★     | 日常对话（最常用）   |
| 하오체 | 格式体准敬语 | ★★★      | 古装剧、老年人       |
| 하게체 | 格式体准半语 | ★★       | 长辈对晚辈(文学)     |
| 해체   | 非格式体半语 | ★        | 朋友、晚辈           |
| 해라체 | 格式体半语   | ★        | 书面/日记/独白       |

### 实际使用频率

- **해요체** (80%) — 日常大部分场合
- **합쇼체** (15%) — 正式场合/客服
- **해체/해라체** (5%) — 朋友/写作

---

## 二、합쇼체 (格式体敬语) ★★★★★

### 陈述句 -(스)ㅂ니다

| 条件   | 形式             | 例子                          |
| ------ | ---------------- | ----------------------------- |
| 有收音 | -습니다          | 먹**습니다**, 있**습니다**    |
| 无收音 | -ㅂ니다          | 가**ㅂ니다** → 갑니다, 합니다 |
| ㄹ收音 | (ㄹ脱落) -ㅂ니다 | 삽니다(살다), 압니다(알다)    |

### 疑问句 -(스)ㅂ니까

| 例句                       | 翻译         |
| -------------------------- | ------------ |
| 어디에 가**십니까**?       | 您去哪里？   |
| 한국어를 공부하**십니까**? | 您学韩语吗？ |
| 이것이 뭐**입니까**?       | 这是什么？   |

### 命令句 -(으)십시오

| 例句                   | 翻译         |
| ---------------------- | ------------ |
| 앉**으십시오**.        | 请坐。       |
| 조용히 하**십시오**.   | 请安静。     |
| 이쪽으로 오**십시오**. | 请往这边来。 |

### 共动句 -(으)ㅂ시다

| 例句               | 翻译             |
| ------------------ | ---------------- |
| 같이 **갑시다**.   | 一起走吧。       |
| 시작**합시다**.    | 开始吧。         |
| 잠깐 쉬**읍시다**. | 稍微休息一下吧。 |

## 三、해요체 (非格式体敬语) ★★★★★

**所有语气都用 -아/어요，靠语调区分**

### 陈述 -아/어요 (↘)

| 例句                 | 翻译         |
| -------------------- | ------------ |
| 저는 학생이에요↘.    | 我是学生。   |
| 오늘 날씨가 좋아요↘. | 今天天气好。 |
| 매일 운동해요↘.      | 每天运动。   |

### 疑问 -아/어요? (↗)

| 例句              | 翻译         |
| ----------------- | ------------ |
| 어디 가요↗?       | 你去哪？     |
| 뭐 먹어요↗?       | 吃什么？     |
| 한국 사람이에요↗? | 是韩国人吗？ |

### 命令 -아/어요 (→)

| 例句          | 翻译                     |
| ------------- | ------------------------ |
| 빨리 오세요→. | 请快来。(更礼貌用 -세요) |
| 여기 앉아요→. | 坐这里。                 |

### 共动 -아/어요 (↘↗)

| 例句         | 翻译       |
| ------------ | ---------- |
| 같이 가요.   | 一起走吧。 |
| 우리 먹어요. | 我们吃吧。 |

### -(으)세요 — 命令/请求的礼貌形式

| 例句                      | 翻译           |
| ------------------------- | -------------- |
| 들어오**세요**.           | 请进。         |
| 잠깐 기다리**세요**.      | 请稍等。       |
| 여기에 이름을 쓰**세요**. | 请在这写名字。 |
| 안녕히 가**세요**.        | 请慢走。       |

## 四、해체 (非格式体半语) ★★★★

**和 해요체 一样但去掉 요**

| 语气 | 例子                | 翻译     |
| ---- | ------------------- | -------- |
| 陈述 | 나 간다→가          | 我走了   |
| 疑问 | 어디 가?            | 去哪？   |
| 命令 | 빨리 와!            | 快来！   |
| 共动 | 같이 가 / 같이 가자 | 一起走吧 |
| 感叹 | 진짜 예쁘다!        | 真漂亮！ |

---

## 五、해라체 (格式体半语/书面体) ★★★★

### 陈述句 -(ㄴ/는)다

| 条件        | 形式           | 例子               |
| ----------- | -------------- | ------------------ |
| 动词 无收音 | -ㄴ다          | 간다, 온다         |
| 动词 有收音 | -는다          | 먹는다, 읽는다     |
| 形容词      | -다            | 크다, 좋다, 예쁘다 |
| 이다        | -이다 / -다    | 학생이다, 의사다   |
| 过去        | -았/었다       | 갔다, 먹었다       |
| 将来        | -(으)ㄹ 것이다 | 갈 것이다          |

### 疑问句 -(으)냐 / -니

| 例句           | 翻译     |
| -------------- | -------- |
| 밥 먹었**냐**? | 吃了吗？ |
| 어디 가**니**? | 去哪？   |
| 괜찮**으냐**?  | 没事吧？ |

### 命令句 -아/어라

| 例句             | 翻译   |
| ---------------- | ------ |
| 빨리 와**라**!   | 快来！ |
| 공부해**라**!    | 学习！ |
| 조용히 해**라**. | 安静！ |

### 共动句 -자

| 例句           | 翻译       |
| -------------- | ---------- |
| 같이 가**자**. | 一起走吧。 |
| 밥 먹**자**.   | 吃饭吧。   |
| 시작하**자**.  | 开始吧。   |

### 感叹句 -는구나 / -(으)ㄴ구나

| 例句               | 翻译             |
| ------------------ | ---------------- |
| 비가 오**는구나**! | 下雨了啊！       |
| 많이 컸**구나**!   | 长大了啊！       |
| 정말 맛있**구나**! | 真好吃啊！       |
| 벌써 봄이**구나**! | 已经是春天了啊！ |

## 六、하오체 和 하게체 (了解即可)

### 하오체 — 古装剧/老式敬语

| 语气 | 形式     | 例子               |
| ---- | -------- | ------------------ |
| 陈述 | -(으)오  | 나는 가겠**소**.   |
| 疑问 | -(으)오? | 어디로 가시**오**? |
| 命令 | -(으)오  | 들어오**시오**.    |

### 하게체 — 学术论文/长辈口吻

| 语气 | 形式              | 例子               |
| ---- | ----------------- | ------------------ |
| 陈述 | -(으)네           | 자네가 왔**네**.   |
| 疑问 | -(으)나/-(으)ㄴ가 | 자네 괜찮**은가**? |
| 命令 | -게               | 앉**게**.          |

## 七、特殊终结语尾 ★★★★

### -네(요) — 发现/惊讶

| 例句                   | 翻译               |
| ---------------------- | ------------------ |
| 비가 오**네요**.       | (刚发现)下雨了呢。 |
| 맛있**네요**.          | (尝后)好吃呢。     |
| 한국어 잘하시**네요**. | 韩语说得好呢。     |

### -군요 / -는군요 — 感叹/原来如此

| 例句                   | 翻译               |
| ---------------------- | ------------------ |
| 많이 바쁘시**군요**.   | 原来很忙啊。       |
| 벌써 도착했**군요**.   | 已经到了啊。       |
| 한국에서 오셨**군요**. | 原来从韩国来的啊。 |

### -거든(요) — 补充说明/理由

| 例句                                  | 翻译                       |
| ------------------------------------- | -------------------------- |
| 저 내일 못 가요. 약속이 있**거든요**. | 我明天去不了，因为有约。   |
| 이거 맛있어요. 직접 만들었**거든요**. | 这个好吃，因为是自己做的。 |

### -잖아(요) — 不是...嘛（提醒对方已知）

| 例句                   | 翻译             |
| ---------------------- | ---------------- |
| 내가 말했**잖아**.     | 我不是说了嘛。   |
| 오늘 시험이**잖아요**. | 今天不是考试嘛。 |
| 바쁘다고 했**잖아요**. | 不是说忙了嘛。   |

### -ㄹ게(요) — 说话者承诺/意志

| 例句                                | 翻译           |
| ----------------------------------- | -------------- |
| 제가 하**ㄹ게요**. → 할게요         | 我来做。       |
| 내일 전화하**ㄹ게요**. → 전화할게요 | 明天打电话。   |
| 잘 먹**을게요**.                    | 我会好好吃的。 |

### -(으)ㄹ래(요) — 意愿/打算

| 例句              | 翻译         |
| ----------------- | ------------ |
| 뭐 마실**래요**?  | 想喝什么？   |
| 나 먼저 갈**래**. | 我先走了。   |
| 같이 갈**래요**?  | 要一起去吗？ |

### -지(요) — 确认

| 例句                    | 翻译           |
| ----------------------- | -------------- |
| 맛있**지요**? (맛있죠?) | 好吃吧？       |
| 알**지**?               | 知道吧？       |
| 내일 오시는 거**죠**?   | 明天来的对吧？ |

## 八、语体选择指南

| 场景       | 推荐语体        | 原因         |
| ---------- | --------------- | ------------ |
| 初次见面   | 해요체          | 安全/礼貌    |
| 工作汇报   | 합쇼체          | 正式         |
| 朋友聊天   | 해체            | 亲切         |
| 写日记     | 해라체          | 书面独白     |
| 新闻播报   | 합쇼체          | 权威         |
| 论文       | 해라체(书面)    | 客观         |
| TOPIK写作  | 해라체/합쇼체   | 考试要求     |
| 给长辈写信 | 합쇼체          | 尊重         |
| 演讲开场   | 합쇼체 → 해요체 | 由正式到亲切 |

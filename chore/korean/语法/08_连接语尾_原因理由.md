# 连接语尾：原因理由

> 表示"因为、所以、由于"等因果关系的连接语尾

---

## 一、-아/어서 ★★★★★

### 接续

词干 + -아/어서 (按阴阳元音)

### 意思

1. **原因/理由** — 因为...所以...
2. **顺序/方式** — ...然后...

### 原因用法

| 例句                           | 翻译                 |
| ------------------------------ | -------------------- |
| 배가 아파**서** 병원에 갔어요. | 因为肚子疼去了医院。 |
| 비가 와**서** 집에 있었어요.   | 因为下雨在家待着了。 |
| 시간이 없어**서** 못 갔어요.   | 因为没时间没去成。   |
| 만나**서** 반갑습니다.         | 见到你很高兴。       |

### 顺序用法

| 例句                           | 翻译               |
| ------------------------------ | ------------------ |
| 손을 씻어**서** 밥을 먹었어요. | 洗了手然后吃了饭。 |
| 시장에 가**서** 과일을 샀어요. | 去市场买了水果。   |
| 택시를 타**서** 공항에 갔어요. | 坐出租车去了机场。 |

### 限制 ★★★

1. **前句不能加过去时** -았/었-
   - ~~배가 아팠어서~~ ✗ → 배가 아파서 ✓
2. **后句不能用命令/共动**
   - ~~비가 와서 우산을 가져오세요~~ ✗ → 비가 오니까 우산을 가져오세요 ✓
3. **不能用于主观判断/推测后句**
   - ~~바빠서 내일 올 거예요~~ ✗ → 바쁘니까 내일 올 거예요 ✓

## 二、-(으)니까 ★★★★★

### 接续

- 有收音 + -으니까 / 无收音 + -니까
- 可加过去时：-았/었으니까

### 意思

因为...所以... （比 -아/어서 更强调原因，语气更重）

| 例句                             | 翻译                   |
| -------------------------------- | ---------------------- |
| 시간이 없**으니까** 빨리 하세요. | 因为没时间，请快点。   |
| 비가 오**니까** 우산을 가져가.   | 因为下雨，带伞去。     |
| 피곤하**니까** 좀 쉬세요.        | 因为累了，请休息一下。 |
| 내가 했**으니까** 너는 하지 마.  | 因为我做了，你别做了。 |
| 맛있**으니까** 먹어 봐.          | 因为好吃，尝尝看。     |

### 特征 ★

1. **后句可以用命令/共动** ✓
2. **前句可以加过去时** ✓
3. **说话者主观强调原因**

---

## 三、-아/어서 vs -(으)니까 对比 ★★★★★

|           | -아/어서      | -(으)니까 |
| --------- | ------------- | --------- |
| 过去时    | 前句 ✗        | 前句 ✓    |
| 命令/共动 | 后句 ✗        | 后句 ✓    |
| 语感      | 客观陈述      | 主观强调  |
| 感谢/道歉 | ✓ (감사해서)  | ✗         |
| 固定搭配  | 만나서 반갑다 | —         |
| 时间顺序  | ✓ (去了然后)  | ✗         |

### 练习：选择正确的

| 句子                        | 答案                                                            |
| --------------------------- | --------------------------------------------------------------- |
| 비가 ( ) 우산을 가져가.     | <details><summary>答案</summary>오니까 (后句是命令)</details>   |
| 배가 아파( ) 병원에 갔어요. | <details><summary>答案</summary>아파서 (客观陈述原因)</details> |
| 늦었( ) 택시를 탑시다.      | <details><summary>答案</summary>으니까 (后句是共动)</details>   |
| 만나( ) 반갑습니다.         | <details><summary>答案</summary>서 (固定搭配)</details>         |

## 四、-기 때문에 ★★★★

### 接续

动词/形容词 + **-기 때문에**
名词 + **(이)기 때문에** 或 **때문에**

### 意思

因为...（书面/正式）

| 例句                                    | 翻译                     |
| --------------------------------------- | ------------------------ |
| 비가 오**기 때문에** 경기가 취소되었다. | 因为下雨比赛被取消了。   |
| 경제가 어렵**기 때문에** 취업이 힘들다. | 因为经济困难，就业很难。 |
| 학생이**기 때문에** 할인을 받았다.      | 因为是学生得到了折扣。   |
| 시간 **때문에** 못 갔다.                | 因为时间去不了。         |

### 过去时

-았/었기 때문에

| 例句                                     | 翻译                         |
| ---------------------------------------- | ---------------------------- |
| 많이 연습했**기 때문에** 잘할 수 있었다. | 因为练习了很多，所以能做好。 |

## 五、-(으)므로 ★★★

### 接续

- 有收音 + -으므로 / 无收音 + -므로

### 意思

因为...（**纯书面语**，论文/公告常用）

| 例句                                     | 翻译                       |
| ---------------------------------------- | -------------------------- |
| 공사 중이**므로** 출입을 금합니다.       | 因为施工中，禁止出入。     |
| 날씨가 추우**므로** 따뜻하게 입으세요.   | 因为天气冷，请穿暖和。     |
| 수요가 증가하였**으므로** 가격이 올랐다. | 因为需求增加，价格上涨了。 |

---

## 六、-느라(고) ★★★

### 接续

动词 + -느라(고)

### 意思

因为忙于做...（导致不好的结果）

### 限制

1. **前后主语必须一致**
2. **后句通常是消极结果**
3. **只接动词**

| 例句                              | 翻译                         |
| --------------------------------- | ---------------------------- |
| 공부하**느라고** 잠을 못 잤어요.  | 因为忙着学习，没睡好觉。     |
| 전화하**느라고** 수업에 늦었어요. | 因为忙着打电话，上课迟到了。 |
| 준비하**느라고** 고생했어요.      | 忙于准备辛苦了。             |
| 일하**느라고** 밥도 못 먹었어요.  | 因为忙着工作，连饭都没吃。   |

---

## 七、-는 바람에 ★★★

### 接续

动词 + -는 바람에

### 意思

因为（突发/意外的原因导致不好的结果）

| 例句                                              | 翻译                           |
| ------------------------------------------------- | ------------------------------ |
| 비가 오**는 바람에** 소풍을 못 갔어요.            | 因为下了雨没能去野餐。         |
| 늦게 일어나**는 바람에** 지각했어요.              | 因为起晚了迟到了。             |
| 갑자기 손님이 오시**는 바람에** 정리를 못 했어요. | 因为突然来了客人没来得及整理。 |
| 차가 고장 나**는 바람에** 택시를 탔어요.          | 因为车坏了坐了出租车。         |

---

## 八、-길래 ★★★

### 接续

动词/形容词 + -길래

### 意思

因为看到/听到...（所以做了某事）— 第一人称根据观察做的行动

| 例句                               | 翻译                   |
| ---------------------------------- | ---------------------- |
| 맛있어 보이**길래** 사 봤어요.     | 因为看着好吃就买了。   |
| 비가 오**길래** 우산을 가져왔어요. | 因为看到下雨带了伞。   |
| 세일하**길래** 많이 샀어요.        | 因为看到打折买了很多。 |
| 친구가 부르**길래** 갔어요.        | 因为朋友叫我就去了。   |

---

## 九、-는 통에 ★★

### 接续

动词 + -는 통에

### 意思

因为混乱的/不好的情况（导致不好的结果）

| 例句                                      | 翻译                     |
| ----------------------------------------- | ------------------------ |
| 아이가 우**는 통에** 잠을 못 잤다.        | 因为孩子哭没睡着觉。     |
| 전화가 계속 오**는 통에** 집중을 못 했다. | 因为电话一直来没能集中。 |

---

## 十、-거든 (连接用法) ★★

### 接续

动词/形容词 + -거든

### 意思

如果...的话（放在前面作为条件/原因）

| 例句                         | 翻译                   |
| ---------------------------- | ---------------------- |
| 시간이 있**거든** 연락해.    | 如果有时间就联系我。   |
| 서울에 오**거든** 꼭 만나자. | 如果来首尔一定要见面。 |

> 注意：-거든요 作终结语尾是"因为…"的意思（见文件07）

## 十一、原因语尾总览对比

| 语尾       | 口语/书面 | 过去时 | 命令/共动 | 特征       |
| ---------- | --------- | ------ | --------- | ---------- |
| -아/어서   | 口语      | 前句✗  | 后句✗     | 客观、顺序 |
| -(으)니까  | 口语      | 前句✓  | 后句✓     | 主观强调   |
| -기 때문에 | 口语+书面 | ✓      | —         | 正式       |
| -(으)므로  | 纯书面    | ✓      | —         | 论文/公告  |
| -느라(고)  | 口语      | ✗      | ✗         | 忙于→消极  |
| -는 바람에 | 口语      | ✗      | ✗         | 突发→消极  |
| -길래      | 口语      | ✓      | ✗         | 观察→行动  |
| -는 통에   | 口语      | ✗      | ✗         | 混乱→消极  |

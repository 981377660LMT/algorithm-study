# 敬语体系 (경어법)

> 韩语敬语分三大系统：主体敬语、客体敬语、相对敬语

---

## 一、主体敬语 (주체 높임법) ★★★★★

### 核心规则

当主语是需要尊敬的人（长辈、上级、老师等）时，谓语加 **-(으)시-**

### 接续

- 有收音 + -으시- / 无收音 + -시-
- ㄹ收音 → ㄹ脱落 + -시-

| 原形   | 敬语形             | 例句                        |
| ------ | ------------------ | --------------------------- |
| 가다   | 가**시**다         | 선생님이 가**십니다**.      |
| 먹다   | 먹다→드시다 ★      | 아버지가 밥을 **드십니다**. |
| 오다   | 오**시**다         | 할머니가 오**셨어요**.      |
| 읽다   | 읽**으시**다       | 어머니가 책을 읽**으세요**. |
| 살다   | 사**시**다(ㄹ脱)   | 부모님이 서울에 사**세요**. |
| 만들다 | 만드**시**다(ㄹ脱) | 어머니가 만드**셨어요**.    |
| 알다   | 아**시**다(ㄹ脱)   | 선생님이 아**세요**?        |

### 特殊敬语词 ★★★★★

| 普通          | 敬语                      | 翻译   |
| ------------- | ------------------------- | ------ |
| 먹다 / 마시다 | **드시다** / **잡수시다** | 吃/喝  |
| 자다          | **주무시다**              | 睡觉   |
| 있다          | **계시다**                | 在/有  |
| 없다          | **안 계시다**             | 不在   |
| 말하다        | **말씀하시다**            | 说话   |
| 죽다          | **돌아가시다**            | 去世   |
| 아프다        | **편찮으시다**            | 不舒服 |
| 나이          | **연세**                  | 年龄   |
| 이름          | **성함**                  | 名字   |
| 밥            | **진지**                  | 饭     |
| 집            | **댁**                    | 家     |
| 생일          | **생신**                  | 生日   |
| 말            | **말씀**                  | 话     |

### 主格敬语助词 ★★★

| 普通  | 敬语       |
| ----- | ---------- |
| 이/가 | **께서**   |
| 은/는 | **께서는** |

| 例句                                               |
| -------------------------------------------------- |
| 선생님**께서** 오셨습니다. (老师来了。)            |
| 할아버지**께서** 주무시고 계십니다. (爷爷在睡觉。) |

## 二、客体敬语 (객체 높임법) ★★★★

### 核心

当宾语/间接宾语是尊敬对象时使用

### 与格敬语助词

| 普通 | 敬语   |
| ---- | ------ |
| 에게 | **께** |
| 한테 | **께** |

### 特殊谦让动词 ★★★★

| 普通         | 客体敬语(谦让)          | 翻译           |
| ------------ | ----------------------- | -------------- |
| 주다         | **드리다**              | 给(尊敬对象)   |
| 데리다       | **모시다**              | 带领(尊敬对象) |
| 만나다       | **뵙다** (뵈다)         | 拜见           |
| 묻다         | **여쭙다** / **여쭈다** | 请问           |
| 말하다(对上) | **말씀드리다**          | 说(对上)       |
| 보다         | **뵙다**                | 拜见           |

| 例句                                   | 翻译             |
| -------------------------------------- | ---------------- |
| 선생님**께** 선물을 **드렸어요**.      | 给老师送了礼物。 |
| 할머니를 **모시고** 갔다.              | 带奶奶去了。     |
| 처음 **뵙겠습니다**.                   | 初次拜见。       |
| 선생님**께** **여쭤볼** 것이 있습니다. | 有想请教老师的。 |

## 三、相对敬语 (상대 높임법) ★★★★★

### 核心

根据**听话者**的身份选择语体

| 语体              | 使用对象      | 例子       |
| ----------------- | ------------- | ---------- |
| 합쇼체 (-ㅂ니다)  | 上级/正式场合 | 감사합니다 |
| 해요체 (-아/어요) | 初次/一般尊重 | 감사해요   |
| 해체 (-아/어)     | 朋友/晚辈     | 고마워     |
| 해라체 (-다)      | 书面/独白     | 고맙다     |

（详见文件07\_终结语尾）

## 四、敬语的综合运用 ★★★★★

### 场景1：对老师说话

| 要素     | 实现          |
| -------- | ------------- |
| 主体敬语 | -(으)시-      |
| 客体敬语 | 께, 드리다    |
| 相对敬语 | 합쇼체/해요체 |

> 선생님**께서** 말씀하**시**는 내용을 잘 이해했**습니다**.
> (老师说的内容我理解了。)

### 场景2：跟朋友说关于老师的事

| 要素     | 实现                  |
| -------- | --------------------- |
| 主体敬语 | -(으)시- (提到老师时) |
| 相对敬语 | 해체 (对朋友时)       |

> 선생님**이** 내일 오**신**대. (老师说明天来。)

### 场景3：给老板发消息

| 要素     | 实现        |
| -------- | ----------- |
| 全部使用 | 최고 존댓말 |

> 부장님**께서** 말씀하**신** 대로 보고서를 수정하여 **드리겠습니다**.
> (按部长您说的修改报告呈上。)

## 五、谦让语 (자기 낮춤) ★★★

| 普通   | 谦让           | 翻译       |
| ------ | -------------- | ---------- |
| 나     | **저**         | 我         |
| 우리   | **저희**       | 我们       |
| 먹다   | **먹겠습니다** | 我吃(客气) |
| 있다   | **있겠습니다** | 我在(客气) |
| 말하다 | **말씀드리다** | 我说(对上) |
| 보다   | **뵙다**       | 拜见       |
| 묻다   | **여쭙다**     | 请问       |
| 주다   | **드리다**     | 呈上       |
| 데리다 | **모시다**     | 陪同(上级) |

## 六、常用敬语表达 ★★★★★

| 表达                     | 翻译           | 场景           |
| ------------------------ | -------------- | -------------- |
| 안녕하세요.              | 你好。         | 通用问候       |
| 안녕하십니까?            | 您好。         | 正式问候       |
| 감사합니다.              | 谢谢。         | 感谢(正式)     |
| 죄송합니다.              | 抱歉。         | 道歉(正式)     |
| 실례합니다.              | 打扰了。       | 请求前         |
| 성함이 어떻게 되세요?    | 请问尊姓大名？ | 问名字         |
| 연세가 어떻게 되세요?    | 请问贵庚？     | 问年龄         |
| 어디에 사세요?           | 住在哪里？     | —              |
| 뭐 드시겠어요?           | 想吃/喝什么？  | 餐厅           |
| 잘 먹겠습니다.           | 我开动了。     | 吃饭前         |
| 잘 먹었습니다.           | 我吃好了。     | 吃饭后         |
| 진지 잡수셨어요?         | 您用过饭了吗？ | 问长辈         |
| 안녕히 가세요.           | 请慢走。       | 送别(留的人说) |
| 안녕히 계세요.           | 请留步。       | 告别(走的人说) |
| 주무셨어요?              | 睡得好吗？     | 早安(对长辈)   |
| 편찮으세요?              | 不舒服吗？     | 问健康         |
| 말씀 좀 여쭤봐도 될까요? | 能请教一下吗？ | 提问前         |

## 七、敬语使用禁忌 ★★

| 错误                              | 原因               | 正确                     |
| --------------------------------- | ------------------ | ------------------------ |
| ~~저는 가시겠습니다~~             | 不能对自己用 -시-  | 저는 가겠습니다          |
| ~~우리 팀장님에게 말했다~~ (对外) | 对外不给自己人加敬 | 저희 팀장에게 말했습니다 |
| ~~커피 드시겠습니까?~~ (对同级)   | 过度使用敬语       | 커피 마실래요?           |
| ~~할아버지 밥 먹었어?~~           | 对长辈用半语       | 할아버지 진지 드셨어요?  |

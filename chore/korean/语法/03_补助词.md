# 补助词 (보조사)

> 补助词不标记语法角色，而是添加额外语义（主题、也、只、甚至…）
> 可以替代或叠加在格助词之后

---

## 一、은/는 — 主题/对比 ★★★★★

### 基本形式

- 有收音 + **은** / 无收音 + **는**

### 用法1：主题标记（已知信息）

| 例句                               | 说明                                 |
| ---------------------------------- | ------------------------------------ |
| 저**는** 한국어를 배우고 있습니다. | "我"是话题：关于我嘛，（我）在学韩语 |
| 한국**은** 사계절이 뚜렷합니다.    | "韩国"是话题                         |
| 이 책**은** 매우 재미있습니다.     | "这本书"是话题                       |

### 用法2：对比/强调 ★

| 例句                                     | 翻译                     | 对比点           |
| ---------------------------------------- | ------------------------ | ---------------- |
| 사과**는** 좋아하지만 배**는** 싫어해요. | 苹果喜欢，但梨不喜欢。   | 苹果 vs 梨       |
| 읽기**는** 잘하지만 쓰기**는** 못해요.   | 阅读擅长，但写作不行。   | 读 vs 写         |
| 오늘**은** 쉬겠습니다.                   | 今天（跟平时不同）休息。 | 今天 vs 其他时候 |

### 은/는 vs 이/가 对比 ★★★★★

| 情况            | 이/가 | 은/는 |
| --------------- | ----- | ----- |
| 新信息/初次提及 | ✓     | ✗     |
| 已知信息/话题   | ✗     | ✓     |
| 回答"谁/什么"   | ✓     | ✗     |
| 对比两个事物    | ✗     | ✓     |
| 描述/说明       | ✗     | ✓     |

经典对话：

> A: 누가 학생이에요? → B: **철수가** 학생이에요. (이/가 = 新信息)
> A: 철수는 뭐 해요? → B: **철수는** 학생이에요. (은/는 = 已知话题)

## 二、도 — 也/都 ★★★★★

| 例句                             | 翻译               |
| -------------------------------- | ------------------ |
| 나**도** 가고 싶다.              | 我也想去。         |
| 여기**도** 좋고 저기**도** 좋다. | 这里也好那里也好。 |
| 하나**도** 없다.                 | 一个都没有。       |
| 아무것**도** 모른다.             | 什么都不知道。     |

### 特殊组合

| 组合            | 意思     | 例子                  |
| --------------- | -------- | --------------------- |
| 하나도 + 否定   | 一个都不 | 하나**도** 안 먹었다. |
| 아무도 + 否定   | 谁都不   | 아무**도** 없었다.    |
| 아무것도 + 否定 | 什么都不 | 아무것**도** 몰랐다.  |
| 어디도 + 否定   | 哪里都不 | 어디**도** 안 갔다.   |
| 조금도 + 否定   | 一点都不 | 조금**도** 안 아프다. |

## 三、만 — 只/仅 ★★★★

| 例句                      | 翻译                 |
| ------------------------- | -------------------- |
| 나**만** 알고 있다.       | 只有我知道。         |
| 커피**만** 마신다.        | 只喝咖啡。           |
| 한 번**만** 더 해 보세요. | 再试一次就好。       |
| 이것**만** 있으면 된다.   | 只要有这个就行。     |
| 공부**만** 한다.          | 只学习（不做别的）。 |

### 만 + 其他助词

| 组合   | 例子                         | 翻译               |
| ------ | ---------------------------- | ------------------ |
| 에서만 | 한국**에서만** 살았다.       | 只在韩国住过。     |
| 에게만 | 너**에게만** 말할게.         | 只跟你说。         |
| 으로만 | 버스**로만** 다닌다.         | 只坐公交。         |
| 만이   | 시간**만이** 해결할 수 있다. | 只有时间才能解决。 |

## 四、까지 — 到/甚至 ★★★★

### 用法1：范围终点（到...为止）

| 例句                        | 翻译           |
| --------------------------- | -------------- |
| 서울**까지** 얼마나 걸려요? | 到首尔要多久？ |
| 5시**까지** 오세요.         | 请5点之前来。  |
| 처음부터 끝**까지**         | 从头到尾       |

### 用法2：甚至/连...都 ★

| 例句                        | 翻译               |
| --------------------------- | ------------------ |
| 친구**까지** 나를 배신했다. | 连朋友都背叛我了。 |
| 비**까지** 오기 시작했다.   | 甚至开始下雨了。   |
| 밥**까지** 못 먹었다.       | 连饭都没吃。       |

| 例句                      | 翻译         |

| 아침**부터** 저녁**까지** | 从早到晚     |

## 六、밖에 — 只（+ 否定） ★★★★

**必须搭配否定形式使用**

| 例句                          | 翻译           |
| ----------------------------- | -------------- |
| 시간이 10분**밖에** 없다.     | 只有10分钟。   |
| 돈이 천 원**밖에** 안 남았다. | 只剩一千元。   |
| 한국어**밖에** 모른다.        | 只懂韩语。     |
| 혼자**밖에** 못 간다.         | 只能一个人去。 |
| 이것**밖에** 할 수 없다.      | 只能做这个。   |

### 만 vs 밖에 区别 ★

|      | 만                           | 밖에                                   |
| ---- | ---------------------------- | -------------------------------------- |
| 搭配 | 肯定/否定都可                | 必须 + 否定                            |
| 语气 | 中性                         | 含"不够/遗憾"                          |
| 例子 | 커피**만** 마신다 (只喝咖啡) | 커피**밖에** 없다 (只有咖啡[别的没有]) |

## 七、마저 — 连...都/甚至 ★★★

| 例句                          | 翻译                   |
| ----------------------------- | ---------------------- |
| 마지막 희망**마저** 사라졌다. | 连最后的希望都消失了。 |
| 건강**마저** 잃었다.          | 连健康都失去了。       |
| 부모님**마저** 반대했다.      | 连父母都反对了。       |
| 남은 돈**마저** 다 썼다.      | 连剩下的钱都花光了。   |

## 八、조차 — 连...都（更强烈） ★★★

| 例句                         | 翻译                 |
| ---------------------------- | -------------------- |
| 이름**조차** 모른다.         | 连名字都不知道。     |
| 밥 먹을 시간**조차** 없다.   | 连吃饭的时间都没有。 |
| 가족**조차** 몰랐다.         | 连家人都不知道。     |
| 기본적인 것**조차** 안 된다. | 连基本的都不行。     |

### 까지 vs 마저 vs 조차 区别 ★★★

|      | 까지            | 마저                | 조차                |
| ---- | --------------- | ------------------- | ------------------- |
| 语气 | 中性~强调       | 遗憾/消极           | 极端消极            |
| 搭配 | 肯定/否定       | 多搭配消极          | 几乎只搭配消极      |
| 意思 | 连/甚至         | 连...也             | 连...都(最不可能的) |
| 例子 | 비**까지** 왔다 | 친구**마저** 떠났다 | 이름**조차** 모른다 |

## 九、(이)나 — 或/大约/竟 ★★★

### 用法1：或者（选择）

- 커피**나** 차를 마시다. (喝咖啡或茶。)
- 영화**나** 볼까? (看个电影什么的？)

### 用法2：大约/多达

- 세 시간**이나** 기다렸다. (等了足足三小时。)
- 열 명**이나** 왔다. (竟然来了十个人。)

### 用法3：表示不满/退步（随便...算了）

- 집에서 텔레비전**이나** 보자. (在家随便看看电视吧。)
- 밥**이나** 먹으러 가자. (随便去吃个饭吧。)

## 十四、(으)로서 vs (으)로써 ★★★

### (으)로서 — 作为（身份/资格）

| 例句                                 | 翻译                   |
| ------------------------------------ | ---------------------- |
| 학생**으로서** 열심히 공부해야 한다. | 作为学生应该努力学习。 |
| 친구**로서** 충고한다.               | 作为朋友忠告你。       |
| 선생님**으로서** 책임감이 있다.      | 作为老师有责任感。     |

### (으)로써 — 用...（手段/工具）

| 例句                          | 翻译                 |
| ----------------------------- | -------------------- |
| 노력**으로써** 성공했다.      | 通过努力成功了。     |
| 대화**로써** 문제를 해결했다. | 通过对话解决了问题。 |
| 이것**으로써** 끝이다.        | 到此(以此)结束。     |

### 区分口诀

- **로서** = **身份** (서 → 서다 站 → 站在什么身份上)
- **로써** = **工具** (써 → 쓰다 使用 → 用什么手段)

## 十五、补助词总览

| 补助词   | 意思          | 替代的格助词 | 例子                |
| -------- | ------------- | ------------ | ------------------- |
| 은/는    | 主题/对比     | 이/가, 을/를 | 나**는** 학생이다   |
| 도       | 也/都         | 이/가, 을/를 | 나**도** 간다       |
| 만       | 只            | 이/가, 을/를 | 나**만** 안다       |
| 밖에     | 只(+否定)     | 이/가, 을/를 | 이것**밖에** 없다   |
| 까지     | 到/甚至       | 叠加         | 서울**까지**        |
| 부터     | 从            | 叠加         | 9시**부터**         |
| 마저     | 连...都       | 이/가, 을/를 | 희망**마저** 없다   |
| 조차     | 连...都(极端) | 이/가, 을/를 | 이름**조차** 모른다 |
| (이)나   | 或/竟         | 이/가, 을/를 | 커피**나** 마시자   |
| (이)라도 | 哪怕          | 이/가, 을/를 | 물**이라도** 마셔   |
| 마다     | 每            | 叠加         | 날**마다**          |
| 대로     | 按照          | 叠加         | 계획**대로**        |

# 表达习惯·经验·回忆 (습관·경험·회상 표현)

---

## 一、-곤 하다 (常常/时不时) ★★★★★

表示习惯性或反复发生的行为

| 例句                                  | 翻译               |
| ------------------------------------- | ------------------ |
| 주말에 영화를 보**곤 해요**.          | 周末常常看电影。   |
| 어렸을 때 할머니 집에 가**곤 했다**.  | 小时候常去奶奶家。 |
| 가끔 혼자 산책하**곤 한다**.          | 偶尔一个人散步。   |
| 학생 때 밤늦게까지 공부하**곤 했다**. | 学生时常熬夜学习。 |

---

## 二、-(으)ㄴ 적(이) 있다/없다 (做过/没做过) ★★★★★

经验表达

| 例句                            | 翻译             |
| ------------------------------- | ---------------- |
| 한국에 간 **적이 있어요**.      | 去过韩国。       |
| 김치를 만들어 본 **적이 있다**. | 做过泡菜。       |
| 한 번도 늦은 **적이 없다**.     | 一次都没迟到过。 |
| 그런 말을 들은 **적이 없어요**. | 从没听过那种话。 |

### -아/어 본 적이 있다 (试做过)

| 例句                               | 翻译           |
| ---------------------------------- | -------------- |
| 스키를 타 **본 적이 있어요**.      | 滑过雪。       |
| 일본 음식을 먹어 **본 적이 있다**. | 吃过日本料理。 |
| 번지점프를 해 **본 적이 없다**.    | 没蹦过极。     |

---

## 三、-더라 / -더라고(요) (回忆叙述) ★★★★★

回忆自己亲身经历或观察的事实

### -더라 (반말/独白)

| 例句                     | 翻译                   |
| ------------------------ | ---------------------- |
| 그 영화 재미있**더라**.  | (我看了)那电影挺有趣。 |
| 한국 음식이 맵**더라**.  | 韩国菜确实辣。         |
| 그 사람 키가 크**더라**. | 那个人确实挺高。       |

### -더라고(요) (해요체)

| 例句                             | 翻译                 |
| -------------------------------- | -------------------- |
| 가 보니까 사람이 많**더라고요**. | 去了发现人很多。     |
| 맛있**더라고요**.                | (尝了)挺好吃的。     |
| 생각보다 어렵**더라고요**.       | 比想象难。           |
| 아이가 벌써 많이 컸**더라고요**. | 孩子已经长大了很多。 |

### 注意事项

- 必须是说话者亲眼所见/亲身经历
- 主语一般是**第三人称**（不能说自己的主观感情时用 -더라）
- 例外：새로 알게 된 자기 감정 → 나도 그게 좋**더라고요** (我也觉得那个好)

---

## 四、-던 (回忆中的/过去未完的) ★★★★

过去某状态/进行中的事物（冠形词形）

| 例句                  | 翻译           |
| --------------------- | -------------- |
| 어렸을 때 살**던** 집 | 小时候住的房子 |
| 자주 가**던** 식당    | 常去的餐厅     |
| 읽**던** 책           | (读了一半的)书 |
| 좋아하**던** 노래     | 以前喜欢的歌   |

### -던 vs -았/었던

| -던                        | -았/었던                       |
| -------------------------- | ------------------------------ |
| 未完成/习惯                | 已完成(隔得更远)               |
| 읽**던** 책 (读了一半的书) | 읽**었던** 책 (之前读完的书)   |
| 가**던** 식당 (常去的餐厅) | 갔**던** 식당 (之前去过的餐厅) |

---

## 五、-더니 (回忆→结果) ★★★★

### (1) 第三人称：观察到…所以/结果

| 例句                              | 翻译                         |
| --------------------------------- | ---------------------------- |
| 아까 울**더니** 지금은 웃는다.    | 刚才还哭，现在笑了。         |
| 열심히 공부하**더니** 1등을 했다. | 一直用功学习，结果得了第一。 |
| 많이 먹**더니** 배탈이 났다.      | 吃了很多，结果闹肚子了。     |

### (2) 第一人称：过去做了…现在

| 예문                              | 翻译                 |
| --------------------------------- | -------------------- |
| 어제 늦게 자**더니** 지금 졸리다. | 昨天晚睡，现在犯困。 |
| 많이 걸었**더니** 다리가 아프다.  | 走了很多路，腿疼了。 |

---

## 六、-았/었더라면 (如果当时…的话) ★★★★

对过去的后悔/假设

| 例句                                     | 翻译                           |
| ---------------------------------------- | ------------------------------ |
| 좀 더 열심히 했**더라면** 합격했을 텐데. | 如果当时再努力一点就会合格了。 |
| 그때 말했**더라면** 달라졌을 거야.       | 如果那时候说了就会不一样了。   |
| 일찍 출발했**더라면** 안 늦었을 거다.    | 如果早出发就不会迟到了。       |

---

## 七、-(으)ㄴ/는 편이다 (算是/偏向) ★★★★

| 例句                             | 翻译             |
| -------------------------------- | ---------------- |
| 매운 음식을 잘 먹**는 편이다**.  | 算是能吃辣的。   |
| 한국어 실력이 좋**은 편이에요**. | 韩语水平算好的。 |
| 운동을 자주 하**는 편이에요**.   | 算是经常运动的。 |
| 키가 큰 **편이다**.              | 算是高的。       |
| 성격이 조용한 **편이에요**.      | 性格算安静的。   |

---

## 八、-게 마련이다 (必然/注定) ★★★

表示必然的结果

| 例句                               | 翻译                   |
| ---------------------------------- | ---------------------- |
| 노력하면 성공하**게 마련이다**.    | 努力就必然会成功。     |
| 시간이 지나면 잊**게 마련이다**.   | 时间过了就注定会忘。   |
| 거짓말을 하면 들키**게 마련이다**. | 说谎的话注定会被发现。 |
| 사람은 실수하**게 마련이다**.      | 人注定会犯错。         |

---

## 九、-(으)ㄴ/는 척하다 (假装) ★★★★

| 例句                | 翻译         |
| ------------------- | ------------ |
| 모르**는 척했다**.  | 假装不知道。 |
| 자**는 척하다**.    | 假装在睡觉。 |
| 안 들은 **척하다**. | 假装没听到。 |
| 괜찮은 **척하다**.  | 假装没事。   |
| 바**쁜 척하다**.    | 假装很忙。   |

---

## 十、-다(가) 보니(까) (做着做着发现) ★★★★

在做某事的过程中发现/得到结果

| 例句                                                    | 翻译                               |
| ------------------------------------------------------- | ---------------------------------- |
| 살**다 보니** 좋은 일도 있다.                           | 活着活着也有好事。                 |
| 한국어를 공부하**다 보니** 한국 문화에도 관심이 생겼다. | 学着韩语，也对韩国文化产生了兴趣。 |
| 혼자 여행하**다 보니** 자신감이 생겼다.                 | 一个人旅行着旅行着，产生了自信。   |

---

## 十一、-(으)ㄹ 때마다 (每次…的时候) ★★★

| 例句                                | 翻译                     |
| ----------------------------------- | ------------------------ |
| 그 노래를 들**을 때마다** 생각나요. | 每次听那首歌都会想起来。 |
| 만날 **때마다** 반갑다.             | 每次见面都很高兴。       |
| 시험을 볼 **때마다** 긴장된다.      | 每次考试都紧张。         |

---

## 十二、-아/어 본 결과 (试了之后结果) ★★★

| 例句                                      | 翻译                             |
| ----------------------------------------- | -------------------------------- |
| 여러 방법을 써 **본 결과** 이게 최선이다. | 试了多种方法之后，这个是最好的。 |
| 조사해 **본 결과** 문제가 없었다.         | 调查之后没有问题。               |

---

## 十三、-기도 하다 (也做…/有时…) ★★★★

### (1) 有时也…

| 例句                         | 翻译                 |
| ---------------------------- | -------------------- |
| 주말에 등산**하기도 해요**.  | 周末也有时爬山。     |
| 혼자 영화를 보**기도 한다**. | 有时也一个人看电影。 |

### (2) 强调确实…

| 例句                    | 翻译         |
| ----------------------- | ------------ |
| 정말 힘들**기도 하다**. | 确实很辛苦。 |
| 보고 싶**기도 하다**.   | 也确实想念。 |

---

## 十四、总览

| 表达              | 核心           | 例句                 |
| ----------------- | -------------- | -------------------- |
| -곤 하다          | 常常(习惯)     | 산책하곤 했다        |
| -(으)ㄴ 적이 있다 | 做过(经验)     | 가 본 적이 있다      |
| -더라고요         | 回忆叙述       | 맛있더라고요         |
| -던               | 回忆中的(冠形) | 살던 집              |
| -더니             | 回忆→结果      | 울더니 웃는다        |
| -았/었더라면      | 如果当时…      | 했더라면 좋았을 텐데 |
| -(으)ㄴ/는 편이다 | 算是/偏向      | 좋은 편이다          |
| -게 마련이다      | 必然           | 잊게 마련이다        |
| -는 척하다        | 假装           | 모르는 척하다        |
| -다 보니          | 做着发现       | 하다 보니 알게 됐다  |
| -기도 하다        | 也有时/确实    | 하기도 한다          |

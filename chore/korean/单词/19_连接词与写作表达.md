# TOPIK II 高级 连接词与写作表达 (~200个)

> 写作(쓰기)必备表达，尤其是53题图表题和54题议论文
> ★ = 写作高频/必背

---

## 一、接续词/连接副词 (접속 부사)

### A. 顺接/因果

| 표현        | 含义         | 用法说明         |
| ----------- | ------------ | ---------------- |
| 따라서      | 因此 ★       | 前因后果，书面语 |
| 그러므로    | 因此 ★       | 比따라서更正式   |
| 그래서      | 所以         | 口语化，写作少用 |
| 그 결과     | 结果 ★       | 引出结果         |
| 이에 따라   | 据此 ★       | 根据前文         |
| 이로 인해   | 因此 ★       | 由此导致         |
| 그런 까닭에 | 因为那个缘故 |                  |
| 결국        | 结局 ★       | 最终             |
| 결과적으로  | 结果上 ★     |                  |

### B. 转折/对比

| 표현      | 含义       | 用法说明     |
| --------- | ---------- | ------------ |
| 그러나    | 然而 ★     | 最正式的转折 |
| 하지만    | 但是 ★     | 稍口语       |
| 그렇지만  | 但是       |              |
| 그런데    | 可是       | 较口语       |
| 반면(에)  | 反面 ★     | 对比两个方面 |
| 한편      | 另一方面 ★ | 引出另一观点 |
| 이에 반해 | 与此相反 ★ |              |
| 그에 비해 | 与此相比 ★ |              |
| 오히려    | 反而 ★     |              |

### C. 并列/递进

| 표현        | 含义       | 用法说明       |
| ----------- | ---------- | -------------- |
| 또한        | 此外 ★     | 补充说明       |
| 게다가      | 而且 ★     | 递进           |
| 더불어      | 同时 ★     |                |
| 아울러      | 同时 ★     | 比더불어更正式 |
| 나아가      | 进而 ★     | 更进一步       |
| 뿐만 아니라 | 不仅 ★     |                |
| 이 밖에도   | 此外 ★     |                |
| 더 나아가   | 更进一步 ★ |                |
| 마찬가지로  | 同样地 ★   |                |

### D. 举例/补充

| 표현          | 含义       | 用法说明 |
| ------------- | ---------- | -------- |
| 예를 들어     | 例如 ★     | 最常用   |
| 예컨대        | 例如 ★     | 更正式   |
| 이를테면      | 比如说     |          |
| 가령          | 假如，比如 |          |
| 특히          | 特别是 ★   |          |
| 구체적으로    | 具体来说 ★ |          |
| 즉            | 即 ★       | 解释     |
| 다시 말해(서) | 换句话说 ★ |          |
| 바꿔 말하면   | 换个说法   |          |

### E. 总结/归纳

| 표현                      | 含义           | 用法说明   |
| ------------------------- | -------------- | ---------- |
| 결론적으로                | 结论性地 ★     | 结尾段必用 |
| 종합하면                  | 综合来看 ★     |            |
| 요컨대                    | 总之 ★         |            |
| 요약하면                  | 概括地说       |            |
| 정리하면                  | 整理来看       |            |
| 이상에서 살펴본 바와 같이 | 如上所述 ★     |            |
| 지금까지 살펴본 것처럼    | 如目前所看到的 |            |

## 二、写作专用句型

### A. 开头 (서론) 表达

| 표현                                              | 含义/用途                      |
| ------------------------------------------------- | ------------------------------ |
| 최근/오늘날 ...이/가 사회적 관심을 받고 있다      | 最近...受到了社会关注 ★        |
| ...에 대한 논의가 활발하게 이루어지고 있다        | 关于...的讨论正在活跃地进行 ★  |
| ...은/는 현대 사회의 중요한 과제이다              | ...是现代社会的重要课题 ★      |
| ...이/가 큰 화제가 되고 있다                      | ...成为了热门话题              |
| ...에 대해 다양한 의견이 제시되고 있다            | 关于...提出了多样的意见 ★      |
| ...은/는 우리 사회에서 빈번하게 논의되는 주제이다 | ...是我们社会中频繁讨论的话题  |
| 이 글에서는 ...에 대해 살펴보고자 한다            | 本文将探讨... ★                |
| ...에 대해 긍정적/부정적 시각이 공존하고 있다     | 关于...存在着正面/负面的看法 ★ |

### B. 表达观点

| 표현                          | 含义/用途         |
| ----------------------------- | ----------------- |
| ...다고 생각한다 / 본다       | 我认为... ★       |
| ...다고 할 수 있다            | 可以说... ★       |
| ...다고 여겨진다              | 被认为...         |
| ...다고 판단된다              | 被判断为... ★     |
| ...이/가 바람직하다           | ...是可取的 ★     |
| ...이/가 필요하다/필요로 하다 | ...是必要的 ★     |
| ...이/가 불가피하다           | ...是不可避免的 ★ |
| 필자는 ...다는 입장이다       | 笔者的立场是...   |

### C. 论证表达

| 표현                              | 含义/用途              |
| --------------------------------- | ---------------------- |
| 우선/먼저                         | 首先 ★                 |
| ...기 때문이다                    | 因为... ★              |
| 그 이유는 ...기 때문이다          | 其原因是因为... ★      |
| ...(으)ㄹ 뿐만 아니라 ...도       | 不仅...而且... ★       |
| ...는 점에서                      | 从...的角度来看 ★      |
| ...는 측면에서 볼 때              | 从...方面来看          |
| ...는 반면(에)                    | ...的另一面 ★          |
| 이는 ...다는 것을 의미한다        | 这意味着... ★          |
| 이를 통해 ...다는 것을 알 수 있다 | 由此可知... ★          |
| ...다는 조사/연구 결과가 있다     | 有...的调查/研究结果 ★ |
| 실제로 ...한 사례가 있다          | 实际上有...的事例 ★    |

### D. 反驳/让步

| 표현                                     | 含义/用途                   |
| ---------------------------------------- | --------------------------- |
| 물론 ...다는 반론/의견도 있다            | 当然也有...的反论/意见 ★    |
| 일부에서는 ...다고 주장하기도 한다       | 一些人也主张... ★           |
| 이러한 주장은 ...다는 점에서 한계가 있다 | 这种主张在...方面存在局限 ★ |
| ...다고 볼 수도 있지만                   | 虽然也可以看作...但是 ★     |
| 확실히 ...한 측면이 있다. 그러나...      | 确实有...的一面。但是... ★  |
| 비록 ...다 하더라도                      | 即使...也 ★                 |
| 설령 ...다 할지라도                      | 纵使...也                   |

### E. 结尾 (결론) 表达

| 표현                                     | 含义/用途                     |
| ---------------------------------------- | ----------------------------- |
| 결론적으로 ...는 것이 바람직하다         | 总之...是可取的 ★             |
| 이상의 내용을 종합하면                   | 综合以上内容 ★                |
| 앞으로 ...기 위해서는 ~이/가 필요하다    | 今后为了...需要... ★          |
| (개인/사회/정부)의 노력이 요구된다       | 需要(个人/社会/政府)的努力 ★  |
| ...에 대한 사회적 논의가 지속되어야 한다 | 关于...的社会讨论应该持续下去 |
| ...기 위한 제도적 장치가 마련되어야 한다 | 应该建立...的制度性措施 ★     |
| 이를 위해 다각적인 노력이 필요할 것이다  | 为此需要多方面的努力 ★        |

## 三、53题 图表描述专用表达

### A. 调查介绍

| 표현                                    | 含义                        |
| --------------------------------------- | --------------------------- |
| ...에 대한 조사/설문 결과이다           | 是关于...的调查/问卷结果 ★  |
| ...을/를 대상으로 조사한 결과           | 以...为对象进行调查的结果 ★ |
| 아래 표/그래프는 ...을/를 나타낸 것이다 | 下表/图展示了... ★          |

### B. 数据描述

| 표현                                       | 含义                       |
| ------------------------------------------ | -------------------------- |
| ...이/가 가장 높은/낮은 것으로 나타났다    | ...被表明为最高/最低 ★     |
| ...%를 차지하다/차지하는 것으로 조사되었다 | 占...% ★                   |
| ...는 ...%인 반면, ...는 ...%에 불과하다   | ...是...%，而...仅为...% ★ |
| ...에서 ...로 증가/감소하였다              | 从...增加/减少到... ★      |
| ...에 비해 ...% 높게/낮게 나타났다         | 比...高/低...% ★           |
| 전년 대비 ...% 증가/감소하였다             | 与去年相比增加/减少了...%  |

### C. 变化趋势

| 표현                                         | 含义                           |
| -------------------------------------------- | ------------------------------ |
| 꾸준히/지속적으로 증가하고 있다              | 持续增加 ★                     |
| 점차 감소하는 추세이다                       | 正在逐渐减少 ★                 |
| 급격히/급속히 늘어났다                       | 急剧增加 ★                     |
| 완만한 증가세를 보이고 있다                  | 呈缓慢增长趋势                 |
| ...년을 기점으로 증가/감소 추세로 전환되었다 | 以...年为转折点转为增/减趋势 ★ |
| 큰 변화 없이 유지되고 있다                   | 没有大变化地维持着             |

### D. 原因与展望

| 표현                                               | 含义                              |
| -------------------------------------------------- | --------------------------------- |
| 이러한 결과가 나타난 이유는 ...기 때문으로 보인다  | 出现这种结果的原因看来是因为... ★ |
| 이는 ...에 따른 것으로 분석된다                    | 这被分析为是由于...               |
| 향후 이러한 추세는 계속될 것으로 전망된다          | 今后这种趋势预计将持续 ★          |
| 앞으로 ...에 대한 관심이 더욱 높아질 것으로 보인다 | 今后对...的关注预计将更高         |

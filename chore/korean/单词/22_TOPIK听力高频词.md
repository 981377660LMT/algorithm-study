| 어휘             | 含义           | 出现场景                            |

| 일치하는 것      | 一致的         | "내용과 일치하는 것" ★              |
| 같은 것          | 相同的         | ★                                   |
| 다른 것          | 不同的         | ★                                   |
| 맞는 것          | 正确的         | ★                                   |
| 이유             | 原因/理由      | ★                                   |
| 목적             | 目的           | ★                                   |
| 태도             | 态度           | "남자의 태도로 알맞은 것" ★         |
| 의도             | 意图           | ★                                   |

# TOPIK II 高级词汇学习方法

## 一、记忆策略

### 1. 间隔重复法 (Spaced Repetition)

- 使用 **Anki** 制作单词卡片
- 正面：韩语单词 + 例句；背面：中文意思 + 词性
- 每日节奏：**新词 30个 + 复习 100个**
- 艾宾浩斯复习周期：学习后 → 1天 → 2天 → 4天 → 7天 → 15天 → 30天

### 2. 汉字词突破法 (中文母语者最大优势)

- 韩语约 **60%~70%** 是汉字词
- 系统学习"汉字 → 韩语音"对应规律，一通百通
- 例：学(학), 生(생), 教(교), 育(육) → 교육(教育), 학생(学生)
- 参见 `09_汉字词对照表.md`

### 3. 词根词缀法

- 掌握高频接头词/接尾词，快速猜测生词含义
- 例：불(不) + 가능(可能) = 불가능(不可能)
- 参见 `08_接头词接尾词.md`

### 4. 主题聚类法

- 按话题分组记忆，形成语义网络
- 如"环境"主题：오염, 배출, 재활용, 생태계, 멸종...
- 本文件夹按主题分类，逐个攻破

### 5. 语境记忆法

- 每个单词至少记1个例句
- 在真题中标注生词，回到语境复习
- 阅读韩语新闻/文章时遇到的生词立即记录

## 二、每日学习流程

```
1. 晨间 (15min)：Anki 复习昨日单词
2. 通勤 (15min)：听单词音频/泛听有声词汇
3. 午间 (15min)：学习新词 15个 (看本文件夹对应主题)
4. 晚间 (15min)：学习新词 15个 + Anki 录入
5. 睡前 (10min)：快速过一遍今日新词
```

## 三、阶段目标

| 阶段  | 目标词汇量  | 重点内容                             |
| ----- | ----------- | ------------------------------------ |
| 第1月 | 3000 → 4000 | 高频动词/形容词/副词 + 汉字词规律    |
| 第2月 | 4000 → 5000 | 社会/经济/教育/文化 主题名词         |
| 第3月 | 5000 → 6000 | 科技/环境/医学 + 惯用语/俗语         |
| 第4月 | 6000 → 7000 | 四字成语 + 接头词接尾词 + 拟声拟态词 |
| 第5月 | 7000 → 8000 | 写作高级表达 + 易混淆词辨析          |
| 第6月 | 8000+       | 全面复习 + 查漏补缺                  |

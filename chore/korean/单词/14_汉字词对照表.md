# TOPIK II 高级 汉字词对照表 (~300组)

> 中文母语者最大优势：韩语60-70%是汉字词
> 掌握汉字音对应规律，遇到生词可以直接推断含义
> ★ = 高收益汉字音

## 三、快速推测技巧

### 常见双字组合模式

| 模式   | 韩语  | 中文  | 例                               |
| ------ | ----- | ----- | -------------------------------- |
| X + 화 | ...化 | ...化 | 세계화, 도시화, 산업화, 자동화 ★ |
| X + 성 | ...性 | ...性 | 가능성, 필요성, 중요성, 다양성 ★ |
| X + 적 | ...的 | ...的 | 사회적, 경제적, 근본적, 실질적 ★ |
| X + 력 | ...力 | ...力 | 경쟁력, 영향력, 노동력, 생산력 ★ |
| X + 률 | ...率 | ...率 | 성장률, 실업률, 출생률, 투표률 ★ |
| X + 감 | ...感 | ...感 | 책임감, 만족감, 소속감, 위기감 ★ |
| 불 + X | 不... | 不... | 불안, 불법, 불가능, 불합리 ★     |
| 무 + X | 無... | 无... | 무관심, 무책임, 무의미, 무한 ★   |
| 비 + X | 非... | 非... | 비정상, 비합리, 비공식 ★         |
| 재 + X | 再... | 再... | 재활용, 재건, 재발 ★             |

# TOPIK II 写作高频词 쓰기 고빈도 어휘 (~250个)

> 写作（特别是 51~54 题）中需要主动使用的词汇与表达
> ★ = 쓰기 必须会写级别

---

## A. 开头段 (서론) 表达

| 표현                                   | 含义                     | 用途           |
| -------------------------------------- | ------------------------ | -------------- |
| 현대 사회에서                          | 在现代社会中             | 引出话题 ★     |
| 오늘날                                 | 如今                     | ★              |
| 최근 들어                              | 最近以来                 | ★              |
| 과거에 비해                            | 与过去相比               | 对比引入 ★     |
| 일반적으로                             | 一般来说                 | ★              |
| 흔히                                   | 常常/通常                | ★              |
| ~에 대한 관심이 높아지고 있다          | 对...的关注在增加        | 引出社会现象 ★ |
| ~이/가 중요한 문제로 대두되고 있다     | ...成为重要问题          | ★              |
| ~에 대해 다양한 의견이 있다            | 关于...有各种意见        | 议论文 ★       |
| ~을/를 둘러싸고 논의가 이루어지고 있다 | 围绕...正在进行讨论      | ★              |
| ~의 의미/중요성에 대해 살펴보고자 한다 | 想要探讨...的意义/重要性 | 点明主旨 ★     |
| ~에 대해 논의해 보겠다                 | 就...进行讨论            | ★              |

## B. 正文段 (본론) 展开

| 표현        | 含义     | 用途       |
| ----------- | -------- | ---------- |
| 먼저        | 首先     | 第一论点 ★ |
| 우선        | 首先     | ★          |
| 첫째        | 第一     | ★          |

| 더 나아가   | 更进一步 | ★          |

### 举例/论证

| 표현                     | 含义            | 用途       |

| ~을/를 통해 알 수 있듯이 | 如通过...可知   | ★          |
| 이는 ~을/를 의미한다     | 这意味着...     | ★          |
| ~(이)라는 점에서         | 从...这一点来看 | ★          |

### 对比/转折

| 표현              | 含义      | 用途             |
| ----------------- | --------- | ---------------- |
| 그러나            | 但是/然而 | 转折 ★           |
| 하지만            | 但是      | 口语化，写作少用 |
| 반면(에)          | 另一方面  | 对比 ★           |
| 이와 달리         | 与此不同  | ★                |
| 한편              | 另一方面  | ★                |
| 그럼에도 불구하고 | 尽管如此  | 让步 ★           |
| ~에도 불구하고    | 尽管      | ★                |
| 물론 ~지만        | 当然...但 | 让步 ★           |

### 因果

| 표현              | 含义       | 用途   |
| ----------------- | ---------- | ------ |
| 따라서            | 因此       | ★      |

| 이로 인해         | 由此       | ★      |
| 그 결과           | 结果       | ★      |
| ~(으)므로         | 因为(书面) | ★      |
| ~기 때문이다      | 因为       | ★      |
| ~(으)ㄴ/는 까닭에 | 由于(书面) | ★      |
| ~에 기인하다      | 归因于     | 高级 ★ |

## F. 书面语替换词 (口语→写作)

| 口语      | 写作替换                        | 含义      |
| --------- | ------------------------------- | --------- |
| 그래서    | 따라서/그러므로                 | 因此 ★    |
| 그런데    | 그러나/하지만                   | 但是 ★    |
| 아주/너무 | 매우/상당히                     | 非常 ★    |
| 많이      | 상당히/다수                     | 很多 ★    |
| 좋다      | 바람직하다/효과적이다           | 好 ★      |
| 나쁘다    | 부정적이다/부적절하다           | 不好 ★    |
| 해야 한다 | ~해야 할 것이다/~할 필요가 있다 | 应该 ★    |
| 것 같다   | ~(으)ㄹ 것으로 보인다           | 看起来 ★  |
| 이렇게    | 이러한 방식으로/이와 같이       | 这样 ★    |

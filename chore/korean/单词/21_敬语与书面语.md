# TOPIK II 高级 敬语与书面语 (~150个)

> TOPIK写作必须使用正式书面体 (-ㅂ니다/습니다体 或 -다体)
> 阅读理解中也常出现书面语/文语表达
> ★ = 写作/阅读高频

---

## 一、口语 → 书面语 对照

### A. 常见词汇替换

| 口语/일상어 | 书面语/문어체                | 含义        |
| ----------- | ---------------------------- | ----------- |

| 좀          | 다소, 약간                   | 稍微 ★      |
| 되게        | 상당히                       | 相当        |
| 빨리        | 신속히, 조속히               | 迅速 ★      |
| 바로        | 즉시, 곧바로                 | 立即 ★      |
| 요즘        | 최근, 근래                   | 最近 ★      |
| 옛날        | 과거, 이전                   | 过去 ★      |
| 나중에      | 추후, 향후                   | 今后 ★      |
| 그때        | 당시                         | 当时 ★      |
| 아마        | 아마도                       | 大概        |
| 왜냐하면    | 그 이유는 ...기 때문이다     | 因为 ★      |
| 그래서      | 따라서, 그러므로             | 因此 ★      |
| 그런데      | 그러나, 하지만               | 但是 ★      |
| 근데        | 그런데 → 그러나              | 但是        |
| 그리고      | 또한, 아울러, 게다가         | 而且 ★      |

| 고치다      | 개선하다, 수정하다           | 改善 ★      |
| 쓸모        | 용도, 가치                   | 用途        |

### B. 缩略语 → 完整形

| 缩略  | 完整形     | 说明                  |
| ----- | ---------- | --------------------- |
| 안 ~  | ~지 않다   | 书面语用 ~지 않다 ★   |
| 못 ~  | ~지 못하다 | 书面语用 ~지 못하다 ★ |
| ~거   | ~것        | 写作中不能缩略 ★      |
| ~건   | ~것은      | 写作中不能缩略        |
| ~걸   | ~것을      | 写作中不能缩略        |
| ~됐다 | ~되었다    | 写作中尽量不缩略      |

## 二、敬语体系 높임말

### A. 主体敬语 (주체 높임)

| 일반형 | 높임형          | 含义          |
| ------ | --------------- | ------------- |
| 먹다   | 드시다/잡수시다 | 吃 ★          |
| 마시다 | 드시다          | 喝 ★          |
| 자다   | 주무시다        | 睡 ★          |
| 죽다   | 돌아가시다      | 去世 ★        |
| 있다   | 계시다          | 在 ★          |
| 없다   | 안 계시다       | 不在          |
| 말하다 | 말씀하시다      | 说 ★          |
| 주다   | 주시다          | 给(他给我)    |
| 묻다   | 여쭙다          | 问(我问他) ★  |
| 보다   | 보시다          | 看            |
| 알다   | 아시다          | 知道          |
| 아프다 | 편찮으시다      | 生病/不舒服 ★ |
| 나이   | 연세            | 年龄 ★        |
| 이름   | 성함            | 名字 ★        |
| 밥     | 진지            | 饭 ★          |
| 말     | 말씀            | 话 ★          |
| 집     | 댁              | 家 ★          |
| 생일   | 생신            | 生日          |

### B. 格式体与非格式体

| 种类            | 句型           | 使用场景                   |
| --------------- | -------------- | -------------------------- |
| 格式体 합쇼체   | -ㅂ니다/습니다 | 最正式：新闻、演讲、面试 ★ |
| 格式体 해라체   | -(으)ㄴ다/는다 | 书面叙述体：论文、报告 ★   |
| 非格式体 해요体 | -아/어요       | 日常礼貌体                 |
| 非格式体 해体   | -아/어         | 朋友之间                   |

> **TOPIK 쓰기 必须用**：해라체 (-다体) 或 합쇼체 (-ㅂ니다体)
> 绝对不能用 해요体！

### C. TOPIK写作文体统一示范

| 错误 (해요体) | 正确 (해라체)                | 正确 (합쇼체)       |
| ------------- | ---------------------------- | ------------------- |
| 중요해요      | 중요하다                     | 중요합니다          |
| 필요해요      | 필요하다                     | 필요합니다          |
| 생각해요      | 생각한다                     | 생각합니다          |
| 증가했어요    | 증가하였다/증가했다          | 증가하였습니다      |
| 줄어들었어요  | 줄어들었다                   | 줄어들었습니다      |
| ~인 것 같아요 | ~인 것으로 보인다            | ~인 것으로 보입니다 |
| ~해야 돼요    | ~해야 한다 / ~할 필요가 있다 | ~해야 합니다        |

## 三、书面语高级句型

### A. 被动/使动表达

| 句型                    | 含义          | 例                             |
| ----------------------- | ------------- | ------------------------------ |
| -아/어지다              | 变得... ★     | 환경이 악화되어지다 → 악화되다 |
| -(으)ㄹ 수 있다/없다    | 能/不能 ★     | 해결할 수 있다                 |
| -게 되다                | 变成... ★     | 참여하게 되다                  |
| -도록 하다              | 使...做 ★     | 학생들이 이해하도록 하다       |
| -(으)ㄹ 것으로 보이다   | 被看作...将 ★ | 증가할 것으로 보인다           |
| -(으)ㄹ 것으로 예상되다 | 被预期将 ★    | 감소할 것으로 예상된다         |
| -(으)ㄹ 것으로 전망되다 | 被展望为将 ★  | 지속될 것으로 전망된다         |
| -는 것으로 나타나다     | 被表明为 ★    | 증가하는 것으로 나타났다       |
| -는 것으로 조사되다     | 被调查为 ★    | 선호하는 것으로 조사되었다     |

### B. 引用/间接表达

| 句型                | 含义          | 例                        |
| ------------------- | ------------- | ------------------------- |
| -다고 하다          | 说... ★       | 중요하다고 하다           |
| -다고 주장하다      | 主张说... ★   | 필요하다고 주장하다       |
| -다고 지적하다      | 指出说... ★   | 문제가 있다고 지적하다    |
| -다는 점에서        | 从...的角度 ★ | 효과적이라는 점에서       |
| -다는 견해가 있다   | 有...的见解 ★ | 불가피하다는 견해가 있다  |
| -(으)ㄹ 필요가 있다 | 有...的必要 ★ | 제도를 개선할 필요가 있다 |
| -(으)ㄹ 수밖에 없다 | 不得不... ★   | 반대할 수밖에 없다        |
| -(으)ㅁ에 따라      | 随着... ★     | 기술이 발전함에 따라      |
| -(으)ㄴ/는 바 있다  | 曾经...       | 언급한 바 있다            |

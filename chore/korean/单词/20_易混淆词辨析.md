# TOPIK II 高级 易混淆词辨析 (~100组)

> 每组列出易混淆的2-3个词，说明区别和正确用法
> ★ = TOPIK常考混淆点

---

## A. 动词混淆

### 1. 넓히다 vs 늘리다 vs 늘다 ★

| 词     | 含义            | 区别                           |
| ------ | --------------- | ------------------------------ |
| 넓히다 | 扩大(面积/范围) | 시야를 넓히다, 영토를 넓히다   |
| 늘리다 | 增加(数量/长度) | 생산량을 늘리다, 수출을 늘리다 |
| 늘다   | (自动)增加      | 인구가 늘다, 실력이 늘다       |

### 2. 줄이다 vs 줄다 vs 낮추다 ★

| 词     | 含义            | 区别                           |
| ------ | --------------- | ------------------------------ |
| 줄이다 | 减少(他动)      | 비용을 줄이다                  |
| 줄다   | 减少(自动)      | 인구가 줄다                    |
| 낮추다 | 降低(程度/水平) | 목소리를 낮추다, 가격을 낮추다 |

### 3. 바꾸다 vs 변하다 vs 변화하다 ★

| 词       | 含义               | 区别                         |
| -------- | ------------------ | ---------------------------- |
| 바꾸다   | 改换(他动，具体)   | 옷을 바꾸다, 이름을 바꾸다   |
| 변하다   | 变(自动，自然变化) | 날씨가 변하다, 마음이 변하다 |
| 변화하다 | 变化(自动，较抽象) | 사회가 변화하다              |

### 4. 맞다 vs 맞추다 ★

| 词     | 含义            | 区别                           |
| ------ | --------------- | ------------------------------ |
| 맞다   | 对(被动/状态)   | 답이 맞다, 비를 맞다           |
| 맞추다 | 使...对上(他动) | 시간을 맞추다, 눈높이를 맞추다 |

### 5. 잡다 vs 잡히다 vs 붙잡다

| 词     | 含义       | 区别          |
| ------ | ---------- | ------------- |
| 잡다   | 抓(主动)   | 기회를 잡다   |
| 잡히다 | 被抓(被动) | 경찰에 잡히다 |
| 붙잡다 | 紧抓不放   | 팔을 붙잡다   |

### 6. 걸리다 vs 걸다 ★

| 词     | 含义                 | 区别                         |
| ------ | -------------------- | ---------------------------- |
| 걸다   | 挂/打(电话)          | 전화를 걸다, 그림을 걸다     |
| 걸리다 | 被挂/花(时间)/得(病) | 시간이 걸리다, 감기에 걸리다 |

### 7. 이루다 vs 이루어지다 ★

| 词         | 含义         | 区别                                 |
| ---------- | ------------ | ------------------------------------ |
| 이루다     | 实现(他动)   | 꿈을 이루다, 목표를 이루다           |
| 이루어지다 | 被实现(自动) | 논의가 이루어지다, 발전이 이루어지다 |

### 8. 받다 vs 받아들이다 ★

| 词         | 含义            | 区别                                 |
| ---------- | --------------- | ------------------------------------ |
| 받다       | 收到(物理/被动) | 편지를 받다, 치료를 받다             |
| 받아들이다 | 接受(心理)      | 의견을 받아들이다, 현실을 받아들이다 |

## B. 形容词混淆

### 9. 다르다 vs 틀리다 ★★

| 词     | 含义           | 区别                     |
| ------ | -------------- | ------------------------ |
| 다르다 | 不同(neutral)  | 의견이 다르다 = 意见不同 |
| 틀리다 | 错误(negative) | 답이 틀리다 = 答案错了   |

> 注意：很多韩国人口语中也混用，但TOPIK要区分

### 10. 쓸데없다 vs 소용없다 vs 필요없다

| 词       | 含义                     | 区别              |
| -------- | ------------------------ | ----------------- |
| 쓸데없다 | 无聊的，没必要的         | 쓸데없는 걱정     |
| 소용없다 | 没有用(已经做了但没效果) | 후회해도 소용없다 |
| 필요없다 | 不需要(本身就不需要)     | 설명이 필요없다   |

### 11. 익숙하다 vs 친숙하다 vs 낯설다 ★

| 词       | 含义           | 区别          |
| -------- | -------------- | ------------- |
| 익숙하다 | 熟悉(通过经验) | 일에 익숙하다 |
| 친숙하다 | 亲切，亲近     | 친숙한 분위기 |
| 낯설다   | 陌生           | 낯선 환경     |

### 12. 어렵다 vs 힘들다 ★

| 词     | 含义               | 区别                     |
| ------ | ------------------ | ------------------------ |
| 어렵다 | 困难(能力上做不到) | 문제가 어렵다            |
| 힘들다 | 辛苦(体力/精力累)  | 일이 힘들다, 살기 힘들다 |

### 13. 부족하다 vs 모자라다

| 词       | 含义             | 区别            |
| -------- | ---------------- | --------------- |
| 부족하다 | 不足(抽象，书面) | 경험이 부족하다 |
| 모자라다 | 不够(具体，口语) | 돈이 모자라다   |

### 14. 정확하다 vs 명확하다 vs 분명하다 ★

| 词       | 含义       | 区别          |
| -------- | ---------- | ------------- |
| 정확하다 | 正确，精确 | 정확한 데이터 |
| 명확하다 | 明确       | 명확한 기준   |
| 분명하다 | 分明，明显 | 분명한 사실   |

### 15. 적절하다 vs 적합하다 ★

| 词       | 含义                 | 区别            |
| -------- | -------------------- | --------------- |
| 적절하다 | 适当(程度、方法合适) | 적절한 조치     |
| 적합하다 | 适合(条件匹配)       | 역할에 적합하다 |

## C. 名词混淆

### 16. 과정 vs 과정 vs 결과 ★

| 词   | 含义 | 区别        |
| ---- | ---- | ----------- |
| 과정 | 过程 | 발전 과정   |
| 결과 | 结果 | 조사 결과   |
| 원인 | 原因 | 문제의 원인 |

### 17. 비율 vs 비중 vs 비례 ★

| 词   | 含义            | 区别               |
| ---- | --------------- | ------------------ |
| 비율 | 比率 (A:B)      | 남녀 비율          |
| 비중 | 比重 (占的份额) | 큰 비중을 차지하다 |
| 비례 | 比例 (正/反比)  | 수입에 비례하다    |

### 18. 대상 vs 대안 vs 대책 ★

| 词   | 含义     | 区别            |
| ---- | -------- | --------------- |
| 대상 | 对象     | 조사 대상       |
| 대안 | 替代方案 | 대안을 제시하다 |
| 대책 | 对策     | 대책을 마련하다 |

### 19. 목적 vs 목표 ★

| 词   | 含义             | 区别        |
| ---- | ---------------- | ----------- |
| 목적 | 目的(为什么做)   | 교육의 목적 |
| 목표 | 目标(要达到的点) | 매출 목표   |

### 20. 의미 vs 의의 ★

| 词   | 含义       | 区别        |
| ---- | ---------- | ----------- |
| 의미 | 意思，含义 | 단어의 의미 |
| 의의 | 意义，价值 | 역사적 의의 |

### 21. 장점 vs 이점 vs 강점

| 词   | 含义         | 区别           |
| ---- | ------------ | -------------- |
| 장점 | 优点(一般性) | 이 방법의 장점 |
| 이점 | 益处(利益性) | 경제적 이점    |
| 강점 | 强项(竞争力) | 기업의 강점    |

## D. 副词混淆

### 22. 아직 vs 여전히 vs 그래도 ★

| 词     | 含义               | 区别                 |
| ------ | ------------------ | -------------------- |
| 아직   | 还(没完成，含期待) | 아직 안 끝났다       |
| 여전히 | 仍然(不变)         | 여전히 변함이 없다   |
| 그래도 | 即便如此           | 그래도 포기하지 않다 |

### 23. 거의 vs 전혀 vs 결코 ★

| 词   | 含义           | 区别            |
| ---- | -------------- | --------------- |
| 거의 | 几乎(接近100%) | 거의 완성되었다 |
| 전혀 | 完全(+否定=0%) | 전혀 모르다     |
| 결코 | 绝不(强调否定) | 결코 쉽지 않다  |

### 24. 오히려 vs 도리어 vs 되레 ★

| 词     | 含义            | 区别                |
| ------ | --------------- | ------------------- |
| 오히려 | 反而(标准书面)  | 오히려 악화되었다 ★ |
| 도리어 | 反而(稍旧/文学) |                     |
| 되레   | 反而(口语)      |                     |

### 25. 단지 vs 단순히 vs 불과 ★

| 词     | 含义           | 区别                  |
| ------ | -------------- | --------------------- |
| 단지   | 只是(限定范围) | 단지 그뿐이다         |
| 단순히 | 单纯地(方式)   | 단순히 무시할 수 없다 |
| 불과   | 仅仅(强调少)   | 불과 10%에 불과하다   |

## E. 语法/助词混淆

### 26. -(으)로 인해 vs -기 때문에 vs -는 바람에 ★

| 语法         | 含义             | 区别                        |
| ------------ | ---------------- | --------------------------- |
| -(으)로 인해 | 由于(书面，原因) | 사고로 인해 지연되다        |
| -기 때문에   | 因为(通用)       | 바쁘기 때문에 못 간다       |
| -는 바람에   | 因为(意外/负面)  | 비가 오는 바람에 취소되었다 |

### 27. -에 대해(서) vs -에 관해(서) ★

| 语法     | 含义              | 区别                    |
| -------- | ----------------- | ----------------------- |
| -에 대해 | 关于(一般)        | 환경에 대해 논의하다    |
| -에 관해 | 关于(更正式/学术) | 이 문제에 관해 연구하다 |

### 28. -도록 vs -(으)려고 ★

| 语法      | 含义           | 区别                         |
| --------- | -------------- | ---------------------------- |
| -도록     | 以便(结果导向) | 학생들이 이해하도록 설명하다 |
| -(으)려고 | 为了(意图)     | 공부하려고 도서관에 가다     |

### 29. -(으)며 vs -고 ★

| 语法    | 含义       | 区别              |
| ------- | ---------- | ----------------- |
| -(으)며 | 并且(书面) | 연구하며 가르치다 |
| -고     | 并且(通用) | 밥을 먹고 가다    |

### 30. -는 데 vs -는데 ★

| 语法   | 含义                     | 区别                      |
| ------ | ------------------------ | ------------------------- |
| -는 데 | ...的地方/方面(依存名词) | 적응하는 데 시간이 걸리다 |
| -는데  | 转折/背景(连接语尾)      | 비가 오는데 우산이 없다   |

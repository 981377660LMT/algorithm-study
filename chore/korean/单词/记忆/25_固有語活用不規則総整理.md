# 固有語活用不規則総整理 — 全7パターン完全攻略

> 固有語の最大の壁 = 不規則活用。漢字語動詞(-하다)には不規則がない
> TOPIK文法問題・쓰기で不規則活用の正確さが点差を作る
> ★ = 高頻度語　⚠️ = 間違えやすい

---

## 一、ㅂ不規則（最重要・最多）

### 規則：パッチムㅂ → 우/오 に変化（母音語尾の前）

```
ㅂ + 아/어 → 워 (ㅂ脱落 + 우 + 어 = 워)
例外：돕다 → 도와 / 곱다 → 고와 (우+아=와)
```

### 不規則活用する語（暗記必須）

| 原形           | -아/어요   | -으면      | -(으)ㄴ  | 日本語                  |
| -------------- | ---------- | ---------- | -------- | ----------------------- |
| 덥다 ★         | 더워요     | 더우면     | 더운     | 暑い                    |

## 二、ㄷ不規則

### 規則：パッチムㄷ → ㄹ に変化（母音語尾の前）

| 原形           | -아/어요 | -으면    | -(으)ㄴ | 日本語 |
| -------------- | -------- | -------- | ------- | ------ |
| 듣다 ★         | 들어요   | 들으면   | 들은    | 聞く   |
| 묻다(聞く) ★⚠️ | 물어요   | 물으면   | 물은    | 尋ねる |
| 걷다 ★         | 걸어요   | 걸으면   | 걸은    | 歩く   |
| 싣다 ★         | 실어요   | 실으면   | 실은    | 載せる |
| 깨닫다 ★       | 깨달아요 | 깨달으면 | 깨달은  | 悟る   |

### 規則活用する語（ㄷだけど規則！）

| 原形             | -아/어요 | 日本語 | 注意       |
| ---------------- | -------- | ------ | ---------- |
| 묻다(埋める) ★⚠️ | 묻어요   | 埋める | ㄷが残る！ |
| 닫다 ★           | 닫아요   | 閉める | ㄷが残る！ |
| 받다 ★           | 받아요   | もらう | ㄷが残る！ |
| 믿다 ★           | 믿어요   | 信じる | ㄷが残る！ |

> **⚠️最大トラップ**: 묻다 が2つ！ 聞く=불규칙(물어요) / 埋める=규칙(묻어요)

## 七、ㅎ不規則

### 規則：語幹末ㅎ → 脱落し、語尾が変化

```
ㅎ + 아/어 → 해/래 (ㅎ脱落、母音融合)
ㅎ + (으)ㄴ → ㄴ (ㅎ脱落)
ㅎ + (으)면 → 면 (ㅎ脱落)
```

## 八、不規則判定フローチャート

```
語幹末のパッチムは何？
│
├─ ㅂ → 돕다/곱다→와, 잡다/좁다等→規則, 他→워
├─ ㄷ → 듣다/걷다/묻다(聞)→ㄹ化, 닫다/받다/믿다→規則
├─ ㅅ → 짓다/잇다/낫다→脱落, 웃다/씻다/벗다→規則
├─ ㄹ → ㄴ/ㅂ/ㅅの前で脱落 (全てのㄹ語幹)
├─ 르 → 르→ㄹㄹ化 (모르다·다르다等), 一部規則(따르다等)
├─ ㅎ → 色彩(-갛/-랗等)と-렇다→融合, 놓다/좋다→規則
└─ 으 → 으脱落 (쓰다·크다·슬프다等)

※ 漢字語(-하다動詞)は全て規則活用！不規則は固有語だけの話！
```

## 九、TOPIK不規則活用 頻出問題パターン

### パターン1: 빈칸に適切な活用形を入れよ

```
[例] 요즘 날씨가 ( ).
① 더워요  ② 더어요  ③ 덥어요  ④ 더보요
正解: ① 더워요 (ㅂ不規則)
```

### パターン2: 正しい文を選べ

```
[例] 다음 중 맞는 것은?
① 이 드라마가 재미있어서 웃었어요 (○ 規則)
② 이 문제를 듣었어요 (✕ → 들었어요 ㄷ不規則)
③ 집을 짓었어요 (✕ → 지었어요 ㅅ不規則)
④ 비가 와서 우산을 씻었어요 (内容おかしいが文法は○)
正解: ①
```

> **最終まとめ**: 不規則活用7種のうち、ㅂ不規則とㄹ脱落が最頻出。この2つだけで不規則問題の70%を占める。まずこの2つを完璧にし、次にㄷ・르・으、最後にㅅ・ㅎの順で攻略すべし。

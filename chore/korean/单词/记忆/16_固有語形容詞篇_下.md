## 二、評価系 — 良い/悪い/正しい/間違っている

### 良い/優れている

| 韓語     | 意味                      | 使い分け                     | ★   |
| -------- | ------------------------- | ---------------------------- | --- |
| 좋다     | 良い/好きだ               | 最汎用                       | ★   |
| 훌륭하다 | 立派だ/素晴らしい         | 人格・作品                   | ★📖 |
| 뛰어나다 | 優れている/抜きん出ている | 能力・才能                   | ★📖 |
| 탁월하다 | 卓越している              | 卓越 → 漢字語                | ★   |
| 멋지다   | 素敵だ/かっこいい         | 멋(趣)+지다                  | ★   |
| 대단하다 | すごい/大したものだ       | 大端→漢字由来                | ★   |
| 기특하다 | けなげだ/感心だ           | 奇特→漢字由来                |     |
| 알차다   | 中身が充実している        | 알(実)+차다(満)=実が詰まった | ★   |

| 韓語     | 意味              | 使い分け                    | ★   |
| -------- | ----------------- | --------------------------- | --- |
| 나쁘다   | 悪い              | 最汎用                      | ★   |
| 못되다   | 性悪だ            | 人の性格                    |     |
| 형편없다 | ひどい/お粗末だ   | 形便=状況+없다→話にならない | ★📖 |

### 正しい/間違っている

| 韓語     | 意味              | 記憶法                       | ★     |
| -------- | ----------------- | ---------------------------- | ----- |
| 옳다     | 正しい            | 옳=オルバルン→올바르다と同族 | ★📖   |
| 올바르다 | 正しい(道徳的に)  | 올(正)+바르다(正しい)        | ★📖📝 |
| 바르다   | まっすぐだ/正しい | -                            | ★     |
| 그르다   | 間違っている      | 옳다の反対                   |       |
| 틀리다   | 間違っている      | 틀(枠)+리→枠からはずれる     | ★     |
| 다르다   | 異なる(≠間違い)   | ⚠ 틀리다≠다르다              | ★     |

> **注意**: 틀리다(間違い) vs 다르다(違い)。日本語の「違う」は両方の意味だが、韓国語は区別する。

## 五、-답다/-스럽다/-롭다 接尾辞系

### -답다 (〜らしい/〜にふさわしい)

| 韓語     | 意味       | 出来元          |
| -------- | ---------- | --------------- |
| 사람답다 | 人間らしい | 사람(人)+답다   |
| 어른답다 | 大人らしい | 어른(大人)+답다 |
| 남자답다 | 男らしい   | 남자(男)+답다   |
| 학생답다 | 学生らしい | 학생(学生)+답다 |

> 日本語の「〜らしい」と完全一致

### -스럽다 (〜っぽい/〜らしい/〜めいた)

| 韓語                      | 意味         | 出来元            |
| ------------------------- | ------------ | ----------------- | --- |
| 사랑스럽다                | 愛らしい     | 사랑(愛)+스럽다   | ★   |
| 자연스럽다                | 自然な       | 자연(自然)+스럽다 | ★   |
| 자랑스럽다                | 誇らしい     | 자랑(自慢)+스럽다 | ★   |
| 걱정스럽다                | 心配な       | 걱정(心配)+스럽다 | ★   |
| 만족스럽다                | 満足な       | 만족(満足)+스럽다 | ★   |
| 실망스럽다                | 失望的な     | 실망(失望)+스럽다 | ★   |
| 의심스럽다                | 疑わしい     | 의심(疑心)+스럽다 | ★   |
| 싱겁다→맛스럽다×→탐스럽다 | 食欲をそそる | 탐(貪)+스럽다     |

### -롭다 (〜らしい/〜に満ちた)

| 韓語     | 意味   | 出来元          |
| -------- | ------ | --------------- | --- |
| 평화롭다 | 平和だ | 평화(平和)+롭다 | ★   |
| 자유롭다 | 自由だ | 자유(自由)+롭다 | ★   |
| 슬기롭다 | 賢い   | 슬기(知恵)+롭다 | ★   |
| 향기롭다 | 芳しい | 향기(香気)+롭다 |
| 해롭다   | 有害だ | 해(害)+롭다     | ★   |
| 이롭다   | 有益だ | 이(利)+롭다     | ★   |
| 외롭다   | 孤独だ | 외(一人)+롭다   | ★   |
| 새롭다   | 新しい | 새(新)+롭다     | ★   |

> **使い分け**: -답다=「本来の姿に合致」、-스럽다=「そういう性質が感じられる」、-롭다=「その状態に満ちている」

## 六、TOPIK읽기 選択肢の固有語形容詞 — 速攻暗記

> 読解の選択肢で漢字語の代わりに出る固有語。知らないと選択肢が読めない。

| 韓語         | TOPIK的意味         | 漢字語パラフレーズ      |
| ------------ | ------------------- | ----------------------- |
| 바람직하다 ★ | 望ましい            | 적절하다/합당하다       |
| 뚜렷하다 ★   | 明瞭だ/明確だ       | 명확하다/분명하다       |
| 두드러지다 ★ | 顕著だ/際立つ       | 현저하다/눈에 띄다      |
| 꾸준하다 ★   | 着実だ/持続的だ     | 지속적이다/한결같다     |
| 한결같다 ★   | 一貫している        | 일관되다/변함없다       |
| 알맞다 ★     | 適切だ              | 적절하다/적합하다       |
| 마땅하다 ★   | 当然だ/しかるべきだ | 당연하다/합당하다       |
| 가파르다 ★   | 急だ(坂/上昇)       | 급격하다                |
| 완만하다 ★   | 緩やかだ            | 완만=漢字語だが固有語的 |
| 드물다 ★     | まれだ/珍しい       | 희귀하다                |
| 잦다 ★       | 頻繁だ              | 빈번하다                |
| 고르다 ★     | 均一だ/揃っている   | 균일하다/일정하다       |
| 고루         | 均等に(副詞)        | 균등하게                |
| 미미하다 ★   | 微々たる            | 미세하다/작다           |

# 固有語副詞篇 — 漢字語では表現できないニュアンスの宝庫

> 固有語副詞は**感覚的・直感的**なものが多く、日本語の副詞と1:1対応しにくい
> 文脈でニュアンスを掴むしかないものを体系的に分類
> ★ = TOPIK必須 / 📖 = 읽기頻出 / 📝 = 쓰기頻出

---

## 一、程度副詞 — 「どのくらい」

### 強調 (非常に/とても)

| 韓語   | 意味          | ニュアンス               | 日本語           | ★   |
| ------ | ------------- | ------------------------ | ---------------- | --- |
| 매우   | 非常に        | 書面的・客観的           | 非常に           | ★📝 |
| 아주   | とても        | 口語的・完全に           | とても           | ★   |
| 너무   | あまりにも    | 過度(口語では「とても」) | あまりにも       | ★   |
| 무척   | とても/非常に | やや感情的               | ひどく           | ★   |
| 몹시   | ひどく        | 不快なことに             | ひどく/甚だ      | ★   |
| 되게   | すごく        | 口語限定                 | めっちゃ         |     |
| 엄청   | ものすごく    | 口語限定・驚き           | 半端なく         |     |
| 대단히 | 大変に        | やや書面的               | 大変に           | ★   |
| 상당히 | 相当に        | 客観的・数値的           | 相当             | ★📝 |
| 꽤     | かなり        | 予想以上                 | かなり           | ★   |
| 제법   | なかなか      | 予想を超えて良い         | なかなか         | ★   |
| 자못   | かなり/相当   | 書面的・古風             | なかなかどうして | 📖  |
| 사뭇   | まるで/かなり | 書面的                   | すっかり         | 📖  |

> **TOPIK쓰기での使い分け**: 매우/상당히(書面OK) > 대단히(やや堅い) > 아주(ギリOK) > 너무/되게/엄청(口語→NG)

## 七、擬声擬態 副詞 — 固有語の真骨頂

> 05\_拟声拟态词完整 を参照。ここでは副詞として特によく使うものだけ抽出。

### 動作擬態

| 韓語     | 意味                        | 母音のイメージ    |
| -------- | --------------------------- | ----------------- |
| 살살     | そっと/やさしく             | ㅏ=明るい         |
| 슬슬     | そろそろ/じわじわ           | ㅡ=控えめ         |
| 팔팔     | ぐらぐら(沸騰)/元気         | ㅏ=明るい・激しい |
| 펄펄     | ぼうぼう(大きな火)/ガンガン | ㅓ=大きく暗い     |
| 반짝반짝 | きらきら                    | ㅏ=明るい光       |
| 번쩍번쩍 | ぴかぴか(大きな光)          | ㅓ=大きい光       |
| 바들바들 | ぶるぶる(小さく)            | ㅏ=小さい         |
| 부들부들 | ブルブル(大きく)            | ㅜ=大きい         |
| 깡충깡충 | ぴょんぴょん                | 小さい動物        |
| 껑충껑충 | ドスンドスン                | 大きい動物        |

> **母音交替法則**(陽母音 ㅏ/ㅗ = 小さい・明るい ↔ 陰母音 ㅓ/ㅜ = 大きい・暗い)は05篇を参照

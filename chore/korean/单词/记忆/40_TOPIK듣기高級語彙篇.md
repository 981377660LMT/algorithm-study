## 七、듣기 選択肢の言い換えパターン

> 正答は本文と同じ内容を「言い換え」で出す

| 本文表現              | 選択肢での言い換え               |
| --------------------- | -------------------------------- |
| 점점 늘어나고 있다    | 증가하는 추세이다 ★★             |
| 없어서는 안 된다      | 필수적이다/필요불가결하다 ★★     |
| 좋다고만은 할 수 없다 | 부정적 측면이 있다 ★★            |
| ~해야 한다            | ~할 필요가 있다 ★★               |
| 큰 도움이 된다        | 효과적이다/유용하다 ★★           |
| ~와/과 다르다         | 차이가 있다/구별된다 ★           |
| 잘 못한다             | 미숙하다/부족하다 ★              |
| 바꿔야 한다           | 개선이 필요하다/전환해야 한다 ★★ |
| 자주 일어난다         | 빈번하다/흔하다 ★                |
| 꼭 ~는 것은 아니다    | 반드시 ~(이)라고 볼 수 없다 ★★   |

> **듣기の鉄則**: 選択肢を先に読み、本文との「言い換え」を見抜く。一字一句同じ選択肢は罠(오답)の可能性大。

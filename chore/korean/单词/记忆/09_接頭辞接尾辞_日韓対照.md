# 接頭辞・接尾辞 完整版 — 日韓漢字音対照

> 接辞を制する者は未知語を制す
> 日本語の接辞と1:1で対応 → 一つ覚えれば派生語が芋づる式に理解できる
> ★ = 高頻度/高収益

| 접두사 | 意味          | 例                                         | 記憶法                              |
| ------ | ------------- | ------------------------------------------ | ----------------------------------- |

### C. 動詞/形容詞化 — 活用接尾辞

| 접미사  | 意味      | 韓語例                             | 日本語説明                         | ★   |
| ------- | --------- | ---------------------------------- | ---------------------------------- | --- |
| -하다   | ~する     | 공부하다, 연구하다, 발전하다       | 漢字語+하다=「~する」              | ★   |
| -되다   | ~される   | 해결되다, 발견되다, 개선되다       | 漢字語+되다=「~される」(受身)      | ★   |
| -시키다 | ~させる   | 발전시키다, 변화시키다, 감소시키다 | 漢字語+시키다=「~させる」(使役)    | ★   |
| -답다   | ~らしい   | 인간답다, 어른답다, 학생답다       | ~답다=「~にふさわしい」            |     |
| -스럽다 | ~のようだ | 자연스럽다, 사랑스럽다, 자랑스럽다 | ~な感じがする(ポジティブ傾向)      | ★   |
| -롭다   | ~に満ちた | 자유롭다, 평화롭다, 해롭다, 이롭다 | 自由な, 平和な, 有害な, 有益な     | ★   |
| -거리다 | 繰り返し~ | 두근거리다, 투덜거리다, 중얼거리다 | ドキドキ, ブツブツ, ブツブツ(擬態) |     |

> **-하다 / -되다 / -시키다 三位一体**:
>
> - 발전**하다** = 発展**する** (能動)
> - 발전**되다** = 発展**される** (受動)
> - 발전**시키다** = 発展**させる** (使役)
>   日本語の「する/される/させる」と完全パラレル。

## 三、接辞パワー — 未知語推測練習

> 接辞力で未知の韓語を瞬時に解読するトレーニング

| 韓語(未知語) | 分解                         | 日本語推測   | 正解        |
| ------------ | ---------------------------- | ------------ | ----------- |
| 비효율적     | 비(非)+효율(効率)+적(的)     | 非効率的     | ✅ 非効率的 |
| 재검토       | 재(再)+검토(検討)            | 再検討       | ✅ 再検討   |
| 탈산업화     | 탈(脱)+산업(産業)+화(化)     | 脱産業化     | ✅ 脱産業化 |
| 무분별하다   | 무(無)+분별(分別)+하다(する) | 無分別にする | ✅ 無分別だ |
| 불가피하다   | 불(不)+가(可)+피(避)+하다    | 不可避だ     | ✅ 不可避だ |
| 최첨단       | 최(最)+첨단(尖端)            | 最先端       | ✅ 最先端   |
| 자율성       | 자(自)+율(律)+성(性)         | 自律性       | ✅ 自律性   |
| 과보호       | 과(過)+보호(保護)            | 過保護       | ✅ 過保護   |
| 생산력       | 생산(生産)+력(力)            | 生産力       | ✅ 生産力   |
| 상호의존     | 상호(相互)+의존(依存)        | 相互依存     | ✅ 相互依存 |

> **結論**: N1の漢字力があれば、韓語の漢字接辞は日本語とほぼ100%対応する。覚えるべきは固有語接辞(맨/풋/햇/헛/짓/되/군/꾼/쟁이/답다/스럽다/롭다/거리다)のみ。

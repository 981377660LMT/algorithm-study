# 敬語與書面語 完整版 — 日韓対照記憶法

> TOPIK쓰기は해라体(다体)か합쇼体(ㅂ니다体)のみ。해요体は絶対禁止！
> 日本語の「です・ます/である/敬語」体系で整理
> ★ = 쓰기・읽기 必須

---

## 一、口語→書面語 即時変換表

### A. 副詞・程度表現

| 口語 ❌   | 書面語 ✅      | 日本語        | 記憶法                                                  |
| --------- | -------------- | ------------- | ------------------------------------------------------- | --- |
| 아주/정말 | 매우, 극히     | 非常に/極めて | 매우=「毎回うそのように多い」→非常に。극히=極(きわ)めて | ★   |
| 엄청      | 상당히, 대단히 | 相当/大いに   | 상당(相当)히, 대단(大段)히 → 漢字そのまま               | ★   |
| 좀        | 다소, 약간     | 多少/若干     | 다소(多少)=たしょう, 약간(若干)=じゃっかん              | ★   |
| 되게      | 상당히         | 相当に        | 상당(相当)                                              |     |
| 빨리      | 신속히, 조속히 | 迅速に/早速に | 신속(迅速)→じんそく, 조속(早速)→そうそく                | ★   |
| 바로      | 즉시, 곧바로   | 即時に/直ちに | 즉시(即時)→そくじ                                       | ★   |
| 요즘      | 최근, 근래     | 最近/近来     | 최근(最近)→さいきん, 근래(近来)→きんらい                | ★   |
| 옛날      | 과거, 이전     | 過去/以前     | 과거(過去), 이전(以前) → 完全対応                       | ★   |
| 나중에    | 추후, 향후     | 追って/今後   | 추후(追後), 향후(向後)                                  | ★   |
| 그때      | 당시           | 当時          | 당시(当時)→とうじ                                       | ★   |
| 처음      | 최초, 당초     | 最初/当初     | 최초(最初), 당초(当初)                                  | ★   |
| 많이      | 상당수, 다수   | 相当数/多数   | 상당수(相当数), 다수(多数)                              | ★   |

### B. 動詞・表現

| 口語 ❌  | 書面語 ✅          | 日本語              | 記憶法                 |
| -------- | ------------------ | ------------------- | ---------------------- | --- |
| 쓰다     | 사용하다, 활용하다 | 使用する/活用する   | 사용(使用), 활용(活用) | ★   |
| 보다     | 관찰하다, 살펴보다 | 観察する/考察する   | 관찰(観察)             | ★   |
| 주다     | 제공하다, 부여하다 | 提供する/付与する   | 제공(提供), 부여(付与) | ★   |
| 알다     | 인지하다, 파악하다 | 認知する/把握する   | 인지(認知), 파악(把握) | ★   |
| 모르다   | 인식하지 못하다    | 認識できない        | 인식(認識)+하지 못하다 |     |
| 바뀌다   | 변화하다, 전환되다 | 変化する/転換される | 변화(変化), 전환(転換) | ★   |
| 늘다     | 증가하다           | 増加する            | 증가(増加)→ぞうか      | ★   |
| 줄다     | 감소하다           | 減少する            | 감소(減少)→げんしょう  | ★   |
| 없어지다 | 사라지다, 소멸되다 | 消滅する            | 소멸(消滅)→しょうめつ  | ★   |
| 생기다   | 발생하다, 나타나다 | 発生する/現れる     | 발생(発生)→はっせい    | ★   |
| 고치다   | 개선하다, 수정하다 | 改善する/修正する   | 개선(改善), 수정(修正) | ★   |

### C. 接続・論理

| 口語 ❌     | 書面語 ✅                    | 日本語                    |
| ----------- | ---------------------------- | ------------------------- | --- |
| 왜냐하면    | 그 이유는 ...기 때문이다     | その理由は...からだ       | ★   |
| 그래서      | 따라서, 그러므로             | したがって                | ★   |
| 그런데/근데 | 그러나, 하지만               | しかし                    | ★   |
| 그리고      | 또한, 아울러, 게다가         | また/さらに               | ★   |
| ~거 같다    | ~것으로 보인다/판단된다      | ~と見られる/判断される    | ★   |
| ~해야 된다  | ~해야 한다 / ~할 필요가 있다 | ~すべきだ/~する必要がある | ★   |

### D. 名詞

| 口語 ❌ | 書面語 ✅        | 日本語          |
| ------- | ---------------- | --------------- | --- |
| 생각    | 견해, 소견, 의견 | 見解/所見/意見  | ★   |
| 문제    | 사안, 현안       | 事案/懸案       | ★   |
| 끝      | 종료, 마무리     | 終了/締めくくり |     |
| 일      | 업무, 과업       | 業務/課業       |     |
| 쓸모    | 용도, 가치       | 用途/価値       |     |

### E. 縮略語→完全形 (쓰기で縮略禁止)

| 縮略 ❌ | 完全形 ✅  | 説明             |
| ------- | ---------- | ---------------- | --- |
| 안 ~    | ~지 않다   | 書面語は후치否定 | ★   |
| 못 ~    | ~지 못하다 | 書面語は후치否定 | ★   |
| ~거     | ~것        | ㅓ→것            | ★   |
| ~건     | ~것은      |                  |     |
| ~걸     | ~것을      |                  |     |
| ~됐다   | ~되었다    | 縮約しない       |     |

> **日本語対比**: 日本語論文でも「じゃない」→「ではない」、「してる」→「している」のように縮約を避ける。同じ発想。

## 二、敬語体系 → 日本語尊敬語・謙譲語

### A. 主体敬語 (주체 높임) — 尊敬語

| 일반형 | 높임형          | 日本語尊敬語     | 記憶法                           |
| ------ | --------------- | ---------------- | -------------------------------- | --- |
| 먹다   | 드시다/잡수시다 | 召し上がる       | 드시다は「差し上がる」の音に近い | ★   |
| 마시다 | 드시다          | お飲みになる     |                                  | ★   |
| 자다   | 주무시다        | お休みになる     | 주무(主務→お休み)                | ★   |
| 죽다   | 돌아가시다      | お亡くなりになる | 돌아가다(帰られる)→婉曲          | ★   |
| 있다   | 계시다          | いらっしゃる     | 계(界にいらっしゃる)             | ★   |
| 말하다 | 말씀하시다      | おっしゃる       | 말씀(お言葉)                     | ★   |
| 묻다   | 여쭙다          | お伺いする(謙譲) | 여쭙다は自分が尊者に聞く時       | ★   |
| 아프다 | 편찮으시다      | ご不調           | 편찮다=「不便だ」の尊敬          | ★   |

### B. 敬語名詞

| 일반형 | 높임형 | 日本語      | 記憶法              |
| ------ | ------ | ----------- | ------------------- | --- |
| 나이   | 연세   | お年/ご年齢 | 연세(年歳)→ねんさい | ★   |
| 이름   | 성함   | お名前      | 성함(姓銜)          | ★   |
| 밥     | 진지   | お食事      | 진지(珍味→お食事)   | ★   |
| 말     | 말씀   | お言葉      |                     | ★   |
| 집     | 댁     | お宅        | 댁(宅)→たく         | ★   |
| 생일   | 생신   | お誕生日    | 생신(生辰)          |     |

### C. 格式体 vs 非格式体 → 日本語文体対照

| 種類   | 韓語句型       | 日本語文体   | 使用場面                 |
| ------ | -------------- | ------------ | ------------------------ | --- |
| 합쇼체 | -ㅂ니다/습니다 | です・ます体 | ニュース・面接・スピーチ | ★   |
| 해라체 | -(으)ㄴ다/는다 | である体     | 論文・報告書・TOPIK쓰기  | ★   |
| 해요体 | -아/어요       | (丁寧口語)   | 日常会話のみ             |     |
| 해体   | -아/어         | (友達口語)   | 友達間のみ               |     |

> **TOPIK쓰기 鉄則**: 해라体(-다体) か 합쇼体(-ㅂ니다体) のみ使用。해요体は0点リスク！

### D. 文体統一 練習

| ❌ 해요体     | ✅ 해라体         | ✅ 합쇼体           | 日本語(である/です)   |
| ------------- | ----------------- | ------------------- | --------------------- |
| 중요해요      | 중요하다          | 중요합니다          | 重要だ/重要です       |
| 필요해요      | 필요하다          | 필요합니다          | 必要だ/必要です       |
| 생각해요      | 생각한다          | 생각합니다          | 考える/考えます       |
| 증가했어요    | 증가하였다        | 증가하였습니다      | 増加した/増加しました |
| ~인 것 같아요 | ~인 것으로 보인다 | ~인 것으로 보입니다 | ~と見られる           |
| ~해야 돼요    | ~해야 한다        | ~해야 합니다        | ~すべきだ/~すべきです |

## 三、書面語高級句型 → 日本語対照

### A. 被動/使動

| 韓語句型                | 日本語等価            | ★   |
| ----------------------- | --------------------- | --- |
| -아/어지다              | ~くなる/~になる(変化) | ★   |
| -(으)ㄹ 수 있다/없다    | ~できる/できない      | ★   |
| -게 되다                | ~することになる       | ★   |
| -도록 하다              | ~するようにする       | ★   |
| -(으)ㄹ 것으로 보이다   | ~するものと見られる   | ★   |
| -(으)ㄹ 것으로 예상되다 | ~すると予想される     | ★   |
| -(으)ㄹ 것으로 전망되다 | ~すると見込まれる     | ★   |
| -는 것으로 나타나다     | ~することが示された   | ★   |
| -는 것으로 조사되다     | ~することが調査された | ★   |

### B. 引用/間接表現

| 韓語句型            | 日本語等価                  | ★   |
| ------------------- | --------------------------- | --- |
| -다고 하다          | ~と言う/~とされる           | ★   |
| -다고 주장하다      | ~と主張する                 | ★   |
| -다고 지적하다      | ~と指摘する                 | ★   |
| -다는 점에서        | ~という点で                 | ★   |
| -다는 견해가 있다   | ~という見解がある           | ★   |
| -(으)ㄹ 필요가 있다 | ~する必要がある             | ★   |
| -(으)ㄹ 수밖에 없다 | ~せざるを得ない             | ★   |
| -(으)ㅁ에 따라      | ~するに伴い/~につれて       | ★   |
| -(으)ㄴ/는 바 있다  | ~したことがある(フォーマル) |     |

> **核心原理**: 韓語書面被動の「~것으로 보이다/예상되다/전망되다/나타나다/조사되다」→ 日本語の「~と見られる/予想される/見込まれる/示された/調査された」。受身の論述表現は日韓で1:1対応。

## 四、敬語 速記まとめ

### スピーキング用 敬語変換チェック

```
食べる: 먹다 → 드시다 (잡수시다)
飲む:   마시다 → 드시다
寝る:   자다 → 주무시다
亡くなる: 죽다 → 돌아가시다
いる:   있다 → 계시다
言う:   말하다 → 말씀하시다
聞く(謙): 묻다 → 여쭙다
具合悪い: 아프다 → 편찮으시다
年齢:   나이 → 연세
名前:   이름 → 성함
食事:   밥 → 진지
家:     집 → 댁
```

> **記憶のコツ**: 日本語の尊敬語「召し上がる/いらっしゃる/おっしゃる」と同様に、韓語の敬語は「別の語」に変わるパターン。数は限られている(約12組)ので丸暗記が最短。

# 固有語多義語詳解篇 — 一語多義を制する者がTOPIKを制す

> 固有語は一語に5〜10個の意味がある多義語が多い
> 漢字語なら字面で区別できるが、固有語では文脈判断力が必須
> ★ = TOPIK必須義 📖 = 읽기で頻出

---

## 四、TOPIK읽기 多義語出題パターン

### パターン1: 밑줄 친 부분이 의미하는 것은?（下線部の意味は？）

```
[例題] "이 연구는 새로운 길을 열었다"에서 '길'의 의미는?
① 도로 (道路)
② 방법 (方法) ★正解
③ 거리 (距離)
④ 과정 (過程)
```

### パターン2: 比喩的意味を問う

```
[例題] "그 사업에 손을 대지 마세요"에서 '손을 대다'의 의미는?
① 만지다 (触る)
② 관여하다 (関与する) ★正解
③ 도와주다 (手伝う)
④ 시작하다 (始める)
```

### 即効対策: 共通イメージ把握法

```
들다 → 「中に入る」が全ての義を貫くイメージ
지다 → 「下へ」が全ての義を貫くイメージ
짓다 → 「形にする」が全ての義を貫くイメージ
풀다 → 「束縛を解く」が全ての義を貫くイメージ
타다 → 「上に乗る」が全ての義を貫くイメージ
걸다 → 「引っ掛ける」が全ての義を貫くイメージ
세우다 → 「垂直にする」が全ての義を貫くイメージ
내다 → 「中から外へ」が全ての義を貫くイメージ
```

> **多義語攻略の鉄則**: 個別の意味を暗記するのではなく、最も抽象的な「核心イメージ(핵심 이미지)」を一つ掴めば、全ての派生義が自然に理解できる。

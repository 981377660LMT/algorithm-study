# 固有語 vs 漢字語 完全対照篇 — 同じ意味の二重体系

> 韓国語の最大の特徴：同じ概念に「固有語」と「漢字語」の2つの表現がある
> 固有語 = 柔らかい/口語的/感情的
> 漢字語 = 硬い/書面語的/客観的
> TOPIK쓰기では漢字語が求められ、듣기では固有語が飛んでくる

---

## 四、TOPIK쓰기 書き換え練習 — 固有語→漢字語に直す

> 쓰기54番(論述)では、固有語を漢字語に書き換える能力が点数に直結

### 練習 10 題

| #   | 固有語(口語)            | →   | 漢字語(쓰기向け)                          |
| --- | ----------------------- | --- | ----------------------------------------- |
| 1   | 사람들이 점점 늘고 있다 | →   | 인구가 지속적으로 증가하고 있다 ★         |
| 2   | 환경이 나빠지고 있다    | →   | 환경이 악화되고 있다 ★                    |
| 3   | 여러 가지 문제가 생겼다 | →   | 다양한 문제가 발생했다 ★                  |
| 4   | 서로 도와야 한다        | →   | 상호 협력해야 한다 ★                      |
| 5   | 이것을 고치려면         | →   | 이를 개선하기 위해서는 ★                  |
| 6   | 알 수 있듯이            | →   | 파악할 수 있듯이 ★                        |
| 7   | 모든 사람이 함께        | →   | 전체 구성원이 공동으로 ★                  |
| 8   | 좋은 점과 나쁜 점       | →   | 장점과 단점 / 긍정적 측면과 부정적 측면 ★ |
| 9   | 점점 더 커지고 있다     | →   | 점차 확대되고 있다 ★                      |
| 10  | 쉽게 이루어지지 않는다  | →   | 용이하게 달성되지 않는다 ★                |

### 쓰기公式変換パターン

```
固有語パターン        →    漢字語パターン
---                       ---
-가 늘다              →    -가 증가하다
-가 줄다              →    -가 감소하다
-가 커지다            →    -가 확대되다
-가 어져지다          →    -가 악화되다
-가 나아지다          →    -가 개선되다
많은 사람들           →    다수의 사람들 / 대부분의 국민
요즘/요새             →    최근/현재
옛날                  →    과거/전통적으로
잘/못                 →    효과적으로/적절히 / 부적절하게
꼭                   →    반드시/필수적으로
함께                  →    공동으로/협력하여
서로                  →    상호
여러 가지             →    다양한
점점                  →    점차/점진적으로
그래서                →    따라서/그러므로
그런데                →    그러나/반면에
-고 싶다              →    -고자 하다 / -기를 바라다
-해야 한다            →    -할 필요가 있다 / -해야 할 것이다
-하면 좋겠다          →    -하는 것이 바람직하다
```

## 五、ニュアンス差 — 固有語と漢字語では「温度」が違う

> **核心**: 固有語は**感情・主観的**表現に強く、漢字語は**論理・客観的**表現に強い。TOPIK듣기では固有語、쓰기では漢字語が求められる。읽기では両方出るので、「同義の対(ペア)」として覚えることが最も効率的。

# 固有語コロケーション篇 — 「名詞+動詞」決まった組合せ

> 固有語最大の壁: 単語を知っていても**搭配**を知らないと使えない
> 日本語の「傘をさす」「首を振る」のように韓国語にも固定搭配がある
> ★ = TOPIK必須

## 五、固有語コロケーション vs 漢字語コロケーション

> 同じ意味でも固有語搭配のほうが生き生きした表現になる

| 固有語表現 | 漢字語表現     | 意味       | ニュアンス差                |
| ---------- | -------------- | ---------- | --------------------------- |
| 힘쓰다     | 노력하다(努力) | 努力する   | 固有語=体感的/漢字語=抽象的 |
| 알아내다   | 발견하다(発見) | 見つけ出す | 固有語=プロセス/漢字語=結果 |
| 뒤처지다   | 낙후하다(落後) | 遅れをとる | 固有語=動的/漢字語=状態的   |
| 앞서다     | 선행하다(先行) | 先行する   | 固有語=能動的/漢字語=形式的 |
| 마련하다   | 준비하다(準備) | 準備する   | 固有語=政策文/漢字語=日常   |

> **TOPIK쓰기上級tip**: 固有語コロケーションを漢字語の間に挟むと、文章に**立体感**が出て高得点。

# 文法総整理篇 — TOPIK6級 必須文法50+間違えやすい文法

> 읽기・듣기の選択肢に直結する文法を厳選
> 쓰기で使うと高得点の文法も収録
> ★★ = 6級必出 ★ = 高得点狙い

## 一、接続語尾(연결어미) — 因果・理由

| 文法                 | 意味               | レベル | 例文                    | 注意点             |
| -------------------- | ------------------ | ------ | ----------------------- | ------------------ |

## 六、冠形詞形語尾(관형사형) — 읽기の核心

| 文法               | 時制         | 例                        |
| ------------------ | ------------ | ------------------------- |
| -(으)ㄴ + 名詞 ★★  | 過去/完了    | 읽**은** 책               |
| -는 + 名詞 ★★      | 現在/進行    | 읽**는** 책               |
| -(으)ㄹ + 名詞 ★★  | 未来/推量    | 읽**을** 책               |
| -던 + 名詞 ★★      | 回想(未完了) | 읽**던** 책(読んでいた本) |
| -았/었던 + 名詞 ★★ | 回想(完了)   | 읽**었던** 책(読んだ本)   |

**-던 vs -았/었던 の違い(6級の罠):**

- 가던 길 = 行っていた(途中の)道
- 갔던 길 = (以前)行った(ことのある)道

## 七、間違えやすい文法ペア

### 7-1. -(으)로서 vs -(으)로써

| 文法         | 意味                   | 例                        |
| ------------ | ---------------------- | ------------------------- |
| -(으)로서 ★★ | 〜として(資格・身分)   | 학생**으로서** 해야 할 일 |
| -(으)로써 ★★ | 〜をもって(手段・道具) | 대화**로써** 해결하다     |

### 7-2. -ㄴ/는 반면(에) vs -ㄴ/는 대신(에)

| 文法        | 意味                 | 例                        |
| ----------- | -------------------- | ------------------------- |
| 반면(에) ★★ | 〜な一方で(対比)     | 편리한 **반면** 위험하다  |
| 대신(에) ★★ | 〜する代わりに(代替) | 돈을 받는 **대신** 일한다 |

### 7-3. -아/어 놓다 vs -아/어 두다

| 文法           | 意味             | ニュアンス       |
| -------------- | ---------------- | ---------------- |
| -아/어 놓다 ★★ | 〜しておく(放置) | 結果状態の維持   |
| -아/어 두다 ★★ | 〜しておく(準備) | 将来のための準備 |

### 7-4. -도록 vs -게

| 文法     | 意味                    | 例                       |
| -------- | ----------------------- | ------------------------ |
| -도록 ★★ | 〜するように(目的/程度) | 잊지 않**도록** 메모하다 |
| -게 ★★   | 〜く/〜するように(結果) | 보기 좋**게** 정리하다   |

### 7-5. -(으)ㄴ 채(로) vs -은/는 그대로

| 文法              | 意味       | 例                          |
| ----------------- | ---------- | --------------------------- |
| -(으)ㄴ 채(로) ★★ | 〜したまま | 신발을 신**은 채** 들어왔다 |

## 八、쓰기54番 高得点文法パターン

### 序論で使う

| パターン                              | 例                              |
| ------------------------------------- | ------------------------------- |
| ~(이)라는 점에서                      | 환경 문제라는 점에서            |
| ~에 대해 ~(으)ㄴ/는지 살펴보고자 한다 | 원인이 무엇인지 살펴보고자 한다 |

### 本論で使う

| パターン                     | 例                          |
| ---------------------------- | --------------------------- |
| ~(으)ㄹ 뿐만 아니라 ~도      | 개인뿐만 아니라 사회도      |
| ~음에도 불구하고             | 노력함에도 불구하고         |
| ~(으)ㄴ/는 반면(에)          | 장점이 있는 반면            |
| ~(이)기 때문이다 (이유 제시) | 변화가 필요하기 때문이다    |
| ~에서 비롯된 것이다          | 인식 부족에서 비롯된 것이다 |
| ~(으)ㄹ 수밖에 없다          | 영향을 미칠 수밖에 없다     |

### 結論で使う

| パターン                             | 例                                     |
| ------------------------------------ | -------------------------------------- |
| ~할 때 비로소 ~(으)ㄹ 수 있을 것이다 | 노력할 때 비로소 해결될 수 있을 것이다 |
| ~하는 것이 바람직하다                | 제도를 마련하는 것이 바람직하다        |
| ~(으)ㄹ 것으로 기대된다              | 효과를 거둘 것으로 기대된다            |

---

> **6級合格の文法戦略**:
>
> 1. 읽기: 관형사형(-ㄴ/-는/-ㄹ/-던)を瞬時に判別できること
> 2. 듣기: 格式体(-ㅂ니다/습니다)の口語縮約に慣れること
> 3. 쓰기: -(으)므로, -음에도 불구하고, -(으)ㄹ 뿐만 아니라 を必ず使うこと

## 一、接续词/连接副词 → 日本語接続詞

### A. 順接/因果 — したがって系

| 韓語        | 日本語等価      | 記憶ポイント                         | ★   |
| ----------- | --------------- | ------------------------------------ | --- |
| 따라서      | したがって      | 따르다(従う)→したがって              | ★   |
| 그러므로    | ゆえに/それゆえ | 그러하므로(そうであるから)           | ★   |
| 그래서      | だから          | 口語的。쓰기では따라서に置換         |     |
| 그 결과     | その結果        | 結(결)果(과)=けっか                  | ★   |
| 이에 따라   | これに伴い      | 이(此)+에+따라(従い)                 | ★   |
| 이로 인해   | これにより      | 이(此)+로+인(因)+해 → 原因(げんいん) | ★   |
| 그런 까닭에 | そのわけで      | 까닭=わけ/理由                       |     |
| 결국        | 結局            | けっきょく→결국 音対応               | ★   |
| 결과적으로  | 結果的に        | けっかてきに→결과적으로              | ★   |

### B. 転折/対比 — しかし系

| 韓語      | 日本語等価     | 記憶ポイント                 | ★   |
| --------- | -------------- | ---------------------------- | --- |
| 그러나    | しかし         | 最もフォーマル。쓰기で最頻出 | ★   |
| 하지만    | しかし/だが    | 그러나よりやや口語的         | ★   |
| 그렇지만  | そうだけれども | 그렇다+지만                  |     |
| 그런데    | ところが       | 口語的。쓰기では그러나に置換 |     |
| 반면(에)  | 反面/一方で    | はんめん→반면 完全対応       | ★   |
| 한편      | 一方(いっぽう) | 한(一)+편(方)                | ★   |
| 이에 반해 | これに反して   | 반(反)+해=はんして           | ★   |
| 그에 비해 | それに比べて   | 비(比)+해=ひして             | ★   |
| 오히려    | むしろ         | 固有語。音で覚える           | ★   |

### C. 並列/遞進 — さらに系

| 韓語        | 日本語等価       | 記憶ポイント                      | ★   |
| ----------- | ---------------- | --------------------------------- | --- |
| 또한        | また/さらに      | 또(また)+한(一つ)                 | ★   |
| 게다가      | そのうえ/しかも  | 固有語。거기에다가の縮約          | ★   |
| 더불어      | あわせて/ともに  | 더불다(加わる)の活用              | ★   |
| 아울러      | あわせて         | 더불어より格式高い                | ★   |
| 나아가      | さらに進んで     | 나아가다(進む)→ひいては           | ★   |
| 뿐만 아니라 | のみならず       | ~뿐(だけ)+만(だけ)+아니라(でなく) | ★   |
| 이 밖에도   | このほかにも     | 이(此)+밖(外)+에도                | ★   |
| 더 나아가   | さらに一歩進んで | 더(もっと)+나아가                 | ★   |
| 마찬가지로  | 同様に           | 마찬가지(同じこと)+로             | ★   |

### D. 例示/補充 — たとえば系

| 韓語          | 日本語等価    | 記憶ポイント                  | ★   |
| ------------- | ------------- | ----------------------------- | --- |
| 예를 들어     | 例を挙げると  | 예(例)+를+들어(挙げて)        | ★   |
| 예컨대        | たとえば      | 예(例)+컨대=言うなれば        | ★   |
| 이를테면      | いわば        | 이를(これを)+테면(言えば)     |     |
| 가령          | 仮に/たとえば | 仮令(かりに)→가령             |     |
| 특히          | とくに        | 特(とく)→특(특) 音対応        | ★   |
| 구체적으로    | 具体的に      | ぐたいてきに→구체적으로       | ★   |
| 즉            | すなわち      | 即(そく)→즉                   | ★   |
| 다시 말해(서) | 言い換えれば  | 다시(再び)+말해(言えば)       | ★   |
| 바꿔 말하면   | 言い換えると  | 바꾸다(換える)+말하면(言えば) |     |

### E. 総結/帰納 — まとめると系

| 韓語                      | 日本語等価             | 記憶ポイント                             | ★   |
| ------------------------- | ---------------------- | ---------------------------------------- | --- |
| 결론적으로                | 結論的に               | けつろんてきに→결론적으로                | ★   |
| 종합하면                  | 総合すると             | そうごうすると→종합하면                  | ★   |
| 요컨대                    | 要するに               | 要(よう)→요 + 건대(言えば)               | ★   |
| 요약하면                  | 要約すると             | ようやくすると→요약하면                  |     |
| 정리하면                  | 整理すると             | せいりすると→정리하면                    |     |
| 이상에서 살펴본 바와 같이 | 以上で見てきたように   | 以上(이상)+살펴보다(見る)+바(こと)       | ★   |
| 지금까지 살펴본 것처럼    | これまで見てきたように | 지금까지(今まで)+것(こと)+처럼(のように) |     |

## 二、쓰기 句型 → 日本語論述文パターン

### A. 序論 (서론) — 導入パターン

| 韓語句型                                      | 日本語等価パターン                         | ★   |
| --------------------------------------------- | ------------------------------------------ | --- |
| 최근 ...이/가 사회적 관심을 받고 있다         | 近年...が社会的関心を集めている            | ★   |
| ...에 대한 논의가 활발하게 이루어지고 있다    | ...をめぐる議論が活発に行われている        | ★   |
| ...은/는 현대 사회의 중요한 과제이다          | ...は現代社会の重要な課題である            | ★   |
| ...이/가 큰 화제가 되고 있다                  | ...が大きな話題となっている                |     |
| ...에 대해 다양한 의견이 제시되고 있다        | ...について多様な意見が出されている        | ★   |
| 이 글에서는 ...에 대해 살펴보고자 한다        | 本稿では...について考察したい              | ★   |
| ...에 대해 긍정적/부정적 시각이 공존하고 있다 | ...について肯定的/否定的見方が共存している | ★   |

> **記憶法**: 서론の骨格は「最近+テーマ+社会的関注/議論が活発」→ 일본어の小論文の序論と完全に同じ構造

### B. 観点表明 — 主張パターン

| 韓語句型                | 日本語等価            | ★   |
| ----------------------- | --------------------- | --- |
| ...다고 생각한다 / 본다 | ...と考える/見る      | ★   |
| ...다고 할 수 있다      | ...と言える           | ★   |
| ...다고 여겨진다        | ...と思われる         |     |
| ...다고 판단된다        | ...と判断される       | ★   |
| ...이/가 바람직하다     | ...が望ましい         | ★   |
| ...이/가 필요하다       | ...が必要だ           | ★   |
| ...이/가 불가피하다     | ...が不可避だ         | ★   |
| 필자는 ...다는 입장이다 | 筆者は...という立場だ |     |

> **핵심**: 바람직하다(望ましい) と 불가피하다(不可避だ) はTOPIK結論段の最頻出表現

### C. 論証 — 根拠提示パターン

| 韓語句型                          | 日本語等価                        | ★   |
| --------------------------------- | --------------------------------- | --- |
| 우선/먼저                         | まず/はじめに                     | ★   |
| 그 이유는 ...기 때문이다          | その理由は...からだ               | ★   |
| ...(으)ㄹ 뿐만 아니라 ...도       | ...のみならず...も                | ★   |
| ...는 점에서                      | ...という点で                     | ★   |
| 이는 ...다는 것을 의미한다        | これは...ということを意味する     | ★   |
| 이를 통해 ...다는 것을 알 수 있다 | これを通じて...ということがわかる | ★   |
| ...다는 조사/연구 결과가 있다     | ...という調査/研究結果がある      | ★   |
| 실제로 ...한 사례가 있다          | 実際に...した事例がある           | ★   |

### D. 反駁/譲歩 — もちろん...しかし

| 韓語句型                                 | 日本語等価                              | ★   |
| ---------------------------------------- | --------------------------------------- | --- |
| 물론 ...다는 반론도 있다                 | もちろん...という反論もある             | ★   |
| 일부에서는 ...다고 주장하기도 한다       | 一部では...と主張することもある         | ★   |
| 이러한 주장은 ...다는 점에서 한계가 있다 | このような主張は...という点で限界がある | ★   |
| ...다고 볼 수도 있지만                   | ...と見ることもできるが                 | ★   |
| 확실히 ...한 측면이 있다. 그러나...      | 確かに...という側面がある。しかし...    | ★   |
| 비록 ...다 하더라도                      | たとえ...だとしても                     | ★   |
| 설령 ...다 할지라도                      | 仮に...だとしても                       |     |

> **黄金パターン**: 물론 A. 그러나 B = もちろんA。しかしB → TOPIK54題の譲歩段で毎回使える

### E. 結論 (결론) — まとめパターン

| 韓語句型                                 | 日本語等価                                  | ★   |
| ---------------------------------------- | ------------------------------------------- | --- |
| 결론적으로 ...는 것이 바람직하다         | 結論として...することが望ましい             | ★   |
| 이상의 내용을 종합하면                   | 以上の内容を総合すると                      | ★   |
| 앞으로 ...기 위해서는 ~이/가 필요하다    | 今後...するためには~が必要だ                | ★   |
| (개인/사회/정부)의 노력이 요구된다       | (個人/社会/政府)の努力が求められる          | ★   |
| ...기 위한 제도적 장치가 마련되어야 한다 | ...するための制度的仕組みが整備されるべきだ | ★   |
| 이를 위해 다각적인 노력이 필요할 것이다  | そのため多角的な努力が必要であろう          | ★   |

## 六、쓰기 模板速記

### 54題 議論文テンプレート

```
[서론]
최근 A에 대한 관심이 높아지고 있다.
A에 대해 다양한 의견이 있는데, 이에 대해 살펴보고자 한다.

[본론1]
먼저, B라는 점에서 A는 중요하다고 할 수 있다.
그 이유는 C이기 때문이다. 실제로 D한 사례가 있다.

[본론2 - 반론 양보]
물론 E라는 반론도 있다. 확실히 F한 측면이 있다.
그러나 이러한 주장은 G라는 점에서 한계가 있다.

[결론]
결론적으로 A는 바람직하다고 할 수 있다.
앞으로 H를 위해 개인과 사회의 노력이 필요할 것이다.
```

> **日本語等価テンプレ**: 近年Aへの関心→多様な意見→まずB→反論もちろんE→しかし限界→結論Aは望ましい→今後の努力

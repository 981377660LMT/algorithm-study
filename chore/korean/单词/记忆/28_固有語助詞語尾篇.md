# 固有語助詞・語尾篇 — 固有語を支える「接着剤」

> 韓国語の助詞と語尾は100%固有語。漢字語は一つもない
> TOPIK文法問題の80%がここから出る
> ★ = TOPIK必須 📖 = 읽기 ✍️ = 쓰기

---

## 一、格助詞 — 日本語助詞との対照

| 機能       | パッチムあり | パッチムなし | 日本語 |
| ---------- | ------------ | ------------ | ------ |
| 主語       | -이 ★        | -가 ★        | が     |
| 主題       | -은 ★        | -는 ★        | は     |
| 目的       | -을 ★        | -를 ★        | を     |
| 場所(静)   | -에 ★        | -에 ★        | に/で  |
| 場所(動)   | -에서 ★      | -에서 ★      | で     |
| 方向       | -으로 ★      | -로 ★        | へ/で  |
| と(並列)   | -과          | -와          | と     |
| と(口語)   | -이랑 💬     | -랑 💬       | と     |
| の         | -의 ★        | -의 ★        | の     |
| に(人)     | -에게 ★      | -에게 ★      | に     |
| に(口語)   | -한테 💬★    | -한테 💬★    | に     |
| から(人)   | -에게서 ★    | -에게서 ★    | から   |
| から(口語) | -한테서 💬   | -한테서 💬   | から   |
| まで       | -까지 ★      | -까지 ★      | まで   |
| から(起点) | -부터 ★      | -부터 ★      | から   |

### 日本語との決定的な違い

```
1. 「は」と「が」の使い分けは日本語と95%同じ
2. 「-에」は場所・時間兼用（日本語の「に」に近い）
3. 「-에서」は動作の場所（日本語の「で」に近い）
4. 「-로/으로」は方向+手段+材料（日本語の「へ」+「で」）
5. 「-의」は口語ではほぼ省略される → 일본의 문화 → 일본 문화
```

## 二、補助助詞 — ニュアンスを変える武器

| 助詞        | 意味              | 例                | 日本語             |
| ----------- | ----------------- | ----------------- | ------------------ |
| -도 ★       | 〜も              | 나도 가고 싶다    | 私も行きたい       |
| -만 ★       | 〜だけ            | 나만 몰랐다       | 私だけ知らなかった |
| -밖에 ★     | 〜しか(+否定)     | 하나밖에 없다     | 一つしかない       |
| -마다 ★     | 〜ごとに          | 날마다            | 毎日               |
| -마저 ★📖   | 〜さえ/〜までも   | 희망마저 사라졌다 | 希望さえ消えた     |
| -조차 ★📖   | 〜さえ(否定的)    | 이름조차 모른다   | 名前さえ知らない   |
| -까지 ★     | 〜まで/〜さえ     | 너까지 그래?      | 君までそう？       |
| -나/이나 ★  | 〜でも/〜も(驚き) | 커피나 한잔       | コーヒーでも一杯   |
| -(이)라도 ★ | 〜でも(妥協)      | 물이라도 주세요   | 水でもください     |
| -부터 ★     | 〜から(まず)      | 먼저 이것부터     | まずこれから       |
| -치고 📖    | 〜にしては        | 한국 사람치고     | 韓国人にしては     |
| -대로 ★✍️   | 〜通りに          | 생각대로          | 思い通りに         |
| -만큼 ★✍️📖 | 〜ほど/〜くらい   | 노력한 만큼       | 努力しただけ       |
| -커녕 ★📖   | 〜どころか        | 감사커녕 불만이다 | 感謝どころか不満だ |

### TOPIK頻出: 마저 vs 조차 vs 까지

```
마저: 最後の一つまでもなくなる(枯渇感)
  → 마지막 희망마저 사라졌다 (最後の希望さえ消えた)

조차: 一番簡単なはずのものすらダメ(失望感)
  → 이름조차 기억 못한다 (名前さえ覚えられない)

까지: 予想外のものまで(驚き/追加)
  → 친구까지 나를 의심했다 (友達まで私を疑った)
```

## 三、接続語尾 — 文をつなぐ固有語の力

### 理由・原因

| 語尾           | ニュアンス           | 例                       | 文体        |
| -------------- | -------------------- | ------------------------ | ----------- |
| -아/어서 ★     | 〜なので(自然な因果) | 배가 아파서 못 갔어요    | 口語/書面   |
| -(으)니까 ★    | 〜だから(主張の根拠) | 비가 오니까 우산 가져가  | 口語中心    |
| -기 때문에 ★✍️ | 〜ために             | 시간이 없기 때문에       | 書面語 ✍️   |
| -(으)므로 ★✍️  | 〜であるため         | 위험하므로 주의해야 한다 | 書面語 ✍️📖 |
| -는 바람에 ★   | 〜せいで(予想外)     | 비가 오는 바람에 못 갔다 | 否定的結果  |
| -는 덕분에 ★   | 〜おかげで           | 도와준 덕분에 성공했다   | 肯定的結果  |
| -는 탓에 ★📖   | 〜せいで             | 날씨 탓에 취소되었다     | 否定強調    |

### 逆接・対比

| 語尾                  | ニュアンス         | 例                             | 文体        |
| --------------------- | ------------------ | ------------------------------ | ----------- |
| -지만 ★               | 〜けど             | 어렵지만 재미있다              | 汎用        |
| -(으)ㄴ데/는데 ★      | 〜のに/〜だけど    | 공부했는데 떨어졌다            | 口語/前置き |
| -아/어도 ★            | 〜ても             | 아무리 먹어도 배가 고프다      | 汎用        |
| -(으)나 ★✍️📖         | 〜であるが         | 노력했으나 실패했다            | 書面語      |
| -ㄴ/는 반면(에) ★✍️📖 | 〜する反面         | 장점이 있는 반면에 단점도 있다 | 書面語      |
| -(으)ㄹ지라도 ★📖     | たとえ〜ても       | 실패할지라도 도전해야 한다     | 書面語      |
| -더라도 ★             | たとえ〜たとしても | 힘들더라도 포기하지 마         | 汎用        |

### 条件・仮定

| 語尾               | ニュアンス         | 例                    | 文体   |
| ------------------ | ------------------ | --------------------- | ------ |
| -(으)면 ★          | 〜れば/たら        | 시간이 있으면 가겠다  | 汎用   |
| -거든 ★            | 〜なら(事実判明時) | 도착하거든 연락해     | 口語   |
| -다면 ★✍️          | もし〜なら         | 가능하다면 참석하겠다 | 書面語 |
| -(으)ㄹ 경우 ★✍️📖 | 〜する場合         | 문제가 생길 경우      | 書面語 |
| -다가는 ★          | 〜し続けたら(警告) | 이러다가는 큰일 난다  | 口語   |

### 目的・意図

| 語尾               | ニュアンス       | 文体      |
| ------------------ | ---------------- | --------- |
| -(으)려고 ★        | 〜しようと       | 口語/書面 |
| -(으)러 ★          | 〜しに(移動目的) | 口語      |
| -기 위해(서) ★✍️📖 | 〜するために     | 書面語    |
| -고자 ★✍️          | 〜しようと(意志) | 書面語    |
| -도록 ★✍️📖        | 〜するように     | 書面語    |

## 四、終結語尾 — 文末の固有語バリエーション

### 丁寧さのスケール

```
(最もフォーマル)
-ㅂ니다/습니다 ★    ニュース・ビジネス・公式
       ↓
-아요/어요 ★        日常丁寧(最も使う)
       ↓
-아/어 ★💬          タメ口
       ↓
-다 ★               日記・独白・書面
(最もカジュアル/中性)
```

### 感嘆・驚き

| 語尾           | ニュアンス        | 例                |
| -------------- | ----------------- | ----------------- |
| -네(요) ★      | 〜ですね(新発見)  | 맛있네요!         |
| -구나/군(요) ★ | 〜なんだ(納得)    | 그렇구나...       |
| -다니! ★       | 〜だなんて!(驚き) | 합격하다니!       |
| -는걸(요)      | 〜のに(意外)      | 생각보다 잘하는걸 |

### 推量・意志

| 語尾                   | ニュアンス            | 例                 |
| ---------------------- | --------------------- | ------------------ |
| -(으)ㄹ 것이다 ★✍️     | 〜するだろう/つもりだ | 증가할 것이다      |
| -겠- ★                 | 〜だろう/〜します     | 알겠습니다         |
| -(으)ㄹ까(요) ★        | 〜しましょうか        | 같이 갈까요?       |
| -(으)ㄹ게(요) ★        | 〜しますね(約束)      | 내가 할게          |
| -(으)ㄹ 텐데 ★         | 〜するはずなのに      | 벌써 도착했을 텐데 |
| -(으)ㄹ 수 있다/없다 ★ | 〜できる/できない     | —                  |

## 五、連体形語尾 — 名詞修飾の固有語システム

| 時制      | 動詞      | 形容詞    | 名詞+이다     |
| --------- | --------- | --------- | ------------- |
| 過去      | -(으)ㄴ ★ | —         | -이었던/-였던 |
| 現在      | -는 ★     | -(으)ㄴ ★ | -인 ★         |
| 未来/推量 | -(으)ㄹ ★ | -(으)ㄹ   | —             |

### 例で理解

```
動詞「먹다(食べる)」:
  過去: 먹은 밥 (食べたご飯)
  現在: 먹는 밥 (食べているご飯)
  未来: 먹을 밥 (食べるご飯)

形容詞「크다(大きい)」:
  現在: 큰 집 (大きい家) ← 過去形と同じ形！
  推量: 클 것 같다
```

> ⚠️ **最大のトラップ**: 動詞の現在連体形は -는、形容詞の現在連体形は -(으)ㄴ。混同すると全て不正解になる

## 六、쓰기54番必須語尾チェックリスト

### 序論

| 表現                    | 意味             | 例                             |
| ----------------------- | ---------------- | ------------------------------ |
| -에 대해 ★✍️            | 〜について       | 환경 문제에 대해 논하고자 한다 |
| -에 관해 ✍️             | 〜に関して       | —                              |
| -(으)ㄹ 필요가 있다 ★✍️ | 〜する必要がある | 대책을 마련할 필요가 있다      |
| -아/어야 한다 ★         | 〜すべきだ       | —                              |
| -(이)라고 할 수 있다 ✍️ | 〜と言える       | —                              |

### 本論

| 表現                      | 意味               |
| ------------------------- | ------------------ |
| -(으)ㄹ 뿐만 아니라 ★✍️📖 | 〜するだけでなく   |
| -(으)ㄹ수록 ★✍️📖         | 〜すればするほど   |
| -기 마련이다 ★✍️📖        | 〜するものだ(当然) |
| -(으)ㄴ/는 셈이다 ★📖     | 〜したも同然だ     |
| -는 데 기여하다 ✍️        | 〜するのに寄与する |
| -(으)ㅁ에 따라 ✍️📖       | 〜するにつれて     |
| -다는 점에서 ★✍️📖        | 〜するという点で   |

### 結論

| 表現                       | 意味                       |
| -------------------------- | -------------------------- |
| -(으)ㄹ 것으로 기대된다 ✍️ | 〜すると期待される         |
| -는 것이 바람직하다 ★✍️    | 〜することが望ましい       |
| -아/어야 할 것이다 ★✍️     | 〜しなければならないだろう |
| -(으)ㄹ 수 있을 것이다 ✍️  | 〜することができるだろう   |

> **結論**: 助詞と語尾は固有語の中でも最も「体系的」な領域。パターンを掴めば暗記ではなく論理で運用できる。

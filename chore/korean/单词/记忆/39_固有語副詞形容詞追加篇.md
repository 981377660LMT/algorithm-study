# 固有語副詞・形容詞追加篇 — TOPIK읽기で意味を取りにくい固有語

> 漢字語は日本語から推測できるが、固有語副詞・形容詞は推測不可
> 覚えていないと「읽기の文章が読めない」直結ジャンル
> 感覚イメージ+例文で定着させる

---

## 一、程度・強調の副詞(40語)

### 1-1. 「とても・非常に」系

| 副詞      | 意味           | ニュアンス             | 例文            |
| --------- | -------------- | ---------------------- | --------------- |
| 매우 ★★   | 非常に         | 書き言葉(最も格式高い) | 매우 중요하다   |
| 아주 ★★   | とても         | 口語・文語両用         | 아주 좋다       |
| 몹시 ★★   | ひどく・激しく | やや否定的             | 몹시 춥다       |
| 무척 ★★   | たいへん       | 口語寄り               | 무척 기쁘다     |
| 굉장히 ★★ | ものすごく     | 感嘆                   | 굉장히 맛있다   |
| 상당히 ★★ | かなり         | 客観的(書き言葉)       | 상당히 높다     |
| 심히 ★    | 甚だ           | 堅い文語               | 심히 우려된다   |
| 지극히 ★  | 極めて         | 最上級                 | 지극히 당연하다 |
| 더없이 ★  | この上なく     | 最高表現               | 더없이 행복하다 |
| 한없이 ★  | 限りなく       | 際限ない               | 한없이 넓다     |

### 1-2. 「少し・わずか」系

| 副詞      | 意味              | 例文               |
| --------- | ----------------- | ------------------ |
| 다소 ★★   | 多少              | 다소 부족하다      |
| 약간 ★★   | やや・若干        | 약간 다르다        |
| 조금 ★★   | 少し              | 조금 기다려 주세요 |
| 살짝 ★    | そっと/ちょっと   | 살짝 미소 짓다     |
| 슬쩍 ★    | こっそり/ちらっと | 슬쩍 보다          |
| 살며시 ★  | そっと(静かに)    | 살며시 문을 열다   |
| 가만히 ★★ | じっと/そっと     | 가만히 생각하다    |
| 겨우 ★★   | かろうじて/やっと | 겨우 도착하다      |
| 간신히 ★★ | 辛うじて          | 간신히 살아남다    |
| 비로소 ★★ | 初めて(ようやく)  | 비로소 깨닫다      |

### 1-3. 「全く・完全に」系

| 副詞        | 意味               | 例文                  |
| ----------- | ------------------ | --------------------- |
| 전혀 ★★     | 全く(否定と共起)   | 전혀 몰랐다           |
| 도무지 ★★   | どうしても(否定)   | 도무지 이해가 안 된다 |
| 도저히 ★★   | とうてい(否定)     | 도저히 할 수 없다     |
| 결코 ★★     | 決して(否定)       | 결코 포기하지 않겠다  |
| 절대(로) ★★ | 絶対に(否定)       | 절대 안 돼            |
| 통 ★        | さっぱり(否定)     | 통 모르겠다           |
| 아예 ★★     | 最初から/いっそ    | 아예 포기하다         |
| 고스란히 ★  | そっくりそのまま   | 고스란히 남아 있다    |
| 온통 ★★     | すっかり/一面      | 온통 하얗다           |
| 오로지 ★★   | ひたすら/ただ~だけ | 오로지 공부만 하다    |

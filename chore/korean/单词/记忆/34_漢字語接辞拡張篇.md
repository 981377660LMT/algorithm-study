# 漢字語接辞拡張篇 — 接頭辞・接尾辞で語彙力を爆発的に増やす

> 漢字語は「接頭辞+語根」「語根+接尾辞」の組み合わせ
> 接辞パターンを覚えれば初見の単語も推測可能
> 日本語漢字の知識を最大限活用
> ★★ = 6級必出 ★ = 高頻度

## 四、動作系接辞パターン

### 4-1. 하다動詞化する漢字語パターン

| パターン             | 例             | 意味           |
| -------------------- | -------------- | -------------- |
| 語根+하다 ★★         | 발전하다(發展) | 発展する       |
| 語根+되다(受身) ★★   | 발전되다       | 発展される     |
| 語根+시키다(使役) ★★ | 발전시키다     | 発展させる     |
| 語根+받다 ★          | 영향받다(影響) | 影響を受ける   |
| 語根+짓다            | —              | (固有語動詞化) |

**하다 → 되다 → 시키다 三位一体で覚える：**

| 하다(能動)             | 되다(受動) | 시키다(使役) |
| ---------------------- | ---------- | ------------ |
| 변화하다(変化する)     | 변화되다   | 변화시키다   |
| 강화하다(強化する)     | 강화되다   | 강화시키다   |
| 개선하다(改善する)     | 개선되다   | 개선시키다   |
| 확대하다(拡大する)     | 확대되다   | 확대시키다   |
| 축소하다(縮小する)     | 축소되다   | 축소시키다   |
| 활성화하다(活性化する) | 활성화되다 | 활성화시키다 |

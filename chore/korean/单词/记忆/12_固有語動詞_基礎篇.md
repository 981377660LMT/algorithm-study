# 固有語動詞 基礎篇 — 語根から芋づる式に覚える

> 漢字速推が効かない固有語動詞を **語根・語源・日本語対応** で攻略
> ★ = TOPIK必須 / 🔑 = 語根キーワード

---

## 一、存在・移動の語根

### 🔑 가다 (行く) → 日: 行く

| 韓語       | 意味         | 語根分解           | 日本語          |
| ---------- | ------------ | ------------------ | --------------- | --- |
| 가다       | 行く         | -                  | 行く            | ★   |
| 나가다     | 出て行く     | 나(外)+가다        | 出ていく        |
| 들어가다   | 入る         | 들(中)+어+가다     | 入っていく      |
| 올라가다   | 上がっていく | 올(上)+라+가다     | 上っていく      |
| 내려가다   | 降りていく   | 내(下)+려+가다     | 下りていく      |
| 돌아가다   | 帰る/回る    | 돌(回)+아+가다     | 回っていく/帰る |
| 지나가다   | 通り過ぎる   | 지나(過)+가다      | 通り過ぎる      |
| 건너가다   | 渡っていく   | 건너(渡)+가다      | 渡る            |
| 찾아가다   | 訪ねて行く   | 찾(探)+아+가다     | 訪ねる          |
| 따라가다   | ついて行く   | 따르(従)+아+가다   | ついていく      |
| 빠져나가다 | 抜け出す     | 빠지(抜)+어+나가다 | 抜け出す        |
| 물러가다   | 退く         | 물러(退)+가다      | 退く            |
| 옮겨가다   | 移っていく   | 옮기(移)+어+가다   | 移る            |

### 🔑 오다 (来る) → 日: 来る

| 韓語              | 意味         | 語根分解           | 日本語       |
| ----------------- | ------------ | ------------------ | ------------ | --- |
| 오다              | 来る         | -                  | 来る         | ★   |
| 나오다            | 出て来る     | 나(外)+오다        | 出てくる     |
| 들어오다          | 入って来る   | 들(中)+어+오다     | 入ってくる   |
| 올라오다          | 上がって来る | 올(上)+라+오다     | 上ってくる   |
| 내려오다          | 降りて来る   | 내(下)+려+오다     | 下りてくる   |
| 돌아오다          | 帰って来る   | 돌(回)+아+오다     | 帰ってくる   |
| 찾아오다          | 訪ねて来る   | 찾(探)+아+오다     | 訪ねてくる   |
| 다가오다          | 近づく       | 다가(近)+오다      | 迫ってくる   | ★   |
| 따라오다          | ついて来る   | 따르(従)+아+오다   | ついてくる   |
| 밀려오다          | 押し寄せる   | 밀리(押)+어+오다   | 押し寄せる   |
| 떠오르다 / 떠오다 | 浮かぶ       | 떠(浮)+오르다/오다 | 浮かんでくる |

> **体系**: 가다 vs 오다 = 日本語の「行く vs 来る」と完全一致。前に方向接頭辞(나/들/올/내/돌)をつけるだけ。

### 🔑 서다 (立つ) → 日: 立つ

| 韓語        | 意味              | 語根分解         | 日本語              |
| ----------- | ----------------- | ---------------- | ------------------- | --- |
| 서다        | 立つ              | -                | 立つ                | ★   |
| 앞서다      | 先立つ/リードする | 앞(前)+서다      | 先頭に立つ          | ★   |
| 나서다      | 名乗り出る        | 나(出)+서다      | 出て立つ→名乗り出る | ★   |
| 멈추어 서다 | 止まって立つ      | 멈추(止)+어+서다 | 立ち止まる          |
| 일어서다    | 立ち上がる        | 일(起)+어+서다   | 起き上がって立つ    |
| 맞서다      | 立ち向かう        | 맞(対)+서다      | 向かい立つ          | ★   |
| 우뚝 서다   | そびえ立つ        | 우뚝(高く)+서다  | 堂々と立つ          |
| 들어서다    | 入る/差し掛かる   | 들(中)+어+서다   | 入って立つ→着手する |

## 二、知覚・認知の語根

### 🔑 보다 (見る) → 日: 見る

| 韓語       | 意味            | 語根分解           | 日本語              |
| ---------- | --------------- | ------------------ | ------------------- | --- |
| 보다       | 見る            | -                  | 見る                | ★   |
| 바라보다   | 見つめる/眺める | 바라(遠く)+보다    | 眺めて見る          | ★   |
| 돌아보다   | 振り返る        | 돌아(回)+보다      | 回って見る→振り返る | ★   |
| 되돌아보다 | 回顧する        | 되(再)+돌아+보다   | 再び振り返る        | ★   |
| 내다보다   | 見通す/予測する | 내(外)+다+보다     | 外を見通す          | ★   |
| 살펴보다   | 調べてみる      | 살피(検)+어+보다   | 詳しく見る          | ★   |
| 알아보다   | 調べる/認識する | 알(知)+아+보다     | 知って見る→調べる   |
| 들여다보다 | 覗く            | 들여(中に)+다+보다 | 中を覗いて見る      |
| 내려다보다 | 見下ろす        | 내려(下)+다+보다   | 下を見る            |
| 올려다보다 | 見上げる        | 올려(上)+다+보다   | 上を見る            |
| 지켜보다   | 見守る          | 지키(守)+어+보다   | 守って見る          |
| 엿보다     | 窺う            | 엿(隙)+보다        | 盗み見る            |

> **体系**: 보다の前に方向語+다 で「〜方向に見る」。日本語と構造完全一致。

### 🔑 지다 (〜になる/負う) → 日: 〜になる

| 韓語            | 意味             | 語根分解     | 日本語      |
| --------------- | ---------------- | ------------ | ----------- | --- |
| 지다            | 負う/負ける/沈む | -            | 負う・沈む  |
| 사라지다        | 消える           | 사라+지다    | 消え去る    | ★   |
| 이루어지다      | 成し遂げられる   | 이루+어+지다 | 成される    | ★   |
| 어우러지다      | 調和する         | 어울+어+지다 | 溶け合う    | ★   |
| 무너지다        | 崩れる           | 무너+지다    | 崩れる      | ★   |
| 쏟아지다        | 降り注ぐ         | 쏟+아+지다   | 注がれる    |
| 빠져들다→빠지다 | 陥る/溺れる      | 빠지+다      | 抜ける→嵌る |

> **構造**: 語幹+아/어+지다 = 自動詞化(〜くなる)。日本語の「〜になる」と同じ。예쁘다→예뻐지다(きれいになる)

## 十、方向接頭辞まとめ — 動詞の万能鍵

| 接頭辞     | 意味   | 日本語 | 組合せ例                           |
| ---------- | ------ | ------ | ---------------------------------- |
| **나**     | 外へ   | 出     | 나가다, 나오다, 나서다, 나타나다   |
| **들**     | 中へ   | 入     | 들어가다, 들어오다, 들어서다       |
| **올**     | 上へ   | 上     | 올라가다, 올라오다, 올리다         |
| **내(려)** | 下へ   | 下     | 내려가다, 내려오다, 내리다         |
| **돌**     | 回転   | 回     | 돌아가다, 돌아오다, 돌아보다       |
| **되**     | 再び   | 再     | 되돌아보다, 되풀이하다, 되찾다     |
| **치**     | 強く上 | 激     | 치솟다, 치켜들다, 치밀다           |
| **건너**   | 越えて | 渡     | 건너가다, 건너오다, 건너뛰다       |
| **빠져**   | 抜けて | 脱     | 빠져나가다, 빠져들다               |
| **밀어**   | 押して | 押     | 밀어내다, 밀어붙이다               |
| **끌어**   | 引いて | 引     | 끌어들이다, 끌어올리다, 끌어당기다 |

> **核心法則**: 固有語動詞の70%は「方向接頭辞 + 基本動詞」の組合せ。10個の接頭辞 × 10個の基本動詞 = 100個の複合動詞が自動生成される。これが固有語の体系的記憶法。

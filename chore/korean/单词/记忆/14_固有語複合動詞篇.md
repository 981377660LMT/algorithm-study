# 固有語複合動詞篇 — 「動詞+動詞」で語彙爆発

> 固有語の核心：基本動詞同士の組合せで無限に増殖する複合動詞
> 12篇・13篇で学んだ基本動詞の「掛け算」
> ★ = TOPIK必須 / 📖 = 読解頻出 / 📝 = 作文頻出

---

## 一、내다 (出す) 系 — 「やり遂げる」ニュアンス

> 내다 = 「外に出す」→ 動作の完遂・達成を表す

| 韓語                | 意味              | 分解          | 日本語                      | ★   |
| ------------------- | ----------------- | ------------- | --------------------------- | --- |
| 만들어 내다         | 作り出す/生み出す | 만들다+내다   | 作り出す                    | ★📖 |
| 찾아내다            | 見つけ出す        | 찾다+내다     | 見つけ出す                  | ★📖 |
| 알아내다            | 突き止める        | 알다+내다     | 知り出す→解明する           | ★📖 |
| 이겨내다            | 打ち勝つ          | 이기다+내다   | 勝ち出す→克服する           | ★📖 |
| 견뎌내다            | 耐え抜く          | 견디다+내다   | 堪え出す→持ちこたえる       | ★📖 |
| 이루어 내다         | 成し遂げる        | 이루다+내다   | 成し出す                    | ★📝 |
| 밀어내다            | 押しのける        | 밀다+내다     | 押し出す                    | ★   |
| 끌어내다            | 引き出す          | 끌다+내다     | 引き出す                    | ★📖 |
| 쏟아내다            | 吐き出す/注ぎ出す | 쏟다+내다     | 注ぎ出す                    |
| 뽑아내다            | 引き抜く/選出する | 뽑다+내다     | 抜き出す                    | ★   |
| 몰아내다            | 追い出す          | 몰다+내다     | 追い出す                    |
| 불러내다            | 呼び出す          | 부르다+내다   | 呼び出す                    |
| 꺼내다              | 取り出す          | 꺼+내다       | 中から出す                  | ★   |
| 들어내다/들추어내다 | 暴き出す          | 들추다+내다   | 持ち上げて出す→暴露する     |
| 빼내다              | 抜き出す          | 빼다+내다     | 引き抜く                    |
| 날려 내다           | 吹き飛ばす        | 날리다+내다   | 飛ばし出す                  |
| 길러내다            | 育て上げる        | 기르다+내다   | 育て出す                    |
| 짜내다              | 絞り出す          | 짜다+내다     | 絞り出す(아이디어를 짜내다) | ★   |
| 쥐어짜내다          | しぼり尽くす      | 쥐어짜다+내다 | ぎゅっと絞り出す            |

> **法則**: 動詞+내다 = 日本語の「〜出す」「〜切る」にほぼ対応。**完遂・達成**のニュアンス。

## 二、들이다/들다 (入れる/入る) 系 — 「中に入り込む」

| 韓語       | 意味                | 分解            | 日本語                        | ★   |
| ---------- | ------------------- | --------------- | ----------------------------- | --- |
| 끌어들이다 | 引き込む            | 끌다+들이다     | 引いて入れる                  | ★📖 |
| 빠져들다   | のめり込む          | 빠지다+들다     | 抜けて入る→嵌り込む           | ★📖 |
| 뛰어들다   | 飛び込む            | 뛰다+들다       | 跳んで入る                    | ★   |
| 파고들다   | 食い込む/掘り下げる | 파다(掘)+들다   | 掘って入る→深く追究する       | ★📖 |
| 스며들다   | 染み込む            | 스미다(染)+들다 | 染みて入る→浸透する           | ★📖 |
| 밀려들다   | 押し寄せる          | 밀리다+들다     | 押されて入る                  | ★   |
| 몰려들다   | 殺到する            | 몰리다+들다     | 追われて入る→どっと押し寄せる | ★   |
| 달려들다   | 飛びかかる          | 달리다+들다     | 走って入る→飛びかかる         |
| 불어들이다 | 呼び入れる          | 부르다+들이다   | 呼んで入れる                  |
| 받아들이다 | 受け入れる          | 받다+들이다     | 受けて入れる→受容する         | ★📖 |
| 들여다보다 | 覗き込む            | 들여+다+보다    | 中に入って見る                |

> **法則**: 動詞+들다/들이다 = 日本語の「〜込む」。内部浸透・深化。

| 韓語        | 意味           | 分解          | 日本語              | ★   |
| ----------- | -------------- | ------------- | ------------------- | --- |
| 내려놓다    | 降ろす/手放す  | 내리다+놓다   | 下に置く→手放す     | ★   |

| 풀어놓다    | 解き放つ       | 풀다+놓다     | 解いて放す          |
| 떼어놓다    | 引き離す       | 떼다+놓다     | 剥がして置く        |
| 쌓아놓다    | 積み上げておく | 쌓다+놓다     | 積んで置く          |
| 벌려놓다    | 広げておく     | 벌리다+놓다   | 広げて置く          |

| 늘어놓다    | 並べ立てる     | 늘다+놓다     | 伸ばして置く→並べる | ★   |

## 十、核心ルール総括

### 複合動詞の構造方程式

```
[方向接頭辞] + [基本動詞A] + 어/아 + [補助動詞B] = 複合動詞
```

### 補助動詞B の意味マップ

| 補助動詞        | 核心意味     | 日本語対応            |
| --------------- | ------------ | --------------------- |
| **내다**        | 完遂・達成   | 〜出す/〜切る         |
| **들다/들이다** | 浸透・内部化 | 〜込む                |
| **붙이다**      | 密着・強行   | 〜付ける              |
| **놓다**        | 保持・放任   | 〜ておく              |
| **버리다**      | 完了・残念   | 〜てしまう            |
| **대다**        | 接触・開始   | 〜始める/〜触れる     |
| **지다**        | 自動詞化     | 〜になる/〜れる       |
| **보다**        | 試行         | 〜てみる              |
| **가다**        | 進行・離去   | 〜ていく              |
| **오다**        | 経験・到来   | 〜てくる              |
| **주다**        | 利他         | 〜てあげる/〜てくれる |

> **結論**: 基本動詞20個 × 補助動詞11個 × 方向接頭辞10個 = 理論上**2200通り**の組合せ。実際に使われるのは約300語。この300語が固有語動詞の実質的な全体。上の体系を理解すれば、未知の複合動詞も文脈で推測可能。

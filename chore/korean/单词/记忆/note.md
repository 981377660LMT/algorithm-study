# 韓国語語彙 速成記憶法 — 日本語 N1 学習者向け

> **核心思路**：韓語 60-70% 是漢字詞，對日語 N1 來說幾乎「免費」。本文件系統化利用日韓語言學關聯，讓你用最少的精力記住最多的詞。
>
> - 第一章：日韓漢字音系統對應 → 秒殺全部漢字詞（約 3000 詞"零成本"）
> - 第二章：固有詞語源分解 → 用詞根、中世韓語、韓日同源詞攻克固有詞
> - 第三章：擬聲擬態詞日韓對照 → 利用日語 オノマトペ 記憶
> - 第四章：慣用語·俗語·四字成語 日語等價表
> - 第五章：易混淆詞 日語區分法

# 第一章 日韓漢字音 系統對應規則

> **原理**：日語音読み和韓語漢字音都來自中古漢語，存在高度系統性的音韻對應。
> 掌握這套規則後，聽到日語音読み就能直接推出韓語讀音，反之亦然。

## 1.1 聲母（子音）對應總表

| 日語音読み子音 | 韓語子音 | 代表例                                                       |
| -------------- | -------- | ------------------------------------------------------------ |
| k- (カ行)      | ㄱ       | 学(がく→학), 国(こく→국), 教(きょう→교), 経(けい→경), 家(か→가) |
| s- (サ行)      | ㅅ       | 社(しゃ→사), 生(せい→생), 市(し→시), 成(せい→성), 心(しん→심) |
| t- (タ行)      | ㄷ/ㅌ    | 大(だい→대), 地(ち→지), 東(とう→동), 統(とう→통), 短(たん→단) |
| n- (ナ行)      | ㄴ       | 南(なん→남), 年(ねん→년), 内(ない→내), 能(のう→능)            |
| h- (ハ行)      | ㅎ       | 学(はく→학)\*, 韓(かん→한), 化(か→화), 現(げん→현)           |
| m- (マ行)      | ㅁ       | 民(みん→민), 文(ぶん→문), 名(めい→명), 面(めん→면)           |
| r- (ラ行)      | ㄹ       | 力(りょく→력), 理(り→리), 論(ろん→론), 料(りょう→료)          |
| b- (バ行)      | ㅂ       | 北(ほく→북), 不(ふ→불), 分(ぶん→분), 法(ほう→법)             |
| j-/ch- (ジャ行) | ㅈ/ㅊ    | 政(せい→정), 主(しゅ→주), 自(じ→자), 中(ちゅう→중), 長(ちょう→장) |

> \* 注意日語ハ行在漢字音中有特殊歷史：古代日語 p→ɸ→h 的音變導致部分ハ行字實際對應韓語 ㅂ 或 ㅎ。

### 重要例外

| 日語          | 韓語     | 說明                                             |
| ------------- | -------- | ------------------------------------------------ |
| ハ行→ㅂ       | 法(ほう→법), 服(ふく→복), 佛(ぶつ→불) | 日語 h- 來自古日語 p-，所以對應韓語 ㅂ |
| ワ行→ㅇ/없    | 話(わ→화), 和(わ→화)                  | 日語ワ行多對應韓語 ㅎ+ㅘ              |
| 零聲母(ア行)→ㅇ | 安(あん→안), 一(いち→일), 音(おん→음) | 日語ア行 = 韓語零聲母ㅇ               |

## 1.2 韻母（母音+收音）對應總表

### 收音(받침)與日語的對應 — 最核心的規律

| 韓語받침 | 日語對應特徵 | 代表例                                                       |
| -------- | ------------ | ------------------------------------------------------------ |
| **ㄱ**   | 日語 -ku/-ki + 促音(ッ) | 学(がく→학), 国(こく→국), 作(さく→작), 特(とく→특), 食(しょく→식) |
| **ㄴ**   | 日語 -n(撥音ン)          | 人(じん→인), 安(あん→안), 間(かん→간), 文(ぶん→문), 信(しん→신) |
| **ㄹ**   | 日語 -tsu/-chi + 促音     | 発(はつ→발), 物(ぶつ→물), 実(じつ→실), 日(にち→일), 鉄(てつ→철) |
| **ㅂ**   | 日語 -pu/-bu + 促音       | 合(がっ→합/ごう→합), 法(ほう→법), 入(にゅう→입), 業(ぎょう→업) |
| **ㅇ**   | 日語 長音(-う/-い) 或 -ng | 生(せい→생), 中(ちゅう→중), 東(とう→동), 公(こう→공), 経(けい→경) |
| **없음** | 日語 短母音              | 社(しゃ→사), 地(ち→지), 自(じ→자), 可(か→가)                 |

> **記憶口訣**：
> - 日語有**促音(ッ)**的 → 韓語大概率有**ㄱ/ㄹ/ㅂ收音**
> - 日語有**撥音(ン)**的 → 韓語有**ㄴ/ㅁ/ㅇ收音**
> - 日語有**長音**的(-う/-い) → 韓語有**ㅇ收音**或雙母音

## 1.4 日韓偽友（同形異義/意味ズレ）

> 以下漢字詞日韓寫法相同但意義有微妙或重大差異，需要特別注意。

| 漢字詞   | 日語意義            | 韓語意義           | 陷阱                     |
| -------- | ------------------- | ------------------ | ------------------------ |
| 約束(약속) | 約束（承諾/約定）   | 約定（偏"約定"義） | 日語偏"約束/束縛"，韓語偏"約定" |
| 人間(인간) | 人類                | 人類/人品          | 韓語多了"人品"義          |
| 愛人(애인) | 情人（曖昧/不倫意） | 戀人（正式義）     | 日語偏負面，韓語中性       |
| 丈夫(장부) | 沒問題(大丈夫)      | 大丈夫/好漢        | 日語「大丈夫」≠韓語장부    |
| 工夫(공부) | 下功夫/鑽研         | 學習/讀書          | 韓語공부=勉強              |
| 新聞(신문) | 報紙                | 報紙               | 一致，但韓語也指"审问"     |
| 外出(외출) | 外出                | 外出               | 一致 ✓                    |
| 無理(무리) | 不合理/勉強         | 不合理/不可能      | 基本一致                   |
| 気分(기분) | 心情                | 心情               | 一致 ✓                    |
| 迷惑(미혹) | 麻煩/打擾           | 迷惑/被迷住        | 意思完全不同！              |
| 勉強(면강) | 學習                | 勉強(強迫義)       | 日語的"勉強"=韓語的"공부"  |
| 親切(친절) | 親切                | 親切               | 一致 ✓                    |
| 検討(검토) | 檢討/審查           | 檢討/審查          | 一致 ✓                    |
| 放送(방송) | 播放/廣播           | 播放/廣播          | 一致 ✓                    |

# 第二章 固有詞語源分解記憶法

> **固有詞（고유어）是韓語學習者的最大難關。** 漢字詞可以靠日韓音對應秒殺，但固有詞沒有漢字可依靠。
> 本章提供三大武器：**① 韓日同源詞** ② **詞根/詞素分解** ③ **中世韓語語源故事**

## 2.1 韓日同源詞（Proto-Koreanic / Proto-Japonic）

> 韓語和日語在類型學上有很多相似之處（SOV語序、助詞膠著、敬語體系），部分學者認為存在遠古同源關係。
> 以下列表**不做學術判斷**，僅作為**記憶工具**使用。標注 ⚠ 的為學術爭議較大者。

| 韓語       | 日語       | 語義       | 記憶線索                                     |
| ---------- | ---------- | ---------- | -------------------------------------------- |
| 나 (na)    | な/汝 (na) | 我/你      | 古日語「な」=你，韓語나=我。人稱代詞同源 ⚠    |
| 너 (neo)   | —          | 你         | 對比：나=我、너=你                            |
| 나라 (nara) | 奈良 (nara) | 國家       | 奈良(なら)：古都名，可能源自"國" ⚠            |
| 섬 (seom)  | 島/しま (shima) | 島嶼   | s-ㅓm ↔ sh-i-ma，子音框架 s-m 一致 ⚠         |
| 곰 (gom)   | 熊/くま (kuma) | 熊      | g-o-m ↔ k-u-ma，子音框架 k/g-m 一致           |
| 바다 (bada) | 海/わた (wata) | 海     | 古日語「わた」=海(渡し)，b↔w 對應 ⚠           |
| 길 (gil)   | 道/ち (chi) | 路       | 微妙，但 みち(道)的ち可能與길有遠古聯繫 ⚠     |
| 물 (mul)   | 水/みず (mizu) | 水     | m-u-l ↔ m-i-zu，首音 m 一致 ⚠                 |
| 불 (bul)   | 火/ひ (hi)  | 火        | 古日語 p→h：古形 *pui ↔ 불 *bul ⚠             |
| 돌 (dol)   | —           | 石頭      | （無明確日語同源）                             |
| 달 (dal)   | 月/つき (tsuki) | 月    | 較遠，但月光=달빛，つきのひかり ⚠              |
| 별 (byeol) | 星/ほし (hoshi) | 星    | （無明確同源，但可聯想「ほし→별」發音距離）     |
| 무릎 (mureup) | 膝/ひざ (hiza) | 膝蓋 | m-u-r ↔ h-i-z，較遠 ⚠                        |
| 배 (bae)   | 腹/はら (hara) | 肚子   | b↔h(古p)，a↔a 母音一致 ⚠                     |
| 붓 (but)   | 筆/ふで (fude) | 筆     | b-u-t ↔ f-u-de(古 p-u-de) ⚠                  |
| 겨울 (gyeoul) | 冬 (fuyu) | 冬天    | 較遠，但部分學者提出 ⚠                        |

> **使用方法**：不需要相信它們真的同源，只需要用這些聯想作為記憶鉤子。

## 2.2 固有動詞 詞根分解

> 韓語固有動詞很多是複合詞，由 **基礎詞根 + 補助詞素** 組成。
> 把複合詞拆開，理解每個部件的意思，就能理解整個詞為什麼是那個意思。

### 2.2.2 TOPIK 高頻固有動詞 逐詞分解

| 動詞              | 分解                    | 語源故事 / 記憶線索                                  | 日語對譯         |
| ----------------- | ----------------------- | ---------------------------------------------------- | ---------------- |
| 깨닫다 ★          | 깨(醒/破) + 닫다(關)    | "打破封閉的狀態" → 領悟。닫다的古形有"觸及"義         | 悟る、気づく     |
| 알아채다           | 알다(知) + 채다(感知)   | "知道+感知" → 察覺                                   | 気づく、察する   |
| 두드러지다 ★       | 두드리다(敲) 的變體     | "被敲打出來的" → 突出、顯著                          | 目立つ           |
| 뛰어나다 ★         | 뛰다(跳) + 어 + 나다(出) | "跳出來" → 出眾                                     | 優れる、抜きん出る |
| 이루다 ★           | 이-(連) + 루다          | "把事物連成" → 實現、達成                            | 成し遂げる、達成する |
| 이끌다 ★           | 이-(連) + 끌다(拉)     | "連著拉" → 引領                                      | 導く、率いる     |
| 드러나다 ★         | 들-(入/出) + 어 + 나다(出) | "顯出來" → 暴露、顯現                              | 明らかになる、露わになる |

## 2.3 中世韓語語源故事

> 部分固有詞的現代形式已看不出語源，需要追溯到中世韓語(15-16世紀)才能理解。

### 아름답다 (美麗)
- 中世韓語：ᄋᆞᄅᆞᆷ(ᄃᆞᆸ다)
- ᄋᆞᄅᆞᆷ = "自身/自己"（cf. 현대 "아우" = 弟弟，同族語根）
- 답다 = "如同...一般"
- → **"如同自身般珍貴/本質性的"** → 美麗
- 記憶：아름답다 = "自身般珍貴" → 美

### 사랑 (愛)
- 中世韓語：ᄉᆞ랑
- 可能源自 "ᄉᆞᆯ다"(活/生) + 名詞化
- → **"使人活著的東西"** → 愛
- 記憶：사랑 = "生命之物" → 愛

### 슬기 (智慧) → 슬기롭다 (明智的)
- 中世韓語：슬기 = 明智、才能
- 可能與 "살다"(活) 有遠古聯繫
- -롭다 = "充滿...的" 形容詞化尾
- 記憶：슬기롭다 = "充滿智慧的" → 明智的

### 어여쁘다/예쁘다 (漂亮)
- 中世韓語：어엿브다 → 어여쁘다 → 예쁘다
- 古義為 "令人憐愛的/可愛可憐的"
- 記憶：예쁘다 是從"惹人憐愛"演變為"美麗"。cf. 日語かわいい也有"可哀（い）想"→"可愛い" 的語義變遷。

### 보람 (有意義/成就感)
- 中世韓語：보ᄅᆞᆷ
- 可能源自 보다(看) + 名詞化 -ᄅᆞᆷ
- → **"值得看的東西/成果"** → 有意義、值得
- 記憶：보람 = "值得一看的" → 有意義感
- 日語：やりがい、生きがい

### 마음 (心)
- 中世韓語：ᄆᆞᄋᆞᆷ
- 可能與 "맘"(口語縮略形) 同源
- → 韓語最基本的固有詞之一，記住即可
- 日語：心、気持ち

### 끈기 (毅力)
- 끈 = 繩子/帶子
- 끈기 = "像繩子一樣韌" → 毅力、堅韌
- 記憶：끈기 = "繩索般的堅韌" → 粘り強さ

### 겸손 (謙虛) — 漢字詞（謙遜）但常被誤認為固有詞
- 겸(謙) + 손(遜) = 日語そのまま「謙遜」
- 記憶：直接等於日語「けんそん」

# 第三章 擬聲擬態詞（의성어/의태어）日韓對照

> 日語和韓語都是「オノマトペ大國」。很多擬聲擬態詞可以找到日韓對應，利用已知的日語記憶韓語。

## 3.1 元音象徵法則（核心規律）

> 韓語的擬聲擬態詞有一套**母音交替**體系，通過換母音就能改變語感。日語也有類似的一套（清音/濁音交替）。

| 母音類型 | 韓語母音 | 語感                   | 類比日語                 |
| -------- | -------- | ---------------------- | ------------------------ |
| 陽性母音 | ㅏ, ㅗ   | 小、輕、明亮、可愛      | 清音(キラキラ)           |
| 陰性母音 | ㅓ, ㅜ   | 大、重、暗沉、粗野      | 濁音(ギラギラ)           |

### 母音交替實例

# 第五章 易混淆詞 日語區分法

> 韓語中許多近義詞讓學習者頭疼，但通過日語翻譯的差異就能輕鬆區分。

## 5.1 動詞混淆

| 混淆組              | 韓語差異                  | 日語對應（一看就懂）                | 記憶法 |
| ------------------- | ------------------------- | ----------------------------------- | ------ |
| 넓히다 vs 늘리다 ★  | 넓=面積/範圍, 늘=數量/長度 | **広げる** vs **増やす/伸ばす**     | 広 vs 増 |
| 줄이다 vs 낮추다 ★  | 줄=數量, 낮추=程度/水平    | **減らす** vs **下げる/低める**     | 減 vs 下 |
| 바꾸다 vs 변하다 ★  | 바꾸다=他動(具體), 변하다=自動(自然) | **変える** vs **変わる**    | 変える vs 変わる(他動/自動) |
| 맞다 vs 맞추다 ★    | 맞다=對(狀態), 맞추다=使對(他動) | **合う** vs **合わせる**           | 合う vs 合わせる |
| 이루다 vs 이루어지다 ★ | 이루다=他動, 이루어지다=自動 | **成し遂げる** vs **成し遂げられる/実現する** | 能動 vs 受動 |
| 걸다 vs 걸리다 ★    | 걸다=他動(掛/打電話), 걸리다=自動(被掛/花時間/得病) | **掛ける** vs **掛かる** | 掛ける vs 掛かる |
| 받다 vs 받아들이다 ★ | 받다=物理接收, 받아들이다=心理接受 | **受ける/もらう** vs **受け入れる** | 受取る vs 受け入れる |

# 固有語形容詞篇 (上) — 感覚・状態・性質

> 漢字語形容詞(~적이다/~하다)と異なり、語幹変化が複雑な固有語形容詞を体系化
> ★ = TOPIK必須 / 📖 = 읽기頻出 / 📝 = 쓰기頻出

---

## 一、大小・長短・広狭 — 基本対立ペア

| 大/長/広 | 意味         | 小/短/狭          | 意味              | 記憶法        |
| -------- | ------------ | ----------------- | ----------------- | ------------- |
| 크다 ★   | 大きい       | 작다 ★            | 小さい            | 日: 大/小     |
| 길다 ★   | 長い         | 짧다 ★            | 短い              | 日: 長/短     |
| 넓다 ★   | 広い         | 좁다 ★            | 狭い              | 日: 広/狭     |
| 높다 ★   | 高い         | 낮다 ★            | 低い              | 日: 高/低     |
| 깊다 ★   | 深い         | 얕다 ★            | 浅い              | 日: 深/浅     |
| 두껍다 ★ | 厚い         | 얇다 ★            | 薄い              | 日: 厚/薄     |
| 굵다     | 太い         | 가늘다            | 細い              | 日: 太/細     |
| 무겁다 ★ | 重い         | 가볍다 ★          | 軽い              | 日: 重/軽     |
| 넉넉하다 | ゆとりがある | 빠듯하다/빡빡하다 | ぎりぎりだ/きつい | 넉넉=ゆったり |

> **派生規則**: 形容詞→動詞化
>
> - 높다(高い) → 높이다(高める) → 높아지다(高くなる)
> - 넓다(広い) → 넓히다(広げる) → 넓어지다(広くなる)
> - 깊다(深い) → 깊이(深さ/深く) → 깊어지다(深くなる)

## 二、温度・明暗・乾湿

### 温度系

| 韓語       | 意味            | 強弱 | 記憶法            |
| ---------- | --------------- | ---- | ----------------- |
| 뜨겁다 ★   | 熱い(触覚)      | 強   | 뜨거=焼けるような |
| 덥다 ★     | 暑い(気温)      | 中   | 夏は 덥다         |
| 따뜻하다 ★ | 暖かい          | 弱   | 따뜻=ぽかぽか     |
| 미지근하다 | ぬるい          | 微   | 미지근=なまぬるい |
| 시원하다 ★ | 涼しい/すっきり | 弱冷 | 시원=さわやか     |
| 차갑다 ★   | 冷たい(触覚)    | 中冷 | 차갑=冷え冷え     |
| 춥다 ★     | 寒い(気温)      | 強冷 | 冬は 춥다         |

> **対比**: 덥다(暑い)⇔춥다(寒い) = 気温。뜨겁다(熱い)⇔차갑다(冷たい) = 触覚。日本語と同じ区別。

## 九、形容詞→名詞化 パターン

| 形容詞 | 名詞(-ㅁ/-음)  | 名詞(-이)       | 名詞(-기)    |
| ------ | -------------- | --------------- | ------------ |
| 슬프다 | 슬픔(悲しみ)   | -               | -            |
| 기쁘다 | 기쁨(喜び)     | -               | -            |
| 아프다 | 아픔(痛み)     | -               | -            |
| 즐겁다 | 즐거움(楽しみ) | -               | -            |
| 두렵다 | 두려움(恐れ)   | -               | -            |
| 외롭다 | 외로움(寂しさ) | -               | -            |
| 깊다   | 깊음           | 깊이(深さ/深く) | -            |
| 높다   | 높음           | 높이(高さ/高く) | -            |
| 넓다   | 넓음           | 넓이(広さ/面積) | -            |
| 길다   | 길음           | 길이(長さ)      | -            |
| 크다   | -              | -               | 크기(大きさ) |
| 넓다   | -              | -               | 넓기         |

> **法則**: 感情系→ -ㅁ/음、寸法系→ -이、程度系→ -기

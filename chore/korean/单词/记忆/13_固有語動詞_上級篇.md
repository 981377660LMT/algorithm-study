## 九、受動・使役の体系 — 固有語動詞の核心文法

### 受動(被動) — 語幹 + 이/히/리/기

| ------------ | ------ | -------- | -------------- |

| 먹다(食べる) | 먹히다 | -히      | 食べられる     |
| 읽다(読む)   | 읽히다 | -히      | 読まれる       |
| 밀다(押す)   | 밀리다 | -리      | 押される       |
| 끌다(引く)   | 끌리다 | -리      | 引かれる       |
| 듣다(聞く)   | 들리다 | -리      | 聞こえる       |
| 풀다(解く)   | 풀리다 | -리      | 解ける         |
| 열다(開ける) | 열리다 | -리      | 開く           |
| 보다(見る)   | 보이다 | -이      | 見える         |
| 놓다(置く)   | 놓이다 | -이      | 置かれる       |
| 쓰다(使う)   | 쓰이다 | -이      | 使われる       |

> **규칙**: ㄱ/ㅂ語幹→이, ㄷ/ㅈ/ㅂ語幹→히, ㄹ語幹→리, ㄴ/ㅁ/ㅇ語幹→기（例外あり）

### 使役(使動) — 語幹 + 이/히/리/기/우/추

| ------------ | ------ | -------- | ----------------- |

| 죽다(死ぬ)   | 죽이다 | -이      | 殺す(死なせる)    |
| 살다(生きる) | 살리다 | -리      | 生かす            |

| 늦다(遅い)   | 늦추다 | -추      | 遅らせる          |

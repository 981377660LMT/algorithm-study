## 五、速推の核心ルール (note.md第1章の要約)

> これだけ覚えれば全漢字語が推測可能

### 子音対応

| 日本語  | 韓語  | 例                       |
| ------- | ----- | ------------------------ |
| か行(k) | ㄱ    | 科(か→과), 国(こく→국)   |
| た行(t) | ㄷ/ㅌ | 大(だい→대), 統(とう→통) |
| さ行(s) | ㅅ    | 社(しゃ→사), 生(せい→생) |
| は行(h) | ㅎ    | 学(がく→학), 化(か→화)   |
| ま行(m) | ㅁ    | 民(みん→민), 文(ぶん→문) |
| な行(n) | ㄴ    | 年(ねん→년), 内(ない→내) |

### 漢字音速推公式

```
日本語の音読み → 子音対応表で初声変換 → 母音対応表で中声変換 → 完成
例: 教育(きょういく) → 교(きょう→교) + 육(いく→육) = 교육 ✅
```

> **本ファイルの全漢字語は、この速推公式で日本語から導出可能。個別暗記の必要なし。**

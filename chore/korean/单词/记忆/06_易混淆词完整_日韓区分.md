# 易混淆詞 完整辨析 — 日語區分法（全 ~100 組）

> **核心方法**：用日語翻譯的差異來區分韓語近義詞。日語 N1 學習者一看就懂。

---

## A. 動詞混淆

### 1. 넓히다 vs 늘리다 vs 늘다 ★

| 詞     | 語義            | 日語              | 記憶 |
| ------ | --------------- | ----------------- | ---- |
| 넓히다 | 擴大(面積/範圍) | **広げる**        | 広   |
| 늘리다 | 增加(數量/長度) | **増やす/伸ばす** | 増   |
| 늘다   | (自動)增加      | **増える**        | 自動 |

### 2. 줄이다 vs 줄다 vs 낮추다 ★

| 詞     | 語義            | 日語              | 記憶 |
| ------ | --------------- | ----------------- | ---- |
| 줄이다 | 減少(他動)      | **減らす**        | 他動 |
| 줄다   | 減少(自動)      | **減る**          | 自動 |
| 낮추다 | 降低(程度/水平) | **下げる/低める** | 下   |

### 3. 바꾸다 vs 변하다 vs 변화하다 ★

| 詞       | 語義            | 日語              | 記憶 |
| -------- | --------------- | ----------------- | ---- |
| 바꾸다   | 改換(他動/具體) | **変える/替える** | 他動 |
| 변하다   | 變化(自動/自然) | **変わる**        | 自動 |
| 변화하다 | 變化(自動/抽象) | **変化する**      | 漢語 |

### 4. 맞다 vs 맞추다 ★

| 詞     | 語義          | 日語            | 記憶 |
| ------ | ------------- | --------------- | ---- |
| 맞다   | 對(狀態/被動) | **合う/当たる** | 自動 |
| 맞추다 | 使對上(他動)  | **合わせる**    | 他動 |

### 5. 잡다 vs 잡히다 vs 붙잡다

| 詞     | 語義       | 日語              | 記憶 |
| ------ | ---------- | ----------------- | ---- |
| 잡다   | 抓(主動)   | **捕まえる/掴む** | 能動 |
| 잡히다 | 被抓(被動) | **捕まる**        | 受動 |
| 붙잡다 | 緊抓不放   | **しがみつく**    | 強調 |

### 6. 걸다 vs 걸리다 ★

| 詞     | 語義             | 日語       | 記憶 |
| ------ | ---------------- | ---------- | ---- |
| 걸다   | 掛/打電話        | **掛ける** | 他動 |
| 걸리다 | 被掛/花時間/得病 | **掛かる** | 自動 |

### 7. 이루다 vs 이루어지다 ★

| 詞         | 語義         | 日語                        | 記憶 |
| ---------- | ------------ | --------------------------- | ---- |
| 이루다     | 實現(他動)   | **成し遂げる**              | 能動 |
| 이루어지다 | 被實現(自動) | **実現する/成し遂げられる** | 受動 |

### 8. 받다 vs 받아들이다 ★

| 詞         | 語義       | 日語              | 記憶 |
| ---------- | ---------- | ----------------- | ---- |
| 받다       | 接收(物理) | **受ける/もらう** | 物理 |
| 받아들이다 | 接受(心理) | **受け入れる**    | 心理 |

## B. 形容詞混淆

### 9. 다르다 vs 틀리다 ★★(最常考！)

| 詞     | 語義       | 日語            | 記憶   |
| ------ | ---------- | --------------- | ------ |
| 다르다 | 不同(中性) | **違う/異なる** | 違い   |
| 틀리다 | 錯誤(負面) | **間違う**      | 間違い |

> 핵심：의견이 다르다(意見不同) ≠ 답이 틀리다(答案錯了)

### 10. 쓸데없다 vs 소용없다 vs 필요없다

| 詞       | 語義          | 日語                  | 記憶       |
| -------- | ------------- | --------------------- | ---------- |
| 쓸데없다 | 無聊的/沒必要 | **くだらない/無駄な** | 無駄       |
| 소용없다 | 做了也沒用    | **無駄だ(結果的)**    | 甲斐がない |
| 필요없다 | 本就不需要    | **必要ない**          | 不要       |

### 11. 익숙하다 vs 친숙하다 vs 낯설다 ★

| 詞       | 語義         | 日語             | 記憶   |
| -------- | ------------ | ---------------- | ------ |
| 익숙하다 | 通過經驗熟悉 | **慣れている**   | 慣れ   |
| 친숙하다 | 親近/親切感  | **親しみがある** | 親しみ |
| 낯설다   | 陌生         | **見慣れない**   | 陌生   |

### 12. 어렵다 vs 힘들다 ★

| 詞     | 語義       | 日語            | 記憶 |
| ------ | ---------- | --------------- | ---- |
| 어렵다 | 能力上困難 | **難しい**      | 難   |
| 힘들다 | 體力上辛苦 | **辛い/大変だ** | 辛   |

### 13. 부족하다 vs 모자라다

| 詞       | 語義            | 日語         | 記憶 |
| -------- | --------------- | ------------ | ---- |
| 부족하다 | 不足(抽象/書面) | **不足する** | 書面 |
| 모자라다 | 不夠(具體/口語) | **足りない** | 口語 |

### 14. 정확하다 vs 명확하다 vs 분명하다 ★

| 詞       | 語義      | 日語         | 記憶 |
| -------- | --------- | ------------ | ---- |
| 정확하다 | 正確/精確 | **正確だ**   | 正確 |
| 명확하다 | 明確      | **明確だ**   | 明確 |
| 분명하다 | 分明/明顯 | **明らかだ** | 明白 |

### 15. 적절하다 vs 적합하다 ★

| 詞       | 語義           | 日語         | 記憶 |
| -------- | -------------- | ------------ | ---- |
| 적절하다 | 適當(程度方法) | **適切だ**   | 適切 |
| 적합하다 | 適合(條件匹配) | **適合する** | 適合 |

## C. 名詞混淆

### 16. 과정 vs 결과 vs 원인 ★

| 詞   | 日語     | 記憶 |
| ---- | -------- | ---- |
| 과정 | **過程** | 過程 |
| 결과 | **結果** | 結果 |
| 원인 | **原因** | 原因 |

### 17. 비율 vs 비중 vs 비례 ★

| 詞   | 語義         | 日語              | 記憶       |
| ---- | ------------ | ----------------- | ---------- |
| 비율 | 比率(A:B)    | **比率**          | rate       |
| 비중 | 比重(份額)   | **比重/ウェイト** | weight     |
| 비례 | 比例(正反比) | **比例**          | proportion |

### 18. 대상 vs 대안 vs 대책 ★

| 詞   | 語義     | 日語     | 記憶 |
| ---- | -------- | -------- | ---- |
| 대상 | 對象     | **対象** | 象   |
| 대안 | 替代方案 | **代案** | 案   |
| 대책 | 對策     | **対策** | 策   |

### 19. 목적 vs 목표 ★

| 詞   | 語義           | 日語     | 記憶 |
| ---- | -------------- | -------- | ---- |
| 목적 | 目的(為什麼做) | **目的** | why  |
| 목표 | 目標(達到的點) | **目標** | what |

### 20. 의미 vs 의의 ★

| 詞   | 語義      | 日語     | 記憶 |
| ---- | --------- | -------- | ---- |
| 의미 | 意思/含義 | **意味** | 意味 |
| 의의 | 意義/價值 | **意義** | 意義 |

### 21. 장점 vs 이점 vs 강점

| 詞   | 語義         | 日語              | 記憶 |
| ---- | ------------ | ----------------- | ---- |
| 장점 | 優點(一般)   | **長所/メリット** | 長所 |
| 이점 | 益處(利益)   | **利点**          | 利点 |
| 강점 | 強項(競爭力) | **強み**          | 強み |

## D. 副詞混淆

### 22. 아직 vs 여전히 vs 그래도 ★

| 詞     | 語義       | 日語                      | 記憶 |
| ------ | ---------- | ------------------------- | ---- |
| 아직   | 還(含期待) | **まだ**                  | まだ |
| 여전히 | 仍然(不變) | **依然として/相変わらず** | 依然 |
| 그래도 | 即便如此   | **それでも**              | けど |

### 23. 거의 vs 전혀 vs 결코 ★

| 詞   | 語義           | 日語            | 記憶 |
| ---- | -------------- | --------------- | ---- |
| 거의 | 幾乎(≈100%)    | **ほとんど**    | ≈100 |
| 전혀 | 完全(+否定=0%) | **全く~ない**   | =0   |
| 결코 | 絕不(強調否定) | **決して~ない** | 強調 |

### 24. 오히려 vs 도리어 vs 되레 ★

| 詞     | 語義            | 日語                | 記憶 |
| ------ | --------------- | ------------------- | ---- |
| 오히려 | 反而(標準書面)  | **むしろ/かえって** | 書面 |
| 도리어 | 反而(稍舊/文學) | **かえって**        | 文語 |
| 되레   | 反而(口語)      | **かえって**        | 口語 |

### 25. 단지 vs 단순히 vs 불과 ★

| 詞     | 語義         | 日語                    | 記憶   |
| ------ | ------------ | ----------------------- | ------ |
| 단지   | 只是(限定)   | **ただ/単に**           | only   |
| 단순히 | 單純地(方式) | **単に/単純に**         | simply |
| 불과   | 僅僅(強調少) | **わずか/〜に過ぎない** | merely |

## E. 語法/助詞混淆

### 26. -(으)로 인해 vs -기 때문에 vs -는 바람에 ★

| 語法         | 語義            | 日語         | 場景     |
| ------------ | --------------- | ------------ | -------- |
| -(으)로 인해 | 由於(書面)      | **〜により** | 正式寫作 |
| -기 때문에   | 因為(通用)      | **〜ため**   | 萬能     |
| -는 바람에   | 因為(意外/負面) | **〜せいで** | 意外     |

### 27. -에 대해(서) vs -에 관해(서) ★

| 語法     | 日語           | 場景   |
| -------- | -------------- | ------ |
| -에 대해 | **〜について** | 一般   |
| -에 관해 | **〜に関して** | 更正式 |

### 28. -도록 vs -(으)려고 ★

| 語法      | 日語                 | 焦點     |
| --------- | -------------------- | -------- |
| -도록     | **〜ように**         | 結果導向 |
| -(으)려고 | **〜(しよ)うとして** | 意圖導向 |

### 29. -(으)며 vs -고 ★

| 語法    | 日語                    | 場景 |
| ------- | ----------------------- | ---- |
| -(으)며 | **〜であり/〜しながら** | 書面 |
| -고     | **〜して**              | 通用 |

### 30. -는 데 vs -는데 ★

| 語法          | 日語                      | 語義                |
| ------------- | ------------------------- | ------------------- |
| -는 데 (空格) | **〜するのに/〜する点で** | 依存名詞(場所/方面) |
| -는데 (連寫)  | **〜が/〜けど**           | 連接語尾(轉折/背景) |

---

> **總結**：大部分韓語混淆詞都能通過日語翻譯的差異秒殺區分。
> 特別注意 **다르다≠틀리다**（每次TOPIK必考！）

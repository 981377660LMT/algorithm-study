# 固有語擬声擬態拡張篇 — 母音交替で芋づる式200語

> 05篇の拡張版。韓国語の擬声語擬態語は母音を入れ替えるだけで微妙に意味が変わる
> この「母音交替(모음교체)」ルールを理解すれば暗記量が半分になる
> ★ = TOPIK必須 💬 = 日常会話

---

## 一、母音交替の法則 — 陽母音 vs 陰母音

```
陽母音(양모음): ㅏ, ㅗ → 明るい・軽い・小さい・速い
陰母音(음모음): ㅓ, ㅜ → 暗い・重い・大きい・遅い

例: 반짝반짝(キラキラ/小) ↔ 번쩍번쩍(ピカピカ/大)
```

## 八、擬声擬態語 → 動詞化パターン

### -하다 を付ける場合

| 擬態語 | 形容詞化   | 意味         |
| ------ | ---------- | ------------ |
| 답답   | 답답하다 ★ | もどかしい   |
| 찡     | 찡하다 ★   | じーんとくる |
| 뻐근   | 뻐근하다   | ずきずきする |
| 허전   | 허전하다 ★ | ぽっかり空虚 |
| 아찔   | 아찔하다 ★ | くらっとする |
| 찜찜   | 찜찜하다   | もやもやする |

> **記憶のコツ**: 擬声擬態語は「音と感覚の直結」なので、**声に出して体で覚える**のが最も効果的。黙読では絶対に身につかない。

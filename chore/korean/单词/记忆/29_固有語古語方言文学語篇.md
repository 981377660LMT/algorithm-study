> TOPIK읽기後半(43-50番)には文学作品(소설/수필)が出る
> 文学作品には日常会話では使わない古風な固有語や方言が登場する
> ★ = TOPIK읽기出題 📖 = 文学頻出

### 마침내 vs 드디어 vs 결국

```
마침내: 長い過程の末に(文語的・格調高い) ★📖
  → 마침내 자유를 얻었다 (ついに自由を得た)

드디어: 待望のことが実現(期待感) ★
  → 드디어 합격했다! (とうとう合格した！)

결국: 結果的に(良くも悪くも) ★📖
  → 결국 실패로 끝났다 (結局失敗に終わった)
```

## 六、TOPIK읽기 文学問題の攻略パターン

### 出題形式

```
[43-44番] 次の文を読んで問いに答えよ
→ 小説の一場面(3-5行)
→ 問1: 登場人物の心情・態度を選べ
→ 問2: 内容と一致するものを選べ
```

### 心情を表す固有語選択肢

| 選択肢に出る語 | 意味               | 頻度       |
| -------------- | ------------------ | ---------- |
| 서운하다 ★     | 寂しい・物足りない | 非常に高い |
| 섭섭하다 ★     | 名残惜しい・残念   | 高い       |
| 후련하다 ★     | すっきりする       | 高い       |
| 뿌듯하다 ★     | 誇らしい           | 高い       |
| 허전하다 ★     | ぽっかり空虚       | 高い       |
| 안타깝다 ★     | 気の毒・もどかしい | 非常に高い |
| 뻔뻔하다       | 図々しい           | 中         |
| 떨떠름하다     | 気乗りしない       | 中         |
| 찜찜하다       | もやもやする       | 中         |
| 씁쓸하다 ★     | ほろ苦い           | 高い       |
| 서글프다       | もの悲しい         | 中         |
| 막막하다 ★     | 途方に暮れる       | 高い       |
| 답답하다 ★     | もどかしい         | 非常に高い |
| 무안하다       | きまりが悪い       | 中         |
| 난감하다 ★     | 困惑する・参る     | 高い       |

> **攻略法**: これらの固有語形容詞は全て「微妙な感情」を表す。漢字語では表現しきれない「韓国語特有の感情語彙」であり、日本語の「せつない」「もどかしい」「やるせない」に近い粒度の違いがある。文脈の「温度感」で判断すること。

## 七、ことわざ・慣用句に残る古語

| ことわざ                             | 古語要素          | 現代語 | 意味                     |
| ------------------------------------ | ----------------- | ------ | ------------------------ |
| 가는 말이 고와야 오는 말이 곱다 ★    | 곱다=美しい       | —      | 売り言葉に買い言葉       |
| 우물 안 개구리 ★                     | 우물=井戸         | —      | 井の中の蛙               |
| 세 살 버릇 여든까지 간다 ★           | 여든=80(固有数詞) | —      | 三つ子の魂百まで         |
| 티끌 모아 태산 ★                     | 티끌=塵(固有語)   | —      | 塵も積もれば山となる     |
| 돌다리도 두들겨 보고 건너라 ★        | 두들기다=叩く     | —      | 石橋を叩いて渡れ         |
| 하늘이 무너져도 솟아날 구멍이 있다 ★ | 솟아나다=湧き出る | —      | 捨てる神あれば拾う神あり |

> **最終メモ**: 文学作品の固有語は「感覚で味わう」もの。一語一語の暗記より、文脈の「温度」「色」「重さ」を感じ取る訓練が、TOPIK읽기最後の5点を左右する。

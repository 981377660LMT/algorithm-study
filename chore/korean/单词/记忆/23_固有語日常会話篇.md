# 固有語日常会話篇 — 口語でしか出会えない固有語200選

> 教科書では漢字語で習っても、実際の韓国人は固有語で話す
> 日本語の「書き言葉 vs 話し言葉」と同じ現象
> ★ = 듣기必須 💬 = 口語限定

## 五、口語限定の固有語フレーズ

### 「-거든(요)」=「〜なんですよ」(固有語語尾)

```
내일 시험이거든요. → 明日試験なんですよ
```

### 「-잖아(요)」=「〜じゃないですか」

```
어제 말했잖아요. → 昨日言ったじゃないですか ★
```

### 「뭐 어쩌겠어」=「しょうがないじゃん」

```
뭐+어쩌다(どうする)+겠+어 → まあどうしようもない 💬
```

## 七、固有語の程度表現（口語グラデーション）

### 「とても」の固有語バリエーション

```
약간 < 좀 < 꽤 < 제법 < 무척 < 매우 < 아주 < 엄청(나게) < 되게 < 겁나 < 개(-)
 少し  ちょっと  かなり  なかなか  とても  非常に  すごく  めちゃ  すごく← 口語  激←俗語
```

| レベル | 固有語         | 文体     | 日本語       |
| ------ | -------------- | -------- | ------------ |
| 1      | 약간           | 普通     | 若干         |
| 2      | 좀 ★           | 口語     | ちょっと     |
| 3      | 꽤 ★           | 普通     | かなり       |
| 4      | 제법 ★         | やや硬い | なかなか     |
| 5      | 무척 ★         | 普通     | とても       |
| 6      | 매우 ★         | 書き言葉 | 非常に       |
| 7      | 아주 ★         | 普通     | すごく       |
| 8      | 엄청(나게) ★💬 | 口語     | めちゃくちゃ |
| 9      | 되게 💬        | 口語     | すっごく     |
| 10     | 겁나 💬        | 俗語     | やばいくらい |

### 「全然」の固有語バリエーション

```
否定文のみ使用：
전혀 < 도무지 < 도통 < 통 < 영
★📖    📖     📖    💬   💬

전혀 모르겠다 → 全く分からない
도통 모르겠다 → さっぱり分からない
```

## 八、듣기頻出 固有語の聞き取りトラップ

| 聞こえる音   | 実際の単語           | 意味               | トラップ                |
| ------------ | -------------------- | ------------------ | ----------------------- |
| 모르겠는데요 | 모르다+겠+는데+요    | 分からないんですが | 모르겠+ㄴ데 の連音      |
| 괜찮으시면   | 괜찮다+으시+면       | よろしければ       | 시の挿入を聞き逃す      |
| 어쩔 수 없지 | 어쩌다+ㄹ 수 없다+지 | しょうがないよ     | 어쩌→어쩔 の活用        |
| 뭐라고요?    | 뭐+라고+요           | なんて言いました？ | 뭐라고 が一語に聞こえる |
| 그럴 리가    | 그렇+ㄹ 리가         | そんなはずが       | 그러→그럴 の活用        |
| 어떡해       | 어떻게 하다          | どうしよう         | 어떻게→어떡 の縮約      |
| 그랬구나     | 그렇+했+구나         | そうだったんだ     | 그래→그랬 の過去        |
| 됐다         | 되다+었+다           | よし/もういい      | 되었다→됐다             |

> **듣기対策**: 固有語は縮約が激しい。原形を知っていても「縮約形を聞いて分かる」訓練が必須。上の表の音声変化パターンを体に覚え込ませること。

# 固有語語根・派生篇 — 一つの語根から10個の単語を芋づる式に

> 固有語の「漢字に当たるもの」= **語根(어근)**。語根を掴めば未知語も推測可能
> ★ = TOPIK必須

## 四、語根組合せ法 — 未知語を推測する

### 例題：「뒷받침하다」を分解せよ

```
뒷 + 받침 + 하다
뒤(後ろ) + 받치다(支える) + ㅁ(名詞化) + 하다(する)
= 後ろから支える → 裏付ける ★
```

### 例題：「걸림돌」を分解せよ

```
걸리(引っかかる) + ㅁ(名詞化) + 돌(石)
= 引っかかる石 → 障害物 ★
```

### 例題：「밑거름」を分解せよ

```
밑(下) + 거름(肥料 ← 거르다=濾す/肥やす)
= 下の肥料 → 下地・土台 ★
```

### 例題：「마중물」を分解せよ

```
마중(出迎え) + 물(水)
= 迎え水 → 呼び水(最初に入れると後から出てくる)
```

## 五、全体系総括 — 固有語攻略ロードマップ

```
STEP 1: 基本動詞20個を完璧に (12篇)
    ↓
STEP 2: 方向接頭辞10個をマスター (12篇・十章)
    ↓ → ここだけで複合動詞100個を自動生成
STEP 3: 補助動詞11個で複合動詞を理解 (14篇)
    ↓ → 理論上2200通り、実用300語が視野に
STEP 4: 受動/使役の接辞パターン (13篇・九章)
    ↓ → 能動↔受動↔使役の三角関係で×3倍
STEP 5: 形容詞の対立ペア+不規則活用 (15-16篇)
    ↓
STEP 6: 副詞の程度/時間/否定/様態分類 (17篇)
    ↓
STEP 7: 名詞(具象→抽象→比喩) (18-19篇)
    ↓
STEP 8: コロケーション(名詞+動詞の固定搭配) (20篇)
    ↓
STEP 9: 語根+接頭辞/接尾辞で未知語推測 (本篇)
    ↓
GOAL: 固有語約1500語を体系的に把握 → TOPIK6級合格ライン
```

> **最終結論**: 固有語は「語根30個 + 方向接頭辞10個 + 補助動詞11個 + 固有語接頭辞12個 + 固有語接尾辞9個」= 合計72個の「部品」で**1500語の90%以上**が推測可能。漢字語が「漢字を知れば推測可能」なのと同様に、固有語も「語根を知れば推測可能」。

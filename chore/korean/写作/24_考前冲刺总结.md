## 三、Q53 图表作文 四段结构

```
① 도입: ~에 대해 살펴보겠다.
② 그래프1: 조사에 따르면 ~(으)ㄴ 것으로 나타났다.
③ 그래프2: 또한/한편/반면 ~(으)ㄴ 것으로 조사되었다.
④ 원인/전망: 이는 ~기 때문인 것으로 보인다/분석된다.
```

**数据表达速查**:

- 增: 증가하다 / 상승하다 / 늘어나다
- 减: 감소하다 / 하락하다 / 줄어들다
- 最: 가장 높은/낮은 비율을 차지하다
- 对比: ~(으)ㄴ 반면 / ~에 비해

## 四、Q54 议论文 万能框架

### 서론 (80~100자) → 3~4句

```
최근 ~에 대한 관심이 높아지고 있다. (现象导入)
이에 대해 다양한 의견이 있지만, 나는 ~(이)라고 생각한다. (立场)
이에 대해 살펴보겠다. (预告)
```

### 본론1 (200~250자) → 논점1

```
~의 첫 번째 이유/원인은 ~기 때문이다. (主题句)
(부연 설명) (展开)
예를 들어, ~ (举例)
이처럼 ~ (小结)
```

### 본론2 (200~250자) → 논점2

```
뿐만 아니라/한편 ~ (过渡)
(새로운 논점 + 근거) (展开)
물론 ~다는 의견도 있다. 그러나 ~ (让步反驳, 可选)
```

### 결론 (80~100자) → 3~4句

```
이상으로 ~에 대해 살펴보았다. (总结)
~다는 점은 분명하다. (再强调)
따라서 ~해야 할 것이다. (呼吁/展望)
```

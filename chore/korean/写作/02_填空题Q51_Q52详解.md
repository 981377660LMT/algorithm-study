# 填空题 Q51-Q52 详解 (빈칸 채우기)

---

## 一、Q51 题型特征

| 项目 | 内容                                           |
| ---- | ---------------------------------------------- |
| 类型 | 实用文 (편지/공지/안내/초대/감사/사과/건의 등) |
| 分值 | 10分                                           |
| 要求 | 根据上下文填写 1~2 个句子                      |
| 体裁 | **격식체 (格式体)** ~(스)ㅂ니다                |

### Q51 常见场景

| 场景 | 韩语      | 开头/结尾套语            |
| ---- | --------- | ------------------------ |
| 邀请 | 초대      | ~에 초대하고자 합니다    |
| 感谢 | 감사      | ~에 감사드립니다         |
| 道歉 | 사과      | ~에 대해 사과드립니다    |
| 通知 | 공지/안내 | ~을/를 알려 드립니다     |
| 申请 | 신청/지원 | ~에 지원하고자 합니다    |
| 建议 | 건의/제안 | ~을/를 건의드립니다      |
| 祝贺 | 축하      | ~을/를 축하드립니다      |
| 请求 | 부탁/요청 | ~을/를 부탁드립니다      |
| 拒绝 | 거절      | ~하기 어려울 것 같습니다 |

---

## 二、Q51 解题步骤

```
Step 1: 确认文体 → 편지? 공지? 안내?
Step 2: 读懂上下文 → 빈칸 전후 내용 파악
Step 3: 判断빈칸角色 → 原因? 目的? 请求? 致谢?
Step 4: 用격식체写出 → ~(스)ㅂ니다 体
```

---

## 三、Q51 高频场景范例 (8个)

### 场景1：邀请 (초대)

```
안녕하십니까? ○○대학교 한국어학과입니다.
다음 주 금요일에 '한국 문화 체험의 날' 행사가 열립니다.
(                                          )
많은 참여 부탁드립니다.
```

**填写**: 관심 있는 분들은 누구나 참가하실 수 있으니 많이 오시기 바랍니다.

---

### 场景2：感谢 (감사)

```
안녕하십니까? ○○ 봉사 센터입니다.
지난주 진행된 봉사 활동에 참여해 주셔서 감사합니다.
(                                          )
앞으로도 많은 관심과 참여를 부탁드립니다.
```

**填写**: 여러분의 도움 덕분에 어려운 이웃들에게 따뜻한 겨울을 선물할 수 있었습니다.

---

### 场景3：通知 (안내)

```
안녕하십니까? 아파트 관리 사무소입니다.
오는 월요일부터 엘리베이터 교체 공사가 진행됩니다.
(                                          )
주민 여러분의 양해 부탁드립니다.
```

**填写**: 공사 기간 동안 엘리베이터를 사용하실 수 없으니 계단을 이용해 주시기 바랍니다.

---

### 场景4：申请 (지원)

```
안녕하십니까? 저는 한국어학과 3학년 김민수입니다.
(                                          )
관련 경험이 있으며 성실하게 활동하겠습니다.
지원서를 첨부했으니 검토 부탁드립니다.
```

**填写**: 이번에 모집하시는 한국어 교육 봉사 활동에 지원하고자 이렇게 글을 씁니다.

---

### 场景5：建议 (건의)

```
안녕하십니까? 저는 ○○도서관을 자주 이용하는 시민입니다.
요즘 도서관 내에서 소음이 심해 공부하기 어렵습니다.
(                                          )
검토해 주시면 감사하겠습니다.
```

**填写**: 방음이 되는 통화 부스를 설치하거나 소음 관리 규정을 강화해 주실 것을 건의드립니다.

---

### 场景6：道歉 (사과)

```
안녕하십니까? ○○쇼핑몰 고객 센터입니다.
고객님께서 주문하신 상품의 배송이 지연되어
(                                          )
빠른 시일 내에 배송될 수 있도록 하겠습니다.
```

**填写**: 불편을 드려 대단히 죄송합니다.

---

### 场景7：祝贺 (축하)

```
사랑하는 영수 씨에게.
이번 승진 소식을 듣고 정말 기뻤습니다.
(                                          )
앞으로도 좋은 일만 가득하길 바랍니다.
```

**填写**: 그동안 열심히 노력한 결과라고 생각하며 진심으로 축하드립니다.

---

### 场景8：拒绝 (거절)

```
안녕하십니까? 김민수입니다.
이번 주말 모임에 초대해 주셔서 감사합니다.
(                                          )
다음에 꼭 참석하도록 하겠습니다.
```

**填写**: 그런데 그날 선약이 있어서 참석하기 어려울 것 같습니다.

## 四、Q52 题型特征

| 项目 | 内容                             |
| ---- | -------------------------------- |
| 类型 | 说明文/论说文 빈칸               |
| 分值 | 10分                             |
| 要求 | 根据上下文补写2个空              |
| 体裁 | **~(으)ㄴ/는다 体** (書面格式体) |

### Q52 형식 (典型格式)

```
              [主题]
... (              ) ...문장1...
... 문장2 ... (              ) ...문장3...
```

---

## 五、Q52 解题步骤

```
Step 1: 读标题 → 确认主题
Step 2: 通读全文 → 理解整体论点
Step 3: 빈칸①逻辑 → 与后文什么关系?(因果/转折/递进)
Step 4: 빈칸②逻辑 → 与前文什么关系?
Step 5: 用 ~(으)ㄴ/는다 体 写出
```

---

## 六、Q52 题型练习 (4题)

### 练习1

```
            행복의 조건

사람들은 돈이 많으면 행복할 것이라고 생각한다.
그러나 연구에 따르면 (                               ).
일정 수준 이상의 소득이 되면 돈이 더 늘어도 행복감은 크게 변하지 않는다.
오히려 (                               ).
가족과의 대화, 취미 활동, 건강 등이 행복에 더 큰 영향을 미친다.
```

**빈칸①**: 소득과 행복은 반드시 비례하지 않는다
**빈칸②**: 물질적 조건보다 정서적 만족감이 행복에 더 중요하다

---

### 练习2

```
            독서의 중요성

독서는 지식을 넓히는 가장 좋은 방법 중 하나이다.
(                               ) 다양한 분야의 책을 읽으면
세상을 보는 시야가 넓어지고 사고력도 향상된다.
또한 독서는 (                               ).
스트레스를 해소하고 마음의 안정을 찾는 데 도움이 된다.
```

**빈칸①**: 특히 독서는 단순한 정보 습득을 넘어 사고의 깊이를 더해 준다.
**빈칸②**: 정서적 안정에도 효과가 있다

---

### 练习3

```
            계절과 음식

한국에서는 계절마다 즐겨 먹는 음식이 다르다.
(                               ) 여름에는 시원한 냉면이나
콩국수를 즐기고, 겨울에는 따뜻한 국물 요리를 먹는다.
이처럼 (                               ).
이를 통해 건강을 유지하는 지혜가 담겨 있다.
```

**빈칸①**: 봄에는 나물, 가을에는 햇곡식을 이용한 음식을 먹는다.
**빈칸②**: 한국의 음식 문화는 계절의 변화와 밀접한 관련이 있다

---

### 练习4

```
            여행의 가치

여행은 새로운 장소를 방문하는 것 이상의 의미가 있다.
(                               ) 낯선 환경에서
다양한 사람들을 만나고 문화를 경험하면서 시야가 넓어진다.
또한 일상에서 벗어나 (                               ).
이러한 이유로 여행은 현대인에게 꼭 필요한 활동이라고 할 수 있다.
```

**빈칸①**: 여행을 통해 자신을 되돌아보고 새로운 가치를 발견할 수 있다.
**빈칸②**: 스트레스를 해소하고 재충전할 수 있는 기회가 된다

## 七、Q51-Q52 必记套语

### Q51 信件/通知 万能表达

| 用途     | 表达                          |
| -------- | ----------------------------- |
| 开头问候 | 안녕하십니까? / ~입니다.      |
| 说明目的 | ~을/를 알려 드리고자 합니다   |
| 请求参与 | 많은 관심과 참여 부탁드립니다 |
| 请求理解 | 양해 부탁드립니다             |
| 表示感谢 | ~에 감사드립니다              |
| 表示歉意 | 불편을 드려 죄송합니다        |
| 请求检查 | 검토해 주시면 감사하겠습니다  |
| 结尾期望 | ~하시기 바랍니다              |

### Q52 说明文 万能连接

| 位置         | 常用连接                             |
| ------------ | ------------------------------------ |
| 빈칸引出原因 | 왜냐하면 ~기 때문이다                |
| 빈칸表转折   | 그러나/하지만 ~(으)ㄴ/는 것은 아니다 |
| 빈칸表递进   | 뿐만 아니라 ~도 ~(으)ㄴ/는다         |
| 빈칸表总结   | 이처럼/이렇듯 ~(으)ㄴ/는 것이다      |

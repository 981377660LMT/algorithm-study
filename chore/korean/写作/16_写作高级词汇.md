# 写作高级词汇 (쓰기 고급 어휘)

> 初级词 → 高级词 替换指南。把"小学生作文"升级为"6级水平"。

---

## 一、动词替换

| --------------- | ------------------------------ | ----------------------------------- |
| 하다 (做)       | 수행하다 / 시행하다            | 정부가 정책을 **시행해야** 한다     |
| 주다 (给)       | 제공하다 / 부여하다            | 교육 기회를 **제공해야** 한다       |

| 알다 (知道)     | 인식하다 / 파악하다            | 문제의 심각성을 **인식해야** 한다   |
| 바꾸다 (换)     | 변화시키다 / 전환하다          | 사회 인식을 **전환해야** 한다       |
| 만들다 (做)     | 조성하다 / 형성하다 / 구축하다 | 좋은 환경을 **조성해야** 한다       |
| 나누다 (分)     | 구분하다 / 분류하다            | 두 가지로 **구분할** 수 있다        |
| 줄이다 (减)     | 감소시키다 / 절감하다          | 비용을 **절감해야** 한다            |
| 늘리다 (增)     | 확대하다 / 증가시키다          | 지원을 **확대해야** 한다            |
| 없애다 (消除)   | 제거하다 / 철폐하다            | 차별을 **철폐해야** 한다            |
| 넣다 (放)       | 도입하다 / 적용하다            | 새로운 제도를 **도입해야** 한다     |
| 찾다 (找)       | 모색하다 / 탐색하다            | 해결 방안을 **모색해야** 한다       |

| 생기다 (产生)   | 발생하다 / 초래하다            | 심각한 문제가 **발생하고** 있다     |

## 五、改写练习

将下面的"初级版"改成"高级版"：

**初级版** ❌:
요즘 환경 문제가 아주 나빠지고 있다. 사람들이 쓰레기를 많이 만들기 때문이다. 이 문제를 없애려면 좋은 방법을 찾아야 한다. 나라에서 돈을 많이 써서 환경을 지켜야 한다.

**高级版** ⭕:
최근 환경 문제가 매우 심각해지고 있다. 대량 소비로 인해 폐기물이 급증하고 있기 때문이다. 이 문제를 해결하기 위해서는 효과적인 대책을 모색해야 한다. 국가 차원에서 충분한 재원을 확보하여 환경을 보존해야 한다.

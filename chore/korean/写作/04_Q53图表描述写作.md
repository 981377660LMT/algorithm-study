# Q53 图表描述写作 (자료 설명하는 글쓰기)

---

## 一、Q53 题型概述

| 项目 | 内容                         |
| ---- | ---------------------------- |
| 分值 | 30分                         |
| 字数 | 200~300字                    |
| 时间 | 13分钟                       |
| 体裁 | ~(으)ㄴ/는다 体 (書面体)     |
| 内容 | 描述图表数据 + 分析原因/影响 |

---

## 二、常见图表类型

| 类型   | 韩语          | 描述重点         |
| ------ | ------------- | ---------------- |
| 柱状图 | 막대그래프    | 比较各项数值大小 |
| 折线图 | 꺾은선 그래프 | 描述变化趋势     |
| 饼图   | 원형 그래프   | 描述比例构成     |
| 表格   | 표            | 数据对比分析     |
| 双图   | 두 개의 자료  | 分别描述+找关联  |

---

## 三、Q53 万能结构 (四段式)

```
【第1句】 주제 도입 — 介绍调查/图表主题
【第2~3句】 자료 1 설명 — 描述第一个图表的主要数据
【第4~5句】 자료 2 설명 — 描述第二个图表的主要数据
【第6~7句】 원인/영향/전망 — 分析原因或展望
```

---

## 四、各部分万能模板

### 第1句：主题导入

| 模板                                            | 翻译                      |
| ----------------------------------------------- | ------------------------- |
| ~에 대해 조사한 결과가 다음과 같다.             | 关于…的调查结果如下。     |
| ~에 대한 조사를 실시한 결과는 다음과 같다.      | 实施关于…的调查结果如下。 |
| 아래의 자료는 ~에 대해 조사한 것이다.           | 以下资料是关于…的调查。   |
| ~에 대한 설문 조사 결과를 살펴보면 다음과 같다. | 关于…的问卷调查结果如下。 |

### 第2~5句：数据描述

#### 趋势描述

| 模板                                            | 翻译                      |
| ----------------------------------------------- | ------------------------- |
| ~(이)가 ~년에 ~%였으나 ~년에는 ~%로 증가하였다. | …从~年的~%增加到~年的~%。 |
| ~(이)가 ~에서 ~(으)로 감소하였다.               | …从~减少到~。             |
| ~(이)가 꾸준히 증가/감소하는 추세이다.          | …呈持续增长/减少趋势。    |
| ~(이)가 ~년을 기점으로 급격히 변화하였다.       | …以~年为节点急剧变化。    |

#### 比较描述

| 模板                                            | 翻译              |
| ----------------------------------------------- | ----------------- |
| ~(이)가 가장 높은/낮은 것으로 나타났다.         | 显示…最高/低。    |
| ~(이)가 ~%로 1위를 차지하였다.                  | …以~%排名第一。   |
| ~에 비해 ~(이)가 ~배 높은/낮은 것으로 나타났다. | 与~相比高/低~倍。 |
| 반면(에) ~(이)가 가장 낮았다.                   | 而…最低。         |

#### 比例描述

| 模板                                | 翻译             |
| ----------------------------------- | ---------------- |
| ~(이)가 전체의 ~%를 차지하였다.     | …占整体的~%。    |
| ~(이)가 과반수를 넘었다/차지하였다. | …超过/占了半数。 |
| ~(이)가 ~에 불과했다.               | …仅为~。         |

### 第6~7句：原因/影响/展望

| 模板                                           | 翻译                    |
| ---------------------------------------------- | ----------------------- |
| 이러한 결과는 ~(으)ㄴ/는 것으로 분석된다.      | 这种结果被分析为…       |
| 이는 ~기 때문인 것으로 보인다.                 | 这似乎是因为…           |
| 이러한 추세는 앞으로도 계속될 것으로 전망된다. | 这种趋势预计将持续。    |
| 따라서 ~에 대한 대책이 필요할 것으로 보인다.   | 因此看来需要对…的对策。 |
| 이에 따라 ~는 변화가 예상된다.                 | 随之预计会有…的变化。   |

## 五、Q53 实战示范 (3篇)

### 示范1：折线图+柱状图

**题目**: 아래는 '한국의 1인 가구 수 변화'와 '1인 가구의 주된 이유'에 대한 조사 결과입니다. 자료를 보고 글을 쓰십시오. (200~300자)

```
[그래프1: 1인 가구 수]
2010년: 400만  →  2015년: 500만  →  2020년: 600만  →  2023년: 700만

[그래프2: 1인 가구의 주된 이유]
취업/학업: 35%,  개인 선호: 30%,  이혼/사별: 20%,  기타: 15%
```

**范文**:

한국의 1인 가구 수 변화와 1인 가구의 주된 이유에 대한 조사 결과는 다음과 같다. 먼저 1인 가구 수의 변화를 살펴보면, 2010년 400만 가구에서 2023년 700만 가구로 꾸준히 증가하였다. 1인 가구의 주된 이유를 살펴보면, '취업/학업'이 35%로 가장 높았으며, '개인 선호'가 30%로 그 뒤를 이었다. 반면 '이혼/사별'은 20%, '기타'는 15%를 차지하였다. 이러한 결과는 취업과 학업을 위해 독립하는 사람이 많고, 혼자 사는 것을 선호하는 문화가 확산되었기 때문인 것으로 보인다. 이에 따라 1인 가구를 위한 주거 및 복지 정책이 필요할 것으로 전망된다. (약 270자)

### 示范2：饼图+饼图

**题目**: 아래는 '대학생의 취미 활동'에 관한 조사 결과입니다. (200~300자)

```
[2018년]  운동 40%, 독서 25%, 게임 20%, 음악 감상 15%
[2023년]  게임 35%, 운동 30%, 음악 감상 20%, 독서 15%
```

**范文**:

대학생의 취미 활동에 대한 조사 결과는 다음과 같다. 2018년에는 '운동'이 40%로 가장 높았으며, '독서'가 25%, '게임'이 20%, '음악 감상'이 15%를 차지하였다. 반면 2023년에는 '게임'이 35%로 1위를 차지하였고, '운동'은 30%로 2위로 떨어졌다. '음악 감상'은 20%로 소폭 증가한 반면, '독서'는 15%로 크게 줄었다. 이러한 변화는 스마트폰과 온라인 게임의 보급이 확대되면서 디지털 기반의 취미가 늘어났기 때문인 것으로 분석된다. 앞으로도 디지털 매체를 활용한 취미 활동이 더욱 증가할 것으로 전망된다. (약 270자)

### 示范3：表格

**题目**: 아래는 '직장인의 스트레스 원인'과 '스트레스 해소 방법'에 대한 조사 결과입니다. (200~300자)

```
[스트레스 원인]     업무 과다 45%, 인간관계 25%, 경제적 문제 20%, 기타 10%
[해소 방법]         운동 30%, 수면 25%, 취미 활동 20%, 음주 15%, 기타 10%
```

**范文**:

직장인의 스트레스 원인과 해소 방법에 대한 조사 결과는 다음과 같다. 스트레스 원인으로는 '업무 과다'가 45%로 가장 높았으며, '인간관계'가 25%, '경제적 문제'가 20%로 뒤를 이었다. 스트레스 해소 방법으로는 '운동'이 30%로 1위를 차지하였고, '수면'이 25%, '취미 활동'이 20% 순이었다. '음주'는 15%에 그쳤다. 이를 통해 직장인들은 주로 업무 부담으로 스트레스를 받으며, 건강한 방법으로 해소하려는 노력을 하고 있음을 알 수 있다. 기업에서는 근무 환경을 개선하고 업무량을 적절히 조절하는 방안이 필요할 것으로 보인다. (약 280자)

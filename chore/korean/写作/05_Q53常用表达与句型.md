---

## 一、调查/主题导入句

| #   | 句型                                                     | 翻译                              |
| --- | -------------------------------------------------------- | --------------------------------- |
| 1   | ~에 대해 조사한 결과는 다음과 같다.                      | 关于…的调查结果如下。             |
| 2   | ~에 대한 설문 조사를 실시한 결과를 살펴보면 다음과 같다. | 实施关于…的问卷调查结果如下。     |
| 3   | 아래의 자료는 ~을/를 보여 주고 있다.                     | 下面的资料显示…                   |
| 4   | ~에 관한 통계 자료를 분석한 결과이다.                    | 对关于…的统计资料进行分析的结果。 |

## 二、过渡·引导句

| 用途     | 句型                          | 翻译            |
| -------- | ----------------------------- | --------------- |
| 引出图1  | 먼저 ~을/를 살펴보면          | 首先看…         |
| 引出图2  | 다음으로 ~을/를 살펴보면      | 接下来看…       |
| 对比引出 | 반면(에)                      | 另一方面/而     |
| 补充     | 또한/게다가                   | 而且/此外       |
| 引出原因 | 이는 ~기 때문인 것으로 보인다 | 这似乎是因为…   |
| 引出展望 | 이러한 추세는 앞으로도 ~      | 这种趋势今后也… |

| #   | 句型                                            | 翻译                          |
| --- | ----------------------------------------------- | ----------------------------- |
| 1   | 이러한 결과는 ~기 때문인 것으로 분석된다        | 这种结果被分析为因为…         |
| 2   | 이는 ~(으)ㄴ/는 것과 관련이 있다                | 这与…有关                     |
| 3   | ~(이)가 ~(으)ㄴ/는 것이 원인으로 보인다         | …似乎是原因                   |
| 4   | 이는 ~에 대한 인식이 변화했기 때문으로 풀이된다 | 这被解释为对…的认识发生了变化 |
| 5   | ~의 영향으로 ~(이)가 변화한 것으로 보인다       | 受…的影响似乎发生了变化       |

## 六、Q53 常用연결 표현 组合

### 模式A：增长型

```
~에 대한 조사 결과는 다음과 같다.
먼저 ~을 살펴보면, ~(이)가 ~에서 ~(으)로 꾸준히 증가하였다.
다음으로 ~을 보면, ~(이)가 ~%로 가장 높았으며, ~(이)가 ~% 순이었다.
이는 ~기 때문인 것으로 보인다.
이러한 추세는 앞으로도 계속될 것으로 전망된다.
```

### 模式B：比较型

```
~에 대한 조사 결과는 다음과 같다.
~(이)가 ~%로 가장 높았으며, 그 뒤를 이어 ~(이)가 ~%를 차지하였다.
반면 ~(이)가 ~%로 가장 낮은 것으로 나타났다.
이러한 결과는 ~(으)ㄴ/는 것과 관련이 있는 것으로 분석된다.
따라서 ~에 대한 관심이 필요할 것으로 보인다.
```

### 模式C：变化对比型

```
~에 대한 조사 결과는 다음과 같다.
~년에는 ~(이)가 ~%였으나 ~년에는 ~%로 크게 증가하였다.
반면 ~(이)가 ~%에서 ~%로 감소하였다.
이는 ~(으)ㄴ/는 변화가 영향을 미친 것으로 보인다.
앞으로 ~에 대한 대책이 필요할 것이다.
```

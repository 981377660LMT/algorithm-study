# 标点与原稿纸格式 (문장 부호와 원고지 작성법)

---

## 一、TOPIK 쓰기 答题纸 (원고지) 基本规则

### 1. 격자 기본 규칙

- 한 칸에 한 글자 (包括标点)
- 제목 없이 바로 첫 줄부터 쓰기 (Q54)
- 첫 문단은 첫 칸 비우기 (들여쓰기 1칸)
- 새 문단 시작할 때도 첫 칸 비우기

### 2. 들여쓰기 (缩进)

```
[올바른 예]
□최근□저출산□문제가□심각한□사회□
문제로□대두되고□있다.□출산율의□급
격한□감소는...
□□저출산의□가장□큰□원인은□경제적
         ↑ 새 문단: 첫 칸 비움
```

> Q51, Q52: 빈칸 채우기이므로 원고지 신경 쓸 필요 없음
> Q53: 200~300자, 들여쓰기 하는 것이 좋음
> Q54: 600~700자, **반드시** 문단 구분 + 들여쓰기

## 二、标点符号 用法

### 마침표 (句号) .

| 规则 | 说明             |
| ---- | ---------------- |
| 위치 | 한 칸 차지       |
| 用法 | 서술문 끝에 사용 |

```
|최|근|환|경|문|제|가|심|각|하|다|.|
```

### 쉼표 (逗号) ,

| 规则 | 说明                              |
| ---- | --------------------------------- |
| 위치 | 앞 글자와 같은 칸 또는 한 칸 차지 |
| 用法 | 나열, 연결, 삽입                  |

```
|첫|째|,| |환|경|을|...
또는
|첫|째|,|환|경|을|...
```

### 물음표 (问号) ?

| 规则 | 说明                        |
| ---- | --------------------------- |
| 위치 | 한 칸 차지, 뒤에 한 칸 띄기 |
| 주의 | Q54에서 거의 사용하지 않음  |

### 느낌표 (感叹号) !

| 规则 | 说明                      |
| ---- | ------------------------- |
| 주의 | Q54에서 **사용하지 않음** |

### 따옴표 (引号)

| 종류         | 기호 | 用法            |
| ------------ | ---- | --------------- |
| 큰따옴표 ' ' | " "  | 직접 인용, 속담 |
| 작은따옴표   | ' '  | 강조, 특수 용어 |

```
|'|워|라|밸|'|이|란|일|과|삶|의|...
```

### 괄호 (括号) ( )

```
|인|공|지|능|(|A|I|)|의|발|달|로|...
```

## 三、쓰기 답안지 작성 팁

### 행의 마지막 칸 처리

| 상황                        | 처리법                           |
| --------------------------- | -------------------------------- |
| 마지막 칸에 문장 끝         | 마침표를 마지막 글자와 같은 칸에 |
| 마지막 칸이 빈칸            | 다음 줄 첫 칸부터 이어서         |
| 마침표가 첫 칸에 오게 될 때 | 앞 줄 마지막 칸에 글자와 함께    |

```
[마침표 처리]
|사|회|문|제|로|대|두|되|고|있|다.|
                              ↑ 마침표는 마지막 글자와 같은 칸 OK

[줄 끝 처리 - 올바른 예]
|사|회|문|제|로|대|두|되|고|있|다|.|   ← OK: 마침표 마지막 칸
|사|회|문|제|로|대|두|되|고|있|다.|   ← OK: 글자+마침표 같은 칸
```

## 四、字数计算法

### TOPIK 원고지 字数 ≠ 띄어쓰기 제외

| 计入字数的        | 不计入字数的 |
| ----------------- | ------------ |
| 한글              | -            |
| 숫자              | -            |
| 영문자            | -            |
| 标点符号          | -            |
| 띄어쓰기(空格) ✅ | -            |

> **TOPIK에서는 빈칸(공백)도 글자 수에 포함됩니다!**

### Q53 字数管理 (200~300자)

- 目标: 250~270자
- 4개 단락 (도입/그래프1/그래프2/원인·전망)
- 每段约 60~70자

### Q54 字数管理 (600~700자)

- 目标: 630~670자
- 4개 단락 기준:
  - 서론: 80~100자 (약 4~5행)
  - 본론1: 200~250자 (약 10~12행)
  - 본론2: 200~250자 (약 10~12행)
  - 결론: 80~100자 (약 4~5행)

### 快速估算法

- 원고지 한 줄 = 약 20자 (한칸 한글자 기준)
- Q53: 약 12~15줄
- Q54: 약 30~35줄

## 五、띄어쓰기 (隔写法) 常见错误

### 必须隔写的

| 错误 ❌      | 正确 ⭕        | 规则               |
| ------------ | -------------- | ------------------ |
| 할수있다     | 할 수 있다     | 의존 명사 띄어쓰기 |
| 할수밖에없다 | 할 수밖에 없다 | 의존 명사 띄어쓰기 |
| 할것이다     | 할 것이다      | 의존 명사 띄어쓰기 |
| 할때         | 할 때          | 의존 명사 띄어쓰기 |
| 할뿐이다     | 할 뿐이다      | 의존 명사 띄어쓰기 |
| 하는것       | 하는 것        | 의존 명사 띄어쓰기 |
| 한바있다     | 한 바 있다     | 의존 명사 띄어쓰기 |
| 해야할것이다 | 해야 할 것이다 | 의존 명사 2개      |

### 不隔写的 (连写)

| 错误 ❌      | 正确 ⭕     | 规则      |
| ------------ | ----------- | --------- |
| 그러 므로    | 그러므로    | 연결 부사 |
| 뿐 만 아니라 | 뿐만 아니라 | 보조사    |
| 결론 적으로  | 결론적으로  | 접미사    |

## 六、원고지 작성 실전 연습

아래 문장을 원고지에 옮겨 쓰는 연습을 하세요:

**연습1** (서론):

```
최근 환경 오염에 대한 관심이 높아지고 있다. 무분별한 개발로 인해 자연환경이 빠르게 파괴되고 있으며, 이에 대한 대책이 시급하다.
```

**원고지:**

```
| |최|근| |환|경| |오|염|에| |대|한| |관|심|이| |높|아|
|지|고| |있|다|.| |무|분|별|한| |개|발|로| |인|해| |자|
|연|환|경|이| |빠|르|게| |파|괴|되|고| |있|으|며|,| |이|
|에| |대|한| |대|책|이| |시|급|하|다|.|
```

→ 약 73자 (공백 포함)

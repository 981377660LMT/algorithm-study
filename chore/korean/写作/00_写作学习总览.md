## 五、每日学习法 (1小时)

| 时间    | 内容     | 方法                         |
| ------- | -------- | ---------------------------- |
| 0~15分  | 句型背诵 | 每天背5个写作句型，抄写+造句 |
| 15~35分 | 范文精读 | 读1篇范文，分析结构+标注好句 |
| 35~50分 | 限时练习 | 每天写1题(Q51/52/53轮流)     |
| 50~60分 | 对照修改 | 对照范文自我批改，记录错误   |

## 六、写作提分核心原则

| #   | 原则               | 说明                                |
| --- | ------------------ | ----------------------------------- |
| 1   | **格式体(격식체)** | 写作必须用 ~(스)ㅂ니다/~(으)ㄴ다 体 |
| 2   | **不用口语**       | 禁止 ~거든요, ~잖아요, ~는데요      |
| 3   | **连接词丰富**     | 따라서/그러나/뿐만 아니라/이처럼    |
| 4   | **书面词汇**       | 用 실시하다(非 하다), 견해(非 생각) |
| 5   | **字数达标**       | Q53: 200~300字, Q54: 600~700字      |
| 6   | **结构清晰**       | 서론→본론→결론 三段必须分明         |
| 7   | **论点明确**       | 开头就亮观点，不要绕弯子            |

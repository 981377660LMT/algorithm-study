## 十一、连接词高频使用模板

### 서론 常用

```
최근 ~에 대한 관심이 높아지고 있다.
이에 대해 다양한 의견이 있지만,
나는 ~(이)라고 생각한다.
이에 대해 자세히 살펴보겠다.
```

### 본론1 → 본론2 过渡

```
[본론1 결미]
이처럼 ~다는 것을 알 수 있다.

[과도]
뿐만 아니라 / 한편 / 더 나아가

[본론2 시작]
~도 중요한 문제이다 / ~도 간과할 수 없다.
```

### 결론 常用

```
이상으로 ~에 대해 살펴보았다.
앞서 살펴본 바와 같이 ~.
따라서 ~해야 할 것이다.
```

## 十二、连接词使用频率参考

一篇 600~700字的 Q54 中，建议使用连接表达 **8~12个**：

| 位置  | 建议数量 | 推荐                                |
| ----- | -------- | ----------------------------------- |
| 서론  | 2~3个    | 최근, 이에 대해, ~(이)라고 생각한다 |
| 본론1 | 2~3个    | 첫째/우선, 예를 들어, 또한          |
| 过渡  | 1个      | 뿐만 아니라 / 한편                  |
| 본론2 | 2~3个    | 둘째/또한, 실제로, 더 나아가        |
| 결론  | 2~3个    | 이상으로, 따라서, ~할 것이다        |

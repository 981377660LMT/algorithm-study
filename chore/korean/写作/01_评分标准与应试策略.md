## 一、TOPIK II 写作官方评分维度

### Q51~Q52 (各10分)

| 分数  | 标准                         |
| ----- | ---------------------------- |
| 10分  | 语法正确、内容完整、表达自然 |
| 7~9分 | 有小错误但意思可理解         |
| 4~6分 | 语法或词汇有明显错误         |
| 1~3分 | 内容偏离或严重错误           |
| 0分   | 空白或完全不相关             |

### Q53 (30分)

| 维度           | 分值 | 要求                       |
| -------------- | ---- | -------------------------- |
| 内容与课题执行 | 12分 | 是否完整描述了图表信息     |
| 文章展开结构   | 6分  | 结构是否清晰、逻辑连贯     |
| 语言使用       | 12分 | 语法准确、词汇丰富、格式体 |

### Q54 (50分)

| 维度           | 分值 | 要求                        |
| -------------- | ---- | --------------------------- |
| 内容与课题执行 | 16分 | 是否围绕主题、论点是否明确  |
| 文章展开结构   | 12分 | 서론-본론-결론 是否完整     |
| 语言使用       | 22分 | 语法·词汇·书面体·表达多样性 |

## 二、扣分项详解

### 必扣分项 (감점 요소)

| 错误类型   | 扣分程度  | 例                         |
| ---------- | --------- | -------------------------- |
| 口语体     | 严重      | ~거든요, ~잖아요, ~는데요  |
| 非格式体   | 严重      | ~해요 代替 ~합니다/~한다   |
| 字数不足   | 严重      | Q53不足150字, Q54不足500字 |
| 跑题       | 致命      | 完全偏离题目要求           |
| 无结构     | 较重      | 没有分段、没有논리 전개    |
| 助词错误   | 中等      | 을/를, 은/는, 이/가 混用   |
| 拼写错误   | 轻微~中等 | 받침 错误, 띄어쓰기        |
| 原稿纸格式 | 轻微      | 标点位置、缩进             |

### 加分项 (가산 요소)

| 要素       | 说明                                      |
| ---------- | ----------------------------------------- |
| 高级语法   | 使用 ~(으)ㄹ 뿐만 아니라, ~기 마련이다 等 |
| 书面词汇   | 실시하다, 견해, 기여하다, 초래하다 等     |
| 连接词多样 | 不只用 그래서, 要用 따라서/이에 따라 等   |
| 引用/例证  | 举例说明论点 (예를 들어~)                 |
| 让步转折   | 물론 ~지만 / ~(으)ㄴ 반면                 |

## 三、时间分配策略

### 50分钟分配

```
Q51 (3分钟) → Q52 (4分钟) → Q53 (13分钟) → Q54 (30分钟)
```

### Q54 内部时间分配 (30分钟)

| 阶段  | 时间  | 任务               |
| ----- | ----- | ------------------ |
| 构思  | 3分钟 | 确定论点、列大纲   |
| 서론  | 5分钟 | 写开头段 (3~4句)   |
| 본론1 | 7分钟 | 第一个论点+论据    |
| 본론2 | 7分钟 | 第二个论点+论据    |
| 결론  | 5分钟 | 总结+展望          |
| 检查  | 3分钟 | 格式体、拼写、字数 |

## 五、字数管理

### 原稿纸字数计算

- 每行 20 格
- Q53 要求 200~300字 → 约 10~15 行
- Q54 要求 600~700字 → 约 30~35 行

### 字数不够怎么办？

| 方法     | 例                             |
| -------- | ------------------------------ |
| 添加举例 | 예를 들어 ~는 경우가 있다      |
| 添加让步 | 물론 ~다는 의견도 있지만       |
| 扩展解释 | 즉, ~(이)라는 것이다           |
| 补充影响 | 이는 ~에 큰 영향을 미칠 것이다 |

### 字数太多怎么办？

| 方法     | 说明                   |
| -------- | ---------------------- |
| 删举例   | 保留论点，删次要例子   |
| 合并句子 | 两个短句合成一个复句   |
| 删重复   | 检查是否说了同样的意思 |

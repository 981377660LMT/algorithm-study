# Q54 议论文结构 (논술문 구조)

---

## 一、Q54 题型概述

| 项目 | 内容                         |
| ---- | ---------------------------- |
| 分值 | 50分 (最大题)                |
| 字数 | 600~700字                    |
| 时间 | 约30分钟                     |
| 体裁 | ~(으)ㄴ/는다 体 (書面体)     |
| 内容 | 围绕社会话题，阐述自己的观点 |

---

## 二、题目常见提问方式

| 韩语                                        | 翻译                      |
| ------------------------------------------- | ------------------------- |
| ~에 대해 자신의 생각을 쓰시오               | 请写出关于…的想法         |
| ~의 원인과 해결 방안에 대해 쓰시오          | 请写关于…的原因和解决方案 |
| ~의 장단점에 대해 쓰시오                    | 请写关于…的优缺点         |
| ~에 대한 자신의 견해를 쓰시오               | 请写出关于…的见解         |
| 아래의 내용을 중심으로 ~에 대한 글을 쓰시오 | 以下面的内容为中心写关于… |

### 题目中的提示 (하위 질문)

题目通常给出 2~3 个小问题，必须全部回答：

```
'인터넷 실명제'에 대해 아래의 내용을 중심으로 자신의 생각을 쓰시오.
- 인터넷 실명제란 무엇인가?
- 인터넷 실명제의 장점과 단점은 무엇인가?
- 인터넷 실명제에 대한 자신의 의견은 무엇인가?
```

## 三、三段式结构 (기본형)

```
【서론】 2~3句 (약 80~100자)
  → 화제 도입 + 자신의 입장 제시

【본론】 10~14句 (약 400~500자)
  → 논점1 + 근거/예시
  → 논점2 + 근거/예시
  → (선택) 반론과 재반박

【결론】 2~3句 (약 80~100자)
  → 요약 + 강조/전망
```

---

## 四、四段式结构 (고급형)

```
【서론】 화제 도입 + 입장   (80~100자)
【본론1】 첫 번째 논점      (150~200자)
【본론2】 두 번째 논점       (150~200자)
【결론】 요약 + 제안/전망    (80~100자)
```

## 五、各部分详细模板

### 서론 (开头段)

```
최근 ~에 대한 관심이 높아지고 있다.              ← 话题引入
~에 대해서는 다양한 의견이 있지만,                ← 话题展开
나는 ~(이)라고 생각한다.                          ← 立场表明
이에 대해 ~의 원인과 해결 방안을 살펴보겠다.      ← 预告结构
```

### 본론1 (第一论点)

```
먼저, ~는 ~(으)ㄴ/는 문제가 있다.                ← 论点提出
왜냐하면 ~기 때문이다.                           ← 原因说明
예를 들어 ~는 경우가 있다.                        ← 举例论证
이처럼 ~(이)가 ~에 영향을 미친다.                ← 小结
```

### 본론2 (第二论点)

```
또한/뿐만 아니라, ~도 중요한 문제이다.            ← 递进/补充
~에 따르면 ~는 것으로 나타났다.                   ← 数据/权威引用
이는 ~(으)ㄴ/는 것을 보여 준다.                   ← 分析
따라서 ~는 것이 필요하다.                         ← 解决方案
```

### 결론 (结尾段)

```
이상에서 살펴본 바와 같이,                        ← 概括语
~는 ~(으)ㄴ/는 문제이며 이를 해결하기 위해서는    ← 总结
~는 노력이 필요하다.                              ← 主张重申
~에 대한 사회적 관심과 제도적 지원이 함께         ← 展望
이루어져야 할 것이다.
```

## 六、5种展开模式

### 模式A：原因→解决 (원인-해결형)

适用题目："~의 원인과 해결 방안에 대해"

```
서론: 문제 소개 + 입장
본론1: 원인 분석 (2~3개)
본론2: 해결 방안 (2~3개)
결론: 요약 + 전망
```

### 模式B：赞反→立场 (찬반형)

适用题目："~에 대한 찬반 의견과 자신의 생각"

```
서론: 주제 소개 + 입장 예고
본론1: 찬성 의견 (또는 반대 의견) 소개
본론2: 반대 의견 (또는 찬성 의견) + 자신의 판단
결론: 최종 입장 강조
```

### 模式C：优缺点→评价 (장단점형)

适用题目："~의 장점과 단점"

```
서론: 주제 소개
본론1: 장점 (2~3개)
본론2: 단점 (2~3개)
결론: 종합 평가 + 방향 제시
```

### 模式D：定义→重要性 (정의-중요성형)

适用题目："~란 무엇이며, 왜 중요한가"

```
서론: 정의 + 입장
본론1: 중요한 이유 1
본론2: 중요한 이유 2
결론: 요약 + 실천 촉구
```

### 模式E：现状→问题→方案 (현상-방안형)

适用题目: 综合型

```
서론: 현상 (사회 변화) 소개
본론1: 문제점 분석
본론2: 해결 방안
결론: 전망
```

## 八、论据类型

| 类型 | 说明              | 例                             |
| ---- | ----------------- | ------------------------------ |
| 举例 | 구체적 사례       | 예를 들어 ~는 경우가 있다      |
| 数据 | 통계/연구 인용    | 연구에 따르면 ~것으로 나타났다 |
| 对比 | 反面论证          | ~와 달리 / ~(으)ㄴ 반면에      |
| 权威 | 전문가 의견       | 전문가들은 ~다고 지적한다      |
| 因果 | 원인-결과         | ~기 때문에 ~게 된다            |
| 让步 | 반론 수용 후 반박 | 물론 ~지만 ~                   |

# 书面体与格式体 (서면체와 격식체)

> TOPIK 写作最大扣分点之一：**文体(체)混用**。Q51用격식체，Q52-54用 ~(으)ㄴ/는다体。

---

## 一、TOPIK 写作中的四种文体

| 文体                  | 终结词尾                      | TOPIK用途    | 正式度 |
| --------------------- | ----------------------------- | ------------ | ------ |
| 격식체 (合니다体)     | ~(스)ㅂ니다 / ~(스)ㅂ니까     | Q51 실용문 ★ | 最高   |
| 비격식체 (~요体)      | ~아/어요                      | ❌ 写作禁用  | 中     |
| 해라体 (~(으)ㄴ/는다) | ~(으)ㄴ/는다 / ~(으)ㄹ 것이다 | Q52-54 ★     | 書面   |
| 해体 (~아/어)         | ~아/어                        | ❌ 写作禁用  | 最低   |

---

## 二、Q51 격식체 (~습니다/~ㅂ니다) 详解

### 陈述句

| 词类          | 规则            | 例                           |
| ------------- | --------------- | ---------------------------- |
| 动词(收音O)   | ~습니다         | 먹다 → 먹**습니다**          |
| 动词(收音X)   | ~ㅂ니다         | 가다 → 가**ㅂ니다** → 갑니다 |
| 形容词(收音O) | ~습니다         | 좋다 → 좋**습니다**          |
| 形容词(收音X) | ~ㅂ니다         | 크다 → 크**ㅂ니다** → 큽니다 |
| 名词+이다     | ~입니다         | 학생**입니다**               |
| 名词+아니다   | ~이/가 아닙니다 | 학생**이 아닙니다**          |
| 过去时        | ~았/었습니다    | 갔**습니다**                 |

### 疑问句

| 规则          | 例                    |
| ------------- | --------------------- |
| ~(스)ㅂ니까?  | 가십니까? / 좋습니까? |
| ~았/었습니까? | 다녀오셨습니까?       |

### Q51 中의 격식체 应用

```
[邀请函]
안녕하십니까? 한국어학과 학생회입니다.
다음과 같이 한국어 말하기 대회를 개최합니다.
많은 참여 부탁드립니다.

[感谢信]
보내 주신 선물 잘 받았습니다.
덕분에 즐거운 시간을 보냈습니다.
진심으로 감사드립니다.
```

## 三、Q52-54 해라体 (~(으)ㄴ/는다) 详解

### 陈述句

| 词类        | 规则                          | 例                                 |
| ----------- | ----------------------------- | ---------------------------------- |
| 动词(收音O) | ~는다                         | 먹다 → 먹**는다**                  |
| 动词(收音X) | ~ㄴ다                         | 가다 → 가**ㄴ다** → 간다           |
| 形容词      | ~다 (原形)                    | 좋다 → 좋**다** / 크다 → 크**다**  |
| 名词+이다   | ~(이)다                       | 중요한 문제**이다** / 행복**이다** |
| 名词+아니다 | ~이/가 아니다                 | 선택**이 아니다**                  |
| 过去时      | ~았/었다                      | 갔**다** / 좋았**다**              |
| 推测        | ~(으)ㄹ 것이다                | 증가할 **것이다**                  |
| 可能        | ~(으)ㄹ 수 있다               | 해결할 수 **있다**                 |
| 义务        | ~아/어야 한다                 | 노력해야 **한다**                  |
| 引用        | ~다고 한다 / ~다고 할 수 있다 | 필요하다고 **할 수 있다**          |

### 疑问句 (很少用于Q54正文)

| 规则          | 例        |
| ------------- | --------- |
| ~(으)ㄴ/는가? | 필요한가? |
| ~(으)ㄹ까?    | 가능할까? |

### Q52 中의 해라체 应用

```
사람들은 여행을 통해 새로운 경험을 한다.
낯선 곳에서의 경험은 일상에서 벗어나
새로운 시각으로 세상을 보게 해 준다.
이러한 이유로 많은 사람들이 여행을 즐긴다.
```

### Q54 中의 해라체 应用

```
최근 저출산 문제가 심각한 사회 문제로 대두되고 있다.
출산율의 급격한 감소는 인구 구조의 변화를 초래하고 있으며,
이에 대한 원인 분석과 해결 방안이 시급하다.
```

## 四、最容易混用的 10 种错误

| ❌ 错误 (混用)              | ⭕ 正确                     | 说明         |
| --------------------------- | --------------------------- | ------------ |
| 문제가 심각합니다 (Q54에서) | 문제가 심각하**다**         | Q54는 해라체 |
| 살펴보겠습니다 (Q54에서)    | 살펴보**겠다**              | Q54는 해라체 |
| 생각해요 (Q54에서)          | 생각한**다**                | ~요体 금지   |
| 좋은 거 같아요              | 바람직하**다**              | 口语 금지    |
| 중요하다 (Q51에서)          | 중요합니**다**              | Q51는 격식체 |
| 보내 주셔서 감사하다 (Q51)  | 감사**드립니다**            | Q51는 격식체 |
| 해야 됩니다 → Q54           | 해야 한**다**               | Q54는 해라체 |
| 있는 것 같습니다 → Q54      | 있**다**                    | Q54는 해라체 |
| 중요한 문제이다 → Q51       | 중요한 문제**입니다**       | Q51는 격식체 |
| 함께 노력하자 → Q54         | 함께 노력**해야 할 것이다** | 청유형 금지  |

## 五、주요 활용형 变换表

### "중요하다" (形容词) 的文体变化

| 文体   | 변환                             |
| ------ | -------------------------------- |
| 격식체 | 중요합니다 / 중요하지 않습니다   |
| 해라체 | 중요하다 / 중요하지 않다         |
| 연결형 | 중요하며 / 중요하고 / 중요하지만 |
| 관형형 | 중요한 (문제)                    |

### "필요하다" (形容词) 的文体变化

| 文体     | 변환                |
| -------- | ------------------- |
| 격식체   | 필요합니다          |
| 해라체   | 필요하다            |
| 연결형   | 필요하며 / 필요하고 |
| ~기 때문 | 필요하기 때문이다   |

### "증가하다" (动词) 的文体变化

| 文体        | 변환                            |
| ----------- | ------------------------------- |
| 격식체      | 증가합니다 / 증가했습니다       |
| 해라체 현재 | 증가한다 / 증가하고 있다        |
| 해라체 과거 | 증가했다 / 증가하였다           |
| 해라체 미래 | 증가할 것이다                   |
| 관형형      | 증가하는 (추세) / 증가한 (수치) |

## 六、文体自检练习

请将以下混用文体的文段改为统一的 해라体：

**❌ 混用版**:
최근 환경 오염이 심각합니다. 많은 사람들이 환경에 관심이 없어요. 이 문제를 해결해야 됩니다. 정부가 노력해야 하지 않을까요? 우리 모두 함께 노력합시다.

**⭕ 统一版** (해라체):
최근 환경 오염이 심각하**다**. 많은 사람들이 환경에 관심이 없**다**. 이 문제를 해결**해야 한다**. 정부가 노력**해야 할 것이다**. 우리 모두 함께 노력**해야 한다**.

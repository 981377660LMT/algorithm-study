# 新闻标题与报道类 (신문 기사 제목/보도)

---

## 一、新闻标题特征

韩语新闻标题有独特的缩写/省略规则：

### 1. 省略助词

| 完整句                                          | 标题                      |
| ----------------------------------------------- | ------------------------- |
| 한국 경제**가** 성장률**이** 하락**했다**       | 한국 경제 성장률 하락     |
| 서울시**에서** 무료 와이파이**를** 확대**한다** | 서울시 무료 와이파이 확대 |

### 2. 名词化终结

| 完整句        | 标题用词    |
| ------------- | ----------- |
| 증가하다/늘다 | 증가 / 급증 |
| 감소하다/줄다 | 감소 / 급감 |
| 실시하다      | 실시        |
| 발표하다      | 발표        |
| 결정하다      | 결정        |
| 논란이 되다   | 논란        |
| 개최하다      | 개최        |
| 추진하다      | 추진        |
| 도입하다      | 도입        |
| 시행하다      | 시행        |
| 합의하다      | 합의        |
| 체결하다      | 체결        |

### 3. 标题标点

| 符号       | 含义      | 例                     |
| ---------- | --------- | ---------------------- |
| … (省略号) | 后续未知  | 경제 전망 불투명…      |
| " " (引号) | 直接引语  | 대통령 "경제 살리겠다" |
| ~ (波浪号) | 范围/大约 | 내년~2028년            |
| ↑ / ↓      | 增/减     | 물가 ↑, 실업률 ↓       |

## 三、新闻报道常见句型

### 开头

| 句型                | 翻译         |
| ------------------- | ------------ |
| ~것으로 나타났다    | (调查)显示   |
| ~것으로 밝혀졌다    | (查明)原来是 |
| ~것으로 알려졌다    | 据悉/据了解  |
| ~것으로 보인다      | 看来/似乎    |
| ~에 따르면          | 根据…        |
| ~에 의하면          | 根据…        |
| ~라고 밝혔다/말했다 | 表示/说      |
| ~라고 전했다        | 转达说       |

### 结尾

| 句型                   | 翻译     |
| ---------------------- | -------- |
| ~할 방침이다           | 计划…    |
| ~할 예정이다           | 预定…    |
| ~할 계획이다           | 打算…    |
| ~에 나설 것으로 보인다 | 看来将…  |
| ~이/가 주목된다        | 受到关注 |
| ~이/가 기대된다        | 令人期待 |
| ~이/가 우려된다        | 令人担忧 |

## 四、题型：选标题 实战 (5题)

### 练习1

정부는 내년부터 중소기업에 대한 세금을 10% 인하하기로 했다. 이번 감세 조치는 경기 침체로 어려움을 겪고 있는 중소기업을 지원하기 위한 것이다. 정부 관계자는 "이번 조치로 약 50만 개의 중소기업이 혜택을 받을 것"이라고 말했다.

**이 글의 제목으로 알맞은 것은?**
① 대기업 세금 인상 추진
② 중소기업 세금 10% 인하 결정
③ 경기 침체 장기화 전망
④ 중소기업 수 50만 돌파

**答案**: ②

### 练习2

최근 조사에 따르면 20대의 1인 가구 비율이 10년 전에 비해 두 배 이상 증가한 것으로 나타났다. 취업난과 높은 집값이 주요 원인으로 분석된다. 전문가들은 1인 가구를 위한 주거 지원 정책이 필요하다고 지적했다.

**이 글의 제목으로 알맞은 것은?**
① 20대 결혼률 소폭 증가
② 집값 안정세 지속
③ 20대 1인 가구 10년 새 두 배 급증
④ 주거 지원 정책 효과 미미

**答案**: ③

### 练习3

서울시는 오는 9월부터 버스 노선을 대폭 개편한다. 이번 개편으로 노선 수가 현재 350개에서 400개로 늘어나며, 특히 출퇴근 시간대에 배차 간격이 줄어들 예정이다. 시민 불편을 최소화하기 위해 한 달간 시범 운행을 거칠 계획이다.

**이 글의 제목으로 알맞은 것은?**
① 서울 지하철 요금 인상
② 서울시 9월부터 버스 노선 대폭 개편
③ 버스 파업으로 시민 불편 가중
④ 서울시 택시 요금 변경

**答案**: ②

### 练习4

한국 영화 '빛의 도시'가 이번 칸 영화제에서 대상을 수상했다. 이 영화는 한국 사회의 빈부 격차를 다룬 작품으로, 해외 평론가들로부터 높은 평가를 받았다. 감독은 "한국 사회의 현실을 세계에 알릴 수 있어 기쁘다"고 소감을 밝혔다.

**이 글의 제목으로 알맞은 것은?**
① 칸 영화제 올해 개최 무산
② 한국 영화 '빛의 도시' 칸 대상 수상
③ 한국 영화 산업 침체 위기
④ 칸 영화제 한국 작품 출품 금지

**答案**: ②

### 练习5

연구팀이 해양 미세플라스틱의 양이 최근 5년간 3배 증가했다는 연구 결과를 발표했다. 미세플라스틱은 해양 생물의 체내에 축적되어 먹이사슬을 통해 인간에게도 영향을 미칠 수 있다. 연구팀은 일회용 플라스틱 사용 규제를 더욱 강화해야 한다고 강조했다.

**이 글의 제목으로 알맞은 것은?**
① 해양 오염 감소 추세
② 해양 미세플라스틱 5년간 3배 급증
③ 플라스틱 재활용률 증가
④ 해양 생물 보호구역 확대

**答案**: ②

## 五、新闻标题还原练习

将标题还原为完整句子：

| 标题                        | 完整意思                                               |
| --------------------------- | ------------------------------------------------------ |
| 한국 관광객 일본행 급증     | 한국 관광객의 일본행이 급증하고 있다                   |
| 정부, 최저임금 인상 결정    | 정부가 최저임금 인상을 결정했다                        |
| AI 도입 기업 생산성 향상    | AI를 도입한 기업의 생산성이 향상되었다                 |
| 서울 미세먼지 '나쁨' 이틀째 | 서울의 미세먼지가 '나쁨' 수준으로 이틀째 이어지고 있다 |
| 출산율 역대 최저…대책 시급  | 출산율이 역대 최저를 기록했으며 대책이 시급하다        |

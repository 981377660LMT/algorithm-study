# 看图选句类 (그림/그래픽 보고 내용 고르기) — 3~4题

---

## 一、题型说明

给出图片(标识/图表/场景图)，选与图片内容一致的句子。

---

## 三、解题步骤

```
Step 1: 仔细观察图片中的文字/数字/人物动作
Step 2: 在选项中找与图片完全一致的描述
Step 3: 排除部分正确但有细节错误的选项
```

## 四、实战练习

### 练习1：标识牌

**[图片描述]**: 门上贴着"화요일 휴무"的标识

**正确答案句式选择**:
① 이 가게는 월요일에 쉽니다.
② 이 가게는 화요일에 쉽니다.
③ 이 가게는 매일 운영합니다.
④ 이 가게는 주말에 쉽니다.

**答案**: ②
**解析**: 화요일 휴무 = 周二休息

### 练习2：时间表

**[图片描述]**: 도서관 운영 시간표

```
평일: 09:00~21:00
토요일: 09:00~18:00
일요일·공휴일: 휴관
```

**보기**:
① 도서관은 주말에 문을 열지 않는다.
② 도서관은 토요일에 9시에 문을 닫는다.
③ 도서관은 토요일에도 평일과 같은 시간에 운영한다.
④ 도서관은 일요일과 공휴일에 문을 열지 않는다.

**答案**: ④
**解析**: ①错在"주말"包含토요일(토요일开門)；②错在닫는 시간；③错在"같은 시간"；④正确。

### 练习3：柱状图

**[图片描述]**: 韩国人最喜欢的运动 调查结果

```
축구: 35%
야구: 25%
농구: 15%
수영: 15%
기타: 10%
```

**보기**:
① 야구를 좋아하는 사람이 가장 많다.
② 축구를 좋아하는 사람이 가장 많다.
③ 농구와 수영을 좋아하는 사람이 같다.
④ 수영을 좋아하는 사람이 가장 적다.

**答案**: ② (축구 35% 最高)
**注意**: ③也正确(都是15%)，但题目问"가장 맞는 것"时 ② 更核心; 若选"모두 맞는 것" 则 ②③皆可。实际考题只设一个正确答案。

### 练习4：场景图

**[图片描述]**: 一个人在우체국柜台前，手里拿着包裹

**보기**:
① 남자가 편지를 읽고 있다.
② 남자가 소포를 보내려고 한다.
③ 남자가 우표를 사고 있다.
④ 남자가 은행에서 돈을 찾고 있다.

**答案**: ②
**解析**: 우체국 + 包裹 = 寄包裹

### 练习5：饼图

**[图片描述]**: 대학생 여가 활동 조사

```
인터넷/SNS: 40%
운동: 20%
독서: 15%
여행: 15%
기타: 10%
```

**보기**:
① 대학생들은 여가 시간에 주로 독서를 한다.
② 대학생들은 여가 시간에 인터넷을 가장 많이 한다.
③ 운동을 하는 대학생이 여행을 하는 대학생보다 적다.
④ 독서를 하는 대학생이 가장 적다.

**答案**: ②
**解析**: 인터넷/SNS 40%最高

## 五、图表题必备读图词汇

### 增减趋势

| 韩语                   | 意思      |
| ---------------------- | --------- |
| 증가하다/늘다/늘어나다 | 增加      |
| 감소하다/줄다/줄어들다 | 减少      |
| 변화가 없다            | 没有变化  |
| 유지되다               | 维持      |
| 가장 많다/높다         | 最多/最高 |
| 가장 적다/낮다         | 最少/最低 |
| ~보다 많다/적다        | 比…多/少  |
| 두 배                  | 两倍      |

### 比较表达

| 韩语                             | 意思          |
| -------------------------------- | ------------- |
| A이/가 B보다 높다/많다           | A比B高/多     |
| A와/과 B가 같다/비슷하다         | A和B相同/相似 |
| A이/가 가장 높은 비율을 차지하다 | A占最高比例   |
| A 다음으로 B가 많다              | 其次是B多     |

# 说话者意图·心情类 (화자 의도·심정 고르기)

---

## 一、题型说明

常见提问：

- 밑줄 친 부분에 나타난 심정으로 알맞은 것을 고르십시오.
- 이 글을 쓴 목적으로 알맞은 것을 고르십시오.
- 화자의 태도/의도로 가장 알맞은 것을 고르십시오.

## 三、心情推断线索

### 1. 通过动作/行为推断

| 动作              | 推断心情                       |
| ----------------- | ------------------------------ |
| 눈물을 흘리다     | 슬프다/감동적이다/기쁘다(감격) |
| 한숨을 쉬다       | 답답하다/걱정되다/안타깝다     |
| 미소를 짓다       | 기쁘다/뿌듯하다/반갑다         |
| 가슴이 두근거리다 | 설레다/긴장되다                |
| 고개를 숙이다     | 부끄럽다/미안하다              |
| 발을 구르다       | 답답하다/화나다                |
| 멍하니 서 있다    | 당황하다/막막하다              |
| 빙그레 웃다       | 기쁘다/뿌듯하다                |

### 2. 通过感叹词推断

| 感叹                | 推断心情          |
| ------------------- | ----------------- |
| 아... 세월이 빠르다 | 그립다/아쉽다     |
| 역시!               | 만족하다/뿌듯하다 |
| 어쩌면 좋지?        | 당황하다/걱정되다 |
| 다행이다            | 안도하다/후련하다 |
| 이럴 수가!          | 당황하다/놀라다   |

### 3. 通过语境推断

| 语境            | 推断心情            |
| --------------- | ------------------- |
| 오랜만의 재회   | 반갑다/그립다       |
| 어린 시절 추억  | 그립다/아쉽다       |
| 약속 취소       | 아쉽다/서운하다     |
| 갑작스러운 사건 | 당황하다/황당하다   |
| 자녀의 성장     | 뿌듯하다/대견하다   |
| 이별            | 슬프다/허전하다     |
| 시험 앞         | 긴장되다/초조하다   |
| 실패 후         | 후회스럽다/막막하다 |
| 도움 받았을 때  | 감사하다/고맙다     |

## 四、实战练习 (8题)

### 练习1

오늘 처음으로 내가 만든 요리를 엄마에게 대접했다. 엄마는 한 입 먹더니 "이게 네가 만든 거야? 정말 맛있다!"라고 하셨다. 그 말을 듣는 순간 가슴이 따뜻해졌다.

**밑줄 친 부분에서 '나'의 심정은?**
① 답답하다
② 뿌듯하다
③ 긴장되다
④ 속상하다

**答案**: ②
**解析**: 妈妈称赞自己做的菜 → 自豪满足

### 练习2

친구와 같이 콘서트에 가기로 했는데 친구가 갑자기 못 간다고 했다. 이 콘서트를 위해 한 달 전부터 준비했는데... 혼자 가야 할지 아니면 포기해야 할지 모르겠다.

**'나'의 심정은?**
① 설레다
② 반갑다
③ 아쉽다
④ 부끄럽다

**答案**: ③

### 练习3

약속 시간이 다 되어 가는데 택시가 잡히지 않았다. 전화로 "거의 다 왔어"라고 말했지만 사실 아직 집 앞이었다. 손에 땀이 나고 마음이 급했다.

**'나'의 심정은?**
① 후련하다
② 초조하다
③ 행복하다
④ 그립다

**答案**: ②

### 练习4

20년 만에 고향에 돌아왔다. 어릴 때 뛰어놀던 골목이 그대로 있었다. 그때 같이 놀던 친구들은 지금 어디서 뭘 하고 있을까. 예전의 기억이 하나둘 떠올랐다.

**'나'의 심정은?**
① 화가 나다
② 짜증나다
③ 그립다
④ 당황하다

**答案**: ③

### 练习5

3년간 준비한 사업이 드디어 성공했다. 힘든 시간이 많았지만 포기하지 않은 자신이 대견하다. 그동안 도와준 사람들에게 감사한 마음이 가득하다.

**'나'의 심정은?**
① 뿌듯하고 감사하다
② 답답하고 초조하다
③ 허전하고 외롭다
④ 부끄럽고 후회스럽다

**答案**: ①

### 练习6

채용 면접을 보러 갔는데 면접관이 나의 이력서를 한참 보더니 "경험이 좀 부족하시네요"라고 했다. 열심히 준비했는데 그 한마디에 마음이 무너지는 것 같았다.

**'나'의 심정은?**
① 기쁘다
② 실망스럽다
③ 반갑다
④ 홀가분하다

**答案**: ②

### 练习7

여행 마지막 날, 공항으로 가는 버스 안에서 창밖을 바라보았다. 일주일간의 여행이 꿈처럼 빨리 지나갔다. 다음에 다시 올 수 있을까. 가슴 한쪽이 텅 빈 것 같았다.

**'나'의 심정은?**
① 든든하다
② 허전하다
③ 부끄럽다
④ 황당하다

**答案**: ②

### 练习8

내일이 시험인데 아직 공부를 많이 하지 못했다. 이제 와서 후회해도 소용없다는 걸 알지만, 왜 일찍 시작하지 않았을까 하는 생각에 자꾸 한숨이 나온다.

**'나'의 심정은?**
① 설레다
② 반갑다
③ 후회스럽다
④ 감동적이다

**答案**: ③

## 五、写作目的类 常见选项

## 二、各题型核心解法

### 1~2题：实用文选主题 (광고/안내/초대)

**解法**: 找关键名词(活动名/商品名/场所) → 排除无关选项

- 看标题/大字
- 看日期/场所/对象
- 不需逐字读

### 3~4题：看图选句

**解法**: 观察图中 → 场景(어디?) + 动作(뭐 해?) + 关系(누구?)

- 看人物表情/动作
- 看背景场所
- 选描述最准确的句子

### 5~8题：划线替换 (밑줄 = ?)

**解法**: 理解划线部分语法含义 → 选近义表达

- 核心考点：语法同义替换
- 例：-곤 하다 = 자주 -는다 / -는 경우가 많다
- 例：-는 셈이다 = -는 것과 같다

### 9~12题：排序/插入

**解法**: 找逻辑线索词

- 指示词（이, 그, 이러한, 그런）
- 接续词（그래서, 하지만, 또한）
- 代词（그것, 이것）
- 主语/话题一致性

### 13~18题：主题/填空

**解法**:

- 主题题 → 找反复出现的核心词
- 填空题 → 读前后文推断逻辑关系(因果/转折/递进)

### 19~24题：内容一致

**解法**: 逐选项对照原文

- 注意数字/时间/条件的细微差异
- 注意否定词/限定词
- "모두/항상/반드시" 等绝对表述通常是陷阱

### 25~27题：新闻标题

**解法**: 理解新闻标题省略特征

- 省略助词：한국 경제 성장률 하락 (省了이/가)
- 名词化：증가 → 증가하다, 실시 → 실시하다
- 被动/使动缩写

### 28~50题：中高级综合

**解法**: 按文体分类应对

- 说明文 → 找中心句(通常在开头或结尾)
- 论述文 → 找论点+论据关系
- 新闻 → 5W1H(누가/언제/어디서/무엇을/왜/어떻게)
- 文学 → 感受人物情感+象征意义

## 三、阅读选项排除法

### 绝对表述 = 高概率错误

| 危险词        | 意思 | 为什么危险                |
| ------------- | ---- | ------------------------- |
| 모두/모든     | 所有 | 原文通常只说"很多/大部分" |
| 항상/언제나   | 总是 | 原文通常有条件限制        |
| 반드시/꼭     | 一定 | 原文通常是"可能/建议"     |
| 절대(로)      | 绝对 | 过于极端                  |
| 전혀          | 完全 | 否定极端化                |
| 오직/유일하게 | 唯一 | 限制过强                  |

### 部分表述 = 更可能正确

| 安全词          | 意思     |
| --------------- | -------- |
| 대부분          | 大部分   |
| 주로            | 主要     |
| 대체로          | 大体上   |
| -(으)ㄹ 수 있다 | 有可能   |
| 경우가 많다     | 情况很多 |

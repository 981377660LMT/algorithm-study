## 六、真题常见选项措辞

### "内容一致" 题正确选项常见措辞

| 原文             | 正确选项变体                  |
| ---------------- | ----------------------------- |
| 증가하고 있다    | 늘어나고 있다 / 많아지고 있다 |
| 바람직하다       | 좋다 / 필요하다               |
| 문제가 있다      | 문제점이 지적되고 있다        |
| ~에 도움이 된다  | ~에 효과가 있다 / 긍정적이다  |
| ~것으로 나타났다 | ~는 것을 알 수 있다           |

### "内容不一致" 题错误选项常见手法

| 手法        | 说明                  |
| ----------- | --------------------- |
| 숫자 변경   | 25% → 50%             |
| 주어 교체   | 학생 → 교사           |
| 시제 변경   | 예정이다 → 이미 했다  |
| 정도 변경   | 줄었다 → 사라졌다     |
| 긍부정 변환 | 가능하다 → 불가능하다 |
| 인과 역전   | A때문에B → B때문에A   |
| 무중생유    | 원문에 없는 내용      |

## 阅读学习方法论

### 每日训练方案 (60分钟)

| 阶段     | 时间 | 内容                       |
| -------- | ---- | -------------------------- |
| 精读     | 20분 | 1篇文章逐句分析(语法+词汇) |
| 题型练习 | 20분 | 做5~8道分题型练习          |
| 速读     | 10분 | 限时快速阅读1~2篇          |
| 词汇积累 | 10분 | 整理生词+复习              |

### 阅读能力四层次

```
Level 1: 字面理解 (글자 그대로) → 能看懂每个句子
Level 2: 推理理解 (추론) → 能理解言外之意
Level 3: 结构理解 (구조) → 能把握文章脉络
Level 4: 批判理解 (비판) → 能判断作者立场态度
```

# 填空选择类 (빈칸 채우기) — 중급

---

## 一、题型说明

给出一段短文，其中有 ( ) 空白，选择最合适的词/短语/句子填入。

常见提问：

- 다음 ( )에 들어갈 내용으로 알맞은 것을 고르십시오.

---

## 二、填空逻辑类型

### Type 1: 因果填空（前因→填结果 / 填原因←后果）

**标志词**: 그래서, 따라서, -기 때문에, -(으)므로, 그 결과

| 位置       | 关系   | 例                                        |
| ---------- | ------ | ----------------------------------------- |
| 原因 → ( ) | 填结果 | 비가 왔다. 그래서 ( ) → 소풍을 못 갔다    |
| ( ) → 结果 | 填原因 | ( ) 때문에 시험에 떨어졌다 → 공부를 안 한 |

### Type 2: 转折填空（前文→但是→填相反内容）

**标志词**: 그러나, 하지만, -(으)ㄴ/는데, -지만

| 位置                 | 关系        |
| -------------------- | ----------- |
| A (积极). 하지만 ( ) | 填消极/相反 |
| A (消极). 그러나 ( ) | 填积极/相反 |

### Type 3: 递进填空

**标志词**: 또한, 게다가, 뿐만 아니라, -는데다가

| 位置            | 关系       |
| --------------- | ---------- |
| A好. 게다가 ( ) | 填同方向好 |
| A坏. 게다가 ( ) | 填同方向坏 |

### Type 4: 举例填空

**标志词**: 예를 들어, 이를테면, 가령

| 位置                | 关系       |
| ------------------- | ---------- |
| 观点. 예를 들어 ( ) | 填具体例子 |
| ( ) 이처럼 总结     | 填具体例子 |

### Type 5: 中心句填空（通常在开头或结尾）

前后内容都围绕一个Theme → ( ) 应该是核心概括

## 三、实战练习 (10题)

### 练习1

사람들은 보통 첫인상으로 상대방을 판단한다. 처음 만났을 때 좋은 인상을 주면 관계가 ( ). 따라서 첫 만남에서 미소를 짓고 상대의 눈을 보며 이야기하는 것이 좋다.

① 어려워질 수 있다
② 복잡해질 수 있다
③ 좋아질 수 있다
④ 나빠질 수 있다

**答案**: ③
**解析**: 좋은 인상 → 관계가 좋아진다

### 练习2

운동은 건강에 좋다. 규칙적으로 운동을 하면 체력이 좋아지고 스트레스도 줄어든다. 그러나 ( ). 갑자기 무리하게 운동을 하면 오히려 몸에 해로울 수 있다.

① 운동을 매일 해야 한다
② 과도한 운동은 주의해야 한다
③ 운동의 종류는 상관없다
④ 운동을 좋아하는 사람이 많다

**答案**: ②
**解析**: "그러나" → 转折; 后文说无理运动有害 → 填"过度运动要注意"

### 练习3

음식이 맛있으려면 좋은 재료가 필요하다. ( ) 아무리 요리 실력이 좋아도 재료가 나쁘면 맛있는 음식을 만들기 어렵다. 그래서 유명한 요리사들은 재료 선택에 많은 시간을 투자한다.

① 또한 가격도 중요하다
② 맛은 사람마다 다르다
③ 즉 재료가 음식의 맛을 결정하는 것이다
④ 하지만 요리법도 중요하다

**答案**: ③
**解析**: 前说需要好材料，后说材料不好就做不好 → 填"材料决定味道"

### 练习4

여행을 갈 때 많은 사람들이 유명한 관광지를 찾는다. 그러나 최근에는 ( ). 잘 알려지지 않은 작은 마을이나 골목을 걸으며 현지인의 생활을 체험하는 것을 선호하는 사람이 늘고 있다.

① 여행 비용이 줄어들고 있다
② 여행을 가는 사람이 점점 줄고 있다
③ 유명하지 않은 곳을 여행하는 사람이 많아지고 있다
④ 혼자 여행하는 것이 위험해지고 있다

**答案**: ③

### 练习5

꿀벌은 생태계에서 매우 중요한 역할을 한다. ( ) 꿀벌이 식물의 꽃가루를 옮겨 주기 때문이다. 만약 꿀벌이 사라진다면 많은 식물이 열매를 맺지 못하게 될 것이다.

① 꿀벌은 꿀을 만든다
② 꿀벌 없이는 많은 식물이 번식할 수 없다
③ 꿀벌의 수가 줄고 있다
④ 꿀벌은 사람에게 위험할 수 있다

**答案**: ②
**解析**: 后文解释꽃가루 옮기기 = 帮助繁殖

### 练习6

성공한 사람들의 공통점 중 하나는 실패를 두려워하지 않는다는 것이다. 그들은 실패를 ( ) 생각한다. 실패에서 교훈을 얻고 그것을 바탕으로 다시 도전하기 때문이다.

① 피해야 할 것으로
② 성공으로 가는 과정으로
③ 부끄러운 일로
④ 남의 탓으로

**答案**: ②

### 练习7

혼자 사는 1인 가구가 증가하면서 ( ). 작은 크기의 가전제품, 1인분 식품, 소형 가구 등이 인기를 끌고 있다.

① 결혼하는 사람이 늘고 있다
② 1인 가구를 위한 상품이 다양해지고 있다
③ 가족과의 관계가 중요해지고 있다
④ 대형 마트의 매출이 증가하고 있다

**答案**: ②

### 练习8

봉사 활동은 다른 사람을 돕는 일이지만, 실제로는 봉사하는 사람 자신에게도 여러 가지 이점이 있다. 봉사를 하면서 ( ). 또한 새로운 사람들을 만나 인간관계를 넓힐 수 있다.

① 시간이 부족해진다
② 보람과 행복감을 느낄 수 있다
③ 돈을 벌 수 있다
④ 체력이 약해질 수 있다

**答案**: ②

### 练习9

( ) 그래서 잠을 잘 자는 것이 중요하다. 잠이 부족하면 기억력이 떨어지고 감정 조절도 어려워진다. 전문가들은 하루에 7~8시간의 수면을 권장한다.

① 잠은 몸과 뇌가 회복하는 시간이다.
② 낮잠을 자면 밤에 잠을 못 잔다.
③ 잠을 많이 자면 게으른 사람이다.
④ 수면제를 먹으면 잠을 잘 잘 수 있다.

**答案**: ①
**解析**: 后文"그래서 잘 자는 것이 중요" → 前面应该是"睡眠让身体大脑恢复"

### 练习10

지구 온난화로 인해 극지방의 빙하가 녹고 있다. 이로 인해 해수면이 상승하여 저지대 지역이 물에 잠길 위험이 있다. ( ) 우리 모두가 탄소 배출을 줄이기 위해 노력해야 한다.

① 따라서
② 그런데
③ 예를 들어
④ 반면에

**答案**: ①
**解析**: 前=问题陈述 → "따라서" → 结论建议

## 四、填空题解题口诀

```
1. 先看空格前后各1~2句
2. 判断逻辑关系(因果/转折/递进/总结)
3. 预测填入内容的方向(积极/消极, 具体/概括)
4. 对照选项，排除方向不对的
5. 确认填入后上下文通顺
```

# 句子排序与插入类 (글 순서/문장 넣기) — 9~12题

---

## 一、题型说明

### A. 순서 배열 (排序)

给出(가)(나)(다)(라)四个句子，选正确顺序。

### B. 문장 넣기 (句子插入)

给出一段文字中有( )空白处，选择应该填入的句子。

---

## 二、排序解题核心：逻辑线索词

### 1. 指示词（前面一定有被指代的内容）

| 指示词        | 意思           | 位置规律          |
| ------------- | -------------- | ----------------- |
| 이/이것/이런  | 这/这个/这样的 | 紧跟在被指对象后  |
| 그/그것/그런  | 那/那个/那样的 | 前面有被指的内容  |
| 이러한/그러한 | 如此的/那样的  | 同上              |
| 여기/거기     | 这里/那里      | 前面提过场所      |
| 이때          | 这时           | 前面提过时间/场景 |

**规律**: 含指示词的句子绝不是第一句！

### 2. 接续词

| 接续词                  | 功能          | 前后关系         |
| ----------------------- | ------------- | ---------------- |
| 그래서/따라서/그러므로  | 因果          | 前=原因，后=结果 |
| 그러나/하지만/그렇지만  | 转折          | 前后相反         |
| 그런데                  | 转折/话题转换 | 补充新信息       |
| 또한/게다가/뿐만 아니라 | 递进          | 同方向补充       |
| 즉/다시 말해            | 换言之        | 前后同义         |
| 예를 들어               | 举例          | 前=观点，后=例子 |
| 이처럼/이와 같이        | 如此          | 前=具体，后=总结 |
| 반면(에)                | 对比          | 前后对比         |
| 그럼에도 불구하고       | 尽管如此      | 前=困难，后=克服 |
| 오히려                  | 反而          | 与预期相反       |
| 결국/마침내             | 最后/终于     | 结尾             |

### 3. 话题引入标记（经常是第一句）

| 标记       | 示例                     | 位置           |
| ---------- | ------------------------ | -------------- |
| 定义句     | ~(이)란 ~을/를 말한다    | 开头           |
| 背景介绍   | 요즘/최근/현대 사회에서  | 开头           |
| 疑问句     | ~(으)ㄹ까?               | 开头(引起思考) |
| 一般性论述 | 사람들은 보통/일반적으로 | 开头           |

### 4. 结尾标记（经常是最后一句）

| 标记             | 示例              | 位置       |
| ---------------- | ----------------- | ---------- |
| 따라서/그러므로  | 따라서 ~해야 한다 | 结尾       |
| 이처럼/이와 같이 | 이처럼 ~인 것이다 | 结尾       |
| 결론적으로       | 결론적으로 ~다    | 结尾       |
| -(으)ㄹ 것이다   | ~할 것이다        | 结尾(展望) |
| -아/어야 한다    | ~해야 한다        | 结尾(建议) |

## 三、排序实战 (5题)

### 练习1

**(가)** 그래서 요즘은 건강을 위해 걷기 운동을 하는 사람이 늘고 있다.
**(나)** 현대인들은 운동 부족으로 여러 가지 건강 문제를 겪고 있다.
**(다)** 걷기는 특별한 장비 없이 누구나 할 수 있다는 장점이 있다.
**(라)** 이처럼 걷기 운동은 현대인에게 가장 적합한 운동이라 할 수 있다.

**答案**: (나)-(가)-(다)-(라)
**解析**:

- (나) 배경 도입(현대인+문제)
- (가) "그래서" → 因果结果(所以走路的人增加)
- (다) 걷기의 장점 补充
- (라) "이처럼" → 总结

### 练习2

**(가)** 그런데 최근 연구에 따르면 아침 식사를 거르는 것이 건강에 좋지 않다고 한다.
**(나)** 이러한 결과를 보면 바쁘더라도 아침 식사를 하는 것이 좋겠다.
**(다)** 아침에 식사를 하지 않는 사람이 점점 많아지고 있다.
**(라)** 아침을 먹지 않으면 집중력이 떨어지고 비만의 위험도 커진다고 한다.

**答案**: (다)-(가)-(라)-(나)
**解析**:

- (다) 현상 도입(不吃早饭的人增加)
- (가) "그런데" → 转折(研究说不吃不好)
- (라) 구체적 근거(集中力下降+肥胖风险)
- (나) "이러한 결과를 보면" → 总结建议

### 练习3

**(가)** 예를 들어 커피에는 항산화 물질이 풍부하게 들어 있다.
**(나)** 하지만 커피를 너무 많이 마시면 수면 장애를 일으킬 수 있다.
**(다)** 커피는 적당히 마시면 건강에 도움이 된다고 알려져 있다.
**(라)** 따라서 하루에 2~3잔 정도가 적당하다.

**答案**: (다)-(가)-(나)-(라)
**解析**:

- (다) 主题导入(커피와 건강)
- (가) "예를 들어" → 举例
- (나) "하지만" → 转折(过度不好)
- (라) "따라서" → 结论

### 练习4

**(가)** 이때 중요한 것은 자신의 감정을 솔직하게 표현하는 것이다.
**(나)** 사람들은 살면서 다양한 갈등을 겪게 된다.
**(다)** 이를 잘 해결하기 위해서는 대화가 필요하다.
**(라)** 솔직한 대화를 통해 서로를 이해할 수 있기 때문이다.

**答案**: (나)-(다)-(가)-(라)
**解析**:

- (나) 일반적 도입(갈등을 겪다)
- (다) "이를" → 指代갈등 → 대화 필요
- (가) "이때" → 此时重要的是…
- (라) "~이기 때문이다" → 补充原因

### 练习5

**(가)** 그런 사람들을 위해 동물 로봇이 개발되었다.
**(나)** 반려동물을 키우면 외로움을 줄일 수 있다는 연구 결과가 있다.
**(다)** 하지만 여러 가지 이유로 동물을 키우지 못하는 사람도 있다.
**(라)** 이 로봇은 실제 동물처럼 반응하여 주인에게 정서적 안정감을 준다.

**答案**: (나)-(다)-(가)-(라)

## 四、句子插入实战 (5题)

### 练习6

다음 글에서 \<보기>의 문장이 들어가기에 가장 알맞은 곳을 고르십시오.

**\<보기>**: 그러나 독서의 효과는 이것만이 아니다.

( ① ) 독서는 우리에게 다양한 지식을 준다. ( ② ) 책을 읽으면서 우리는 모르던 세계를 알게 된다. ( ③ ) 독서는 스트레스를 줄여 주고 집중력을 높여 준다는 연구 결과도 있다. ( ④ ) 이처럼 독서는 우리 삶에 큰 도움이 된다.

**答案**: ②
**解析**: "그러나 ~이것만이 아니다" → 前面说了一个好处(지식) → 这里转折說不只如此 → 后面补充其他好处(스트레스/집중력)

### 练习7

**\<보기>**: 이 때문에 많은 나라에서 일회용품 사용을 줄이려는 노력을 하고 있다.

( ① ) 편리한 생활을 위해 일회용품이 많이 사용되고 있다. ( ② ) 하지만 일회용품은 환경에 큰 피해를 준다. ( ③ ) 일회용품은 잘 분해되지 않아 토양과 바다를 오염시킨다. ( ④ )

**答案**: ④
**解析**: "이 때문에"指代前面的环境问题(土壤/海洋污染) → 所以各国努力减少一次性用品

### 练习8

**\<보기>**: 그렇다면 어떻게 해야 시간을 잘 관리할 수 있을까?

( ① ) 현대인들은 항상 시간이 부족하다고 느낀다. ( ② ) 해야 할 일은 많은데 시간은 한정되어 있기 때문이다. ( ③ ) 먼저 해야 할 일의 우선순위를 정하는 것이 중요하다. ( ④ ) 그리고 계획을 세워 실천하면 시간을 효율적으로 사용할 수 있다.

**答案**: ②(即放在②和③之间)
**解析**: 보기是一个疑问句(怎样才能管理好时间) → 前面是问题描述(时间不够) → 后面是解答(先定优先顺序)

## 五、排序/插入 黄金法则

```
1. 找第一句: 无指示词 + 有背景/定义/引入
2. 找最后一句: 总结词(이처럼/따라서) + 结论/建议
3. 找紧密对: 指示词必须紧跟被指内容
4. 接续词确认: 因果/转折/递进逻辑是否通顺
5. 主语连贯: 同一主题的句子应连续
```

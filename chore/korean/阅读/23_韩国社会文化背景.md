## 一、韩国社会热点话题

| 关键词      | 翻译       |
| ----------- | ---------- |
| 저출산      | 低出生率   |

## 四、TOPIK 阅读中高频社会话题

| 话题    | 常见论点                        |
| ------- | ------------------------------- |
| 저출산  | 원인(양육비/주거비)→정부 지원책 |
| 고령화  | 노인 빈곤/복지/일자리           |
| 교육    | 사교육 과열→공교육 강화         |
| 환경    | 기후 변화→탄소 중립→개인 실천   |
| AI/기술 | 편의 vs 일자리 대체/윤리 문제   |
| 문화    | 전통 보존 vs 현대화             |
| 경제    | 최저임금/부동산/청년 실업       |
| 건강    | 정신 건강/운동/식단             |
| 미디어  | 가짜 뉴스/디지털 리터러시       |
| 다문화  | 이민자 증가/편견 해소           |

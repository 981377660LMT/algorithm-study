# 文章目的与态度类 (글의 목적·글쓴이 태도)

---

## 一、题型说明

常见提问：

- 이 글의 목적으로 알맞은 것을 고르십시오.
- 글쓴이의 태도로 알맞은 것을 고르십시오.
- 글쓴이의 의견으로 알맞은 것을 고르십시오.

---

## 二、글의 목적 (文章目的) 分类

| 목적                 | 翻译         | 文中标志                         |
| -------------------- | ------------ | -------------------------------- |
| 정보를 전달하다      | 传达信息     | 客观叙述, 数据, ~것으로 나타났다 |
| 문제를 제기하다      | 提出问题     | 질문형, ~문제가 있다/심각하다    |
| 해결 방안을 제시하다 | 提出解决方案 | ~해야 한다, ~는 것이 필요하다    |
| 주장하다             | 主张         | ~다고 생각하다, ~아/어야 한다    |
| 비판하다             | 批评         | 부정적 어휘, ~문제가 있다        |
| 홍보/광고하다        | 宣传         | 장점 나열, ~해 보세요            |
| 안내/공지하다        | 通知         | 정보 전달, 날짜/장소/방법        |
| 건의/제안하다        | 建议         | ~하는 것이 좋겠다, 제안          |
| 소개하다             | 介绍         | 대상 설명, 특징 나열             |
| 감사/축하하다        | 感谢/祝贺    | 감사하다, 축하하다               |
| 사과하다             | 道歉         | 죄송하다, 양해 부탁              |
| 경험을 나누다        | 分享经验     | 주관적, 감정 표현, 일화          |
| 교훈을 주다          | 给予教训     | 우화, 이야기+교훈                |

## 四、态度判断标志

### 赞成/积极的标志

| 标志语                 | 翻译           |
| ---------------------- | -------------- |
| ~는 것이 바람직하다    | …是可取的      |
| ~의 효과/장점이 있다   | 有…的效果/优点 |
| ~에 도움이 된다        | 对…有帮助      |
| ~가 기대된다           | 令人期待       |
| 긍정적인 영양을 미치다 | 产生积极影响   |

### 反对/消极的标志

| 标志语                 | 翻译         |
| ---------------------- | ------------ |
| ~는 문제가 있다        | …有问题      |
| ~에 대한 우려가 있다   | 有对…的担忧  |
| ~는 바람직하지 않다    | …不可取      |
| 부정적인 영향을 미치다 | 产生消极影响 |
| ~를 경계해야 한다      | 要警惕…      |
| 부작용이 우려된다      | 担心副作用   |

### 中立/客观的标志

| 标志语                          | 翻译             |
| ------------------------------- | ---------------- |
| ~는 장단점이 있다               | …有优缺点        |
| 한편으로는 ~, 다른 한편으로는 ~ | 一方面…另一方面… |
| 찬반 양론이 있다                | 有赞反两论       |
| ~에 대해 다양한 의견이 있다     | 有各种意见       |

## 五、实战练习 (8题)

### 练习1

인터넷 쇼핑이 편리하기는 하지만 개인 정보 유출의 위험이 있다. 실제로 해마다 개인 정보가 유출되는 사건이 증가하고 있다. 따라서 온라인에서 물건을 살 때는 안전한 결제 방법을 사용하고, 비밀번호를 정기적으로 변경하는 등 주의가 필요하다.

**글쓴이의 태도는?**
① 인터넷 쇼핑의 편리함을 강조하고 있다
② 인터넷 쇼핑의 위험성을 경고하고 있다
③ 인터넷 쇼핑을 금지해야 한다고 주장하고 있다
④ 인터넷 쇼핑의 발전을 기대하고 있다

**答案**: ②

### 练习2

최근 많은 학교에서 AI를 활용한 교육을 도입하고 있다. AI를 이용하면 학생 개인의 수준에 맞는 맞춤형 교육이 가능하다. 그러나 지나치게 AI에 의존하면 학생들의 사고력이 저하될 수 있다는 의견도 있다. AI 교육의 장점을 살리면서도 학생들의 주체적인 사고력을 키우는 방향으로 발전해야 할 것이다.

**글쓴이의 의견은?**
① AI 교육을 전면 도입해야 한다
② AI 교육을 중단해야 한다
③ AI 교육의 장단점을 균형 있게 활용해야 한다
④ 학생들이 AI를 직접 개발해야 한다

**答案**: ③

### 练习3

매년 봄이 되면 한강에서 벚꽃 축제가 열린다. 축제 기간에는 야간 조명이 설치되어 밤에도 아름다운 벚꽃을 감상할 수 있다. 또한 다양한 문화 공연, 먹거리 장터, 사진 전시회 등이 진행된다. 올해는 4월 5일부터 4월 15일까지 개최될 예정이니 많은 관심 바랍니다.

**이 글의 목적은?**
① 벚꽃의 생태를 설명하려고
② 벚꽃 축제를 홍보하려고
③ 한강의 역사를 소개하려고
④ 봄 여행지를 추천하려고

**答案**: ②

### 练习4

플라스틱 빨대는 연간 수십억 개가 사용되며 해양 생태계를 심각하게 위협하고 있다. 일부 국가에서는 이미 플라스틱 빨대 사용을 금지하고 있다. 우리도 종이 빨대나 스테인리스 빨대 같은 대안을 적극적으로 사용해야 한다. 작은 실천이 지구를 살릴 수 있다.

**글쓴이의 태도는?**
① 플라스틱 빨대의 편리함을 강조하고 있다
② 플라스틱 빨대 대안 사용을 촉구하고 있다
③ 플라스틱 빨대 금지 정책을 비판하고 있다
④ 해양 생태계 연구를 소개하고 있다

**答案**: ②

### 练习5

동물원은 멸종 위기 동물을 보호한다는 긍정적 측면이 있다. 하지만 좁은 공간에 갇혀 사는 동물들이 스트레스를 받고 자연스러운 행동을 하지 못한다는 비판도 있다. 동물의 복지를 위해 동물원의 사육 환경을 개선하는 노력이 필요하다.

**글쓴이의 의견은?**
① 동물원을 모두 폐쇄해야 한다
② 동물원은 문제가 없다
③ 동물원의 환경을 개선해야 한다
④ 동물원에 더 많은 동물을 데려와야 한다

**答案**: ③

### 练习6

지난 10년간 한국의 커피 소비량은 매년 꾸준히 증가해 왔다. 한국인 1인당 연간 커피 소비량은 약 350잔으로 세계 평균의 두 배에 달한다. 커피 전문점의 수도 빠르게 늘어나고 있으며, 커피 관련 산업의 규모도 계속 커지고 있다.

**이 글의 목적은?**
① 커피의 건강 효과를 알리려고
② 커피 소비 현황에 대한 정보를 전달하려고
③ 커피를 너무 많이 마시지 말라고 경고하려고
④ 좋은 커피를 고르는 방법을 알려 주려고

**答案**: ②

### 练习7

요즘 아파트 층간소음 문제가 심각하다. 윗집의 발소리, 아이들이 뛰어다니는 소리 때문에 이웃 간 갈등이 발생하는 경우가 많다. 이 문제를 해결하기 위해서는 건축 단계부터 방음 설계를 강화하고, 입주민 간의 배려 문화도 함께 만들어 나가야 한다.

**글쓴이의 태도는?**
① 층간소음 문제에 대해 구체적인 해결 방안을 제시하고 있다
② 층간소음이 심각하지 않다고 주장하고 있다
③ 아파트에 살지 말라고 권유하고 있다
④ 층간소음에 대한 법률을 소개하고 있다

**答案**: ①

### 练习8

자원봉사에 참여한 학생들은 처음에는 의무감으로 시작했지만, 활동을 하면서 큰 보람을 느꼈다고 한다. 봉사를 통해 사회에 대한 이해와 책임감이 깊어졌다는 학생도 많다. 이처럼 자원봉사는 도움을 받는 사람뿐만 아니라 봉사하는 사람에게도 의미 있는 경험이 된다.

**이 글의 목적은?**
① 자원봉사의 양면적 가치를 소개하려고
② 자원봉사를 의무화해야 한다고 주장하려고
③ 자원봉사의 문제점을 지적하려고
④ 학생들의 봉사 시간을 줄이자고 제안하려고

**答案**: ①

## 六、태도·목적 판断速查

| 문中 有…                      | 태도/목적 판단      |
| ----------------------------- | ------------------- |
| 객관적 data + ~으로 나타났다  | 정보 전달           |
| ~해야 한다, ~는 것이 필요하다 | 주장/해결 방안 제시 |
| ~는 문제가 있다, 우려된다     | 문제 제기/경고      |
| 장점 + 단점 + 균형            | 균형 잡힌 시각      |
| ~해 보세요, ~추천합니다       | 홍보/제안           |
| ~가 바람직하다                | 긍정적 주장         |
| 감정 표현 + 개인 경험         | 경험 공유           |

# 图表数据解读 (도표·그래프 읽기)

---

## 一、TOPIK II 图表题概述

图表题通常出现在：

- Q3~4 看图选句题
- 中后段的说明文中包含数据解读

常见图表类型：

| 类型   | 韩语                  | 常见内容 |
| ------ | --------------------- | -------- |
| 柱状图 | 막대그래프            | 数量比较 |
| 折线图 | 꺾은선 그래프         | 变化趋势 |
| 饼图   | 원형 그래프/파이 차트 | 比例构成 |
| 表格   | 표                    | 数据对比 |

| ----------- | -------- | ---- |
| 증가하다    | 增加     | ↑    |
| 늘어나다    | 增多     | ↑    |
| 상승하다    | 上升     | ↑    |
| 올라가다    | 上去     | ↑    |
| 높아지다    | 变高     | ↑    |
| 감소하다    | 减少     | ↓    |
| 줄어들다    | 减少     | ↓    |
| 하락하다    | 下降     | ↓    |
| 떨어지다    | 掉落     | ↓    |
| 낮아지다    | 变低     | ↓    |

## 三、图表描述句型

### 描述趋势

| 句型                         | 翻译        | 例                            |
| ---------------------------- | ----------- | ----------------------------- |
| ~(이)가 증가하는 추세이다    | …呈增长趋势 | 인구가 증가하는 추세이다      |
| ~(이)가 해마다 줄어들고 있다 | …逐年减少   | 출생률이 해마다 줄어들고 있다 |
| ~(이)가 꾸준히 상승하고 있다 | …持续上升   | 물가가 꾸준히 상승하고 있다   |
| ~(이)가 급격히 감소했다      | …急剧减少   | 매출이 급격히 감소했다        |

### 描述比较

| 句型                              | 翻译               |
| --------------------------------- | ------------------ |
| A가 B보다 ~배 높다                | A比B高~倍          |
| A의 비중이 가장 크다/작다         | A的比重最大/小     |
| A와 B의 격차가 점점 벌어지고 있다 | A和B的差距越来越大 |
| A와 B가 비슷한 수준이다           | A和B处于相似水平   |

### 描述极值

| 句型                                   | 翻译           |
| -------------------------------------- | -------------- |
| ~년에 최고/최저를 기록했다             | 在~年创最高/低 |
| ~(이)가 가장 높은/낮은 것으로 나타났다 | 显示~最高/低   |
| ~가 ~중에서 1위를 차지했다             | ~在~中排名第一 |

## 四、图表阅读实战 (6题)

### 练习1: 柱状图

```
[한국인이 좋아하는 운동] (응답자 500명)

축구 ████████████ 30%
농구 ██████ 15%
수영 ████████ 20%
야구 ██████████ 25%
기타 ████ 10%
```

**Q1. 위 그래프의 내용과 같은 것은?**
① 수영이 가장 인기가 많다
② 축구를 좋아하는 사람이 가장 많다
③ 야구보다 농구를 좋아하는 사람이 많다
④ 기타를 선택한 사람이 20%이다

**答案**: ②

> 축구 30%로 1위.

---

### 练习2: 折线图

```
[연도별 1인 가구 비율]

2010년: 23%
2015년: 27%
2020년: 31%
2023년: 35%
```

**Q2. 위 자료의 내용과 같은 것은?**
① 1인 가구 비율은 줄어들고 있다
② 2020년의 1인 가구 비율은 35%이다
③ 1인 가구 비율이 꾸준히 증가하고 있다
④ 2010년과 2023년의 차이는 5%이다

**答案**: ③

> 23→27→31→35% 꾸준히 증가.

---

### 练习3: 饼图

```
[대학생의 아르바이트 이유] (원형 그래프)

생활비 마련: 45%
경험 쌓기: 25%
용돈 벌기: 20%
기타: 10%
```

**Q3. 위 그래프의 내용과 같은 것은?**
① 경험을 쌓기 위해 아르바이트를 하는 학생이 가장 많다
② 생활비 마련이 아르바이트의 가장 큰 이유이다
③ 용돈을 벌기 위한 학생이 절반 이상이다
④ 기타 이유가 25%를 차지한다

**答案**: ②

> 생활비 마련 45%로 1위.

---

### 练习4: 表格

```
[국가별 연간 커피 소비량 (1인당, 잔)]

| 국가 | 2020년 | 2023년 |
|------|--------|--------|
| 한국 | 353    | 405    |
| 일본 | 280    | 310    |
| 미국 | 330    | 350    |
```

**Q4. 위 표의 내용과 같은 것은?**
① 일본의 커피 소비량이 가장 많다
② 2023년 기준 한국의 커피 소비량이 가장 많다
③ 미국의 커피 소비량은 감소했다
④ 한국의 2020년 소비량은 405잔이다

**答案**: ②

> 2023년 한국 405 > 미국 350 > 일본 310.

---

### 练习5: 数据解读 (文中图表)

조사에 따르면 직장인의 **62%**가 '워라밸이 연봉보다 중요하다'고 응답했다. 반면 '연봉이 더 중요하다'는 응답은 **28%**에 그쳤다. 나머지 **10%**는 '잘 모르겠다'고 답했다.

**Q5. 위 조사 결과와 같은 것은?**
① 연봉이 중요하다는 응답이 과반수이다
② 워라밸이 더 중요하다는 응답이 가장 많다
③ 잘 모르겠다는 응답이 28%이다
④ 워라밸보다 연봉을 선택한 사람이 더 많다

**答案**: ②

> 워라밸 62%로 1위. 과반수(50% 이상).

---

### 练习6: 종합

아래는 '한국인의 여가 활동' 조사 결과이다.

```
| 활동 | 20대 | 50대 |
|------|------|------|
| 운동 | 30%  | 45%  |
| 독서 | 15%  | 20%  |
| 게임 | 35%  | 5%   |
| TV   | 20%  | 30%  |
```

**Q6. 위 표의 내용과 같은 것은?**
① 20대는 운동을 가장 좋아한다
② 50대에서 가장 인기 있는 여가 활동은 게임이다
③ 20대와 50대 모두 게임을 가장 좋아한다
④ 50대는 운동을 가장 많이 하고, 20대는 게임을 가장 많이 한다

**答案**: ④

> 50대: 운동 45%(1위), 20대: 게임 35%(1위).

## 五、图表题常见陷阱

| 陷阱       | 例                 |
| ---------- | ------------------ |
| 数字替换   | 30% → 50%          |
| 年份替换   | 2020年 → 2023年    |
| 排名替换   | 1위 → 2위          |
| 趋势反转   | "증가" → "감소"    |
| 主体混淆   | A国数据安到B国上   |
| "가장"误用 | 不是最高却说"가장" |

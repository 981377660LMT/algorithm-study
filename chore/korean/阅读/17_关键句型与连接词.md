# 关键句型与连接词 (핵심 문형·접속 표현)

---

## 一、逻辑连接词大全

### 1. 因果关系 (인과)

| 韩语       | 翻译         | 用法                   |
| ---------- | ------------ | ---------------------- |
| 그래서     | 所以         | 원인 뒤에 결과 연결    |
| 따라서     | 因此         | 공식적 글에 사용       |
| 그러므로   | 因此/故      | 格式体                 |
| 때문에     | 因为         | N+때문에 / V-기 때문에 |
| 덕분에     | 多亏         | 긍정적 결과            |
| 탓에       | 怪/由于      | 부정적 결과            |
| ~(으)므로  | 因为         | 書面体                 |
| ~는 바람에 | 因为(非意愿) | 부정적/의외의 결과     |
| 그 결과    | 其结果       | 접속부사               |
| 이에 따라  | 随之         | 공식 문서              |

### 2. 转折关系 (전환·대조)

| 韩语                | 翻译      | 用法                 |
| ------------------- | --------- | -------------------- |
| 그러나              | 但是      | 格式体               |
| 하지만              | 但是      | 一般体               |
| 그렇지만            | 但是/不过 | 일상                 |
| 그런데              | 可是/不过 | 话题转换+转折        |
| 반면(에)            | 反之      | 对比两事物           |
| 반대로              | 相反地    | 완전한 반대          |
| 한편                | 另一方面  | 새 관점 도입         |
| ~(으)나             | 但        | 書面체, V/A+(으)나   |
| ~지만               | 虽然…但   | V/A+지만             |
| ~(으)ㄴ/는 반면(에) | 相反      | A+(으)ㄴ/V+는 반면에 |
| 물론 ~지만          | 当然…但   | 양보+전환            |
| ~에도 불구하고      | 尽管      | N+에도 불구하고      |
| 그럼에도 불구하고   | 尽管如此  | 접속부사             |

### 3. 递进关系 (첨가·나열)

| 韩语                | 翻译      | 用法             |
| ------------------- | --------- | ---------------- |
| 그리고              | 而且/和   | 가장 기본        |
| 또한                | 而且/此外 | 격식체           |
| 게다가              | 不仅如此  | 추가 강조        |
| 뿐만 아니라         | 不仅      | ~뿐만 아니라 ~도 |
| 더불어              | 同时/加上 | 공식             |
| 아울러              | 同时      | 공문서           |
| 더 나아가           | 进一步    | 논설             |
| ~(으)ㄹ 뿐만 아니라 | 不仅…     | 뿐만 아니라, ~도 |

### 4. 举例 (예시)

| 韩语            | 翻译      |
| --------------- | --------- |
| 예를 들어       | 例如      |

| 구체적으로      | 具体来说  |
| 이를테면        | 比方说    |
| ~는 경우가 있다 | 有…的情况 |

### 5. 总结·概括 (정리·요약)

### 6. 条件·假设 (조건·가정)

## 二、文章中的关键句型

### 主题句常见句型

| 句型                          | 翻译            | 例                                  |
| ----------------------------- | --------------- | ----------------------------------- |
| ~에 대해 살펴보겠다           | 来看看关于…     | 오늘은 환경 문제에 대해 살펴보겠다. |
| ~에 대한 관심이 높아지고 있다 | 对…的关注在增加 | 건강에 대한 관심이 높아지고 있다.   |
| ~가 문제가 되고 있다          | …成为问题       | 저출산이 문제가 되고 있다.          |
| ~(으)ㄹ 필요가 있다           | 有…的必要       | 교육을 개선할 필요가 있다.          |

### 主张句常见句型

| 句型                       | 翻译             |
| -------------------------- | ---------------- |
| ~아/어야 한다              | 应该             |
| ~는 것이 중요하다/필요하다 | …是重要的/必要的 |
| ~는 것이 바람직하다        | …是可取的        |
| ~해야 할 것이다            | 应当…            |
| ~지 않으면 안 된다         | 不…不行          |
| ~(으)ㄹ 수밖에 없다        | 只能/不得不      |

### 定义句常见句型

| 句型                      | 翻译     | 例                                                             |
| ------------------------- | -------- | -------------------------------------------------------------- |
| ~(이)란 ~(을/를) 말한다   | …是指    | 인플레이션이란 물가가 지속적으로 오르는 현상을 말한다.         |
| ~(이)란 ~(을/를) 뜻한다   | …意味着  | 지속 가능하다는 것은 미래 세대도 사용할 수 있다는 것을 뜻한다. |
| ~(이)란 ~(을/를) 의미한다 | …意味着  |                                                                |
| ~(으)로 정의된다          | 被定义为 |                                                                |

### 数据引用句型

| 句型               | 翻译       |
| ------------------ | ---------- |
| 조사에 따르면      | 据调查     |
| 통계에 의하면      | 据统计     |
| 연구 결과에 따르면 | 据研究结果 |
| ~것으로 나타났다   | 显示为…    |
| ~것으로 밝혀졌다   | 被证明为…  |
| ~것으로 조사되었다 | 被调查为…  |
| ~것으로 추정된다   | 被推测为…  |

## 三、指示词指代关系

| 指示词     | 指向位置   | 功能               |
| ---------- | ---------- | ------------------ |
| 이것 / 이  | 紧邻前文   | 指代刚刚提到的事物 |
| 그것 / 그  | 前文稍远处 | 指代之前提到的事物 |
| 이러한 + N | 前文整体   | 概括前面的内容     |
| 그러한 + N | 前文整体   | 概括之前的内容     |
| 이에       | 前文事态   | "对此"             |
| 이를       | 前文事物   | "把这个" (宾格)    |

### 指代链判断练习

원거리 학습이 확산되면서 학생들의 자기 주도 학습 능력이 중요해졌다. **이러한 변화**에 대응하기 위해 학교에서는 새로운 교육 방법을 도입하고 있다. **이에** 학생들도 적극적으로 참여해야 할 것이다.

| 指示语      | 指代                                   |
| ----------- | -------------------------------------- |
| 이러한 변화 | 원거리 학습 확산 + 자기 주도 학습 중요 |
| 이에        | 학교가 새로운 교육 방법을 도입한 것    |

## 四、对比·并列结构

| 结构                            | 翻译             |
| ------------------------------- | ---------------- |
| A는 ~인 반면(에) B는 ~이다      | A…而B…           |
| A와 달리 B는 ~이다              | 与A不同，B…      |
| A뿐만 아니라 B도 ~이다          | 不仅A，B也…      |
| A는 물론이고 B까지              | 不用说A，连B都…  |
| A와 B의 공통점은 ~              | A和B的共同点是   |
| A와 B의 차이점은 ~              | A和B的差异是     |
| 한편으로는 ~, 다른 한편으로는 ~ | 一方面…另一方面… |

## 五、阅读中常见「陷阱句型」

| 出现在选项中       | 可能是陷阱                  |
| ------------------ | --------------------------- |
| 항상/언제나/반드시 | 绝对词 → 多半不选           |
| 모든/전부/전혀     | 极端词 → 仔细核对           |
| ~기만 하면 된다    | "只要…就行" → 过于简单化    |
| ~와/과 관계없다    | "与…无关" → 多半是有关的    |
| ~(으)ㄹ 수 없다    | "不能" → 原文可能说的是"难" |

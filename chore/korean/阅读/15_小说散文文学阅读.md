# 小说散文文学阅读 (소설·수필 문학 읽기)

---

## 一、TOPIK II 文学题出题概况

考试最后 2~4 题(约 Q47~50)通常为文学类长文，难度 ★★★★★。

| 体裁        | 特征                       | 常见提问                          |
| ----------- | -------------------------- | --------------------------------- |
| 소설 (小说) | 인물/사건/배경, 대화, 묘사 | 인물의 심정, 밑줄 의미, 내용 일치 |
| 수필 (散文) | 개인 경험, 감정, 깨달음    | 글쓴이의 태도, 밑줄 의미, 주제    |

## 二、文学阅读核心要素

### 1. 인물 심정 (人物心情)

#### 积极心情

| 韩语       | 翻译       | 常见场景          |
| ---------- | ---------- | ----------------- |
| 기쁘다     | 高兴       | 좋은 소식, 성공   |
| 뿌듯하다   | 感到欣慰   | 성취, 자녀 성장   |
| 설레다     | 激动期待   | 첫 경험, 만남     |
| 감격스럽다 | 感动       | 재회, 감동적 장면 |
| 홀가분하다 | 轻松       | 짐을 내려놓음     |
| 후련하다   | 痛快/释然  | 오래된 문제 해결  |
| 든든하다   | 心里踏实   | 의지할 곳이 있음  |
| 벅차다     | 激动到满溢 | 감격+기쁨         |

| 서글프다 | 凄凉          | 외로움, 이별       |
| 허전하다 | 空虚/若有所失 | 이별, 상실         |
| 한스럽다 | 抱恨/遗恨     | 과거 후회          |

| 초조하다 | 焦躁          | 결과 기다림        |

### 2. 밑줄 친 부분의 의미 (画线部分的意思)

常见模式：

| 类型 | 说明         | 例                            |
| ---- | ------------ | ----------------------------- |
| 比喻 | 用A形容B     | "마음에 비가 내리다" = 슬프다 |
| 象征 | 事物代表含义 | "겨울" = 시련, "봄" = 희망    |
| 反语 | 说反话       | "잘하셨네요" (실제로는 비꼼)  |
| 夸张 | 程度放大     | "세상이 무너졌다" = 큰 충격   |
| 对比 | 通过反差凸显 | 밝은 기억 vs 어두운 현실      |
| 委婉 | 间接表达     | "떠나다" = 죽다               |

---

### 3. 서술 방식 (叙述方式)

| 韩语 | 翻译 | 特征                        |
| ---- | ---- | --------------------------- |
| 회상 | 回忆 | 과거 이야기를 현재 시점에서 |
| 묘사 | 描写 | 인물/풍경의 세부적 표현     |
| 대화 | 对话 | 인물 간 대화로 전개         |
| 독백 | 独白 | 인물의 내면 생각            |
| 관찰 | 观察 | 제3자 시점으로 서술         |
| 대비 | 对比 | 두 대상을 비교              |
| 반어 | 反语 | 진짜 의미와 반대되는 표현   |
| 풍자 | 讽刺 | 사회/인물 비판              |

## 三、文学阅读实战 (5篇)

### 소설 1

어머니는 항상 새벽에 일어나셨다. 아직 어둑한 부엌에서 된장찌개를 끓이는 소리가 들리면 나는 이불 속에서 고개를 돌려 어머니의 뒷모습을 바라보곤 했다. 어머니의 어깨는 언제부터인가 점점 좁아지고 있었다. 나는 그 모습을 보면서 ⓐ **마음 한쪽이 무거워지는 것을 느꼈다.** 학교를 졸업하고 도시로 떠나온 뒤에야 그때의 그 된장찌개 냄새가 세상에서 가장 그리운 것이 되었다는 걸 알았다.

**Q1. ⓐ의 의미로 가장 알맞은 것은?**
① 어머니의 음식이 맛이 없어서 속이 답답했다
② 어머니의 나이 드신 모습에 안타까움과 미안함을 느꼈다
③ 새벽에 깨서 짜증이 났다
④ 학교에 가기 싫어서 우울했다

**答案**: ②

> "마음 한쪽이 무거워지는 것" = 沉重感 → 看到母亲衰老的背影感到心痛和愧疚.

**Q2. 이 글의 내용과 같은 것은?**
① 글쓴이는 현재 고향에 살고 있다
② 어머니는 늦게 일어나셨다
③ 글쓴이는 도시로 떠난 뒤 어머니가 해 주신 음식을 그리워했다
④ 어머니의 어깨가 점점 넓어지고 있었다

**答案**: ③

---

### 소설 2

민수는 면접 결과를 기다리며 하루하루를 보냈다. 합격 발표일 아침, 손이 떨려 스마트폰을 제대로 잡지도 못했다. 화면에 '합격'이라는 글자가 뜨는 순간, 민수는 한동안 멍하니 화면만 바라보았다. 그리고 천천히 눈물이 흘러내렸다. ⓐ **반년 동안의 긴 겨울이 끝나고 드디어 봄이 온 것 같았다.**

**Q3. ⓐ에 나타난 민수의 심정은?**
① 합격했지만 기쁘지 않다
② 오랜 고생 끝에 벅찬 감격을 느끼고 있다
③ 불합격에 절망하고 있다
④ 새로운 회사에 대한 걱정이 크다

**答案**: ②

> "긴 겨울이 끝나고 봄이 온 것 같다" → 겨울=고생, 봄=합격의 기쁨. 벅찬 감격.

**Q4. ⓐ에서 '긴 겨울'이 의미하는 것은?**
① 추운 날씨
② 오랜 준비 기간의 힘든 시간
③ 면접 장소가 추웠던 것
④ 결과가 겨울에 나온 것

**答案**: ②

---

### 수필 1

나는 오래된 사진 한 장을 서랍에서 꺼냈다. 초등학교 졸업식 날, 친구들과 함께 찍은 사진이었다. 사진 속의 우리는 모두 해맑게 웃고 있었다. 지금은 어디에서 무엇을 하고 있을지 모를 그 친구들의 얼굴을 하나하나 바라보다가, 문득 그때로 돌아갈 수만 있다면 하는 생각이 들었다. 세월은 이렇게 소리 없이 흘러 다시는 돌아오지 않는 것이다.

**Q5. 글쓴이의 심정은?**
① 그리움과 아쉬움
② 분노와 실망
③ 기대와 설렘
④ 감사와 기쁨

**答案**: ①

> "돌아갈 수만 있다면", "다시는 돌아오지 않는" → 그리움, 아쉬움.

---

### 수필 2

정년퇴직을 한 지 한 달이 지났다. 30년 동안 다니던 회사를 떠나니 매일 아침 빈 손으로 거실에 앉아 있는 자신이 낯설었다. 가족들은 이제 쉬라고 했지만, ⓐ **벽시계의 초침 소리가 이렇게 크게 들린 적이 없었다.** 어느 날 아내가 동네 도서관에서 자원봉사를 해 보라고 권했다. 처음 가는 날은 어색했지만, 아이들에게 동화책을 읽어 주는 동안 오랜만에 가슴이 따뜻해지는 것을 느꼈다.

**Q6. ⓐ의 의미는?**
① 벽시계가 고장이 나서 소리가 커졌다
② 시간이 느리게 흘러 무료하고 공허한 마음을 의미한다
③ 퇴직 후 귀가 밝아졌다는 것이다
④ 집이 조용해서 좋다는 것이다

**答案**: ②

> 은퇴 후 할 일이 없어 시간이 천천히 흐르는 느낌 → 무료함, 허전함의 비유.

---

### 소설 3

할아버지는 매일 저녁 마당에 나와 하늘의 별을 올려다보셨다. 할머니가 돌아가신 뒤부터였다. 누군가 왜 별을 보냐고 물으면 할아버지는 웃으며 말씀하셨다. "이 별들 중에 할머니 별이 있을 거야." 나는 어릴 때 그 말을 정말로 믿었다. 지금은 알고 있다. 할아버지가 별을 보고 있었던 것이 아니라, ⓐ **별 너머의 사람을 그리워하고 있었다는 것을.**

**Q7. ⓐ의 의미로 알맞은 것은?**
① 할아버지는 천문학에 관심이 있었다
② 할아버지는 돌아가신 할머니를 그리워하고 있었다
③ 할아버지는 다른 가족을 기다리고 있었다
④ 별이 할머니가 되었다고 믿었다

**答案**: ②

> "별 너머의 사람" = 할머니. 별을 通해 할머니를 그리워하는 것.

**Q8. 이 글의 내용과 같은 것은?**
① 할아버지는 할머니가 돌아가시기 전부터 별을 보셨다
② 글쓴이는 어릴 때 할아버지의 말을 믿었다
③ 할아버지는 별을 보며 울었다
④ 할머니는 아직 살아 계신다

**答案**: ②

## 四、文学类解题口诀

| 순서 | 행동                                             |
| ---- | ------------------------------------------------ |
| 1    | 밑줄 부분 → **비유인가? 상징인가?** 즉시 판단    |
| 2    | 인물 감정 → **행동(울다/웃다/떨다) + 묘사 단서** |
| 3    | 전후 문맥 → 밑줄의 **앞 2문장 + 뒤 1문장** 집중  |
| 4    | 선택지 대조 → 극단적 감정(분노/절망) 은 보통 ✗   |

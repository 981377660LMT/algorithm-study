# 广告通知类 (광고·안내문) — 1~2题

---

## 一、题型说明

给出一段广告/通知/邀请函，选择"이 글은 무엇에 대한 글입니까?"(这篇文章是关于什么的？)

---

## 二、常见广告类型与核心词汇

### 1. 모집 (招募)

| 词汇      | 意思      |
| --------- | --------- |
| 모집하다  | 招募      |
| 지원하다  | 申请      |
| 접수하다  | 受理      |
| 자격/조건 | 资格/条件 |
| 모집 기간 | 招募期间  |
| 모집 인원 | 招募人数  |
| ~명       | ~名       |
| 지원 방법 | 申请方法  |
| 서류      | 材料/文件 |
| 면접      | 面试      |

**典型格式**:

```
[OO 모집 안내]
모집 기간: 2026.4.1~4.30
모집 인원: 00명
지원 자격: ...
지원 방법: 홈페이지에서 접수
문의: 02-XXX-XXXX
```

### 2. 행사/이벤트 (活动)

| 词汇          | 意思          |
| ------------- | ------------- |
| 행사          | 活动          |
| 축제          | 庆典          |
| 전시회        | 展览会        |
| 공연          | 演出          |
| 콘서트        | 音乐会        |
| 강연/강좌     | 讲座          |
| 체험          | 体验          |
| 참가비/입장료 | 参加费/入场费 |
| 무료/유료     | 免费/收费     |
| 사전 신청     | 提前申请      |
| 선착순        | 先到先得      |

**典型格式**:

```
[제5회 OO 축제]
일시: 2026년 5월 3일(토) 10:00~17:00
장소: OO공원
내용: 공연, 체험, 전시
참가비: 무료 (사전 신청 필요)
```

### 3. 수강/교육 (课程/教育)

| 词汇        | 意思     |
| ----------- | -------- |
| 수강생 모집 | 招收学员 |
| 강좌        | 课程     |
| 수업료      | 学费     |
| 교재비      | 教材费   |
| 수료증      | 结业证   |
| 개강일      | 开课日   |
| 수강 기간   | 上课期间 |
| 수강 신청   | 课程申请 |
| 정원        | 名额     |

### 4. 할인/판매 (打折/销售)

| 词汇          | 意思      |
| ------------- | --------- |
| 세일/할인     | 打折      |
| ~% 할인       | 打~折     |
| 특가          | 特价      |
| 무료 배송     | 免费配送  |
| 구매하다/사다 | 购买      |
| 1+1           | 买一送一  |
| 한정 판매     | 限量销售  |
| 사은품        | 赠品      |
| 쿠폰          | 优惠券    |
| 교환/환불     | 换货/退款 |

### 5. 안내/공지 (通知/公告)

| 词汇                   | 意思                 |
| ---------------------- | -------------------- |
| 공지 사항              | 公告事项             |
| 안내 말씀 드립니다     | 通知您               |
| 변경/변경되다          | 变更                 |
| 이전하다               | 搬迁                 |
| 휴무/휴관              | 休息/休馆            |
| 점검/수리              | 检查/修理            |
| 양해 부탁드립니다      | 请谅解               |
| 불편을 드려 죄송합니다 | 给您带来不便深表歉意 |

### 6. 초대 (邀请)

| 词汇          | 意思        |
| ------------- | ----------- |
| 초대합니다    | 诚邀        |
| 초대장        | 邀请函      |
| 참석하다      | 出席        |
| 축하하다      | 祝贺        |
| 결혼식/돌잔치 | 婚礼/周岁宴 |
| 개업식        | 开业典礼    |
| 참석 여부     | 是否出席    |

---

## 三、解题步骤

```
Step 1: 看标题/大字 → 确定文体(广告? 通知? 邀请?)
Step 2: 扫关键名词 → 핵심 주제어
Step 3: 对照选项 → 选最匹配的主题
```

## 四、实战练习

### 练习1

```
[한국어 말하기 대회 참가자 모집]
일시: 2026년 6월 15일(토) 14:00
장소: 서울대학교 대강당
참가 자격: 한국어를 배우는 외국인
신청 기간: 5월 1일~5월 31일
신청 방법: 홈페이지(www.xxx.kr)에서 접수
상금: 1등 100만 원
문의: 02-880-XXXX
```

**이 글은 무엇에 대한 글입니까?**
① 한국어 교재 판매
② 한국어 강좌 안내
③ 말하기 대회 안내
④ 유학 상담 안내

**答案**: ③
**解析**: 标题"말하기 대회 참가자 모집" → 口语大赛招募参赛者

### 练习2

```
알려 드립니다
5월 2일(금)~5월 4일(일)은 어린이날 연휴로
도서관을 운영하지 않습니다.
5월 5일(월)부터 정상 운영합니다.
이용에 불편을 드려 죄송합니다.
- 서울시립도서관 -
```

**이 글은 무엇에 대한 글입니까?**
① 도서관 이전 안내
② 도서관 휴관 안내
③ 도서관 행사 안내
④ 도서관 회원 모집

**答案**: ②
**解析**: "운영하지 않습니다" = 不运营 → 休馆通知

### 练习3

```
봄맞이 가전 세일!
냉장고·세탁기·에어컨 최대 40% 할인
기간: 3월 15일~3월 31일
매장 방문 고객에게 사은품 증정!
무이자 할부 12개월 가능
※ 온라인 주문 시 무료 배송
```

**이 글은 무엇에 대한 글입니까?**
① 가전제품 수리 안내
② 가전제품 할인 행사
③ 가전제품 사용 방법
④ 가전제품 교환 안내

**答案**: ②
**解析**: "세일", "40% 할인" = 打折活动

### 练习4

```
제10회 서울 국제 사진 전시회
세계 각국 사진작가 200명의 작품을 만나 보세요!
일시: 2026년 7월 1일~7월 20일 (10:00~18:00)
장소: 서울시립미술관
입장료: 성인 10,000원 / 학생 5,000원
문의: 02-XXX-XXXX
```

**이 글은 무엇에 대한 글입니까?**
① 사진 강좌 모집
② 사진 전시회 안내
③ 미술관 이전 안내
④ 사진 공모전 안내

**答案**: ②

### 练习5

```
행복한 우리 아이 돌잔치에 초대합니다
아이 이름: 김하은
일시: 2026년 4월 20일(일) 12:00
장소: 그랜드호텔 3층 연회장
오셔서 축하해 주시면 감사하겠습니다.
참석 여부를 4월 15일까지 알려 주세요.
```

**이 글은 무엇에 대한 글입니까?**
① 아이 생일 파티 초대
② 결혼식 초대
③ 돌잔치 초대
④ 회사 행사 안내

**答案**: ③
**解析**: "돌잔치" = 周岁宴

## 五、实用文高频表达总结

| 表达                        | 意思           | 语境 |
| --------------------------- | -------------- | ---- |
| ~을/를 모집합니다           | 招募…          | 招募 |
| ~에 참가/참석해 주세요      | 请参加/出席    | 活动 |
| ~까지 신청하시기 바랍니다   | 请在…之前申请  | 申请 |
| 선착순 ~명                  | 先到先得~名    | 限额 |
| 문의:                       | 咨询:          | 联系 |
| ~부터 ~까지                 | 从…到…         | 时间 |
| 많은 관심 부탁드립니다      | 请多多关注     | 宣传 |
| 자세한 내용은 홈페이지 참조 | 详情请参考官网 | 引导 |

# 论述文深度阅读 (논설문 깊이 읽기)

---

## 一、论述文在 TOPIK II 中的地位

论述文(논설문)在 TOPIK II 阅读后半段(约 Q43~48)频繁出现，难度 ★★★★★。

| 特征     | 说明                               |
| -------- | ---------------------------------- |
| 篇幅     | 通常 200~350 字                    |
| 结构     | 问题→论点→论据→结论                |
| 常见主题 | 社会、科技、环境、教育、经济、文化 |
| 考查方式 | 주장 파악, 빈칸, 내용 일치, 태도   |

---

## 二、论述文结构分析

### 模式1：주장 → 근거

```
[주장 제시] → [근거 1] → [근거 2] → [재강조/결론]
```

### 模式2：문제 → 원인 → 주장

```
[문제 제기] → [원인 분석] → [해결 주장] → [기대 효과]
```

### 模式3：양면 분석 → 입장

```
[찬성 의견] → [반대 의견] → [글쓴이 입장] → [근거]
```

### 模式4：현상 → 비판

```
[현상 소개] → [문제점 지적] → [비판적 주장]
```

## 三、论述文核心词汇

### 主张类

| 韩语                | 翻译      |
| ------------------- | --------- |
| 주장하다            | 主张      |
| ~아/어야 한다       | 应该      |
| ~는 것이 필요하다   | 有…的必要 |
| ~는 것이 바람직하다 | …是可取的 |
| ~할 필요가 있다     | 有必要    |
| ~하는 것이 중요하다 | …是重要的 |
| ~해야 할 것이다     | 应当      |
| ~ 않으면 안 된다    | 不…不可   |

### 论据类

| 韩语                    | 翻译             |
| ----------------------- | ---------------- |
| 왜냐하면 ~기 때문이다   | 因为             |
| 그 이유는 ~에 있다      | 原因在于         |
| 연구 결과에 따르면      | 据研究结果       |
| 통계에 의하면           | 据统计           |
| 전문가들은 ~고 지적한다 | 专家指出         |
| 실험을 통해 밝혀졌다    | 通过实验得以证明 |
| 이를 뒷받침하는 사례로  | 作为支撑的案例   |
| 사례를 보면             | 看案例           |

### 转折/让步

| 韩语                      | 翻译            |
| ------------------------- | --------------- |
| 물론 ~지만                | 当然…但是       |
| ~다는 의견도 있지만       | 虽然也有…的意见 |
| 일부에서는 ~다고 하지만   | 一些人说…但     |
| 그럼에도 불구하고         | 尽管如此        |
| 그러나 이는 사실과 다르다 | 但这与事实不符  |

### 结论类

| 韩语                          | 翻译         |
| ----------------------------- | ------------ |
| 결론적으로                    | 综上所述     |
| 따라서                        | 因此         |
| 이러한 점에서                 | 从这一点来看 |
| 앞으로 ~해 나가야 한다        | 今后应当     |
| ~에 더욱 관심을 기울여야 한다 | 应更加关注   |

## 四、论述文阅读实战 (6篇)

### 论述文1：교육

최근 학교에서 코딩 교육을 의무화해야 한다는 목소리가 높아지고 있다. 4차 산업혁명 시대에 코딩 능력은 단순한 기술이 아니라 논리적 사고를 키우는 도구이기 때문이다. 실제로 코딩 교육을 도입한 학교에서 학생들의 문제 해결 능력이 향상되었다는 연구 결과도 있다. 물론 교사 인력이 부족하고 교육 인프라가 미흡하다는 문제가 있지만, 이는 단계적으로 해결해 나갈 수 있는 부분이다. 미래 세대를 위해 코딩 교육의 확대는 선택이 아닌 필수이다.

**Q1. 글쓴이의 주장은?**
① 코딩 교육은 시기상조이다
② 코딩 교육을 확대해야 한다
③ 교사 인력 문제를 먼저 해결해야 한다
④ 코딩보다 수학 교육이 더 중요하다

**答案**: ②

**Q2. 글쓴이가 코딩 교육을 주장하는 근거로 맞지 않는 것은?**
① 논리적 사고를 키울 수 있다
② 문제 해결 능력이 향상된다는 연구가 있다
③ 4차 산업혁명 시대에 필요한 능력이다
④ 코딩 교육을 하면 취업률이 높아진다

**答案**: ④

> 본문에 취업률에 대한 언급 없음.

### 论述文2：사회

한국 사회에서 '워라밸(Work-Life Balance)'에 대한 관심이 높아지고 있다. 과거에는 장시간 근무를 미덕으로 여기는 분위기가 있었지만, 이제는 개인의 삶의 질을 중시하는 문화로 변화하고 있다. 정부는 주 52시간 근무제를 도입하여 근로자의 과로를 줄이고자 했다. 그러나 일부에서는 이 제도가 소득 감소로 이어질 수 있다며 반대하고 있다. 그럼에도 불구하고 장기적으로 볼 때 워라밸은 근로자의 건강과 생산성을 모두 높이는 방향으로 나아가야 한다.

**Q3. 글쓴이의 태도는?**
① 워라밸에 대해 부정적이다
② 주 52시간 근무제에 반대하고 있다
③ 워라밸의 필요성을 긍정하면서도 균형을 강조하고 있다
④ 장시간 근무가 미덕이라고 주장하고 있다

**答案**: ③

> "장기적으로 볼 때 ~건강과 생산성을 모두 높이는 방향" → 긍정 + 균형.

---

### 论述文3：환경

일회용 플라스틱의 사용을 줄이려는 노력이 세계적으로 확산되고 있다. 하지만 일회용품을 완전히 없애는 것은 현실적으로 어렵다. 의료 분야에서 일회용 주사기나 장갑 등은 위생상 필수적이며, 식품 포장에서도 일회용품이 필요한 경우가 있다. 따라서 무조건적인 금지보다는 ( ) 예를 들어 생분해성 소재를 활용한 일회용품 개발이나, 재활용 시스템의 효율화 등이 이에 해당한다.

**Q4. ( )에 들어갈 내용은?**
① 일회용품 사용을 더 늘리는 것이 좋다
② 일회용품을 대체할 수 있는 지속 가능한 방안을 찾는 것이 중요하다
③ 의료 분야에서만 일회용품을 사용해야 한다
④ 플라스틱의 장점을 적극 활용해야 한다

**答案**: ②

> "따라서 무조건적인 금지보다는" → 대안 제시. 뒤에 "생분해성 소재, 재활용" 등이 힌트.

---

### 论述文4：문화

전통 문화를 보존하는 것은 중요하지만, 현대 사회에 맞게 변화시키는 것도 필요하다. 한복을 예로 들면, 전통 한복은 생활하기 불편하다는 이유로 일상에서 거의 입지 않게 되었다. 하지만 최근 젊은 디자이너들이 한복의 전통적 아름다움을 살리면서도 실용성을 더한 '생활 한복'을 만들면서 다시 관심을 받고 있다. 이처럼 전통 문화에 현대적 감각을 더하는 것이 전통을 지키는 가장 좋은 방법이 될 수 있다.

**Q5. 글쓴이의 주장은?**
① 전통 문화를 원래 그대로 보존해야 한다
② 전통 한복은 불편하므로 입지 말아야 한다
③ 전통에 현대적 요소를 더해 계승하는 것이 바람직하다
④ 생활 한복은 전통 한복이 아니다

**答案**: ③

---

### 论述文5：경제

최저임금 인상에 대한 논쟁이 계속되고 있다. 인상을 찬성하는 측은 저소득 노동자의 생활 안정과 소비 증가에 도움이 된다고 주장한다. 반면 반대하는 측은 기업의 인건비 부담이 늘어나 오히려 고용이 줄어들 수 있다고 우려한다. 양측의 주장 모두 일리가 있으므로, 최저임금 결정 시에는 경제 상황과 중소기업의 여건을 종합적으로 고려하여 신중하게 결정해야 할 것이다.

**Q6. 글쓴이의 태도는?**
① 최저임금 인상에 찬성하고 있다
② 최저임금 인상에 반대하고 있다
③ 최저임금을 폐지해야 한다고 주장하고 있다
④ 양측 의견을 고려한 신중한 결정을 촉구하고 있다

**答案**: ④

### 论述文6：과학 기술

유전자 편집 기술(CRISPR)은 질병 치료에 혁명적인 변화를 가져올 것으로 기대되고 있다. 실제로 이 기술을 활용하면 유전 질환을 미리 예방하거나 치료할 수 있다. 그러나 유전자를 인위적으로 조작하는 것은 윤리적 문제를 야기할 수 있다. 특히 '맞춤형 아기'와 같은 비의료적 목적의 유전자 편집은 사회적 불평등을 심화시킬 우려가 있다. 따라서 유전자 편집 기술의 의료적 활용은 적극적으로 추진하되, 비의료적 활용에 대해서는 엄격한 규제와 사회적 합의가 필요하다.

**Q7. 글쓴이의 주장으로 알맞은 것은?**
① 유전자 편집 기술을 전면 금지해야 한다
② 유전자 편집은 윤리적 문제가 없다
③ 의료 목적의 활용은 추진하고 비의료 목적은 엄격히 규제해야 한다
④ 맞춤형 아기도 허용해야 한다

**答案**: ③

## 五、论述文读后策略

| 读完后确认              | 方法                             |
| ----------------------- | -------------------------------- |
| 주장은 무엇인가?        | 마지막 1~2문장에 주목            |
| 근거는 몇 개? 무엇?     | 중간 부분에서 찾기               |
| 양보/반론이 있는가?     | "물론, 일부에서는" 확인          |
| 글쓴이의 진정한 입장은? | 양보 뒤의 "그러나/그럼에도" 주목 |
| 빈칸은 어떤 역할?       | 주장/전환/총결인지 판단          |

## 六、옳고 그름 판단 핵심

| 함정 유형          | 설명                      | 대처법                              |
| ------------------ | ------------------------- | ----------------------------------- |
| 과도한 일반화      | "모든/항상/전혀"          | 원문에 이런 극단 표현이 있는지 확인 |
| 주장 ≠ 근거 혼동   | 근거를 주장으로 착각      | 마지막 문장 = 주장인 경우 많음      |
| 양보문을 주장으로  | "물론~"의 내용을 선택     | "그러나" 뒤가 진짜 주장             |
| 언급되지 않은 내용 | 선택지에 원문에 없는 내용 | 반드시 원문에서 근거 찾기           |

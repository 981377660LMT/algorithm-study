# 句子位置插入类（高级）(문장이 들어갈 위치 찾기)

---

## 一、题型说明

> 다음을 읽고 ( ) 에 들어갈 내용으로 가장 알맞은 것을 고르십시오.
> 或 다음 글에서 &lt;보기&gt;의 문장이 들어가기에 가장 적절한 곳을 고르십시오.

高级阅读(Q39~44)中的句子插入题比中级更复杂，段落更长，逻辑更隐蔽。

---

## 二、核心解题策略

### 策略1：分析插入句的逻辑角色

| 角色 | 标志                      | 例             |
| ---- | ------------------------- | -------------- |
| 转折 | 그러나, 하지만, 반면에    | 前后意思相反   |
| 因果 | 따라서, 그래서, 그 결과   | 紧跟原因之后   |
| 举例 | 예를 들어, 가령           | 紧跟概括之后   |
| 补充 | 게다가, 뿐만 아니라, 또한 | 同一方向追加   |
| 总结 | 이처럼, 이렇듯, 결국, 즉  | 放在论述之后   |
| 换言 | 즉, 다시 말하면           | 解释前面的难句 |

### 策略2：寻找「指示词锚点」

| 指示词      | 指向           | 位置规则             |
| ----------- | -------------- | -------------------- |
| 이것/이러한 | 紧接前文的名词 | 必须在被指代对象之后 |
| 그것/그러한 | 稍远的前文     | 与前文有一定距离     |
| 이런 현상   | 前面描述的现象 | 先有现象，后出现指示 |

### 策略3：代词 · 주어 一致性

插入句的主语通常与前后文保持一致。

| 检查项       | 说明                               |
| ------------ | ---------------------------------- |
| 主语是否一致 | 插入后，该句主语要能与前后自然衔接 |
| 时态是否一致 | 过去/现在/将来要匹配上下文         |
| 话题是否跳跃 | 插入后不能产生话题突变             |

---

## 三、段落结构模式

### 模式A：总→分→总

```
[주제 제시] → [세부 설명 1] → [세부 설명 2] → [결론]
```

插入句往往：

- 例句 → 放在具体描述之间
- 总结句 → 放在最后

### 模式B：问题→原因→解决

```
[문제 제기] → [원인 분석] → [해결 방안]
```

插入句往往：

- 原因补充 → 放在原因之后
- 解决方案 → 放在原因之后、结论之前

### 模式C：现象→解释→评价

```
[현상 소개] → [구체적 설명] → [평가/전망]
```

## 四、实战练习 (8题)

### 练习1

( ㉠ ) 사람들은 흔히 성공한 사람들이 특별한 재능을 가지고 있다고 생각한다. ( ㉡ ) 그러나 많은 연구에 따르면 성공의 가장 큰 요인은 재능이 아니라 꾸준한 노력이다. ( ㉢ ) 실제로 세계적인 음악가들도 하루에 최소 4시간 이상 연습을 한다고 한다. ( ㉣ ) 따라서 무엇이든 포기하지 않고 꾸준히 하는 것이 가장 중요하다.

**&lt;보기&gt;**: 즉, 타고난 능력보다 얼마나 오래 노력하느냐가 더 중요하다는 것이다.

**위치는?**
① ㉠ ② ㉡ ③ ㉢ ④ ㉣

**答案**: ③

> 즉(换言之) 是对 ㉡㉢ 内容的概括，放在 ㉢ 之后、结论 ㉣ 之前最自然。

---

### 练习2

( ㉠ ) 한국어에는 높임법이 발달해 있어 상대방의 나이나 지위에 따라 말하는 방식이 달라진다. ( ㉡ ) 이러한 높임법은 한국 사회의 서열 문화와 밀접한 관련이 있다. ( ㉢ ) 최근에는 직장에서도 수평적 호칭을 사용하는 기업이 늘고 있다. ( ㉣ ) 이는 조직 문화가 변화하고 있음을 보여 주는 것이다.

**&lt;보기&gt;**: 하지만 시대가 변하면서 이런 문화에도 변화가 나타나고 있다.

**위치는?**
① ㉠ ② ㉡ ③ ㉢ ④ ㉣

**答案**: ②

> "하지만(但是)"是转折，㉠㉡讲传统文化，㉢㉣讲变化。插入句是过渡句，放在 ㉡ 后引出变化。

---

### 练习3

( ㉠ ) 도시에서 생활하다 보면 자연을 접할 기회가 적다. ( ㉡ ) 그래서 최근 도시 농업에 대한 관심이 높아지고 있다. ( ㉢ ) 옥상이나 베란다에서 채소를 키우는 사람들이 늘고 있는 것이다. ( ㉣ ) 도시 농업은 신선한 먹거리를 얻을 수 있을 뿐만 아니라 정서적 안정에도 도움이 된다.

**&lt;보기&gt;**: 또한 이웃과 함께 텃밭을 가꾸면서 공동체 의식을 회복하는 효과도 있다.

**위치는?**
① ㉠ ② ㉡ ③ ㉢ ④ ㉣

**答案**: ④

> "또한(而且)"是递进，补充说明 ㉣ 提到的好处，放在 ㉣ 后最自然。

---

### 练习4

( ㉠ ) 현대 사회에서 스마트폰은 생활 필수품이 되었다. ( ㉡ ) 그런데 스마트폰을 장시간 사용하면 목과 어깨에 통증이 생길 수 있다. ( ㉢ ) 이를 '거북목 증후군'이라고 한다. ( ㉣ ) 전문가들은 1시간에 한 번씩 스트레칭을 하고 바른 자세를 유지할 것을 권장한다.

**&lt;보기&gt;**: 특히 고개를 숙이고 스마트폰을 보는 자세가 문제인데, 이 자세는 목에 큰 부담을 준다.

**위치는?**
① ㉠ ② ㉡ ③ ㉢ ④ ㉣

**答案**: ②

> 插入句具体说明 ㉡ 的原因（低头看手机），放在 ㉡ 后、指示词 "이를" 之前。

---

### 练习5

( ㉠ ) 기후 변화로 인해 전 세계적으로 이상 기후 현상이 자주 발생하고 있다. ( ㉡ ) 한국에서도 여름철 폭염과 겨울철 한파가 해마다 심해지고 있다. ( ㉢ ) 이러한 문제를 해결하기 위해 탄소 배출을 줄이는 것이 급선무이다. ( ㉣ ) 개인적으로는 대중교통 이용과 에너지 절약 습관을 길러야 한다.

**&lt;보기&gt;**: 예를 들어 지난해에는 기록적인 폭우로 큰 피해가 발생하기도 했다.

**위치는?**
① ㉠ ② ㉡ ③ ㉢ ④ ㉣

**答案**: ②

> "예를 들어(例如)"是举例，具体论证 ㉡ 的异常气候现象，放在 ㉡ 后。

---

### 练习6

( ㉠ ) 한국의 전통 시장은 오랜 역사를 가지고 있다. ( ㉡ ) 그런데 대형 마트와 온라인 쇼핑의 등장으로 전통 시장의 매출이 크게 줄었다. ( ㉢ ) 이에 정부는 전통 시장을 살리기 위해 다양한 지원책을 마련하고 있다. ( ㉣ ) 예를 들어 온누리 상품권 할인, 시장 시설 현대화, 문화 행사 개최 등이 있다.

**&lt;보기&gt;**: 그 결과 많은 상인들이 생계에 어려움을 겪고 있다.

**위치는?**
① ㉠ ② ㉡ ③ ㉢ ④ ㉣

**答案**: ②

> "그 결과(其结果)"紧接 ㉡ 的原因，放在 ㉡ 后说明结果，然后 ㉢ 提出解决方案。

---

### 练习7

( ㉠ ) 반려동물을 키우는 가구가 점점 늘어나고 있다. ( ㉡ ) 반려동물은 사람에게 정서적 안정과 행복감을 준다. ( ㉢ ) 그러나 반려동물의 유기 문제도 심각해지고 있다. ( ㉣ ) 반려동물을 키우기 전에 충분한 책임감과 경제적 여건을 갖추었는지 진지하게 고려해야 한다.

**&lt;보기&gt;**: 반면에 반려동물을 충동적으로 입양했다가 버리는 사람도 많다.

**위치는?**
① ㉠ ② ㉡ ③ ㉢ ④ ㉣

**答案**: ②

> "반면에(反之)"是转折，从 ㉡ 的正面转向 ㉢ 的负面。放在 ㉡ 后、㉢(유기) 前最恰当。

---

### 练习8

( ㉠ ) 인공지능의 발달로 많은 직업이 사라질 것이라는 전망이 나오고 있다. ( ㉡ ) 단순 반복 업무는 로봇이 대체할 가능성이 높다. ( ㉢ ) 그래서 창의력과 감성 능력이 더욱 중요해질 것이다. ( ㉣ ) 미래 사회에서 살아남기 위해서는 끊임없이 배우고 변화하는 자세가 필요하다.

**&lt;보기&gt;**: 반대로 창의적 사고가 필요한 직업은 오히려 가치가 높아질 수 있다.

**위치는?**
① ㉠ ② ㉡ ③ ㉢ ④ ㉣

**答案**: ②

> "반대로(相反)" 用于与 ㉡ 对比——单纯工作消失 vs 创意工作价值升高，放在 ㉡ 后、㉢ 前。

## 五、高频逻辑连接词总表

| 连接词               | 功能   | 位置倾向   |
| -------------------- | ------ | ---------- |
| 그러나 / 하지만      | 转折   | 正→反 之间 |
| 따라서 / 그래서      | 因果   | 原因→结果  |
| 예를 들어 / 가령     | 举例   | 总论→具体  |
| 즉 / 다시 말하면     | 换言   | 难句→解释  |
| 게다가 / 뿐만 아니라 | 递进   | 论点→补充  |
| 이처럼 / 이렇듯      | 总结   | 论述→总结  |
| 그 결과              | 结果   | 行为→结果  |
| 반면에 / 반대로      | 对比   | A→B 对立   |
| 한편                 | 另一面 | 视角转换处 |
| 그런데               | 转话题 | 话题切换处 |

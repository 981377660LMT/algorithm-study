# 相近表达选择类 (비슷한 의미 고르기) — 5~8题

> 밑줄 친 부분과 의미가 비슷한 것을 고르십시오.

---

## 一、核心考点：语法同义替换

### A. 习惯/反复

| 原文           | 同义替换                         |
| -------------- | -------------------------------- |
| -곤 하다       | 자주 -ㄴ다/는다, -는 경우가 많다 |
| -기 일쑤이다   | 자주/늘 -ㄴ다/는다               |
| -(으)ㄹ 때마다 | -면 항상/늘                      |

例：주말에 등산을 하**곤 한다** = 주말에 자주 등산을 **한다**

### B. 推测/判断

| 原文              | 同义替换                       |
| ----------------- | ------------------------------ |
| -(으)ㄹ 것 같다   | -(으)ㄹ 듯하다, -(으)ㄹ 것이다 |
| -나 보다          | -(으)ㄴ/는 모양이다            |
| -아/어 보이다     | -(으)ㄴ/는 것 같다             |
| -(으)ㄹ 리가 없다 | 절대 -(으)ㄹ 수 없다           |

例：피곤한**가 보다** = 피곤한 **모양이다**

### C. 因果

| 原文         | 同义替换              |
| ------------ | --------------------- |
| -아/어서     | -기 때문에, -(으)니까 |
| -는 바람에   | -아/어서 (消极结果)   |
| -는 탓에     | -기 때문에 (消极)     |
| -는 덕분에   | -기 때문에 (积极)     |
| -(으)로 인해 | -때문에               |

例：비가 오**는 바람에** 못 갔다 = 비가 **와서** 못 갔다

### D. 让步/转折

| 原文             | 同义替换                     |
| ---------------- | ---------------------------- |
| -아/어도         | -더라도, -(으)ㄹ지라도       |
| -지만            | -(으)나, -(으)ㄴ/는 반면(에) |
| -(으)ㄴ/는데(도) | -지만                        |

例：비싸**지만** 좋다 = 비싸**나** 좋다

### E. 程度

| 原文                    | 同义替换      |
| ----------------------- | ------------- |
| -(으)ㄹ 정도로          | -(으)ㄹ 만큼  |
| 얼마나 ~(으)ㄴ지 모르다 | 매우/아주 ~다 |
| -기 짝이 없다           | 매우/너무 ~다 |

例：**얼마나** 기쁜**지 몰라요** = **매우** 기쁘다

### F. 必然/不得不

| 原文                | 同义替换                        |
| ------------------- | ------------------------------- |
| -(으)ㄹ 수밖에 없다 | -아/어야 한다, -지 않을 수 없다 |
| -기 마련이다        | 당연히/반드시 -게 되다          |
| -는 법이다          | -기 마련이다                    |

例：인정할 **수밖에 없다** = 인정**하지 않을 수 없다**

### G. 不仅/递进

| 原文                | 同义替换                       |
| ------------------- | ------------------------------ |
| -(으)ㄹ 뿐만 아니라 | -(으)ㄹ뿐더러, -는 것은 물론   |
| -기는커녕           | -(으)ㄹ 뿐만 아니라 ~도 못하다 |
| -는데다가           | -(으)ㄹ 뿐만 아니라            |

例：맛있을 **뿐만 아니라** 건강에도 좋다 = 맛있을**뿐더러** 건강에도 좋다

### H. 近似/等同

| 原文              | 同义替换                               |
| ----------------- | -------------------------------------- |
| -는 셈이다        | -는 것과 같다, -는 것이나 마찬가지이다 |
| -다시피 하다      | 거의 ~다                               |
| -(으)ㄴ/는 편이다 | 비교적/대체로 ~다                      |

例：거의 다 한 **셈이다** = 거의 다 한 **것과 같다**

### I. 意图/目的

| 原文       | 同义替换              |
| ---------- | --------------------- |
| -(으)려고  | -기 위해(서)          |
| -(으)러    | -(으)려고, -기 위해   |
| -도록      | -게, -기 위해         |
| -(으)ㄹ 겸 | -(으)려는 목적도 있고 |

例：건강해지**려고** 운동한다 = 건강해지**기 위해** 운동한다

### J. 条件

| 原文         | 同义替换                |
| ------------ | ----------------------- |
| -아/어야     | -(으)면 -(으)ㄹ 수 있다 |
| -기만 하면   | -(으)면 반드시/바로     |
| -(으)ㄹ 경우 | -(으)면                 |

### K. 非意愿变化

| 原文       | 同义替换      |
| ---------- | ------------- |
| -게 되다   | -아/어지다    |
| -아/어지다 | 점점 ~게 되다 |

例：한국어를 좋아하**게 됐다** = 한국어가 좋아**졌다**

## 二、实战练习 (20题)

### 练习1

밑줄 친 부분과 의미가 비슷한 것을 고르십시오.

그 사람은 주말마다 산에 가**곤 했다**.
① 산에 간 적이 있었다
② 산에 가야 했다
③ 산에 자주 갔다
④ 산에 가고 싶어했다

**答案**: ③

### 练习2

날씨가 추워**진** 탓에 감기 환자가 늘었다.
① 추워지면서
② 추워지려고
③ 추워졌기 때문에
④ 추워질 뿐만 아니라

**答案**: ③

### 练习3

이 일은 나 **아니면** 할 수 있는 사람이 없다.
① 나밖에
② 나라도
③ 나조차
④ 나마저

**答案**: ①

### 练习4

아무리 바빠**도** 아침은 꼭 먹는다.
① 바쁘면
② 바쁘니까
③ 바쁘**더라도**
④ 바쁘기 때문에

**答案**: ③

### 练习5

요즘 한국어 실력이 많이 좋아**졌다**.
① 좋아지곤 했다
② 좋아지게 됐다
③ 좋아질 뻔했다
④ 좋아지려고 했다

**答案**: ②

### 练习6

그 영화는 볼 **만했다**.
① 볼 수 없었다
② 볼 가치가 있었다
③ 봐야 했다
④ 보기 어려웠다

**答案**: ②

### 练习7

그 소식을 듣고 **얼마나** 놀랐는**지 모른다**.
① 조금 놀랐다
② 놀라지 않았다
③ 매우 놀랐다
④ 놀랄 뻔했다

**答案**: ③

### 练习8

노력하면 좋은 결과가 나오**기 마련이다**.
① 나올 수 없다
② 나올 리가 없다
③ 반드시 나오게 된다
④ 나오기 어렵다

**答案**: ③

### 练习9

감사하**기는커녕** 오히려 화를 냈다.
① 감사하지 않을 뿐만 아니라
② 감사하니까
③ 감사해서
④ 감사한 편이다

**答案**: ①

### 练习10

시험을 앞두고 긴장하**지 않을 수 없었다**.
① 긴장하지 않았다
② 긴장할 수밖에 없었다
③ 긴장할 필요가 없었다
④ 긴장하곤 했다

**答案**: ②

### 练习11

그 식당은 비싼 **반면** 맛이 좋다.
① 비싸고
② 비싸서
③ 비싸**지만**
④ 비싸니까

**答案**: ③

### 练习12

저도 모르게 울**게 됐다**.
① 울어야 했다
② 울고 싶었다
③ 자연스럽게 울**었다**
④ 울지 않았다

**答案**: ③

### 练习13

이 영화는 거의 다 본 **셈이다**.
① 본 것과 같다
② 보고 싶다
③ 본 적이 있다
④ 볼 수 있다

**答案**: ①

### 练习14

건강을 위해 매일 운동하**는 편이다**.
① 반드시 운동한다
② 비교적 자주 운동한다
③ 운동을 안 한다
④ 운동을 시작했다

**答案**: ②

### 练习15

피곤한**가 봐요**.
① 피곤할 거예요
② 피곤한 **모양이에요**
③ 피곤해야 해요
④ 피곤하면 좋겠어요

**答案**: ②

### 练习16

수영할 **줄 안다**.
① 수영을 잘한다
② 수영할 **수 있다**
③ 수영을 좋아한다
④ 수영을 배웠다

**答案**: ②

### 练习17

이렇게 하**다가는** 큰일 난다.
① 이렇게 하면
② 이렇게 하니까
③ 이렇게 계속 하**면** 결국
④ 이렇게 해서

**答案**: ③

### 练习18

도서관에 책을 빌리**러** 갔다.
① 빌리**려고** 갔다
② 빌리**면서** 갔다
③ 빌리**고** 갔다
④ 빌리**다가** 갔다

**答案**: ①

### 练习19

그가 거짓말을 할 **리가 없다**.
① 할 수도 있다
② **절대** 하지 **않을 것이다**
③ 할 줄 모른다
④ 한 적이 없다

**答案**: ②

### 练习20

맛있을 **뿐만 아니라** 가격도 싸다.
① 맛있**는 데다가** 가격도 싸다
② 맛있기 때문에 가격도 싸다
③ 맛있지만 가격이 비싸다
④ 맛있는 반면 가격도 싸다

**答案**: ①

# TOPIK II 高级 (5-6级) 备考计划

**方案结构：**

1. **考试概况** — TOPIK II 题型、分值、时间分配，特别详解了写作4道题的要求
2. **学习方法论** — 듣기/읽기/쓰기/어휘文法 四大板块各自的核心问题和针对性训练策略
3. **推荐资源** — 教材(연세5/6、真题集)、在线资源(官方真题、TTMIK、세바시)、工具(Anki、Papago、맞춤법 검사기)
4. **6个月分阶段计划**：
   - Phase 1 (1-2月)：夯实词汇语法基础，精听精读
   - Phase 2 (3-4月)：限时强化，攻克写作
   - Phase 3 (5-6月)：冲刺模考，查漏补缺
5. **每日作息示例** — 适合工作日，每天2.5小时
6. **评分策略** — 保5级(200分)和冲6级(235分)的各科目标分配
7. **高频写作话题 + 议论文万能框架** — 8大常考主题 + 서론/본론/결론模板
8. **进度追踪表** — 真题成绩记录和每周复盘区域

**核心备考建议：**

- 듣기+읽기 是基本盘，靠刷真题 + 限时训练稳拿 80+
- 쓰기 是决定5级/6级的分水岭，每周至少写2篇议论文并找人批改
- 词汇用 Anki 间隔重复，重点掌握汉字词对应规律（中文母语者的巨大优势）
- 至少完整模拟考5次以上，适应3小时连续考试节奏

## 一、考试概况

| 项目         | 内容                                                 |
| ------------ | ---------------------------------------------------- |
| **考试名称** | TOPIK II (한국어능력시험 II)                         |
| **目标等级** | 5级 (190~229分) / 6级 (230~300分)                    |
| **满分**     | 300分 (듣기100 + 읽기100 + 쓰기100)                  |
| **考试时间** | 第1场：듣기(60min) + 쓰기(50min)；第2场：읽기(70min) |
| **题型**     | 듣기50题(选择) + 읽기50题(选择) + 쓰기4题(主观/写作) |
| **考试频率** | 韩国境内每年6次；海外每年约3-4次                     |

### 쓰기(写作) 题型详解 — 拉分关键

| 题号 | 类型            | 分值 | 要求                                 |
| ---- | --------------- | ---- | ------------------------------------ |
| 51   | 实用文填空      | 10分 | 根据上下文填写2个句子                |
| 52   | 说明文填空      | 10分 | 根据上下文填写2个句子                |
| 53   | 图表/数据说明文 | 30分 | 200~300字，根据图表/调查数据写说明文 |
| 54   | 议论文          | 50分 | 600~700字，给定话题写议论文          |

## 二、学习方法论

### 1. 듣기 (听力) — 目标 ≥ 80分

**核心问题**：语速快、词汇学术化、长对话/讲演逻辑链长

**方法**：

- **精听 + 泛听 结合**
  - 精听：逐句听写历年真题音频 → 对照原文 → 标注生词和语法 → 跟读模仿
  - 泛听：KBS 뉴스, EBS 다큐멘터리, 팟캐스트(세바시 TED Korean) 作为日常背景音
- **题型专项突破**
  - 1~20题 (日常对话)：重点掌握惯用表达和连接语
  - 21~36题 (中长对话)：训练"先听问题 → 预判关键词"的应试技巧
  - 37~50题 (讲演/讨论)：练习边听边记笔记，抓主旨、论点、态度
- **影子跟读法 (Shadowing)**：每天15分钟，跟读新闻或讲座音频，提升语感和反应速度

### 2. 읽기 (阅读) — 目标 ≥ 80分

**核心问题**：时间紧 (70分钟50题，平均84秒/题)、高级词汇密集、长文逻辑复杂

**方法**：

- **限时训练**：从一开始就严格计时做题，培养"扫读 → 定位 → 精读"的习惯
- **题型策略**
  - 1~12题 (语法/词汇填空)：靠平时积累，30秒/题快速过
  - 13~24题 (排序/主旨/态度)：先看选项，再读文章
  - 25~50题 (长文理解)：先读问题 → 带着问题读文章 → 排除法选答案
- **词汇体系构建**
  - 基础词汇：TOPIK 高级必备词汇 (약 3000~5000개) — 用 Anki 间隔重复
  - 汉字词：系统学习韩语汉字音对应规律 (e.g., 学 → 학, 生 → 생)，可以快速猜词
  - 惯用语/四字成语/俗语：每天积累5个，做笔记本

### 3. 쓰기 (写作) — 目标 ≥ 60分 (高级拉分项)

**核心问题**：唯一主观题，直接决定能否过5级/6级

**方法**：

- **51-52题 (填空)**：背模板句式，练习根据上下文推断和衔接
  - 常见功能：원인/이유, 조건, 대조, 예시, 결론
  - 高频语法：-기 때문에, -(으)ㄹ 뿐만 아니라, -는 반면에, -(으)므로, -다고 볼 수 있다
- **53题 (图表说明文)**：固定模板 + 数据描述
  - 开头：조사/설문의 주제와 대상 소개
  - 中间：수치 변화 묘사 (증가하다, 감소하다, 차지하다, 나타나다)
  - 结尾：정리 및 전망
- **54题 (议论文)**：论证结构 + 高级表达
  - 结构：서론(话题引入+立场) → 본론1(论点+论据) → 본론2(反驳/补充) → 결론(总结+展望)
  - **每周写2篇 → 找韩语老师/语伴批改 → 整理错误 → 背诵好句**
  - 积累高频议论文主题：환경, 기술, 교육, 사회문제, 문화, 경제

### 4. 어휘/문법 (词汇/语法) — 贯穿始终

- **语法清单**：系统学习 TOPIK II 高级语法约 150~200个
  - 推荐教材：《한국어능력시험 TOPIK II 문법》/ 연세 한국어 5-6
  - 每天学3个新语法 → 造句 → 在真题中标注出现位置
- **词汇记忆**：
  - Anki/Quizlet 间隔重复，每天新词30个 + 复习100个
  - 按主题分组：정치, 경제, 사회, 문화, 과학, 환경, 교육, 의학
  - 重点记忆：접두사/접미사 (e.g., 불-, 비-, -적, -성, -화)

## 四、分阶段学习计划 (6个月)

> 假设每天可投入 **2~3小时**，周末 **4~5小时**

### Phase 1: 基础夯实 (第1~2月)

**目标**：建立词汇/语法体系，熟悉题型

| 时间段 | 内容                                      | 每日时长 |
| ------ | ----------------------------------------- | -------- |
| 每天   | Anki 词汇 (新词30 + 复习)                 | 30min    |
| 每天   | 语法学习 (3个/天) + 造句练习              | 30min    |
| 每天   | 精听1套真题듣기 (逐句听写)                | 45min    |
| 每天   | 읽기 精读2篇长文 (标注生词语法)           | 30min    |
| 周末   | 쓰기 练习 (51-52题模板背诵，53题图表描述) | 1h       |
| 周末   | 做1套完整真题 (不计时，重在理解)          | 2h       |

**里程碑**：

- [ ] 掌握高级核心语法 100个
- [ ] 词汇量达到 5000+
- [ ] 完成 5套真题精析
- [ ] 能写出基本的53题图表说明文

### Phase 2: 强化训练 (第3~4月)

**目标**：提速提正确率，攻克写作

| 时间段  | 内容                             | 每日时长 |
| ------- | -------------------------------- | -------- |
| 每天    | Anki 词汇 + 惯用语/俗语积累      | 30min    |
| 每天    | 限时듣기训练 (1套60min)          | 60min    |
| 每天    | 限时읽기训练 (25题/35min)        | 40min    |
| 每天    | 泛听 (新闻/세바시) + 影子跟读    | 15min    |
| 每周2次 | 54题议论文写作 + 批改            | 1h/次    |
| 周末    | 完整模拟考 (严格计时) + 错题分析 | 3h       |

**里程碑**：

- [ ] 듣기 稳定 75+/100
- [ ] 읽기 稳定 75+/100 且70分钟内完成
- [ ] 能独立写出600字议论文，结构完整
- [ ] 掌握高级语法 180+
- [ ] 完成 10套真题

### Phase 3: 冲刺模考 (第5~6月)

**目标**：查漏补缺，稳定发挥

| 时间段  | 内容                             | 每日时长 |
| ------- | -------------------------------- | -------- |
| 每天    | 错题本复习 (듣기+읽기高频错误点) | 20min    |
| 每天    | Anki 词汇维护 (以复习为主)       | 20min    |
| 每天    | 限时做半套真题 (듣기或읽기)      | 40min    |
| 每天    | 쓰기 背诵范文好句 + 默写         | 30min    |
| 每周2次 | 完整模拟考 (严格计时，模拟考场)  | 3h/次    |
| 每周1次 | 54题议论文限时写作 (50min)       | 50min    |
| 考前2周 | 每天1套完整真题冲刺              | 3h       |

**里程碑**：

- [ ] 模拟考总分稳定 200+ (冲5级) / 240+ (冲6级)
- [ ] 쓰기 54题能在40分钟内完成结构完整的议论文
- [ ] 错题本中反复出错的知识点清零
- [ ] 完成 20套真题

## 六、评分策略与心态

### 各科目标分 (5级: 190分 / 6级: 230分)

| 目标  | 듣기 | 읽기 | 쓰기 | 总分 |
| ----- | ---- | ---- | ---- | ---- |
| 保5级 | 75   | 75   | 50   | 200  |
| 冲6级 | 85   | 85   | 65   | 235  |

### 关键策略

1. **듣기 + 읽기 是基本盘**：客观题更容易拿稳定分，目标 80+
2. **쓰기 是分水岭**：51-52题必须拿满 (20分)，53题争取25+ (30分)，54题争取30+ (50分)
3. **不要死磕难题**：듣기/읽기遇到不确定的快速标记跳过，回头再看
4. **考前模拟节奏**：严格模拟至少5次完整考试，适应连续3小时的考试强度

### 写作评分标准 (阅卷重点)

- **내용 (内容)**: 是否切题、论点充分
- **구성 (结构)**: 段落逻辑、衔接词使用
- **언어 사용 (语言)**: 语法准确度、词汇多样性、书面语风格
- **사회언어학적 능력 (社会语言能力)**: 是否使用恰当的正式书面体 (-ㅂ니다/습니다体)

## 七、高频写作话题与框架

### 54题常考主题

1. 현대 사회에서 기술의 역할 (技术的角色)
2. 환경 보호와 경제 발전 (环保vs经济)
3. 교육 제도의 문제점과 개선 방안 (教育问题)
4. 세대 간 갈등의 원인과 해결 (代际冲突)
5. 소셜 미디어의 영향 (社交媒体)
6. 전통문화의 보존 (传统文化保存)
7. 일과 삶의 균형 (工作生活平衡)
8. 인공지능과 인간의 미래 (AI与人类)

### 议论文万能框架

```
[서론] 최근/오늘날 ... (으)ㄴ/는 것이 사회적으로 큰 관심을 받고 있다.
       이에 대해 ...다는 의견과 ...다는 의견이 대립하고 있는데,
       나는 ...다고 생각한다.

[본론1] 우선/먼저, ...기 때문이다.
        예를 들어, ... (구체적 사례)
        따라서 ...다고 할 수 있다.

[본론2] 또한/뿐만 아니라, ...도 고려해야 한다.
        물론 ...다는 반론도 있지만,
        이는 ...다는 점에서 한계가 있다.

[결론] 결론적으로/종합하면, ...는 것이 바람직하다.
       앞으로 ...기 위해서는 (개인/사회/정부)의 노력이 필요하다.
```

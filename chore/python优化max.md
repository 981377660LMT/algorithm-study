- 变量查找
  len()、int()和更多内置函数也是如此；每次使用它们都需要进行昂贵的查找。
- 动态属性查找
  即使在单个类中，解释器也`不知道该类的对象上存在哪些属性`(所以 slots 非常关键)。
  这意味着它需要对象的动态表示来跟踪属性。在 Python 中查找属性非常快，但仍然比静态属性列表要慢得多。
- 变量类型查找
  解释器不知道任何变量的类型。
  这意味着对变量的任何操作都需要查找它是什么类型，以便弄清楚如何对该类型执行操作。

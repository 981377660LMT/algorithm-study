#2078 **两栋颜色不同且距离最远的房子**  
简单
`next活用`

```Python
class Solution:
    def maxDistance(self, colors: List[int]) -> int:
      n = len(colors)
      cand1 = next((i for i in range(n) if colors[i] != colors[-1]), 0)
      cand2 = next((i for i in range(n - 1, -1, -1) if colors[i] != colors[0]), n - 1)
      return max(n - 1 - cand1, cand2)
```

- ReAct 是 Agent 的心脏：思考一步，行动一步，观察结果，再思考下一步
- ReAct：Reasoning + Acting，推理与行动的交织循环
- Reason（思考）→ Act（行动）→ Observe（观察）

- 为什么有效：让 LLM 能获取新信息、修正错误、追溯过程
- 终止条件：任务完成 / 结果收敛 / 最大轮数 / 用户中断 + 预算/超时护栏
  - 生产环境建议 MaxIterations = 10-15
  - 强制 MinIterations = 1，确保至少做一次真正的工具调用
  - 提前停（不是等烧完 MaxIterations）
  - 相似性检测：如果连续两次观察结果高度相似，强制停止或换策略
  - 在 Prompt 里明确："你必须使用工具获取信息，不能直接回答"

这份文档介绍了如何在本地运行 LangGraph 应用程序。以下是核心步骤总结：

### 1. 准备工作
确保你拥有一个 [LangSmith API Key](https://smith.langchain.com/settings)。

### 2. 创建并初始化项目
使用脚手架创建新项目，进入目录并安装依赖：

```bash
# 1. 创建项目
npm create langgraph

# 2. 进入目录
cd path/to/your/app

# 3. 安装依赖
npm install
```

### 3. 配置环境变量
在项目根目录创建 `.env` 文件（参考 `.env.example`），并填入你的 API Key：
```bash
LANGSMITH_API_KEY=lsv2...
```

### 4. 启动本地服务器
使用 LangGraph CLI 启动开发服务器：
```bash
npx @langchain/langgraph-cli dev
```
启动成功后，API 默认运行在 `http://localhost:2024`。

### 5. 测试与调试
*   **可视化调试 (Studio)**：访问终端输出中提供的 URL（通常是 `https://smith.langchain.com/studio/?baseUrl=http://127.0.0.1:2024`）来使用 LangGraph Studio Web UI。
    *   *注意：Safari 用户需添加 `--tunnel` 参数启动。*
*   **API 测试**：你可以使用 `@langchain/langgraph-sdk` 或 `curl` 向本地服务器发送请求。

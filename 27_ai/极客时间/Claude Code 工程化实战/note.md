- 能力的天平正在从“执行力”向“创意力”和“组织力”倾斜。—— 而这些能力往往并不属于传统的“程序员”，而是程序员或开发者的加强版——也就是极客
- 极客精神的核心从来不是“会写代码”，而是“用技术解决问题”
- Claude Code 的真正身份是：`一个可编程、可扩展、可组合的 AI Agent 框架。`
